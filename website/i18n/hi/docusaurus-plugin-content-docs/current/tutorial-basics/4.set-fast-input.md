# त्वरित इनपुट कीबोर्ड सक्षम करें

फोन कनेक्ट होने के बाद स्वचालित रूप से त्वरित इनपुट कीबोर्ड पर स्विच हो जाएगा।

## त्वरित इनपुट कीबोर्ड क्यों सेट करें

त्वरित इनपुट कीबोर्ड TikMatrix का आवश्यक उपकरण है, स्वचालित बहुभाषी समर्थन प्रदान करता है। इसके बिना, TikMatrix की टेक्स्ट इनपुट सुविधाएं (जैसे कमेंट, संदेश और पोस्ट शीर्षक) ठीक से काम नहीं कर सकती हैं।

## त्वरित इनपुट कीबोर्ड को मैन्युअल रूप से कैसे सक्षम करें

1. त्वरित इनपुट कीबोर्ड का उपयोग करने वाले डिवाइस चुनें।
2. `कस्टम ADB कमांड` - `त्वरित इनपुट सक्षम करें` पर क्लिक करें।

## टेक्स्ट कैसे इनपुट करें

TikMatrix में, आप कई तरीकों से टेक्स्ट इनपुट कर सकते हैं:

### विधि 1: सीधे कीबोर्ड इनपुट ⌨️ (नई सुविधा)

⭐ **कीबोर्ड इनपुट नियंत्रण** - अब आप सीधे कंप्यूटर कीबोर्ड का उपयोग करके फोन पर टेक्स्ट इनपुट कर सकते हैं, जिससे टेक्स्ट इनपुट अधिक तेज़ और सुविधाजनक हो जाता है।

1. उपयोग करने के लिए डिवाइस चुनें।
2. फोन स्क्रीन पर किसी भी टेक्स्ट इनपुट बॉक्स पर क्लिक करें।
3. सीधे कंप्यूटर कीबोर्ड पर टाइप करना शुरू करें - टेक्स्ट फोन पर रियल-टाइम में प्रदर्शित होगा।

### विधि 2: इनपुट पैनल

1. फोन स्क्रीन पर क्लिक करें, बड़ी स्क्रीन विस्तारित करें, और दाईं ओर टूलबार में `इनपुट` बटन पर क्लिक करें।
2. इनपुट बॉक्स में टेक्स्ट दर्ज करें, भेजें पर क्लिक करें।

## फोन और कंप्यूटर के बीच टेक्स्ट कैसे कॉपी-पेस्ट करें

### फोन क्लिपबोर्ड टेक्स्ट को कंप्यूटर क्लिपबोर्ड पर कॉपी करें

1. कॉपी करने के लिए फोन चुनें, `Ctrl+C` कुंजी दबाएं, कॉपी सफल

### कंप्यूटर क्लिपबोर्ड टेक्स्ट को फोन इनपुट बॉक्स में पेस्ट करें

1. पेस्ट करने के लिए डिवाइस चुनें, `Ctrl+V` कुंजी दबाएं, पेस्ट सफल
