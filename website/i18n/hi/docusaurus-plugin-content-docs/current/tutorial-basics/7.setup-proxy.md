# प्रॉक्सी सेट करें

आप प्रत्येक फोन के लिए स्वतंत्र प्रॉक्सी IP पता सेट कर सकते हैं।

> **अस्वीकरण:** इस पृष्ठ पर अनुशंसित सभी वेबसाइटें इंटरनेट से आती हैं, इस साइट से संबंधित नहीं हैं। उपयोग करने से पहले कृपया स्वयं उनकी प्रभावशीलता सत्यापित करें। हम कोई गारंटी प्रदान नहीं करते हैं।

## 1. आवासीय VPS प्रॉक्सी सेट करें (यदि आपने प्रॉक्सी IP खरीदी है, तो इस चरण को छोड़ सकते हैं)

1. लक्ष्य देश से आवासीय प्रॉक्सी IP VPS खरीदें (मूल IP और दोहरी ISP VPS बेहतर है)।
2. VPS पर VPN त्वरित रूप से सेट करने के लिए [https://github.com/MHSanaei/3x-ui](https://github.com/MHSanaei/3x-ui) का उपयोग करें:

    ```shell
    bash <(curl -Ls https://raw.githubusercontent.com/MHSanaei/3x-ui/refs/tags/v2.6.0/install.sh)
    # प्रॉम्प्ट का पालन करके VPN सेटअप पूर्ण करें।
    ```

3. इंस्टॉलेशन पूर्ण होने के बाद, x-ui वेब इंटरफ़ेस पर जाएं। आप इंस्टॉलेशन के बाद टर्मिनल आउटपुट में URL और लॉगिन क्रेडेंशियल पा सकते हैं।
4. x-ui में नया उपयोगकर्ता बनाएं, प्रोटोकॉल को `SOCKS5` पर सेट करें, और आवश्यकतानुसार पोर्ट और अन्य सेटिंग्स कॉन्फ़िगर करें।
5. उपयोगकर्ता बनाने के बाद, आप x-ui इंटरफ़ेस में प्रॉक्सी IP और पोर्ट देखेंगे।
6. फोन पर NekoBox एप्लिकेशन में प्रॉक्सी IP और पोर्ट का उपयोग करें।

## 2. आवासीय प्रॉक्सी IP खरीदें (यदि आपने VPS सेट किया है, तो इस चरण को छोड़ सकते हैं)

आप VPS सेट किए बिना सीधे आवासीय प्रॉक्सी IP भी खरीद सकते हैं।

## 3. फोन पर प्रॉक्सी सक्षम करें

1. फोन पर [NekoBoxForAndroid](https://github.com/MatsuriDayo/NekoBoxForAndroid) क्लाइंट APK डाउनलोड और इंस्टॉल करें।
2. NekoBox एप्लिकेशन खोलें, नया प्रॉक्सी जोड़ने के लिए `+` बटन पर क्लिक करें।
3. `SOCKS5` या अन्य प्रॉक्सी प्रकार चुनें।
4. प्रॉक्सी IP और पोर्ट दर्ज करें, फिर `सहेजें` पर क्लिक करें।
5. प्रॉक्सी सक्षम करने के लिए `प्रारंभ` बटन पर क्लिक करें।
6. प्रॉक्सी सक्षम करने के बाद, आप NekoBox एप्लिकेशन में कनेक्शन स्थिति जांच सकते हैं।

### चीनी उपयोगकर्ताओं के लिए नोट

यदि आप चीन में हैं, तो `SOCKS5` प्रोटोकॉल ठीक से काम नहीं कर सकता। आपको फ्रंट-एंड प्रॉक्सी के रूप में एन्क्रिप्टेड प्रॉक्सी (`VLESS`) जोड़ने की आवश्यकता है। आप निम्नलिखित उपकरणों का उपयोग कर सकते हैं:

1. NekoBox एप्लिकेशन में `VLESS` प्रॉक्सी जोड़ें।
2. NekoBox एप्लिकेशन में `SOCKS5` प्रॉक्सी जोड़ें।
3. NekoBox एप्लिकेशन में `चेन` प्रॉक्सी जोड़ें। फ्रंट-एंड प्रॉक्सी के रूप में `VLESS` प्रॉक्सी चुनें, बैक-एंड प्रॉक्सी के रूप में `SOCKS5` प्रॉक्सी।
