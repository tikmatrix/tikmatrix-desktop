# Work Profile कार्य प्रोफ़ाइल कॉन्फ़िगरेशन

TikMatrix प्रत्येक डिवाइस के लिए अलग से Work Profile उपयोगकर्ता कॉन्फ़िगर करने का समर्थन करता है, यह एंटरप्राइज़ डिवाइस या डुअल-ओपन ऐप का उपयोग करने के लिए बहुत उपयोगी है।

## Work Profile क्या है

Work Profile (कार्य प्रोफ़ाइल) Android की एक सुविधा है, जो एक ही डिवाइस पर एक स्वतंत्र कार्य वातावरण बनाने की अनुमति देती है। विभिन्न उपयोगकर्ता ID कॉन्फ़िगर करके, आप कर सकते हैं:

- एंटरप्राइज़ प्रबंधित डिवाइस पर TikMatrix का सामान्य रूप से उपयोग करें
- विभिन्न ऐप वातावरणों के लिए विभिन्न उपयोगकर्ता कॉन्फ़िगरेशन सेट करें
- अधिक सटीक डिवाइस प्रबंधन और अनुमति नियंत्रण प्राप्त करें

## Shelter टूल का उपयोग करके ऐप्स क्लोन करें

Work Profile कॉन्फ़िगर करने से पहले, आपको Shelter टूल का उपयोग करके TikTok और TikMatrix ऐप्स को क्लोन करना होगा:

### Shelter क्या है

Shelter एक ओपन सोर्स एप्लिकेशन है जो Android डिवाइस पर Work Profile बना और प्रबंधित कर सकता है। यह आपको पृथक कार्य वातावरण में डुप्लिकेट ऐप चलाने की अनुमति देता है।

### Shelter इंस्टॉल करें

1. [F-Droid](https://f-droid.org/packages/net.typeblog.shelter/) या [Google Play Store](https://play.google.com/store/apps/details?id=net.typeblog.shelter) से Shelter डाउनलोड करें
2. डिवाइस पर Shelter इंस्टॉल और खोलें
3. Work Profile बनाने के लिए सेटअप विज़ार्ड का पालन करें

### आवश्यक ऐप्स क्लोन करें

Shelter सेट करने के बाद, आपको TikTok और TikMatrix ऐप्स को क्लोन करना होगा:

1. **TikTok ऐप क्लोन करें**:
   - Shelter खोलें और "मुख्य इंटरफ़ेस" टैब पर जाएं
   - ऐप सूची में TikTok खोजें
   - "कार्य प्रोफ़ाइल में क्लोन करें" बटन पर क्लिक करें
   - क्लोन प्रक्रिया पूर्ण होने तक प्रतीक्षा करें

2. **TikMatrix ऐप क्लोन करें**:
   - Shelter में, ऐप सूची में TikMatrix खोजें
   - "कार्य प्रोफ़ाइल में क्लोन करें" बटन पर क्लिक करें
   - क्लोन संचालन की पुष्टि करें

### क्लोन सफलता की पुष्टि करें

सफल क्लोन के बाद:

- आप ऐप ड्रॉअर में ब्रीफकेस आइकन के साथ TikTok और TikMatrix देखेंगे
- ये ऐप के Work Profile संस्करण हैं
- मुख्य प्रोफ़ाइल में मूल ऐप अपरिवर्तित रहते हैं

## Work Profile कैसे कॉन्फ़िगर करें

### 1. डिवाइस टूलबार खोलें

जब आपका डिवाइस कनेक्ट हो और TikMatrix मुख्य इंटरफ़ेस में प्रदर्शित हो:

1. बड़ी स्क्रीन मोड में प्रवेश करने के लिए डिवाइस कार्ड पर डबल-क्लिक करें
2. डिवाइस स्क्रीन के दाईं ओर एक टूलबार दिखाई देगा
3. टूलबार डिफ़ॉल्ट रूप से संकुचित अवस्था में होता है, माउस होवर करने पर स्वचालित रूप से विस्तारित होता है

### 2. Work Profile बटन खोजें

टूलबार के नीचे, आप ब्रीफकेस आइकन वाला बटन देखेंगे, यह Work Profile कॉन्फ़िगरेशन बटन है।

### 3. उपयोगकर्ता ID सेट करें

1. ब्रीफकेस आइकन बटन पर क्लिक करें
2. पॉप-अप संवाद में उपयोगकर्ता ID दर्ज करें (उदाहरण: 10)
3. "सहेजें" बटन पर क्लिक करें

### 4. कॉन्फ़िगरेशन की पुष्टि करें

सफल कॉन्फ़िगरेशन के बाद, सिस्टम "कार्य प्रोफ़ाइल उपयोगकर्ता सेटिंग्स सहेजी गई" का संदेश प्रदर्शित करेगा।

## उपयोगकर्ता ID विवरण

### सामान्य उपयोगकर्ता ID

- **0**: मुख्य उपयोगकर्ता (डिफ़ॉल्ट उपयोगकर्ता)
- **10**: पहला कार्य प्रोफ़ाइल उपयोगकर्ता
- **11**: दूसरा कार्य प्रोफ़ाइल उपयोगकर्ता
- अधिक उपयोगकर्ता ID इसी तरह

### उपयोगकर्ता ID कैसे खोजें

यदि आप डिवाइस पर उपयोगकर्ता ID के बारे में अनिश्चित हैं, तो निम्नलिखित तरीके से खोज सकते हैं:

```bash
adb shell pm list users
```

या TikMatrix के डीबग टूल में निष्पादित करें:

```bash
pm list users
```

आउटपुट उदाहरण:

```text
Users:
  UserInfo{0:Owner:c13} running
  UserInfo{10:Work profile:1030} running
```

## कॉन्फ़िगरेशन फ़ाइल स्टोरेज

Work Profile कॉन्फ़िगरेशन स्वचालित रूप से `data/work_profile_user.json` फ़ाइल में सहेजा जाएगा, प्रारूप इस प्रकार है:

```json
{
  "डिवाइस_सीरियल_नंबर1": "10",
  "डिवाइस_सीरियल_नंबर2": "0",
  "डिवाइस_सीरियल_नंबर3": "11"
}
```

## डिवाइस कॉन्फ़िगरेशन प्रबंधित करें

### वर्तमान कॉन्फ़िगरेशन देखें

प्रत्येक डिवाइस का Work Profile कॉन्फ़िगरेशन स्वतंत्र है, आप कर सकते हैं:

1. प्रत्येक डिवाइस के लिए विभिन्न उपयोगकर्ता ID सेट करें
2. किसी भी समय मौजूदा डिवाइस के उपयोगकर्ता कॉन्फ़िगरेशन को संशोधित करें
3. कॉन्फ़िगरेशन साफ़ करें (खाली मान दर्ज करें और सहेजें ताकि कॉन्फ़िगरेशन हटा सकें)

### बैच प्रबंधन

यदि आपको बड़ी संख्या में डिवाइस प्रबंधित करने की आवश्यकता है, तो सीधे `data/work_profile_user.json` फ़ाइल संपादित कर सकते हैं:

1. TikMatrix ऐप बंद करें
2. कॉन्फ़िगरेशन फ़ाइल खोलें
3. JSON प्रारूप के अनुसार डिवाइस कॉन्फ़िगरेशन जोड़ें या संशोधित करें
4. TikMatrix पुनरारंभ करें

## समस्या निवारण

### सामान्य समस्याएं

#### Q: Work Profile सेट करने के बाद कमांड निष्पादन विफल

A: कृपया पुष्टि करें:

- उपयोगकर्ता ID सही है या नहीं
- डिवाइस पर संबंधित उपयोगकर्ता मौजूद है या नहीं
- उस उपयोगकर्ता तक पहुंचने के लिए पर्याप्त अनुमतियां हैं या नहीं

#### Q: Work Profile कॉन्फ़िगरेशन कैसे रद्द करें

A: कॉन्फ़िगरेशन संवाद में उपयोगकर्ता ID इनपुट बॉक्स साफ़ करें, फिर सहेजें पर क्लिक करें।

#### Q: कॉन्फ़िगरेशन खो गया तो क्या करें

A: कॉन्फ़िगरेशन स्थानीय JSON फ़ाइल में संग्रहीत है, यदि खो जाता है तो फिर से सेट कर सकते हैं, या बैकअप से `data/work_profile_user.json` फ़ाइल पुनर्स्थापित कर सकते हैं।

#### Q: Shelter संबंधित समस्याएं

A: यदि Shelter संबंधित समस्याओं का सामना करना पड़े:

- **क्लोन विफल**: सुनिश्चित करें कि आपके पास व्यवस्थापक अनुमतियां और पर्याप्त स्टोरेज स्थान है
- **क्लोन ऐप दिखाई नहीं दे रहा**: Shelter में Work Profile सही तरीके से सक्रिय है या नहीं जांचें
- **Work Profile में ऐप क्रैश**: ऐप को फिर से क्लोन करने या Shelter अपडेट करने का प्रयास करें
- **क्लोन ऐप नहीं मिल रहा**: ऐप ड्रॉअर में ब्रीफकेस आइकन वाले ऐप खोजें

## सर्वोत्तम प्रथाएं

### एंटरप्राइज़ वातावरण उपयोग

1. **एकीकृत प्रबंधन**: सभी एंटरप्राइज़ डिवाइस के लिए समान उपयोगकर्ता ID सेट करें
2. **अनुमति पृथक्करण**: विभिन्न अनुमति स्तरों को अलग करने के लिए विभिन्न उपयोगकर्ता ID का उपयोग करें
3. **बैकअप कॉन्फ़िगरेशन**: नियमित रूप से `work_profile_user.json` फ़ाइल बैकअप लें

### व्यक्तिगत उपयोग

1. **ऐप अलगाव**: विभिन्न उद्देश्यों के लिए ऐप्स के लिए विभिन्न उपयोगकर्ता वातावरण सेट करें
2. **परीक्षण वातावरण**: ऐप परीक्षण के लिए स्वतंत्र उपयोगकर्ता ID का उपयोग करें
3. **गोपनीयता सुरक्षा**: उपयोगकर्ता पृथक्करण के माध्यम से गोपनीयता सुरक्षा बढ़ाएं

### Shelter टूल प्रबंधन

1. **नियमित अपडेट**: संगतता सुनिश्चित करने के लिए Shelter ऐप को नवीनतम रखें
2. **ऐप सिंक**: सुनिश्चित करें कि Work Profile कॉन्फ़िगर करने से पहले TikTok और TikMatrix क्लोन किए गए हैं
3. **Shelter सेटिंग्स बैकअप**: आसान पुनर्प्राप्ति के लिए Shelter कॉन्फ़िगरेशन निर्यात और बैकअप करें
4. **ऐप अपडेट मॉनिटर**: जब TikTok या TikMatrix अपडेट होते हैं, तो आपको क्लोन संस्करण अपडेट करने की आवश्यकता हो सकती है

## तकनीकी विवरण

Work Profile सुविधा ADB कमांड में `--user` पैरामीटर जोड़कर प्राप्त की जाती है:

```bash
# Work Profile का उपयोग नहीं करना
adb shell input tap 100 200

# Work Profile का उपयोग करना (उपयोगकर्ता ID: 10)
adb shell --user 10 input tap 100 200
```

यह सुनिश्चित करता है कि कमांड सही उपयोगकर्ता वातावरण में निष्पादित हों, अनुमति समस्याओं और वातावरण संघर्ष से बचें।

---

Work Profile को उचित रूप से कॉन्फ़िगर करके, आप विभिन्न जटिल डिवाइस वातावरणों में सुचारू रूप से TikMatrix का उपयोग कर सकते हैं, कार्य दक्षता और प्रबंधन सुविधा में सुधार कर सकते हैं।
