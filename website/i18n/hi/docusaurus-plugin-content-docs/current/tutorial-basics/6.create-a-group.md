# समूह बनाएं

**समूह** उपकरणों का संग्रह है, जो कई उपकरणों को एकीकृत प्रबंधन और संचालन करने के लिए अधिक सुविधाजनक बनाता है।

## चरण

1. बाएं साइडबार में `समूह जोड़ें` बटन पर क्लिक करें।
2. समूह नाम दर्ज करें।
3. समूह सहेजने के लिए `Enter` कुंजी दबाएं।
4. आप इस समूह में डिवाइस को स्थानांतरित करने के लिए चुन सकते हैं।
