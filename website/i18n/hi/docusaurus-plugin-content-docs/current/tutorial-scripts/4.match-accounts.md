# अकाउंट मैच करें

अकाउंट मैच स्क्रिप्ट का उपयोग फोन से अकाउंट पहचानने और उन्हें अकाउंट सूची में जोड़ने के लिए किया जाता है।

## चरण

1. वह डिवाइस चुनें जिसके अकाउंट को मैच करना है।
2. `स्क्रिप्ट` - `अकाउंट मैच करें` बटन पर क्लिक करें।
3. `शुरू करें` बटन पर क्लिक करके अकाउंट मैच टास्क शुरू करें।
4. स्क्रिप्ट स्वचालित रूप से फोन से अकाउंट पहचानेगी और उन्हें अकाउंट सूची में जोड़ देगी।

## नोट

* अकाउंट मैच स्क्रिप्ट 100% सफल नहीं है, यदि कुछ अकाउंट पहचाने नहीं गए हैं, तो आपको स्क्रिप्ट को कई बार चलाना पड़ सकता है।
