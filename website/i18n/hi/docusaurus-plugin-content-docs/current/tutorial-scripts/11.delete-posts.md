# पोस्ट डिलीट करें

पोस्ट डिलीट स्क्रिप्ट का उपयोग व्यू काउंट के आधार पर पोस्ट डिलीट करने के लिए किया जाता है।

## चरण

1. वह डिवाइस चुनें जिस पर आप स्क्रिप्ट चलाना चाहते हैं।
2. `स्क्रिप्ट` > `पोस्ट डिलीट करें` पर क्लिक करें।
3. टास्क सेटिंग्स कॉन्फ़िगर करें:
    - **अधिकतम व्यू काउंट**: उन पोस्ट की अधिकतम व्यू काउंट दर्ज करें जिन्हें आप डिलीट करना चाहते हैं।
4. `स्क्रिप्ट शुरू करें` पर क्लिक करके निष्पादन शुरू करें।

## नोट

- पोस्ट डिलीट स्क्रिप्ट हमेशा सफल नहीं हो सकती; यदि आवश्यक हो तो विफल टास्क को पुनः प्रयास करें।
