# सॉफ़्टवेयर स्टार्टअप त्रुटि

## सामान्य सॉफ़्टवेयर स्टार्टअप त्रुटि समस्या निवारण चरण

### चरण 1: सिस्टम आवश्यकताएं जांचें

1. सुनिश्चित करें कि आपका सिस्टम न्यूनतम आवश्यकताओं को पूरा करता है:
   - Windows 10 या उच्चतर
   - कम से कम 4GB मेमोरी
   - कम से कम 1GB उपलब्ध डिस्क स्थान

### चरण 2: Microsoft Visual C++ Redistributable जांचें

आमतौर पर, `vcruntime140.dll नहीं मिली` या `एप्लिकेशन सामान्य रूप से प्रारंभ नहीं हो सकता (0xc000007b)` त्रुटि Microsoft Visual C++ Redistributable स्थापित नहीं होने के कारण होती है।

1. `Windows + R` कुंजी दबाएं, `cmd` दर्ज करें और Enter दबाएं
2. `dir "C:\Windows\System32\vcruntime*.dll"` दर्ज करें और Enter दबाएं
3. यदि आउटपुट है, तो इसका मतलब है कि Visual C++ Redistributable का संबंधित संस्करण स्थापित है। फ़ाइल नाम में vcruntime प्रत्यय आमतौर पर संस्करण संख्या के साथ आता है, जैसे vcruntime140.dll (2015-2022 के अनुरूप)।
4. यदि कोई आउटपुट नहीं है, तो कृपया Microsoft Visual C++ Redistributable डाउनलोड और इंस्टॉल करें: [नवीनतम Microsoft Visual C++ Redistributable संस्करण](https://learn.microsoft.com/en-us/cpp/windows/latest-supported-vc-redist?view=msvc-170)

### चरण 3: सिस्टम समय जांचें

कंप्यूटर घड़ी गलत होने पर "Agent Start Timeout" त्रुटि हो सकती है:

1. स्क्रीन के निचले दाएं कोने में समय प्रदर्शन पर राइट-क्लिक करें
2. "तारीख/समय समायोजित करें" चुनें
3. यदि अभी तक सक्षम नहीं है, तो "स्वचालित रूप से समय सेट करें" सक्षम करें
4. यदि समय अभी भी गलत है, तो "अभी सिंक करें" पर क्लिक करें या मैन्युअल रूप से सही तारीख और समय सेट करें
5. समय सही करने के बाद TikMatrix को पुनः प्रारंभ करें

### चरण 4: एप्लिकेशन लॉग जांचें

1. `सेटिंग्स` - `ऐप डायरेक्टरी खोलें` बटन पर क्लिक करें
2. `logs` फ़ोल्डर पर नेविगेट करें
3. नवीनतम लॉग फ़ाइल में त्रुटि जानकारी की जांच करें

### यदि समस्या बनी रहती है

### चरण 5: macOS — Agent InvalidStateError

कुछ macOS मशीनों पर, Agent प्रारंभ नहीं हो सकता और निम्नलिखित त्रुटि दिखाई दे सकती है:

```text
Agent start error InvalidStateError: The object is in an invalid state.
```

संभावित कारण:

- आमतौर पर ऐप और वर्तमान macOS सिस्टम संस्करण या पुराने हार्डवेयर/फर्मवेयर के बीच असंगति के कारण होता है।

समाधान / कार्य योजना:

1. "सिस्टम सेटिंग्स" > "सामान्य" > "सॉफ़्टवेयर अपडेट" के माध्यम से macOS को नवीनतम संस्करण में अपडेट करें। अपडेट के बाद Mac को पुनरारंभ करें और TikMatrix को फिर से प्रारंभ करें।
2. यदि अपग्रेड नहीं कर सकते या त्रुटि बनी रहती है, तो नए macOS संस्करण वाले दूसरे Mac पर ऐप चलाने का प्रयास करें।

यदि उपरोक्त विधियों को आजमाने के बाद भी समस्या बनी रहती है, तो कृपया ऐप लॉग एकत्र करें और [Telegram](https://t.me/tikmatrix_agent_bot) के माध्यम से हमसे संपर्क करें, अपना macOS संस्करण और लॉग फ़ाइल संलग्न करें।

कृपया हमारी सहायता टीम [Telegram](https://t.me/tikmatrix_agent_bot) से संपर्क करें, और निम्नलिखित जानकारी प्रदान करें:

- Windows संस्करण
- एप्लिकेशन लॉग फ़ाइल
- आपके द्वारा पहले से प्रयास किए गए चरण
