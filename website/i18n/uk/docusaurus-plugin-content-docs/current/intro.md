# Вступний підручник

Відкрийте для себе **TikMatrix** та **IgMatrix** менш ніж за 5 хвилин та дізнайтеся про основні функції та використання.

## Презентація програмного забезпечення

**TikMatrix** - це настільний додаток для ПК, який автоматизує завдання TikTok на пристроях Android.  
**IgMatrix** - це відповідний інструмент для автоматизації завдань Instagram на пристроях Android.

**Примітка:** **TikMatrix** та **IgMatrix** - це окремі програми, і вони не можуть працювати одночасно на одному комп'ютері.

## Функції

- **Перемикання кількох облікових записів**: Керуйте кількома обліковими записами на пристрій та автоматично перемикайтеся під час виконання завдань.
- **Підтримка клонованих додатків**: Підтримка клонованих додатків TikTok/Instagram з власними назвами пакетів, щоб різні облікові записи могли використовувати різні клони та автоматично відкривати цільовий додаток.
- ~~**Автоматична реєстрація**~~ **[ПРИПИНЕНО]**
- **Автоматичний вхід**: Instagram підтримує автоматичний вхід з 2FA; TikTok може все ще вимагати ручного втручання в деяких потоках перевірки (captcha, перевірка електронної пошти).
- **Автоматичне заповнення профілю**: Автоматично встановлюйте аватар, псевдонім, біографію та ім'я користувача.
- **Автоматичне розпізнавання облікових записів**: Виявлення та імпорт облікових записів, які вже увійшли в телефон, одним кліком.
- **Розігрів / зростання облікового запису**: Шукайте ключові слова, дивіться відео, ставте лайки, підписуйтесь, коментуйте та додавайте в улюблене для розігріву облікових записів.
- **Автоматичне публікування**: Публікуйте відео або зображення (підтримує порядок та пакетні числа) з налаштовуваною музикою, заголовком, описом, хештегами та посиланнями на продукти.
- **Автоматичне видалення**: Видаляйте неефективні пости за правилами кількості переглядів.
- **Супер маркетинг**: Імітуйте людські шляхи перегляду (відвідуйте профілі, переглядайте пости, випадково підписуйтесь/ставте лайки/коментуйте/додавайте в улюблене/ділитесь/надсилайте DM) та додатково використовуйте OpenAI або сумісні API для генерації персоналізованих DM/коментарів.
- **Пакетне підвищення (прямий ефір)**: Масово ставити лайки та коментувати прямі трансляції.
- **Збір користувачів**: Збирайте користувачів за ключовими словами, списками підписників або підписок.
- **Масова взаємна підписка та вітальні DM**: Масово підписуйтесь на підписників та надсилайте вітальні повідомлення.
- **Планування завдань на рівні хвилин**: Підтримка повторних спроб, інтервального виконання та максимальних налаштувань повторних спроб.
- **Багатопристрійний паралелізм**: Один ПК може підключити та керувати багатьма пристроями Android, що виконують завдання паралельно.
- **Дзеркалювання екрана**: Дзеркальте кілька екранів пристроїв на Windows / macOS та керуйте ними за допомогою миші та клавіатури.
- **Імпорт / експорт облікових записів**: Excel-сумісний масовий імпорт/експорт.
- **Багатомовна підтримка**: Програмне забезпечення підтримує англійську, російську та китайську мови. Примітка: Деякі функції вимагають, щоб додаток TikTok/Instagram на вашому телефоні був встановлений на англійську мову для оптимальної сумісності.
- **Підтримка та квитки**: Забезпечує підтримку через людські квитки для проблем.

## Системні вимоги

- **Операційна система**: Windows 10+ (64-bit) або macOS 10.15+
- **Пристрої**: Пристрої Android з Android 7.0+ з увімкненим USB-налагодженням
- **Підключення**: Wi-Fi або дротове підключення для управління пристроями
- **Пам'ять**: Принаймні 8ГБ оперативної пам'яті (рекомендується 16ГБ+ для >20 пристроїв)
