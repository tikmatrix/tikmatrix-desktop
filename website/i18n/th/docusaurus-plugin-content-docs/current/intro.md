# บทนำการใช้งาน

ค้นพบ **TikMatrix** และ **IgMatrix** ในเวลาไม่ถึง 5 นาที และเรียนรู้ฟีเจอร์หลักและวิธีการใช้งาน

## แนะนำซอฟต์แวร์

**TikMatrix** คือแอปพลิเคชันเดสก์ท็อปสำหรับ PC ที่ทำงานอัตโนมัติสำหรับ TikTok บนอุปกรณ์ Android  
**IgMatrix** คือเครื่องมือที่สอดคล้องกันสำหรับทำงานอัตโนมัติบน Instagram บนอุปกรณ์ Android

**หมายเหตุ:** **TikMatrix** และ **IgMatrix** เป็นแอปพลิเคชันที่แยกกันและไม่สามารถทำงานพร้อมกันบนคอมพิวเตอร์เครื่องเดียวกันได้

## ฟีเจอร์

- **การสลับหลายบัญชี**: จัดการหลายบัญชีต่ออุปกรณ์และสลับอัตโนมัติระหว่างการทำงาน
- **รองรับแอปโคลน**: รองรับแอป TikTok/Instagram ที่โคลนมาพร้อมชื่อแพ็คเกจที่กำหนดเอง เพื่อให้บัญชีต่างๆ สามารถใช้แอปโคลนต่างกันและเปิดแอปเป้าหมายโดยอัตโนมัติ
- ~~**การลงทะเบียนอัตโนมัติ**~~ **[ยกเลิกแล้ว]**
- **เข้าสู่ระบบอัตโนมัติ**: Instagram รองรับการเข้าสู่ระบบอัตโนมัติด้วย 2FA; TikTok อาจยังต้องการการแทรกแซงด้วยตนเองในบางขั้นตอนการตรวจสอบ (captcha, การตรวจสอบอีเมล)
- **กรอกโปรไฟล์อัตโนมัติ**: ตั้งค่าอวตาร์, ชื่อเล่น, ประวัติ และชื่อผู้ใช้โดยอัตโนมัติ
- **การจดจำบัญชีอัตโนมัติ**: ตรวจจับและนำเข้าบัญชีที่เข้าสู่ระบบแล้วบนโทรศัพท์ด้วยคลิกเดียว
- **การอุ่นเครื่อง / การเติบโตของบัญชี**: ค้นหาคีย์เวิร์ด, ดูวิดีโอ, กดไลค์, ติดตาม, แสดงความคิดเห็น และบันทึกเพื่ออุ่นเครื่องบัญชี
- **โพสต์อัตโนมัติ**: เผยแพร่วิดีโอหรือรูปภาพ (รองรับการเรียงลำดับและจำนวนแบทช์) พร้อมเพลง, ชื่อเรื่อง, คำอธิบาย, แฮชแท็ก และลิงก์ผลิตภัณฑ์ที่กำหนดเองได้
- **ลบอัตโนมัติ**: ลบโพสต์ที่มีประสิทธิภาพต่ำตามกฎจำนวนการดู
- **Super marketing**: จำลองเส้นทางการเรียกดูแบบมนุษย์ (เยี่ยมชมโปรไฟล์, เรียกดูโพสต์, สุ่มติดตาม/ไลค์/แสดงความคิดเห็น/บันทึก/แชร์/ส่ง DM) และสามารถใช้ OpenAI หรือ API ที่เข้ากันได้เพื่อสร้าง DM/ความคิดเห็นที่เป็นส่วนตัว
- **การบูสต์แบบแบทช์ (Live)**: กดไลค์และแสดงความคิดเห็นในไลฟ์สตรีมแบบกลุ่ม
- **การขูดข้อมูลผู้ใช้**: รวบรวมผู้ใช้ตามคีย์เวิร์ด, รายชื่อผู้ติดตามหรือรายการติดตาม
- **ติดตามกลับจำนวนมาก & DM ต้อนรับ**: ติดตามผู้ติดตามแบบกลุ่มและส่งข้อความต้อนรับ
- **การกำหนดเวลางานระดับนาที**: รองรับการลองใหม่, การทำงานเป็นช่วง และการตั้งค่าการลองใหม่สูงสุด
- **อุปกรณ์หลายเครื่องแบบขนาน**: PC เครื่องเดียวสามารถเชื่อมต่อและจัดการอุปกรณ์ Android หลายเครื่องที่ทำงานแบบขนานได้
- **Screen mirroring**: มิเรอร์หน้าจออุปกรณ์หลายเครื่องไปยัง Windows / macOS และควบคุมด้วยเมาส์และคีย์บอร์ด
- **นำเข้า / ส่งออกบัญชี**: รองรับการนำเข้า/ส่งออกแบบกลุ่มที่เข้ากันได้กับ Excel
- **รองรับหลายภาษา**: ซอฟต์แวร์รองรับภาษาอังกฤษ, รัสเซีย และจีน หมายเหตุ: บางฟีเจอร์ต้องการให้แอป TikTok/Instagram บนโทรศัพท์ของคุณตั้งค่าเป็นภาษาอังกฤษเพื่อความเข้ากันได้ที่ดีที่สุด
- **การสนับสนุนและตั๋ว**: ให้การสนับสนุนตั๋วจากมนุษย์สำหรับปัญหา

**รายละเอียดการชำระเงิน**:

- การเรียกเก็บเงินรายปีได้ส่วนลด 30%
- วิธีการชำระเงิน: บัตรเครดิต, Alipay, cryptocurrency
- แต่ละใบอนุญาตผูกกับคอมพิวเตอร์เครื่องเดียว; รองรับการเปลี่ยนคอมพิวเตอร์ฟรี
- โทรศัพท์ที่เชื่อมต่อสูงสุดอาจถูกจำกัดโดยฮาร์ดแวร์ (เช่น เมนบอร์ดทั่วไปสามารถเชื่อมต่ออุปกรณ์ USB ได้ ~40 เครื่อง)

กฎการอัปเกรดแผน: ระบบจะคำนวณส่วนต่างของราคาโดยอัตโนมัติตามเวลาที่เหลืออยู่ เมื่ออัปเกรด คุณจะจ่ายส่วนต่างระหว่างราคาแผนใหม่และมูลค่าที่เหลืออยู่ของแผนปัจจุบันของคุณ

หากคุณต้องการเปลี่ยนคอมพิวเตอร์ โปรดดูคู่มือการย้ายใบอนุญาตของเรา: [คู่มือการย้ายใบอนุญาต](tutorial-basics/license-migration.md)

## สิ่งที่คุณต้องการ

### โทรศัพท์ Android

คุณสามารถใช้อุปกรณ์ Android ใดก็ได้เพื่อรัน TikMatrix รวมถึง:

- **อุปกรณ์ Android จริง**
- **[อุปกรณ์ Android แบบเมนบอร์ด](http://www.niaozun.shop?cid=934ec2fe)**
- **[Cloud phones](https://www.geelark.cn?invite_code=XHY6a8)**
- **Emulators**
- **Virtual machines**

อุปกรณ์ Android ต้องการอย่างน้อย API 21 (Android 5.0) และไม่ต้องการ root โปรดเปิดใช้งาน USB debugging (Developer Options) ในบางรุ่น (เช่น Xiaomi) คุณอาจต้องเปิดใช้งาน "USB debugging (Security Settings)" เพื่ออนุญาตการควบคุมด้วยคีย์บอร์ด/เมาส์

### คอมพิวเตอร์

#### Windows: Windows 10 ขึ้นไป

#### Mac: macOS 15.0 ขึ้นไป (รองรับทั้ง Apple Silicon & Intel)

## การติดตั้ง

1. ดาวน์โหลดตัวติดตั้งล่าสุดจากหน้า [Download TikMatrix](https://tikmatrix.com/Download) หรือ [Download IgMatrix](https://igmatrix.com/Download-IgMatrix)
2. รันตัวติดตั้ง
3. รอให้การติดตั้งเสร็จสิ้น
4. เปิดแอป **TikMatrix** หรือ **IgMatrix** บนคอมพิวเตอร์ของคุณ
