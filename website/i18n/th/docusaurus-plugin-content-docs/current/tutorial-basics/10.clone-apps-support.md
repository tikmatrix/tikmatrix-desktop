# รองรับแอปโคลน

TikMatrix และ IgMatrix รองรับการรันแอปโคลนหลายตัว (เช่น GB TikTok, GB Instagram, Instagram Plus หรือเวอร์ชันที่ปรับแต่งอื่นๆ) บนอุปกรณ์เดียวกัน โดยแต่ละตัวมีชื่อแพ็คเกจของตัวเอง คุณสมบัตินี้ช่วยให้คุณจัดการหลายบัญชีได้อย่างมีประสิทธิภาพมากขึ้นโดยใช้อินสแตนซ์แอปที่แตกต่างกัน

:::danger คำเตือนความเสี่ยง
การใช้แอปโคลนอาจละเมิดข้อกำหนดการให้บริการของ TikTok/Instagram และอาจส่งผลให้บัญชีถูกระงับหรือแบน เราให้การสนับสนุนทางเทคนิคสำหรับคุณสมบัตินี้เท่านั้น และ**ไม่รับผิดชอบ**ต่อปัญหาที่เกี่ยวข้องกับบัญชี รวมถึงการแบน การระงับ หรือข้อจำกัดที่เกิดจากการใช้แอปโคลน ใช้ความเสี่ยงของคุณเอง
:::

## ชื่อแพ็คเกจคืออะไร?

แอป Android ทุกตัวมีตัวระบุเฉพาะที่เรียกว่า **ชื่อแพ็คเกจ** แอปอย่างเป็นทางการใช้ชื่อแพ็คเกจมาตรฐาน:

- TikTok: `com.zhiliaoapp.musically`
- Instagram: `com.instagram.android`

แอปโคลนใช้ชื่อแพ็คเกจที่แตกต่างกัน เช่น:

- GB TikTok: `com.zhiliaoapp.musically.gb`
- GB Instagram: `com.instagram.gb.android`
- Instagram Plus: `com.instagram.plus.android`

TikMatrix/IgMatrix สามารถตรวจจับและใช้ชื่อแพ็คเกจที่ถูกต้องสำหรับแต่ละบัญชีได้โดยอัตโนมัติ

## การตั้งค่าแอปโคลน

### ขั้นตอนที่ 1: ติดตั้งแอปโคลน

1. ติดตั้งแอปโคลนที่คุณต้องการบนอุปกรณ์หรือ CloudPhone ของคุณ
2. ตรวจสอบให้แน่ใจว่าแอปโคลนแต่ละตัวมีชื่อแพ็คเกจที่ไม่ซ้ำกันเพื่อหลีกเลี่ยงความขัดแย้ง
3. เข้าสู่ระบบบัญชีที่แตกต่างกันในอินสแตนซ์แอปแต่ละตัว

### ขั้นตอนที่ 2: กำหนดค่าชื่อแพ็คเกจ

คุณมีสองตัวเลือกในการกำหนดค่าชื่อแพ็คเกจ:

#### ตัวเลือก A: การกำหนดค่าระดับบัญชี (แนะนำ)

ตั้งค่าชื่อแพ็คเกจเฉพาะสำหรับแต่ละบัญชี:

1. เปิด **Accounts** ในแถบนำทางด้านซ้าย
2. คลิกที่บัญชีเพื่อแก้ไข หรือคลิก **Add** เพื่อสร้างบัญชีใหม่
3. ในฟิลด์ **Package Name** คุณสามารถ:
   - **กรอกด้วยตนเอง** ชื่อแพ็คเกจของแอปโคลน (เช่น `com.instagram.gb.android`)
   - **ตรวจจับอัตโนมัติ** โดยคลิกปุ่ม **Detect Package** (ต้องเลือกอุปกรณ์ก่อน)
4. คลิก **Save**

:::tip ตรวจจับชื่อแพ็คเกจอัตโนมัติ
ปุ่ม **Detect Package** จะตรวจจับแอปที่เปิดอยู่บนอุปกรณ์ที่เลือกโดยอัตโนมัติ นี่คือวิธีที่ง่ายที่สุดในการรับชื่อแพ็คเกจที่ถูกต้อง:

1. เลือกอุปกรณ์สำหรับบัญชี
2. เปิดแอปโคลนบนอุปกรณ์นั้น
3. คลิกปุ่ม **Detect Package**
4. ชื่อแพ็คเกจจะถูกกรอกโดยอัตโนมัติ

คุณสมบัตินี้ใช้งานได้กับแอป Android ใดๆ รวมถึงแอปโคลน TikTok/Instagram, แอป App Cloner และแอป NomixCloner
:::

เมื่อคุณรันสคริปต์สำหรับบัญชีนี้ TikMatrix/IgMatrix จะใช้ชื่อแพ็คเกจที่ระบุโดยอัตโนมัติ

#### ตัวเลือก B: ค่าเริ่มต้นทั่วทั้งระบบ

ตั้งค่าชื่อแพ็คเกจเริ่มต้นสำหรับบัญชีทั้งหมดที่ไม่มีการกำหนดเฉพาะ:

1. เปิด **Settings** ในแถบนำทางด้านซ้าย
2. ค้นหาฟิลด์ **TikTok Package Name** หรือ **Instagram Package Name**
3. ป้อนชื่อแพ็คเกจเริ่มต้น
4. คลิก **Save**

:::tip ลำดับความสำคัญ
การตั้งค่าระดับบัญชีจะแทนที่ค่าเริ่มต้นทั่วทั้งระบบเสมอ หากบัญชีมีการกำหนดค่าชื่อแพ็คเกจของตัวเอง จะใช้ชื่อนั้นแทนค่าเริ่มต้น
:::

## วิธีการทำงาน

เมื่อคุณรันสคริปต์อัตโนมัติใดๆ:

1. TikMatrix/IgMatrix จะตรวจสอบว่าบัญชีเป้าหมายมีการกำหนดค่าชื่อแพ็คเกจที่กำหนดเองหรือไม่
2. ถ้าใช่ ระบบจะใช้ชื่อแพ็คเกจนั้นสำหรับการดำเนินการอัตโนมัติ UI ทั้งหมด
3. ถ้าไม่ใช่ ระบบจะกลับไปใช้ชื่อแพ็คเกจเริ่มต้นทั่วทั้งระบบ
4. resource ID และคิวรี XPath ทั้งหมดจะถูกจัดรูปแบบด้วยชื่อแพ็คเกจที่ถูกต้องโดยอัตโนมัติ

## สคริปต์ที่รองรับ

สคริปต์ต่อไปนี้รองรับชื่อแพ็คเกจแบบไดนามิกอย่างสมบูรณ์:

**สคริปต์ TikMatrix:**

- Login
- Profile Editing
- Post Creation
- Account Warmup
- Mass DM
- Super Marketing
- คุณสมบัติอัตโนมัติอื่นๆ ทั้งหมด

**สคริปต์ IgMatrix:**

- Login
- Profile Editing
- Post Creation
- Account Warmup
- Super Marketing
- Boost Posts
- Mass DM
- Match Accounts
- คุณสมบัติอัตโนมัติอื่นๆ ทั้งหมด

## แนวทางปฏิบัติที่ดีที่สุด

### 1. ใช้ชื่อผู้ใช้ที่สื่อความหมาย

เนื่องจากคุณอาจมีหลายบัญชีในแอปโคลนต่างๆ ให้ใช้ชื่อผู้ใช้ที่สื่อความหมายเพื่อระบุว่าบัญชีแต่ละบัญชีอยู่ในแอปใด:

```text
username_gb      → GB TikTok/Instagram
username_plus    → TikTok/Instagram Plus
username         → Official App
```

### 2. จัดระเบียบบัญชีตามกลุ่ม

สร้างกลุ่มอุปกรณ์ตามประเภทแอปโคลน:

- **กลุ่ม A**: บัญชีทั้งหมดที่ใช้แอป GB
- **กลุ่ม B**: บัญชีทั้งหมดที่ใช้แอป Plus
- **กลุ่ม C**: บัญชีทั้งหมดที่ใช้แอปอย่างเป็นทางการ

### 3. ทดสอบก่อนการดำเนินการแบบกลุ่ม

ก่อนรันสคริปต์กับบัญชีทั้งหมด:

1. ทดสอบกับบัญชีเดียวก่อน
2. ตรวจสอบว่าใช้แอปที่ถูกต้อง
3. ยืนยันว่าการดำเนินการทั้งหมดทำงานตามที่คาดหวัง

### 4. รักษาความสม่ำเสมอของชื่อแพ็คเกจ

เมื่อคุณตั้งค่าชื่อแพ็คเกจสำหรับบัญชี อย่าเปลี่ยนเว้นแต่คุณจะเปลี่ยนไปใช้แอปโคลนอื่น ความสม่ำเสมอช่วยหลีกเลี่ยงความสับสนและข้อผิดพลาด

## การแก้ไขปัญหา

### สคริปต์เปิดแอปผิด

**ปัญหา**: สคริปต์เปิดแอปผิดแทนที่จะเป็นแอปโคลน

**วิธีแก้ไข**:

1. ตรวจสอบว่าฟิลด์ **Package Name** ของบัญชีถูกตั้งค่าอย่างถูกต้อง
2. ตรวจสอบว่าติดตั้งแอปโคลนแล้วและชื่อแพ็คเกจตรงกัน
3. ใช้ปุ่ม **Detect Package** เพื่อรับชื่อแพ็คเกจที่ถูกต้องโดยอัตโนมัติ
4. รีสตาร์ทอุปกรณ์หากเพิ่งติดตั้งแอป

### ไม่พบองค์ประกอบ UI

**ปัญหา**: สคริปต์รายงานข้อผิดพลาด "element not found"

**วิธีแก้ไข**:

1. แอปโคลนต่างๆ อาจมีเลย์เอาต์ UI ที่แตกต่างกันเล็กน้อย
2. ตรวจสอบว่าเวอร์ชันของแอปโคลนเข้ากันได้หรือไม่
3. ลองอัปเดตเป็นเวอร์ชันล่าสุดของ TikMatrix/IgMatrix
4. ติดต่อฝ่ายสนับสนุนหากปัญหายังคงมีอยู่

### หลายบัญชีในแอปเดียวกัน

**ปัญหา**: ต้องการสลับระหว่างหลายบัญชีในอินสแตนซ์แอปเดียวกัน

**วิธีแก้ไข**:

- TikTok/Instagram รองรับหลายบัญชีที่เข้าสู่ระบบแล้วโดยธรรมชาติ
- ใช้คุณสมบัติ **Switch Account** ใน TikMatrix/IgMatrix
- ตั้งค่าชื่อแพ็คเกจเดียวกันสำหรับบัญชีทั้งหมดในอินสแตนซ์แอปนั้น
- TikMatrix/IgMatrix จะจัดการการสลับบัญชีโดยอัตโนมัติ

### ปุ่ม Detect Package ไม่ทำงาน

**ปัญหา**: ปุ่ม **Detect Package** ไม่ทำงานหรือแสดงข้อผิดพลาด

**วิธีแก้ไข**:

1. ตรวจสอบให้แน่ใจว่าคุณเลือกอุปกรณ์ก่อน
2. ตรวจสอบว่าแอปเปิดอยู่บนอุปกรณ์ในขณะนี้
3. ตรวจสอบว่าการเชื่อมต่ออุปกรณ์มีเสถียรภาพ
4. ลองเปิดแอปอีกครั้งแล้วคลิกตรวจจับ

## การค้นหาชื่อแพ็คเกจ

หากคุณต้องการตรวจสอบชื่อแพ็คเกจบนอุปกรณ์ของคุณด้วยตนเอง:

### ใช้คำสั่ง ADB

```bash
# แสดงรายการแพ็คเกจที่เกี่ยวข้องกับ TikTok ทั้งหมด
adb shell pm list packages | grep musically

# แสดงรายการแพ็คเกจที่เกี่ยวข้องกับ Instagram ทั้งหมด
adb shell pm list packages | grep instagram
```

### ใช้คุณสมบัติตรวจจับอัตโนมัติ

วิธีที่ง่ายที่สุดคือใช้ปุ่ม **Detect Package** ที่มีอยู่แล้ว:

1. เปิดหน้าต่างแก้ไขบัญชี
2. เลือกอุปกรณ์
3. เปิดแอปโคลนบนอุปกรณ์
4. คลิก **Detect Package**
5. ชื่อแพ็คเกจจะถูกตรวจจับและกรอกโดยอัตโนมัติ

วิธีนี้ใช้งานได้กับแอป Android ใดๆ และเป็นวิธีที่แนะนำ

## คำถามที่พบบ่อย

**คำถาม: ฉันสามารถใช้แอปโคลนหลายตัวบนอุปกรณ์เดียวกันได้หรือไม่?**

ตอบ: ได้ ตราบใดที่แอปโคลนแต่ละตัวมีชื่อแพ็คเกจที่ไม่ซ้ำกัน คุณสามารถรันหลายอินสแตนซ์แอปบนอุปกรณ์เดียวกันได้

**คำถาม: ฉันต้องติดตั้ง TikMatrix/IgMatrix ใหม่หรือไม่?**

ตอบ: ไม่ คุณสมบัตินี้มีอยู่ในเวอร์ชันล่าสุดแล้ว เพียงอัปเดตเป็นเวอร์ชันล่าสุดและกำหนดค่าชื่อแพ็คเกจ

**คำถาม: สคริปต์อัตโนมัติที่มีอยู่ของฉันจะยังคงทำงานหรือไม่?**

ตอบ: ใช่ สคริปต์ที่มีอยู่ทั้งหมดเข้ากันได้อย่างสมบูรณ์ บัญชีที่ไม่มีชื่อแพ็คเกจที่กำหนดเองจะใช้ชื่อแพ็คเกจเริ่มต้น

**คำถาม: คุณสมบัตินี้ใช้ได้กับทั้ง TikTok และ Instagram หรือไม่?**

ตอบ: ใช่ ทั้ง TikMatrix และ IgMatrix รองรับการตรวจจับและอัตโนมัติของแอปโคลนอย่างสมบูรณ์

**คำถาม: คุณสมบัติตรวจจับอัตโนมัติทำงานอย่างไร?**

ตอบ: เมื่อคุณคลิก **Detect Package**, TikMatrix/IgMatrix จะตรวจสอบว่าแอปใดอยู่ในพื้นหน้าบนอุปกรณ์ที่เลือกและดึงชื่อแพ็คเกจโดยอัตโนมัติ

**คำถาม: ฉันสามารถใช้แอปจาก App Cloner หรือ NomixCloner ได้หรือไม่?**

ตอบ: ได้ คุณสมบัติชื่อแพ็คเกจใช้งานได้กับเครื่องมือโคลนแอป Android ใดๆ รวมถึง App Cloner และ NomixCloner ใช้คุณสมบัติตรวจจับอัตโนมัติเพื่อรับชื่อแพ็คเกจที่ถูกต้องได้ง่ายๆ
