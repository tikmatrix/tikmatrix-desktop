# เปิดใช้งานแป้นพิมพ์ Fast Input

เมื่อโทรศัพท์ของคุณเชื่อมต่อแล้ว TikMatrix จะเปลี่ยนไปใช้แป้นพิมพ์ Fast Input โดยอัตโนมัติ

## ทำไมต้องใช้แป้นพิมพ์ Fast Input?

แป้นพิมพ์ Fast Input เป็นสิ่งจำเป็นสำหรับ TikMatrix โดยให้การรองรับหลายภาษาอัตโนมัติ หากไม่มีแป้นพิมพ์นี้ ฟีเจอร์การป้อนข้อความ เช่น การแสดงความคิดเห็น การส่งข้อความ และการเผยแพร่หัวเรื่องอาจไม่ทำงานอย่างถูกต้อง

## วิธีเปิดใช้งานแป้นพิมพ์ Fast Input ด้วยตนเอง

1. เลือกอุปกรณ์ที่คุณต้องการใช้
2. คลิก `ADB Commands` > `Enable Fast Input`

## วิธีป้อนข้อความ

คุณสามารถป้อนข้อความใน TikMatrix โดยใช้หลายวิธี:

### วิธีที่ 1: การป้อนข้อมูลจากแป้นพิมพ์โดยตรง ⌨️ (ฟีเจอร์ใหม่)

⭐ **การควบคุมการป้อนข้อมูลจากแป้นพิมพ์** - ตอนนี้คุณสามารถพิมพ์โดยตรงบนโทรศัพท์โดยใช้แป้นพิมพ์คอมพิวเตอร์ของคุณ ทำให้การป้อนข้อความเร็วและสะดวกยิ่งขึ้น

1. เลือกอุปกรณ์ที่คุณต้องการใช้
2. คลิกในช่องข้อความใดๆ บนหน้าจอโทรศัพท์
3. เริ่มพิมพ์โดยตรงบนแป้นพิมพ์คอมพิวเตอร์ของคุณ - ข้อความจะปรากฏบนโทรศัพท์ของคุณแบบเรียลไทม์

### วิธีที่ 2: แผงป้อนข้อมูล

1. คลิกที่หน้าจอโทรศัพท์เพื่อขยาย จากนั้นคลิกปุ่ม `Input` ในแถบเครื่องมือทางขวา
2. ป้อนข้อความของคุณในกล่องป้อนข้อมูลและคลิกส่ง

## คัดลอกและวางระหว่างโทรศัพท์และคอมพิวเตอร์

### คัดลอก Clipboard จากโทรศัพท์ไปยังคอมพิวเตอร์

1. เลือกโทรศัพท์ กด `Ctrl+C` เพื่อคัดลอก clipboard จากโทรศัพท์ไปยังคอมพิวเตอร์ของคุณ

### วาง Clipboard จากคอมพิวเตอร์ไปยังโทรศัพท์

1. เลือกอุปกรณ์ กด `Ctrl+V` เพื่อวาง clipboard จากคอมพิวเตอร์ของคุณลงในกล่องป้อนข้อมูลของโทรศัพท์
