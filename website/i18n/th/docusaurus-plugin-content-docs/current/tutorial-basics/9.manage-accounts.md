# การจัดการบัญชี

การจัดการบัญชีช่วยให้ข้อมูลประจำตัวของ TikTok หรือ Instagram แต่ละบัญชีซิงค์กับอุปกรณ์ที่ใช้งาน ข้อมูลการเข้าสู่ระบบที่จำเป็น และข้อมูลที่ระบบอัตโนมัติของคุณต้องการ รายการบัญชีที่เป็นระเบียบจะทำให้ง่ายต่อการมอบหมายอุปกรณ์ ตรวจสอบสถานะของบัญชี และเตรียมแคมเปญโดยไม่ต้องจัดการแต่ละโทรศัพท์ด้วยตนเอง

## สร้างบัญชีด้วยตนเอง

1. เปิด **Accounts** ในแถบนำทางด้านซ้าย
2. คลิก **Add** เพื่อเปิดหน้าต่างแก้ไข
3. กรอกข้อมูลพื้นฐาน:
   - **Email**: ที่อยู่อีเมลที่ใช้โดยแอป TikMatrix/IgMatrix จะพยายามเข้าสู่ระบบด้วย email + password ก่อน หากช่อง email ว่างเปล่า ระบบจะเปลี่ยนไปใช้ username + password โดยอัตโนมัติ
   - **Password**: ข้อมูลการเข้าสู่ระบบที่ใช้กับแอป IgMatrix ยังรับรหัสผ่านที่ขยายด้วย 2FA secret (ดูด้านล่าง)
   - **Username** — ชื่อผู้ใช้ที่แสดงใน TikTok/Instagram นี่คือฟิลด์เดียวที่จำเป็นต้องกรอก
   - สลับ **Login Status** เพื่อทำเครื่องหมายว่าเซสชันของแอปได้รับการยืนยันตัวตนบนอุปกรณ์แล้วหรือไม่ (1 = เข้าสู่ระบบแล้ว, 0 = ยังไม่ได้เข้าสู่ระบบ)
   - สลับ **Status** เพื่อเปิดใช้งานบัญชี (0 = เปิดใช้งาน) หรือปิดใช้งาน (1 = ปิดใช้งาน)
4. เลือกหมายเลขอุปกรณ์เพื่อผูกบัญชีกับโทรศัพท์หรือ CloudPhone หากไม่เลือกอุปกรณ์ บัญชีจะยังไม่ถูกมอบหมาย
5. คลิก **Save** รายการจะรีเฟรชโดยอัตโนมัติ

## การเข้าสู่ระบบ

- เมื่อระบุทั้ง **Email** และ **Password** TikMatrix/IgMatrix จะเข้าสู่ระบบด้วยคู่ข้อมูลนี้บนอุปกรณ์ที่กำหนด
- หากช่อง **Email** ว่างเปล่า ระบบจะลองใช้การผสม **Username** + **Password** แทน
- ตรวจสอบให้แน่ใจว่าได้เลือกดัชนีอุปกรณ์แล้ว เพื่อให้สคริปต์การเข้าสู่ระบบรู้ว่าจะรันที่ไหน
- ระบบอัตโนมัติของการเข้าสู่ระบบจะตรวจจับเซสชันที่ยืนยันตัวตนแล้วและข้ามไปโดยอัตโนมัติ

## รองรับ 2FA (IgMatrix เท่านั้น)

IgMatrix สามารถจัดการการยืนยันตัวตนแบบสองขั้นตอนระหว่างการเข้าสู่ระบบอัตโนมัติ

### รูปแบบรหัสผ่าน 2FA

ในการเปิดใช้งาน 2FA ให้บันทึกรหัสผ่านในรูปแบบต่อไปนี้:

```text
your_password:your_2fa_secret
```

**ตัวอย่าง:**

```text
mypassword123:JBSWY3DPEHPK3PXP
```

## การนำเข้าและส่งออกบัญชี

- **Export** จะเขียนทุกบัญชีลงใน `accounts.xlsx` ในโฟลเดอร์ดาวน์โหลดของ TikMatrix/IgMatrix ใช้เป็นข้อมูลสำรองที่อัปเดตได้ หรือเป็นเทมเพลตสำหรับการแก้ไขแบบกลุ่ม
- **Import** ให้คุณสร้างหรืออัปเดตบัญชีแบบกลุ่มจากไฟล์ Excel นั้น หากคุณไม่แน่ใจเกี่ยวกับรูปแบบหัวตาราง ให้คลิก **Export** ก่อน แก้ไขไฟล์ที่ดาวน์โหลด แล้วนำเข้ากลับมา
- ในระหว่างการนำเข้า แถวใดก็ตามที่มี `id` จะอัปเดตระเบียนที่มีอยู่ แถวที่ไม่มี `id` จะสร้างบัญชีใหม่

> สำหรับการเข้าสู่ระบบ 2FA ของ IgMatrix ให้กรอก `username`, `pwd` (ใช้รูปแบบ `password:2fa_secret`) และคอลัมน์ `device_index`

### ข้อมูลอ้างอิงคอลัมน์ Excel

| คอลัมน์ | จำเป็น | คำอธิบาย |
| --- | --- | --- |
| `id` | ไม่ | ตัวระบุภายใน รวมไว้เฉพาะเมื่อคุณต้องการอัปเดตบัญชีที่มีอยู่ |
| `email` | ไม่ | อีเมลสำหรับเข้าสู่ระบบที่ใช้โดยแอป เป็นประโยชน์สำหรับการติดต่อและการกู้คืน |
| `pwd` | ไม่ | รหัสผ่านที่บันทึกไว้เพื่ออ้างอิงอย่างรวดเร็วหรือการเข้าสู่ระบบอัตโนมัติ |
| `username` | ใช่ | ชื่อผู้ใช้ TikTok/Instagram ต้องไม่ซ้ำกันภายในพื้นที่ทำงาน |
| `device` | ไม่ | หมายเลขซีเรียลอุปกรณ์หรือตัวระบุ CloudPhone ที่ผูกกับบัญชี |
| `device_index` | ไม่ | ป้ายอุปกรณ์ที่เป็นมิตรที่แสดงใน UI ส่วนใหญ่สำหรับการอ้างอิง |
| `logined` | ไม่ | ตัวบ่งชี้เซสชัน (`1` = เข้าสู่ระบบแล้ว, `0` = ยังไม่ได้เข้าสู่ระบบ) |
| `status` | ไม่ | ตัวบ่งชี้ความพร้อมใช้งาน (`0` = เปิดใช้งาน, `1` = ปิดใช้งาน) บัญชีที่ปิดใช้งานจะไม่เข้าร่วมการรันสคริปต์เลย |
| `nickname` | ไม่ | ชื่อเล่นล่าสุดที่ดึงมาจากการซิงค์ข้อมูล TikTok |
| `country` | ไม่ | ประเทศที่อนุมานจากโปรไฟล์ TikTok |
| `followers` | ไม่ | จำนวนผู้ติดตามที่ซิงค์จาก TikTok |
| `following` | ไม่ | จำนวนบัญชีที่โปรไฟล์นี้ติดตาม |
| `hearts` | ไม่ | จำนวนไลค์ทั้งหมด (hearts) บนโปรไฟล์ |
| `videos` | ไม่ | จำนวนวิดีโอที่เผยแพร่ |
| `friends` | ไม่ | จำนวนเพื่อนร่วมกันเมื่อมีข้อมูล |

### ฟิลด์ TikMatrix เทียบกับ IgMatrix

- **TikMatrix** จะกรอกฟิลด์สถิติ (`nickname`, `country`, `followers`, `following`, `hearts`, `videos`, `friends`) หลังจากที่คุณรัน **Sync TikTok Data** หรือใช้ปุ่มคิวรีข้อมูล
- **IgMatrix** ใช้เฉพาะคอลัมน์พื้นฐาน (`email`, `pwd`, `username`, `device`, `logined`, `status`) และละเว้นคอลัมน์วิเคราะห์

## พฤติกรรมเพิ่มเติม

- บัญชีที่ทำเครื่องหมายว่า **Disabled** จะถูกยกเว้นจากสคริปต์อัตโนมัติทั้งหมด งานที่กำหนดเวลา และการดำเนินการแบบกลุ่ม เปิดใช้งานอีกครั้งก่อนเริ่มงานใหม่
- สคริปต์ **Match Accounts** จะสแกนอุปกรณ์เพื่อหาเซสชันแอปที่เข้าสู่ระบบแล้ว สร้างรายการที่เกี่ยวข้องในรายการบัญชีโดยอัตโนมัติ และเชื่อมโยงกับอุปกรณ์ที่ตรวจพบ นี่คือวิธีที่เร็วที่สุดในการเพิ่มโทรศัพท์ที่เตรียมไว้ด้วยตนเอง
