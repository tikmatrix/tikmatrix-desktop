# คำสั่ง ADB ที่กำหนดเอง

TikMatrix ช่วยให้คุณสร้างและจัดการคำสั่ง ADB ที่กำหนดเองเพื่อดำเนินการทั่วไปได้อย่างรวดเร็ว ปรับปรุงขั้นตอนการทำงานและเพิ่มประสิทธิภาพ

## คำสั่ง ADB ที่กำหนดเองคืออะไร?

คำสั่ง ADB ที่กำหนดเองช่วยให้คุณบันทึกคำสั่ง ADB ที่ใช้บ่อยและดำเนินการได้ด้วยการคลิกเพียงครั้งเดียว โดยไม่ต้องป้อนคำสั่งด้วยตนเองทุกครั้ง

## การเข้าถึงอินเทอร์เฟซคำสั่งที่กำหนดเอง

1. ในอินเทอร์เฟซหลักของ TikMatrix คลิก **Custom Commands** ในแถบด้านข้างซ้าย
2. อินเทอร์เฟซจะแสดงคำสั่งที่บันทึกไว้ทั้งหมด พร้อมตัวเลือกในการเพิ่มหรือรีเซ็ตคำสั่ง

## คำสั่งที่ตั้งไว้ล่วงหน้า

TikMatrix ให้คำสั่งที่ตั้งไว้ล่วงหน้าที่มีประโยชน์หลายคำสั่ง:

- **Home**: กลับไปยังหน้าจอหลักของอุปกรณ์
- **Back**: ทำการย้อนกลับ
- **Enable TCP**: เปิดใช้งานโหมดการเชื่อมต่อ TCP/IP
- **Enable Fast Input**: เปิดใช้งานวิธีการป้อนข้อมูลแบบเร็ว
- **Open Gallery**: เปิดแกลเลอรีของอุปกรณ์
- **Open Settings**: เปิดการตั้งค่าอุปกรณ์
- **Open NekoBox**: เปิดแอป NekoBox (หากติดตั้งแล้ว)

## การเพิ่มคำสั่งที่กำหนดเอง

1. คลิก **Add Command** ที่ด้านบนของอินเทอร์เฟซ
2. ในกล่องโต้ตอบ กรอกข้อมูล:
   - **Command Name**: ชื่อที่สามารถจดจำได้สำหรับคำสั่งของคุณ
   - **Command Arguments**: พารามิเตอร์คำสั่ง ADB (ไม่จำเป็นต้องใส่คำนำหน้า "adb")
3. คลิก **Add Command** เพื่อบันทึก

## การรองรับคำสั่งหลายบรรทัด

คำสั่งที่กำหนดเองรองรับการป้อนข้อมูลหลายบรรทัด แต่ละบรรทัดจะถูกจัดการเป็นคำสั่ง ADB แยกต่างหากและดำเนินการตามลำดับ
