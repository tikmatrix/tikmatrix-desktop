# สร้างกลุ่ม

**กลุ่ม** คือชุดของอุปกรณ์ ทำให้ง่ายต่อการจัดการและดำเนินการหลายอุปกรณ์พร้อมกัน

## วิธีสร้างกลุ่ม

1. คลิกปุ่ม `Add Group` ในแถบด้านข้างซ้าย
2. ป้อนชื่อกลุ่ม
3. กด `Enter` เพื่อบันทึกกลุ่ม
4. เลือกอุปกรณ์เพื่อย้ายเข้าสู่กลุ่ม
