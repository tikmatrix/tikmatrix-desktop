# คู่มือการหมุนเวียนพร็อกซีอย่างง่าย

การหมุนเวียนพร็อกซีช่วยให้ TikMatrix เปลี่ยนที่อยู่ IP ที่โทรศัพท์ใช้โดยอัตโนมัติ ใช้เมื่อคุณต้องการเครือข่ายที่สะอาดก่อนเข้าสู่ระบบ สลับบัญชี หรือรันงานที่ละเอียดอ่อน

## สิ่งที่คุณต้องการ

- บริการพร็อกซีที่รองรับ "rotate IP" หรือ "change IP" โดยคลิกลิงก์หรือเรียก URL ง่ายๆ
- ลิงก์หมุนเวียนพร็อกซี (บางครั้งเรียกว่า *rotation URL*, *refresh URL* หรือ *change IP URL*)
- ตัวเลือก: รหัสผ่านหรือโทเค็นที่ผู้ให้บริการพร็อกซีของคุณต้องการ

## บันทึกลิงก์หมุนเวียนสำหรับอุปกรณ์

1. เปิด **Manage Devices** ในแอป TikMatrix desktop และสลับไปที่มุมมองตาราง
2. ค้นหาอุปกรณ์ที่คุณต้องการควบคุมและคลิก **Configure** ในคอลัมน์ *Proxy rotation URL*
3. กรอกแบบฟอร์ม:
   - **Proxy rotation URL**: วางลิงก์จากผู้ให้บริการพร็อกซีของคุณ TikMatrix จะเปิดลิงก์นี้ทุกครั้งที่ต้องการ IP ใหม่
   - **Request method**: เก็บไว้เป็น `GET` เว้นแต่ผู้ให้บริการของคุณจะบอกให้คุณใช้ `POST` อย่างชัดเจน
   - **Headers** และ **Body**: เว้นว่างไว้เว้นแต่ผู้ให้บริการของคุณให้ตัวอย่างสั้นๆ ที่ใช้พวกมัน หากจำเป็น ให้คัดลอกตัวอย่างของพวกเขาลงในกล่องตามที่แสดง
   - **Timeout**: ระยะเวลาที่ TikMatrix รอให้บริการพร็อกซีตอบกลับ ค่าเริ่มต้น 10 วินาทีใช้ได้กับบริการส่วนใหญ่
4. คลิก **Save** TikMatrix จะจัดเก็บทุกอย่างโดยอัตโนมัติ—คุณไม่ต้องแก้ไขไฟล์ใดๆ

หลังจากบันทึก ตารางจะแสดงผลการหมุนเวียนครั้งล่าสุดถัดจากอุปกรณ์ ป้ายสีเขียวหมายความว่าการหมุนเวียนครั้งล่าสุดสำเร็จ ป้ายสีเทาหมายความว่ายังไม่ได้รัน และป้ายสีแดงหมายความว่าบริการพร็อกซีส่งคืนข้อผิดพลาด

## ตรวจสอบว่าลิงก์ทำงาน

1. อยู่ในแถวตารางเดียวกันและคลิก **Test Proxy Rotation**
2. TikMatrix จะเรียกลิงก์ทันที  
   - หากใช้งานได้ คุณจะเห็นข้อความสำเร็จและเวลา "Last rotated" จะอัปเดต  
   - หากล้มเหลว ให้สังเกตข้อความ ตรวจสอบลิงก์และรหัสผ่านอีกครั้ง จากนั้นลองอีกครั้ง

เคล็ดลับ: ผู้ให้บริการพร็อกซีหลายรายจำกัดความถี่ในการหมุนเวียน หากการทดสอบล้มเหลว รอสักครู่แล้วลองอีกครั้ง

## ใช้การหมุนเวียนพร็อกซีระหว่างสคริปต์

1. เมื่อรันสคริปต์ ให้เช็ค **Proxy Rotation** ก่อน TikMatrix จะเปลี่ยน IP สำหรับแต่ละอุปกรณ์ทีละอันก่อนเปิด TikTok  
2. คุณสามารถคลิกปุ่มล็อกทางด้านขวาของแต่ละอุปกรณ์เพื่อดูข้อมูลโดยละเอียดเกี่ยวกับการหมุนเวียนแต่ละครั้ง

## คำถามทั่วไป

- **ฉันต้องหมุนเวียนสำหรับทุกสคริปต์หรือไม่?** ไม่ ใช้เมื่อคุณต้องการให้ TikTok เห็น IP ใหม่—ตัวอย่างเช่น ก่อนสลับระหว่างหลายบัญชี
- **การตั้งค่าจัดเก็บไว้ที่ไหน?** TikMatrix จัดการการจัดเก็บ คุณเพียงแค่กรอกแบบฟอร์มครั้งเดียวต่ออุปกรณ์
- **ลิงก์ผู้ให้บริการดูซับซ้อน จะยังคงใช้งานได้หรือไม่?** ได้ ตราบใดที่ลิงก์มาจากผู้ให้บริการของคุณและใช้งานได้ในเบราว์เซอร์ TikMatrix ก็สามารถเปิดได้เช่นกัน
- **ใครสามารถช่วยได้หากยังคงล้มเหลว?** ส่งคำแนะนำของผู้ให้บริการและข้อความข้อผิดพลาดไปยังฝ่ายสนับสนุน TikMatrix เพื่อให้เราสามารถตรวจสอบร่วมกัน
