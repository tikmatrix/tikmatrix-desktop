# การสลับบัญชีหลายบัญชี

TikTok และ Instagram ทั้งสองรองรับการเข้าสู่ระบบหลายบัญชีบนอุปกรณ์เดียวกัน — นี่เป็นฟีเจอร์พื้นฐานของแอปทางการ TikMatrix และ IgMatrix มีการสลับบัญชีอัตโนมัติเพื่อให้คุณสามารถรันงานข้ามบัญชีที่เข้าสู่ระบบทั้งหมดโดยไม่ต้องดำเนินการด้วยตนเอง

## สคริปต์ที่รองรับ

สคริปต์ต่อไปนี้รองรับการสลับบัญชีอัตโนมัติ:

- **Profile** — ดูข้อมูลโปรไฟล์บัญชี
- **Account Warmup** — อบอุ่นบัญชีด้วยกิจกรรมที่เป็นธรรมชาติ
- **Post** — เผยแพร่โพสต์
- **Mass DM** — ส่งข้อความส่วนตัวเป็นกลุ่ม
- **Follow Back** — ติดตามผู้ใช้ที่ติดตามคุณกลับ
- **Unfollow All** — ยกเลิกการติดตามบัญชีทั้งหมด
- **Delete Post** — ลบโพสต์ที่เผยแพร่
- **Boost Users** — เพิ่มการมีส่วนร่วมกับโปรไฟล์ผู้ใช้
- **Boost Posts** — เพิ่มการมีส่วนร่วมกับโพสต์
- **Boost Lives** — เพิ่มการมีส่วนร่วมกับสตรีมสด
- **Mass Comment** — โพสต์ความคิดเห็นเป็นกลุ่ม
- **Boost Comments** — เพิ่มการมีส่วนร่วมกับความคิดเห็น
- **Super Marketing** — ระบบอัตโนมัติการตลาดขั้นสูง

## การตั้งค่าเริ่มต้น: จับคู่บัญชี

ก่อนใช้ฟีเจอร์การสลับบัญชีหลายบัญชี คุณต้องจับคู่บัญชีจากอุปกรณ์ของคุณก่อน

1. เปิดแท็บ **Scripts** ในเมนูด้านซ้าย
2. ค้นหาและรันสคริปต์ **Match Accounts**
3. สคริปต์นี้จะสแกนอุปกรณ์เพื่อหาบัญชีที่เข้าสู่ระบบและเพิ่มเข้าสู่รายการบัญชีของคุณโดยอัตโนมัติ
4. หลังจากสคริปต์เสร็จสมบูรณ์ คลิกปุ่ม **Accounts** เพื่อยืนยันว่าบัญชีอุปกรณ์ของคุณถูกเพิ่มเรียบร้อยแล้ว

## ทดสอบการสลับบัญชี

หลังจากรัน **Match Accounts** คุณสามารถทดสอบฟังก์ชันการสลับบัญชี:

1. รันสคริปต์ **Switch Account** จากแท็บ Scripts
2. สคริปต์นี้ไม่มีตรรกะอื่น — เพียงตรวจสอบว่าขั้นตอนการสลับบัญชีทำงานอย่างถูกต้อง
3. หากสคริปต์สำเร็จ ฟีเจอร์การสลับบัญชีพร้อมใช้งาน

## ใช้โหมดหลายบัญชี

เมื่อคุณทำการตั้งค่าเริ่มต้นและทดสอบเสร็จแล้ว:

1. เมื่อรันสคริปต์ที่รองรับใดๆ ให้เปิดใช้งาน **Multi-Account Mode** ในการตั้งค่าสคริปต์
2. ระบบจะสร้างงานอิสระสำหรับแต่ละบัญชีในรายการบัญชีของคุณ
3. งานจะถูกจัดคิวและดำเนินการตามลำดับ — ระบบจะสลับไปยังบัญชีเป้าหมายและรันระบบอัตโนมัติโดยอัตโนมัติ
4. ไม่ต้องดำเนินการด้วยตนเอง

## เมื่อใดควรรัน Match Accounts ใหม่

สคริปต์ **Match Accounts** และ **Switch Account** โดยทั่วไปต้องรันเพียงครั้งเดียวในระหว่างการตั้งค่าเริ่มต้น อย่างไรก็ตาม คุณควรรัน **Match Accounts** ใหม่หาก:

- คุณเข้าสู่ระบบบัญชีใหม่บนอุปกรณ์
- คุณออกจากระบบบัญชีที่มีอยู่
- บัญชีบนอุปกรณ์มีการเปลี่ยนแปลง

เพื่อรีเฟรชรายการบัญชี:

1. ลบรายการบัญชีด้วยตนเองในส่วน **Accounts**
2. รันสคริปต์ **Match Accounts** ใหม่เพื่อตรวจจับบัญชีปัจจุบันบนอุปกรณ์

---

การสลับบัญชีหลายบัญชีช่วยลดความจำเป็นในการสลับระหว่างบัญชีด้วยตนเอง ทำให้ง่ายต่อการขยายระบบอัตโนมัติของคุณข้ามบัญชีทั้งหมดบนอุปกรณ์เดียว
