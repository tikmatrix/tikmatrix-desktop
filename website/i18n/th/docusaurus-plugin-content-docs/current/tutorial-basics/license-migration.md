# การย้ายไลเซนส์

โอนไลเซนส์ TikMatrix ของคุณจากคอมพิวเตอร์เครื่องหนึ่งไปยังอีกเครื่องหนึ่ง นี่มีประโยชน์เมื่ออัปเกรดฮาร์ดแวร์หรือเปลี่ยนคอมพิวเตอร์

## ข้อกำหนด

- ไลเซนส์ที่ใช้งานอยู่บนคอมพิวเตอร์ปัจจุบัน (รหัสเปิดใช้งานหรือการสมัครสมาชิก Stripe)
- คอมพิวเตอร์เป้าหมายที่ไม่มีไลเซนส์ TikMatrix อยู่
- อนุญาตให้ย้ายสูงสุด 5 ครั้งต่อเดือน

## ขั้นตอนการย้าย

### ขั้นตอนที่ 1: เปิดไดอะล็อกการย้าย

1. เปิด TikMatrix บนคอมพิวเตอร์ปัจจุบันของคุณ
2. คลิก **ไอคอน License** ในแถบชื่อ
3. คลิกปุ่ม **"Migrate License"**

![License Migration Button](../img/migrate-button.webp)

### ขั้นตอนที่ 2: รับ Machine ID ของเครื่องเป้าหมาย

บนคอมพิวเตอร์เป้าหมายของคุณ:

1. ติดตั้งและเปิด TikMatrix
2. คลิก **ไอคอน License** ในแถบชื่อ
3. คัดลอก **Machine ID**
4. ส่ง ID นี้ไปยังคอมพิวเตอร์ปัจจุบันของคุณ

![Target Machine ID](../img/target-machine-id.webp)

### ขั้นตอนที่ 3: ตรวจสอบและย้าย

กลับมาที่คอมพิวเตอร์ปัจจุบันของคุณ:

1. วาง **Target Machine ID** ในไดอะล็อกการย้าย
2. คลิก **"Validate"** เพื่อตรวจสอบความเข้ากันได้
3. ตรวจสอบรายละเอียดไลเซนส์ที่แสดง

![Validation Success](../img/validation-success.webp)

1. ทำเครื่องหมายในช่องยืนยัน
2. คลิก **"Migrate License"** และยืนยัน

![Migration Confirmation](../img/migration-confirm.webp)

### ขั้นตอนที่ 4: ทำการตั้งค่าให้เสร็จสมบูรณ์

1. รอให้การย้ายเสร็จสมบูรณ์
2. บนคอมพิวเตอร์เป้าหมายของคุณ รีสตาร์ท TikMatrix
3. ไลเซนส์ของคุณตอนนี้ใช้งานบนคอมพิวเตอร์เครื่องใหม่

![Migration Success](../img/migration-success.webp)

## คำเตือนสำคัญ

⚠️ **การย้ายไลเซนส์ไม่สามารถยกเลิกได้**

- ไลเซนส์จะย้ายจากต้นทางไปยังเป้าหมายอย่างสมบูรณ์
- คอมพิวเตอร์เครื่องเก่าของคุณจะสูญเสียการเข้าถึงทันที
- สูงสุด 5 ครั้งการย้ายต่อเดือน
- คอมพิวเตอร์ทั้งสองต้องมีอินเทอร์เน็ตที่เสถียร

## สิ่งที่ถูกย้าย

### สำหรับรหัสเปิดใช้งาน

- สถานะไลเซนส์และวันที่เหลืออยู่
- ข้อมูลรหัสไลเซนส์

### สำหรับการสมัครสมาชิก Stripe

- สถานะการสมัครสมาชิกและข้อมูลการเรียกเก็บเงิน
- วันที่ต่ออายุและรายละเอียดแผน

## การแก้ไขปัญหา

### ข้อความแสดงข้อผิดพลาดทั่วไป

#### "Target machine already has a license"

คอมพิวเตอร์เป้าหมายมีไลเซนส์ที่ใช้งานอยู่แล้ว การย้ายใช้งานได้เฉพาะกับคอมพิวเตอร์ที่ไม่มีไลเซนส์อยู่เท่านั้น

#### "Monthly migration limit exceeded"

คุณสามารถย้ายได้เพียง 5 ครั้งต่อเดือน รอจนถึงเดือนถัดไปหรือติดต่อฝ่ายสนับสนุน

#### "Invalid machine ID format"

ตรวจสอบให้แน่ใจว่าคุณคัดลอก Machine ID ที่สมบูรณ์อย่างถูกต้อง ควรมีความยาวอย่างน้อย 10 ตัวอักษร

#### "Migration validation failed"

ตรวจสอบว่า:

- ไลเซนส์ปัจจุบันของคุณใช้งานอยู่และไม่หมดอายุ
- Target Machine ID ถูกต้อง
- คอมพิวเตอร์ทั้งสองมีการเข้าถึงอินเทอร์เน็ต

### การรับการสนับสนุน

ติดต่อ [Telegram Support](https://t.me/tikmatrix_agent_bot) พร้อม:

- ภาพหน้าจอของข้อความแสดงข้อผิดพลาด
- Machine ID ปัจจุบันและเป้าหมายของคุณ
- คำอธิบายของปัญหา

## คำถามที่พบบ่อย

**ฉันสามารถย้ายกลับไปยังคอมพิวเตอร์เดิมได้หรือไม่?**

ได้ แต่จะนับเป็นการย้ายอีกครั้งเข้าสู่ขีดจำกัดรายเดือนของคุณ

**การเชื่อมต่ออุปกรณ์ของฉันเป็นอย่างไร?**

การเชื่อมต่ออุปกรณ์ผูกกับคอมพิวเตอร์ คุณจะต้องเชื่อมต่ออุปกรณ์ใหม่บนคอมพิวเตอร์เครื่องใหม่

**ฉันสามารถย้ายไลเซนส์ทดลองได้หรือไม่?**

ไม่ได้ สามารถย้ายได้เฉพาะไลเซนส์ที่ชำระเงินเท่านั้น

**การย้ายส่งผลต่อวันที่เหลืออยู่ของไลเซนส์หรือไม่?**

ไม่ วันที่เหลืออยู่ของคุณจะเหมือนเดิมหลังการย้าย
