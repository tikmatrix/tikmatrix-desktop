# ตั้งค่า Proxy

คุณสามารถตั้งค่าที่อยู่ IP proxy ที่ไม่ซ้ำกันสำหรับแต่ละโทรศัพท์ได้

> **คำปฏิเสธความรับผิดชอบ:** เว็บไซต์ทั้งหมดที่แนะนำที่นี่มาจากอินเทอร์เน็ตและไม่มีความเกี่ยวข้องกับเว็บไซต์นี้ กรุณาตรวจสอบความถูกต้องก่อนใช้งาน เราไม่ให้การรับประกันใดๆ

## 1. ตั้งค่า Residential VPS Proxy (หากคุณซื้อ Proxy IP แล้ว สามารถข้ามขั้นตอนนี้ได้)

1. ซื้อ VPS proxy IP แบบ residential จากประเทศเป้าหมายของคุณ (ควรเป็น native IP และ dual ISP VPS)
2. ใช้ [https://github.com/MHSanaei/3x-ui](https://github.com/MHSanaei/3x-ui) เพื่อตั้งค่า VPN บน VPS อย่างรวดเร็ว:

    ```shell
    bash <(curl -Ls https://raw.githubusercontent.com/MHSanaei/3x-ui/refs/tags/v2.6.0/install.sh)
    # ทำตามคำแนะนำเพื่อทำการตั้งค่า VPN ให้เสร็จสมบูรณ์
    ```

3. หลังจากติดตั้งแล้ว เข้าถึงเว็บอินเทอร์เฟซ x-ui คุณสามารถหา URL และข้อมูลการเข้าสู่ระบบได้จากผลลัพธ์ในเทอร์มินัลหลังการติดตั้ง
4. สร้างผู้ใช้ใหม่ใน x-ui ตั้งค่าโปรโตคอลเป็น `SOCKS5` และกำหนดค่าพอร์ตและการตั้งค่าอื่นๆ ตามต้องการ
5. หลังจากสร้างผู้ใช้แล้ว คุณจะเห็น proxy IP และพอร์ตในอินเทอร์เฟซ x-ui
6. ใช้ proxy IP และพอร์ตในแอป NekoBox บนโทรศัพท์ของคุณ

> ผู้ให้บริการ VPS ที่แนะนำ

- [คลิกที่นี่](/advertising)

## 2. ซื้อ Residential Proxy IP (หากคุณตั้งค่า VPS แล้ว สามารถข้ามขั้นตอนนี้ได้)

คุณยังสามารถซื้อ residential proxy IP โดยตรงโดยไม่ต้องตั้งค่า VPS

> ผู้ให้บริการ Proxy ที่แนะนำ

- [คลิกที่นี่](/advertising)

> เครื่องมือตรวจสอบ IP

- [ipinfo](https://ipinfo.io) — ตรวจสอบว่า IP เป็นแบบ residential หรือไม่
- [scamalytics](https://scamalytics.com/ip) — ตรวจสอบระดับความเสี่ยงของ IP

## 3. เปิดใช้งาน Proxy บนโทรศัพท์

1. ดาวน์โหลด APK และติดตั้งไคลเอนต์ [NekoBoxForAndroid](https://github.com/MatsuriDayo/NekoBoxForAndroid) บนโทรศัพท์ของคุณ
2. เปิดแอป NekoBox และคลิกปุ่ม `+` เพื่อเพิ่ม proxy ใหม่
3. เลือก `SOCKS5` หรือประเภท proxy อื่นๆ
4. ใส่ proxy IP และพอร์ต จากนั้นคลิก `Save`
5. คลิกปุ่ม `Start` เพื่อเปิดใช้งาน proxy
6. หลังจากเปิดใช้งาน proxy แล้ว คุณสามารถตรวจสอบสถานะการเชื่อมต่อในแอป NekoBox ได้

### สำหรับผู้ใช้ในจีน

หากคุณอยู่ในประเทศจีน โปรโตคอล `SOCKS5` อาจไม่ทำงานได้ดี คุณจำเป็นต้องเพิ่ม encryption proxy (`VLESS`) เป็น frontend proxy คุณสามารถใช้เครื่องมือต่อไปนี้:

1. เพิ่ม `VLESS` proxy ในแอป NekoBox
2. เพิ่ม `SOCKS5` proxy ในแอป NekoBox
3. เพิ่ม `Chain` proxy ในแอป NekoBox และเลือก `VLESS` proxy เป็น frontend proxy และ `SOCKS5` proxy เป็น backend proxy
