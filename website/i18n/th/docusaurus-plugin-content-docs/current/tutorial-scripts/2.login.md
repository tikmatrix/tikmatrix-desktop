# เข้าสู่ระบบ

สคริปต์เข้าสู่ระบบใช้สำหรับเข้าสู่ระบบบัญชีโซเชียลมีเดีย

## ขั้นตอน

1. ตรวจสอบให้แน่ใจว่าบัญชีถูกเพิ่มในรายการบัญชีแล้ว (คลิก `Accounts` เพื่อเพิ่มบัญชี) และเลือกดัชนีอุปกรณ์ที่คุณต้องการใช้สำหรับบัญชีนี้
2. เลือกอุปกรณ์เดี่ยวหรือหลายอุปกรณ์เพื่อเรียกใช้สคริปต์เข้าสู่ระบบ
3. คลิก `Script` > `Login`
4. คลิก `Start Script` เพื่อเริ่มต้น

## ลำดับความสำคัญการเข้าสู่ระบบ

ระบบใช้ลำดับความสำคัญการเข้าสู่ระบบดังนี้:

1. **อีเมล + รหัสผ่าน**: หากมีการกำหนดค่าอีเมลสำหรับบัญชี จะใช้อีเมลในการเข้าสู่ระบบก่อน
2. **ชื่อผู้ใช้ + รหัสผ่าน**: หากไม่มีการกำหนดค่าอีเมล จะใช้ชื่อผู้ใช้ในการเข้าสู่ระบบ

## การรองรับ 2FA (IgMatrix เท่านั้น)

IgMatrix รองรับการเข้าสู่ระบบด้วยการยืนยันตัวตนสองขั้นตอน (2FA) อัตโนมัติ:

### รูปแบบรหัสผ่านสำหรับ 2FA

ในการใช้ 2FA ให้จัดรูปแบบรหัสผ่านของคุณดังนี้:

```plaintext
your_password:your_2fa_secret_key
```

**ตัวอย่าง:**

```plaintext
mypassword123:JBSWY3DPEHPK3PXP
```

### วิธีการทำงานของ 2FA

1. **การแยกวิเคราะห์รหัสผ่าน**: ระบบจะตรวจจับอัตโนมัติว่ารหัสผ่านของคุณมีเครื่องหมายจุดคู่ (`:`) คั่นหรือไม่
2. **การสร้างโทเค็น**: เมื่อต้องการ 2FA ระบบจะใช้คีย์ลับของคุณเพื่อขอรหัส TOTP จากบริการ 2FA ของเรา
3. **การป้อนอัตโนมัติ**: รหัส 6 หลักที่สร้างขึ้นจะถูกป้อนลงในช่อง 2FA ของ Instagram โดยอัตโนมัติ
4. **การเข้าสู่ระบบที่ราบรื่น**: กระบวนการ 2FA ทั้งหมดเกิดขึ้นโดยอัตโนมัติโดยไม่ต้องมีการแทรกแซงด้วยตนเอง

### คีย์ลับ 2FA

คีย์ลับ 2FA โดยทั่วไปเป็นสตริงที่เข้ารหัส Base32 แบบ 32 ตัวอักษรที่คุณได้รับเมื่อตั้งค่า 2FA ใน Instagram เป็นคีย์เดียวกับที่ใช้โดยแอพยืนยันตัวตนเช่น Google Authenticator หรือ Authy

### จะหาบัญชี Instagram ที่เปิดใช้งาน 2FA ได้ที่ไหน

คุณสามารถซื้อบัญชี Instagram ที่รองรับ 2FA ได้จาก: [AccsMarket](https://accsmarket.com/en/?ref=802647)

## หมายเหตุ

- การเข้าสู่ระบบ TikTok ไม่สามารถทำอัตโนมัติ 100% คุณอาจต้องป้อนรหัสยืนยันอีเมลด้วยตนเองหรือทำขั้นตอนการยืนยันอื่นๆ เราไม่สามารถข้ามความท้าทายการเข้าสู่ระบบเฉพาะของ TikTok (เช่น CAPTCHA รูปภาพหรือการยืนยันตัวบุคคล) หรือแก้ไขข้อความแสดงข้อผิดพลาดบางอย่างเช่น "ความพยายามเข้าสู่ระบบมากเกินไป"
- Instagram สามารถบรรลุการเข้าสู่ระบบอัตโนมัติ 100% ผ่าน 2FA สำหรับบัญชี Instagram ที่เปิดใช้งาน 2FA ตรวจสอบให้แน่ใจว่าใช้รูปแบบคีย์ลับที่ถูกต้องในรหัสผ่านของคุณ
