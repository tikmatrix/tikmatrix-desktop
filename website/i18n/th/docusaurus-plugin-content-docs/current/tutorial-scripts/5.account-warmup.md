# เพิ่มความอบอุ่นให้บัญชี

สคริปต์เพิ่มความอบอุ่นให้บัญชีใช้สำหรับเพิ่มความอบอุ่นให้บัญชีโดยการค้นหาคีย์เวิร์ด ดูวิดีโอ ติดตาม กดถูกใจ แสดงความคิดเห็น และบันทึก

## ขั้นตอน

1. เลือกอุปกรณ์ที่จะเรียกใช้สคริปต์
2. คลิก `Script` > `Account Warmup`
3. กำหนดค่าการตั้งค่างาน:
   - **ระยะเวลาการดู**: ระยะเวลาในการดูแต่ละวิดีโอ
   - **คีย์เวิร์ดการค้นหา**: หัวข้อในการค้นหาวิดีโอ
   - **ระยะเวลางาน**: ระยะเวลารวมของงานเพิ่มความอบอุ่น
   - **เนื้อหาความคิดเห็น**: ใช้ความคิดเห็นที่เตรียมไว้ล่วงหน้าหรือความคิดเห็นที่สร้างโดย ChatGPT จากคำบรรยายโพสต์
   - **ความน่าจะเป็นการโต้ตอบ**: ความน่าจะเป็นของการติดตาม กดถูกใจ แสดงความคิดเห็น และบันทึก
4. คลิก `Start Script` เพื่อเริ่มต้น

## หมายเหตุ

- สคริปต์เพิ่มความอบอุ่นอาจไม่สำเร็จเสมอไป ลองงานที่ล้มเหลวอีกครั้งหากจำเป็น
- รองรับการกำหนดเวลา ดังนั้นคุณสามารถตั้งให้งานเพิ่มความอบอุ่นทำงานในเวลาที่กำหนด
- รองรับการข้ามโฆษณา การถ่ายทอดสด และวิดีโอที่ถูกใจไปแล้ว
