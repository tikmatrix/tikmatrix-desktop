# จับคู่บัญชี

สคริปต์จับคู่บัญชีใช้สำหรับจดจำบัญชี TikTok จากโทรศัพท์และเพิ่มลงในรายการบัญชี

## ขั้นตอน

1. เลือกอุปกรณ์ที่จะเรียกใช้สคริปต์
2. คลิก `Script` > `Match Accounts`
3. คลิก `Start Script` เพื่อเริ่มต้น
4. สคริปต์จะจดจำบัญชีบนโทรศัพท์โดยอัตโนมัติและเพิ่มลงในรายการบัญชี

## หมายเหตุ

- สคริปต์จับคู่บัญชีอาจไม่สามารถจดจำบัญชีทั้งหมดในการรันครั้งเดียว หากมีบัญชีที่ไม่ได้ถูกจดจำ ให้ลองเรียกใช้สคริปต์อีกครั้ง
