# ลบโพสต์

สคริปต์ลบโพสต์ใช้สำหรับลบโพสต์ตามจำนวนการดู

## ขั้นตอน

1. เลือกอุปกรณ์ที่จะเรียกใช้สคริปต์
2. คลิก `Script` > `Delete Posts`
3. กำหนดค่าการตั้งค่างาน:
    - **จำนวนการดูสูงสุด**: ป้อนจำนวนการดูสูงสุดสำหรับโพสต์ที่คุณต้องการลบ
4. คลิก `Start Script` เพื่อเริ่มต้น

## หมายเหตุ

- สคริปต์ลบโพสต์อาจไม่สำเร็จเสมอไป ลองงานที่ล้มเหลวอีกครั้งหากจำเป็น
