# การตั้งค่าความเป็นส่วนตัว

สคริปต์**การตั้งค่าความเป็นส่วนตัว**ใช้สำหรับอัปเดตการตั้งค่าความเป็นส่วนตัวสำหรับโพสต์ TikTok เป็นจำนวนมากตามตัวกรองจำนวนการดู

:::info
สคริปต์นี้ใช้ได้เฉพาะกับบัญชี TikTok เท่านั้น
:::

## ขั้นตอน

1. เลือกอุปกรณ์ที่จะเรียกใช้สคริปต์
2. คลิก `Script` > `Privacy Settings`
3. กำหนดค่าการตั้งค่างาน:
    - **พารามิเตอร์ตัวกรองโพสต์**:
      - **จำนวนการดูสูงสุด**: ป้อนจำนวนการดูสูงสุดเพื่อกรองโพสต์ เฉพาะโพสต์ที่มีการดู ≤ จำนวนนี้จะถูกอัปเดต
      - **โพสต์สูงสุดที่จะประมวลผล**: ป้อนจำนวนโพสต์สูงสุดที่จะประมวลผล ตั้งค่าเป็น 0 สำหรับการประมวลผลไม่จำกัด
    - **พารามิเตอร์การตั้งค่าความเป็นส่วนตัว**:
      - **ใครสามารถดูวิดีโอนี้**: เลือกหนึ่งตัวเลือก:
        - `ทุกคน`: ทุกคนสามารถดูวิดีโอได้
        - `เพื่อน`: เฉพาะเพื่อนเท่านั้นที่สามารถดูวิดีโอได้
        - `เฉพาะคุณ`: เฉพาะคุณเท่านั้นที่สามารถดูวิดีโอได้
    - **พารามิเตอร์สวิตช์**:
      - **อนุญาตความคิดเห็น**: เปิดหรือปิดความคิดเห็นในโพสต์
      - **อนุญาตการนำเนื้อหากลับมาใช้**: เปิดหรือปิดการนำเนื้อหากลับมาใช้ รวมถึง Duet, Stitch, Sticker และ Add to Story
4. คลิก `Start Script` เพื่อเริ่มต้น

## วิธีการทำงาน

สคริปต์ปฏิบัติตามขั้นตอนนี้:

1. ไปที่แท็บวิดีโอ
2. สแกนโพสต์และแยกวิเคราะห์จำนวนการดู
3. ค้นหาโพสต์ที่ตรงกับเกณฑ์ตัวกรอง (การดู ≤ max_views)
4. เข้าสู่หน้ารายละเอียดโพสต์
5. คลิกปุ่ม "การตั้งค่าความเป็นส่วนตัว"
6. รอให้กล่องโต้ตอบการตั้งค่าปรากฏ
7. ตั้งค่าสิทธิ์การดูตามการเลือกของคุณ
8. สลับสวิตช์ "อนุญาตความคิดเห็น" อย่างชาญฉลาด
9. สลับสวิตช์ "อนุญาตการนำกลับมาใช้" อย่างชาญฉลาด
10. กลับไปที่รายการและดำเนินการประมวลผลโพสต์ถัดไปต่อ
11. ทำซ้ำจนกว่าโพสต์ที่ตรงกันทั้งหมดจะถูกประมวลผล

## หมายเหตุ

- สคริปต์การตั้งค่าความเป็นส่วนตัวอาจไม่สำเร็จเสมอไป ลองงานที่ล้มเหลวอีกครั้งหากจำเป็น
- สคริปต์ประมวลผลเฉพาะโพสต์ที่ตรงกับตัวกรองจำนวนการดูเท่านั้น
- การเปลี่ยนแปลงจะถูกนำไปใช้ทันทีและไม่สามารถยกเลิกผ่านสคริปต์ได้
- ตรวจสอบให้แน่ใจว่าได้ตรวจสอบการตั้งค่าความเป็นส่วนตัวของคุณก่อนเรียกใช้สคริปต์บนหลายโพสต์
