# เพิ่มไลฟ์

สคริปต์เพิ่มไลฟ์ช่วยให้คุณควบคุมบัญชีฟาร์มเพื่อดูและอาจจะกดถูกใจหรือแสดงความคิดเห็นในไลฟ์สตรีมเป้าหมาย คุณสามารถตั้งค่าระยะเวลาการดูและช่วงเวลาการกดถูกใจ/แสดงความคิดเห็นเพื่อปรับแต่งพฤติกรรมการเพิ่ม

## ขั้นตอน

1. เลือกอุปกรณ์ที่จะเรียกใช้สคริปต์
2. คลิก `Scripts` > `Boost Lives`
3. กำหนดค่าการตั้งค่างาน:
    - **ชื่อผู้ใช้**: ป้อนชื่อผู้ใช้เป้าหมาย หนึ่งรายการต่อบรรทัด
    - **วิธีการเข้า**: เลือกระหว่างวิธี **Search** และ **Direct**
    - **ตัวเลือกการเพิ่ม**: เปิดใช้งานการกดถูกใจและ/หรือการแสดงความคิดเห็น
    - **เข้าร่วมแฟนคลับ**: เปิดใช้งานการเข้าร่วมแฟนคลับของสตรีมเมอร์ถ้ามี
    - **เช็คอินรายวัน**: เปิดใช้งานกิจกรรมเช็คอินรายวันระหว่างการถ่ายทอดสด
    - **ช่วงเวลาการกดถูกใจ**: ตั้งค่าช่วงเวลาระหว่างแต่ละการกดถูกใจ
    - **จำนวนการแตะกดถูกใจ**: จำนวนครั้งที่จะแตะปุ่มกดถูกใจ
    - **ช่วงเวลาความคิดเห็น**: ตั้งค่าช่วงเวลาระหว่างแต่ละการแสดงความคิดเห็น
    - **เนื้อหาความคิดเห็น**: ป้อนหนึ่งความคิดเห็นต่อบรรทัด
    - **แทรกอีโมจิ**: เลือกว่าจะแทรกอีโมจิลงในความคิดเห็นหรือไม่
    - **ลำดับความคิดเห็น**: เลือกลำดับสุ่มหรือตามลำดับสำหรับความคิดเห็น
    - **ความคิดเห็นต่อบัญชี**: ตั้งค่าจำนวนความคิดเห็นที่จะโพสต์ต่อบัญชี
    - **ระยะเวลาการดู**: ตั้งค่าระยะเวลาที่จะอยู่ในไลฟ์สตรีม
    - **ช่วงเวลางาน**: ตั้งค่าช่วงเวลาระหว่างแต่ละงาน
4. คลิก `Start Script` เพื่อเริ่มต้น

## หมายเหตุ

- สคริปต์เพิ่มไลฟ์อาจไม่สำเร็จเสมอไป ลองงานที่ล้มเหลวอีกครั้งหากจำเป็น
