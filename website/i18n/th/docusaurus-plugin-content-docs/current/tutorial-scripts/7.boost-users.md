# เพิ่มผู้ใช้

:::warning ประกาศยุติการใช้งาน
ฟังก์ชันนี้ได้รับการรวมเข้ากับ [Super Marketing](./14.super-marketing.md) โปรดใช้สคริปต์ Super Marketing สำหรับคุณสมบัติการเพิ่มผู้ใช้ที่ทรงพลังและยืดหยุ่นมากขึ้น
:::

ฟีเจอร์เพิ่มผู้ใช้เพิ่มการเปิดเผยและการมีส่วนร่วมโดยการติดตามหรือเลิกติดตามบัญชีเป้าหมายเป็นจำนวนมาก ระบบพยายามจำลองพฤติกรรมผู้ใช้จริงเพื่อลดความเสี่ยงในการตรวจจับ แต่คุณควรปฏิบัติตามกฎของแพลตฟอร์มและใช้งานอย่างพอประมาณ

## การใช้ฟีเจอร์เพิ่มผู้ใช้ใน Super Marketing

1. เปิดสคริปต์: เลือก "Super Marketing"
2. เลือกประเภทแหล่งข้อมูล "User account list"
3. วางชื่อผู้ใช้เป็นจำนวนมากหรือนำเข้าจากไฟล์ข้อความ (หนึ่งชื่อผู้ใช้ต่อบรรทัด) หลังจากนำเข้าเลือก "Append" หรือ "Overwrite"
4. เลือกกลยุทธ์การใช้ข้อมูล:

    - Shared pool: บัญชีบอททั้งหมดโต้ตอบกับเป้าหมายเดียวกัน

    - Single-consumption: แต่ละบัญชีบอทจัดการเป้าหมายที่แตกต่างกัน ดังนั้นแต่ละเป้าหมายจะถูกประมวลผลโดยบัญชีเดียวเท่านั้น

5. ตั้งค่าจำนวนการทำซ้ำต่อเป้าหมายใน "User Actions — Repeats" (โดยทั่วไปตั้งค่าเป็น 1)
6. เลือกประเภทการกระทำ: Follow หรือ Unfollow คุณสามารถกำหนดค่าตัวเลือกเพิ่มเติมใน **User Actions**:

        - วิธีการนำทางโปรไฟล์: เลือก **Search User** หรือ **Direct Open Profile**

        - ส่งข้อความโดยตรง (ทางเลือก): หากเปิดใช้งาน วางเทมเพลต DM แบบหลายบรรทัดที่สร้างไว้ล่วงหน้า (หนึ่งเทมเพลตต่อบรรทัดหรือย่อหน้า) กำหนดค่าลำดับการใช้เทมเพลต:

                - **Random**: เลือกเทมเพลตแบบสุ่มแต่ละครั้ง
                - **Sequential**: ใช้เทมเพลตตามลำดับที่วาง/นำเข้า (สามารถดำเนินการต่อข้ามการรันงาน)

            คุณยังสามารถเลือกที่จะแทรกอีโมจิลงในเทมเพลตโดยอัตโนมัติเพื่อเพิ่มความเป็นธรรมชาติ

7. กำหนดค่าพารามิเตอร์สคริปต์อื่นๆ (การทำงานพร้อมกัน ความล่าช้า กลุ่มบัญชี) ตรวจสอบการตั้งค่า จากนั้นเริ่มสคริปต์

## คำแนะนำและหมายเหตุ

- ทดสอบกับเป้าหมายจำนวนน้อยก่อนเพื่อให้แน่ใจว่าการตั้งค่าไม่กระตุ้นการป้องกันแพลตฟอร์ม
- ตั้งค่าความล่าช้าและการทำงานพร้อมกันที่เหมาะสมเพื่อเลียนแบบพฤติกรรมมนุษย์และลดความเสี่ยง
- สำหรับการรันที่ยาวนาน ติดตามสุขภาพบัญชีและบันทึกงาน และปรับกลยุทธ์ตามความจำเป็น
- การดำเนินการเพิ่มผู้ใช้ไม่รับประกันความสำเร็จ 100% ลองงานที่ล้มเหลวอีกครั้งหรือตรวจสอบสถานะบัญชีเป้าหมายหากจำเป็น
