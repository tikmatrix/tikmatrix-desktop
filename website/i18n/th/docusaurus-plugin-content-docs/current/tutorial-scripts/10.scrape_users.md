# ดึงข้อมูลผู้ใช้

สคริปต์ดึงข้อมูลผู้ใช้ใช้สำหรับดึงชื่อผู้ใช้ TikTok ฟีเจอร์ดึงข้อมูลผู้ใช้รองรับสามโหมด: รวบรวมรายการผู้ติดตามของผู้ใช้ที่ระบุ รวบรวมรายการผู้ที่กำลังติดตามของผู้ใช้ที่ระบุ หรือรวบรวมผู้ใช้โดยการค้นหาด้วยคีย์เวิร์ด

## ขั้นตอน

1. เลือกอุปกรณ์ที่จะเรียกใช้สคริปต์ หากคุณเลือกหลายอุปกรณ์ งานดึงข้อมูลจะถูกกระจายอย่างเท่าเทียมกันระหว่างอุปกรณ์ที่เลือก
2. คลิก `Script` > `Scrape Users`
3. กำหนดค่าการตั้งค่างาน:
    - **โหมดดึงข้อมูล**: เลือกโหมดสำหรับดึงข้อมูลผู้ใช้
      - **Scrape by Followers**: รวบรวมผู้ใช้จากรายการผู้ติดตามของบัญชีที่ระบุ
      - **Scrape by Following**: รวบรวมผู้ใช้จากรายการผู้ที่กำลังติดตามของบัญชีที่ระบุ
      - **Scrape by Keyword**: ค้นหาและรวบรวมผู้ใช้โดยใช้คีย์เวิร์ด
    - **ชื่อผู้ใช้บัญชีเป้าหมาย**: ป้อนชื่อผู้ใช้ของบัญชีที่จะดึงข้อมูลผู้ใช้ หนึ่งรายการต่อบรรทัด (สำหรับโหมด Followers และ Following)
    - **คีย์เวิร์ดค้นหา**: ป้อนคีย์เวิร์ดเพื่อค้นหาผู้ใช้ (สำหรับโหมด Keyword)
    - **จำนวนการดึงข้อมูลสูงสุด**: ตั้งค่าจำนวนผู้ใช้สูงสุดที่จะดึงข้อมูล

4. คลิก `Start Script` เพื่อเริ่มต้น

## หมายเหตุ

- ข้อมูลผู้ใช้ที่ดึงมาจะถูกบันทึกในไฟล์ตามโหมดการดึงข้อมูล:
- สำหรับโหมด Followers: `exported_users_following_tiktok_<username>_<count>.txt`
- สำหรับโหมด Following: `exported_users_followers_tiktok_<username>_<count>.txt`
- สำหรับโหมด Keyword: `exported_users_search_tiktok_<keyword>_<count>.txt`
- TikTok จำกัดจำนวนผู้ใช้ที่สามารถดูได้เท่านั้น ดังนั้นสามารถดึงข้อมูลได้ประมาณ 50 รายการต่อการดำเนินการเท่านั้น
