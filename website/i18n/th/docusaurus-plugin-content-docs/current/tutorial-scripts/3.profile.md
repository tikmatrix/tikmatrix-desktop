# โปรไฟล์

สคริปต์โปรไฟล์ใช้สำหรับตั้งค่าโปรไฟล์บัญชี

## ขั้นตอน

1. เลือกอุปกรณ์ที่จะเรียกใช้สคริปต์
2. คลิก `Script` > `Fill Profile`
3. กำหนดค่าการตั้งค่างาน:
    - **ชื่อเล่น**: หนึ่งชื่อเล่นต่อบรรทัด
    - **ชื่อผู้ใช้**: หนึ่งชื่อผู้ใช้ต่อบรรทัด
    - **ประวัติส่วนตัว**: หนึ่งประวัติต่อบรรทัด (รองรับการกล่าวถึงผู้ใช้และการขึ้นบรรทัดใหม่)
      - ในการเพิ่มการขึ้นบรรทัดใหม่ในประวัติ ให้ใช้ `\n` ในข้อความของคุณ ตัวอย่าง: `บรรทัดแรก\nบรรทัดที่สอง\nบรรทัดที่สาม`
    - **เส้นทางรูปโปรไฟล์**: เส้นทางไฟล์ไปยังโฟลเดอร์รูปภาพโปรไฟล์
    - **ลำดับการเลือก**: เลือกวิธีการเลือกเนื้อหาสำหรับแต่ละงาน:
      - **สุ่ม**: เลือกแบบสุ่มจากตัวเลือกที่มีสำหรับแต่ละงาน (ค่าเริ่มต้น)
      - **ตามลำดับ**: เลือกเนื้อหาตามลำดับตามจำนวนงาน ใช้โหมดนี้เพื่อให้แน่ใจว่าแต่ละบัญชีได้รับข้อมูลที่ไม่ซ้ำกัน
4. คลิก `Start Script` เพื่อเริ่มต้น

## คู่มือลำดับการเลือก

### โหมดสุ่ม

- เนื้อหา (ชื่อเล่น, ชื่อผู้ใช้, ประวัติ, รูปโปรไฟล์) ถูกเลือกแบบสุ่มสำหรับแต่ละงาน
- เนื้อหาเดียวกันอาจถูกใช้หลายครั้งในบัญชีที่แตกต่างกัน
- เหมาะสมเมื่อความเป็นเอกลักษณ์ของเนื้อหาไม่สำคัญ

### โหมดตามลำดับ

- เนื้อหาถูกเลือกตามลำดับ: งานที่ 1 ได้รับรายการที่ 1, งานที่ 2 ได้รับรายการที่ 2, เป็นต้น
- เมื่อรายการทั้งหมดถูกใช้แล้ว จะวนกลับไปที่จุดเริ่มต้น
- **สำคัญ**: เพื่อให้แน่ใจว่าแต่ละบัญชีได้รับข้อมูลที่ไม่ซ้ำกัน ตรวจสอบให้แน่ใจว่าคุณมีรายการมากกว่าจำนวนบัญชีทั้งหมดในการรันชุดเดียว
  - ตัวอย่าง: หากรัน 100 บัญชี ให้มีชื่อเล่น, ชื่อผู้ใช้, ประวัติ และรูปภาพโปรไฟล์ที่ไม่ซ้ำกันอย่างน้อย 100 รายการ

## หมายเหตุ

- แนะนำให้ล้างแกลเลอรีก่อนตั้งค่าโปรไฟล์ เนื่องจากสคริปต์ใช้รูปแรกเป็นรูปโปรไฟล์
