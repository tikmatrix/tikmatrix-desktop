# Ta Bort Inlägg

Ta Bort Inlägg-skriptet används för att ta bort inlägg baserat på deras visningsantal.

## Steg

1. Välj enheten för att köra skriptet.
2. Klicka på `Skript` > `Ta Bort Inlägg`.
3. Konfigurera uppgiftsinställningarna:
    - **Maximalt Visningsantal**: Ange maximalt visningsantal för inlägg du vill ta bort.
4. Klicka på `Starta Skript` för att börja.

## Notering

- Ta bort inlägg-skriptet kanske inte alltid lyckas; försök igen misslyckade uppgifter vid behov.
