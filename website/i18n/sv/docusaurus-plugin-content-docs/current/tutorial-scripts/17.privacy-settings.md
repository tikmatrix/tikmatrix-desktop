# Integritetsinställningar

**Integritetsinställningar**-skriptet används för att massuppdatera integritetsinställningar för TikTok-inlägg baserat på visningsantalfilter.

:::info
Detta skript är endast tillgängligt för TikTok-konton.
:::

## Steg

1. Välj enheten för att köra skriptet.
2. Klicka på `Skript` > `Integritetsinställningar`.
3. Konfigurera uppgiftsinställningarna:
    - **Inläggsfilterparametrar**:
      - **Maximalt Visningsantal**: Ange maximalt visningsantal för att filtrera inlägg. Endast inlägg med visningar ≤ detta nummer kommer att uppdateras.
      - **Maximala Inlägg att Bearbeta**: Ange maximalt antal inlägg att bearbeta. Ställ in på 0 för obegränsad bearbetning.
    - **Integritetsinställningsparametrar**:
      - **Vem kan titta på denna video**: Välj ett alternativ:
        - `Alla`: Vem som helst kan titta på videon
        - `Vänner`: Endast vänner kan titta på videon
        - `Endast du`: Endast du kan titta på videon
    - **Växlingsparametrar**:
      - **Tillåt kommentarer**: Aktivera eller inaktivera kommentarer på inlägget.
      - **Tillåt återanvänd innehåll**: Aktivera eller inaktivera återanvändning av innehåll, inklusive Duett, Stitch, Klistermärke och Lägg till i Berättelse.
4. Klicka på `Starta Skript` för att börja.

## Hur det Fungerar

Skriptet följer detta arbetsflöde:

1. Navigera till fliken Videor
2. Skanna inlägg och tolka visningsantal
3. Hitta inlägg som matchar filterkriterier (visningar ≤ max_visningar)
4. Gå in på inläggsdetaljsidan
5. Klicka på knappen "Integritetsinställningar"
6. Vänta på att inställningsdialogen ska visas
7. Ställ in visningstillståndet enligt ditt val
8. Växla intelligent växeln "Tillåt kommentarer"
9. Växla intelligent växeln "Tillåt återanvändning"
10. Återgå till listan och fortsätt bearbeta nästa inlägg
11. Upprepa tills alla matchande inlägg har bearbetats

## Notering

- Integritetsinställningar-skriptet kanske inte alltid lyckas; försök igen misslyckade uppgifter vid behov.
- Skriptet bearbetar endast inlägg som matchar visningsantalfiltret.
- Ändringar tillämpas omedelbart och kan inte ångras genom skriptet.
- Se till att granska dina integritetsinställningar innan du kör skriptet på flera inlägg.
