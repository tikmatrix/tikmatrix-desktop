# Logga in

Login-skriptet används för att logga in på sociala mediekonton.

## Steg

1. Se till att kontot är tillagt i kontolistan (klicka på `Konton` för att lägga till konton) och välj det enhetsindex du vill använda för detta konto.
2. Välj en eller flera enheter för att köra login-skriptet.
3. Klicka på `Skript` > `Logga in`.
4. Klicka på `Starta skript` för att börja.

## Inloggningsprioritet

Systemet använder följande inloggningsprioritet:

1. **E-post + Lösenord**: Om en e-postadress är konfigurerad för kontot kommer den att användas för inloggning först.
2. **Användarnamn + Lösenord**: Om ingen e-post är konfigurerad kommer användarnamnet att användas för inloggning.

## 2FA-stöd (endast IgMatrix)

IgMatrix stöder automatisk tvåfaktorsautentisering (2FA) vid inloggning:

### Lösenordsformat för 2FA

För att använda 2FA, formatera ditt lösenord enligt följande:

```plaintext
ditt_lösenord:din_2fa_hemliga_nyckel
```

**Exempel:**

```plaintext
mittlösenord123:JBSWY3DPEHPK3PXP
```

### Hur 2FA fungerar

1. **Lösenordsparsning**: Systemet identifierar automatiskt om ditt lösenord innehåller en kolon (`:`) separator.
2. **Token-generering**: När 2FA krävs använder systemet din hemliga nyckel för att begära en TOTP-kod från vår 2FA-tjänst.
3. **Automatisk inmatning**: Den genererade 6-siffriga koden matas automatiskt in i Instagrams 2FA-fält.
4. **Sömlös inloggning**: Hela 2FA-processen sker automatiskt utan manuell åtgärd.

### 2FA hemlig nyckel

Den hemliga 2FA-nyckeln är vanligtvis en 32-teckens Base32-kodad sträng som du får när du ställer in 2FA i Instagram. Det är samma nyckel som används av autentiseringsappar som Google Authenticator eller Authy.

### Var man kan köpa Instagram-konton med 2FA

Du kan köpa Instagram-konton med 2FA-stöd från: [AccsMarket](https://accsmarket.com/en/?ref=802647)

## Obs

- TikTok-inloggning är inte 100% helt automatisk. Du kan behöva manuellt ange e-postverifieringskoder eller slutföra andra verifieringssteg. Vi kan inte kringgå TikTok-specifika inloggningsutmaningar (såsom bild-CAPTCHA eller mänsklig verifiering) eller lösa vissa felmeddelanden som "för många inloggningsförsök".
- Instagram kan uppnå 100% automatiserad inloggning via 2FA. För 2FA-aktiverade Instagram-konton, se till att använda rätt hemlig nyckelformat i ditt lösenord.
