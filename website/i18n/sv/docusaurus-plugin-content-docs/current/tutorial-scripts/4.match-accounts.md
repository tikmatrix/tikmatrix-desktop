# Matcha konton

Skriptet Matcha konton används för att känna igen TikTok-konton från telefonen och lägga till dem i kontolistan.

## Steg

1. Välj enheten för att köra skriptet.
2. Klicka på `Skript` > `Matcha konton`.
3. Klicka på `Starta skript` för att börja.
4. Skriptet kommer automatiskt att känna igen konton på telefonen och lägga till dem i kontolistan.

## Obs

- Skriptet för att matcha konton kanske inte känner igen alla konton i en körning. Om några konton missas, försök köra skriptet igen.
