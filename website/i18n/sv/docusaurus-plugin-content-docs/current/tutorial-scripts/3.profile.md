# Profil

Profilskriptet används för att konfigurera kontoprofiler.

## Steg

1. Välj enheten för att köra skriptet.
2. Klicka på `Skript` > `Fyll i profil`.
3. Konfigurera uppgiftsinställningarna:
    - **Smeknamn**: Ett smeknamn per rad.
    - **Användarnamn**: Ett användarnamn per rad.
    - **Biografier**: En biografi per rad (stöder att nämna användare och radbrytningar).
      - För att lägga till radbrytningar i biografin, använd `\n` i din text. Till exempel: `Första raden\nAndra raden\nTredje raden`
    - **Avatar-sökväg**: Filsökväg till avatar-bildmappen.
    - **Urvalsordning**: Välj hur innehåll väljs för varje uppgift:
      - **Slumpmässig**: Väljer slumpmässigt från tillgängliga alternativ för varje uppgift (standard).
      - **Sekventiell**: Väljer innehåll i ordning baserat på uppgiftsantal. Använd detta läge för att säkerställa att varje konto får unik information.
4. Klicka på `Starta skript` för att börja.

## Guide för urvalsordning

### Slumpmässigt läge

- Innehåll (smeknamn, användarnamn, biografier, avatarer) väljs slumpmässigt för varje uppgift
- Samma innehåll kan användas flera gånger för olika konton
- Lämpligt när innehållets unikhet inte är kritisk

### Sekventiellt läge

- Innehåll väljs i ordning: 1:a uppgiften får 1:a objektet, 2:a uppgiften får 2:a objektet, etc.
- När alla objekt har använts börjar det om från början
- **Viktigt**: För att säkerställa att varje konto får unik information, se till att du tillhandahåller fler objekt än det totala antalet konton i en enskild batchkörning
  - Exempel: Om du kör 100 konton, tillhandahåll minst 100 unika smeknamn, användarnamn, biografier och avatar-bilder

## Obs

- Det rekommenderas att rensa galleriet innan du ställer in profilen, eftersom skriptet använder det första fotot som avatar.
