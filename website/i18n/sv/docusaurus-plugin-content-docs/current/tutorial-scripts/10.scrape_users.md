# Skrapa Användare

Skrapa Användare-skriptet används för att skrapa TikTok-användarnamn. Användarskapningsfunktionen stöder tre lägen: samla in följarlistan för en angiven användare, samla in följ-listan för en angiven användare, eller samla in användare genom att söka med nyckelord.

## Steg

1. Välj enheten för att köra skriptet. Om du väljer flera enheter kommer skapningsuppgifterna att fördelas jämnt mellan de valda enheterna.
2. Klicka på `Skript` > `Skrapa Användare`.
3. Konfigurera uppgiftsinställningarna:
    - **Skapningsläge**: Välj läge för att skrapa användare.
      - **Skrapa efter Följare**: Samla användare från följarlistan för angivna konton.
      - **Skrapa efter Följer**: Samla användare från följ-listan för angivna konton.
      - **Skrapa efter Nyckelord**: Sök och samla användare med hjälp av nyckelord.
    - **Målkonto Användarnamn**: Ange användarnamnet för kontot att skrapa användare från, ett per rad (för Följare och Följer-lägen).
    - **Söknyckelord**: Ange nyckelord för att söka efter användare (för Nyckelord-läge).
    - **Max skapningsantal**: Ställ in maximalt antal användare att skrapa.

4. Klicka på `Starta Skript` för att börja.

## Notering

- Den skrapade användardatan kommer att sparas i en fil baserad på skapningsläget:
- För Följare-läge: `exported_users_following_tiktok_<användarnamn>_<antal>.txt`
- För Följer-läge: `exported_users_followers_tiktok_<användarnamn>_<antal>.txt`
- För Nyckelord-läge: `exported_users_search_tiktok_<nyckelord>_<antal>.txt`
- TikTok begränsar endast antalet användare som kan visas, så endast cirka 50 poster kan hämtas per operation.
