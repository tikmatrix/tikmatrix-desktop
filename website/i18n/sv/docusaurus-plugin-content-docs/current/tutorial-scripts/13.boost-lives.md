# Boosta Livesändningar

Boosta Livesändningar-skriptet låter dig kontrollera farmkonton för att titta på och valfritt gilla eller kommentera mållivesändningar. Du kan ställa in visningstiden och gilla/kommentarsintervaller för att anpassa boostbeteendet.

## Steg

1. Välj enheten för att köra skriptet.
2. Klicka på `Skript` > `Boosta Livesändningar`.
3. Konfigurera uppgiftsinställningarna:
    - **Användarnamn**: Ange målanvändarnamn, ett per rad.
    - **Inträdesmetod**: Välj mellan **Sök** och **Direkt** metoder.
    - **Boostalternativ**: Aktivera gillande och/eller kommentering.
    - **Gå Med i Fanklubb**: Aktivera att gå med i streamerns fanklubb om tillgänglig.
    - **Checka In Dagligen**: Aktivera dagliga incheckningsaktiviteter under livesändningen.
    - **Gillarintervall**: Ställ in intervallet mellan varje gillaråtgärd.
    - **Gillaknappstryckningar**: Antal gånger att trycka på gillaknappen.
    - **Kommentarsintervall**: Ställ in intervallet mellan varje kommentarsåtgärd.
    - **Kommentarsinnehåll**: Ange en kommentar per rad.
    - **Infoga Emojis**: Välj om emojis ska infogas i kommentarer.
    - **Kommentarsordning**: Välj slumpmässig eller sekventiell ordning för kommentarer.
    - **Kommentarer per Konto**: Ställ in antalet kommentarer att posta per konto.
    - **Visningstid**: Ställ in hur länge man ska stanna i livesändningen.
    - **Uppgiftsintervall**: Ställ in intervallet mellan varje uppgift.
4. Klicka på `Starta Skript` för att börja.

## Noteringar

- Boosta livesändningar-skriptet kanske inte alltid lyckas; försök igen misslyckade uppgifter vid behov.
