# Kontouppvärmning

Skriptet för kontouppvärmning används för att värma upp konton genom att söka nyckelord, titta på videor, följa, gilla, kommentera och favorisera.

## Steg

1. Välj enheten för att köra skriptet.
2. Klicka på `Skript` > `Kontouppvärmning`.
3. Konfigurera uppgiftsinställningarna:
   - **Visningstid**: Hur länge ska varje video tittas på.
   - **Söknyckelord**: Ämnen att söka efter videor för.
   - **Uppgiftslängd**: Total varaktighet för uppvärmningsuppgiften.
   - **Kommentarsinnehåll**: Använd färdiga kommentarer eller ChatGPT-genererade kommentarer från inläggstexter.
   - **Interaktionssannolikhet**: Sannolikhet för att följa, gilla, kommentera och favorisera.
4. Klicka på `Starta skript` för att börja.

## Obs

- Uppvärmningsskriptet kanske inte alltid lyckas; försök misslyckade uppgifter igen vid behov.
- Stöder schemaläggning, så du kan ställa in uppvärmningsuppgiften att köras vid en specifik tid.
- Stöder att hoppa över annonser, liveströmmar och redan gillade videor.
