# Masskommentera

:::warning
Denna funktion har slagits samman i [Super Marketing-skriptet](./14.super-marketing.md). Föredra Super Marketing för mer kraftfulla och flexibla masskommenteringsfunktioner och engagemangsfunktioner.
:::

Masskommentera används för att posta engagemangssvar över många TikTok-inlägg—användbart för kontouppfostran, marknadsföring eller aktivitetsökning. Systemet stöder både förgenererade mallar och realtidsgenerering, och det försöker efterlikna mänsklig tidpunkt för att minska detektionsrisk.

## Använda Masskommentera inuti Super Marketing

1. Öppna Super Marketing-skriptet.
2. Välj datakälltyp: "Inläggs-URL-lista".
3. Klistra in inläggs-URL:er i bulk eller importera en `.txt`-fil (en URL per rad). Efter import kan du välja att "Lägg till" eller "Ersätt" den aktuella listan.
4. Välj en dataanvändningsstrategi:

   - Delad pool: alla bot-konton kommer att kommentera från samma målpool.

   - Engångsförbrukning: varje bot-konto bearbetar olika mål så att varje inlägg får en kommentar från endast ett konto.

5. Ställ in hur många kommentarer som ska postas per mål (vanligtvis 1 för att undvika duplicerade kommentarer).
6. Konfigurera kommentarsinställningar:

   - Förgenererade mallar: klistra in flera rader eller stycken av förberedda kommentarskopior. Varje rad/stycke behandlas som en kandidatkommentar.

   - Kommentarsordning: välj "Slumpmässig" eller "Sekventiell".

   - Slumpmässig: varje sändning väljer en slumpmässig mall.

   - Sekventiell: rotera mallar i inklistrad/importordning (användbart när du behöver en reproducerbar dedupliceringsstrategi).

   - Infoga emojis: aktivera automatisk infogning av emojis för att öka naturligheten.

   - Infoga enhetsnummer (valfritt): bifoga enhetens numeriska etikett till kommentarer för spårning (använd med försiktighet; avsett för felsökning/granskning).

   - Realtidskommentargenerering (valfritt, ChatGPT API):

      - När aktiverat kan skriptet anropa ChatGPT eller ett liknande API för att generera kontextuella och unika kommentarer baserade på inläggstiteln/beskrivningen eller tillgängligt sammanhang—lämpligt när du behöver många unika kommentarer.

      - Notering: att aktivera realtidsgenerering kräver en giltig API-nyckel och du måste överväga hastighetsbegränsningar, kostnader och integritet (undvik att skicka känslig data till tredjepartstjänster).

   - Två vanliga sätt att garantera unika kommentarer:

      1. Använd sekventiell rotation av förgenererade mallar (varje mall används en gång tills den är förbrukad).

      2. Använd ChatGPT realtidsgenerering för att skapa unika kommentarer per mål.

7. Konfigurera andra skriptparametrar (samtidighet, uppgiftsintervall, kontopool, etc.) och starta skriptet när det är klart.

:::tip Om uppgiftsintervall
Ställ in ett minimum- och maximumintervall och aktivera slumpmässiga fördröjningar. Förskjutna tidpunkter minskar risken för synkroniserade aktivitetstoppar och sänker detektionsrisk.
:::

## Rekommendationer och noteringar

- Testa kommentarsstrategier på en liten batch först för att validera kvalitet och plattformssvar.
- Håll kommentarstext kompatibel och varierad; undvik att skicka många identiska meddelanden på kort tid.
- Om du använder realtidsgenerering, övervaka strikt API-användning och kostnader och undvik att skicka känslig information.
- Uppgifter kan misslyckas; kontrollera loggar och försök igen misslyckade rader. Bearbetade mål kommer att hoppas över vid efterföljande körningar.
