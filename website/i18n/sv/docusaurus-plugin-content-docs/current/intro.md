# Handledning Introduktion

Upptäck **TikMatrix** och **IgMatrix** på mindre än 5 minuter och lär dig kärnfunktioner och användning.

## Programvaruintroduktion

**TikMatrix** är en PC-skrivbordsapplikation som automatiserar TikTok-uppgifter på Android-enheter.  
**IgMatrix** är motsvarande verktyg för att automatisera Instagram-uppgifter på Android-enheter.

**Notera:** **TikMatrix** och **IgMatrix** är separata applikationer och kan inte köras samtidigt på samma dator.

## Funktioner

- **Byte av flera konton**: Hantera flera konton per enhet och växla automatiskt under uppgifter.
- **Stöd för klonade appar**: Stöd för klonade TikTok/Instagram-appar med anpassade paketnamn så att olika konton kan använda olika kloner och automatiskt öppna målappen.
- ~~**Automatisk registrering**~~ **[AVVECKLAD]**
- **Automatisk inloggning**: Instagram stöder automatisk 2FA-inloggning; TikTok kan fortfarande kräva manuell intervention i vissa verifieringsflöden (captcha, e-postverifiering).
- **Automatisk profilfyllning**: Ställ in avatar, smeknamn, biografi och användarnamn automatiskt.
- **Automatisk kontoigenkänning**: Identifiera och importera konton som redan är inloggade på telefonen med ett klick.
- **Kontouppvärmning / tillväxt**: Sök nyckelord, titta på videor, gilla, följ, kommentera och favoritmarkera för att värma upp konton.
- **Automatisk publicering**: Publicera videor eller bilder (stöder sortering och batch-antal) med anpassningsbar musik, titel, beskrivning, hashtaggar och produktlänkar.
- **Automatisk borttagning**: Ta bort underpresterande inlägg enligt visningsregler.
- **Supermarknadsföring**: Simulera människoliknande surfvägar (besök profiler, bläddra i inlägg, slumpmässig följ/gilla/kommentera/favoritmarkera/dela/DM) och använd valfritt OpenAI eller kompatibla API:er för att generera personliga DM:er/kommentarer.
- **Batch-förstärkning (Live)**: Gilla och kommentera livestreams i bulk.
- **Användarinhämtning**: Samla in användare efter nyckelord, följare eller följande listor.
- **Massåterfölj & välkomst-DM:er**: Följ följare i bulk och skicka välkomstmeddelanden.
- **Minutnivå uppgiftsschemaläggning**: Stöd för återförsök, intervallkörning och maximalt antal återförsök.
- **Multi-enhet parallellt**: En PC kan ansluta och hantera många Android-enheter som kör uppgifter parallellt.
- **Skärmspegeling**: Spegla flera enhetsskärmar till Windows / macOS och kontrollera dem med mus och tangentbord.
- **Importera / Exportera konton**: Excel-vänlig bulkimport/export.
- **Flerspråkigt stöd**: Programvaran stöder engelska, ryska och kinesiska. Notera: Vissa funktioner kräver att TikTok/Instagram-appen på din telefon är inställd på engelska för optimal kompatibilitet.
- **Support & biljetter**: Tillhandahåller mänsklig biljettsupport för problem.

## Vad du behöver

### Android-telefoner

Du kan använda vilken Android-enhet som helst för att köra TikMatrix, inklusive:

- **Riktiga Android-enheter**
- **[Moderkort Android-enheter](http://www.niaozun.shop?cid=934ec2fe)**
- **[Molntelefoner](https://www.geelark.cn?invite_code=XHY6a8)**
- **Emulatorer**
- **Virtuella maskiner**

Android-enheter kräver minst API 21 (Android 5.0) och kräver inte root. Vänligen aktivera USB-felsökning (Utvecklaralternativ). På vissa modeller (t.ex. Xiaomi) kan du behöva aktivera "USB-felsökning (Säkerhetsinställningar)" för att tillåta tangentbords-/muskontroll.

### Dator

#### Windows: Windows 10 eller senare

#### Mac: macOS 15.0 eller senare (Apple Silicon & Intel stöds)

## Installation

1. Ladda ner den senaste installationsprogrammet från [Ladda ner TikMatrix](https://tikmatrix.com/Download) eller [Ladda ner IgMatrix](https://igmatrix.com/Download-IgMatrix) sidan.
2. Kör installationsprogrammet.
3. Vänta på att installationen ska slutföras.
4. Öppna **TikMatrix** eller **IgMatrix** appen på din dator.
