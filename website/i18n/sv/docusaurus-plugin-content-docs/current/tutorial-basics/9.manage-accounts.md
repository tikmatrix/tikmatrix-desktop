# Kontohantering

Kontohantering håller varje TikTok- eller Instagram-identitet synkroniserad med de enheter de körs på, de uppgifter de behöver och den data din automatisering är beroende av. En ren kontolista gör det enkelt att delegera enheter, övervaka kontohälsa och förbereda kampanjer utan att röra varje telefon manuellt.

## Skapa ett konto manuellt

1. Öppna **Konton** i vänster navigation.
2. Klicka på **Lägg till** för att öppna redigeringsdialogen.
3. Fyll i grundläggande fält:
   - **E-post**: adressen som används av appen. TikMatrix/IgMatrix försöker först en e-post + lösenordsinloggning; om e-posten är tom, återgår den automatiskt till användarnamn + lösenord.
   - **Lösenord**: uppgifterna som används i appen. IgMatrix accepterar också ett lösenord utökat med en 2FA-hemlighet (se nedan).
   - **Användarnamn** — handtaget som visas i TikTok/Instagram. Detta är det enda obligatoriska fältet.
   - Växla **Inloggningsstatus** för att markera om app-sessionen redan är autentiserad på enheten (1 = Inloggad, 0 = Inte inloggad).
   - Växla **Status** för att hålla kontot aktivt (0 = Aktiverat) eller inaktivera det (1 = Inaktiverat).
4. Välj en enhetsbricka för att binda kontot till en telefon eller CloudPhone. Om ingen enhet väljs förblir kontot otilldelat.
5. Klicka på **Spara**. Listan uppdateras automatiskt.

## Inloggning

- När både **E-post** och **Lösenord** tillhandahålls kommer TikMatrix/IgMatrix att logga in med det paret på den tilldelade enheten.
- Om fältet **E-post** är tomt kommer det att prova kombinationen **Användarnamn** + **Lösenord** istället.
- Se till att ett enhetsindex är valt så att inloggningsskriptet vet var det ska köras.
- Inloggningsautomatiseringen upptäcker redan autentiserade sessioner och hoppar över dem automatiskt.

## 2FA-stöd (endast IgMatrix)

IgMatrix kan hantera tvåfaktorsautentisering under automatiserade inloggningar.

### 2FA-lösenordsformat

För att aktivera 2FA, lagra lösenordet i följande format:

```text
ditt_lösenord:din_2fa_hemlighet
```

**Exempel:**

```text
mypassword123:JBSWY3DPEHPK3PXP
```

## Importera och exportera konton

- **Exportera** skriver varje konto till `accounts.xlsx` i din TikMatrix/IgMatrix-nedladdningsmapp. Använd detta som en levande säkerhetskopia eller som mall för massredigeringar.
- **Importera** låter dig skapa eller uppdatera konton i bulk från det Excel-arket. Om du är osäker på huvudformatet, klicka först på **Exportera**, redigera den nedladdade filen och importera sedan tillbaka den.
- Under import uppdaterar varje rad som innehåller ett `id` den befintliga posten. Rader utan `id` skapar nya konton.

> För IgMatrix 2FA-inloggningar, fyll i `username`, `pwd` (med formatet `password:2fa_secret`) och kolumnen `device_index`.

### Excel-kolumnreferens

| Kolumn | Obligatorisk | Beskrivning |
| --- | --- | --- |
| `id` | Nej | Intern identifierare. Inkludera den endast när du avser att uppdatera ett befintligt konto. |
| `email` | Nej | Inloggnings-e-post som används av appen. Användbart för kontakt och återställning. |
| `pwd` | Nej | Lagrat lösenord för snabb referens eller automatiserade inloggningar. |
| `username` | Ja | TikTok/Instagram-handtag. Måste vara unikt inom arbetsytan. |
| `device` | Nej | Enhetsserie eller CloudPhone-identifierare bunden till kontot. |
| `device_index` | Nej | Vänlig enhetsetikett som visas i gränssnittet; främst för referens. |
| `logined` | Nej | Sessionsflagga (`1` = inloggad, `0` = inte inloggad). |
| `status` | Nej | Tillgänglighetsflagga (`0` = aktiverad, `1` = inaktiverad). Inaktiverade konton deltar aldrig i skriptkörningar. |
| `nickname` | Nej | Senaste smeknamn hämtat från TikTok-datasynkronisering. |
| `country` | Nej | Land härlett från TikTok-profilen. |
| `followers` | Nej | Antal följare synkroniserade från TikTok. |
| `following` | Nej | Antal konton denna profil följer. |
| `hearts` | Nej | Totalt antal gillningar (hjärtan) på profilen. |
| `videos` | Nej | Antal publicerade videor. |
| `friends` | Nej | Antal ömsesidiga vänner när tillgängligt. |

### TikMatrix vs. IgMatrix-fält

- **TikMatrix** fyller i statistikfälten (`nickname`, `country`, `followers`, `following`, `hearts`, `videos`, `friends`) efter att du kör **Synkronisera TikTok-data** eller använder datafrågeknappen.
- **IgMatrix** använder endast grundkolumnerna (`email`, `pwd`, `username`, `device`, `logined`, `status`) och ignorerar analyskolumnerna.

## Ytterligare beteenden

- Konton markerade som **Inaktiverade** exkluderas från varje automatiseringsskript, schemalagd uppgift och massåtgärd. Återaktivera dem innan du startar nya jobb.
- Skriptet **Matcha konton** skannar enheter efter app-sessioner som redan är inloggade, skapar automatiskt motsvarande poster i kontolistan och länkar dem till den upptäckta enheten. Detta är det snabbaste sättet att introducera telefoner som har förberetts manuellt.
