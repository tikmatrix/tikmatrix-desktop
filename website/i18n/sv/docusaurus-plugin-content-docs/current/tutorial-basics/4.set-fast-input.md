# Aktivera snabbinmatningstangentbord

När din telefon är ansluten kommer TikMatrix automatiskt att växla till snabbinmatningstangentbordet.

## Varför använda snabbinmatningstangentbordet?

Snabbinmatningstangentbordet är avgörande för TikMatrix och ger automatiskt flerspråksstöd. Utan det kan textinmatningsfunktioner som kommentarer, meddelanden och publicering av titlar inte fungera korrekt.

## Hur man manuellt aktiverar snabbinmatningstangentbordet

1. Välj enheten du vill använda.
2. Klicka på `ADB Commands` > `Enable Fast Input`.

## Hur man matar in text

Du kan mata in text i TikMatrix med flera metoder:

### Metod 1: Direkt tangentbordsinmatning ⌨️ (Ny funktion)

⭐ **Tangentbordsinmatningskontroll** - Du kan nu skriva direkt på dina telefoner med hjälp av datorns tangentbord, vilket gör textinmatning snabbare och mer bekväm.

1. Välj enheten du vill använda.
2. Klicka på vilket textfält som helst på telefonskärmen.
3. Börja skriva direkt på datorns tangentbord - texten kommer att visas på din telefon i realtid.

### Metod 2: Inmatningspanel

1. Klicka på telefonskärmen för att expandera den, klicka sedan på knappen `Input` i höger verktygsfält.
2. Ange din text i inmatningsrutan och klicka på skicka.

## Kopiera och klistra in mellan telefon och dator

### Kopiera telefonens urklipp till datorn

1. Välj telefonen, håll ned `Ctrl+C` för att kopiera telefonens urklipp till din dator.

### Klistra in datorns urklipp till telefonen

1. Välj enheten, håll ned `Ctrl+V` för att klistra in datorns urklipp i telefonens inmatningsruta.
