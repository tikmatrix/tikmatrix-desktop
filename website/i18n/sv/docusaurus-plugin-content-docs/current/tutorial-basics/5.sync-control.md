# Synkroniseringskontroll

Synkroniseringskontroll låter dig synkronisera operationer över flera enheter.

## Hur man använder

1. Öppna skärmen på en telefon.
2. Välj flera enheter från listan till vänster.
3. Alla skärmoperationer kommer att synkroniseras till de valda enheterna.
