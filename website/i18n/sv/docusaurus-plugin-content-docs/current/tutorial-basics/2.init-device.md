# Initialisera enheter

:::warning
Denna applikation installeras via Androids officiella ADB-felsökningskanal för enhetsautomationskontroll. Vissa enheter kan fråga om att skicka ett okänt appexempel till Google; detta är en Android-systemsäkerhetsprompt som inte har med appsäkerhet att göra och påverkar inte normal användning. Klicka på "Don't send" för att slutföra installationen.
:::

TikMatrix APK är en mobil agent som krävs för automatiseringsuppgifter. **När du ansluter din telefon till TikMatrix för första gången kommer systemet automatiskt att initialisera din enhet och installera TikMatrix-appen** - ingen manuell intervention krävs.

Under den automatiska initialiseringsprocessen kommer TikMatrix att installera två APK-filer: TikMatrix och TikMatrix Test (inget gränssnitt). Om din telefon ber om installationsauktorisering, tryck helt enkelt på "Tillåt" (särskilt viktigt på Xiaomi-enheter).

## Automatisk vs manuell installation

**Automatisk installation (Rekommenderad)**: Som standard initialiserar TikMatrix automatiskt enheter när de ansluts första gången. Detta är den enklaste och mest praktiska metoden.

**Manuell installation/uppdatering**: Behövs endast i specialfall eller för uppdateringar.

## Automatisk initialiseringsprocess

När du ansluter en ny enhet till TikMatrix:

1. **Enhetsdetektering**: TikMatrix detekterar automatiskt den anslutna enheten.
2. **Automatisk installation**: Systemet kommer automatiskt att installera TikMatrix-appen utan användarintervention.
3. **Auktorisering**: Om din enhet ber om installationsbehörighet, tryck helt enkelt på "Tillåt".
4. **Slutförande**: Initialiseringsprocessen slutförs på cirka 10 sekunder.
5. **Redo att användas**: Din enhet är nu redo för automatiseringsuppgifter.

Denna automatiska process eliminerar behovet av manuell installation i de flesta fall.

## Manuell installation/uppdatering

För att manuellt installera eller uppdatera TikMatrix APK, stäng av växlaren `Auto WakeUP Agent` i Inställningar. Följ sedan dessa steg:

1. Välj din telefon och klicka på `General` > `Init Agent`.
2. TikMatrix kommer automatiskt att installera båda APK-filerna på din telefon.
3. Om du blir uppmanad, tryck på "Tillåt" för att auktorisera installationen.
4. Om den blockeras av Google Play Protect, tryck på "Install anyway" för att fortsätta.
5. Vänta tills installationen är klar (cirka 10 sekunder).
6. Slå på växlaren `Auto WakeUP Agent` för att aktivera automatisk appuppvakning när telefonen ansluts.
7. Klicka på `General` > `Open Agent` för att testa installationen. Om appen inte startar, kontakta teknisk support.
