# Installera TikTok

Vi rekommenderar att ladda ner de specifika TikTok-versionerna nedan, eftersom nyare versioner kanske inte är kompatibla. Stödda versioner kommer att tillkännages i gruppen efter uppdateringar.

## Ladda ner TikTok APK

- Global version: [com.zhiliaoapp.musically](https://apkpure.com/tiktok-musically-2024/com.zhiliaoapp.musically)
- Asiatisk version: [com.ss.android.ugc.trill](https://apkpure.com/tiktok/com.ss.android.ugc.trill)

## Installationssteg

1. Ladda ner APK från länkarna ovan.
2. Välj alla enheter i TikMatrix.
3. Klicka på `General` > `Install APK` och välj den nedladdade APK-filen.
4. Vänta tills installationen är klar (kan ta 1-3 minuter per enhet beroende på modell; klicka inte upprepade gånger).
5. Efter installationen, klicka på `Settings` och verifiera att TikTok-versionen matchar din installerade app.
6. Klicka på `Open TikTok` för att kontrollera om appen är korrekt installerad.
7. Se till att TikToks språk är inställt på engelska; annars kanske skript inte fungerar korrekt. Ändra språket i TikTok-inställningarna om det behövs.
