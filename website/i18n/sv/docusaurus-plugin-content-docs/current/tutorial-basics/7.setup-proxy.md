# Konfigurera proxy

Du kan ställa in en unik proxy-IP-adress för varje telefon.

> **Ansvarsfriskrivning:** Alla webbplatser som rekommenderas här kommer från internet och är inte anslutna till denna webbplats. Vänligen verifiera deras giltighet före användning. Vi tillhandahåller ingen garanti.

## 1. Konfigurera residential VPS-proxy (Om du köper en proxy-IP kan du hoppa över detta steg)

1. Köp en residential proxy-IP VPS från ditt målland (helst med en nativ IP och dual ISP VPS).
2. Använd [https://github.com/MHSanaei/3x-ui](https://github.com/MHSanaei/3x-ui) för att snabbt konfigurera en VPN på VPS:

    ```shell
    bash <(curl -Ls https://raw.githubusercontent.com/MHSanaei/3x-ui/refs/tags/v2.6.0/install.sh)
    # Följ instruktionerna för att slutföra VPN-konfigurationen.
    ```

3. Efter installation, öppna x-ui webbgränssnittet. Du hittar URL:en och inloggningsuppgifterna i terminalutskriften efter installationen.
4. Skapa en ny användare i x-ui, ställ in protokollet till `SOCKS5` och konfigurera porten och andra inställningar efter behov.
5. Efter att ha skapat användaren ser du proxy-IP och port i x-ui-gränssnittet.
6. Använd proxy-IP och port i NekoBox-appen på din telefon.

## 2. Köp residential proxy-IP (Om du konfigurerat en VPS kan du hoppa över detta steg)

Du kan också köpa residential proxy-IP direkt utan att konfigurera en VPS.

> Rekommenderade proxy-leverantörer

- [KlickaHär](/advertising)

> Verktyg för IP-kontroll

- [ipinfo](https://ipinfo.io) — Kontrollera om IP:n är residential.
- [scamalytics](https://scamalytics.com/ip) — Kontrollera IP:ns risknivå.

## 3. Aktivera proxy på telefonen

1. Ladda ner APK:n och installera [NekoBoxForAndroid](https://github.com/MatsuriDayo/NekoBoxForAndroid)-klienten på din telefon.
2. Öppna NekoBox-appen och klicka på `+`-knappen för att lägga till en ny proxy.
3. Välj `SOCKS5` eller andra proxy-typer.
4. Ange proxy-IP och port, klicka sedan på `Spara`.
5. Klicka på `Starta`-knappen för att aktivera proxyn.
6. Efter att proxyn är aktiverad kan du kontrollera anslutningsstatus i NekoBox-appen.

### För kinesiska användare

Om du är i Kina kanske `SOCKS5`-protokollet inte fungerar bra. Du behöver lägga till en krypterad proxy (`VLESS`) som frontend-proxy. Du kan använda följande verktyg:

1. Lägg till en `VLESS`-proxy i NekoBox-appen.
2. Lägg till en `SOCKS5`-proxy i NekoBox-appen.
3. Lägg till en `Chain`-proxy i NekoBox-appen. Och välj `VLESS`-proxyn som frontend-proxy och `SOCKS5`-proxyn som backend-proxy.
