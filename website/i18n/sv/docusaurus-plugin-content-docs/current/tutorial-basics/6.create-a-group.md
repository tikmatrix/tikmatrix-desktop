# Skapa grupp

En **grupp** är en samling enheter som gör det enklare att hantera och använda flera enheter tillsammans.

## Hur man skapar en grupp

1. Klicka på knappen `Lägg till grupp` i den vänstra sidopanelen.
2. Ange ett gruppnamn.
3. Tryck `Enter` för att spara gruppen.
4. Välj enheter för att flytta dem till gruppen.
