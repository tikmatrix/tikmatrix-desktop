# Anpassade ADB-kommandon

TikMatrix låter dig skapa och hantera anpassade ADB-kommandon för snabb exekvering av vanliga operationer, vilket effektiviserar ditt arbetsflöde och förbättrar effektiviteten.

## Vad är anpassade ADB-kommandon?

Anpassade ADB-kommandon låter dig spara ofta använda ADB-instruktioner och exekvera dem med ett enda klick, vilket eliminerar behovet av att manuellt ange kommandon varje gång.

## Åtkomst till gränssnittet för anpassade kommandon

1. I TikMatrix huvudgränssnitt, klicka på **Anpassade kommandon** i den vänstra sidopanelen.
2. Gränssnittet visar alla dina sparade kommandon, med alternativ för att lägga till eller återställa kommandon.

## Förinställda kommandon

TikMatrix tillhandahåller flera användbara förinställda kommandon:

- **Home**: Återvänd till enhetens hemskärm
- **Back**: Utför en tillbaka-operation
- **Enable TCP**: Aktivera TCP/IP-anslutningsläge
- **Enable Fast Input**: Aktivera snabbinmatningsmetoden
- **Open Gallery**: Öppna enhetens galleri
- **Open Settings**: Öppna enhetsinställningar
- **Open NekoBox**: Öppna NekoBox-appen (om installerad)

## Lägga till anpassade kommandon

1. Klicka på **Lägg till kommando** överst i gränssnittet.
2. I dialogrutan, fyll i:
   - **Kommandonamn**: Ett igenkännligt namn för ditt kommando
   - **Kommandoargument**: ADB-kommandoparametrarna (behöver inte inkludera "adb"-prefixet)
3. Klicka på **Lägg till kommando** för att spara.

## Stöd för flerradiga kommandon

Anpassade kommandon stöder flerradig inmatning; varje rad behandlas som ett separat ADB-kommando och exekveras i sekvens.
