# خطأ بدء تشغيل البرنامج

## خطوات استكشاف أخطاء بدء تشغيل البرنامج الشائعة

### الخطوة 1: التحقق من متطلبات النظام

1. تأكد من أن نظامك يلبي الحد الأدنى من المتطلبات:
   - Windows 10 أو أحدث
   - 4 جيجابايت على الأقل من ذاكرة الوصول العشوائي
   - 1 جيجابايت على الأقل من مساحة القرص المتاحة

### الخطوة 2: التحقق من Microsoft Visual C++ Redistributable

عادةً، خطأ `vcruntime140.dll غير موجود` أو خطأ `فشل تشغيل التطبيق بشكل صحيح (0xc000007b)` ناتج عن عدم تثبيت Microsoft Visual C++ Redistributable.

1. اضغط على مفاتيح `Windows + R`، واكتب `cmd` واضغط على Enter
2. اكتب `dir "C:\Windows\System32\vcruntime*.dll"` واضغط على Enter
3. إذا كان هناك إخراج، فهذا يعني أن إصدارًا ذا صلة من Visual C++ Redistributable مثبت. عادةً ما يحتوي اسم الملف على لاحقة vcruntime مع رقم الإصدار، مثل vcruntime140.dll (يقابل 2015-2022).
4. إذا لم يكن هناك إخراج، يرجى تنزيل وتثبيت Microsoft Visual C++ Redistributable: [أحدث إصدار من Microsoft Visual C++ Redistributable](https://learn.microsoft.com/en-us/cpp/windows/latest-supported-vc-redist?view=msvc-170)

### الخطوة 3: التحقق من وقت النظام

عدم دقة ساعة الكمبيوتر سيؤدي إلى خطأ "Agent Start Timeout":

1. انقر بزر الماوس الأيمن على عرض الوقت في الزاوية السفلية اليمنى من الشاشة
2. حدد "ضبط التاريخ/الوقت"
3. إذا لم يتم تمكينه بعد، قم بتشغيل "ضبط الوقت تلقائيًا"
4. إذا كان الوقت لا يزال غير صحيح، انقر على "مزامنة الآن" أو اضبط التاريخ والوقت الصحيحين يدويًا
5. أعد تشغيل TikMatrix بعد تصحيح الوقت

### الخطوة 4: التحقق من سجلات التطبيق

1. انقر على زر `الإعدادات` - `فتح دليل التطبيق`
2. انتقل إلى مجلد `logs`
3. تحقق من رسائل الخطأ في أحدث ملف سجل

### الخطوة 5: macOS — خطأ Agent InvalidStateError

على بعض أجهزة macOS، قد يفشل Agent في البدء ويظهر الخطأ التالي:

```text
Agent start error InvalidStateError: The object is in an invalid state.
```

الأسباب المحتملة:

- عادةً ما يكون ذلك بسبب عدم التوافق بين التطبيق وإصدار نظام macOS الحالي أو الأجهزة/البرامج الثابتة القديمة.

الحل / الحل البديل:

1. قم بتحديث macOS إلى أحدث إصدار من خلال "إعدادات النظام" > "عام" > "تحديث البرنامج". بعد التحديث، أعد تشغيل Mac وأعد تشغيل TikMatrix.
2. إذا كان التحديث غير ممكن أو استمر الخطأ، حاول تشغيل التطبيق على Mac آخر يعمل بإصدار أحدث من macOS.

إذا استمرت المشكلة بعد تجربة الطرق المذكورة أعلاه، يرجى جمع سجلات التطبيق والاتصال بنا عبر [Telegram](https://t.me/tikmatrix_agent_bot) مع إصدار macOS الخاص بك وملفات السجل.

يرجى الاتصال بفريق الدعم لدينا عبر [Telegram](https://t.me/tikmatrix_agent_bot) وتقديم المعلومات التالية:

- إصدار Windows
- ملفات سجل التطبيق
- الخطوات التي جربتها بالفعل
