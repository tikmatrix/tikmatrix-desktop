# Permission Denied - Unable to Save Draft Videos

## 解决手机屏幕出现权限拒绝تلميح的问题

### 问题الوصف

عندما运行post脚本时，手机屏幕可能会出现فيما يليتلميح：

```text
Permission denied. Unable to save draft videos.
```

### 解决方案

这个问题نعم由于TikTok应用缺少存储权限导致的。请按照فيما يلي步骤解决：

#### 步骤1：手动点击النشر按钮

1. 在手机屏幕上找到并点击**النشر**按钮
2. 继续进行النشر流程

#### 步骤2：允许系统权限

1. عندما系统弹出权限申请对话框时，点击**允许**
2. 确保授予TikTok应用所需的存储权限

### ملاحظة مهمة

:::info
这个权限设置步骤只需要执行一次。一旦授予权限，后续的post脚本运行将不会再出现此问题。
:::

### 预防措施

为了避免此问题，建议在首次استخدامTikMatrix之前：

1. 手动打开TikTok应用
2. 尝试录制أو保存一个فيديو草稿
3. 在权限申请时选择**允许**
4. 这样可以预先التكوين好所需的权限
