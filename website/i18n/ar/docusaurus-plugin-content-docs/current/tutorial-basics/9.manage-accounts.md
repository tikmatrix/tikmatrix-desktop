# إدارة الحسابات

يمكن لإدارة الحسابات الحفاظ على اتساق كل هوية TikTok / Instagram مع جهاز التشغيل الخاص بها وبيانات اعتماد تسجيل الدخول والبيانات التي تعتمد عليها الأتمتة. تتيح لك قائمة الحسابات النظيفة تخصيص الأجهزة ومراقبة حالة الحساب والتحضير للأنشطة دون الحاجة إلى تشغيل الهواتف واحدًا تلو الآخر.

## إنشاء حساب يدويًا

1. افتح **الحسابات** في التنقل الأيسر.
2. انقر على **إضافة** لفتح نافذة التحرير المنبثقة.
3. املأ الحقول الأساسية بالتسلسل:
   - **البريد الإلكتروني**: البريد الإلكتروني المستخدم لتسجيل الدخول إلى التطبيق. سيحاول TikMatrix/IgMatrix أولاً تسجيل الدخول باستخدام البريد الإلكتروني + كلمة المرور؛ إذا كان البريد الإلكتروني فارغًا، فسيتحول تلقائيًا إلى اسم المستخدم + كلمة المرور.
   - **كلمة المرور**: كلمة المرور المستخدمة لتسجيل الدخول إلى التطبيق. يدعم IgMatrix إلحاق مفتاح 2FA بكلمة المرور، راجع التنسيق أدناه.
   - **اسم المستخدم**: اسم الحساب المعروض في TikTok/Instagram، هذا هو الحقل الإلزامي الوحيد.
   - **حالة تسجيل الدخول**: تحديد ما إذا كان قد تم تسجيل الدخول بالفعل على الجهاز الحالي (`1` = تم تسجيل الدخول، `0` = لم يتم تسجيل الدخول).
   - **الحالة**: التحكم في ما إذا كان الحساب يشارك في المهام (`0` = مفعل، `1` = معطل).
4. حدد علامة جهاز لربط الحساب بهاتف أو هاتف سحابي. إذا لم يتم التحديد، يظل الحساب غير مخصص.
5. انقر على **حفظ**، وستتحدث القائمة تلقائيًا.

## تسجيل الدخول

- إذا تم ملء **البريد الإلكتروني** و **كلمة المرور** للحساب، فسيحاول TikMatrix/IgMatrix استخدام هذه المعلومات لتسجيل الدخول إلى تطبيق TikTok/Instagram.
- إذا كان **البريد الإلكتروني** فارغًا، فسيحاول تسجيل الدخول باستخدام **اسم المستخدم** و **كلمة المرور**.
- يجب تحديد رقم تسلسل الجهاز حتى ينفذ السكريبت عملية تسجيل الدخول على الجهاز المقابل.
- سيتعرف سكريبت تسجيل الدخول تلقائيًا على حالة تسجيل الدخول للحساب، ويتخطى الحسابات التي تم تسجيل دخولها بالفعل.

## دعم 2FA (IgMatrix فقط)

يدعم IgMatrix تسجيل الدخول التلقائي بالمصادقة الثنائية (2FA):

### تنسيق كلمة مرور 2FA

لاستخدام 2FA، يرجى تعيين كلمة المرور بالتنسيق التالي:

```
كلمة_المرور_الخاصة_بك:مفتاح_2FA_الخاص_بك
```

**مثال:**

```
mypassword123:JBSWY3DPEHPK3PXP
```

## استيراد وتصدير الحسابات

- سيقوم **التصدير** بكتابة جميع الحسابات في `accounts.xlsx` إلى دليل التنزيل الخاص بـ TikMatrix/IgMatrix، ويمكن استخدامه كنسخة احتياطية أو قالب تحرير دفعي.
- يدعم **الاستيراد** إنشاء أو تحديث الحسابات بشكل جماعي من ملف Excel. إذا لم تكن متأكدًا من رؤوس الجدول، يرجى النقر على **تصدير** أولاً، ثم تحرير الملف المنزل واستيراده.
- عند الاستيراد، سيتم تحديث الصفوف التي تحتوي على `id`، وسيتم إضافة صفوف جديدة للصفوف التي لا تحتوي على `id`.

> يتطلب تسجيل الدخول IgMatrix 2FA فقط ملء `username` و `password`، والتنسيق هو `كلمة_المرور_الخاصة_بك:مفتاح_2FA_الخاص_بك`، `device_index` رقم تسلسل الجهاز.

### مقارنة حقول Excel

| الحقل | إلزامي | الوصف |
| --- | --- | --- |
| `id` | لا | المعرف الفريد الداخلي، يُحفظ فقط عند الحاجة إلى تحديث حساب موجود. |
| `email` | لا | البريد الإلكتروني لتسجيل الدخول إلى التطبيق، مفيد للاتصال والاسترداد. |
| `pwd` | لا | كلمة مرور تسجيل الدخول، يمكن استخدامها للرجوع السريع أو تسجيل الدخول التلقائي. |
| `username` | نعم | اسم المستخدم في TikTok/Instagram، يجب أن يكون فريدًا داخل مساحة العمل. |
| `device` | لا | الرقم التسلسلي للجهاز المربوط أو معرف الهاتف السحابي. |
| `device_index` | لا | الاسم المستعار للجهاز المعروض في الواجهة، يُستخدم بشكل أساسي للرجوع. |
| `logined` | لا | علامة الجلسة (`1` = تم تسجيل الدخول، `0` = لم يتم تسجيل الدخول). |
| `status` | لا | علامة التوفر (`0` = مفعل، `1` = معطل). لن تشارك الحسابات المعطلة في مهام السكريبت. |
| `nickname` | لا | أحدث لقب تم الحصول عليه من خلال مزامنة بيانات TikTok. |
| `country` | لا | البلد/المنطقة المستنتجة من ملف TikTok الشخصي. |
| `followers` | لا | عدد المتابعين المتزامن. |
| `following` | لا | عدد الأشخاص الذين يتابعهم الحساب. |
| `hearts` | لا | إجمالي عدد الإعجابات التي حصل عليها الحساب. |
| `videos` | لا | عدد مقاطع الفيديو المنشورة. |
| `friends` | لا | عدد الأصدقاء المتابعين المتبادلين (إذا كان متاحًا). |

### الفروقات في الحقول بين TikMatrix و IgMatrix

- سيملأ **TikMatrix** حقول الإحصائيات (`nickname`، `country`، `followers`، `following`، `hearts`، `videos`، `friends`) بعد تنفيذ **مزامنة بيانات TikTok** أو النقر على زر استعلام البيانات.
- يستخدم **IgMatrix** فقط الحقول الأساسية (`email`، `pwd`، `username`، `device`، `logined`، `status`)، ويتجاهل حقول الإحصائيات.

## سلوكيات أخرى

- لن تشارك الحسابات المحددة كـ **معطلة** في أي سكريبتات أتمتة أو مهام مجدولة أو عمليات دفعية. يمكن متابعة تنفيذ المهام بعد إعادة التفعيل.
- سيقوم سكريبت **مطابقة الحسابات** بفحص جلسات التطبيق التي تم تسجيل دخولها على الجهاز، وإنشاء إدخالات مقابلة تلقائيًا في قائمة الحسابات وربط الأجهزة المكتشفة، وهي طريقة للوصول السريع إلى الهواتف المُعدة يدويًا.
