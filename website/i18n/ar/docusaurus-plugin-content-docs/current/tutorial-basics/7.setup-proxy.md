# إعداد البروكسي

يمكنك إعداد عنوان IP بروكسي مستقل لكل هاتف.

> **إخلاء المسؤولية:** جميع المواقع الموصى بها في هذه الصفحة مصدرها الإنترنت ولا علاقة لها بهذا الموقع. يرجى التحقق من صلاحيتها بنفسك قبل الاستخدام. نحن لا نقدم أي ضمانات.

## 1. إعداد بروكسي VPS السكني (إذا اشتريت بروكسي IP، يمكنك تخطي هذه الخطوة)

1. قم بشراء VPS بروكسي IP سكني من الدولة المستهدفة (يفضل IP أصلي و VPS مزدوج ISP).
2. استخدم [https://github.com/MHSanaei/3x-ui](https://github.com/MHSanaei/3x-ui) لإعداد VPN بسرعة على VPS:

    ```shell
    bash <(curl -Ls https://raw.githubusercontent.com/MHSanaei/3x-ui/refs/tags/v2.6.0/install.sh)
    # اتبع المطالبات لإكمال إعداد VPN.
    ```

3. بعد اكتمال التثبيت، قم بالوصول إلى واجهة الويب x-ui. يمكنك العثور على عنوان URL وبيانات اعتماد تسجيل الدخول في مخرجات الطرفية بعد التثبيت.
4. أنشئ مستخدمًا جديدًا في x-ui، واضبط البروتوكول على `SOCKS5`، وقم بتكوين المنفذ والإعدادات الأخرى حسب الحاجة.
5. بعد إنشاء المستخدم، سترى IP البروكسي والمنفذ في واجهة x-ui.
6. استخدم IP البروكسي والمنفذ في تطبيق NekoBox على الهاتف.

> مزودو VPS الموصى بهم

- [انقر هنا](/advertising)

## 2. شراء بروكسي IP سكني (إذا قمت بإعداد VPS، يمكنك تخطي هذه الخطوة)

يمكنك أيضًا شراء بروكسي IP سكني مباشرة دون الحاجة إلى إعداد VPS.

> مزودو البروكسي الموصى بهم

- [انقر هنا](/advertising)

> أدوات اختبار IP

- [ipinfo](https://ipinfo.io) — تحقق مما إذا كان IP سكنيًا.
- [scamalytics](https://scamalytics.com/ip) — تحقق من مستوى مخاطر IP.

## 3. تفعيل البروكسي على الهاتف

1. قم بتنزيل وتثبيت عميل [NekoBoxForAndroid](https://github.com/MatsuriDayo/NekoBoxForAndroid) APK على الهاتف.
2. افتح تطبيق NekoBox، وانقر على زر `+` لإضافة بروكسي جديد.
3. حدد `SOCKS5` أو نوع بروكسي آخر.
4. أدخل IP البروكسي والمنفذ، ثم انقر على `حفظ`.
5. انقر على زر `البدء` لتفعيل البروكسي.
6. بعد تفعيل البروكسي، يمكنك التحقق من حالة الاتصال في تطبيق NekoBox.

### ملاحظة للمستخدمين الصينيين

إذا كنت في الصين، فقد لا يعمل بروتوكول `SOCKS5` بشكل صحيح. تحتاج إلى إضافة بروكسي مشفر (`VLESS`) كبروكسي أمامي. يمكنك استخدام الأدوات التالية:

1. أضف بروكسي `VLESS` في تطبيق NekoBox.
2. أضف بروكسي `SOCKS5` في تطبيق NekoBox.
3. أضف بروكسي `سلسلة` في تطبيق NekoBox. حدد بروكسي `VLESS` كبروكسي أمامي، وبروكسي `SOCKS5` كبروكسي خلفي.
