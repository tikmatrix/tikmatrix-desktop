# التعليق الجماعي

:::warning 下线تلميح
该功能已整合到 [التسويق الفائق脚本](./14.super-marketing.md)。请优先استخدام“التسويق الفائق”脚本以获得更强大、更灵活的التعليق الجماعي与内容互动能力。
:::

التعليق الجماعي用于对大量 TikTok 帖子النشر互动评论（如养号、推广、活动运营）。系统支持预生成模板与实时生成两种消息来源，并会尽量模拟人工发送节奏以降低被检测风险。

## 在“التسويق الفائق”脚本中استخدامالتعليق الجماعي

1. 打开脚本：选择“التسويق الفائق”。
2. 选择数据源النوع为“帖子链接列表”。
3. 批量粘贴帖子链接أو从文本文件导入（每行一个）；导入后اختياري择“追加”أو“覆盖”。
4. 选择数据استخدام策略：

   - 共享池策略：所有 bot 账号对相同帖子进行评论。

   - 单次消耗策略：每个 bot 账号只对不同的帖子进行一次评论，确保每个帖子仅被一个账号处理一次。

5. 设置每个帖子的评论次数（通常为 1 次以避免重复评论）。
6. 在“评论设置”中التكوينفيما يليالخيارات：

   - 预生成评论模板：可粘贴多行أو多段预生成评论文案（每行أو每段视为一条候选评论）。

   - 文案استخدام顺序：选择“随机”أو“按顺序”。

   - 随机：每次从候选文案中随机抽取一条。

   - 按顺序：按照粘贴/导入顺序轮换استخدام（适合需要可复现的去重策略）。

   - 插入表情（emoji）：可开启自动在评论中插入表情以提高自然度。

   - 插入设备编号（اختياري）：在评论末尾加入设备编号以便追踪来源（仅用于调试/审计，استخدام时请谨慎）。

   - 实时生成评论（اختياري，ChatGPT API）：

      - 启用后，脚本可调用 ChatGPT 类 API，根据帖文标题/الوصفأو目标上下文实时生成上下文相关且唯一的评论文案，适合需要大量唯一评论的场景。

      - ملاحظة：启用实时生成需提供有效 API Key，并ملاحظة速率限制、费用与隐私合规（避免发送敏感معلومات到第三方服务）。

   - 保证唯一评论的两种常见方式：

      1. استخدام“按顺序”轮换预生成评论模板（每条模板只استخدام一次直到耗尽）。

      2. استخدام ChatGPT 实时生成每条评论（基于帖文内容动态生成，唯一性更高）。

7. التكوين其他脚本المعاملات（并发数、任务间隔、账号池等），确认无误后启动脚本。

:::tip 关于任务间隔
建议设置最小/最大间隔并启用随机间隔，以避免所有设备同时执行导致异常峰值，从而降低被平台检测أو限制的风险。
:::

## استخدام建议与ملاحظات

- 先在小批量帖子上测试你的评论策略，确认评论质量与平台处理情况。
- 评论内容应合规且多样化，避免短时间内发送大量相同内容。
- 如果استخدام实时生成，请严格监控 API 调用量与费用，并确保不向外泄露敏感معلومات。
- 任务可能失败；遇到失败请重试أو查看日志，系统会自动跳过已完成的目标。
