# 提升用户

:::warning 下线تلميح
该功能已整合到 [التسويق الفائق脚本](./14.super-marketing.md)。请优先استخدام“التسويق الفائق”以获得更强大、更灵活的تعزيز المستخدمين能力。
:::

提升用户功能通过批量关注/取关操作来提升目标用户的曝光与互动。系统会尽量模拟真实用户行为以降低被检测风险，但仍建议遵循平台规则并控制频率。

## 在“التسويق الفائق”脚本中استخدام提升用户功能

1. 打开脚本：选择“التسويق الفائق”。
2. 选择数据源النوع为“用户账号列表”。
3. 批量粘贴用户名أو从文本文件导入（每行一个）；导入后اختياري择“追加”أو“覆盖”。
4. 选择数据استخدام策略：

    - 共享池策略：所有 bot 账号会对相同目标进行关注/取关。

    - 单次消耗策略：每个 bot 账号仅处理不同的目标，确保每个目标只被一个账号操作一次。

5. 在“用户操作 — 重复次数”中设置每个目标的操作次数（通常设置为 1 次）。
6. 选择操作النوع：关注 أو 取消关注。可在“用户操作”中进一步التكوين：

        - 进入用户主页方式：选择“بحث用户”أو“直接打开主页（Direct Open Profile）”。

        - 发送私信（اختياري）：启用后可粘贴预生成的多行私信文案，每行أو按段为一条候选文案；التكوين文案استخدام顺序：

                - 随机：每次استخدام随机选一条文案。
                - 按顺序：按粘贴/导入的顺序依次استخدام（可跨任务轮次继续）。

            另اختياري择نعملا在文案中自动插入表情（emoji）以提高自然度。

7. التكوين其他المعاملات（并发、延时、账号池），确认无误后启动脚本。

## استخدام建议与ملاحظات

- 建议先在少量账号上测试，观察策略与المعاملاتنعملا触发平台风控。
- 合理设置延时与并发，模拟人工行为以降低风险。
- 长期运行时请定期检查账号健康、任务日志，并根据结果调整策略。
- 提升用户操作并非 100% 成功，遇到失败请重试أو排查目标账号状态。
