# تسجيل الدخول

تسجيل الدخوليستخدم البرنامج النصي لـتسجيل الدخول社交媒体账户。

## 步骤

1. 确保账户已添加到账户列表中（点击`账户`添加账户）, 并选择你需要تسجيل الدخول该账户的手机序号。
2. 单选أو者多选要运行تسجيل الدخول脚本的设备。
3. 点击`脚本` > `登录`。
4. 点击`开始脚本`开始执行。

## تسجيل الدخول优先级

系统استخدامفيما يليتسجيل الدخول优先级：

1. **邮箱+密码**：如果账户التكوين了邮箱，将优先استخدام邮箱进行تسجيل الدخول。
2. **用户名+密码**：如果没有التكوين邮箱，则استخدام用户名进行تسجيل الدخول。

## 2FA 支持（仅限 IgMatrix）

IgMatrix 支持自动双因子认证（2FA）تسجيل الدخول：

### 2FA 密码格式

要استخدام 2FA，请按فيما يلي格式设置密码：

```plaintext
你的密码:你的2FA密钥
```

**أمثلة：**

```plaintext
mypassword123:JBSWY3DPEHPK3PXP
```

### 2FA 工作原理

1. **密码解析**：系统自动检测密码中نعملا包含冒号（`:`）分隔符。
2. **验证码生成**：عندما需要 2FA 验证时，系统استخدام您的密钥向我们的 2FA 服务请求 TOTP 验证码。
3. **自动输入**：生成的 6 位数字验证码会自动输入到 Instagram 的 2FA 验证框中。
4. **无缝تسجيل الدخول**：整个 2FA 过程无需手动干预，自动完成。

### 2FA 密钥الوصف

2FA 密钥通常نعم一个 32 位的 Base32 编码字符串，您在 Instagram 中设置 2FA 时会获得此密钥。这与 Google Authenticator أو Authy 等验证器应用استخدام的密钥相同。

### 获取支持2FA的Instagram账号

您可以从فيما يلي网站购买支持2FA的Instagram账号：[AccsMarket](https://accsmarket.com/en/?ref=802647)

## ملاحظات

- TikTok تسجيل الدخول不نعم完全100%自动的。你可能需要手动输入邮箱验证码أو完成其他验证步骤。我们也无法解决 TikTok 的تسجيل الدخول挑战（如صور验证码、人机验证等）以及错误تلميح: تسجيل الدخول太频繁的问题。
- Instagram 可以通过 2FA تسجيل الدخول实现100%全自动تسجيل الدخول。对于启用了 2FA 的 Instagram 账户，请确保在密码中استخدام正确的密钥格式。
