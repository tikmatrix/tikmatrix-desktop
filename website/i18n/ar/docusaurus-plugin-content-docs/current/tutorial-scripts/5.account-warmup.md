# تسخين الحساب

يُستخدم سكريبت تسخين الحساب لتسخين الحساب من خلال البحث عن الكلمات الرئيسية ومشاهدة مقاطع الفيديو والمتابعة والإعجاب والتعليق والحفظ.

## الخطوات

1. اختر الأجهزة لتشغيل السكريبت.
2. انقر على `السكريبتات` > `تسخين الحساب`.
3. قم بتكوين إعدادات المهمة:
   - **مدة المشاهدة**: مدة مشاهدة كل فيديو.
   - **الكلمات الرئيسية للبحث**: موضوع البحث عن الفيديو.
   - **مدة المهمة**: المدة الإجمالية لمهمة التسخين.
   - **محتوى التعليق**: استخدم تعليقات جاهزة أو تعليقات ChatGPT المُنشأة من عنوان المنشور.
   - **احتمالية التفاعل**: احتمالية المتابعة والإعجاب والتعليق والحفظ.
4. انقر على `بدء السكريبت` لبدء التنفيذ.

## ملاحظات

- قد لا ينجح سكريبت التسخين دائمًا؛ أعد المحاولة للمهام الفاشلة إذا لزم الأمر.
- يدعم الجدولة الزمنية، يمكنك تعيين مهام التسخين للتشغيل في وقت محدد.
- يدعم تخطي الإعلانات والبث المباشر ومقاطع الفيديو التي تم الإعجاب بها بالفعل.
