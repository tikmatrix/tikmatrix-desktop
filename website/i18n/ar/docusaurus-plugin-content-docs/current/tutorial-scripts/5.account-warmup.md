# تدفئة الحساب

تدفئة الحسابيستخدم البرنامج النصي لـ通过بحث关键词、观看فيديو、关注、点赞、评论和收藏来预热账号。

## 步骤

1. 选择要运行脚本的设备。
2. 点击`脚本` > `账号预热`。
3. التكوينإعدادات المهام：
   - **观看时长**：观看每个فيديو的时长。
   - **بحث关键词**：بحثفيديو的话题。
   - **任务时长**：预热任务的总时长。
   - **评论内容**：استخدام预制评论أو从帖子标题生成的ChatGPT评论。
   - **互动概率**：关注、点赞、评论和收藏的概率。
4. 点击`开始脚本`开始执行。

## ملاحظات

- 预热脚本可能不总نعم成功；如需要请重试失败的任务。
- 支持定时调度，你可以设置预热任务在指定时间运行。
- 支持跳过广告、直播和已点赞的فيديو。
