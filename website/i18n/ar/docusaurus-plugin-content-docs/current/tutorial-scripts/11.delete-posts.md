# حذف المنشورات

يُستخدم سكريبت حذف المنشورات لحذف المنشورات بناءً على عدد المشاهدات.

## الخطوات

1. اختر الأجهزة لتشغيل السكريبت.
2. انقر على `السكريبتات` > `حذف المنشورات`.
3. قم بتكوين إعدادات المهمة:
    - **الحد الأقصى للمشاهدات**: أدخل الحد الأقصى لعدد المشاهدات للمنشورات التي تريد حذفها.
4. انقر على `بدء السكريبت` لبدء التنفيذ.

## ملاحظة

- قد لا ينجح سكريبت حذف المنشورات دائمًا؛ يرجى إعادة المحاولة للمهام الفاشلة إذا لزم الأمر.
