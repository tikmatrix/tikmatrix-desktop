# حذف المنشورات

حذف المنشوراتيستخدم البرنامج النصي لـ根据浏览量حذف المنشورات。

## 步骤

1. 选择要运行脚本的设备。
2. 点击 `脚本` > `删除帖子`。
3. التكوينإعدادات المهام：
    - **最大浏览量**：输入您想要删除的帖子的最大浏览量。
4. 点击 `开始脚本` 开始执行。

## ملاحظة

- حذف المنشورات脚本可能并不总نعم成功；如有需要请重试失败的任务。
