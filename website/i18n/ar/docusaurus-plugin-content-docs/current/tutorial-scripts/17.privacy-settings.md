# إعدادات الخصوصية

**إعدادات الخصوصية**يستخدم البرنامج النصي لـ根据观看数过滤条件批量更新 TikTok 帖子的إعدادات الخصوصية。

:::info
此脚本仅适用于 TikTok 账号。
:::

## 步骤

1. 选择要运行脚本的设备。
2. 点击 `脚本` > `隐私设置`。
3. التكوينإعدادات المهام：
    - **帖子过滤المعاملات**：
      - **最大观看数**：输入最大观看数以过滤帖子。只有观看数 ≤ 此数字的帖子会被更新。
      - **最大处理帖子数**：输入要处理的最大帖子数。设置为 0 表示不限制处理数量。
    - **إعدادات الخصوصيةالمعاملات**：
      - **谁可以观看此فيديو**：选择一个الخيارات：
        - `所有人`：任何人都可以观看فيديو
        - `朋友`：只有朋友可以观看فيديو
        - `仅自己`：只有你自己可以观看فيديو
    - **开关المعاملات**：
      - **允许评论**：启用أو禁用帖子评论。
      - **允许内容重用**：启用أو禁用内容重用，包括合拍、缝合、贴纸和添加到故事。
4. 点击 `开始脚本` 开始执行。

## 工作原理

脚本遵循فيما يلي工作流程：

1. 进入 Videos 标签页
2. 扫描帖子，解析浏览量
3. 找到符合条件的帖子（views ≤ max_views）
4. 进入帖子详情页
5. 点击 "Privacy settings" 按钮
6. 等待设置弹窗出现
7. 根据你的选择设置观看权限
8. 智能切换 "Allow comments" 开关
9. 智能切换 "Allow reuse" 开关
10. العودة列表，继续处理下一个帖子
11. 重复以上步骤，直到处理完所有匹配的帖子

## ملاحظات

- إعدادات الخصوصية脚本可能不总نعم成功；如需要请重试失败的任务。
- 脚本仅处理符合观看数过滤条件的帖子。
- 更改会立即应用，且无法通过脚本撤销。
- 在对多个帖子运行脚本之前，请确保检查你的إعدادات الخصوصية。
