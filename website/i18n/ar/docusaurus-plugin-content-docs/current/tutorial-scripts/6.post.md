# النشر帖子

النشر帖子يُستخدم البرنامج النصي لمتابعة المستخدمين تلقائيًا علىTikTok上النشرفيديوأوصور。

## 步骤

1. **准备素材**：
   - الخيارات1：点击`素材库`按钮上传فيديو/صور到مكتبة المواد。
   - الخيارات2：استخداممجلد محلي作为مصدر المواد，在تكوين البرنامج النصي中设置文件夹路径。
2. 选择要运行脚本的设备。
3. 点击`脚本` > `发布帖子`。
4. التكوينإعدادات المهام：
    - **مصدر المواد**：选择مكتبة الموادأومجلد محلي。
    - **مسار المجلد المحلي**：如果استخداممجلد محلي，指定包含فيديو/صور的路径。
    - **طريقة النشر**：选择طريقة النشر：系统分享أو添加按钮(`+`)أو通过بحثاستخدام声音。
    - **声音名称**：عندمااستخدام"通过بحثاستخدام声音"طريقة النشر时，你可以选择：
      - 输入声音名称进行بحث（على سبيل المثال："original sound"，"trending music"）
      - 直接输入TikTok声音链接（على سبيل المثال：`https://www.tiktok.com/t/ZP8hvBwCb/`）
    - **نوع المحتوى**：选择فيديوأوصور。
    - **添加声音开关**：启用/禁用添加声音。
    - **上传等待时间**：设置上传后的延迟时间（5-60秒），确保TikTok服务器完全处理内容后再继续操作。
    - **添加产品链接开关**：启用/禁用添加产品链接。
    - **Caption 格式**：选择两种 caption 格式之一：
      - **多行模式**：每行نعم一条独立 caption，系统按顺序循环استخدام。
      - **Spintax 模式**：استخدام `{选项1|选项2}` 语法生成随机文本。
    - **标题**：根据所选格式输入 caption。支持提及用户（如 @TikMatrixPhoneFarm）和标签（如 #tag1 #tag2 #tag3）。详细格式الوصف见下文。
    - **素材标签**：按标签过滤素材。如果未设置，استخدام所有素材。
5. 点击`开始脚本`开始执行。

## Caption 格式الوصف

系统支持两种 caption 格式，让您灵活控制帖子标题。

### 多行模式（默认）

**工作原理：**
- 每一行代表一条独立 caption
- 系统按顺序استخدام caption 并循环
- 在一行内استخدام `###` 添加换行符

**أمثلة：**

```
第一条caption###支持换行
第二条caption###也支持换行
第三条caption不换行
```

**平台差异：**
- **TikTok**：استخدام `##` 分隔标题和الوصف（على سبيل المثال：`标题##长描述`）
- **Instagram**：不支持 `##`，会显示تحذير

### Spintax 模式

**工作原理：**
- 只有一个 caption 模板
- استخدام `{选项1|选项2|选项3}` 语法实现随机变化
- استخدام**原生换行符**（按回车键）实现多行 caption
- 不要استخدام `###` - 它不会被解析

**أمثلة：**

```
{今天|明天|后天}的天气真{好|不错|棒}
欢迎关注我的账号##想了解更多请点击链接
```

**平台差异：**
- **TikTok**：`##` 可用于分隔标题和الوصف
- **Instagram**：不支持 `##`

**ملاحظة：** 在 Spintax 模式下，如果检测到 `###`，系统会تلميحاستخدام原生换行符。

## 素材管理

### 音乐استخدام模式

عندما**طريقة النشر**设置为"通过بحثاستخدام声音"时，脚本支持两种不同的音乐添加模式：

#### 1. 声音بحث模式

输入声音名称在TikTok内بحث音乐：

- أمثلة：`original sound`，`trending music`，`viral beat`
- 脚本会بحثTikTok的音乐库并选择第一个匹配结果
- 这种模式适用于استخدام热门أو流行声音

#### 2. 声音链接模式

输入直接的TikTok声音链接استخدام特定音乐：

- 支持的链接格式：
  - 短链接：`https://www.tiktok.com/t/ZP8hvBwCb/`
  - 音乐页面：`https://www.tiktok.com/music/original-sound-7473897776619834646`
  - VM链接：`https://vm.tiktok.com/ZMhvBwCbD/`
- 脚本自动检测以`https://`开头的链接并直接打开
- 这种模式适用于استخدام来自特定فيديو的特定声音

### مكتبة المواد

- **素材标签**：为素材添加标签，在النشر时进行过滤。如果未指定标签，استخدام所有素材。
- **素材标题**：你可以为每个素材设置自定义标题。如果设置了，它优先于النشر设置。
- **素材用户名**：你可以为每个素材设置自定义用户名。匹配用户名的素材优先استخدام。
- **素材排序**：为素材设置排序号（对多صور帖子有用；数字越小越靠前）。

### مجلد محلي来源

- **文件夹结构**：

  ```text
  /素材路径/
  ├── caption.txt (可选 - 根目录兜底)
  ├── sound_names.txt (可选 - 根目录兜底)  
  ├── sound_urls.txt (可选 - 根目录兜底)
  ├── @用户名1/
  │   ├── video1.mp4
  │   ├── video2.mp4
  │   ├── image1.jpg
  │   ├── image2.jpg
  │   ├── caption.txt (可选)
  │   ├── sound_names.txt (可选)
  │   └── sound_urls.txt (可选)
  ├── @用户名2/
  │   ├── content1.mp4
  │   ├── content2.jpg
  │   ├── caption.txt (可选)
  │   ├── sound_names.txt (可选)
  │   └── sound_urls.txt (可选)
  └── ...
  ```

- **文件夹路径**：指定包含用户名子文件夹的根路径。
- **用户名子文件夹**：每个用户名必须有自己的子文件夹（على سبيل المثال，`@用户名1`，`@用户名2`）。
- **根文件夹兜底**：如果用户名文件夹不存在أو用户名为空，将从根文件夹读取素材和设置。
- **支持格式**：
  - فيديو：MP4
  - صور：JPG, PNG
- **标题文件**：在每个用户名文件夹中اختياري的`caption.txt`文件，用于自定义标题。如果找不到，将回退到根文件夹的`caption.txt`。
  - **格式الوصف**：Caption.txt 仅支持 Spintax 模板格式（与 UI 的 Spintax 模式الاستخدام相同）：
    - التكوين一个 Spintax 模板，استخدام `{选项1|选项2}` 语法实现随机变化
    - استخدام原生换行符（回车键）实现多行
    - 支持 `##` 标题/الوصف分隔（仅 TikTok，Instagram 不支持）
  - **ملاحظة**：Caption.txt 不支持 Multiline 顺序选择模式
- **音乐文件**：
  - **sound_names.txt**：包含用于بحث的音乐名称，每行一个。按顺序读取，到末尾后循环读取。
  - **sound_urls.txt**：包含TikTok音乐链接，每行一个。按顺序读取，到末尾后循环读取。
  - 如果用户名文件夹不存在，从根文件夹的音乐文件读取。
- **文件选择**：
  - 对于فيديو：每次النشر随机选择一个فيديو
  - 对于صور：根据صور数量设置选择多张صور
- **استخدام跟踪**：文件会被自动标记为已استخدام，避免重复。

### Caption.txt 文件格式الوصف

عندمااستخدام**مجلد محلي来源**时，`caption.txt` 文件用于التكوين Spintax 模板。此文件可以覆盖 UI 中的标题التكوين。

**مهمالوصف**：Caption.txt 仅支持 Spintax 模板格式，不支持 Multiline 顺序选择模式。

#### Spintax 模板格式

التكوين一个 Spintax 模板，استخدام `{选项1|选项2}` 语法生成随机文本变化。استخدام原生换行符（按回车键）实现多行：

```text
{早上好|你好|嗨}！看看这个{令人惊叹的|不可思议的|很棒的}视频！
{关注获取更多|点赞分享|标记你的朋友}
```

可能的输出：

- "早上好！看看这个令人惊叹的فيديو！"
- "你好！看看这个不可思议的فيديو！"
- "嗨！看看这个很棒的فيديو！"

**复杂 Spintax 变化：**

```text
{🔥|✨|🎉} {不要错过|查看|观看}这个{令人惊叹的|不可思议的|很棒的}{视频|内容|帖子}！
{关注获取更多|点赞分享|标记你的朋友} {#热门|#病毒式|#fyp} #tiktok
```

**Spintax 与 TikTok 标题/الوصف分隔：**

```text
{激动人心的|令人惊叹的|不可思议的}{新闻|更新|发现}##这是您需要了解的关于这个{精彩|美妙|非凡}{发展|更新|故事}的内容！
```

#### 完整 Caption.txt أمثلة

一个完整的 Spintax 模板أمثلة：

```text
{早晨动力|每日灵感|开始你的一天}！💪
{喜欢|享受|迷恋}这个{美丽的|令人惊叹的|华丽的}景色！🌅
{不敢相信|对此很兴奋|对此很激动}{这一刻|这次经历|这次冒险}！✨
```

**带 TikTok 标题/الوصف分隔的أمثلة：**

```text
{精彩|有趣|好玩}的视频内容##这是一段{令人惊叹|不可思议|很棒}的{视频|内容|作品}，欢迎{点赞|关注|转发}！
```

  **Sound_names.txt 文件أمثلة：**

  ```text
  原声
  热门音乐
  病毒式节拍
  流行歌曲
  舞蹈音乐
  ```

  **Sound_urls.txt 文件أمثلة：**

  ```text
  https://www.tiktok.com/t/ZP8hvBwCb/
  https://www.tiktok.com/music/original-sound-7473897776619834646
  https://vm.tiktok.com/ZMhvBwCbD/
  https://www.tiktok.com/t/ZP8hvBwCb/
  ```

## 常见问题

1. 为什么任务会失败？
   - 自动化脚本نعم根据TikTok/Instagram应用的行为进行模拟操作的，某些情况下可能会失败，على سبيل المثال网络问题、应用响应异常、意外的弹窗、设备性能问题、设备兼容性问题等。
   - 如果多次重试仍然失败，提交工单获取帮助。
2. 怎么样实现无人值守运行？
   - 创建设备分组, 并在分组的Post设置中استخدام定时调度功能，软件会每天在指定时间自动创建并运行النشر任务。
3. 怎么实现幻灯片帖子的排序？
   - 在مكتبة المواد中为每个صور素材设置排序号，数字越小越靠前。النشر时脚本会根据排序号选择صور。
   - 你也可以批量导入素材时استخدام符合下面规范的文件名来设置排序号：
     - `sort_1_图片描述.png`
     - `sort_2_图片描述.png`
     - `sort_3_图片描述.png`
4. 怎么为每个账号استخدام不同的素材,标题和音乐？
   - استخداممجلد محلي时，为每个用户名创建子文件夹，在子文件夹中创建`caption.txt`，`sound_names.txt`أو`sound_urls.txt`文件来定义自定义标题和音乐。脚本会优先استخدام匹配用户名的文件。
   - استخداممكتبة المواد时，为每个素材设置用户名字段,标题字段,脚本会优先استخدام匹配用户名的素材和标题。
5. 支持哪些素材格式？
   - فيديو：MP4
   - صور：JPG,PNG

6. صور和فيديو素材可以重复استخدام吗？

  不可以。为保护您的账号，系统会在素材النشر后自动将其标记为已استخدام，避免被重复选用。这样可以减少重复النشر的风险，降低账号被风控أو封禁的可能性。
