# Gestion des comptes

La gestion des comptes vous permet de maintenir la cohérence entre chaque identité TikTok / Instagram, son appareil d'exécution, ses identifiants de connexion et les données dont dépendent les automatisations. Une liste de comptes bien organisée vous permet d'attribuer des appareils, de surveiller les statuts des comptes et de préparer les activités, sans avoir à manipuler chaque téléphone individuellement.

## Créer un compte manuellement

1. Ouvrez **Comptes** dans la navigation de gauche.
2. Cliquez sur **Nouveau** pour ouvrir la fenêtre d'édition.
3. Remplissez les champs de base dans l'ordre :
   - **Email** : L'email utilisé pour se connecter à l'application. TikMatrix/IgMatrix essaiera d'abord de se connecter avec email + mot de passe ; si l'email est vide, il basculera automatiquement vers nom d'utilisateur + mot de passe.
   - **Mot de passe** : Le mot de passe utilisé pour se connecter à l'application. IgMatrix prend en charge l'ajout d'une clé 2FA dans le mot de passe, voir le format ci-dessous.
   - **Nom d'utilisateur** : Le nom du compte affiché dans TikTok/Instagram, c'est le seul champ obligatoire.
   - **Statut de connexion** : Indique si le compte est déjà connecté sur l'appareil actuel (`1` = connecté, `0` = non connecté).
   - **Statut** : Contrôle si le compte participe aux tâches (`0` = activé, `1` = désactivé).
4. Sélectionnez une étiquette d'appareil pour lier le compte à un téléphone ou un téléphone cloud. Si aucune n'est sélectionnée, le compte restera non attribué.
5. Cliquez sur **Enregistrer**, la liste se rafraîchira automatiquement.

## Connexion

- Si l'**email** et le **mot de passe** du compte sont remplis, TikMatrix/IgMatrix tentera de se connecter à l'application TikTok/Instagram avec ces informations.
- Si l'**email** est vide, il tentera de se connecter avec le **nom d'utilisateur** et le **mot de passe**.
- Vous devez sélectionner un numéro d'appareil pour que le script exécute l'opération de connexion sur l'appareil correspondant.
- Le script de connexion reconnaîtra automatiquement le statut de connexion du compte et ignorera les comptes déjà connectés.

## Support 2FA (IgMatrix uniquement)

IgMatrix prend en charge la connexion automatique avec authentification à deux facteurs (2FA) :

### Format du mot de passe 2FA

Pour utiliser 2FA, configurez votre mot de passe au format suivant :

```
votremotdepasse:votrecle2FA
```

**Exemple :**

```
mypassword123:JBSWY3DPEHPK3PXP
```

## Importer et exporter des comptes

- **Exporter** enregistrera tous les comptes dans `accounts.xlsx` dans le répertoire de téléchargement de TikMatrix/IgMatrix, pouvant servir de sauvegarde ou de modèle d'édition par lots.
- **Importer** permet de créer ou de mettre à jour des comptes en masse à partir de ce fichier Excel. Si vous n'êtes pas sûr des en-têtes, cliquez d'abord sur **Exporter**, puis éditez le fichier téléchargé et importez-le.
- Lors de l'importation, les lignes avec un `id` mettront à jour les comptes existants, celles sans `id` créeront de nouveaux comptes.

> La connexion 2FA IgMatrix nécessite uniquement de remplir `username` et `password`, au format `votremotdepasse:votrecle2FA`, et `device_index` numéro d'appareil.

### Correspondance des champs Excel

| Champ | Obligatoire | Description |
| --- | --- | --- |
| `id` | Non | Identifiant unique interne, à conserver uniquement lors de la mise à jour de comptes existants. |
| `email` | Non | Email de connexion à l'application, utile pour le contact et la récupération. |
| `pwd` | Non | Mot de passe de connexion, utilisable pour référence rapide ou connexion automatique. |
| `username` | Oui | Nom d'utilisateur TikTok/Instagram, doit être unique dans l'espace de travail. |
| `device` | Non | Numéro de série de l'appareil lié ou identifiant du téléphone cloud. |
| `device_index` | Non | Alias de l'appareil affiché dans l'interface, principalement pour référence. |
| `logined` | Non | Marqueur de session (`1` = connecté, `0` = non connecté). |
| `status` | Non | Marqueur de disponibilité (`0` = activé, `1` = désactivé). Les comptes désactivés ne participeront pas aux tâches de script. |
| `nickname` | Non | Dernier surnom obtenu via la synchronisation des données TikTok. |
| `country` | Non | Pays/région déduit du profil TikTok. |
| `followers` | Non | Nombre d'abonnés synchronisé. |
| `following` | Non | Nombre de personnes suivies par le compte. |
| `hearts` | Non | Total des likes reçus par le compte. |
| `videos` | Non | Nombre de vidéos publiées. |
| `friends` | Non | Nombre d'amis mutuellement suivis (si disponible). |

### Différences de champs entre TikMatrix et IgMatrix

- **TikMatrix** remplit les champs statistiques (`nickname`, `country`, `followers`, `following`, `hearts`, `videos`, `friends`) après l'exécution de **Synchroniser les données TikTok** ou en cliquant sur le bouton de requête de données.
- **IgMatrix** utilise uniquement les champs de base (`email`, `pwd`, `username`, `device`, `logined`, `status`) et ignore les champs statistiques.

## Autres comportements

- Les comptes marqués comme **désactivés** ne participeront à aucun script d'automatisation, tâche planifiée ou opération en masse. Ils pourront continuer à exécuter des tâches après réactivation.
- Le script **Matcher les comptes** scannera les sessions d'application connectées sur les appareils, créera automatiquement les entrées correspondantes dans la liste des comptes et les liera aux appareils détectés. C'est un moyen rapide d'intégrer des téléphones préparés manuellement.
