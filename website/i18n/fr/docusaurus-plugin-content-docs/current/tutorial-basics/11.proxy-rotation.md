# Tutoriel de rotation de proxy simplifié

La rotation de proxy permet de changer automatiquement l'IP du téléphone avant le démarrage du script. Cela rend le changement de compte ou la connexion plus sûrs et peut réduire les risques de contrôle.

## Préparation préalable

- Vous avez besoin d'un service de proxy prenant en charge la fonctionnalité de "changement d'IP" ou "rotation d'IP".
- Demandez au fournisseur de service le lien de rotation (également appelé *Rotation URL*, *Refresh URL*, *Change IP URL*).
- Si le fournisseur de service nécessite un mot de passe ou un token supplémentaire, préparez-le à l'avance.

## Enregistrer le lien de rotation pour l'appareil

1. Ouvrez **Manage Devices** sur le bureau et passez à la vue tableau.
2. Trouvez l'appareil nécessitant un changement d'IP, cliquez sur **Configure** dans la colonne *Proxy rotation URL*.
3. Remplissez selon les instructions :
   - **Proxy rotation URL** : Collez le lien de rotation fourni par le fournisseur. TikMatrix l'accédera automatiquement lorsqu'un changement d'IP sera nécessaire.
   - **Request method** : Conservez `GET`, sauf si le fournisseur de service demande explicitement de passer à `POST`.
   - **Headers**, **Body** : Généralement laissés vides. Si le fournisseur de service a fourni un exemple, copiez-le tel quel dans la zone de saisie.
   - **Timeout** : Le temps d'attente de réponse du fournisseur, 10 secondes par défaut suffisent.
4. Cliquez sur **Save**. La configuration sera automatiquement enregistrée, aucune modification manuelle de fichier n'est nécessaire.

Après l'enregistrement, la ligne de l'appareil affichera le résultat de la dernière rotation : vert pour succès, gris pour non encore exécuté, rouge pour erreur retournée par le fournisseur.

## Vérifier si la rotation est effective

1. Dans la même ligne, cliquez sur **Test Proxy Rotation**.
2. TikMatrix accédera immédiatement au lien de rotation.
   - En cas de succès, une notification de succès s'affichera et le temps "Last rotated" sera actualisé.
   - En cas d'échec, notez la notification, vérifiez que le lien et le mot de passe sont corrects, puis réessayez.

Conseil : De nombreux proxies limitent la fréquence de changement d'IP. Si la notification indique un échec, attendez quelques secondes avant de réessayer.

## Utiliser la rotation dans les scripts

1. Lors de l'exécution d'un script, cochez d'abord **Rotation de proxy**, TikMatrix changera l'IP pour chaque appareil avant d'ouvrir TikTok.
2. Vous pouvez cliquer sur le bouton de journal à droite de l'appareil pour consulter les informations détaillées de chaque rotation.

## Questions fréquentes

- **Faut-il faire une rotation pour tous les scripts ?** Non. Seulement lorsque vous souhaitez que TikTok voie une nouvelle IP, par exemple avant de changer fréquemment de compte.
- **Où est enregistrée la configuration ?** TikMatrix la gère automatiquement. Chaque appareil n'a besoin d'être configuré qu'une fois.
- **Le lien semble très long et compliqué, que faire ?** Tant qu'il peut être ouvert dans un navigateur, TikMatrix peut également l'utiliser.
- **Toujours des difficultés ?** Envoyez les instructions du fournisseur et la notification d'erreur au service client TikMatrix, nous vous aiderons à résoudre le problème ensemble.
