# Activer le clavier de saisie rapide

Le téléphone basculera automatiquement vers le clavier de saisie rapide après la connexion.

## Pourquoi configurer le clavier de saisie rapide

Le clavier de saisie rapide est un outil essentiel pour TikMatrix, offrant une prise en charge automatique multilingue. Sans lui, les fonctions de saisie de texte de TikMatrix (telles que les commentaires, les messages et les titres de publication) pourraient ne pas fonctionner correctement.

## Comment activer manuellement le clavier de saisie rapide

1. Sélectionnez l'appareil qui nécessite le clavier de saisie rapide.
2. Cliquez sur `Commandes ADB personnalisées` - `Activer la saisie rapide`.

## Comment saisir du texte

Dans TikMatrix, vous pouvez saisir du texte de plusieurs façons :

### Méthode 1 : Saisie directe au clavier ⌨️ (Nouvelle fonctionnalité)

⭐ **Contrôle de saisie au clavier** - Vous pouvez maintenant utiliser directement le clavier de votre ordinateur pour saisir du texte sur le téléphone, rendant la saisie de texte plus rapide et plus pratique.

1. Sélectionnez l'appareil à utiliser.
2. Cliquez sur n'importe quel champ de saisie de texte sur l'écran du téléphone.
3. Commencez directement à taper sur le clavier de l'ordinateur - le texte s'affichera en temps réel sur le téléphone.

### Méthode 2 : Panneau de saisie

1. Cliquez sur l'écran du téléphone, développez l'affichage grand écran et cliquez sur le bouton `Saisie` dans la barre d'outils à droite.
2. Saisissez le texte dans le champ de saisie et cliquez sur envoyer.

## Comment copier et coller du texte entre le téléphone et l'ordinateur

### Copier le texte du presse-papiers du téléphone vers le presse-papiers de l'ordinateur

1. Sélectionnez le téléphone à copier, maintenez les touches `Ctrl+C`, copie réussie

### Coller le texte du presse-papiers de l'ordinateur dans le champ de saisie du téléphone

1. Sélectionnez l'appareil où coller, maintenez les touches `Ctrl+V`, collage réussi
