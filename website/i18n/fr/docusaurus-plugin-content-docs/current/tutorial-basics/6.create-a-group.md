# Créer un groupe

Un **groupe** est un ensemble d'appareils qui permet de gérer et d'opérer plusieurs appareils de manière unifiée et pratique.

## Étapes

1. Cliquez sur le bouton `Ajouter un groupe` dans la barre latérale gauche.
2. Saisissez le nom du groupe.
3. Appuyez sur la touche `Entrée` pour enregistrer le groupe.
4. Vous pouvez sélectionner des appareils et les déplacer vers ce groupe.
