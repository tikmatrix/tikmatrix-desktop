# Paramètres des tâches

## Règles de planification des tâches

- Les tâches seront exécutées dans les 10 secondes suivant l'heure de début.
- Chaque appareil n'exécutera qu'une seule tâche à la fois, les autres tâches seront automatiquement mises en file d'attente sur cet appareil.
- Le nombre maximum de tâches simultanées est limité par le niveau de licence ; les utilisateurs gratuits n'exécuteront aucune tâche d'automatisation, qui restera affichée comme "en attente".
- Le système ajoutera un délai aléatoire de 5 à 60 secondes avant chaque nouvelle tentative.

Cette boîte de dialogue est utilisée pour contrôler la stratégie de nouvelle tentative et l'ordre d'exécution des tâches. Voici une brève explication de chaque option et de son utilisation.

## Nouvelle tentative automatique

Fonction :

- Réessaie automatiquement les tâches échouées jusqu'au succès ou jusqu'à atteindre la limite de nouvelles tentatives.

Comment l'utiliser :

- Activez lorsque les échecs sont principalement dus à des pannes temporaires (fluctuations réseau, anomalies d'application).
- Vérifiez d'abord le comportement sur un petit ensemble de tâches avant de décider de l'activer pour toutes.

## Nombre maximum de nouvelles tentatives

Fonction :

- Définit le nombre maximum de tentatives pour la nouvelle tentative automatique de tâche.

Comment l'utiliser :

- Il est recommandé de commencer par 2-3, augmentez légèrement si vous rencontrez beaucoup d'échecs temporaires.
- L'interface limitera à une plage raisonnable (par exemple 1-10).

## Ordre aléatoire

Fonction :

- Exécute les tâches dans un ordre aléatoire sur chaque appareil, évitant les comportements synchronisés.

Comment l'utiliser :

- Activez pour les grands pools d'appareils ou lorsque vous souhaitez réduire la détectabilité.

## Prise d'effet et notifications

- Les modifications prennent généralement effet immédiatement ; l'interface affichera une notification de succès ou d'échec de l'enregistrement.

## Où ouvrir

- Ouvrez la page des appareils ou la page des tâches, cliquez sur l'icône de paramètres des tâches (engrenage).

## Suggestions rapides

- Commencez par ajuster `Nombre maximum de nouvelles tentatives` à 2-3.
- Activez `Ordre aléatoire` lors d'exécutions à grande échelle pour réduire les modes synchronisés.
- Testez d'abord le comportement de `Nouvelle tentative automatique` sur un petit ensemble.
