# Contrôle synchronisé

Le contrôle synchronisé permet de contrôler les opérations synchronisées de plusieurs appareils.

## Étapes

1. Cliquez pour ouvrir l'écran d'un téléphone.
2. Sélectionnez plusieurs appareils sur le côté gauche.
3. Toutes les opérations à l'écran seront synchronisées sur plusieurs appareils.
