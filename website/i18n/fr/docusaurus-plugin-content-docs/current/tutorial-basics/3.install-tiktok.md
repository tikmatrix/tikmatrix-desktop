# Installer Douyin/TikTok

Nous recommandons de télécharger les versions spécifiques suivantes, car les versions plus récentes peuvent ne pas être compatibles. Le logiciel sera régulièrement mis à jour, les versions prises en charge seront annoncées dans le groupe après la mise à jour.

## Télécharger le package d'installation Douyin/TikTok

* Version internationale : [com.zhiliaoapp.musically](https://apkpure.com/tiktok-musically-2024/com.zhiliaoapp.musically)
* Version asiatique : [com.ss.android.ugc.trill](https://apkpure.com/tiktok/com.ss.android.ugc.trill)

## Étapes d'installation

1. Téléchargez le package APK depuis les liens ci-dessus.
2. Sélectionnez tous les appareils.
3. Cliquez sur `Général` - `Installer APK` - Sélectionnez le package APK téléchargé.
4. Attendez que l'installation soit terminée (peut prendre 1 à 3 minutes par appareil, selon le modèle de téléphone ; veuillez patienter et ne cliquez pas plusieurs fois sur le bouton « Installer APK »).
5. Une fois l'installation terminée, vous devez cliquer sur le bouton `Paramètres` pour vérifier si la sélection de version TikTok correspond à la version que vous avez installée.
6. Cliquez sur le bouton `Ouvrir TikTok` pour vérifier si l'application TikTok est correctement installée.
