# Changement de compte multiple

TikTok et Instagram prennent tous deux en charge la connexion de plusieurs comptes sur le même appareil, c'est une fonctionnalité native de l'application officielle. TikMatrix et IgMatrix offrent une fonctionnalité de changement automatique de compte, permettant d'exécuter automatiquement des tâches sur tous les comptes connectés, sans intervention manuelle.

## Scripts supportés

Les scripts suivants prennent en charge la fonctionnalité de changement automatique de compte :

- **Profile (Voir le profil)** — Consulter les informations du profil du compte
- **Account Warmup (Préchauffage de compte)** — Préchauffer le compte par une activité naturelle
- **Post (Publier du contenu)** — Publier des posts
- **Mass DM (Messages en masse)** — Envoyer des messages en masse
- **Follow Back (Suivre en retour)** — Suivre en retour les utilisateurs qui vous suivent
- **Unfollow All (Tout désabonner)** — Se désabonner de tous les comptes
- **Delete Post (Supprimer le post)** — Supprimer le contenu publié
- **Boost Users (Promouvoir les utilisateurs)** — Augmenter l'engagement sur la page d'accueil des utilisateurs
- **Boost Posts (Promouvoir les posts)** — Augmenter l'engagement des posts
- **Boost Lives (Promouvoir les lives)** — Augmenter l'engagement de la salle de direct
- **Mass Comment (Commentaires en masse)** — Publier des commentaires en masse
- **Boost Comments (Promouvoir les commentaires)** — Augmenter l'engagement des commentaires
- **Super Marketing (Marketing avancé)** — Automatisation marketing avancée

## Configuration initiale : Matcher les comptes

Avant d'utiliser la fonctionnalité de changement de compte multiple, vous devez d'abord matcher les comptes depuis l'appareil.

1. Ouvrez l'onglet **Scripts** dans la navigation de gauche.
2. Trouvez et exécutez le script **Matcher les comptes**.
3. Ce script scannera les comptes connectés sur l'appareil et les ajoutera automatiquement à la liste des comptes.
4. Une fois l'exécution du script terminée, cliquez sur le bouton **Comptes** pour vérifier que les comptes de l'appareil ont été ajoutés avec succès.

## Tester le changement de compte

Après avoir exécuté le script **Matcher les comptes**, vous pouvez tester la fonctionnalité de changement de compte :

1. Dans l'onglet Scripts, exécutez le script **Changer de compte**.
2. Ce script n'a aucune autre logique — son seul rôle est de vérifier que le processus de changement de compte fonctionne normalement.
3. Si le script s'exécute avec succès, cela signifie que la fonctionnalité de changement de compte est prête.

## Utiliser le mode multi-comptes

Après avoir terminé la configuration initiale et les tests :

1. Lors de l'exécution de tout script supporté, activez le **Mode multi-comptes** dans les paramètres du script.
2. Le système créera des tâches indépendantes pour chaque compte dans la liste des comptes.
3. Les tâches seront mises en file d'attente et exécutées successivement — le système changera automatiquement vers le compte cible et exécutera la tâche d'automatisation.
4. Aucune intervention manuelle n'est nécessaire.

## Quand réexécuter Matcher les comptes

Les scripts **Matcher les comptes** et **Changer de compte** n'ont généralement besoin d'être exécutés qu'une seule fois lors de la configuration initiale. Cependant, vous devriez réexécuter **Matcher les comptes** dans les situations suivantes :

- Connexion de nouveaux comptes sur l'appareil
- Déconnexion de comptes existants
- Modification des comptes sur l'appareil

Étapes pour actualiser la liste des comptes :

1. Effacez manuellement la liste des comptes sur la page **Comptes**.
2. Réexécutez le script **Matcher les comptes** pour détecter les comptes actuels sur l'appareil.

---

La fonctionnalité de changement de compte multiple élimine le besoin de changer manuellement de compte, vous permettant d'étendre facilement l'automatisation à tous les comptes sur un seul appareil.
