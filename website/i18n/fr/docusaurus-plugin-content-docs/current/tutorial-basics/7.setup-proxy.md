# Configurer un proxy

Vous pouvez configurer une adresse IP proxy indépendante pour chaque téléphone.

> **Clause de non-responsabilité :** Tous les sites web recommandés sur cette page proviennent d'Internet et ne sont pas affiliés à ce site. Veuillez vérifier leur validité par vous-même avant utilisation. Nous ne fournissons aucune garantie.

## 1. Configurer un proxy VPS résidentiel (vous pouvez ignorer cette étape si vous avez acheté un proxy IP)

1. Achetez un VPS proxy résidentiel dans le pays cible (de préférence un VPS avec IP native et double ISP).
2. Utilisez [https://github.com/MHSanaei/3x-ui](https://github.com/MHSanaei/3x-ui) pour configurer rapidement un VPN sur le VPS :

    ```shell
    bash <(curl -Ls https://raw.githubusercontent.com/MHSanaei/3x-ui/refs/tags/v2.6.0/install.sh)
    # Suivez les instructions pour terminer la configuration du VPN.
    ```

3. Une fois l'installation terminée, accédez à l'interface Web x-ui. Vous pouvez trouver l'URL et les identifiants de connexion dans la sortie du terminal après l'installation.
4. Dans x-ui, créez un nouvel utilisateur, définissez le protocole sur `SOCKS5` et configurez le port et d'autres paramètres selon vos besoins.
5. Après avoir créé l'utilisateur, vous verrez l'IP proxy et le port dans l'interface x-ui.
6. Utilisez l'IP proxy et le port dans l'application NekoBox sur le téléphone.

## 2. Acheter un proxy IP résidentiel (vous pouvez ignorer cette étape si vous avez configuré un VPS)

Vous pouvez également acheter directement un proxy IP résidentiel sans configurer de VPS.

> Outils de détection d'IP

- [ipinfo](https://ipinfo.io) — Vérifiez si l'IP est une IP résidentielle.
- [scamalytics](https://scamalytics.com/ip) — Vérifiez le niveau de risque de l'IP.

## 3. Activer le proxy sur le téléphone

1. Téléchargez l'APK et installez le client [NekoBoxForAndroid](https://github.com/MatsuriDayo/NekoBoxForAndroid) sur le téléphone.
2. Ouvrez l'application NekoBox, cliquez sur le bouton `+` pour ajouter un nouveau proxy.
3. Sélectionnez `SOCKS5` ou un autre type de proxy.
4. Saisissez l'IP proxy et le port, puis cliquez sur `Enregistrer`.
5. Cliquez sur le bouton `Démarrer` pour activer le proxy.
6. Une fois le proxy activé, vous pouvez vérifier l'état de la connexion dans l'application NekoBox.

### Note pour les utilisateurs en Chine

Si vous êtes en Chine, le protocole `SOCKS5` peut ne pas fonctionner correctement. Vous devrez ajouter un proxy chiffré (`VLESS`) comme proxy frontal. Vous pouvez utiliser les outils suivants :

1. Ajoutez un proxy `VLESS` dans l'application NekoBox.
2. Ajoutez un proxy `SOCKS5` dans l'application NekoBox.
3. Ajoutez un proxy `en chaîne` dans l'application NekoBox. Sélectionnez le proxy `VLESS` comme proxy frontal et le proxy `SOCKS5` comme proxy backend.
