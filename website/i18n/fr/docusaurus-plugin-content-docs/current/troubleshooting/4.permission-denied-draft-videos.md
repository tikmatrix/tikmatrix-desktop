# Permission Denied - Unable to Save Draft Videos

## Résoudre le problème de refus de permission apparaissant sur l'écran du téléphone

### Description du problème

Lors de l'exécution d'un script de publication, le message suivant peut apparaître sur l'écran du téléphone :

```text
Permission denied. Unable to save draft videos.
```

### Solution

Ce problème est causé par l'absence de permission de stockage pour l'application TikTok. Veuillez suivre les étapes suivantes pour le résoudre :

#### Étape 1 : Cliquer manuellement sur le bouton de publication

1. Trouvez et cliquez sur le bouton **Publier** sur l'écran du téléphone
2. Continuez le processus de publication

#### Étape 2 : Autoriser les permissions système

1. Lorsque la boîte de dialogue de demande de permission système apparaît, cliquez sur **Autoriser**
2. Assurez-vous d'accorder les permissions de stockage requises à l'application TikTok

### Note importante

:::info
Cette étape de configuration des permissions ne doit être effectuée qu'une seule fois. Une fois les permissions accordées, les exécutions ultérieures du script de publication ne rencontreront plus ce problème.
:::

### Mesures préventives

Pour éviter ce problème, il est recommandé avant la première utilisation de TikMatrix :

1. Ouvrir manuellement l'application TikTok
2. Essayer d'enregistrer ou de sauvegarder un brouillon vidéo
3. Sélectionner **Autoriser** lors de la demande de permission
4. Cela permettra de préconfigurer les permissions requises
