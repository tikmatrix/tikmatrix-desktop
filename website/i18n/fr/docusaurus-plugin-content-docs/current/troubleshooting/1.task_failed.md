# Étapes de dépannage en cas d'échec de tâche

## Étape 1 : Vérifier l'état de l'Agent (si la tâche n'a pas pu démarrer avec succès)

1. Cliquez sur "Général" dans la barre latérale gauche - "Ouvrir l'Agent"
2. Confirmez que l'application TikMatrix est ouverte sur le téléphone
3. Observez si un état de succès s'affiche dans les 10 secondes
4. Si l'état de l'Agent est en échec, veuillez suivre les étapes du document [Initialiser l'appareil](../tutorial-basics/2.init-device.md) pour effectuer une initialisation manuelle

## Étape 2 : Vérifier l'état de l'application TikTok/Instagram (si le script n'a pas ouvert ou fermé correctement l'application)

1. Si vous utilisez l'application officielle TikTok/Instagram, cliquez sur "Ouvrir TikTok"/"Ouvrir Instagram" et observez si le téléphone ouvre correctement l'application TikTok ou Instagram. Si ce n'est pas le cas, cliquez sur le bouton paramètres dans la barre de titre, sélectionnez la bonne version de TikTok (version internationale/version asiatique). Instagram n'a qu'une seule version officielle
2. Si vous utilisez une version clonée de l'application TikTok/Instagram, assurez-vous d'avoir correctement configuré le **nom du package** de l'application clonée dans la **liste des comptes**. Veuillez vous référer au document [Utiliser des applications TikTok/Instagram clonées](../tutorial-basics/10.clone-apps-support.md) pour la configuration, et assurez-vous d'activer le **mode multi-comptes** lors de l'exécution de la tâche

## Étape 3 : Vérifier les paramètres de langue de TikTok/Instagram

Actuellement, nous ne prenons en charge que TikTok et Instagram en anglais. Si votre application TikTok ou Instagram n'est pas en anglais, veuillez suivre les étapes suivantes :

1. Ouvrez l'application TikTok sur votre téléphone
2. Accédez à votre profil
3. Cliquez sur les trois points en haut à droite pour ouvrir les paramètres
4. Faites défiler jusqu'à la section "Langue"
5. Sélectionnez "Anglais" comme langue
6. Répétez les étapes ci-dessus pour configurer l'application Instagram en anglais

## Étape 4 : Exécuter le script de test

1. Cliquez sur "Scripts" dans la barre latérale gauche - "Script de test"
2. Attendez que le script se termine
3. Si le script de test échoue, consultez les journaux et essayez de résoudre le problème. Pour les problèmes courants, reportez-vous à la section "Problèmes connus" ci-dessous

## Étape 5 : Soumettre un ticket

Si les 4 premières étapes de vérification n'ont révélé aucun problème, vous devez créer un ticket. Nous collecterons automatiquement les journaux et vous aiderons à résoudre le problème. Veuillez suivre les étapes suivantes :

1. Cliquez sur le bouton "Support" dans la barre de titre
2. Cliquez sur le bouton "Créer un ticket"
3. Entrez vos coordonnées et la description du problème. Pour une résolution plus rapide, veuillez essayer d'enregistrer une vidéo de l'écran du téléphone pendant l'exécution du script et de l'ajouter aux pièces jointes
4. Sélectionnez les appareils affectés, vous pouvez en sélectionner plusieurs (important, nous ne pouvons recevoir les journaux pertinents que si vous sélectionnez les appareils affectés)
5. Cliquez sur le bouton "Soumettre"
6. Nous pourrions vous contacter pour obtenir plus d'informations, par exemple pour télécharger le fichier XML de l'interface utilisateur d'une page spécifique : manipulez manuellement le téléphone pour rester sur la dernière page avant l'échec de la tâche, cliquez sur le bouton "Déboguer" sur le côté droit du grand écran du téléphone, puis cliquez sur le bouton "Télécharger XML", envoyez le fichier XML téléchargé à notre adresse e-mail : `support@tikmatrix.com` avec le numéro de ticket
