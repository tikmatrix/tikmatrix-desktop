# Impossible de détecter le téléphone

## Étapes de dépannage pour la non-détection du téléphone

### Première étape : Vérifier l'état de la connexion ADB

1. Dans TikMatrix, cliquez sur `Paramètres` > `Ouvrir le terminal ADB`
2. Exécutez la commande `adb devices` pour vérifier si le téléphone apparaît dans la liste

### Système Windows

```shell
.\adb.exe devices
```

### Système Mac/Linux

```shell
./adb devices
```

### Exemple de sortie

```shell
List of devices attached
<ID de l'appareil>    <état>
```

### Si l'appareil n'apparaît pas dans la liste des appareils ADB

Veuillez suivre les étapes du document [Connecter des appareils](../tutorial-basics/1.connect-to-devices.md) pour le dépannage

### Si l'état de l'appareil est `unauthorized`

La raison possible est que l'appareil n'est pas autorisé. Veuillez cliquer sur `Autoriser le débogage USB` sur le téléphone

### Si l'état de l'appareil est `offline`

La raison possible est que le câble USB est desserré. Veuillez reconnecter le téléphone, débrancher le câble USB et le reconnecter

### Si l'état de l'appareil est `device`

L'appareil est déjà connecté avec succès. Si TikMatrix ne détecte toujours pas le téléphone, veuillez confirmer si TikMatrix est activé. L'essai gratuit nécessite également une activation sur la page d'autorisation

### Si le problème persiste

Veuillez contacter notre équipe de support technique via [Telegram](https://t.me/tikmatrix_agent_bot) et fournir les informations suivantes :

- Capture d'écran de la sortie de la commande adb devices
- Les étapes que vous avez déjà essayées
