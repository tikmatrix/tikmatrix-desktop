# Extraire les utilisateurs

Le script d'extraction d'utilisateurs permet de collecter les noms d'utilisateur TikTok. La fonctionnalité d'extraction d'utilisateurs prend en charge trois modes : collecter la liste de followers d'un utilisateur spécifié, collecter la liste d'abonnements d'un utilisateur spécifié, ou collecter des utilisateurs via une recherche par mots-clés.

## Étapes

1. Sélectionnez les appareils sur lesquels exécuter le script. Si vous sélectionnez plusieurs appareils, les tâches d'extraction seront réparties équitablement entre les appareils sélectionnés.
2. Cliquez sur `Scripts` > `Extraire les utilisateurs`.
3. Configurez les paramètres de tâche :
    - **Mode d'extraction** : sélectionnez le mode d'extraction des utilisateurs.
      - **Extraction par followers** : collecte la liste de followers d'un compte spécifié.
      - **Extraction par abonnements** : collecte la liste d'abonnements d'un compte spécifié.
      - **Extraction par mots-clés** : collecte des utilisateurs via une recherche par mots-clés.
    - **Nom d'utilisateur du compte cible** : entrez le nom d'utilisateur du compte dont vous souhaitez extraire les utilisateurs, un par ligne (pour les modes followers et abonnements).
    - **Mots-clés de recherche** : entrez les mots-clés pour rechercher des utilisateurs (pour le mode mots-clés).
    - **Nombre maximum d'extractions** : définissez le nombre maximum d'utilisateurs à extraire.

4. Cliquez sur `Démarrer le script` pour commencer l'exécution.

## Remarques

- Les données d'utilisateurs extraites seront enregistrées dans un fichier selon le mode d'extraction :
  - Mode followers : `exported_users_followers_tiktok_<nom_utilisateur>_<nombre>.txt`
  - Mode abonnements : `exported_users_following_tiktok_<nom_utilisateur>_<nombre>.txt`
  - Mode mots-clés : `exported_users_search_tiktok_<mot_clé>_<nombre>.txt`
- TikTok limite uniquement le nombre d'utilisateurs visibles, par conséquent, environ 50 enregistrements peuvent être récupérés à chaque opération.
