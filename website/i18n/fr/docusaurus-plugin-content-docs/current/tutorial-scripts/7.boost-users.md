# Booster les utilisateurs

:::warning Avis de dépréciation
Cette fonctionnalité a été intégrée dans le [script Super Marketing](./14.super-marketing.md). Veuillez utiliser en priorité "Super Marketing" pour bénéficier de capacités de boost d'utilisateurs plus puissantes et flexibles.
:::

La fonctionnalité de boost d'utilisateurs améliore l'exposition et l'engagement des utilisateurs cibles via des opérations massives de suivi/désabonnement. Le système simule au maximum le comportement des utilisateurs réels pour réduire le risque de détection, mais il est toujours recommandé de respecter les règles de la plateforme et de contrôler la fréquence.

## Utiliser la fonctionnalité de boost d'utilisateurs dans le script "Super Marketing"

1. Ouvrez le script : sélectionnez "Super Marketing".
2. Sélectionnez le type de source de données "Liste de comptes utilisateurs".
3. Collez en masse les noms d'utilisateur ou importez-les depuis un fichier texte (un par ligne) ; après l'importation, vous pouvez choisir "Ajouter" ou "Remplacer".
4. Sélectionnez la stratégie d'utilisation des données :

    - Stratégie de pool partagé : tous les comptes bot suivront/désabonneront les mêmes cibles.

    - Stratégie de consommation unique : chaque compte bot ne traitera que des cibles différentes, garantissant que chaque cible n'est opérée qu'une seule fois par un compte.

5. Dans "Opération utilisateur — Nombre de répétitions", définissez le nombre d'opérations pour chaque cible (généralement défini à 1 fois).
6. Sélectionnez le type d'opération : Suivre ou Désabonner. Vous pouvez configurer davantage dans "Opération utilisateur" :

        - Méthode d'accès au profil utilisateur : sélectionnez "Rechercher l'utilisateur" ou "Ouvrir directement le profil (Direct Open Profile)".

        - Envoyer un message privé (optionnel) : une fois activé, vous pouvez coller plusieurs lignes de messages privés prégénérés, chaque ligne ou paragraphe étant un message candidat ; configurez l'ordre d'utilisation des messages :

                - Aléatoire : sélectionne aléatoirement un message à chaque fois.
                - Séquentiel : utilise les messages dans l'ordre collé/importé (peut continuer entre les tours de tâches).

            Vous pouvez également choisir d'insérer automatiquement des emojis dans les messages pour améliorer le naturel.

7. Configurez les autres paramètres (concurrence, délais, pool de comptes), confirmez que tout est correct, puis lancez le script.

## Conseils d'utilisation et précautions

- Il est recommandé de tester d'abord sur un petit nombre de comptes pour observer si la stratégie et les paramètres déclenchent les contrôles de la plateforme.
- Configurez raisonnablement les délais et la concurrence pour simuler un comportement humain et réduire les risques.
- Pour une exécution à long terme, vérifiez régulièrement la santé des comptes et les journaux de tâches, et ajustez la stratégie en fonction des résultats.
- Les opérations de boost d'utilisateurs ne réussissent pas à 100%, en cas d'échec, veuillez réessayer ou vérifier l'état du compte cible.
