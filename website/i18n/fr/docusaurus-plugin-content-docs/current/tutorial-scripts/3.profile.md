# Profil

Le script de profil sert à configurer le profil de vos comptes.

## Étapes

1. Sélectionnez les appareils pour lesquels vous souhaitez configurer le profil.
2. Cliquez sur le bouton `Scripts` - `Profil`.
3. Configurez les paramètres de la tâche :
    - **Pseudo** : Un pseudo par ligne.
    - **Nom d'utilisateur** : Un nom d'utilisateur par ligne.
    - **Biographie** : Une biographie par ligne (prend en charge les mentions d'utilisateurs et les sauts de ligne).
      - Pour ajouter un saut de ligne dans la biographie, utilisez `\n` dans le texte. Par exemple : `Première ligne\nDeuxième ligne\nTroisième ligne`
    - **Chemin de l'avatar** : Le chemin du dossier contenant les images d'avatar.
    - **Ordre de sélection** : Choisissez comment le contenu est sélectionné pour chaque tâche :
      - **Aléatoire** : Sélectionne aléatoirement parmi les options disponibles pour chaque tâche (par défaut).
      - **Séquentiel** : Sélectionne le contenu dans l'ordre en fonction du compteur de tâches. Utilisez ce mode pour vous assurer que chaque compte reçoit des informations uniques.
4. Cliquez sur le bouton `Démarrer le script` pour lancer la tâche.

## Explication de l'ordre de sélection

### Mode aléatoire

- Chaque tâche sélectionne aléatoirement le contenu (pseudo, nom d'utilisateur, biographie, avatar)
- Le même contenu peut être réutilisé sur différents comptes
- Adapté aux scénarios où l'unicité du contenu n'est pas nécessaire

### Mode séquentiel

- Sélectionne le contenu dans l'ordre : la 1ère tâche utilise le 1er élément, la 2ème tâche utilise le 2ème élément, et ainsi de suite
- Lorsque tous les éléments ont été utilisés, le système revient au début
- **Note importante** : Pour vous assurer que chaque compte reçoit des informations uniques, assurez-vous que la quantité de contenu fournie est supérieure au nombre total de comptes exécutés dans un lot unique
  - Exemple : Si vous exécutez 100 comptes, fournissez au moins 100 pseudos, noms d'utilisateur, biographies et images d'avatar différents

## Notes

- Il est préférable de vider l'album avant de configurer le profil, car le script utilisera la première photo de l'album comme avatar.
