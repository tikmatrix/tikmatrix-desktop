# Supprimer les publications

Le script de suppression de publications permet de supprimer des publications en fonction du nombre de vues.

## Étapes

1. Sélectionnez les appareils sur lesquels exécuter le script.
2. Cliquez sur `Scripts` > `Supprimer les publications`.
3. Configurez les paramètres de tâche :
    - **Nombre maximum de vues** : entrez le nombre maximum de vues des publications que vous souhaitez supprimer.
4. Cliquez sur `Démarrer le script` pour commencer l'exécution.

## Remarques

- Le script de suppression de publications peut ne pas toujours réussir ; veuillez réessayer les tâches échouées si nécessaire.
