# Connexion

Le script de connexion sert à vous connecter à vos comptes sur les réseaux sociaux.

## Étapes

1. Assurez-vous que les comptes ont été ajoutés à la liste des comptes (cliquez sur `Comptes` pour ajouter un compte), et sélectionnez le numéro de série du téléphone sur lequel vous souhaitez vous connecter à ce compte.
2. Sélectionnez un ou plusieurs appareils sur lesquels vous souhaitez exécuter le script de connexion.
3. Cliquez sur `Scripts` > `Connexion`.
4. Cliquez sur `Démarrer le script` pour lancer l'exécution.

## Priorité de connexion

Le système utilise l'ordre de priorité de connexion suivant :

1. **Email + Mot de passe** : Si un e-mail est configuré pour le compte, il sera prioritairement utilisé pour la connexion.
2. **Nom d'utilisateur + Mot de passe** : Si aucun e-mail n'est configuré, le nom d'utilisateur sera utilisé pour la connexion.

## Support 2FA (IgMatrix uniquement)

IgMatrix prend en charge la connexion automatique avec authentification à deux facteurs (2FA) :

### Format du mot de passe 2FA

Pour utiliser la 2FA, définissez votre mot de passe au format suivant :

```plaintext
votre_mot_de_passe:votre_clé_2FA
```

**Exemple :**

```plaintext
monmotdepasse123:JBSWY3DPEHPK3PXP
```

### Fonctionnement de la 2FA

1. **Analyse du mot de passe** : Le système détecte automatiquement si le mot de passe contient le séparateur deux-points (`:`).
2. **Génération du code de vérification** : Lorsqu'une vérification 2FA est requise, le système utilise votre clé pour demander un code de vérification TOTP à notre service 2FA.
3. **Saisie automatique** : Le code de vérification à 6 chiffres généré est automatiquement saisi dans la boîte de vérification 2FA d'Instagram.
4. **Connexion transparente** : L'ensemble du processus 2FA est automatisé sans intervention manuelle.

### Informations sur la clé 2FA

La clé 2FA est généralement une chaîne de 32 caractères encodée en Base32, que vous obtenez lors de la configuration de la 2FA dans Instagram. C'est la même clé utilisée par les applications d'authentification comme Google Authenticator ou Authy.

### Obtenir des comptes Instagram avec support 2FA

Vous pouvez acheter des comptes Instagram avec support 2FA sur le site suivant : [AccsMarket](https://accsmarket.com/en/?ref=802647)

## Notes

- La connexion TikTok n'est pas automatique à 100 %. Vous devrez peut-être saisir manuellement le code de vérification par e-mail ou effectuer d'autres étapes de vérification. Nous ne pouvons pas non plus résoudre les défis de connexion TikTok (comme les CAPTCHA d'image, la vérification humaine, etc.) ni l'erreur indiquant : connexions trop fréquentes.
- Instagram peut réaliser une connexion 100 % automatique via la 2FA. Pour les comptes Instagram avec 2FA activée, assurez-vous d'utiliser le format de clé correct dans le mot de passe.
