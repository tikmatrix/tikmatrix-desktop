# Booster les Lives

Le script de boost de lives vous permet de contrôler des comptes ferme pour regarder et optionnellement liker ou commenter des lives cibles. Vous pouvez personnaliser le comportement de boost en configurant la durée de visionnage et les intervalles de likes/commentaires.

## Étapes

1. Sélectionnez les appareils sur lesquels exécuter le script.
2. Cliquez sur `Scripts` > `Booster les Lives`.
3. Configurez les paramètres de la tâche :
    - **Nom d'utilisateur** : Entrez les noms d'utilisateurs cibles, un par ligne.
    - **Méthode d'entrée** : Sélectionnez la méthode **Recherche** ou **Directe**.
    - **Options de boost** : Activez les likes et/ou les commentaires.
    - **Rejoindre le fan club** : Activez la fonction pour rejoindre le fan club du streamer (si disponible).
    - **Check-in quotidien** : Activez l'activité de check-in quotidien pendant le live.
    - **Intervalle de likes** : Définissez l'intervalle entre chaque opération de like.
    - **Nombre de clics de like** : Définissez le nombre de fois où cliquer sur le bouton like.
    - **Intervalle de commentaires** : Définissez l'intervalle entre chaque opération de commentaire.
    - **Contenu des commentaires** : Entrez un commentaire par ligne.
    - **Insérer des emojis** : Choisissez d'insérer ou non des emojis dans les commentaires.
    - **Ordre des commentaires** : Sélectionnez l'ordre aléatoire ou séquentiel des commentaires.
    - **Nombre de commentaires par compte** : Définissez le nombre de commentaires publiés par chaque compte.
    - **Durée de visionnage** : Définissez la durée de présence dans le live.
    - **Intervalle entre tâches** : Définissez l'intervalle entre chaque tâche.
4. Cliquez sur `Démarrer le script` pour commencer l'exécution.

## Remarques

- Le script de boost de lives peut ne pas toujours réussir ; réessayez les tâches échouées si nécessaire.
