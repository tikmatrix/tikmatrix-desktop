# Commentaires en masse

:::warning Avis de dépréciation
Cette fonctionnalité a été intégrée dans le [script Super Marketing](./14.super-marketing.md). Veuillez utiliser en priorité le script "Super Marketing" pour bénéficier de capacités de commentaires en masse et d'interaction de contenu plus puissantes et flexibles.
:::

Les commentaires en masse permettent de publier des commentaires d'engagement sur un grand nombre de publications TikTok (comme pour l'échauffement de compte, la promotion, les opérations d'événements). Le système prend en charge deux sources de messages : les modèles prégénérés et la génération en temps réel, et simule au maximum le rythme d'envoi manuel pour réduire le risque de détection.

## Utiliser les commentaires en masse dans le script "Super Marketing"

1. Ouvrez le script : sélectionnez "Super Marketing".
2. Sélectionnez le type de source de données "Liste de liens de publications".
3. Collez en masse les liens de publications ou importez-les depuis un fichier texte (un par ligne) ; après l'importation, vous pouvez choisir "Ajouter" ou "Remplacer".
4. Sélectionnez la stratégie d'utilisation des données :

   - Stratégie de pool partagé : tous les comptes bot commentent les mêmes publications.

   - Stratégie de consommation unique : chaque compte bot ne commente qu'une seule fois des publications différentes, garantissant que chaque publication n'est traitée qu'une seule fois par un compte.

5. Définissez le nombre de commentaires pour chaque publication (généralement 1 fois pour éviter les commentaires en double).
6. Dans "Paramètres de commentaires", configurez les options suivantes :

   - Modèles de commentaires prégénérés : vous pouvez coller plusieurs lignes ou paragraphes de commentaires prégénérés (chaque ligne ou paragraphe est considéré comme un commentaire candidat).

   - Ordre d'utilisation des messages : sélectionnez "Aléatoire" ou "Séquentiel".

   - Aléatoire : tire aléatoirement un message parmi les candidats à chaque fois.

   - Séquentiel : alterne l'utilisation dans l'ordre collé/importé (convient aux stratégies de déduplication reproductibles).

   - Insérer des emojis : vous pouvez activer l'insertion automatique d'emojis dans les commentaires pour améliorer le naturel.

   - Insérer le numéro d'appareil (optionnel) : ajoute le numéro d'appareil à la fin du commentaire pour suivre la source (uniquement pour le débogage/audit, utilisez avec précaution).

   - Génération de commentaires en temps réel (optionnel, API ChatGPT) :

      - Une fois activé, le script peut appeler une API de type ChatGPT pour générer des commentaires contextuels et uniques en temps réel basés sur le titre/la description de la publication ou le contexte cible, adapté aux scénarios nécessitant un grand nombre de commentaires uniques.

      - Remarque : l'activation de la génération en temps réel nécessite de fournir une clé API valide, et il faut faire attention aux limites de taux, aux coûts et à la conformité en matière de confidentialité (éviter d'envoyer des informations sensibles à des services tiers).

   - Deux méthodes courantes pour garantir des commentaires uniques :

      1. Utiliser "Séquentiel" pour alterner les modèles de commentaires prégénérés (chaque modèle n'est utilisé qu'une seule fois jusqu'à épuisement).

      2. Utiliser ChatGPT pour générer chaque commentaire en temps réel (génération dynamique basée sur le contenu de la publication, unicité plus élevée).

7. Configurez les autres paramètres du script (nombre de concurrences, intervalle entre les tâches, pool de comptes, etc.), confirmez que tout est correct, puis lancez le script.

:::tip À propos de l'intervalle entre les tâches
Il est recommandé de définir un intervalle min/max et d'activer l'intervalle aléatoire pour éviter que tous les appareils n'exécutent simultanément, créant un pic anormal, réduisant ainsi le risque de détection ou de limitation par la plateforme.
:::

## Conseils d'utilisation et précautions

- Testez d'abord votre stratégie de commentaires sur un petit lot de publications pour confirmer la qualité des commentaires et le traitement par la plateforme.
- Le contenu des commentaires doit être conforme et diversifié, évitez d'envoyer un grand nombre de contenus identiques en peu de temps.
- Si vous utilisez la génération en temps réel, surveillez strictement le volume d'appels API et les coûts, et assurez-vous de ne pas divulguer d'informations sensibles à l'extérieur.
- Les tâches peuvent échouer ; en cas d'échec, veuillez réessayer ou consulter les journaux, le système ignorera automatiquement les cibles déjà complétées.
