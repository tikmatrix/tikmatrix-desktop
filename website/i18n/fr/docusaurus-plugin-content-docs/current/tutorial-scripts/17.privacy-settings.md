# Paramètres de Confidentialité

Le script **Paramètres de Confidentialité** est utilisé pour mettre à jour en masse les paramètres de confidentialité des publications TikTok en fonction de critères de filtrage du nombre de vues.

:::info
Ce script est uniquement applicable aux comptes TikTok.
:::

## Étapes

1. Sélectionnez les appareils sur lesquels exécuter le script.
2. Cliquez sur `Scripts` > `Paramètres de Confidentialité`.
3. Configurez les paramètres de la tâche :
    - **Paramètres de filtrage des publications** :
      - **Nombre de vues maximal** : Entrez le nombre de vues maximal pour filtrer les publications. Seules les publications avec un nombre de vues ≤ ce nombre seront mises à jour.
      - **Nombre maximal de publications à traiter** : Entrez le nombre maximal de publications à traiter. Définir à 0 signifie aucune limite de traitement.
    - **Paramètres de confidentialité** :
      - **Qui peut voir cette vidéo** : Sélectionnez une option :
        - `Tout le monde` : Tout le monde peut voir la vidéo
        - `Amis` : Seuls les amis peuvent voir la vidéo
        - `Moi uniquement` : Seul vous-même pouvez voir la vidéo
    - **Paramètres de commutateurs** :
      - **Autoriser les commentaires** : Active ou désactive les commentaires sur les publications.
      - **Autoriser la réutilisation du contenu** : Active ou désactive la réutilisation du contenu, y compris les duos, les points, les stickers et l'ajout à l'histoire.
4. Cliquez sur `Démarrer le script` pour commencer l'exécution.

## Fonctionnement

Le script suit le flux de travail suivant :

1. Accède à l'onglet Videos
2. Scanne les publications, analyse le nombre de vues
3. Trouve les publications répondant aux critères (vues ≤ max_views)
4. Accède à la page de détails de la publication
5. Clique sur le bouton "Privacy settings"
6. Attend l'apparition de la fenêtre contextuelle des paramètres
7. Définit les autorisations de visionnage selon votre choix
8. Bascule intelligemment le commutateur "Allow comments"
9. Bascule intelligemment le commutateur "Allow reuse"
10. Retourne à la liste, continue à traiter la publication suivante
11. Répète les étapes ci-dessus jusqu'à ce que toutes les publications correspondantes soient traitées

## Remarques

- Le script de paramètres de confidentialité peut ne pas toujours réussir ; réessayez les tâches échouées si nécessaire.
- Le script ne traite que les publications répondant aux critères de filtrage du nombre de vues.
- Les modifications sont appliquées immédiatement et ne peuvent pas être annulées via le script.
- Avant d'exécuter le script sur plusieurs publications, assurez-vous de vérifier vos paramètres de confidentialité.
