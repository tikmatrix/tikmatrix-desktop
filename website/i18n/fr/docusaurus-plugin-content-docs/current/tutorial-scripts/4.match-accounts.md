# Associer les comptes

Le script d'association de comptes sert à identifier les comptes depuis les téléphones et à les ajouter à la liste des comptes.

## Étapes

1. Sélectionnez les appareils pour lesquels vous souhaitez associer les comptes.
2. Cliquez sur le bouton `Scripts` - `Associer les comptes`.
3. Cliquez sur le bouton `Démarrer` pour lancer la tâche d'association de comptes.
4. Le script identifiera automatiquement les comptes depuis les téléphones et les ajoutera à la liste des comptes.

## Notes

* Le script d'association de comptes n'est pas fiable à 100 %. Si certains comptes ne sont pas identifiés, vous devrez peut-être exécuter le script plusieurs fois.
