# Soporte para aplicaciones clonadas

TikMatrix e IgMatrix admiten ejecutar múltiples aplicaciones clonadas (por ejemplo, GB TikTok, GB Instagram, Instagram Plus u otras versiones modificadas) en el mismo dispositivo, cada aplicación con su propio nombre de paquete independiente. Esta función te permite gestionar múltiples cuentas de manera más eficiente mediante el uso de diferentes instancias de aplicaciones.

:::danger Advertencia de riesgo
El uso de aplicaciones clonadas puede violar los términos de servicio de TikTok/Instagram y puede resultar en la prohibición o restricción de cuentas. Solo proporcionamos soporte técnico para esta función, **no somos responsables** de ningún problema de cuenta causado por el uso de aplicaciones clonadas, incluidas prohibiciones, restricciones u otras sanciones. Asume el riesgo por tu cuenta.
:::

## ¿Qué es un nombre de paquete?

Cada aplicación Android tiene un identificador único llamado **nombre de paquete**. Las aplicaciones oficiales usan nombres de paquete estándar:

- TikTok: `com.zhiliaoapp.musically`
- Instagram: `com.instagram.android`

Las aplicaciones clonadas usan diferentes nombres de paquete, por ejemplo:

- GB TikTok: `com.zhiliaoapp.musically.gb`
- GB Instagram: `com.instagram.gb.android`
- Instagram Plus: `com.instagram.plus.android`

TikMatrix/IgMatrix puede detectar automáticamente y usar el nombre de paquete correcto para cada cuenta.

## Configurar aplicaciones clonadas

### Paso 1: Instalar aplicación clonada

1. Instala la aplicación clonada preferida en tu dispositivo o teléfono en la nube.
2. Asegúrate de que cada aplicación clonada tenga un nombre de paquete único para evitar conflictos.
3. Inicia sesión con diferentes cuentas en cada instancia de la aplicación.

### Paso 2: Configurar nombre de paquete

Tienes dos formas de configurar el nombre de paquete:

#### Método A: Configuración a nivel de cuenta (Recomendado)

Establece un nombre de paquete específico para cada cuenta:

1. Abre **Cuentas** en la navegación izquierda.
2. Haz clic en una cuenta para editarla, o haz clic en **Agregar** para crear una nueva cuenta.
3. En el campo **Nombre de paquete**, puedes:
   - **Ingresar manualmente** el nombre de paquete de la aplicación clonada (por ejemplo `com.instagram.gb.android`)
   - **Detección automática** Haz clic en el botón **Detectar nombre de paquete** (necesitas seleccionar el dispositivo primero)
4. Haz clic en **Guardar**.

:::tip Detección automática de nombre de paquete
El botón **Detectar nombre de paquete** puede detectar automáticamente la aplicación actualmente abierta en el dispositivo seleccionado. Esta es la forma más fácil de obtener el nombre de paquete correcto:

1. Selecciona un dispositivo para la cuenta
2. Abre la aplicación clonada en ese dispositivo
3. Haz clic en el botón **Detectar nombre de paquete**
4. El nombre de paquete se completará automáticamente

Esta función funciona para cualquier aplicación Android, incluidas aplicaciones clonadas de TikTok/Instagram, aplicaciones App Cloner y aplicaciones NomixCloner.
:::

Cuando ejecutes scripts para esta cuenta, TikMatrix/IgMatrix usará automáticamente el nombre de paquete especificado.

#### Método B: Valor predeterminado global del sistema

Establece un nombre de paquete predeterminado para todas las cuentas que no tienen un nombre de paquete específico:

1. Abre **Configuración** en la navegación izquierda.
2. Encuentra el campo **Nombre de paquete de TikTok** o **Nombre de paquete de Instagram**.
3. Ingresa el nombre de paquete predeterminado.
4. Haz clic en **Guardar**.

:::tip Prioridad
La configuración a nivel de cuenta siempre anulará el valor predeterminado global del sistema. Si una cuenta tiene configurado su propio nombre de paquete, se usará ese nombre de paquete en lugar del valor predeterminado.
:::

## Cómo funciona

Cuando ejecutas cualquier script de automatización:

1. TikMatrix/IgMatrix verifica si la cuenta objetivo tiene configurado un nombre de paquete personalizado.
2. Si lo tiene, usa ese nombre de paquete en todas las operaciones de automatización de UI.
3. Si no lo tiene, recurre al nombre de paquete predeterminado global del sistema.
4. Todos los ID de recursos y consultas XPath se formatearán automáticamente usando el nombre de paquete correcto.

## Mejores prácticas

### 1. Usar nombres de usuario descriptivos

Como puedes tener múltiples cuentas en diferentes aplicaciones clonadas, usa nombres de usuario descriptivos para identificar a qué aplicación pertenece cada cuenta:

```text
username_gb      → GB TikTok/Instagram
username_plus    → TikTok/Instagram Plus
username         → Aplicación oficial
```

### 2. Organizar cuentas por grupo

Crea grupos de dispositivos según el tipo de aplicación clonada:

- **Grupo A**: Todas las cuentas que usan aplicaciones GB
- **Grupo B**: Todas las cuentas que usan aplicaciones Plus
- **Grupo C**: Todas las cuentas que usan aplicaciones oficiales

### 3. Probar antes de operaciones por lotes

Antes de ejecutar scripts en todas las cuentas:

1. Primero prueba con una sola cuenta
2. Verifica que se esté usando la aplicación correcta
3. Confirma que todas las operaciones funcionan según lo esperado

### 4. Mantener la consistencia del nombre de paquete

Una vez que hayas establecido un nombre de paquete para una cuenta, no lo cambies a menos que cambies a una aplicación clonada diferente. Mantener la consistencia ayuda a evitar confusión y errores.

## Solución de problemas

### El script abre la aplicación incorrecta

**Problema**: El script abre la aplicación incorrecta en lugar de la aplicación clonada.

**Solución**:

1. Verifica que el campo **Nombre de paquete** de la cuenta esté configurado correctamente
2. Verifica que la aplicación clonada esté instalada y el nombre de paquete coincida
3. Usa el botón **Detectar nombre de paquete** para obtener automáticamente el nombre de paquete correcto
4. Si la aplicación se instaló recientemente, reinicia el dispositivo

### No se encuentran elementos de UI

**Problema**: El script reporta errores de "elemento no encontrado".

**Solución**:

1. Diferentes aplicaciones clonadas pueden tener diseños de UI ligeramente diferentes
2. Verifica que la versión de la aplicación clonada sea compatible
3. Intenta actualizar a la última versión de TikMatrix/IgMatrix
4. Si el problema persiste, contacta con soporte técnico

### Múltiples cuentas en la misma aplicación

**Problema**: Quieres cambiar entre múltiples cuentas en la misma instancia de aplicación.

**Solución**:

- TikTok/Instagram admite nativamente múltiples inicios de sesión de cuenta
- Usa la función **Cambiar cuenta** de TikMatrix/IgMatrix
- Establece el mismo nombre de paquete para todas las cuentas en esa instancia de aplicación
- TikMatrix/IgMatrix manejará automáticamente el cambio de cuenta

### El botón Detectar nombre de paquete no funciona

**Problema**: El botón **Detectar nombre de paquete** no funciona o muestra un error.

**Solución**:

1. Asegúrate de haber seleccionado primero un dispositivo
2. Asegúrate de que la aplicación esté actualmente abierta en el dispositivo
3. Verifica que la conexión del dispositivo sea estable
4. Intenta volver a abrir la aplicación y hacer clic en detectar

## Encontrar nombre de paquete

Si deseas verificar manualmente el nombre de paquete en el dispositivo:

### Usar comando ADB

```bash
# Listar todos los paquetes relacionados con TikTok
adb shell pm list packages | grep musically

# Listar todos los paquetes relacionados con Instagram
adb shell pm list packages | grep instagram
```

### Usar función de detección automática

La forma más fácil es usar el botón integrado **Detectar nombre de paquete**:

1. Abre el cuadro de diálogo de edición de cuenta
2. Selecciona el dispositivo
3. Abre la aplicación clonada en el dispositivo
4. Haz clic en **Detectar nombre de paquete**
5. El nombre de paquete se detectará y completará automáticamente

Este método funciona para cualquier aplicación Android y es el método recomendado.

## Preguntas frecuentes

**P: ¿Puedo usar múltiples aplicaciones clonadas en el mismo dispositivo?**

R: Sí, siempre que cada aplicación clonada tenga un nombre de paquete único, puedes ejecutar múltiples instancias de aplicaciones en el mismo dispositivo.

**P: ¿Necesito reinstalar TikMatrix/IgMatrix?**

R: No, esta función ya está integrada en la última versión. Solo actualiza a la última versión y configura el nombre de paquete.

**P: ¿Seguirán funcionando mis scripts de automatización existentes?**

R: Sí, todos los scripts existentes son totalmente compatibles. Las cuentas sin nombre de paquete personalizado usarán el nombre de paquete predeterminado.

**P: ¿Se admiten tanto TikTok como Instagram?**

R: Sí, tanto TikMatrix como IgMatrix admiten completamente detección y automatización de aplicaciones clonadas.

**P: ¿Cómo funciona la función de detección automática?**

R: Cuando haces clic en **Detectar nombre de paquete**, TikMatrix/IgMatrix verifica la aplicación actualmente en primer plano en el dispositivo seleccionado y obtiene automáticamente su nombre de paquete.

**P: ¿Puedo usar aplicaciones creadas con App Cloner o NomixCloner?**

R: Sí, la función de nombre de paquete funciona con cualquier herramienta de clonación de aplicaciones Android, incluidas App Cloner y NomixCloner. Usa la función de detección automática para obtener fácilmente el nombre de paquete correcto.
