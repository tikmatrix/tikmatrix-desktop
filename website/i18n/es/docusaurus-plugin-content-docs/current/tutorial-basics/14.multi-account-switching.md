# Cambio de múltiples cuentas

Tanto TikTok como Instagram admiten iniciar sesión con múltiples cuentas en el mismo dispositivo, esta es una función nativa de la aplicación oficial. TikMatrix e IgMatrix proporcionan una función de cambio automático de cuenta que puede ejecutar automáticamente tareas en todas las cuentas que han iniciado sesión, sin necesidad de intervención manual.

## Scripts soportados

Los siguientes scripts admiten la función de cambio automático de cuenta:

- **Profile (Ver perfil)** — Ver información del perfil de la cuenta
- **Account Warmup (Calentamiento de cuenta)** — Calentar cuenta a través de actividad natural
- **Post (Publicar contenido)** — Publicar publicaciones
- **Mass DM (Mensaje masivo)** — Enviar mensajes masivos
- **Follow Back (Seguir de vuelta)** — Seguir de vuelta a usuarios que te siguen
- **Unfollow All (Dejar de seguir a todos)** — Dejar de seguir todas las cuentas
- **Delete Post (Eliminar publicación)** — Eliminar contenido publicado
- **Boost Users (Promocionar usuarios)** — Aumentar la interacción en la página principal del usuario
- **Boost Posts (Promocionar publicaciones)** — Aumentar la interacción en publicaciones
- **Boost Lives (Promocionar transmisiones en vivo)** — Aumentar la interacción en salas de transmisión en vivo
- **Mass Comment (Comentario masivo)** — Publicar comentarios masivos
- **Boost Comments (Promocionar comentarios)** — Aumentar la interacción en comentarios
- **Super Marketing (Súper marketing)** — Automatización de marketing avanzado

## Configuración inicial: Coincidir cuentas

Antes de usar la función de cambio de múltiples cuentas, primero debes coincidir las cuentas del dispositivo.

1. Abre la pestaña **Scripts** en la navegación izquierda.
2. Encuentra y ejecuta el script **Coincidir cuenta**.
3. Este script escaneará las cuentas que han iniciado sesión en el dispositivo y las agregará automáticamente a la lista de cuentas.
4. Después de que se complete la ejecución del script, haz clic en el botón **Cuentas** para verificar si las cuentas del dispositivo se agregaron con éxito.

## Probar cambio de cuenta

Después de ejecutar el script **Coincidir cuenta**, puedes probar la función de cambio de cuenta:

1. Ejecuta el script **Cambiar cuenta** en la pestaña de scripts.
2. Este script no tiene ninguna otra lógica — su único propósito es verificar si el proceso de cambio de cuenta funciona normalmente.
3. Si el script se ejecuta con éxito, significa que la función de cambio de cuenta está lista.

## Usar modo de múltiples cuentas

Después de completar la configuración inicial y las pruebas:

1. Al ejecutar cualquier script soportado, habilita el **Modo de múltiples cuentas** en la configuración del script.
2. El sistema creará tareas independientes para cada cuenta en la lista de cuentas.
3. Las tareas se pondrán en cola y se ejecutarán en orden — el sistema cambiará automáticamente a la cuenta objetivo y ejecutará la tarea de automatización.
4. No se requiere ninguna intervención manual.

## Cuándo volver a ejecutar Coincidir cuenta

Los scripts **Coincidir cuenta** y **Cambiar cuenta** generalmente solo necesitan ejecutarse una vez durante la configuración inicial. Sin embargo, debes volver a ejecutar **Coincidir cuenta** en las siguientes situaciones:

- Has iniciado sesión con una nueva cuenta en el dispositivo
- Has cerrado sesión de una cuenta existente
- Las cuentas en el dispositivo han cambiado

Pasos para actualizar la lista de cuentas:

1. Limpia manualmente la lista de cuentas en la página **Cuentas**.
2. Vuelve a ejecutar el script **Coincidir cuenta** para detectar las cuentas actuales en el dispositivo.

---

La función de cambio de múltiples cuentas elimina la necesidad de cambiar manualmente de cuenta, permitiéndote escalar fácilmente la automatización entre todas las cuentas en un solo dispositivo.
