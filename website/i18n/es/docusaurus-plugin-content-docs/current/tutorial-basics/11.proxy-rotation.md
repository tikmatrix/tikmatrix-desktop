# Tutorial de rotación de proxy simple

La rotación de proxy puede cambiar automáticamente a una nueva IP para el teléfono antes de que comience el script. Esto hace que cambiar de cuenta o iniciar sesión sea más seguro y puede reducir las posibilidades de control de riesgo.

## Preparación

- Necesitas un servicio proxy que admita la función de "cambio de IP" o "rotación de IP".
- Solicita el enlace de rotación al proveedor de servicios (también puede llamarse *URL de rotación*, *URL de actualización*, *URL de cambio de IP*).
- Si el proveedor de servicios requiere contraseña adicional o token, prepáralo con anticipación.

## Guardar enlace de rotación para el dispositivo

1. En el escritorio, abre **Gestionar dispositivos** y cambia a la vista de tabla.
2. Encuentra el dispositivo que necesita cambiar IP, haz clic en **Configurar** en la columna *URL de rotación de proxy*.
3. Completa según las instrucciones:
   - **URL de rotación de proxy**: Pega el enlace de rotación proporcionado por el proveedor de servicios. TikMatrix lo accederá automáticamente cuando necesite cambiar IP.
   - **Método de solicitud**: Mantén `GET`, a menos que el proveedor de servicios especifique explícitamente cambiar a `POST`.
   - **Encabezados**, **Cuerpo**: Generalmente déjalos vacíos. Si el proveedor de servicios proporciona un ejemplo, cópialo tal cual en el cuadro de entrada.
   - **Tiempo de espera**: Tiempo de espera para la respuesta del proveedor de servicios, 10 segundos por defecto es suficiente.
4. Haz clic en **Guardar**. La configuración se guardará automáticamente sin necesidad de modificar ningún archivo manualmente.

Después de guardar, la fila del dispositivo mostrará el resultado de la última rotación: verde representa éxito, gris significa que aún no se ha ejecutado, rojo significa que el proveedor de servicios devolvió un error.

## Verificar si la rotación es efectiva

1. En la misma fila, haz clic en **Probar rotación de proxy**.
2. TikMatrix accederá inmediatamente al enlace de rotación.  
   - Si tiene éxito, aparecerá un mensaje de éxito y se actualizará el tiempo de "Última rotación".  
   - Si falla, registra el mensaje, verifica que el enlace y la contraseña sean correctos e inténtalo de nuevo.

Consejo: Muchos proxies limitan la frecuencia de cambio de IP. Si aparece un mensaje de falla, espera unos segundos antes de reintentar.

## Usar rotación en scripts

1. Al ejecutar un script, primero marca **Rotación de proxy**, TikMatrix cambiará la IP para cada dispositivo en orden antes de abrir TikTok.
2. Puedes hacer clic en el botón de registro a la derecha del dispositivo para ver información detallada de cada rotación.

## Preguntas frecuentes

- **¿Todos los scripts necesitan rotación?** No. Solo es necesario cuando quieres que TikTok vea una nueva IP, por ejemplo, antes de cambiar frecuentemente de cuenta.
- **¿Dónde se guarda la configuración?** TikMatrix la gestiona automáticamente. Cada dispositivo solo necesita configurarse una vez.
- **¿Qué pasa si el enlace parece muy largo y complejo?** Siempre que se pueda abrir en el navegador, TikMatrix también puede usarlo.
- **¿Todavía no puedes resolverlo?** Envía las instrucciones del proveedor de servicios y el mensaje de error al servicio al cliente de TikMatrix, te ayudaremos a solucionarlo juntos.
