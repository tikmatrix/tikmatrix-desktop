# Configurar proxy

Puedes configurar una dirección IP proxy independiente para cada teléfono.

> **Descargo de responsabilidad:** Todos los sitios web recomendados en esta página provienen de Internet y no están relacionados con este sitio. Por favor verifica su efectividad antes de usarlos. No proporcionamos ninguna garantía.

## 1. Configurar proxy VPS residencial (si compraste una IP proxy, puedes omitir este paso)

1. Compra un VPS de proxy IP residencial del país objetivo (preferiblemente IP nativa y VPS de doble ISP).
2. Usa [https://github.com/MHSanaei/3x-ui](https://github.com/MHSanaei/3x-ui) para configurar rápidamente una VPN en el VPS:

    ```shell
    bash <(curl -Ls https://raw.githubusercontent.com/MHSanaei/3x-ui/refs/tags/v2.6.0/install.sh)
    # Sigue las instrucciones para completar la configuración de VPN.
    ```

3. Después de la instalación, accede a la interfaz web x-ui. Puedes encontrar la URL y credenciales de inicio de sesión en la salida del terminal después de la instalación.
4. Crea un nuevo usuario en x-ui, establece el protocolo en `SOCKS5` y configura el puerto y otras opciones según sea necesario.
5. Después de crear el usuario, verás la IP y puerto del proxy en la interfaz x-ui.
6. Usa la IP y puerto del proxy en la aplicación NekoBox en el teléfono.

## 3. Habilitar proxy en el teléfono

1. Descarga e instala el cliente [NekoBoxForAndroid](https://github.com/MatsuriDayo/NekoBoxForAndroid) APK en el teléfono.
2. Abre la aplicación NekoBox, haz clic en el botón `+` para agregar un nuevo proxy.
3. Selecciona `SOCKS5` u otro tipo de proxy.
4. Ingresa la IP y puerto del proxy, luego haz clic en `Guardar`.
5. Haz clic en el botón `Iniciar` para habilitar el proxy.
6. Después de habilitar el proxy, puedes verificar el estado de conexión en la aplicación NekoBox.

### Nota para usuarios en China

Si estás en China, el protocolo `SOCKS5` puede no funcionar correctamente. Necesitas agregar un proxy cifrado (`VLESS`) como proxy frontal. Puedes usar las siguientes herramientas:

1. Agrega un proxy `VLESS` en la aplicación NekoBox.
2. Agrega un proxy `SOCKS5` en la aplicación NekoBox.
3. Agrega un proxy en `Cadena` en la aplicación NekoBox. Selecciona el proxy `VLESS` como proxy frontal y el proxy `SOCKS5` como proxy backend.
