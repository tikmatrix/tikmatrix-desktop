# Instalar Douyin/TikTok

Recomendamos descargar las siguientes versiones especificadas, ya que las versiones más recientes pueden no ser compatibles. El software se actualiza periódicamente, y las versiones soportadas se anunciarán en el grupo después de cada actualización.

## Descargar paquete de instalación de Douyin/TikTok

* Versión internacional: [com.zhiliaoapp.musically](https://apkpure.com/tiktok-musically-2024/com.zhiliaoapp.musically)
* Versión asiática: [com.ss.android.ugc.trill](https://apkpure.com/tiktok/com.ss.android.ugc.trill)

## Pasos de instalación

1. Descarga el paquete APK desde los enlaces anteriores.
2. Selecciona todos los dispositivos.
3. Haz clic en `General` - `Instalar APK` - Selecciona el paquete APK descargado.
4. Espera a que se complete la instalación (puede tomar de 1-3 minutos por dispositivo, dependiendo del modelo del teléfono; por favor ten paciencia y no hagas clic repetidamente en el botón "Instalar APK").
5. Después de completar la instalación, necesitas hacer clic en el botón `Configuración` para verificar si la versión de TikTok seleccionada coincide con la versión que instalaste.
6. Haz clic en el botón `Abrir TikTok` para verificar si la aplicación TikTok se instaló correctamente.
