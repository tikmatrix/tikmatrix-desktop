# Inicializar dispositivo

:::warning
Esta aplicación se instala a través del canal oficial de depuración ADB de Android para control automatizado de dispositivos. Algunos dispositivos pueden preguntar si deseas enviar muestras de aplicaciones desconocidas a Google, esto es un aviso de seguridad del sistema Android, no está relacionado con la seguridad de la aplicación y no afectará el uso normal. Haz clic en "Don't send" para completar la instalación.
:::

El APK de TikMatrix es el agente móvil requerido para tareas automatizadas. **Cuando conectas un teléfono a TikMatrix por primera vez, el sistema inicializará automáticamente tu dispositivo e instalará la aplicación TikMatrix** - sin necesidad de operación manual.

Durante el proceso de inicialización automática, TikMatrix instalará dos archivos APK: TikMatrix y TikMatrix Test (sin interfaz). Si tu teléfono solicita autorización de instalación, simplemente haz clic en "Permitir" (especialmente importante en dispositivos Xiaomi).

## Instalación automática vs manual

**Instalación automática (Recomendado)**: Por defecto, TikMatrix inicializará automáticamente el dispositivo cuando lo conectes por primera vez. Esta es la forma más simple y conveniente.

**Instalación/actualización manual**: Úsalo solo en casos especiales o cuando necesites actualizar.

## Flujo de inicialización automática

Cuando conectas un nuevo dispositivo a TikMatrix:

1. **Detección de dispositivo**: TikMatrix detecta automáticamente los dispositivos conectados.
2. **Instalación automática**: El sistema instalará automáticamente la aplicación TikMatrix sin intervención del usuario.
3. **Confirmación de autorización**: Si el dispositivo solicita permisos de instalación, simplemente haz clic en "Permitir".
4. **Instalación completada**: El proceso de inicialización se completa en aproximadamente 10 segundos.
5. **Listo para usar**: Tu dispositivo ahora está preparado para ejecutar tareas automatizadas.

Este flujo automatizado no requiere configuración manual en la mayoría de los casos.

## Instalación/actualización manual

Para instalar o actualizar manualmente el APK de TikMatrix, desactiva el interruptor `Auto despertar Agent` en la configuración. Luego sigue estos pasos:

1. Selecciona tu teléfono y haz clic en `General` > `Inicializar Agent`.
2. TikMatrix instalará automáticamente dos archivos APK en el teléfono.
3. Si se solicita, haz clic en "Permitir" para autorizar la instalación.
4. Si Google Play Protect lo bloquea, haz clic en "Instalar de todos modos" para continuar.
5. Espera a que se complete la instalación (aproximadamente 10 segundos).
6. Activa el interruptor `Auto despertar Agent` para habilitar el despertar automático de la aplicación cuando se conecte el teléfono.
7. Haz clic en `General` > `Abrir Agent` para probar la instalación. Si la aplicación no se inicia, contacta con soporte técnico.
