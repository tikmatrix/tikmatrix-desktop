# Crear grupo

Un **grupo** es una colección de dispositivos que permite una gestión y operaciones unificadas más convenientes para múltiples dispositivos.

## Pasos

1. Haz clic en el botón `Agregar grupo` en la barra lateral izquierda.
2. Ingresa el nombre del grupo.
3. Presiona la tecla `Enter` para guardar el grupo.
4. Puedes seleccionar dispositivos para moverlos a ese grupo.
