# Control sincronizado

El control sincronizado se utiliza para controlar operaciones sincronizadas de múltiples dispositivos.

## Pasos

1. Haz clic para abrir la pantalla de 1 teléfono.
2. Selecciona múltiples dispositivos en el lado izquierdo.
3. Todas las operaciones de pantalla se sincronizarán con múltiples dispositivos.
