# Introducción al Tutorial

Conozca en menos de 5 minutos las funciones principales y métodos de uso de **TikMatrix** e **IgMatrix**.

## Introducción al Software

**TikMatrix** es un software de escritorio para PC que ejecuta automáticamente tareas de TikTok en dispositivos Android.  
**IgMatrix** es un software de escritorio para PC que ejecuta automáticamente tareas de Instagram en dispositivos Android.

**Nota:** **TikMatrix** e **IgMatrix** son dos softwares independientes que no pueden ejecutarse simultáneamente en la misma computadora.

## Características

- **Cambio de múltiples cuentas**: El mismo dispositivo puede iniciar sesión en múltiples cuentas de TikTok/Instagram y cambiar automáticamente para ejecutar tareas.
- **Soporte para aplicaciones clonadas**: Compatible con aplicaciones clonadas de TikTok/Instagram con nombres de paquete personalizados, diferentes cuentas pueden vincularse a diferentes aplicaciones clonadas y abrir automáticamente la aplicación objetivo.
- ~~**Registro automático**~~: **[Fuera de servicio]**
- **Inicio de sesión automático**: Instagram admite inicio de sesión automático con 2FA; TikTok aún requiere intervención manual para algunas verificaciones (como verificación gráfica, verificación de correo electrónico).
- **Completar perfil automáticamente**: Compatible con la configuración de avatar, apodo, biografía y nombre de usuario.
- **Reconocimiento automático**: Reconocer e importar con un clic las cuentas de TikTok/Instagram que ya han iniciado sesión en el teléfono.
- **Calentamiento automático de cuentas**: Compatible con búsqueda de palabras clave, visualización de videos, me gusta, seguir, comentar y guardar, etc.
- **Publicación automática**: Compatible con la publicación de videos o imágenes (puede configurar el orden y la cantidad en lote), compatible con música personalizada, e incluye título, descripción, hashtags y enlaces de productos.
- **Eliminación automática**: Eliminar publicaciones de bajo rendimiento según reproducciones o reglas especificadas.
- **Super Marketing**: Simula rutas de visita reales (visitar página principal, navegar publicaciones, seguir/me gusta/comentar/guardar/compartir/mensaje directo aleatoriamente), compatible con OpenAI o API compatible para generar contenido personalizado de mensajes directos y comentarios.
- **Impulso masivo (transmisión en vivo)**: Dar me gusta y comentar masivamente en transmisiones en vivo.
- **Recolección de usuarios**: Recopilar nombres de usuario objetivo a través de palabras clave, seguidores o listas de seguidos.
- **Seguimiento masivo de vuelta**: Seguir seguidores masivamente y enviar mensajes de bienvenida.
- **Programación de tareas**: Programación con precisión de minutos, compatible con reintentos automáticos, ejecución con intervalos y configuración de máximo de reintentos.
- **Múltiples dispositivos en paralelo**: Una computadora puede conectarse y administrar simultáneamente múltiples dispositivos Android ejecutando tareas en paralelo.
- **Espejo de pantalla**: Compatible con proyección de pantalla de múltiples dispositivos en Windows / macOS, y control mediante mouse y teclado.
- **Importar / Exportar cuentas**: Compatible con importación y exportación masiva en formato Excel.
- **Soporte multiidioma**: El software de computadora admite inglés, ruso y chino. Nota: Algunas funciones requieren que el idioma de la aplicación TikTok/Instagram en el teléfono sea inglés para obtener la mejor compatibilidad.
- **Soporte de tickets**: Proporciona servicio de tickets con atención humana para resolver problemas durante el uso.

## Qué necesita

### Teléfono Android

Puede usar cualquier dispositivo Android para ejecutar TikMatrix, incluyendo:

- **Dispositivos Android reales**
- **[Dispositivos Android de placa base](http://www.niaozun.shop?cid=934ec2fe)**
- **[Teléfonos en la nube](https://www.geelark.cn?invite_code=XHY6a8)**
- **Emuladores**
- **Máquinas virtuales**

Los dispositivos Android requieren al menos API 21 (Android 5.0) y no necesitan permisos root. Habilite la depuración USB (opciones de desarrollador) en el dispositivo. En algunos modelos (por ejemplo, Xiaomi), también puede ser necesario habilitar "Depuración USB (configuración de seguridad)" para admitir control de teclado y mouse.

### Computadora

#### Windows: Windows 10 o superior

#### Mac: macOS 15.0 o superior (compatible con chips Apple Silicon serie M y chips Intel)

## Instalación

1. Descargue el instalador más reciente desde la página [Descargar TikMatrix](https://tikmatrix.com/Download) o [Descargar IgMatrix](https://igmatrix.com/Download-IgMatrix).
2. Ejecute el instalador.
3. Espere a que se complete la instalación.
4. Abra la aplicación **TikMatrix** o **IgMatrix** en su computadora.
