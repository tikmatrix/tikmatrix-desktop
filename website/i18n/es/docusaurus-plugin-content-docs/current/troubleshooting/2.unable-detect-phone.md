# No se puede detectar el teléfono

## Pasos para solucionar problemas de detección del teléfono

### Primer paso: Verificar el estado de la conexión ADB

1. En TikMatrix, haga clic en `Configuración` > `Abrir terminal ADB`  
2. Ejecute el comando `adb devices` para verificar si el teléfono aparece en la lista

### Sistema Windows

```shell
.\adb.exe devices
```

### Sistema Mac/Linux

```shell
./adb devices
```

### Ejemplo de salida

```shell
List of devices attached
<ID del dispositivo>    <estado>
```

### Si el dispositivo no aparece en la lista de dispositivos adb

Siga los pasos del documento [Conectar dispositivos](../tutorial-basics/1.connect-to-devices.md) para solucionar el problema

### Si el estado del dispositivo es `unauthorized`

La posible causa es que el dispositivo no está autorizado. Por favor, toque `Permitir depuración USB` en el teléfono

### Si el estado del dispositivo es `offline`

La posible causa es que el cable USB está suelto. Vuelva a conectar el teléfono, desconecte el cable USB y vuelva a conectarlo

### Si el estado del dispositivo es `device`

El dispositivo ya está conectado correctamente. Si TikMatrix todavía no detecta el teléfono, verifique si TikMatrix está activado. La prueba gratuita también requiere activación en la página de autorización.

### Si el problema persiste

Contacte a nuestro equipo de soporte técnico a través de [Telegram](https://t.me/tikmatrix_agent_bot) y proporcione la siguiente información:

- Captura de pantalla de la salida del comando adb devices
- Los pasos que ya ha intentado
