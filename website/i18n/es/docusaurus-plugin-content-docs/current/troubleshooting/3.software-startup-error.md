# Error de inicio del software

## Pasos para solucionar errores comunes de inicio del software

### Paso 1: Verificar los requisitos del sistema

1. Asegúrese de que su sistema cumple con los requisitos mínimos:
   - Windows 10 o superior
   - Al menos 4GB de memoria
   - Al menos 1GB de espacio libre en disco

### Paso 2: Verificar Microsoft Visual C++ Redistributable

Por lo general, los errores `vcruntime140.dll no encontrado` o `La aplicación no se puede iniciar correctamente (0xc000007b)` se deben a que Microsoft Visual C++ Redistributable no está instalado.

1. Presione las teclas `Windows + R`, escriba `cmd` y presione Enter
2. Escriba `dir "C:\Windows\System32\vcruntime*.dll"` y presione Enter
3. Si hay salida, significa que está instalada la versión correspondiente de Visual C++ Redistributable. El nombre del archivo con el sufijo vcruntime generalmente incluye el número de versión, como vcruntime140.dll (correspondiente a 2015-2022).
4. Si no hay salida, descargue e instale Microsoft Visual C++ Redistributable: [Última versión de Microsoft Visual C++ Redistributable](https://learn.microsoft.com/en-us/cpp/windows/latest-supported-vc-redist?view=msvc-170)

### Paso 3: Verificar la hora del sistema

Un reloj de computadora inexacto causará un error "Agent Start Timeout":

1. Haga clic derecho en la visualización de la hora en la esquina inferior derecha de la pantalla
2. Seleccione "Ajustar fecha/hora"
3. Si aún no está habilitado, active "Establecer hora automáticamente"
4. Si la hora sigue siendo incorrecta, haga clic en "Sincronizar ahora" o configure manualmente la fecha y hora correctas
5. Reinicie TikMatrix después de corregir la hora

### Paso 4: Verificar los registros de la aplicación

1. Haga clic en el botón `Configuración` - `Abrir directorio de la aplicación`
2. Navegue a la carpeta `logs`
3. Revise la información de errores en el archivo de registro más reciente

### Si el problema persiste

### Paso 5: macOS — Agent InvalidStateError

En algunas máquinas macOS, el Agent puede no iniciar y mostrar el siguiente error:

```text
Agent start error InvalidStateError: The object is in an invalid state.
```

Posible causa:

- Generalmente se debe a incompatibilidad entre la aplicación y la versión actual del sistema macOS o hardware/firmware antiguo.

Solución / método de trabajo:

1. Actualice macOS a la última versión a través de "Configuración del sistema" > "General" > "Actualización de software". Después de actualizar, reinicie el Mac e inicie TikMatrix nuevamente.
2. Si no puede actualizar o el error persiste, intente ejecutar la aplicación en otro Mac con una versión más reciente de macOS.

Si el problema persiste después de intentar los métodos anteriores, recopile los registros de la aplicación y contáctenos a través de [Telegram](https://t.me/tikmatrix_agent_bot), adjuntando su versión de macOS y los archivos de registro.

Contacte a nuestro equipo de soporte a través de [Telegram](https://t.me/tikmatrix_agent_bot) y proporcione la siguiente información:

- Versión de Windows
- Archivos de registro de la aplicación
- Los pasos que ya ha intentado
