# Permission Denied - Unable to Save Draft Videos

## Resolver el problema de aviso de permiso denegado en la pantalla del teléfono

### Descripción del problema

Al ejecutar el script de publicación, puede aparecer el siguiente aviso en la pantalla del teléfono:

```text
Permission denied. Unable to save draft videos.
```

### Solución

Este problema se debe a que la aplicación de TikTok carece de permisos de almacenamiento. Siga estos pasos para resolverlo:

#### Paso 1: Tocar manualmente el botón de publicación

1. Encuentre y toque el botón **Publicar** en la pantalla del teléfono
2. Continúe con el proceso de publicación

#### Paso 2: Permitir permisos del sistema

1. Cuando aparezca el cuadro de diálogo de solicitud de permisos del sistema, toque **Permitir**
2. Asegúrese de otorgar los permisos de almacenamiento necesarios a la aplicación de TikTok

### Nota importante

:::info
Este paso de configuración de permisos solo necesita realizarse una vez. Una vez que se otorguen los permisos, las ejecuciones posteriores del script de publicación no volverán a presentar este problema.
:::

### Medidas preventivas

Para evitar este problema, se recomienda antes de usar TikMatrix por primera vez:

1. Abrir manualmente la aplicación de TikTok
2. Intentar grabar o guardar un borrador de video
3. Seleccionar **Permitir** al solicitar permisos
4. Esto permite preconfigurar los permisos necesarios
