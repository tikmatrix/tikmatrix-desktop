# Eliminar publicaciones

El script de eliminar publicaciones se utiliza para eliminar publicaciones según el número de vistas.

## Pasos

1. Seleccione los dispositivos en los que desea ejecutar el script.
2. Haga clic en `Scripts` > `Eliminar publicaciones`.
3. Configure los ajustes de la tarea:
    - **Número máximo de vistas**: Ingrese el número máximo de vistas de las publicaciones que desea eliminar.
4. Haga clic en `Iniciar Script` para comenzar la ejecución.

## Notas

- El script de eliminar publicaciones puede no tener siempre éxito; reintente las tareas fallidas si es necesario.
