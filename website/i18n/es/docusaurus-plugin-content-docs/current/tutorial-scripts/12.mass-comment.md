# Comentarios masivos

:::warning Aviso de descontinuación
Esta funcionalidad ha sido integrada en el [script de Super Marketing](./14.super-marketing.md). Por favor, use preferentemente el script "Super Marketing" para obtener capacidades de comentarios masivos e interacción de contenido más potentes y flexibles.
:::

Los comentarios masivos se utilizan para publicar comentarios de interacción en una gran cantidad de publicaciones de TikTok (como calentamiento de cuenta, promoción, operaciones de actividades). El sistema admite dos fuentes de mensajes: plantillas pregeneradas y generación en tiempo real, e intentará simular el ritmo de envío manual para reducir el riesgo de detección.

## Usar comentarios masivos en el script "Super Marketing"

1. Abra el script: Seleccione "Super Marketing".
2. Seleccione el tipo de fuente de datos como "Lista de enlaces de publicaciones".
3. Pegue enlaces de publicaciones por lotes o impórtelos desde un archivo de texto (uno por línea); después de importar, puede elegir "Agregar" o "Sobrescribir".
4. Seleccione la estrategia de uso de datos:

   - Estrategia de pool compartido: Todas las cuentas bot comentan en las mismas publicaciones.

   - Estrategia de consumo único: Cada cuenta bot solo comenta en publicaciones diferentes una vez, asegurando que cada publicación solo sea procesada una vez por una cuenta.

5. Establezca el número de comentarios para cada publicación (generalmente 1 vez para evitar comentarios duplicados).
6. En "Configuración de comentarios" configure las siguientes opciones:

   - Plantilla de comentarios pregenerados: Puede pegar mensajes de comentarios pregenerados de múltiples líneas o párrafos (cada línea o párrafo se considera un comentario candidato).

   - Orden de uso de mensajes: Seleccione "Aleatorio" o "En orden".

   - Aleatorio: Cada vez extrae aleatoriamente un mensaje de los mensajes candidatos.

   - En orden: Rota el uso en el orden de pegado/importación (adecuado para estrategias de eliminación de duplicados reproducibles).

   - Insertar emojis: Puede habilitar la inserción automática de emojis en los comentarios para aumentar la naturalidad.

   - Insertar número de dispositivo (opcional): Añadir número de dispositivo al final del comentario para rastrear la fuente (solo para depuración/auditoría, use con precaución).

   - Generar comentarios en tiempo real (opcional, ChatGPT API):

      - Una vez habilitado, el script puede llamar a ChatGPT o API similar, generar mensajes de comentarios únicos y relevantes al contexto basados en el título/descripción de la publicación o contexto objetivo en tiempo real, adecuado para escenarios que requieren una gran cantidad de comentarios únicos.

      - Nota: Habilitar la generación en tiempo real requiere proporcionar una clave API válida, y prestar atención a los límites de velocidad, costos y cumplimiento de privacidad (evitar enviar información sensible a servicios de terceros).

   - Dos formas comunes de garantizar comentarios únicos:

      1. Usar "En orden" para rotar plantillas de comentarios pregenerados (cada plantilla solo se usa una vez hasta que se agote).

      2. Usar ChatGPT para generar cada comentario en tiempo real (generación dinámica basada en el contenido de la publicación, mayor unicidad).

7. Configure otros parámetros del script (número de concurrencia, intervalo de tareas, pool de cuentas, etc.), confirme que todo esté correcto y ejecute el script.

:::tip Sobre el intervalo de tareas
Se recomienda establecer intervalos mínimos/máximos y habilitar intervalos aleatorios para evitar que todos los dispositivos ejecuten al mismo tiempo causando picos anormales, reduciendo así el riesgo de ser detectado o limitado por la plataforma.
:::

## Sugerencias de uso y notas

- Primero pruebe su estrategia de comentarios en un pequeño lote de publicaciones, confirme la calidad de los comentarios y la situación de procesamiento de la plataforma.
- El contenido de los comentarios debe ser conforme y diversificado, evitar enviar grandes cantidades del mismo contenido en poco tiempo.
- Si usa generación en tiempo real, monitoree estrictamente la cantidad de llamadas API y los costos, y asegúrese de no filtrar información sensible al exterior.
- Las tareas pueden fallar; si encuentra fallas, reintente o revise los registros, el sistema saltará automáticamente los objetivos ya completados.
