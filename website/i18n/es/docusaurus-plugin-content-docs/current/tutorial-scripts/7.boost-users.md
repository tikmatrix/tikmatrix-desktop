# Impulsar usuarios

:::warning Aviso de descontinuación
Esta funcionalidad ha sido integrada en el [script de Super Marketing](./14.super-marketing.md). Por favor, use preferentemente "Super Marketing" para obtener capacidades de impulso de usuarios más potentes y flexibles.
:::

La funcionalidad de impulsar usuarios aumenta la exposición e interacción de usuarios objetivo mediante operaciones masivas de seguir/dejar de seguir. El sistema intentará simular el comportamiento de usuarios reales para reducir el riesgo de detección, pero aún se recomienda seguir las reglas de la plataforma y controlar la frecuencia.

## Usar la funcionalidad de impulsar usuarios en el script "Super Marketing"

1. Abra el script: Seleccione "Super Marketing".
2. Seleccione el tipo de fuente de datos como "Lista de cuentas de usuario".
3. Pegue nombres de usuario por lotes o impórtelos desde un archivo de texto (uno por línea); después de importar, puede elegir "Agregar" o "Sobrescribir".
4. Seleccione la estrategia de uso de datos:

    - Estrategia de pool compartido: Todas las cuentas bot seguirán/dejarán de seguir los mismos objetivos.

    - Estrategia de consumo único: Cada cuenta bot solo procesará objetivos diferentes, asegurando que cada objetivo solo sea operado una vez por una cuenta.

5. En "Operaciones de usuario — Número de repeticiones" establezca el número de operaciones para cada objetivo (generalmente establecido en 1 vez).
6. Seleccione el tipo de operación: Seguir o Dejar de seguir. Puede configurar más en "Operaciones de usuario":

        - Forma de entrar al perfil del usuario: Seleccione "Buscar usuario" o "Abrir perfil directamente (Direct Open Profile)".

        - Enviar mensaje privado (opcional): Una vez habilitado, puede pegar mensajes privados pregenerados de múltiples líneas, cada línea o párrafo es un mensaje candidato; configurar el orden de uso de mensajes:

                - Aleatorio: Cada vez selecciona aleatoriamente un mensaje.
                - En orden: Usa en el orden de pegado/importación (puede continuar entre rondas de tareas).

            Además, puede elegir si insertar automáticamente emojis en los mensajes para aumentar la naturalidad.

7. Configure otros parámetros (concurrencia, retrasos, pool de cuentas), confirme que todo esté correcto y ejecute el script.

## Sugerencias de uso y notas

- Se recomienda probar primero en una pequeña cantidad de cuentas, observar si la estrategia y los parámetros activan el control de riesgos de la plataforma.
- Configure razonablemente los retrasos y la concurrencia, simule el comportamiento humano para reducir el riesgo.
- Durante la ejecución a largo plazo, verifique regularmente la salud de las cuentas, los registros de tareas y ajuste la estrategia según los resultados.
- Las operaciones de impulsar usuarios no son 100% exitosas, reintente si encuentra fallas o investigue el estado de las cuentas objetivo.
