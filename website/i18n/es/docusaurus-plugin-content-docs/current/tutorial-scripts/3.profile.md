# Perfil

El script de perfil se utiliza para configurar el perfil de las cuentas.

## Pasos

1. Seleccione los dispositivos para configurar el perfil.
2. Haga clic en el botón `Scripts` - `Perfil`.
3. Configure los ajustes de la tarea:
    - **Nombre**: Un nombre por línea.
    - **Nombre de usuario**: Un nombre de usuario por línea.
    - **Biografía**: Una biografía por línea (admite menciones de usuarios y saltos de línea).
      - Para agregar saltos de línea en la biografía, use `\n` en el texto. Por ejemplo: `Primera línea\nSegunda línea\nTercera línea`
    - **Ruta de avatar**: Ruta a la carpeta de imágenes de avatar.
    - **Orden de selección**: Elija cómo se selecciona el contenido para cada tarea:
      - **Aleatorio**: Selecciona opciones disponibles aleatoriamente para cada tarea (predeterminado).
      - **Secuencial**: Selecciona contenido en orden según el conteo de tareas. Use este modo para asegurar que cada cuenta obtenga información única.
4. Haga clic en el botón `Iniciar Script` para comenzar la tarea.

## Descripción del orden de selección

### Modo aleatorio

- Cada tarea selecciona contenido aleatoriamente (nombre, nombre de usuario, biografía, avatar)
- El mismo contenido puede reutilizarse en diferentes cuentas
- Adecuado para escenarios donde no se requiere garantizar la unicidad del contenido

### Modo secuencial

- Selecciona contenido en orden: la primera tarea usa el primer elemento, la segunda tarea usa el segundo elemento, y así sucesivamente
- Cuando se usan todos los elementos, vuelve al principio
- **Nota importante**: Para asegurar que cada cuenta obtenga información única, asegúrese de que la cantidad de contenido proporcionado sea mayor que el número total de cuentas en una sola ejecución por lotes
  - Ejemplo: Si ejecuta 100 cuentas, proporcione al menos 100 nombres, nombres de usuario, biografías e imágenes de avatar diferentes

## Notas

- Es mejor limpiar el álbum antes de configurar el perfil, ya que el script usará la primera foto del álbum como avatar.
