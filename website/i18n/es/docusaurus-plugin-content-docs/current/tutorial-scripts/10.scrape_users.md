# Recopilar usuarios

El script de recopilar usuarios se utiliza para recopilar nombres de usuario de TikTok. La funcionalidad de recopilación de usuarios admite tres modos: recopilar lista de seguidores de usuarios especificados, recopilar lista de seguidos de usuarios especificados o recopilar usuarios mediante búsqueda de palabras clave.

## Pasos

1. Seleccione los dispositivos en los que desea ejecutar el script. Si selecciona varios dispositivos, la tarea de recopilación se distribuirá uniformemente entre los dispositivos seleccionados.
2. Haga clic en `Scripts` > `Recopilar usuarios`.
3. Configure los ajustes de la tarea:
    - **Modo de recopilación**: Seleccione el modo de recopilación de usuarios.
      - **Recopilar mediante seguidores**: Recopilar lista de seguidores de cuentas especificadas.
      - **Recopilar mediante seguidos**: Recopilar lista de seguidos de cuentas especificadas.
      - **Recopilar mediante palabras clave**: Recopilar usuarios usando búsqueda de palabras clave.
    - **Nombre de usuario de cuenta objetivo**: Ingrese el nombre de usuario de la cuenta para recopilar usuarios, uno por línea (para modos de seguidores y seguidos).
    - **Palabras clave de búsqueda**: Ingrese palabras clave para buscar usuarios (para modo de palabras clave).
    - **Cantidad máxima de recopilación**: Establezca la cantidad máxima de usuarios a recopilar.

4. Haga clic en `Iniciar Script` para comenzar la ejecución.

## Notas

- Los datos de usuarios recopilados se guardarán en archivos según el modo de recopilación:
  - Modo seguidores: `exported_users_followers_tiktok_<nombre_usuario>_<cantidad>.txt`
  - Modo seguidos: `exported_users_following_tiktok_<nombre_usuario>_<cantidad>.txt`
  - Modo palabras clave: `exported_users_search_tiktok_<palabra_clave>_<cantidad>.txt`
- TikTok solo limita la cantidad de usuarios que se pueden ver, por lo que cada operación solo puede recuperar aproximadamente 50 registros.
