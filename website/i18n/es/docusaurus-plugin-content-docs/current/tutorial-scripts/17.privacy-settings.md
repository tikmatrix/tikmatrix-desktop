# Configuración de privacidad

El script de **Configuración de privacidad** se utiliza para actualizar masivamente la configuración de privacidad de publicaciones de TikTok según condiciones de filtro de número de vistas.

:::info
Este script solo es aplicable para cuentas de TikTok.
:::

## Pasos

1. Seleccione los dispositivos en los que desea ejecutar el script.
2. Haga clic en `Scripts` > `Configuración de privacidad`.
3. Configure los ajustes de la tarea:
    - **Parámetros de filtro de publicaciones**:
      - **Número máximo de vistas**: Ingrese el número máximo de vistas para filtrar publicaciones. Solo se actualizarán las publicaciones con número de vistas ≤ este número.
      - **Número máximo de publicaciones a procesar**: Ingrese el número máximo de publicaciones a procesar. Establecer en 0 significa sin límite de cantidad de procesamiento.
    - **Parámetros de configuración de privacidad**:
      - **Quién puede ver este video**: Seleccione una opción:
        - `Todos`: Cualquiera puede ver el video
        - `Amigos`: Solo amigos pueden ver el video
        - `Solo yo`: Solo usted mismo puede ver el video
    - **Parámetros de interruptor**:
      - **Permitir comentarios**: Habilitar o deshabilitar comentarios de publicación.
      - **Permitir reutilización de contenido**: Habilitar o deshabilitar reutilización de contenido, incluyendo duetos, costura, stickers y añadir a historia.
4. Haga clic en `Iniciar Script` para comenzar la ejecución.

## Cómo funciona

El script sigue el siguiente flujo de trabajo:

1. Entrar a la pestaña Videos
2. Escanear publicaciones, analizar número de vistas
3. Encontrar publicaciones que cumplan las condiciones (views ≤ max_views)
4. Entrar a la página de detalles de la publicación
5. Hacer clic en el botón "Privacy settings"
6. Esperar a que aparezca la ventana emergente de configuración
7. Establecer permisos de visualización según su elección
8. Cambiar inteligentemente el interruptor "Allow comments"
9. Cambiar inteligentemente el interruptor "Allow reuse"
10. Volver a la lista, continuar procesando la siguiente publicación
11. Repetir los pasos anteriores hasta procesar todas las publicaciones coincidentes

## Notas

- El script de configuración de privacidad puede no tener siempre éxito; reintente las tareas fallidas si es necesario.
- El script solo procesa publicaciones que cumplan las condiciones de filtro de número de vistas.
- Los cambios se aplican inmediatamente y no se pueden deshacer a través del script.
- Antes de ejecutar el script en múltiples publicaciones, asegúrese de verificar su configuración de privacidad.
