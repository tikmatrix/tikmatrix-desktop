# Iniciar sesión

El script de inicio de sesión se utiliza para iniciar sesión en cuentas de redes sociales.

## Pasos

1. Asegúrese de que las cuentas se han agregado a la lista de cuentas (haga clic en `Cuentas` para agregar cuentas) y seleccione el número de serie del teléfono en el que necesita iniciar sesión con esa cuenta.
2. Seleccione uno o varios dispositivos para ejecutar el script de inicio de sesión.
3. Haga clic en `Scripts` > `Iniciar sesión`.
4. Haga clic en `Iniciar Script` para comenzar la ejecución.

## Prioridad de inicio de sesión

El sistema utiliza la siguiente prioridad de inicio de sesión:

1. **Correo + Contraseña**: Si la cuenta tiene un correo configurado, se priorizará el inicio de sesión con correo.
2. **Nombre de usuario + Contraseña**: Si no hay correo configurado, se utilizará el nombre de usuario para iniciar sesión.

## Soporte 2FA (solo IgMatrix)

IgMatrix admite inicio de sesión automático con autenticación de dos factores (2FA):

### Formato de contraseña 2FA

Para usar 2FA, configure la contraseña en el siguiente formato:

```plaintext
tu_contraseña:tu_clave_2FA
```

**Ejemplo:**

```plaintext
mypassword123:JBSWY3DPEHPK3PXP
```

### Cómo funciona 2FA

1. **Análisis de contraseña**: El sistema detecta automáticamente si la contraseña contiene el separador dos puntos (`:`).
2. **Generación de código de verificación**: Cuando se requiere verificación 2FA, el sistema utiliza su clave para solicitar un código de verificación TOTP a nuestro servicio 2FA.
3. **Entrada automática**: El código de verificación de 6 dígitos generado se ingresa automáticamente en el cuadro de verificación 2FA de Instagram.
4. **Inicio de sesión sin interrupciones**: Todo el proceso 2FA se completa automáticamente sin intervención manual.

### Descripción de la clave 2FA

La clave 2FA suele ser una cadena de 32 caracteres codificada en Base32, que obtendrá al configurar 2FA en Instagram. Esta es la misma clave que utilizan aplicaciones de autenticación como Google Authenticator o Authy.

## Notas

- El inicio de sesión de TikTok no es 100% automático. Es posible que deba ingresar manualmente el código de verificación del correo o completar otros pasos de verificación. Tampoco podemos resolver los desafíos de inicio de sesión de TikTok (como captcha de imagen, verificación humana, etc.) ni el error: inicio de sesión demasiado frecuente.
- Instagram puede lograr un inicio de sesión 100% automático mediante 2FA. Para cuentas de Instagram con 2FA habilitado, asegúrese de usar el formato de clave correcto en la contraseña.
