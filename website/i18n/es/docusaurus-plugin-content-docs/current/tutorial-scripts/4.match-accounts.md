# Emparejar cuentas

El script de emparejar cuentas se utiliza para identificar cuentas del teléfono y agregarlas a la lista de cuentas.

## Pasos

1. Seleccione los dispositivos para emparejar cuentas.
2. Haga clic en el botón `Scripts` - `Emparejar cuentas`.
3. Haga clic en el botón `Iniciar` para comenzar la tarea de emparejamiento de cuentas.
4. El script identificará automáticamente las cuentas del teléfono y las agregará a la lista de cuentas.

## Notas

* El script de emparejar cuentas no tiene una tasa de éxito del 100%, si algunas cuentas no son identificadas, es posible que deba ejecutar el script varias veces.
