# Inizializza Dispositivo

:::warning
本Applica通过 Android 官方 ADB 调试通道Installa，用于DispositiviAutomatico化控制。某些Dispositivi可能会SuggerimentoVuoi将未知Applica样本发送给 Google，这Sì Android 系统的安全Suggerimento，与Applica安全性无关，不会影响正常Utilizzo。点击“Don't send”即可CompletatoInstalla。
:::

TikMatrix APKSìAutomatico化Attività所需的移动Proxy。**当您首volte将TelefonoConnessione到TikMatrix时，系统会Automatico初始化您的Dispositivi并InstallaTikMatrixApplica** - 无需ManualeOperazioni。

在Automatico初始化过程中，TikMatrix会Installa两个APK文件：TikMatrix和TikMatrix Test（无界面）。如果您的TelefonoSuggerimentoInstallaAutorizzazione，只需点击"允许"（在小米Dispositivi上尤其Importante）。

## AutomaticoInstalla vs ManualeInstalla

**AutomaticoInstalla（Consigliati）**：Predefinito情况下，TikMatrix在首volteConnetti Dispositivi时会AutomaticoInizializza Dispositivo。这Sì最简单便捷的方法。

**ManualeInstalla/Aggiorna**：仅在特殊情况下或需要Aggiorna时Utilizzo。

## Automatico初始化流程

当您Connessione新Dispositivi到TikMatrix时：

1. **Dispositivi检测**：TikMatrixAutomatico检测Connesso的Dispositivi。
2. **AutomaticoInstalla**：系统会AutomaticoInstallaTikMatrixApplica，无需用户干预。
3. **AutorizzazioneConferma**：如果DispositiviSuggerimentoInstalla权限，只需点击"允许"。
4. **CompletatoInstalla**：初始化过程大约在10secondi内Completato。
5. **可以Utilizzo**：您的Dispositivi现在已准备好EseguiAutomatico化Attività。

这个Automatico化流程在大多数情况下无需ManualeImpostazioni。

## ManualeInstalla/Aggiorna

要ManualeInstalla或AggiornaTikMatrix APK，请在Impostazioni中Chiudi`自动唤醒Agent`开关。然后按照以下步骤Operazioni：

1. 选择你的Telefono并点击`通用` > `初始化Agent`。
2. TikMatrix会Automatico在Telefono上Installa两个APK文件。
3. 如果Suggerimento，点击"允许"AutorizzazioneInstalla。
4. 如果被Google Play保护阻止，点击"仍然Installa"Riprendi。
5. In AttesaInstallaCompletato（大约10secondi钟）。
6. Apri`自动唤醒Agent`开关以在TelefonoConnessione时AbilitaAutomaticoApplica唤醒。
7. 点击`通用` > `打开Agent`测试Installa。如果Applica无法Avvia，请Contatta技术Supporto。
