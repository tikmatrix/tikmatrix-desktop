# Migrazione Licenza

将您的TikMatrixLicenza从一台计算机转移到另一台计算机。这在Aggiorna硬件或更换计算机时非常有用。

## 要求

- Corrente计算机上有有效Licenza（Attivazione码或Stripe订阅）
- 目标计算机没有现有的TikMatrixLicenza
- Ognimesi最多允许5volte迁移

## 迁移步骤

### 步骤1：Apri迁移对话框

1. 在Corrente计算机上AvviaTikMatrix
2. 点击Titolo栏的**Licenza图标**
3. 点击**"迁移Licenza"**按钮

![许可证迁移按钮](../img/migrate-button.webp)

### 步骤2：获取目标机器ID

在目标计算机上：

1. Installa并AvviaTikMatrix
2. 点击Titolo栏的**Licenza图标**
3. Copia**机器ID**
4. 将此ID发送到Corrente计算机

![目标机器ID](../img/target-machine-id.webp)

### 步骤3：验证和迁移

回到Corrente计算机：

1. 在迁移对话框中Incolla**目标机器ID**
2. 点击**"验证"**检查兼容性
3. 查看Mostra的LicenzaDettagli

![验证成功](../img/validation-success.webp)

1. 勾选Conferma复选框
2. 点击**"迁移Licenza"**并Conferma

![迁移确认](../img/migration-confirm.webp)

### 步骤4：CompletatoImpostazioni

1. In Attesa迁移Completato
2. 在目标计算机上重启TikMatrix
3. 您的Licenza现在在新计算机上处于活跃状态

![迁移成功](../img/migration-success.webp)

## ImportanteAvviso

⚠️ **Migrazione Licenza无法撤销**

- Licenza完全从源计算机转移到目标计算机
- 您的旧计算机立即失去访问权限
- Ognimesi最多5volte迁移
- 两台计算机都需要稳定的ReteConnessione

## 迁移Contenuto

### Attivazione码用户

- Licenza状态和剩余giorni数
- Licenza代码信息

### Stripe订阅用户

- 订阅状态和账单信息
- 续费日期和计划Dettagli

## Risoluzione dei Problemi

### 常见Errore信息

#### "目标机器已有Licenza"

目标计算机已经有活跃的Licenza。迁移只能迁移到没有现有Licenza的计算机。

#### "超过mesi度迁移限制"

您Ognimesi只能迁移5volte。请等到下个mesi或ContattaAssistenza。

#### "无效的机器ID格式"

确保您正确Copia了完整的机器ID。它应该至少10个字符长。

#### "迁移验证Fallito"

检查以下Contenuto：

- 您Corrente的LicenzaSì活跃且未过期的
- 目标机器IDSì正确的
- 两台计算机都有ReteConnessione

## Domande Frequenti

**我可以迁移回原来的计算机吗？**

可以，但它会计入您的mesi度迁移限制。

**迁移后我的DispositiviConnessione会怎样？**

DispositiviConnessione与计算机绑定。您需要在新计算机上重新Connetti Dispositivi。

**我可以迁移试用Licenza吗？**

不可以，只有付费Licenza才能迁移。

**迁移会影响剩余的Licenzagiorni数吗？**

不会，迁移后您的剩余giorni数保持不变。
