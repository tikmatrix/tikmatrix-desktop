# Imposta Proxy

您可以为Ogni部TelefonoImpostazioni独立的ProxyIndirizzo IP。

> **免责声明：** 本页面Consigliati的Tutti网站均来源于互联网，与本站无关。Utilizzo前请自行验证其有效性。我们不提供任何保证。

## 1. Impostazioni住宅VPSProxy（如果您Acquista了ProxyIP，可以Salta此步骤）

1. 从目标国家Acquista住宅ProxyIP VPS（最好Sì原生IP和双ISP VPS）。
2. Utilizzo [https://github.com/MHSanaei/3x-ui](https://github.com/MHSanaei/3x-ui) 在VPS上快速ImpostazioniVPN：

    ```shell
    bash <(curl -Ls https://raw.githubusercontent.com/MHSanaei/3x-ui/refs/tags/v2.6.0/install.sh)
    # 按照提示完成VPN设置。
    ```

3. InstallaCompletato后，访问x-ui Web界面。您可以在Installa后的终端输出中找到URL和Accedi凭据。
4. 在x-ui中创建新用户，将ProtocolloImpostazioni为`SOCKS5`，并根据需要ConfigurazionePorta和其他Impostazioni。
5. 创建用户后，您将在x-ui界面中看到ProxyIP和Porta。
6. 在Telefono上的NekoBoxApplica中UtilizzoProxyIP和Porta。

## 3. 在Telefono上AbilitaProxy

1. 在Telefono上ScaricaAPK并Installa [NekoBoxForAndroid](https://github.com/MatsuriDayo/NekoBoxForAndroid) 客户端。
2. ApriNekoBoxApplica，点击`+`按钮Aggiungi新Proxy。
3. 选择`SOCKS5`或其他Proxy类型。
4. 输入ProxyIP和Porta，然后点击`保存`。
5. 点击`开始`按钮AbilitaProxy。
6. AbilitaProxy后，您可以在NekoBoxApplica中检查Connessione状态。

### 中国用户Nota事项

如果您在中国，`SOCKS5`Protocollo可能无法正常工作。您需要Aggiungi加密Proxy（`VLESS`）作为前端Proxy。您可以Utilizzo以下工具：

1. 在NekoBoxApplica中Aggiungi`VLESS`Proxy。
2. 在NekoBoxApplica中Aggiungi`SOCKS5`Proxy。
3. 在NekoBoxApplica中Aggiungi`链式`Proxy。选择`VLESS`Proxy作为前端Proxy，`SOCKS5`Proxy作为后端Proxy。
