# PersonalizzatoADB命令

TikMatrix允许您创建和GestionePersonalizzatoADB命令，以快速Esegui常用Operazioni。这些命令可以简化您的工作流程，提高Operazioni效率。

## 什么SìPersonalizzatoADB命令

PersonalizzatoADB命令Funzionalità让您可以Salva常用的ADB指令，并通过简单的点击Esegui它们，而无需Ognivolte都Manuale输入完整命令。

## 访问Comandi Personalizzati界面

1. 在TikMatrix主界面中，找到并点击左侧边栏中的**Comandi Personalizzati**选项。
2. Comandi Personalizzati界面将Mostra您已Salva的Tutti命令，以及Aggiungi新命令和Ripristina命令的按钮。

## 预设命令

首volteUtilizzo时，TikMatrix已经为您准备了一些实用的预设命令：

- **Home**：IndietroDispositivi主屏幕
- **Back**：EseguiIndietroOperazioni
- **Enable TCP**：AbilitaTCP/IPConnessione模式
- **Enable Fast Input**：AbilitaInput Rapido法
- **Open Gallery**：ApriDispositivi图库
- **Open Settings**：ApriDispositiviImpostazioni
- **Open NekoBox**：ApriNekoBoxApplica（如已Installa）

## AggiungiComandi Personalizzati

1. 点击界面顶部的**Aggiungi命令**按钮。
2. 在弹出的对话框中，填写以下信息：
   - **命令名称**：为您的命令取一个易于识别的名称
   - **命令参数**：输入ADB命令参数（无需包含"adb"前缀）
3. 点击**Aggiungi命令**按钮Salva。

## Supporto多行命令

Comandi PersonalizzatiSupporto多行输入，Ogni行将被视为单独的ADB命令，并按顺序Esegui。
