# 简易Rotazione ProxyTutorial

Rotazione Proxy可以在Script开始前，为TelefonoAutomatico换一个新的 IP。这样Cambio Account或Accedi时更安全，也能减少被风控的机会。

## 开始前准备

- 需要一个Supporto“换 IP”“轮换 IP”Funzionalità的Proxy服务。
- 向服务商索取轮换链接（也可能叫 *Rotation URL*、*Refresh URL*、*Change IP URL*）。
- 若服务商要求额外的密码或 Token，请提前准备好。

## 给DispositiviSalva轮换链接

1. 在桌面端Apri **Manage Devices**，切换到表格视图。
2. 找到需要换 IP 的Dispositivi，在 *Proxy rotation URL* 列点击 **Configure**。
3. 按Suggerimento填写：
   - **Proxy rotation URL**：Incolla服务商提供的轮换链接。TikMatrix 会在需要换 IP 时Automatico访问它。
   - **Request method**：保持为 `GET` 即可，除非服务商明确要求改成 `POST`。
   - **Headers**、**Body**：一般留空。如果服务商提供了Esempio，请原样Copia到输入框中。
   - **Timeout**：In Attesa服务商回应的时间，Predefinito 10 secondi就足够。
4. 点击 **Save**。Configurazione会AutomaticoSalva，无需ManualeModifica任何文件。

Salva后，Dispositivi行里会MostraRecenti一volte轮换的结果：绿色代表Successo，灰色表示尚未Esegui，红色表示服务商IndietroErrore。

## 检查轮换Vuoi生效

1. 仍在同一行点击 **Test Proxy Rotation**。
2. TikMatrix 会立刻访问轮换链接。  
   - 如果Successo，会弹出SuccessoSuggerimento，并Aggiorna“Last rotated”时间。  
   - 如果Fallito，请记录Suggerimento，检查链接和密码Vuoi正确，然后再试一volte。

小Suggerimento：许多Proxy都会限制换 IP 的频率。如果SuggerimentoFallito，先In Attesa几secondi钟再Riprova。

## 在Script中Utilizzo轮换

1. EseguiScript时，先勾选 **Rotazione Proxy**，TikMatrix 就会在Apri TikTok 前，依volte为Ogni台Dispositivi换 IP。
2. 你可以点击Dispositivi右侧的日志按钮，查看Ognivolte轮换的Dettagli。

## 常见问答

- **Sì不SìTuttiScript都要轮换？** 不Sì。只有在希望 TikTok 看到新 IP 时才需要，例如频繁Cambio Account前。
- **ConfigurazioneSalva在哪里？** TikMatrix AutomaticoGestione。Ogni台Dispositivi只需要Configurazione一volte。
- **链接看起来很长很复杂怎么办？** 只要在Visualizzazioni器能Apri，TikMatrix 也可以Utilizzo。
- **还Sì搞不定？** 把服务商给的Istruzioni和ErroreSuggerimento发给 TikMatrix Assistenza，我们会一起帮你排查。
