# Impostazioni Screen Mirroring

TikMatrix 提供小屏与大屏两种Screen Mirroring模式，并允许你调整画质、屏幕在界面中的Mostra大小与行为。本DocumentazioneIstruzioni“Impostazioni Screen Mirroring”对话框中的各项Configurazione。

## 小屏分辨率

小屏分辨率控制Dispositivi卡片内小屏视图的画质。较低的分辨率消耗更少的 CPU 与带宽，较高的分辨率能PotenziaMostra清晰度。

Opzionale预设：

- 非常低（240）
- 极低（360）
- 低（480）
- Standard（720）

在一台主机上Esegui大量Dispositivi时Consigliato选择较低的值；需要看清Dispositivi界面时Opzionale择更高的分辨率，但会Aumenta资源占用。

## 大屏分辨率

大屏分辨率用于在“大屏模式”下ApriDispositivi时（更大的浮动窗口）。大屏通常Utilizzo更高分辨率以获得更清晰的视觉效果。

Opzionale预设：

- 极低（360）
- 低（480）
- Standard（720）
- 高（1080）

如果你经常将单个Dispositivi以大屏查看并需要高细节，Seleziona 720 或 1080；资源受限时选择 360 或 480。

## 屏幕尺寸（网格卡片高度）

“屏幕尺寸”控制小屏视图在Dispositivi网格中的Mostra高度。此Impostazioni仅影响电脑上的渲染大小，不会Modifica真实Telefono分辨率。

- Utilizzo减号/加号调整卡片高度。
- 数值代表卡片高度的像素值，系统会将其限制在合理范围以避免极端大小。

## 大屏行为

选择大屏的Mostra方式：

- Standard（Standard窗口）— 在独立窗口中ApriDispositivi。
- Docked（停靠窗口）— 将Dispositivi在Dispositivi网格内以停靠方式Mostra（适合在保持网格布局同时查看某Dispositivi）。

当你希望DispositiviFisso在网格中时Utilizzo停靠模式，想要独立的大窗口时选择Standard窗口。

## 息屏Screen Mirroring

Abilita此选项后，Dispositivi的物理屏幕会在Screen Mirroring时Chiudi，从而节省电量、降低发热，并可能PotenziaScreen Mirroring稳定性。

Nota：

- 该Funzionalità只影响物理Dispositivi的屏幕状态；投射到电脑的画面不会因此丢失或变更。
- 部分Dispositivi或 ROM 可能不完全Supporto屏幕Chiudi行为，Consigliato先在少量Dispositivi上测试。

## Consigliato与Suggerimento

- 对于大规模Dispositivi池，优先Utilizzo较小的小屏分辨率（360 或 480）以减少Ogni台Dispositivi的资源占用。
- 在进行演示或需要高精度查看时可以Utilizzo高分辨率（1080）。
- 如果在切换分辨率后出现画面不稳定，尝试Chiudi再开启休眠投射或重启该Dispositivi的Screen Mirroring（scrcpy）会话。

## 在哪里Apri这个对话框

在Dispositivi页面中，点击Impostazioni Screen Mirroring（桌面/Mostra器图标）即可Apri对话框并Modifica相关Impostazioni。
