# Supporto Clonazione App

TikMatrix 和 IgMatrix Supporto在同一Dispositivi上Esegui多个Clona Applicazione（例如 GB TikTok、GB Instagram、Instagram Plus 或其他ModificaVersione），Ogni个Applica都有自己独立的包名。此Funzionalità让您能够通过Utilizzo不同的Applica实例更高效地Gestione多个Account。

:::danger 风险Suggerimento
UtilizzoClona Applicazione可能违反 TikTok/Instagram 的服务条款，可能导致Account Bannato禁或限制。我们仅提供此Funzionalità的技术Supporto，**不负责**因UtilizzoClona Applicazione导致的任何Account问题，包括封禁、限制或其他处罚。请自行承担Utilizzo风险。
:::

## 什么Sì包名？

Ogni个 Android Applica都有一个唯一的标识符，称为 **包名**。官方ApplicaUtilizzoStandard包名：

- TikTok: `com.zhiliaoapp.musically`
- Instagram: `com.instagram.android`

Clona ApplicazioneUtilizzo不同的包名，例如：

- GB TikTok: `com.zhiliaoapp.musically.gb`
- GB Instagram: `com.instagram.gb.android`
- Instagram Plus: `com.instagram.plus.android`

TikMatrix/IgMatrix 可以Automatico检测并为Ogni个AccountUtilizzo正确的包名。

## ImpostazioniClona Applicazione

### 第一步：InstallaClona Applicazione

1. 在您的Dispositivi或Cloud Phone上Installa首选的Clona Applicazione。
2. 确保Ogni个Clona Applicazione都有唯一的包名以避免冲突。
3. 在Ogni个Applica实例中Accedi不同的Account。

### 第二步：Configurazione包名

您有两种Configurazione包名的方式：

#### 方式 A：Account级别Configurazione（Consigliati）

为Ogni个Impostazioni Account特定的包名：

1. 在左侧导航Apri **Account**。
2. 点击某个Account进行Modifica，或点击 **新增** 创建新Account。
3. 在 **包名** 字段中，您可以：
   - **Manuale输入** Clona Applicazione的包名（例如 `com.instagram.gb.android`）
   - **Automatico检测** 点击 **检测包名** 按钮（需要先选择Dispositivi）
4. 点击 **Salva**。

:::tip Automatico检测包名
**检测包名** 按钮可以Automatico检测所选Dispositivi上CorrenteApri的Applica。这Sì获取正确包名最简单的方式：

1. 为Account选择一个Dispositivi
2. 在该Dispositivi上ApriClona Applicazione
3. 点击 **检测包名** 按钮
4. 包名将Automatico填入

此Funzionalità适用于任何 Android Applica，包括 TikTok/Instagram Clona Applicazione、App Cloner Applica和 NomixCloner Applica。
:::

当您为此AccountEseguiScript时，TikMatrix/IgMatrix 将AutomaticoUtilizzo指定的包名。

#### 方式 B：系统全局Predefinito值

为Tutti没有特定包名的Impostazioni AccountPredefinito包名：

1. 在左侧导航Apri **Impostazioni**。
2. 找到 **TikTok 包名** 或 **Instagram 包名** 字段。
3. 输入Predefinito包名。
4. 点击 **Salva**。

:::tip 优先级
Account级别的Impostazioni始终会覆盖系统全局Predefinito值。如果AccountConfigurazione了自己的包名，将Utilizzo该包名而不SìPredefinito值。
:::

## 工作原理

当您Esegui任何Automatico化Script时：

1. TikMatrix/IgMatrix 检查目标AccountVuoiConfigurazione了Personalizzato包名。
2. 如果有，则在Tutti UI Automatico化Operazioni中Utilizzo该包名。
3. 如果没有，则回退到系统全局Predefinito包名。
4. Tutti资源 ID 和 XPath 查询都会AutomaticoUtilizzo正确的包名进行格式化。

## 最佳实践

### 1. UtilizzoDescrizione性用户名

由于您可能在不同的Clona Applicazione中有多个Account，请UtilizzoDescrizione性用户名来识别Ogni个Account属于哪个Applica：

```text
username_gb      → GB TikTok/Instagram
username_plus    → TikTok/Instagram Plus
username         → 官方应用
```

### 2. 按组织AccountGruppo

根据Clona Applicazione类型创建Dispositivi组：

- **A 组**：TuttiUtilizzo GB Applica的Account
- **B 组**：TuttiUtilizzo Plus Applica的Account
- **C 组**：TuttiUtilizzo官方Applica的Account

### 3. Operazioni in Massa前先测试

在对TuttiAccountEseguiScript之前：

1. 先用单个Account进行测试
2. 验证Utilizzo的Sì正确的Applica
3. ConfermaTuttiOperazioni都按预期工作

### 4. 保持包名一致

一旦为Impostazioni Account了包名，除非切换到不同的Clona Applicazione，No则不要更改它。保持一致性有助于避免混淆和Errore。

## Risoluzione dei Problemi

### ScriptApri了Errore的Applica

**问题**：ScriptApri了Errore的Applica而不SìClona Applicazione。

**解决方案**：

1. 检查Account的 **包名** 字段Vuoi正确Impostazioni
2. 验证Clona Applicazione已Installa且包名匹配
3. Utilizzo **检测包名** 按钮Automatico获取正确的包名
4. 如果ApplicaSìRecentiInstalla的，请重启Dispositivi

### 找不到 UI 元素

**问题**：Script报告"找不到元素"Errore。

**解决方案**：

1. 不同的Clona Applicazione可能有略微不同的 UI 布局
2. 检查Clona ApplicazioneVersioneVuoi兼容
3. 尝试Aggiorna到UltimiVersione的 TikMatrix/IgMatrix
4. 如果问题持续存在，请Contatta技术Supporto

### 同一Applica中的多个Account

**问题**：想要在同一Applica实例中切换多个Account。

**解决方案**：

- TikTok/Instagram 原生Supporto多AccountAccedi
- Utilizzo TikMatrix/IgMatrix 的 **Cambio Account** Funzionalità
- 为该Applica实例中的TuttiImpostazioni Account相同的包名
- TikMatrix/IgMatrix 将Automatico处理Account切换

### 检测包名按钮不工作

**问题**：**检测包名** 按钮不工作或MostraErrore。

**解决方案**：

1. 确保已先选择Dispositivi
2. 确保ApplicaCorrente在Dispositivi上Apri
3. 检查DispositiviConnessioneVuoi稳定
4. 尝试重新ApriApplica并点击检测

4. 点击 **检测包名**

此方法适用于任何 Android Applica，SìConsigliati的方式。

## Domande Frequenti

**问：我可以在同一Dispositivi上Utilizzo多个Clona Applicazione吗？**

答：可以，只要Ogni个Clona Applicazione都有唯一的包名，您就可以在同一Dispositivi上Esegui多个Applica实例。

**问：我需要重新Installa TikMatrix/IgMatrix 吗？**

答：不需要，此Funzionalità已内置在UltimiVersione中。只需Aggiorna到UltimiVersione并Configurazione包名即可。

**问：我现有的Automatico化Script还能工作吗？**

答：可以，Tutti现有Script都完全兼容。没有Personalizzato包名的Account将UtilizzoPredefinito包名。

**问：TikTok 和 Instagram 都Supporto吗？**

答：Sì的，TikMatrix 和 IgMatrix 都完全SupportoClona Applicazione检测和Automatico化。

**问：Automatico检测FunzionalitàSì如何工作的？**

答：当您点击 **检测包名** 时，TikMatrix/IgMatrix 会检查所选Dispositivi上Corrente前台的Applica，并Automatico获取其包名。

**问：我可以Utilizzo App Cloner 或 NomixCloner 创建的Applica吗？**

答：可以，包名Funzionalità适用于任何 Android Clonazione Applicazione工具，包括 App Cloner 和 NomixCloner。UtilizzoAutomatico检测Funzionalità可以轻松获取正确的包名。
