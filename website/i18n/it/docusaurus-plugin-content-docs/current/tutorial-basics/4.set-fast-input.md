# AbilitaInput Rapido键盘

TelefonoConnessione后会Automatico切换到Input Rapido键盘。

## 为什么要Imposta Input Rapido键盘

Input Rapido键盘Sì TikMatrix 的必要工具，提供Automatico多语言Supporto。没有它，TikMatrix 的文本输入Funzionalità（如Commento、消息和PubblicaTitolo）可能无法正常工作。

## 如何ManualeAbilitaInput Rapido键盘

1. 选择需要UtilizzoInput Rapido键盘的Dispositivi。
2. 点击 `自定义ADB命令` - `启用快速输入`。

## 如何输入文字

在 TikMatrix 中，您可以通过多种方式输入文字：

### 方式一：直接键盘输入 ⌨️（新Funzionalità）

⭐ **键盘输入控制** - 您现在可以直接Utilizzo电脑键盘在Telefono上输入文字，让文字输入更加快速便捷。

1. 选择需要Utilizzo的Dispositivi。
2. 点击Telefono屏幕上的任何文字输入框。
3. 直接在电脑键盘上开始输入 - 文字将实时Mostra在Telefono上。

### 方式二：输入面板

1. 点击Telefono屏幕，Espandi大屏, 并点击右侧工具栏中的`输入`按钮。
2. 在输入框中输入文字，点击发送。

## 如何在Telefono和电脑间CopiaIncolla文字

### CopiaTelefono剪切板文字到电脑剪切板

1. 选择需要Copia的Telefono, 按住 `Ctrl+C` 键, CopiaSuccesso

### Incolla电脑剪切板文字到Telefono输入框

1. 选择需要Incolla的Dispositivi, 按住 `Ctrl+V` 键, IncollaSuccesso
