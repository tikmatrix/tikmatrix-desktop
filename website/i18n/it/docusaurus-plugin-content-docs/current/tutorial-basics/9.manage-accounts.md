# Gestione Account

Gestione Account可以把Ogni个 TikTok / Instagram 身份与其EseguiDispositivi、Accedi凭据以及Automatico化所依赖的数据保持一致。一个干净的Account列表让你无需挨个OperazioniTelefono，就能分配Dispositivi、监控Account状态、为活动预热。

## Manuale创建Account

1. Apri左侧导航中的 **Account**。
2. 点击 **新增** ApriModifica弹窗。
3. 依volte填写Base字段：
   - **邮箱**: ApplicaAccedi所Utilizzo的邮箱。TikMatrix/IgMatrix 会优先尝试Utilizzo 邮箱 + 密码 Accedi；如果邮箱为空，则Automatico切换为 用户名 + 密码。
   - **密码**：ApplicaAccedi所Utilizzo的密码。IgMatrix Supporto在密码中附加 2FA 密钥，格式见下文。
   - **用户名**：TikTok/Instagram 中Mostra的Account名，这Sì唯一Obbligatorio项。
   - **Accedi状态**：标记CorrenteDispositivi上Vuoi已经Accedi（`1` = 已Accedi，`0` = 未Accedi）。
   - **状态**：控制AccountVuoiCoinvolgimentoAttività（`0` = Abilita，`1` = Disabilita）。
4. 选择一个DispositiviTag，将Account绑定到Telefono或Cloud Phone。若未选择，Account保持未分配状态。
5. 点击 **Salva**，列表会AutomaticoAggiorna。

## Accedi

- 如果Account的 **邮箱** 和 **密码** 已填写，TikMatrix/IgMatrix 会尝试Utilizzo该信息Accedi TikTok/Instagram Applica。
- 如果 **邮箱** 为空，则会尝试Utilizzo **用户名** 和 **密码** Accedi。
- Richiesto选择Dispositivi序号，Script才会在对应Dispositivi上EseguiAccediOperazioni。
- AccediScript会Automatico识别Account的Accedi状态，Salta已Accedi的Account。

## 2FA Supporto（仅限 IgMatrix）

IgMatrix SupportoAutomatico双因子认证（2FA）Accedi：

### 2FA 密码格式

要Utilizzo 2FA，请按以下格式Impostazioni密码：

```
你的密码:你的2FA密钥
```

**Esempio：**

```
mypassword123:JBSWY3DPEHPK3PXP
```

## Importa与EsportaAccount

- **Esporta** 会把TuttoAccount写入 `accounts.xlsx` 到 TikMatrix/IgMatrix 的Scarica目录，可作为备份或In MassaModifica模板。
- **Importa** Supporto从该 Excel 文件In Massa创建或AggiornaAccount。如果不确定表头，请先点击 **Esporta**，再ModificaScarica的文件并Importa。
- Importa时带有 `id` 的行会Aggiorna现有Account，没有 `id` 的行则会新增Account。

> IgMatrix 2FA Accedi只需要填写 `username` 和 `password`，格式为 `你的密码:你的2FA密钥`，`device_index` Dispositivi序号。

### Excel 字段对照

| 字段 | VuoiObbligatorio | Istruzioni |
| --- | --- | --- |
| `id` | No | 内部唯一标识，只在需要Aggiorna已有Account时保留。 |
| `email` | No | ApplicaAccedi邮箱，方便Contatta与找回。 |
| `pwd` | No | Accedi密码，Disponibile于快速参考或AutomaticoAccedi。 |
| `username` | Sì | TikTok/Instagram 的用户名，工作区内需保持唯一。 |
| `device` | No | 绑定的Dispositivi序列号或Cloud Phone标识。 |
| `device_index` | No | 在界面中Mostra的Dispositivi别名，主要用于参考。 |
| `logined` | No | 会话标记（`1` = 已Accedi，`0` = 未Accedi）。 |
| `status` | No | Disponibile性标记（`0` = Abilita，`1` = Disabilita）。DisabilitaAccount不会CoinvolgimentoScriptAttività。 |
| `nickname` | No | 通过 TikTok 数据Sincronizzazione得到的Ultimi昵称。 |
| `country` | No | 根据 TikTok 资料推断的国家/地区。 |
| `followers` | No | Sincronizzazione到的Follower数量。 |
| `following` | No | AccountSegui的人数。 |
| `hearts` | No | Account获得的Mi Piace总数。 |
| `videos` | No | 已Pubblica Video数量。 |
| `friends` | No | 互相Segui的好友数量（若Disponibile）。 |

### TikMatrix 与 IgMatrix 的字段差异

- **TikMatrix** 在Esegui **Sincronizzazione TikTok 数据** 或点击数据查询按钮后，会填充统计字段（`nickname`、`country`、`followers`、`following`、`hearts`、`videos`、`friends`）。
- **IgMatrix** 仅UtilizzoBase字段（`email`、`pwd`、`username`、`device`、`logined`、`status`），会忽略统计类字段。

## 其他行为

- 标记为 **Disabilita** 的Account不会Coinvolgimento任何Automatico化Script、定时Attività或Operazioni in Massa。重新Abilita后才能RiprendiEseguiAttività。
- **Abbina Account** Script会扫描Dispositivi上已Accedi的Applica会话，Automatico在Account列表中创建对应条目并绑定检测到的Dispositivi，Sì快速接入Manuale准备Telefono的方式。
