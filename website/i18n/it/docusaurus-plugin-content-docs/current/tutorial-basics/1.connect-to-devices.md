# Connetti Dispositivi

TikMatrixSupportoAndroidDispositivi的USB和TCPConnessione。

## USBConnessione

1. ApriTelefono的开发者Impostazioni并AbilitaUSB调试（参考下方截图）。
   - **小米Telefono用户Nota**：您还Richiesto在开发者选项中Abilita"通过USBInstalla"和"USB调试（安全Impostazioni）"这两个选项。
2. 通过USB将TelefonoConnessione到电脑。
3. 当TelefonoSuggerimento时，点击`允许`以AbilitaUSB调试。
4. In AttesaConnessione建立。

## TCPConnessione

1. 通过USB将TelefonoConnessione到电脑。
2. 在TikMatrix中，点击`ADB命令` > `启用TCP`以AbilitaTCP/IPConnessione。
3. DisconnettiUSB线缆（如果UtilizzoTelefono盒子，可切换到OTG模式）。
4. 点击`扫描TCP设备`按钮并输入IP范围（你可以在路由器Gestione页面找到这个信息）。
5. 点击`扫描`在指定IP范围内CercaDispositivi。
6. In AttesaConnessione建立。

**Cloud Phone用户**：TikMatrix现在SupportoSalvaCloud PhoneConnessione信息以便快速访问。在扫描对话框中切换到`云手机列表`Tag页即可Salva和Gestione您的Cloud PhoneConnessione。详细Istruzioni请参阅[连接云手机](./1.connect-to-cloudphone.md)Guida。

### Suggerimento

- 不同品牌的Operazioni步骤可能有所不同，但基本流程相似。
- 如果没有响应，请尝试更换数据线或Utilizzo电脑后部的USBPorta。

## 如何Connessione主板或机箱

1. 向主板或机箱销售商索要Autorizzazione文件`adbkey`和`adbkey.pub`。
2. 将这些文件Copia到电脑的`C:\Users\你的电脑用户名\.android`目录下。
3. 重启电脑。

### 其他Suggerimento

- 备份你的Autorizzazione文件。如果重装系统或更换电脑，你将再volte需要它们。
- 如果这些文件丢失，你需要在Telefono屏幕上Manuale允许权限。

## 如何ConnessioneAltroUSBDispositivi

💡 理论上，USB最多可Supporto127个Dispositivi。  
但实际上，大多数消费级主板由于固件和芯片组的限制，通常将其限制在约40个Dispositivi。

🖥️ 服务器级主板（例如X79架构）则不同 — 它们通常移除这些USB限制，允许您大规模扩展。

🔧 如何SuccessoConnessioneAltroDispositivi：

1. UtilizzoPC背面的USBPorta — 它们与主板Connessione更为直接。  
2. UtilizzoUSB 2.0Porta（通常Sì黑色的），而不SìUSB 3.0（蓝色），后者对于大型Dispositivi群可能不稳定。  
3. 在BIOSImpostazioni中：  
   - DisabilitaXHCI  
   - AbilitaEHCI

这些步骤将Aiuto您的系统可靠地识别AltroDispositivi。
