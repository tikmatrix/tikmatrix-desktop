# Errore di Avvio del Software

## 常见Errore di Avvio del SoftwareRisoluzione dei Problemi步骤

### 步骤1：检查系统要求

1. 确保您的系统满足最低要求：
   - Windows 10 或更高Versione
   - 至少4GB内存
   - 至少1GBDisponibile磁盘空间

### 步骤2：检查Microsoft Visual C++ Redistributable

通常情况下, `vcruntime140.dll 未找到` 或者`应用程序无法正常启动(0xc000007b)`ErroreSì由于 Microsoft Visual C++ Redistributable 未Installa导致的。

1. 按 `Windows + R` 键, 输入 `cmd` 并按回车
2. 输入 `dir "C:\Windows\System32\vcruntime*.dll"` 并按回车
3. 如果有输出，IstruzioniInstalla了相关Versione的 Visual C++ Redistributable。文件名中的 vcruntime 后缀通常会带Versione号，如 vcruntime140.dll（对应 2015-2022）。
4. 如果没有输出，请Scarica并InstallaMicrosoft Visual C++ Redistributable: [Latest Microsoft Visual C++ Redistributable Version](https://learn.microsoft.com/en-us/cpp/windows/latest-supported-vc-redist?view=msvc-170)

### 步骤3：检查系统时间

电脑时钟不准确会导致"Agent Start Timeout"Errore：

1. 右键点击屏幕右下角的时间Mostra
2. 选择"调整日期/时间"
3. 如果尚未Abilita，请开启"AutomaticoImpostazioni时间"
4. 如果时间仍然不正确，点击"立即Sincronizzazione"或ManualeImpostazioni正确的日期和时间
5. 校正时间后重新AvviaTikMatrix

### 步骤4：检查ApplicaProgramma日志

1. 点击`设置` - `打开应用目录` 按钮
2. 导航到 `logs` 文件夹
3. 检查Ultimi的日志文件中的Errore信息

### 如果问题仍然存在

### 步骤 5：macOS — Agent InvalidStateError

在部分 macOS 机器上，Agent 可能无法Avvia并出现如下Errore：

```text
Agent start error InvalidStateError: The object is in an invalid state.
```

可能原因：

- 通常SìApplica与Corrente macOS 系统Versione或旧硬件/固件之间的不兼容所致。

解决方法 / 工作方案：

1. 通过「系统Impostazioni」>「通用」>「SoftwareAggiorna」将 macOS Aggiorna到Ultimi版。Aggiorna后重启 Mac 并再volteAvvia TikMatrix。
2. 如果无法Aggiorna或Errore仍然存在，尝试在另一台Esegui较新 macOS 的 Mac 上EseguiApplica。

如果在尝试以上方法后问题依旧，请收集Applica日志并通过 [Telegram](https://t.me/tikmatrix_agent_bot) Contattaci，附上您的 macOS Versione与日志文件。

请Contattaci的Supporto团队 [Telegram](https://t.me/tikmatrix_agent_bot) ，并提供以下信息：

- WindowsVersione
- ApplicaProgramma日志文件
- 您已经尝试过的步骤
