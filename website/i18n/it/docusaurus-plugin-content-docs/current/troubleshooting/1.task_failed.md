# AttivitàEseguiFallito排查步骤

## 第1步：检查Agent状态(如果Attività未能SuccessoAvvia)

1. 点击左侧边栏“通用” - “ApriAgent”
2. ConfermaTelefonoVuoi已ApriTikMatrix app
3. 观察Vuoi在10secondi内MostraSuccesso状态
4. 如果Agent状态为Fallito，请按照[初始化设备](../tutorial-basics/2.init-device.md)Documentazione步骤进行Manuale初始化

## 第2步：检查TikTok/InstagramApplica状态(如果Script没有正确Apri或ChiudiApplica)

1. 如果您Utilizzo的Sì官方的TikTok/InstagramApplica，请点击“ApriTikTok”/“ApriInstagram”，观察TelefonoVuoi正确ApriTikTok或InstagramApplica,如果没有，请点击Titolo栏中的Impostazioni按钮, 选择正确的TikTok Versione(国际版/亚洲版), Instagram只有唯一的官方Versione
2. 如果您Utilizzo的Sì克隆版TikTok/InstagramApplica，请确保您已经在**Account列表**中正确Configurazione了Clona Applicazione的**包名**, 请参考[使用克隆版__PROTECTED_1__/__PROTECTED_3__应用](../tutorial-basics/10.clone-apps-support.md)Documentazione进行Configurazione, 并且确保EseguiAttività的时候开启了**多Account模式**

## 第3步：检查TikTok/Instagram语言Impostazioni

目前我们只Supporto英文的TikTok和Instagram。如果您的TikTok或InstagramApplica不Sì英文，请按照以下步骤进行Impostazioni：

1. ApriTelefono上的TikTokApplica
2. 进入Profilo
3. 点击右上角的三个点ApriImpostazioni
4. 滚动到“语言”部分
5. 选择“英语”作为语言
6. 重复以上步骤为InstagramApplicaImpostazioni语言为英语

## 第4步：Esegui测试Script

1. 点击左侧边栏“Script” - “测试Script”
2. In AttesaScriptEseguiCompletato
3. 如果测试ScriptEseguiFallito，请查看日志并尝试解决问题，Domande Frequenti请参考下方的“Problemi Noti”部分

## 第5步：提交工单

如果前4步检查都没有发现问题，需要创建一个工单, 我们会Automatico收集日志并协助您解决问题。请按照以下步骤Operazioni：

1. 点击Titolo栏的“Supporto”按钮
2. 点击“创建工单”按钮
3. 输入您的Contatta方式和问题Descrizione, 为了更快地解决问题, 请尽量拍摄ScriptEsegui过程中Telefono的屏幕录像并Aggiungi到附件中
4. 选择受影响的Dispositivi,可以多选(Importante, 只有选择了受影响的Dispositivi, 我们才能收到相关日志)
5. 点击“提交”按钮
6. 我们可能会进一步Contatta你以获取Altro信息,比如Scarica指定页面的UI xml文件: ManualeOperazioniTelefono停留在Attività Fallita前的最后页面, 点击Telefono大屏右侧的“调试”按钮, 然后点击“ScaricaXML”按钮, 将Scarica的xml文件发送到我们的邮箱:`support@tikmatrix.com`并附上工单号
