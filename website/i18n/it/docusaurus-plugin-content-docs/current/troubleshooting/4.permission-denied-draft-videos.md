# Permission Denied - Unable to Save Draft Videos

## 解决Telefono屏幕出现权限拒绝Suggerimento的问题

### 问题Descrizione

当EseguipostScript时，Telefono屏幕可能会出现以下Suggerimento：

```text
Permission denied. Unable to save draft videos.
```

### 解决方案

这个问题Sì由于TikTokApplica缺少存储权限导致的。请按照以下步骤解决：

#### 步骤1：Manuale点击Pubblica按钮

1. 在Telefono屏幕上找到并点击**Pubblica**按钮
2. Riprendi进行Pubblica流程

#### 步骤2：允许系统权限

1. 当系统弹出权限申请对话框时，点击**允许**
2. 确保授予TikTokApplica所需的存储权限

### ImportanteSuggerimento

:::info
这个权限Impostazioni步骤只需要Esegui一volte。一旦授予权限，后续的postScriptEsegui将不会再出现此问题。
:::

### 预防措施

为了避免此问题，Consigliato在首volteUtilizzoTikMatrix之前：

1. ManualeApriTikTokApplica
2. 尝试录制或Salva一个VideoBozza
3. 在权限申请时选择**允许**
4. 这样可以预先Configurazione好所需的权限
