# 无法检测到Telefono

## 无法检测到Telefono排查步骤

### 第一步：检查ADBConnessione状态

1. 在TikMatrix中点击`设置` > `打开ADB终端`  
2. Esegui`adb devices`命令，检查TelefonoVuoi出现在列表中

### Windows系统

```shell
.\adb.exe devices
```

### Mac/Linux系统

```shell
./adb devices
```

### 输出Esempio

```shell
List of devices attached
<设备ID>    <状态>
```

### 如果Dispositivi没有出现在adbDispositivi列表中

请按照[连接设备](../tutorial-basics/1.connect-to-devices.md)Documentazione步骤进行排查

### 如果Dispositivi状态为`unauthorized`

可能的原因SìDispositivi未Autorizzazione,请在Telefono上点击`允许USB调试`

### 如果Dispositivi状态为`offline`

可能的原因SìUSB线松动,请重新ConnessioneTelefono,拔出USB线,重新Connessione

### 如果Dispositivi状态为`device`

Dispositivi已经ConnessioneSuccesso,如果TikMatrix仍然没有检测到Telefono, 请ConfermaTikMatrixVuoiAttivazione, Prova Gratuita也需要在Autorizzazione页面Attivazione试用

### 如果问题仍然存在

请通过[Telegram](https://t.me/tikmatrix_agent_bot)联系我们的技术支持团队，提供以下信息：

- adb devices命令输出截图
- 你已经尝试过的步骤
