# Potenzia Utenti

:::warning 下线Suggerimento
该Funzionalità已整合到 [超级营销脚本](./14.super-marketing.md)。请优先Utilizzo“Super Marketing”以获得更强大、更灵活的用户Potenzia能力。
:::

Potenzia UtentiFunzionalità通过In MassaSegui/Non Seguire PiùOperazioni来Potenzia目标用户的曝光与Interazioni。系统会尽量模拟真实用户行为以降低被检测风险，但仍Consigliato遵循平台规则并控制频率。

## 在“Super Marketing”Script中UtilizzoPotenzia UtentiFunzionalità

1. ApriScript：选择“Super Marketing”。
2. 选择数据源类型为“用户Account列表”。
3. In MassaIncolla用户名或从文本文件Importa（Ogni行一个）；Importa后Opzionale择“追加”或“覆盖”。
4. 选择数据Utilizzo策略：

    - 共享池策略：Tutti bot Account会对相同目标进行Segui/Non Seguire Più。

    - 单volte消耗策略：Ogni个 bot Account仅处理不同的目标，确保Ogni个目标只被一个AccountOperazioni一volte。

5. 在“用户Operazioni — 重复volte数”中ImpostazioniOgni个目标的Operazionivolte数（通常Impostazioni为 1 volte）。
6. 选择Operazioni类型：Segui 或 Non Seguire Più。可在“用户Operazioni”中进一步Configurazione：

        - 进入用户Home方式：选择“Cerca用户”或“直接ApriHome（Direct Open Profile）”。

        - 发送Messaggio Diretto（Opzionale）：Abilita后可Incolla预生成的多行Messaggio Diretto文案，Ogni行或按段为一条候选文案；Configurazione文案Utilizzo顺序：

                - Casuale：OgnivolteUtilizzoCasuale选一条文案。
                - 按顺序：按Incolla/Importa的顺序依volteUtilizzo（可跨Attività轮volteRiprendi）。

            另Opzionale择Vuoi在文案中Automatico插入表情（emoji）以提高自然度。

7. Configurazione其他参数（并发、延时、Account池），Conferma无误后AvviaScript。

## UtilizzoConsigliato与Nota事项

- Consigliato先在少量Account上测试，观察策略与参数Vuoi触发平台风控。
- 合理Impostazioni延时与并发，模拟人工行为以降低风险。
- 长期Esegui时请定期检查Account健康、Attività日志，并根据结果调整策略。
- Potenzia UtentiOperazioni并非 100% Successo，遇到FallitoRiprova或排查目标Account状态。
