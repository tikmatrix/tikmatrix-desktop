# Raccogli Utenti

Raccogli UtentiScript用于采集TikTok用户名。用户采集FunzionalitàSupporto三种模式：收集指定用户的Follower列表、收集指定用户的Segui列表，或通过关键词Cerca收集用户。

## 步骤

1. 选择要EseguiScript的Dispositivi。如果选择多个Dispositivi，采集Attività将在所选Dispositivi之间平均分配。
2. 点击`脚本` > `采集用户`。
3. ConfigurazioneImpostazioni Attività：
    - **采集模式**：选择Raccogli Utenti的模式。
      - **通过Follower采集**：收集指定Account的Follower列表。
      - **通过Segui采集**：收集指定Account的Segui列表。
      - **通过关键词采集**：Utilizzo关键词Cerca收集用户。
    - **目标Account用户名**：输入要Raccogli Utenti的Account用户名，Ogni行一个（用于Follower和Segui模式）。
    - **Cerca关键词**：输入Cerca用户的关键词（用于关键词模式）。
    - **最大采集数量**：Impostazioni要采集的最大用户数量。

4. 点击`开始脚本`开始Esegui。

## Nota事项

- 采集的用户数据将根据采集模式Salva到文件：
  - Follower模式：`exported_users_followers_tiktok_<用户名>_<数量>.txt`
  - Segui模式：`exported_users_following_tiktok_<用户名>_<数量>.txt`
  - 关键词模式：`exported_users_search_tiktok_<关键词>_<数量>.txt`
- TikTok只限制可查看的用户数量，因此OgnivolteOperazioni只能检索大约50条记录。
