# Potenzia Post

:::warning 下线Suggerimento
该ScriptFunzionalità已整合到 [超级营销脚本](./14.super-marketing.md)。请优先Utilizzo“Super Marketing”Script以获得更强大、更灵活的帖子Potenzia能力。
:::

Potenzia PostFunzionalità可通过In MassaMi Piace、Condividi、Preferiti、Inoltra与Visualizzazioni等Operazioni来Aumenta帖子的Interazioni量。系统会尽量模拟真实用户行为以降低被检测的风险，但Utilizzo时仍应遵循平台规则并适度控制频率。

## 在“Super Marketing”Script中UtilizzoPotenzia PostFunzionalità

1. ApriScript：选择“Super Marketing”。
2. 选择数据源类型为“帖子链接列表”。
3. In MassaIncolla帖子链接或从文本文件Importa；Importa后Opzionale择“追加”或“覆盖”Corrente列表。
4. 选择数据Utilizzo策略：

    - 共享池策略：Tutti bot Account对相同帖子进行Interazioni。

    - 单volte消耗策略：Ogni个 bot Account只对不同的帖子进行一volteInterazioni，确保Ogni个帖子仅被一个Account处理一volte。

5. 在“帖子Operazioni — 重复volte数”中ImpostazioniOgni个帖子的Interazionivolte数。若包含Commento，Consigliato将Commentovolte数Impostazioni为 1，避免出现重复Commento。
6. 在“帖子Operazioni”中选择需要的Interazioni类型：Mi Piace、Condividi、Inoltra、Preferiti、Commento、VisualizzazioniVideo等。

     如果选择“Commento”，请在CommentoImpostazioni中进行以下Configurazione：

     - 预生成Commento模板：可Incolla多行或多段预生成Commento文案（Ogni行或Ogni段视为一条候选Commento）。

     - 文案Utilizzo顺序：选择“Casuale”或“按顺序”Utilizzo。
         - Casuale：Ognivolte从候选文案中Casuale抽取一条。
         - 按顺序：按Incolla/Importa的顺序依volteUtilizzo（适合需要可复现的去重策略）。

     - 插入表情（emoji）：可开启Automatico在Commento中插入表情以提高自然度。

     - 实时生成Commento（Opzionale，ChatGPT API）：
         - Abilita后，Script可调用 ChatGPT 类 API，根据帖子Titolo/Descrizione实时生成上下文相关且唯一的Commento文案。此方式可显著降低重复Commento的风险，适合需要大量唯一Commento的场景。
         - Nota事项：Utilizzo实时生成需提供Disponibile的 API Key，并Nota速率限制、费用与隐私合规（不要发送敏感数据到第三方服务）。

     - 保证唯一Commento的两种常见方式：
         1. Utilizzo“按顺序”加载并轮换预生成Commento模版（保证Ogni条模版只被Utilizzo一volte直到耗尽）。
         2. Utilizzo ChatGPT API 实时生成Ogni条Commento（基于帖文Titolo/ContenutoDinamico生成，giorni然更易保证唯一性）。

     - Consigliato：先用小In Massa帖子测试Commento策略，检查Commento质量与平台处理情况；若Utilizzo实时生成，请监控 API 调用量与成本。
7. Configurazione其他Script参数（如并发数、延时、Account池等），Conferma无误后AvviaScript。

## UtilizzoConsigliato与Nota事项

- Consigliato先在少量帖子上进行测试，Conferma策略与参数不会触发平台风控。
- CommentoContenuto应多样化并保持自然，避免CopiaIncolla相同文本。
- 合理Impostazioni延时与并发，模拟人工行为以降低风险。
- 长期Esegui时请定期检查Account健康与Attività日志，并根据情况调整策略。
