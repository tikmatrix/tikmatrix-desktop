# Potenzia Live

Potenzia LiveScript允许您控制农场AccountVisualizzazioni并Opzionale择性地Mi Piace或Commento目标Live。您可以ImpostazioniVisualizzazioni时长和Mi Piace/Commento间隔来PersonalizzatoPotenzia行为。

## 步骤

1. 选择要EseguiScript的Dispositivi。
2. 点击`脚本` > `提升直播`。
3. ConfigurazioneImpostazioni Attività：
    - **用户名**：输入目标用户名，Ogni行一个。
    - **进入方式**：选择**Cerca**或**直接**方式。
    - **Potenzia选项**：AbilitaMi Piace和/或Commento。
    - **加入Follower团**：Abilita加入主播Follower团Funzionalità（如果Disponibile）。
    - **Ogni日签到**：AbilitaLive期间的Ogni日签到活动。
    - **Mi Piace间隔**：ImpostazioniOgnivolteMi PiaceOperazioni之间的间隔。
    - **Mi Piace点击volte数**：Impostazioni点击Mi Piace按钮的volte数。
    - **Commento间隔**：ImpostazioniOgnivolteCommentoOperazioni之间的间隔。
    - **CommentoContenuto**：Ogni行输入一条Commento。
    - **插入表情符号**：选择Vuoi在Commento中插入表情符号。
    - **Commento顺序**：选择Commento的Casuale或顺序排列。
    - **OgniAccountCommento数**：ImpostazioniOgni个AccountPubblica的Commento数量。
    - **Visualizzazioni时长**：Impostazioni在Live中停留的时间。
    - **Attività间隔**：ImpostazioniOgni个Attività之间的间隔。
4. 点击`开始脚本`开始Esegui。

## Nota事项

- Potenzia LiveScript可能不总SìSuccesso；如需要RiprovaFallito的Attività。
