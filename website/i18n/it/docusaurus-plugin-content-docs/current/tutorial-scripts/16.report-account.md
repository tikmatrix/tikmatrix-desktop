# Segnala Account

:::warning 负责任地Utilizzo
SegnalaFunzionalitàSì一个强力Operazioni。仅当Conferma存在违法或违规（欺诈、诈骗、垃圾信息等）时才Utilizzo。滥用Segnala可能导致Account被限制或其他不良后果。
:::

本GuidaIstruzioni如何Utilizzo Super Marketing Script内置的“Segnala Account”Funzionalità。该Funzionalità会AutomaticoEseguiFisso的Applica内Segnala流程以处理一批用户Account，但不Supporto在 UI 或Script中PersonalizzatoSegnala路径或原因。

## 快速概览

- ScriptUtilizzo的FissoSegnala流程如下（不可Modifica）：

```text
举报 → 举报账号 → Something else → Frauds and scams
```

- Script会尝试进入目标用户Home，ApriAltro/菜单项，按Fisso路径选择Segnala并提交（若Disponibile）。
- 由于Telefono端Applica在不同Versione与Dispositivi上 UI 可能存在差异，ScriptUtilizzo尽力而为的选择器，因此请在小In Massa上先行验证。

## 通过 Super Marketing Esegui的步骤

1. Apri TikMatrix/IgMatrix 并选择要Esegui的Dispositivi（Supporto多选）。
2. Apri `Scripts` → `Super Marketing`。
3. 将 **数据来源（Data source）** Impostazioni为 `Usernames`，Importa文本文件或Incolla用户名（一行一个）；Importa后选择“追加”或“覆盖”。
4. 在 **用户动作（User Actions）** 中Abilita **Segnala Account（Report account）** 开关，并ConfermaMostra的FissoSegnala路径。
5. ConfigurazioneAttività并发、延迟等参数。
6. 点击 `Start Script` 开始处理列表，监控日志与Dispositivi界面以ConfermaOperazione Riuscita或Fallito。

## Consigliato与Nota事项

- 先在 1–5 个Account上测试，ConfermaScript能正确识别并EseguiSegnala流程（UI 文本或位置变化会导致Fallito）。
- 保持保守的节奏与延迟，In MassaAutomaticoSegnala可能触发平台的限流或风控。
- 如果Applica UI 发生变化（按钮文本、布局变动），Script可能无法找到对应元素，需要Aggiorna选择器或调整In Attesa策略。
- 仅对确凿违规ContenutoUtilizzoSegnala，避免骚扰或针对无辜用户滥用该Funzionalità。
- Esegui后检查日志，Script会记录诸如“找不到菜单按钮”或“找不到Segnala选项”的Errore，便于你调整Script或Esegui参数。

## 不ConsigliatoUtilizzo的场景

- 需要为Ogni个Caso d'Uso选择PersonalizzatoSegnala原因或Carica证据的场景：Script不Supporto细粒度Personalizzato。
- 法律/敏感事件需要人工审核或保留证据的情况。

## 故障排查

- 报错 “Could not find Report option” / “Could not find 'Report account' option”：表示Applica UI 文本或布局已变化。请在小In Massa下再volteEsegui并收集Dispositivi截图用于调试。
- 如果ScriptMostra已提交Segnala但平台没有实际处理，平台可能对AutomaticoSegnala作了降权或需要人工审查；针对Importante个案，请Utilizzo人工流程。
