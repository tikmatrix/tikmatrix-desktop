# Impostazioni Privacy

**Impostazioni Privacy**Script用于根据Visualizzazioni数过滤条件In MassaAggiorna TikTok 帖子的Impostazioni Privacy。

:::info
此Script仅适用于 TikTok Account。
:::

## 步骤

1. 选择要EseguiScript的Dispositivi。
2. 点击 `脚本` > `隐私设置`。
3. ConfigurazioneImpostazioni Attività：
    - **帖子过滤参数**：
      - **最大Visualizzazioni数**：输入最大Visualizzazioni数以过滤帖子。只有Visualizzazioni数 ≤ 此数字的帖子会被Aggiorna。
      - **最大处理帖子数**：输入要处理的最大帖子数。Impostazioni为 0 表示不限制处理数量。
    - **Impostazioni Privacy参数**：
      - **谁可以Visualizzazioni此Video**：选择一个选项：
        - `所有人`：任何人都可以VisualizzazioniVideo
        - `朋友`：只有朋友可以VisualizzazioniVideo
        - `仅自己`：只有你自己可以VisualizzazioniVideo
    - **开关参数**：
      - **允许Commento**：Abilita或Disabilita帖子Commento。
      - **允许Contenuto重用**：Abilita或DisabilitaContenuto重用，包括合拍、缝合、贴纸和Aggiungi到故事。
4. 点击 `开始脚本` 开始Esegui。

## 工作原理

Script遵循以下工作流程：

1. 进入 Videos Tag页
2. 扫描帖子，解析Visualizzazioni量
3. 找到符合条件的帖子（views ≤ max_views）
4. 进入帖子Dettagli页
5. 点击 "Privacy settings" 按钮
6. In AttesaImpostazioni弹窗出现
7. 根据你的选择ImpostazioniVisualizzazioni权限
8. 智能切换 "Allow comments" 开关
9. 智能切换 "Allow reuse" 开关
10. Indietro列表，Riprendi处理下一个帖子
11. 重复以上步骤，直到处理完Tutti匹配的帖子

## Nota事项

- Impostazioni PrivacyScript可能不总SìSuccesso；如需要RiprovaFallito的Attività。
- Script仅处理符合Visualizzazioni数过滤条件的帖子。
- 更改会立即Applica，且无法通过Script撤销。
- 在对多个帖子EseguiScript之前，请确保检查你的Impostazioni Privacy。
