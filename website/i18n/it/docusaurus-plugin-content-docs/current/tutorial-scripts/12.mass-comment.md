# Commenti in Massa

:::warning 下线Suggerimento
该Funzionalità已整合到 [超级营销脚本](./14.super-marketing.md)。请优先Utilizzo“Super Marketing”Script以获得更强大、更灵活的Commenti in Massa与ContenutoInterazioni能力。
:::

Commenti in Massa用于对大量 TikTok 帖子PubblicaInterazioniCommento（如Warm-up、Promozione、活动运营）。系统Supporto预生成模板与实时生成两种消息来源，并会尽量模拟人工发送节奏以降低被检测风险。

## 在“Super Marketing”Script中UtilizzoCommenti in Massa

1. ApriScript：选择“Super Marketing”。
2. 选择数据源类型为“帖子链接列表”。
3. In MassaIncolla帖子链接或从文本文件Importa（Ogni行一个）；Importa后Opzionale择“追加”或“覆盖”。
4. 选择数据Utilizzo策略：

   - 共享池策略：Tutti bot Account对相同帖子进行Commento。

   - 单volte消耗策略：Ogni个 bot Account只对不同的帖子进行一volteCommento，确保Ogni个帖子仅被一个Account处理一volte。

5. ImpostazioniOgni个帖子的Commentovolte数（通常为 1 volte以避免重复Commento）。
6. 在“CommentoImpostazioni”中Configurazione以下选项：

   - 预生成Commento模板：可Incolla多行或多段预生成Commento文案（Ogni行或Ogni段视为一条候选Commento）。

   - 文案Utilizzo顺序：选择“Casuale”或“按顺序”。

   - Casuale：Ognivolte从候选文案中Casuale抽取一条。

   - 按顺序：按照Incolla/Importa顺序轮换Utilizzo（适合需要可复现的去重策略）。

   - 插入表情（emoji）：可开启Automatico在Commento中插入表情以提高自然度。

   - 插入Dispositivi编号（Opzionale）：在Commento末尾加入Dispositivi编号以便追踪来源（仅用于调试/审计，Utilizzo时请谨慎）。

   - 实时生成Commento（Opzionale，ChatGPT API）：

      - Abilita后，Script可调用 ChatGPT 类 API，根据帖文Titolo/Descrizione或目标上下文实时生成上下文相关且唯一的Commento文案，适合需要大量唯一Commento的场景。

      - Nota：Abilita实时生成需提供有效 API Key，并Nota速率限制、费用与隐私合规（避免发送敏感信息到第三方服务）。

   - 保证唯一Commento的两种常见方式：

      1. Utilizzo“按顺序”轮换预生成Commento模板（Ogni条模板只Utilizzo一volte直到耗尽）。

      2. Utilizzo ChatGPT 实时生成Ogni条Commento（基于帖文ContenutoDinamico生成，唯一性更高）。

7. Configurazione其他Script参数（并发数、Attività间隔、Account池等），Conferma无误后AvviaScript。

:::tip InfoAttività间隔
ConsigliatoImpostazioni最小/最大间隔并AbilitaCasuale间隔，以避免TuttiDispositivi同时Esegui导致异常峰值，从而降低被平台检测或限制的风险。
:::

## UtilizzoConsigliato与Nota事项

- 先在小In Massa帖子上测试你的Commento策略，ConfermaCommento质量与平台处理情况。
- CommentoContenuto应合规且多样化，避免短时间内发送大量相同Contenuto。
- 如果Utilizzo实时生成，请严格监控 API 调用量与费用，并确保不向外泄露敏感信息。
- Attività可能Fallito；遇到FallitoRiprova或查看日志，系统会AutomaticoSalta已Completato的目标。
