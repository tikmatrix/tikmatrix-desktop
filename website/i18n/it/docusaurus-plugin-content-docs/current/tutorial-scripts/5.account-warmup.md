# Account预热

Account预热Script用于通过Cerca关键词、VisualizzazioniVideo、Segui、Mi Piace、Commento和Preferiti来预热Account。

## 步骤

1. 选择要EseguiScript的Dispositivi。
2. 点击`脚本` > `账号预热`。
3. ConfigurazioneImpostazioni Attività：
   - **Visualizzazioni时长**：VisualizzazioniOgni个Video的时长。
   - **Cerca关键词**：CercaVideo的Hashtag。
   - **Attività时长**：预热Attività的总时长。
   - **CommentoContenuto**：Utilizzo预制Commento或从帖子Titolo生成的ChatGPTCommento。
   - **Interazioni概率**：Segui、Mi Piace、Commento和Preferiti的概率。
4. 点击`开始脚本`开始Esegui。

## Nota事项

- 预热Script可能不总SìSuccesso；如需要RiprovaFallito的Attività。
- Supporto定时调度，你可以Impostazioni预热Attività在指定时间Esegui。
- SupportoSaltaPubblicità、Live和已Mi Piace的Video。
