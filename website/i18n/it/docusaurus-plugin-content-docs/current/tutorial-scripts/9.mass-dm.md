# Messaggi Diretti in Massa

:::warning 下线Suggerimento
该Funzionalità已整合到 [超级营销脚本](./14.super-marketing.md)。请优先Utilizzo“Super Marketing”Script以获得更强大、更灵活的Messaggi Diretti in Massa与消息Gestione能力。
:::

Messaggi Diretti in MassaFunzionalità用于通过In Massa发送Messaggio Diretto（DM）与目标用户建立Contatta，适用于Promozione、Assistenza回访和用户唤醒等场景。系统Supporto预生成模板与实时生成两种消息来源，并会尽量模拟人工发送以降低被检测风险。

## 在“Super Marketing”Script中UtilizzoMessaggi Diretti in MassaFunzionalità

1. ApriScript：选择“Super Marketing”。
2. 选择数据源类型为“用户Account列表（目标名单）”。
3. In MassaIncolla目标用户名或从文本文件Importa（Ogni行一个）；Importa后Opzionale择“追加”或“覆盖”。
4. 选择数据Utilizzo策略：

    - 共享池策略：Tutti bot Account对相同目标名单进行Messaggio Diretto。

    - 单volte消耗策略：Ogni个 bot Account只处理不同的目标，确保Ogni个目标仅由一个Account发送消息一volte。

5. ImpostazioniOgni个目标的消息发送volte数（通常为 1 volte，以避免重复骚扰）。
6. 在“消息Impostazioni”中ConfigurazioneMessaggio DirettoContenuto与发送策略：

    - 预生成消息模板：可Incolla多行或多段预生成Messaggio Diretto文案（Ogni行或Ogni段视为一条候选消息）。

    - 文案Utilizzo顺序：选择“Casuale”或“按顺序”Utilizzo。

        - Casuale：OgnivolteCasuale抽取一条模板发送。

        - 按顺序：按照Incolla/Importa顺序轮换Utilizzo模板（适合需要可复现的去重策略）。

    - 插入表情（emoji）：可开启Automatico在消息中插入表情以提高自然度。

    - 实时生成消息（Opzionale，ChatGPT API）：

        - Abilita后，Script可调用 ChatGPT 或类似 API，根据目标用户名或Disponibile上下文实时生成个性化消息，适合需要大量唯一消息的场景。

        - Nota：Abilita实时生成需提供有效 API Key，并Nota速率限制、费用与隐私合规（避免发送敏感数据）。

    - 保证唯一消息的两种常见方式：

        1. Utilizzo“按顺序”轮换预生成模板（Ogni条模板只Utilizzo一volte直到耗尽）。

        2. Utilizzo ChatGPT 实时生成Ogni条消息（基于目标信息Dinamico生成，唯一性更高）。

7. ConfigurazioneApri用户方式（如需查看用户资料）：选择“Cerca用户”或“直接ApriHome（Direct Open Profile）”。
8. ConfigurazioneAttività间隔（最小/最大，单位：minuti）与并发、Account池等参数，Conferma无误后AvviaScript。

:::tip InfoAttività间隔
ConsigliatoImpostazioni最小/最大间隔并AbilitaCasuale间隔，以避免TuttiDispositivi同时Esegui导致异常峰值，从而降低被平台检测或限制的风险。
:::

## UtilizzoConsigliato与Nota事项

- Messaggio DirettoContenuto应合规且多样化，避免短时间内发送大量相同Contenuto。
- 如Utilizzo实时生成，请严格监控 API 调用量与费用，并确保不向外泄露敏感信息。
- Attività可能Fallito；遇到FallitoRiprova或查看日志，系统会AutomaticoSalta已Completato的目标。
