# Pubblica帖子

Pubblica帖子Script用于在TikTok上Pubblica Video或图片。

## 步骤

1. **准备素材**：
   - 选项1：点击`素材库`按钮CaricaVideo/图片到素材库。
   - 选项2：Utilizzo本地文件夹作为素材来源，在ScriptConfigurazione中Impostazioni文件夹路径。
2. 选择要EseguiScript的Dispositivi。
3. 点击`脚本` > `发布帖子`。
4. ConfigurazioneImpostazioni Attività：
    - **素材来源**：选择素材库或本地文件夹。
    - **本地文件夹路径**：如果Utilizzo本地文件夹，指定包含Video/图片的路径。
    - **Pubblica方式**：选择Pubblica方式：系统Condividi或Aggiungi按钮(`+`)或通过CercaUtilizzo声音。
    - **声音名称**：当Utilizzo"通过CercaUtilizzo声音"Pubblica方式时，你可以选择：
      - 输入声音名称进行Cerca（例如："original sound"，"trending music"）
      - 直接输入TikTok声音链接（例如：`https://www.tiktok.com/t/ZP8hvBwCb/`）
    - **Contenuto类型**：选择Video或图片。
    - **Aggiungi声音开关**：Abilita/DisabilitaAggiungi声音。
    - **CaricaIn Attesa时间**：ImpostazioniCarica后的延迟时间（5-60secondi），确保TikTok服务器完全处理Contenuto后再RiprendiOperazioni。
    - **Aggiungi产品链接开关**：Abilita/DisabilitaAggiungi产品链接。
    - **Caption 格式**：选择两种 caption 格式之一：
      - **多行模式**：Ogni行Sì一条独立 caption，系统按顺序循环Utilizzo。
      - **Spintax 模式**：Utilizzo `{选项1|选项2}` 语法生成Casuale文本。
    - **Titolo**：根据所选格式输入 caption。Supporto提及用户（如 @TikMatrixPhoneFarm）和Tag（如 #tag1 #tag2 #tag3）。详细格式Istruzioni见下文。
    - **素材Tag**：按Tag过滤素材。如果未Impostazioni，UtilizzoTutti素材。
5. 点击`开始脚本`开始Esegui。

## Caption 格式Istruzioni

系统Supporto两种 caption 格式，让您灵活控制帖子Titolo。

### 多行模式（Predefinito）

**工作原理：**
- Ogni一行代表一条独立 caption
- 系统按顺序Utilizzo caption 并循环
- 在一行内Utilizzo `###` Aggiungi换行符

**Esempio：**

```
第一条caption###支持换行
第二条caption###也支持换行
第三条caption不换行
```

**平台差异：**
- **TikTok**：Utilizzo `##` 分隔Titolo和Descrizione（例如：`标题##长描述`）
- **Instagram**：不Supporto `##`，会MostraAvviso

### Spintax 模式

**工作原理：**
- 只有一个 caption 模板
- Utilizzo `{选项1|选项2|选项3}` 语法实现Casuale变化
- Utilizzo**原生换行符**（按回车键）实现多行 caption
- 不要Utilizzo `###` - 它不会被解析

**Esempio：**

```
{今天|明天|后天}的天气真{好|不错|棒}
欢迎关注我的账号##想了解更多请点击链接
```

**平台差异：**
- **TikTok**：`##` Disponibile于分隔Titolo和Descrizione
- **Instagram**：不Supporto `##`

**Nota：** 在 Spintax 模式下，如果检测到 `###`，系统会SuggerimentoUtilizzo原生换行符。

## 素材Gestione

### 音乐Utilizzo模式

当**Pubblica方式**Impostazioni为"通过CercaUtilizzo声音"时，ScriptSupporto两种不同的音乐Aggiungi模式：

#### 1. 声音Cerca模式

输入声音名称在TikTok内Cerca音乐：

- Esempio：`original sound`，`trending music`，`viral beat`
- Script会CercaTikTok的音乐库并选择第一个匹配结果
- 这种模式适用于UtilizzoPopolari或流行声音

#### 2. 声音链接模式

输入直接的TikTok声音链接Utilizzo特定音乐：

- Supporto的链接格式：
  - 短链接：`https://www.tiktok.com/t/ZP8hvBwCb/`
  - 音乐页面：`https://www.tiktok.com/music/original-sound-7473897776619834646`
  - VM链接：`https://vm.tiktok.com/ZMhvBwCbD/`
- ScriptAutomatico检测以`https://`开头的链接并直接Apri
- 这种模式适用于Utilizzo来自特定Video的特定声音

### 素材库

- **素材Tag**：为素材AggiungiTag，在Pubblica时进行过滤。如果未指定Tag，UtilizzoTutti素材。
- **素材Titolo**：你可以为Ogni个素材ImpostazioniPersonalizzatoTitolo。如果Impostazioni了，它优先于PubblicaImpostazioni。
- **素材用户名**：你可以为Ogni个素材ImpostazioniPersonalizzato用户名。匹配用户名的素材优先Utilizzo。
- **素材Ordina**：为素材ImpostazioniOrdina号（对多图片帖子有用；数字越小越靠前）。

### 本地文件夹来源

- **文件夹结构**：

  ```text
  /素材路径/
  ├── caption.txt (可选 - 根目录兜底)
  ├── sound_names.txt (可选 - 根目录兜底)  
  ├── sound_urls.txt (可选 - 根目录兜底)
  ├── @用户名1/
  │   ├── video1.mp4
  │   ├── video2.mp4
  │   ├── image1.jpg
  │   ├── image2.jpg
  │   ├── caption.txt (可选)
  │   ├── sound_names.txt (可选)
  │   └── sound_urls.txt (可选)
  ├── @用户名2/
  │   ├── content1.mp4
  │   ├── content2.jpg
  │   ├── caption.txt (可选)
  │   ├── sound_names.txt (可选)
  │   └── sound_urls.txt (可选)
  └── ...
  ```

- **文件夹路径**：指定包含用户名子文件夹的根路径。
- **用户名子文件夹**：Ogni个用户名Richiesto有自己的子文件夹（例如，`@用户名1`，`@用户名2`）。
- **根文件夹兜底**：如果用户名文件夹不存在或用户名为空，将从根文件夹读取素材和Impostazioni。
- **Supporto格式**：
  - Video：MP4
  - 图片：JPG, PNG
- **Titolo文件**：在Ogni个用户名文件夹中Opzionale的`caption.txt`文件，用于PersonalizzatoTitolo。如果找不到，将回退到根文件夹的`caption.txt`。
  - **格式Istruzioni**：Caption.txt 仅Supporto Spintax 模板格式（与 UI 的 Spintax 模式用法相同）：
    - Configurazione一个 Spintax 模板，Utilizzo `{选项1|选项2}` 语法实现Casuale变化
    - Utilizzo原生换行符（回车键）实现多行
    - Supporto `##` Titolo/Descrizione分隔（仅 TikTok，Instagram 不Supporto）
  - **Nota**：Caption.txt 不Supporto Multiline 顺序选择模式
- **音乐文件**：
  - **sound_names.txt**：包含用于Cerca的音乐名称，Ogni行一个。按顺序读取，到末尾后循环读取。
  - **sound_urls.txt**：包含TikTok音乐链接，Ogni行一个。按顺序读取，到末尾后循环读取。
  - 如果用户名文件夹不存在，从根文件夹的音乐文件读取。
- **文件选择**：
  - 对于Video：OgnivoltePubblicaCasuale选择一个Video
  - 对于图片：根据图片数量Impostazioni选择多张图片
- **Utilizzo跟踪**：文件会被Automatico标记为已Utilizzo，避免重复。

### Caption.txt 文件格式Istruzioni

当Utilizzo**本地文件夹来源**时，`caption.txt` 文件用于Configurazione Spintax 模板。此文件可以覆盖 UI 中的TitoloConfigurazione。

**ImportanteIstruzioni**：Caption.txt 仅Supporto Spintax 模板格式，不Supporto Multiline 顺序选择模式。

#### Spintax 模板格式

Configurazione一个 Spintax 模板，Utilizzo `{选项1|选项2}` 语法生成Casuale文本变化。Utilizzo原生换行符（按回车键）实现多行：

```text
{早上好|你好|嗨}！看看这个{令人惊叹的|不可思议的|很棒的}视频！
{关注获取更多|点赞分享|标记你的朋友}
```

可能的输出：

- "早上好！看看这个令人惊叹的Video！"
- "你好！看看这个不可思议的Video！"
- "嗨！看看这个很棒的Video！"

**复杂 Spintax 变化：**

```text
{🔥|✨|🎉} {不要错过|查看|观看}这个{令人惊叹的|不可思议的|很棒的}{视频|内容|帖子}！
{关注获取更多|点赞分享|标记你的朋友} {#热门|#病毒式|#fyp} #tiktok
```

**Spintax 与 TikTok Titolo/Descrizione分隔：**

```text
{激动人心的|令人惊叹的|不可思议的}{新闻|更新|发现}##这是您需要了解的关于这个{精彩|美妙|非凡}{发展|更新|故事}的内容！
```

## Domande Frequenti

1. 为什么Attività会Fallito？
   - Automatico化ScriptSì根据TikTok/InstagramApplica的行为进行模拟Operazioni的，某些情况下可能会Fallito，例如Rete问题、Applica响应异常、意外的弹窗、Dispositivi性能问题、Dispositivi兼容性问题等。
   - 如果多volteRiprova仍然Fallito，提交工单获取Aiuto。
2. 怎么样实现无人值守Esegui？
   - 创建DispositiviGruppo, 并在Gruppo的PostImpostazioni中Utilizzo定时调度Funzionalità，Software会Ognigiorni在指定时间Automatico创建并EseguiPubblicaAttività。
3. 怎么实现幻灯片帖子的Ordina？
   - 在素材库中为Ogni个图片素材ImpostazioniOrdina号，数字越小越靠前。Pubblica时Script会根据Ordina号选择图片。
   - 你也可以In MassaImporta素材时Utilizzo符合下面规范的文件名来ImpostazioniOrdina号：
     - `sort_1_图片描述.png`
     - `sort_2_图片描述.png`
     - `sort_3_图片描述.png`
4. 怎么为Ogni个AccountUtilizzo不同的素材,Titolo和音乐？
   - Utilizzo本地文件夹时，为Ogni个用户名创建子文件夹，在子文件夹中创建`caption.txt`，`sound_names.txt`或`sound_urls.txt`文件来定义PersonalizzatoTitolo和音乐。Script会优先Utilizzo匹配用户名的文件。
   - Utilizzo素材库时，为Ogni个素材Impostazioni用户名字段,Titolo字段,Script会优先Utilizzo匹配用户名的素材和Titolo。
5. Supporto哪些素材格式？
   - Video：MP4
   - 图片：JPG,PNG

6. 图片和Video素材可以重复Utilizzo吗？

  不可以。为保护您的Account，系统会在素材Pubblica后Automatico将其标记为已Utilizzo，避免被重复选用。这样可以减少重复Pubblica的风险，降低Account被风控或封禁的可能性。
