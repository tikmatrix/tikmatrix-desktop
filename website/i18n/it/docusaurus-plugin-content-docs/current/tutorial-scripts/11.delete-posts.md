# Elimina Post

Elimina PostScript用于根据Visualizzazioni量Elimina Post。

## 步骤

1. 选择要EseguiScript的Dispositivi。
2. 点击 `脚本` > `删除帖子`。
3. ConfigurazioneImpostazioni Attività：
    - **最大Visualizzazioni量**：输入您想要Elimina的帖子的最大Visualizzazioni量。
4. 点击 `开始脚本` 开始Esegui。

## Nota

- Elimina PostScript可能并不总SìSuccesso；如有需要RiprovaFallito的Attività。
