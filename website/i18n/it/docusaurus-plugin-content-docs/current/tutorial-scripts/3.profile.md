# Profilo

ProfiloScript用于ImpostazioniAccount的Profilo。

## 步骤

1. 选择要ImpostazioniProfilo的Dispositivi。
2. 点击 `脚本` - `个人资料` 按钮。
3. ConfigurazioneImpostazioni Attività：
    - **昵称**: Ogni行一个昵称。
    - **用户名**: Ogni行一个用户名。
    - **Bio**: Ogni行一个Bio（Supporto提及用户和换行）。
      - 要在Bio中Aggiungi换行，请在文本中Utilizzo `\n`。例如：`第一行\n第二行\n第三行`
    - **Avatar路径**: Avatar图片文件夹的路径。
    - **选择顺序**: 选择Ogni个Attività如何选取Contenuto：
      - **Casuale**: 为Ogni个AttivitàCasuale选择Disponibile选项（Predefinito）。
      - **顺序**: 根据Attività计数按顺序选择Contenuto。Utilizzo此模式可确保Ogni个Account获得唯一信息。
4. 点击 `开始脚本` 按钮Avvia Attività。

## 选择顺序Istruzioni

### Casuale模式

- Ogni个AttivitàCasuale选择Contenuto（昵称、用户名、Bio、Avatar）
- 同一Contenuto可能在不同Account间重复Utilizzo
- 适用于不需要保证Contenuto唯一性的场景

### 顺序模式

- 按顺序选择Contenuto：第1个AttivitàUtilizzo第1项，第2个AttivitàUtilizzo第2项，依此类推
- 当Tutti项目都Utilizzo完后，会循环回到开头
- **ImportanteSuggerimento**: 要确保Ogni个Account获得唯一信息，请确保提供的Contenuto数量大于单volteIn MassaEsegui的Account总数
  - Esempio：如果Esegui 100 个Account，请至少提供 100 个不同的昵称、用户名、Bio和Avatar图片

## Nota事项

- 在ImpostazioniProfilo之前最好Svuota相册，因为Script会Utilizzo相册中的第一张照片作为Avatar。
