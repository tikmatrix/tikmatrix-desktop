# Accedi

AccediScript用于Accedi社交媒体Account。

## 步骤

1. 确保Account已Aggiungi到Account列表中（点击`账户`AggiungiAccount）, 并选择你需要Accedi该Account的Telefono序号。
2. 单选或者多选要EseguiAccediScript的Dispositivi。
3. 点击`脚本` > `登录`。
4. 点击`开始脚本`开始Esegui。

## Accedi优先级

系统Utilizzo以下Accedi优先级：

1. **邮箱+密码**：如果AccountConfigurazione了邮箱，将优先Utilizzo邮箱进行Accedi。
2. **用户名+密码**：如果没有Configurazione邮箱，则Utilizzo用户名进行Accedi。

## 2FA Supporto（仅限 IgMatrix）

IgMatrix SupportoAutomatico双因子认证（2FA）Accedi：

### 2FA 密码格式

要Utilizzo 2FA，请按以下格式Impostazioni密码：

```plaintext
你的密码:你的2FA密钥
```

**Esempio：**

```plaintext
mypassword123:JBSWY3DPEHPK3PXP
```

### 2FA 工作原理

1. **密码解析**：系统Automatico检测密码中Vuoi包含冒号（`:`）分隔符。
2. **验证码生成**：当需要 2FA 验证时，系统Utilizzo您的密钥向我们的 2FA 服务请求 TOTP 验证码。
3. **Automatico输入**：生成的 6 位数字验证码会Automatico输入到 Instagram 的 2FA 验证框中。
4. **无缝Accedi**：整个 2FA 过程无需Manuale干预，AutomaticoCompletato。

### 2FA 密钥Istruzioni

2FA 密钥通常Sì一个 32 位的 Base32 编码字符串，您在 Instagram 中Impostazioni 2FA 时会获得此密钥。这与 Google Authenticator 或 Authy 等验证器ApplicaUtilizzo的密钥相同。

## Nota事项

- TikTok Accedi不Sì完全100%Automatico的。你可能需要Manuale输入邮箱验证码或Completato其他验证步骤。我们也无法解决 TikTok 的Accedi挑战（如图片验证码、人机验证等）以及ErroreSuggerimento: Accedi太频繁的问题。
- Instagram 可以通过 2FA Accedi实现100%全AutomaticoAccedi。对于Abilita了 2FA 的 Instagram Account，请确保在密码中Utilizzo正确的密钥格式。
