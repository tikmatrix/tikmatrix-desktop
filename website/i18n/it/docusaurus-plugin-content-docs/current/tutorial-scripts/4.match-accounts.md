# Abbina Account

Abbina AccountScript用于从Telefono中识别Account并将其Aggiungi到Account列表中。

## 步骤

1. 选择要Abbina Account的Dispositivi。
2. 点击 `脚本` - `匹配账号` 按钮。
3. 点击 `开始` 按钮开始Abbina AccountAttività。
4. Script将Automatico从Telefono中识别Account并将其Aggiungi到Account列表中。

## Nota事项

* Abbina AccountScript不Sì100%Successo的，如果某些Account未被识别，您可能需要多volteEseguiScript。
