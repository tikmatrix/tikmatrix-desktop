# TikTok installeren

We raden aan om de onderstaande specifieke TikTok-versies te downloaden, aangezien nieuwere versies mogelijk niet compatibel zijn. Ondersteunde versies worden in de groep aangekondigd na updates.

## TikTok APK downloaden

- Wereldwijde versie: [com.zhiliaoapp.musically](https://apkpure.com/tiktok-musically-2024/com.zhiliaoapp.musically)
- Aziatische versie: [com.ss.android.ugc.trill](https://apkpure.com/tiktok/com.ss.android.ugc.trill)

## Installatiestappen

1. Download de APK via de bovenstaande links.
2. Selecteer alle apparaten in TikMatrix.
3. Klik op `General` > `Install APK` en kies het gedownloade APK-bestand.
4. Wacht tot de installatie is voltooid (kan 1-3 minuten per apparaat duren, afhankelijk van het model; klik niet herhaaldelijk).
5. Klik na de installatie op `Settings` en controleer of de TikTok-versie overeenkomt met je geïnstalleerde app.
6. Klik op `Open TikTok` om te controleren of de app correct is geïnstalleerd.
7. Zorg ervoor dat de taal van TikTok is ingesteld op Engels; anders werken scripts mogelijk niet goed. Wijzig indien nodig de taal in de TikTok-instellingen.
