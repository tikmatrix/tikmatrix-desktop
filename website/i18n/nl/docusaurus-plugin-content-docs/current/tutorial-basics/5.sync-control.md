# Synchrone besturing

Synchrone besturing stelt je in staat om bewerkingen over meerdere apparaten te synchroniseren.

## Hoe te gebruiken

1. Open het scherm van één telefoon.
2. Selecteer meerdere apparaten uit de lijst aan de linkerkant.
3. Alle schermbewerkingen worden gesynchroniseerd naar de geselecteerde apparaten.
