# Groep aanmaken

Een **Groep** is een verzameling apparaten, waardoor het gemakkelijker wordt om meerdere apparaten samen te beheren en te bedienen.

## Hoe een groep aan te maken

1. Klik op de knop `Add Group` in de linker zijbalk.
2. Voer een groepsnaam in.
3. Druk op `Enter` om de groep op te slaan.
4. Selecteer apparaten om ze naar de groep te verplaatsen.
