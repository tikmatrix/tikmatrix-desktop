# Registreren

:::warning Script Stopgezet
Het Registreer-script is **stopgezet** en wordt **niet langer onderhouden**. Deze functie is niet meer beschikbaar in TikMatrix.
:::

~~Het Registreer-script wordt gebruikt om nieuwe TikTok-accounts aan te maken.~~

## ~~Stappen~~

~~1. Selecteer het apparaat om het script uit te voeren.~~
~~2. Klik op `Script` > `Registreren`.~~
~~3. Configureer de taakinstellingen:~~
    ~~- **Registratie-e-mails**: Voer één e-mailadres per regel in.~~
    ~~- **Registratiewachtwoord**: Voer het wachtwoord in voor de nieuwe accounts.~~
~~4. Klik op `Script Starten` om te beginnen.~~

## ~~Opmerking~~

~~- Registratie is niet volledig automatisch. U moet de e-mail handmatig verifiëren of CAPTCHA-uitdagingen voltooien.~~
