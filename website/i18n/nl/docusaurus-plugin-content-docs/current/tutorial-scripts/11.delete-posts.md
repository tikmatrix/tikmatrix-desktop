# Berichten Verwijderen

Het Berichten Verwijderen-script wordt gebruikt om berichten te verwijderen op basis van hun weergaveaantal.

## Stappen

1. Selecteer het apparaat om het script uit te voeren.
2. Klik op `Script` > `Berichten Verwijderen`.
3. Configureer de taakinstellingen:
    - **Maximum Weergaveaantal**: Voer het maximale weergaveaantal in voor berichten die u wilt verwijderen.
4. Klik op `Script Starten` om te beginnen.

## Opmerking

- Het berichten verwijderen-script slaagt mogelijk niet altijd; probeer mislukte taken opnieuw indien nodig.
