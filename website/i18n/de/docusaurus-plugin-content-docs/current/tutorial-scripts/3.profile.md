# Profil

Das Profilskript wird zum Konfigurieren des Kontoprofils verwendet.

## Schritte

1. Wählen Sie Geräte aus, um das Profil zu konfigurieren.
2. Klicken Sie auf die Schaltfläche `Skripte` - `Profil`.
3. Konfigurieren Sie Aufgabenparameter:
    - **Spitznamen**: Ein Spitzname pro Zeile.
    - **Benutzernamen**: Ein Benutzername pro Zeile.
    - **Biografien**: Eine Biografie pro Zeile (unterstützt Benutzer-Erwähnungen und Zeilenumbrüche).
      - Um Zeilenumbrüche in einer Biografie hinzuzufügen, verwenden Sie `\n` im Text. Beispiel: `Erste Zeile\nZweite Zeile\nDritte Zeile`
    - **Avatar-Pfad**: Pfad zum Ordner mit Avatar-Bildern.
    - **Auswahlreihenfolge**: Wählen Sie die Methode zur Auswahl von Inhalten für jede Aufgabe:
      - **Zufällig**: Zufällige Auswahl aus verfügbaren Optionen für jede Aufgabe (Standard).
      - **Sequenziell**: Auswahl von Inhalten in der Reihenfolge basierend auf dem Aufgabenzähler. Verwenden Sie diesen Modus, um eindeutige Informationen für jedes Konto sicherzustellen.
4. Klicken Sie auf die Schaltfläche `Skript ausführen`, um die Aufgabe zu starten.

## Leitfaden zur Auswahlreihenfolge

### Zufallsmodus

- Inhalte (Spitznamen, Benutzernamen, Biografien, Avatare) werden für jede Aufgabe zufällig ausgewählt
- Derselbe Inhalt kann mehrmals für verschiedene Konten verwendet werden
- Geeignet, wenn Inhalts-Eindeutigkeit nicht kritisch ist

### Sequenzieller Modus

- Inhalte werden in der Reihenfolge ausgewählt: 1. Aufgabe erhält 1. Element, 2. Aufgabe erhält 2. Element usw.
- Wenn alle Elemente verwendet wurden, beginnt die Auswahl von vorne
- **Wichtig**: Um eindeutige Informationen für jedes Konto zu gewährleisten, stellen Sie sicher, dass Sie mehr Elemente als die Gesamtzahl der Konten in einem Batch-Lauf bereitstellen
  - Beispiel: Wenn Sie 100 Konten ausführen, stellen Sie mindestens 100 eindeutige Spitznamen, Benutzernamen, Biografien und Avatar-Bilder bereit

## Hinweise

- Es ist besser, das Album vor der Profilkonfiguration zu löschen, da das Skript das erste Foto im Album als Avatar verwendet.
