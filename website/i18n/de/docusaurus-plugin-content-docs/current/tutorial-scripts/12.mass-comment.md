# Massen-Kommentare

:::warning
Diese Funktion ist im [Super Marketing Skript](./14.super-marketing.md) integriert. Verwenden Sie Super Marketing für leistungsfähigere und flexiblere Möglichkeiten für Massen-Kommentare und Engagement-Verwaltung.
:::

Massen-Kommentare werden verwendet, um Interaktionen auf einer großen Anzahl von TikTok-Posts zu platzieren – geeignet für Kontoentwicklung, Werbekampagnen und Engagement-Steigerung. Das System unterstützt vorgefertigte Vorlagen und Echtzeitgenerierung und simuliert die menschliche Aktionshäufigkeit, um das Erkennungsrisiko zu verringern.

## Verwendung in Super Marketing

1. Öffnen Sie das Super Marketing Skript.
2. Wählen Sie den Datenquellentyp: „Post-URL-Liste".
3. Fügen Sie Post-Links ein oder importieren Sie eine `.txt` (einen Link pro Zeile). Nach dem Import können Sie „Hinzufügen" (Append) oder „Ersetzen" (Replace) wählen.
4. Wählen Sie eine Datennutzungsstrategie:

   - Gemeinsamer Pool: Alle Bots beziehen Ziele aus einem gemeinsamen Pool.

   - Einmalige Nutzung: Jeder Bot verarbeitet eindeutige Ziele, sodass jeder Post nur einen Kommentar von einem Konto erhält.

5. Legen Sie fest, wie viele Kommentare pro Post veröffentlicht werden (normalerweise 1, um Duplikate zu vermeiden).
6. Konfigurieren Sie die Kommentarparameter:

   - Vorgefertigte Vorlagen: Fügen Sie mehrere Zeilen oder Absätze fertiger Texte ein. Jede Zeile/Absatz wird als separate Vorlage gezählt.

   - Kommentar-Reihenfolge: Wählen Sie „Zufall" oder „Nacheinander".

   - Zufall: Jedes Mal wird eine zufällige Vorlage beim Senden ausgewählt.

   - Nacheinander: Templates werden in der Einfüge-/Importreihenfolge verwendet (praktisch für reproduzierbare Deduplizierungsstrategien).

   - Emoji-Einfügung: Aktivieren Sie das automatische Hinzufügen von Emojis für mehr Natürlichkeit.

   - Gerätenummer einfügen (optional): Hinzufügen der Gerätenummer zum Kommentar für Verfolgung (mit Bedacht verwenden – hauptsächlich für Debugging/Audit).

   - Echtzeit-Kommentargenerierung (optional, ChatGPT API):

      - Wenn aktiviert, kann das Skript ChatGPT oder ähnliche APIs aufrufen, um kontextabhängige und eindeutige Kommentare basierend auf Post-Titel/Beschreibung oder verfügbarem Kontext zu generieren – geeignet, wenn viel eindeutiger Inhalt benötigt wird.

      - Achtung: Für die Arbeit ist ein gültiger API-Schlüssel erforderlich; beachten Sie Anforderungslimitierungen, Kosten und Datenschutzanforderungen (senden Sie keine sensiblen Daten an Drittanbieter).

   - Zwei häufige Möglichkeiten, um Kommentar-Eindeutigkeit sicherzustellen:

      1. Rotation vorgefertigter Vorlagen in Reihenfolge (jede Vorlage wird einmal verwendet, bis sie erschöpft ist).

      2. Echtzeit-Generierung eindeutiger Kommentare mit ChatGPT.

7. Konfigurieren Sie weitere Skript-Parameter (Parallelität, Aufgabenintervall, Konten-Pool usw.) und starten Sie das Skript.

:::tip Über Aufgabenintervalle
Legen Sie Mindest- und Höchstintervall fest und aktivieren Sie zufällige Verzögerungen. Zeitlich verteilte Aktionen reduzieren das Risiko synchroner Spitzen und Plattformbeschränkungen.
:::

## Empfehlungen und Hinweise

- Testen Sie die Strategie auf einer kleinen Auswahl, um die Kommentar-Qualität und Plattformreaktion zu überprüfen.
- Kommentare sollten vielfältig und konform mit Plattformrichtlinien sein; vermeiden Sie die Massensendung identischer Meldungen.
- Bei Verwendung von Echtzeitgenerierung überwachen Sie die API-Nutzung und Kosten, geben Sie keine vertraulichen Daten weiter.
- Wenn Teil der Ziele nicht verarbeitet wurde, überprüfen Sie die Protokolle und starten Sie erneut – bereits ausgeführte Zeilen werden übersprungen.
