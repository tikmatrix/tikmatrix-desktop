# Live-Streams fördern

Mit dem Live-Stream-Förderungs-Skript können Sie Farm-Konten verwalten, um Ziel-Live-Streams anzuschauen und optional zu liken oder zu kommentieren. Sie können die Anzeigedauer und Like-/Kommentar-Intervalle anpassen, um das Förderungsverhalten zu personalisieren.

## Schritte

1. Wählen Sie Geräte zum Ausführen des Skripts.
2. Klicken Sie auf `Skripte` > `Live-Streams fördern`.
3. Konfigurieren Sie die Aufgabenparameter:
    - **Benutzernamen**: Geben Sie Ziel-Benutzernamen ein, einen pro Zeile.
    - **Zugriffsmethode**: Wählen Sie **Suche** oder **Direkt**.
    - **Förderungsoptionen**: Aktivieren Sie Likes und/oder Kommentare.
    - **Fan-Club beitreten**: Aktivieren Sie den Beitritt zum Fan-Club des Streamers, falls verfügbar.
    - **Tägliche Anwesenheit**: Aktivieren Sie die tägliche Anwesenheit während des Live-Streams.
    - **Like-Intervall**: Legen Sie das Intervall zwischen jeder Like-Operation fest.
    - **Like-Klicks**: Legen Sie die Anzahl der Like-Button-Klicks fest.
    - **Kommentar-Intervall**: Legen Sie das Intervall zwischen jeder Kommentaroperation fest.
    - **Kommentarinhalt**: Geben Sie einen Kommentar pro Zeile ein.
    - **Emoji einfügen**: Wählen Sie, ob Emojis in Kommentare eingefügt werden sollen.
    - **Kommentar-Reihenfolge**: Wählen Sie zufällige oder sequenzielle Kommentar-Reihenfolge.
    - **Kommentare pro Konto**: Legen Sie die Anzahl der Kommentare fest, die von jedem Konto veröffentlicht werden.
    - **Anzeigendauer**: Legen Sie die Verweilzeit im Live-Stream fest.
    - **Aufgabenintervall**: Legen Sie das Intervall zwischen jeder Aufgabe fest.
4. Klicken Sie auf `Skript starten`, um die Ausführung zu beginnen.

## Hinweise

- Das Live-Stream-Förderungs-Skript ist nicht immer erfolgreich; wiederholen Sie ggf. fehlgeschlagene Aufgaben.
