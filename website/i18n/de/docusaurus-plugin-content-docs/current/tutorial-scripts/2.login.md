# Anmelden

Das Anmeldeskript wird zum Anmelden bei Social-Media-Konten verwendet.

## Schritte

1. Stellen Sie sicher, dass Konten zur Kontoliste hinzugefügt wurden (klicken Sie auf `Konten` zum Hinzufügen von Konten), und wählen Sie den Geräteindex aus, auf dem Sie sich bei diesem Konto anmelden möchten.
2. Wählen Sie ein oder mehrere Geräte aus, um das Anmeldeskript auszuführen.
3. Klicken Sie auf `Skripte` > `Anmelden`.
4. Klicken Sie auf `Skript ausführen`, um zu beginnen.

## Anmeldepriorität

Das System verwendet die folgende Anmeldepriorität:

1. **E-Mail + Passwort**: Wenn für das Konto eine E-Mail konfiguriert ist, wird diese zuerst für die Anmeldung verwendet.
2. **Benutzername + Passwort**: Wenn keine E-Mail konfiguriert ist, wird der Benutzername für die Anmeldung verwendet.

## 2FA-Unterstützung (nur IgMatrix)

IgMatrix unterstützt automatische Anmeldung mit Zwei-Faktor-Authentifizierung (2FA):

### Passwortformat für 2FA

Um 2FA zu verwenden, formatieren Sie Ihr Passwort wie folgt:

```plaintext
ihr_passwort:ihr_2fa_geheimschlüssel
```

**Beispiel:**

```plaintext
mypassword123:JBSWY3DPEHPK3PXP
```

### Wie 2FA funktioniert

1. **Passwort-Parsing**: Das System erkennt automatisch, ob Ihr Passwort einen Doppelpunkt-Trenner (`:`) enthält.
2. **Token-Generierung**: Wenn 2FA erforderlich ist, verwendet das System Ihren Geheimschlüssel, um einen TOTP-Code von unserem 2FA-Service anzufordern.
3. **Automatische Eingabe**: Der generierte 6-stellige Code wird automatisch in das Instagram-2FA-Feld eingegeben.
4. **Nahtlose Anmeldung**: Der gesamte 2FA-Prozess läuft automatisch ohne manuelle Eingriffe ab.

### 2FA-Geheimschlüssel

Der 2FA-Geheimschlüssel ist normalerweise eine 32-Zeichen-Zeichenfolge in Base32-Codierung, die Sie beim Einrichten von 2FA in Instagram erhalten. Dies ist derselbe Schlüssel, der von Authenticator-Apps wie Google Authenticator oder Authy verwendet wird.

### Wo Sie Instagram-Konten mit 2FA-Unterstützung erhalten

Sie können Instagram-Konten mit 2FA-Unterstützung auf der Website kaufen: [AccsMarket](https://accsmarket.com/en/?ref=802647)

## Hinweise

- Die TikTok-Anmeldung ist nicht zu 100% automatisch. Es kann erforderlich sein, manuell einen E-Mail-Bestätigungscode einzugeben oder andere Verifizierungsschritte durchzuführen. Wir können TikTok-spezifische Prüfungen (z. B. Bild-CAPTCHAs oder Bot-Prüfungen) nicht umgehen oder einige Fehlermeldungen wie „zu viele Anmeldeversuche" beheben.
- Instagram kann vollständig automatische Anmeldung über 2FA gewährleisten. Für Instagram-Konten mit aktiviertem 2FA stellen Sie sicher, dass Sie das richtige Geheimschlüsselformat im Passwort verwenden.
