# Konten-Aufwärmung

Skript zur Konten-Aufwärmung wird verwendet, um Konten durch Schlüsselwortsuche, Videoansichten, Abos, Likes, Kommentare und Favoriten zu aktivieren.

## Schritte

1. Wählen Sie Geräte zum Ausführen des Skripts.
2. Klicken Sie auf `Skripte` > `Konten-Aufwärmung`.
3. Konfigurieren Sie die Aufgabenparameter:
   - **Anzeigedauer**: Zeit zum Ansehen jedes Videos.
   - **Suchschlüsselwörter**: Themen zum Suchen von Videos.
   - **Aufgabendauer**: Gesamtdauer der Aufwärmaufgabe.
   - **Kommentarinhalt**: Verwendung vorgefertigter Kommentare oder ChatGPT-generierte Kommentare basierend auf Post-Titeln.
   - **Interaktionswahrscheinlichkeit**: Wahrscheinlichkeit für Abos, Likes, Kommentare und Favoriten.
4. Klicken Sie auf `Skript starten`, um die Ausführung zu beginnen.

## Hinweise

- Das Aufwärmskript ist nicht immer erfolgreich; wiederholen Sie ggf. fehlgeschlagene Aufgaben.
- Zeitplanung wird unterstützt, Sie können Aufwärmaufgaben für die Ausführung zu bestimmten Zeiten einplanen.
- Werbung, Live-Streams und bereits gelikete Videos können übersprungen werden.
