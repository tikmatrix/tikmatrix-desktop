# Konten zuordnen

Das Skript zum Zuordnen von Konten wird verwendet, um Konten vom Telefon zu erkennen und sie zur Kontoliste hinzuzufügen.

## Schritte

1. Wählen Sie Geräte zum Zuordnen von Konten aus.
2. Klicken Sie auf die Schaltfläche `Skripte` - `Konten zuordnen`.
3. Klicken Sie auf die Schaltfläche `Starten`, um die Aufgabe zum Zuordnen von Konten zu beginnen.
4. Das Skript erkennt automatisch Konten vom Telefon und fügt sie zur Kontoliste hinzu.

## Hinweise

* Das Skript zum Zuordnen von Konten ist nicht zu 100% erfolgreich. Wenn einige Konten nicht erkannt wurden, müssen Sie das Skript möglicherweise mehrmals ausführen.
