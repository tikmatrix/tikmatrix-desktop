# Benutzer sammeln

Skript zum Sammeln von Benutzern wird verwendet, um TikTok-Benutzernamen zu sammeln. Die Benutzerssammel-Funktion unterstützt drei Modi: Sammlung einer Liste von Followern eines angegebenen Benutzers, Sammlung einer Liste von Folgende-Benutzer eines angegebenen Benutzers oder Sammlung von Benutzern durch Schlüsselwortsuche.

## Schritte

1. Wählen Sie Geräte zum Ausführen des Skripts. Wenn mehrere Geräte ausgewählt sind, werden Sammelaufgaben gleichmäßig auf die ausgewählten Geräte verteilt.
2. Klicken Sie auf `Skripte` > `Benutzer sammeln`.
3. Konfigurieren Sie die Aufgabenparameter:
    - **Sammel-Modus**: Wählen Sie einen Benutzersammel-Modus.
      - **Nach Followern sammeln**: Sammlung von Benutzern aus der Follower-Liste angegebener Konten.
      - **Nach folgenden sammeln**: Sammlung von Benutzern aus der Folgende-Liste angegebener Konten.
      - **Nach Schlüsselwörtern sammeln**: Suche und Sammlung von Benutzern mit Schlüsselwörtern.
    - **Zielkonten-Benutzernamen**: Geben Sie Benutzernamen von Konten zum Sammeln ein, einen pro Zeile (für Follower- und Folgende-Modi).
    - **Suchschlüsselwörter**: Geben Sie Schlüsselwörter zur Benutzersuche ein (für Schlüsselwort-Modus).
    - **Maximale Sammelmenge**: Legen Sie die maximale Anzahl von Benutzern zum Sammeln fest.

4. Klicken Sie auf `Skript starten`, um die Ausführung zu beginnen.

## Hinweise

- Gesammelte Benutzerdaten werden in eine Datei gespeichert, je nach Sammel-Modus:
  - Follower-Modus: `exported_users_followers_tiktok_<benutzername>_<menge>.txt`
  - Folgende-Modus: `exported_users_following_tiktok_<benutzername>_<menge>.txt`
  - Schlüsselwort-Modus: `exported_users_search_tiktok_<schlüsselwort>_<menge>.txt`
- TikTok begrenzt die Anzahl der sichtbaren Benutzer, daher kann jeder Vorgang nur etwa 50 Einträge abrufen.
