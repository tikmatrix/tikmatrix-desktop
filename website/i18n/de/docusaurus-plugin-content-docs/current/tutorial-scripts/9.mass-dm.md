# Massen-Direktnachrichten

:::warning
Diese Funktion ist im [Super Marketing Skript](./14.super-marketing.md) integriert. Verwenden Sie Super Marketing für leistungsfähigere und flexiblere Möglichkeiten für Massen-Direktnachrichten und Nachrichtenverwaltung.
:::

Massen-Direktnachrichten (Mass DM) werden verwendet, um Kontakt durch Versenden von Direktnachrichten in großem Umfang herzustellen – geeignet für Werbung, Kundenfeedback und Benutzer-Reaktivierung. Das System unterstützt vorgefertigte Vorlagen und Echtzeit-Nachrichtengenerierung und versucht, den menschlichen Senderhythmus zu simulieren, um das Erkennungsrisiko zu verringern.

## Verwendung in Super Marketing

1. Öffnen Sie das Super Marketing Skript.
2. Wählen Sie den Datenquellentyp: „Benutzernamen-Liste (Zielgruppe)".
3. Fügen Sie eine Liste von Benutzernamen ein oder importieren Sie eine `.txt` Datei (ein Benutzername pro Zeile). Nach dem Import können Sie „Hinzufügen" (Append) oder „Ersetzen" (Replace) wählen.
4. Wählen Sie eine Datennutzungsstrategie:

   - Gemeinsamer Pool: Alle Bots beziehen Ziele aus einem gemeinsamen Pool.

   - Einmalige Nutzung: Jeder Bot verarbeitet eindeutige Ziele, sodass jedes Ziel nur eine Nachricht von einem Konto erhält.

5. Legen Sie fest, wie viele Nachrichten pro Ziel gesendet werden (normalerweise 1, um Spam zu vermeiden).
6. Konfigurieren Sie die Nachrichtenparameter:

   - Vorgefertigte Vorlagen: Fügen Sie mehrere Zeilen oder Absätze fertiger Texte ein. Jede Zeile/Absatz wird als separate Vorlage gezählt.

   - Reihenfolge der Nutzung: Wählen Sie „Zufall" oder „Nacheinander".

   - Zufall: Jedes Mal wird eine zufällige Vorlage beim Senden ausgewählt.

   - Nacheinander: Templates werden in der Einfüge-/Importreihenfolge verwendet (praktisch für reproduzierbare Deduplizierungsstrategien).

   - Emoji-Einfügung: Aktivieren Sie das automatische Hinzufügen von Emojis für mehr Natürlichkeit.

   - Echtzeit-Nachrichtengenerierung (optional, ChatGPT API):

      - Wenn aktiviert, kann das Skript ChatGPT oder ähnliche APIs aufrufen, um personalisierte Nachrichten basierend auf Benutzernamen oder verfügbarem Kontext zu generieren – nützlich, wenn viele eindeutige Nachrichten benötigt werden.

      - Achtung: Für die Arbeit ist ein gültiger API-Schlüssel erforderlich; beachten Sie Anforderungslimitierungen, Kosten und Datenschutzanforderungen (senden Sie keine sensiblen Daten).

   - Zwei häufige Möglichkeiten, um Nachrichten-Eindeutigkeit sicherzustellen:

   1. Rotation vorgefertigter Templates in Reihenfolge (jede Vorlage wird einmal verwendet, bis sie erschöpft ist).

   2. Echtzeit-Generierung eindeutiger Nachrichten über ChatGPT.

7. Wählen Sie die Methode zum Öffnen des Benutzerprofils (falls erforderlich): „Suche" oder „Direktzugang".
8. Konfigurieren Sie das Aufgabenintervall (Min/Max in Minuten), Parallelität und Konten-Pool, dann starten Sie das Skript.

:::tip Über Aufgabenintervalle
Legen Sie Mindest- und Höchstintervall fest und aktivieren Sie zufällige Verzögerungen. Zeitlich verteilte Aktionen reduzieren das Risiko synchroner Spitzen und Plattformbeschränkungen.
:::

## Empfehlungen und Hinweise

- Gestalten Sie Nachrichtentexte vielfältig und konform mit Plattformrichtlinien; vermeiden Sie die Massensendung identischer Nachrichten.
- Bei Verwendung von Echtzeitgenerierung überwachen Sie die API-Nutzung und Kosten, und geben Sie keine vertraulichen Informationen weiter.
- Wenn Teil der Ziele nicht verarbeitet wurde, überprüfen Sie die Protokolle und starten Sie erneut – bereits verarbeitete Zeilen werden übersprungen.
