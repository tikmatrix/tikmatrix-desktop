# Datenschutzeinstellungen

Skript für **Datenschutzeinstellungen** wird verwendet, um Datenschutzeinstellungen von TikTok-Veröffentlichungen massenhaft zu aktualisieren, basierend auf einem Aufrufe-Filter.

:::info
Dieses Skript ist nur für TikTok-Konten verfügbar.
:::

## Schritte

1. Wählen Sie ein Gerät zum Ausführen des Skripts.
2. Klicken Sie auf `Skripte` > `Datenschutzeinstellungen`.
3. Konfigurieren Sie die Aufgabenparameter:
    - **Veröffentlichungs-Filter-Parameter**:
      - **Maximale Aufrufe**: Geben Sie die maximale Anzahl von Aufrufen ein, um Veröffentlichungen zu filtern. Nur Veröffentlichungen mit Aufrufen ≤ diese Zahl werden aktualisiert.
      - **Maximale Anzahl von Veröffentlichungen zur Verarbeitung**: Geben Sie die maximale Anzahl von Veröffentlichungen zur Verarbeitung ein. Legen Sie 0 für unbegrenzte Verarbeitung fest.
    - **Datenschutzeinstellungen-Parameter**:
      - **Wer kann dieses Video ansehen**: Wählen Sie eine Option:
        - `Jeder`: Das Video kann von jedem Benutzer angesehen werden
        - `Freunde`: Das Video kann nur von Freunden angesehen werden
        - `Nur Sie`: Das Video können Sie nur selbst ansehen
    - **Schalter-Parameter**:
      - **Kommentare erlauben**: Aktivieren oder deaktivieren Sie Kommentare zur Veröffentlichung.
      - **Inhalts-Wiederverwendung erlauben**: Aktivieren oder deaktivieren Sie die Inhalts-Wiederverwendung, einschließlich Duett, Stitch, Sticker und Zu Story hinzufügen.
4. Klicken Sie auf `Skript starten`, um die Ausführung zu beginnen.

## So funktioniert es

Das Skript folgt diesem Arbeitsablauf:

1. Zur Video-Registerkarte navigieren
2. Veröffentlichungen scannen und Aufrufe analysieren
3. Veröffentlichungen suchen, die Filter-Kriterien entsprechen (Aufrufe ≤ max_views)
4. Zur Veröffentlichungs-Details-Seite navigieren
5. Auf die Schaltfläche "Datenschutzeinstellungen" klicken
6. Auf das Datenschutzeinstellungs-Dialog warten
7. Anzeigeberechtigungen gemäß Ihrer Auswahl festlegen
8. Intelligentes Umschalten der Option "Kommentare erlauben"
9. Intelligentes Umschalten der Option "Wiederverwendung erlauben"
10. Zurück zur Liste und Fortsetzung der nächsten Veröffentlichung
11. Wiederholen bis alle übereinstimmenden Veröffentlichungen verarbeitet sind

## Hinweise

- Das Datenschutzeinstellungs-Skript ist nicht immer erfolgreich; wiederholen Sie ggf. fehlgeschlagene Aufgaben.
- Das Skript verarbeitet nur Veröffentlichungen, die dem Aufrufe-Filter entsprechen.
- Änderungen werden sofort angewendet und können nicht über das Skript rückgängig gemacht werden.
- Stellen Sie sicher, dass Sie die Datenschutzeinstellungen überprüft haben, bevor Sie das Skript für mehrere Veröffentlichungen ausführen.
