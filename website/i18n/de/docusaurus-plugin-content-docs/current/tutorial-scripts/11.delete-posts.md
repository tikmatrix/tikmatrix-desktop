# Posts löschen

Skript zum Löschen von Posts wird verwendet, um Posts basierend auf der Anzahl der Aufrufe zu löschen.

## Schritte

1. Wählen Sie ein Gerät zum Ausführen des Skripts.
2. Klicken Sie auf `Skripte` > `Posts löschen`.
3. Konfigurieren Sie die Aufgabenparameter:
    - **Maximale Aufrufe**: Geben Sie die maximale Anzahl von Aufrufen für Posts ein, die Sie löschen möchten.
4. Klicken Sie auf `Skript starten`, um die Ausführung zu beginnen.

## Hinweis

- Das Post-Lösch-Skript ist nicht immer erfolgreich; wiederholen Sie ggf. fehlgeschlagene Aufgaben.
