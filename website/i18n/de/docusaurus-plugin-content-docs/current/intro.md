# Tutorial-Einführung

Lernen Sie **TikMatrix** und **IgMatrix** in weniger als 5 Minuten kennen und erfahren Sie mehr über ihre wichtigsten Funktionen.

## Softwareeinführung

**TikMatrix** ist eine Desktop-PC-Anwendung, die TikTok-Aufgaben auf Android-Geräten automatisiert.  
**IgMatrix** ist ein Tool zur Automatisierung von Instagram-Aufgaben auf Android.

**Hinweis:** **TikMatrix** und **IgMatrix** sind zwei separate Anwendungen und können nicht gleichzeitig auf demselben Computer ausgeführt werden.

## Funktionen

- **Multi-Account-Wechsel**: Verwalten Sie mehrere Konten auf einem Gerät mit automatischem Wechsel bei der Aufgabenausführung.
- **Unterstützung für geklonte Apps**: Unterstützung für geklonte TikTok/Instagram-Apps mit benutzerdefinierten Package Names, sodass verschiedene Konten unterschiedliche Klone verwenden und automatisch die Ziel-App öffnen können.
- ~~**Automatische Registrierung**~~ **[EINGESTELLT]**
- **Automatische Anmeldung**: Instagram unterstützt automatische Anmeldung mit 2FA; TikTok kann bei einigen Überprüfungen (Captcha, E-Mail-Bestätigung) manuelles Eingreifen erfordern.
- **Profil-Autofill**: Automatisches Festlegen von Avatar, Nickname, Beschreibung und Benutzername.
- **Automatische Kontoerkennung**: Erkennung und Import von Konten mit einem Klick, die bereits auf dem Telefon angemeldet sind.
- **Account-Pflege**: Suche nach Schlüsselwörtern, Videos ansehen, Likes, Abonnements, Kommentare und Favoriten hinzufügen zur Account-Pflege.
- **Auto-Posting**: Veröffentlichung von Videos oder Bildern (Unterstützung für Sortierung und Batch-Upload) mit anpassbarer Musik, Titel, Beschreibung, Hashtags und Produktlinks.
- **Auto-Löschen**: Löschen von erfolglosen/schwach performenden Posts nach Aufrufregeln.
- **Super-Marketing**: Simulation menschlicher Navigation (Profile ansehen, Posts, zufällige Abonnements/Likes/Kommentare/Favoriten/Teilen/Direktnachrichten) und Option zur Verwendung von OpenAI oder kompatibler API zur Generierung personalisierter Nachrichten/Kommentare.
- **Batch-Promotion (Live)**: Massen-Likes und Kommentare in Live-Streams.
- **Benutzersammlung**: Sammlung von Benutzern nach Schlüsselwörtern, Follower- oder Abonnentenlisten.
- **Massen-Follow-Back und Begrüßungsnachrichten**: Massen-Abonnement von Followern und Versand von Begrüßungsnachrichten.
- **Minutengenaue Aufgabenplanung**: Unterstützung für Wiederholungen, Ausführungsintervalle und maximale Anzahl von Wiederholungen.
- **Multi-User-Parallelität**: Ein PC kann mehrere Android-Geräte verbinden und verwalten und Aufgaben parallel ausführen.
- **Bildschirmspiegelung**: Spiegelung mehrerer Geräte auf Windows / macOS mit Maus- und Tastatursteuerung.
- **Konto-Import / -Export**: Batch-Import/-Export im Excel-Format.
- **Mehrsprachige Unterstützung**: Die Software unterstützt Englisch, Russisch und Chinesisch. Hinweis: Für optimale Kompatibilität erfordern einige Funktionen, dass die TikTok/Instagram-App auf Ihrem Telefon auf Englisch eingestellt ist.
- **Technischer Support**: Wir bieten Support über Tickets und persönliche Unterstützung.

## Was Sie benötigen

### Android-Telefone

Sie können beliebige Android-Geräte für die Arbeit mit TikMatrix verwenden:

- **Echte Android-Geräte**
- **[Android-Mainboard-Geräte](http://www.niaozun.shop?cid=934ec2fe)**
- **[Cloud-Telefone](https://www.geelark.cn?invite_code=XHY6a8)**
- **Emulatoren**
- **Virtuelle Maschinen**

Geräte müssen mindestens API 21 (Android 5.0) haben und benötigen kein Root. Aktivieren Sie USB-Debugging (Entwickleroptionen). Bei einigen Modellen (z. B. Xiaomi) muss möglicherweise USB-Debugging (Sicherheitseinstellungen) aktiviert werden.

### Computer

#### Windows: Windows 10 oder höher

#### Mac: macOS 15.0 oder höher (Apple Silicon und Intel)

## Installation

1. Laden Sie den neuesten Installer von der Seite [TikMatrix herunterladen](https://tikmatrix.com/Download) oder [IgMatrix herunterladen](https://igmatrix.com/Download-IgMatrix) herunter.
2. Führen Sie den Installer aus.
3. Warten Sie, bis die Installation abgeschlossen ist.
4. Öffnen Sie die **TikMatrix**- oder **IgMatrix**-Anwendung auf Ihrem Computer.
