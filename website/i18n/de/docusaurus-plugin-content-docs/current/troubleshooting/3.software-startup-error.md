# Software-Startfehler

## Fehlerbehebungsschritte bei allgemeinen Software-Startfehlern

### Schritt 1: Systemanforderungen prüfen

1. Stellen Sie sicher, dass Ihr System die Mindestanforderungen erfüllt:
   - Windows 10 oder höher
   - Mindestens 4GB RAM
   - Mindestens 1GB freier Speicherplatz

### Schritt 2: Microsoft Visual C++ Redistributable prüfen

Normalerweise wird der Fehler `vcruntime140.dll nicht gefunden` oder `Anwendung kann nicht normal gestartet werden (0xc000007b)` durch fehlendes Microsoft Visual C++ Redistributable verursacht.

1. Drücken Sie `Windows + R`, geben Sie `cmd` ein und drücken Sie Enter
2. Geben Sie `dir "C:\Windows\System32\vcruntime*.dll"` ein und drücken Sie Enter
3. Wenn es eine Ausgabe gibt, bedeutet dies, dass die entsprechende Visual C++ Redistributable-Version installiert ist. Dateinamen mit vcruntime-Suffix enthalten normalerweise eine Versionsnummer, z.B. vcruntime140.dll (entspricht 2015-2022).
4. Wenn keine Ausgabe vorhanden ist, laden Sie Microsoft Visual C++ Redistributable herunter und installieren Sie es: [Neueste unterstützte Microsoft Visual C++ Redistributable](https://learn.microsoft.com/en-us/cpp/windows/latest-supported-vc-redist?view=msvc-170)

### Schritt 3: Systemzeit prüfen

Ungenaue Systemzeit kann den Fehler "Agent Start Timeout" verursachen:

1. Klicken Sie mit der rechten Maustaste auf die Zeitanzeige in der unteren rechten Ecke des Bildschirms
2. Wählen Sie "Datum und Uhrzeit anpassen"
3. Aktivieren Sie "Uhrzeit automatisch festlegen", falls noch nicht geschehen
4. Wenn die Zeit immer noch falsch ist, klicken Sie auf "Jetzt synchronisieren" oder stellen Sie Datum und Uhrzeit manuell ein
5. Starten Sie TikMatrix nach Korrektur der Zeit neu

### Schritt 4: Anwendungsprotokolle prüfen

1. Klicken Sie in TikMatrix auf `Einstellungen` > `App-Ordner öffnen`
2. Navigieren Sie zum `logs`-Ordner
3. Prüfen Sie die neueste Protokolldatei auf Fehlermeldungen

### Wenn das Problem weiterhin besteht

### Schritt 5: macOS — Agent InvalidStateError

Auf einigen macOS-Geräten startet der Agent möglicherweise nicht und zeigt einen Fehler wie:

```text
Agent start error InvalidStateError: The object is in an invalid state.
```

Mögliche Ursache:

- Normalerweise verursacht durch Inkompatibilität zwischen App und macOS-Version oder alter Hardware/Firmware.

Lösungen / Workarounds:

1. Aktualisieren Sie macOS auf die neueste Version über Systemeinstellungen > Allgemein > Software-Update. Nach dem Update starten Sie den Mac neu und versuchen Sie TikMatrix erneut zu starten.
2. Wenn ein Update nicht möglich ist oder der Fehler weiterhin besteht, versuchen Sie, die App auf einem anderen Mac mit neuerer macOS-Version auszuführen.

Wenn das Problem nach diesen Schritten weiterhin besteht, sammeln Sie Anwendungsprotokolle und kontaktieren Sie unseren Support auf [Telegram](https://t.me/tikmatrix_agent_bot), fügen Sie die macOS-Version und Log-Dateien bei.

Kontaktieren Sie unser technisches Support-Team über [Telegram](https://t.me/tikmatrix_agent_bot) und geben Sie folgende Informationen an:

- Screenshot des Fehlers
- Betriebssystemversion
- Schritte, die Sie bereits versucht haben
