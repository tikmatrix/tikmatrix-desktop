# Fehlerbehebungsschritte bei Aufgabenfehlern

## Schritt 1: Agent-Status prüfen (wenn Aufgabe nicht gestartet wurde)

1. Wählen Sie in der Seitenleiste: General → Open Agent.
2. Stellen Sie sicher, dass die TikMatrix-App auf dem Telefon geöffnet ist.
3. Beobachten Sie, ob innerhalb von 10 Sekunden ein Erfolgsstatus erscheint.
4. Wenn der Agent einen Fehlerstatus anzeigt, führen Sie die manuelle Initialisierung durch, wie im Leitfaden Initialize Device beschrieben: ../tutorial-basics/2.init-device.md

## Schritt 2: TikTok/Instagram-App-Konfiguration prüfen (wenn Skript App falsch öffnet/schließt)

1. Wenn Sie die offizielle TikTok/Instagram-App verwenden, klicken Sie auf "Open TikTok" / "Open Instagram" und prüfen Sie, ob die App auf dem Gerät öffnet. Wenn nicht, klicken Sie auf die Einstellungsschaltfläche im Header und wählen Sie die korrekte TikTok-Version (International / Asia). Instagram hat nur die offizielle Version.
2. Wenn Sie eine geklonte Version von TikTok/Instagram verwenden, stellen Sie sicher, dass Sie den Paketnamen der geklonten App in der **Account List** konfiguriert haben. Siehe Anleitung zur Verwendung geklonter Apps: ../tutorial-basics/10.clone-apps-support.md. Stellen Sie außerdem sicher, dass beim Starten von Aufgaben der **Multi-account Mode** aktiviert ist.

## Schritt 3: TikTok/Instagram-Spracheinstellungen prüfen

Derzeit wird nur Englisch für TikTok und Instagram unterstützt. Wenn Ihre Apps nicht auf Englisch eingestellt sind, setzen Sie die App-Sprache auf Englisch wie folgt:

1. Öffnen Sie die TikTok-App auf dem Gerät.
2. Gehen Sie zu Ihrem Profil.
3. Tippen Sie auf die drei Punkte in der oberen rechten Ecke, um die Einstellungen zu öffnen.
4. Scrollen Sie zum Abschnitt "Language".
5. Wählen Sie "English" als App-Sprache.
6. Wiederholen Sie dieselben Schritte für Instagram.

## Schritt 4: Test-Skript ausführen

1. Wählen Sie in der Seitenleiste: Scripts → Test Script.
2. Warten Sie, bis das Skript abgeschlossen ist.
3. Wenn der Test fehlschlägt, überprüfen Sie die Logs und versuchen Sie, das Problem zu identifizieren. Siehe Abschnitt "Known Issues" unten für häufige Probleme.

## Schritt 5: Support-Ticket einreichen

Wenn die vorherigen vier Schritte das Problem nicht identifiziert haben, erstellen Sie ein Support-Ticket. Wir sammeln automatisch Logs und helfen bei der Fehlerbehebung. Schritte:

1. Klicken Sie auf die Schaltfläche "Support" im Header.
2. Klicken Sie auf "Create Ticket".
3. Geben Sie Kontaktinformationen und Problembeschreibung ein. Für schnellere Lösung fügen Sie eine Bildschirmaufnahme der Skriptausführung bei.
4. Wählen Sie betroffene Geräte aus (mehrere auswählbar). Dies ist wichtig – wir können nur Logs für ausgewählte Geräte erhalten.
5. Klicken Sie auf "Submit".
6. Wir könnten um zusätzliche Informationen bitten, z.B. UI-XML für den letzten Bildschirm vor dem Fehler: Navigieren Sie manuell auf dem Telefon zum letzten Bildschirm vor dem Aufgabenfehler, klicken Sie die rechte "Debug"-Taste auf dem Gerätebildschirm, dann klicken Sie "Download XML". Senden Sie die heruntergeladene XML und Ticketnummer an unsere Support-E-Mail: [support@tikmatrix.com](mailto:support@tikmatrix.com)
