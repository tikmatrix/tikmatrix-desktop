# Telefon kann nicht erkannt werden

## Fehlerbehebungsschritte bei Nichterkennung des Telefons

### Erster Schritt: ADB-Verbindungsstatus prüfen

1. Klicken Sie in TikMatrix auf `Einstellungen` > `ADB-Terminal öffnen`.  
2. Führen Sie den Befehl `adb devices` aus und prüfen Sie, ob Ihr Telefon in der Geräteliste erscheint

### Für Windows-Systeme

```shell
.\adb.exe devices
```

### Für Mac/Linux-Systeme

```shell
./adb devices
```

### Beispielausgabe

```shell
List of devices attached
<DEVICE_ID>    <STATUS>
```

### Wenn Gerät nicht in adb-Geräteliste erscheint

Folgen Sie den Schritten in der Dokumentation [Gerät verbinden](../tutorial-basics/1.connect-to-devices.md) zur Fehlerbehebung

### Wenn Gerätestatus `unauthorized` ist

Mögliche Ursache - Gerät ist nicht autorisiert, tippen Sie auf dem Telefon auf `USB-Debugging erlauben`

### Wenn Gerätestatus `offline` ist

Mögliche Ursache - lockeres USB-Kabel, verbinden Sie das Telefon erneut, trennen Sie das USB-Kabel und verbinden Sie es wieder

### Wenn Gerätestatus `device` ist

Gerät erfolgreich verbunden. Wenn TikMatrix das Telefon immer noch nicht erkennt, stellen Sie sicher, dass TikMatrix aktiviert ist. Auch die kostenlose Testversion erfordert Aktivierung auf der Lizenzseite.

### Wenn das Problem weiterhin besteht

Kontaktieren Sie unser technisches Support-Team über [Telegram](https://t.me/tikmatrix_agent_bot) und geben Sie folgende Informationen an:

- Screenshot der adb devices-Befehlsausgabe
- Schritte, die Sie bereits versucht haben
