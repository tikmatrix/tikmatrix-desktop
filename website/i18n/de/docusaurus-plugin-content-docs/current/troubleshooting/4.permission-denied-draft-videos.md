# Berechtigung verweigert - Entwurfsvideos können nicht gespeichert werden

## Lösung des Berechtigungsverweigerungsproblems auf dem mobilen Gerätebildschirm

### Problembeschreibung

Beim Ausführen von Post-Skripten kann folgende Meldung auf dem mobilen Gerätebildschirm erscheinen:

```text
Permission denied. Unable to save draft videos.
```

### Lösung

Dieses Problem tritt aufgrund fehlender Speicherberechtigungen für die TikTok-App auf. Führen Sie folgende Schritte zur Lösung aus:

#### Schritt 1: Veröffentlichen-Schaltfläche manuell drücken

1. Finden und tippen Sie auf die **Veröffentlichen**-Schaltfläche auf dem mobilen Gerätebildschirm
2. Fahren Sie mit dem Veröffentlichungsprozess fort

#### Schritt 2: Systemberechtigungen erteilen

1. Wenn der System-Berechtigungsanfrage-Dialog erscheint, tippen Sie auf **Erlauben**
2. Stellen Sie sicher, dass Sie der TikTok-App die erforderlichen Speicherberechtigungen erteilt haben

### Wichtiger Hinweis

:::info
Dieser Berechtigungseinrichtungsprozess muss nur einmal durchgeführt werden. Nach Erteilung der Berechtigungen werden nachfolgende Post-Skript-Ausführungen nicht mehr auf dieses Problem stoßen.
:::

### Vorbeugende Maßnahmen

Um dieses Problem zu vermeiden, wird empfohlen, vor der ersten Verwendung von TikMatrix folgende Schritte durchzuführen:

1. Öffnen Sie die TikTok-App manuell
2. Versuchen Sie, ein Entwurfsvideo aufzunehmen oder zu speichern
3. Wählen Sie **Erlauben** bei Berechtigungsanfrage
4. Dies richtet die erforderlichen Berechtigungen im Voraus ein
