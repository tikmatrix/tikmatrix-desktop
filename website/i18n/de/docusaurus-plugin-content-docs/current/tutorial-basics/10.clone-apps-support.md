# Unterstützung für geklonte Apps

TikMatrix und IgMatrix unterstützen das Ausführen mehrerer geklonter Apps (wie GB TikTok, GB Instagram, Instagram Plus oder andere modifizierte Versionen) auf einem Gerät, jede mit ihrem eigenen Paketnamen. Diese Funktion ermöglicht eine effizientere Verwaltung mehrerer Konten durch die Verwendung verschiedener App-Instanzen.

:::danger Risikowarnung
Die Verwendung geklonter Apps kann gegen die Nutzungsbedingungen von TikTok/Instagram verstoßen und kann zur Sperrung oder Einschränkung von Konten führen. Wir bieten nur technische Unterstützung für diese Funktion und übernehmen **KEINE Verantwortung** für kontobezogene Probleme, einschließlich Sperrungen, Einschränkungen oder andere Sanktionen, die durch die Verwendung geklonter Apps verursacht werden. Nutzung auf eigenes Risiko.
:::

## Was ist ein Paketname?

Jede Android-App hat eine eindeutige Kennung, den **Paketnamen**. Offizielle Apps verwenden Standard-Paketnamen:

- TikTok: `com.zhiliaoapp.musically`
- Instagram: `com.instagram.android`

Geklonte Apps verwenden unterschiedliche Paketnamen, zum Beispiel:

- GB TikTok: `com.zhiliaoapp.musically.gb`
- GB Instagram: `com.instagram.gb.android`
- Instagram Plus: `com.instagram.plus.android`

TikMatrix/IgMatrix kann automatisch den richtigen Paketnamen für jedes Konto erkennen und verwenden.

## Konfiguration geklonter Apps

### Schritt 1: Geklonte Apps installieren

1. Installieren Sie die bevorzugte geklonte App auf dem Gerät oder CloudPhone.
2. Stellen Sie sicher, dass jede geklonte App einen eindeutigen Paketnamen hat, um Konflikte zu vermeiden.
3. Melden Sie sich bei verschiedenen Konten in jeder App-Instanz an.

### Schritt 2: Paketnamen konfigurieren

Sie haben zwei Optionen zur Konfiguration von Paketnamen:

#### Option A: Konfiguration auf Kontoebene (empfohlen)

Legen Sie einen spezifischen Paketnamen für jedes Konto fest:

1. Öffnen Sie **Konten** im linken Navigationsmenü.
2. Klicken Sie auf ein Konto zum Bearbeiten oder auf **Hinzufügen**, um ein neues zu erstellen.
3. Im Feld **Paketname** können Sie:
   - **Manuell eingeben**: Den Paketnamen der geklonten App (z. B. `com.instagram.gb.android`)
   - **Automatisch erkennen**: Durch Klicken auf die Schaltfläche **Paket erkennen** (erfordert zuerst die Auswahl eines Geräts)
4. Klicken Sie auf **Speichern**.

:::tip Automatische Paketnamen-Erkennung
Die Schaltfläche **Paket erkennen** erkennt automatisch die aktuell geöffnete App auf dem ausgewählten Gerät. Dies ist der einfachste Weg, um den richtigen Paketnamen zu erhalten:

1. Wählen Sie ein Gerät für das Konto aus
2. Öffnen Sie die geklonte App auf diesem Gerät
3. Klicken Sie auf die Schaltfläche **Paket erkennen**
4. Der Paketname wird automatisch ausgefüllt

Diese Funktion funktioniert mit jeder Android-App, einschließlich TikTok/Instagram-Klonen, App Cloner-Apps und NomixCloner.
:::

Beim Ausführen von Skripten für dieses Konto verwendet TikMatrix/IgMatrix automatisch den angegebenen Paketnamen.

#### Option B: Systemweiter Standardwert

Legen Sie einen Standard-Paketnamen für alle Konten fest, die keinen spezifischen haben:

1. Öffnen Sie **Einstellungen** im linken Navigationsmenü.
2. Suchen Sie das Feld **TikTok-Paketname** oder **Instagram-Paketname**.
3. Geben Sie den Standard-Paketnamen ein.
4. Klicken Sie auf **Speichern**.

:::tip Priorität
Einstellungen auf Kontoebene überschreiben immer den systemweiten Standardwert. Wenn ein Konto einen eigenen Paketnamen konfiguriert hat, wird dieser anstelle des Standardwerts verwendet.
:::

## So funktioniert es

Beim Ausführen eines Automatisierungsskripts:

1. TikMatrix/IgMatrix überprüft, ob das Zielkonto einen benutzerdefinierten Paketnamen konfiguriert hat.
2. Wenn ja, verwendet es diesen Paketnamen für alle UI-Automatisierungsoperationen.
3. Wenn nein, wird auf den systemweiten Standard-Paketnamen zurückgegriffen.
4. Alle Ressourcen-IDs und XPath-Abfragen werden automatisch mit dem richtigen Paketnamen formatiert.

## Best Practices

### 1. Verwenden Sie beschreibende Benutzernamen

Da Sie mehrere Konten in verschiedenen geklonten Apps haben können, verwenden Sie beschreibende Benutzernamen, um zu identifizieren, zu welcher App jedes Konto gehört:

```text
username_gb      → GB TikTok/Instagram
username_plus    → TikTok/Instagram Plus
username         → Offizielle App
```

### 2. Organisieren Sie Konten nach Gruppen

Erstellen Sie Gerätegruppen basierend auf dem Typ der geklonten App:

- **Gruppe A**: Alle Konten, die GB-Apps verwenden
- **Gruppe B**: Alle Konten, die Plus-Apps verwenden
- **Gruppe C**: Alle Konten, die offizielle Apps verwenden

### 3. Testen Sie vor Massenoperationen

Bevor Sie Skripte auf allen Konten ausführen:

1. Testen Sie zuerst mit einem Konto
2. Stellen Sie sicher, dass die richtige App verwendet wird
3. Bestätigen Sie, dass alle Operationen wie erwartet funktionieren

### 4. Konsistenz der Paketnamen beibehalten

Nachdem Sie einen Paketnamen für ein Konto festgelegt haben, ändern Sie ihn nicht, es sei denn, Sie wechseln zu einer anderen geklonten App. Konsistenz hilft, Verwirrung und Fehler zu vermeiden.

## Fehlerbehebung

### Skript öffnet falsche App

**Problem**: Skript öffnet die falsche App anstelle der geklonten.

**Lösung**:

1. Überprüfen Sie, ob das Feld **Paketname** des Kontos korrekt eingestellt ist
2. Stellen Sie sicher, dass die geklonte App installiert ist und der Paketname übereinstimmt
3. Verwenden Sie die Schaltfläche **Paket erkennen**, um automatisch den richtigen Paketnamen zu erhalten
4. Starten Sie das Gerät neu, wenn die App kürzlich installiert wurde

### UI-Elemente können nicht gefunden werden

**Problem**: Skript meldet Fehler "Element nicht gefunden".

**Lösung**:

1. Verschiedene geklonte Apps können leicht unterschiedliche UI-Layouts haben
2. Überprüfen Sie, ob die Version der geklonten App kompatibel ist
3. Versuchen Sie, auf die neueste Version von TikMatrix/IgMatrix zu aktualisieren
4. Wenden Sie sich an den Support, wenn das Problem weiterhin besteht

### Mehrere Konten in einer App

**Problem**: Notwendigkeit, zwischen mehreren Konten in einer App-Instanz zu wechseln.

**Lösung**:

- TikTok/Instagram unterstützen nativ mehrere angemeldete Konten
- Verwenden Sie die Funktion **Kontowechsel** in TikMatrix/IgMatrix
- Legen Sie denselben Paketnamen für alle Konten in dieser App-Instanz fest
- TikMatrix/IgMatrix behandelt den Kontowechsel automatisch

### Paket-Erkennen-Schaltfläche funktioniert nicht

**Problem**: Die Schaltfläche **Paket erkennen** funktioniert nicht oder zeigt einen Fehler.

**Lösung**:

1. Stellen Sie sicher, dass zuerst ein Gerät ausgewählt ist
2. Stellen Sie sicher, dass die App derzeit auf dem Gerät geöffnet ist
3. Überprüfen Sie die Stabilität der Geräteverbindung
4. Versuchen Sie, die App erneut zu öffnen und zu erkennen

## Paketnamen finden

Wenn Sie Paketnamen auf dem Gerät manuell überprüfen möchten:

### ADB-Befehl verwenden

```bash
# Liste aller Pakete im Zusammenhang mit TikTok
adb shell pm list packages | grep musically

# Liste aller Pakete im Zusammenhang mit Instagram
adb shell pm list packages | grep instagram
```

### Automatische Erkennungsfunktion verwenden

Der einfachste Weg ist die Verwendung der integrierten Schaltfläche **Paket erkennen**:

1. Öffnen Sie den Kontobearbeitungsdialog
2. Wählen Sie ein Gerät aus
3. Öffnen Sie die geklonte App auf dem Gerät
4. Klicken Sie auf **Paket erkennen**
5. Der Paketname wird automatisch erkannt und ausgefüllt

Diese Methode funktioniert für jede Android-App und ist der empfohlene Ansatz.

## Häufig gestellte Fragen

**F: Kann ich mehrere geklonte Apps auf einem Gerät verwenden?**

A: Ja, solange jede geklonte App einen eindeutigen Paketnamen hat, können Sie mehrere App-Instanzen auf einem Gerät ausführen.

**F: Muss ich TikMatrix/IgMatrix neu installieren?**

A: Nein, diese Funktion ist in der neuesten Version integriert. Aktualisieren Sie einfach auf die neueste Version und konfigurieren Sie die Paketnamen.

**F: Funktionieren meine bestehenden Automatisierungsskripte weiterhin?**

A: Ja, alle vorhandenen Skripte sind vollständig kompatibel. Konten ohne benutzerdefinierten Paketnamen verwenden den Standard-Paketnamen.

**F: Werden sowohl TikTok als auch Instagram unterstützt?**

A: Ja, sowohl TikMatrix als auch IgMatrix unterstützen vollständig die Erkennung geklonter Apps und Automatisierung.

**F: Wie funktioniert die automatische Erkennungsfunktion?**

A: Beim Klicken auf **Paket erkennen** überprüft TikMatrix/IgMatrix, welche App derzeit im Vordergrund auf dem ausgewählten Gerät ist, und ruft automatisch ihren Paketnamen ab.

**F: Kann ich Apps von App Cloner oder NomixCloner verwenden?**

A: Ja, die Paketnamenfunktion funktioniert mit jedem Android-App-Klonierungstool, einschließlich App Cloner und NomixCloner. Verwenden Sie die automatische Erkennungsfunktion, um den richtigen Paketnamen einfach zu erhalten.
