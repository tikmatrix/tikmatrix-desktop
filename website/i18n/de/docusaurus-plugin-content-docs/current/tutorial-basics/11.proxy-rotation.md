# Einfache Anleitung zur Proxy-Rotation

Die Proxy-Rotation hilft TikMatrix, die IP-Adresse des Telefons automatisch zu wechseln. Dies ist praktisch, wenn Sie häufig Konten wechseln oder sensible Aufgaben ausführen und TikTok eine "frische" IP anzeigen möchten.

## Was Sie benötigen

- Einen Proxy-Service mit "IP wechseln" oder "IP rotieren" Funktion.
- Eine Rotations-URL (auch genannt *rotation URL*, *refresh URL* oder *change IP URL*).
- Bei Bedarf – Passwort, Token oder andere Daten für den Zugriff auf die URL.

## Speichern der URL für das Gerät

1. Öffnen Sie auf dem Computer **Geräte verwalten** und wechseln Sie in den Tabellenmodus.
2. Finden Sie das gewünschte Gerät und klicken Sie auf **Konfigurieren** in der Spalte *Proxy-Rotations-URL*.
3. Füllen Sie die Felder aus:
   - **Proxy-Rotations-URL** – Fügen Sie die URL vom Anbieter ein. TikMatrix öffnet sie jedes Mal, wenn die IP aktualisiert werden muss.
   - **Anfragemethode** – Lassen Sie `GET`, wenn der Anbieter nicht `POST` verlangt.
   - **Headers** und **Body** – Meist leer lassen. Wenn der Anbieter ein kurzes Beispiel gezeigt hat, kopieren Sie es unverändert.
   - **Timeout** – Wartezeit für Antwort. Standard 10 Sekunden passen zu den meisten Services.
4. Klicken Sie auf **Speichern**. Das Programm speichert die Einstellungen automatisch, nichts muss manuell bearbeitet werden.

In der Tabelle erscheinen Informationen zur letzten Rotation. Grün bedeutet Erfolg, Grau – Rotation wurde noch nicht gestartet, Rot – Service hat einen Fehler zurückgegeben.

## Überprüfen, ob alles funktioniert

1. Klicken Sie in derselben Zeile auf **Proxy-Rotation testen**.
2. TikMatrix greift sofort auf die URL zu.  
   - Bei Erfolg sehen Sie eine Meldung über erfolgreiche Rotation und die Zeit "Zuletzt rotiert" wird aktualisiert.  
   - Bei Fehler überprüfen Sie die URL und Autorisierungsdaten und versuchen Sie es erneut.

Tipp: Viele Anbieter beschränken die Häufigkeit des IP-Wechsels. Wenn der Test fehlschlägt, warten Sie ein paar Sekunden und versuchen Sie es erneut.

## Verwendung der Rotation in Szenarien

1. Beim Starten eines Skripts aktivieren Sie zuerst **Proxy wechseln**, und TikMatrix wechselt die IP für jedes Gerät nacheinander, bevor TikTok geöffnet wird.  
2. Sie können auf die Log-Schaltfläche rechts neben dem Gerät klicken, um detaillierte Informationen zu jedem IP-Wechsel anzuzeigen.

## Häufige Fragen

- **Muss ich die Rotation für alle Szenarien aktivieren?** Nein. Tun Sie dies nur dort, wo wirklich eine neue IP benötigt wird – zum Beispiel beim Massenwechsel von Konten.
- **Wo werden die Einstellungen gespeichert?** TikMatrix macht alles automatisch. Es reicht, jedes Gerät einmal zu konfigurieren.
- **Die URL sieht verdächtig lang aus. Ist das normal?** Ja. Wenn die URL in einem normalen Browser funktioniert, kann TikMatrix sie auch verwenden.
- **An wen wenden, wenn nichts hilft?** Geben Sie die Anweisungen des Anbieters und den Fehlertext an den TikMatrix-Support weiter – sie helfen bei der Klärung.
