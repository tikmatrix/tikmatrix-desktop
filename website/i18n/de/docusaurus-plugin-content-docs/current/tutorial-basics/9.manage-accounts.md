# Kontenverwaltung

Das Kontenverwaltungsmodul synchronisiert jeden TikTok- oder Instagram-Eintrag mit dem Gerät, auf dem er läuft, mit seinen Anmeldedaten und den für die Automatisierung benötigten Daten. Eine geordnete Kontoliste beseitigt manuelle Aktionen auf jedem Telefon und erleichtert die Gerätezuweisung, Statusüberwachung und Kampagnenvorbereitung.

## Manuelles Erstellen eines Kontos

1. Öffnen Sie den Punkt **Accounts** im linken Bereich.
2. Klicken Sie auf **Add**, um das Bearbeitungsfenster zu öffnen.
3. Füllen Sie die Basisfelder aus:
   - **Email**: Die in der App verwendete Adresse. TikMatrix/IgMatrix versucht zunächst, sich mit dem Paar Email + Password zu authentifizieren; wenn das Email-Feld leer ist, wird automatisch das Schema Username + Password angewendet.
   - **Password**: Login-Passwort. IgMatrix unterstützt ein Passwort, das mit einem 2FA-Secret ergänzt wurde (siehe unten).
   - **Username** — angezeigter TikTok/Instagram-Spitzname. Dies ist das einzige Pflichtfeld.
   - Schalter **Login Status**: `1` — Sitzung ist bereits auf dem Gerät autorisiert, `0` — Login nicht durchgeführt.
   - Schalter **Status**: `0` — Konto aktiv, `1` — deaktiviert.
4. Wählen Sie ein Gerätebadge, um das Konto an ein Telefon oder CloudPhone zu binden. Wenn kein Gerät ausgewählt ist, bleibt das Konto nicht zugeordnet.
5. Klicken Sie auf **Save**. Die Liste wird automatisch aktualisiert.

## Autorisierung

- Wenn die Felder **Email** und **Password** ausgefüllt sind, führt TikMatrix/IgMatrix den Login auf dem angegebenen Gerät genau mit diesem Paar durch.
- Wenn **Email** leer ist, wird die Kombination **Username** + **Password** verwendet.
- Wählen Sie unbedingt einen Geräteindex aus, damit das Skript versteht, wo der Login gestartet werden soll.
- Das Autorisierungsskript überprüft den aktuellen Status und überspringt Konten, die bereits im System angemeldet sind.

## 2FA-Unterstützung (nur IgMatrix)

IgMatrix kann die Zwei-Faktor-Authentifizierung während des Logins automatisch durchlaufen.

### Passwortformat mit 2FA

Um 2FA zu aktivieren, legen Sie das Passwort im folgenden Format fest:

```text
ihr_passwort:ihr_2FA_secret
```

**Beispiel:**

```text
mypassword123:JBSWY3DPEHPK3PXP
```

## Import und Export von Konten

- **Export** speichert alle Konten in der Datei `accounts.xlsx` im TikMatrix/IgMatrix-Download-Ordner. Verwenden Sie dies als Backup und Vorlage für Massenbearbeitung.
- **Import** ermöglicht das massenhafte Erstellen oder Aktualisieren von Einträgen aus Excel. Wenn Sie sich über die Dateistruktur nicht sicher sind, führen Sie zuerst **Export** aus, bearbeiten Sie die exportierte Datei und importieren Sie sie zurück.
- Beim Import aktualisieren Zeilen mit ausgefülltem `id`-Feld vorhandene Einträge. Zeilen ohne `id` erstellen neue Konten.

> Für 2FA in IgMatrix füllen Sie `username`, `pwd` (im Format `passwort:2fa_secret`) und die Spalte `device_index` aus.

### Excel-Spaltenreferenz

| Spalte | Erforderlich | Beschreibung |
| --- | --- | --- |
| `id` | Nein | Interne Kennung. Lassen Sie diese nur, wenn Sie einen vorhandenen Eintrag aktualisieren müssen. |
| `email` | Nein | E-Mail für den App-Login; nützlich für Kommunikation und Wiederherstellung. |
| `pwd` | Nein | Passwort für schnelle Referenz oder automatische Autorisierung. |
| `username` | Ja | TikTok/Instagram-Spitzname. Muss im Arbeitsbereich eindeutig sein. |
| `device` | Nein | Seriennummer des Geräts oder CloudPhone-Kennung, an die das Konto gebunden ist. |
| `device_index` | Nein | Benutzerfreundliches Geräteetikett, das in der Oberfläche angezeigt wird; dient der Bequemlichkeit. |
| `logined` | Nein | Sitzungsflag (`1` — autorisiert, `0` — nicht autorisiert). |
| `status` | Nein | Verfügbarkeitsflag (`0` — aktiviert, `1` — deaktiviert). Deaktivierte Konten nehmen nicht an Skripten teil. |
| `nickname` | Nein | Letzter Spitzname, der nach Synchronisation der TikTok-Daten erhalten wurde. |
| `country` | Nein | Land/Region, bestimmt durch TikTok-Profil. |
| `followers` | Nein | Anzahl der Follower aus TikTok-Synchronisation. |
| `following` | Nein | Wie viele Profile das Konto verfolgt. |
| `hearts` | Nein | Gesamtzahl der Likes (Hearts). |
| `videos` | Nein | Anzahl der veröffentlichten Videos. |
| `friends` | Nein | Anzahl gegenseitiger Freunde, falls TikTok diese Daten bereitstellt. |

### TikMatrix- und IgMatrix-Felder

- **TikMatrix** füllt statistische Felder (`nickname`, `country`, `followers`, `following`, `hearts`, `videos`, `friends`) nach Ausführung der Funktion **Sync TikTok Data** oder Klicken auf die Datenanforderungsschaltfläche.
- **IgMatrix** verwendet nur Basisspalten (`email`, `pwd`, `username`, `device`, `logined`, `status`) und ignoriert analytische Werte.

## Zusätzliche Besonderheiten

- Konten mit Status **Disabled** werden von allen automatisierten Skripten, Zeitplänen und Batch-Aktionen ausgeschlossen. Versetzen Sie sie vor dem Starten von Aufgaben in den Enabled-Status.
- Das Skript **Match Accounts** scannt Geräte, findet bereits in der App autorisierte Profile, erstellt automatisch Einträge in der Kontoliste und verknüpft sie mit dem erkannten Gerät. Dies ist der schnellste Weg, um manuell vorbereitete Telefone zu verbinden.
