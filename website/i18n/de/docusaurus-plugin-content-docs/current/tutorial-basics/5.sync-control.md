# Synchrone Steuerung

Die synchrone Steuerung wird verwendet, um synchrone Operationen auf mehreren Geräten zu steuern.

## Schritte

1. Klicken Sie, um den Bildschirm 1 des Telefons zu öffnen.
2. Wählen Sie links mehrere Geräte aus.
3. Alle Operationen auf dem Bildschirm werden auf mehrere Geräte synchronisiert.
