# Schnelleingabetastatur aktivieren

Nach dem Verbinden des Telefons wird automatisch auf die Schnelleingabetastatur umgeschaltet.

## Warum die Schnelleingabetastatur konfigurieren

Die Schnelleingabetastatur ist ein notwendiges Werkzeug von TikMatrix, das automatische mehrsprachige Unterstützung bietet. Ohne sie können die Texteingabefunktionen von TikMatrix (wie Kommentare, Nachrichten und Post-Titel) möglicherweise nicht korrekt funktionieren.

## So aktivieren Sie manuell die Schnelleingabetastatur

1. Wählen Sie die Geräte aus, die die Schnelleingabetastatur benötigen.
2. Klicken Sie auf `Benutzerdefinierte ADB-Befehle` - `Schnelleingabe aktivieren`.

## So geben Sie Text ein

In TikMatrix können Sie Text auf verschiedene Weise eingeben:

### Methode 1: Direkte Tastatureingabe ⌨️ (Neue Funktion)

⭐ **Tastatureingabesteuerung** - Jetzt können Sie Text direkt auf Telefonen mit der Computertastatur eingeben, was die Texteingabe schneller und bequemer macht.

1. Wählen Sie das Gerät aus, das Sie verwenden möchten.
2. Klicken Sie auf ein beliebiges Textfeld auf dem Telefonbildschirm.
3. Beginnen Sie direkt auf der Computertastatur zu tippen - der Text wird in Echtzeit auf dem Telefon erscheinen.

### Methode 2: Eingabefeld

1. Klicken Sie auf den Telefonbildschirm, erweitern Sie den großen Bildschirm und klicken Sie auf die Schaltfläche `Eingabe` in der Werkzeugleiste rechts.
2. Geben Sie Text in das Eingabefeld ein und klicken Sie auf Senden.

## So kopieren und fügen Sie Text zwischen Telefon und Computer ein

### Text aus der Telefonzwischenablage in die Computerzwischenablage kopieren

1. Wählen Sie das Telefon zum Kopieren aus, halten Sie die Tasten `Strg+C`, Kopieren erfolgreich abgeschlossen

### Text aus der Computerzwischenablage in das Telefoneingabefeld einfügen

1. Wählen Sie das Gerät zum Einfügen aus, halten Sie die Tasten `Strg+V`, Einfügen erfolgreich abgeschlossen
