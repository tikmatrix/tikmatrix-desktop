# Gerät initialisieren

:::warning
Diese Anwendung wird über den offiziellen Android ADB-Debug-Kanal installiert, um die Geräteautomatisierung zu verwalten. Auf einigen Geräten kann eine Abfrage erscheinen, ob eine Probe einer unbekannten Anwendung an Google gesendet werden soll – dies ist eine Android-Systemwarnung, die nicht mit der Anwendungssicherheit zusammenhängt und deren normale Funktion nicht beeinträchtigt. Klicken Sie auf "Don't send" (Nicht senden), um die Installation abzuschließen.
:::

TikMatrix APK ist ein mobiler Agent, der für die Ausführung automatisierter Aufgaben erforderlich ist. **Beim ersten Verbinden des Telefons mit TikMatrix initialisiert das System automatisch Ihr Gerät und installiert die TikMatrix-Anwendung** – keine manuellen Aktionen erforderlich.

Während der automatischen Initialisierung installiert TikMatrix zwei APK-Dateien: TikMatrix und TikMatrix Test (ohne Oberfläche). Wenn Ihr Telefon um Erlaubnis zur Installation bittet, klicken Sie einfach auf "Zulassen" (besonders wichtig auf Xiaomi-Geräten).

## Automatische vs. Manuelle Installation

**Automatische Installation (Empfohlen)**: Standardmäßig initialisiert TikMatrix Geräte automatisch beim ersten Verbinden. Dies ist der einfachste und bequemste Weg.

**Manuelle Installation/Aktualisierung**: Nur in Sonderfällen oder für Updates erforderlich.

## Automatischer Initialisierungsprozess

Beim Verbinden eines neuen Geräts mit TikMatrix:

1. **Geräteerkennung**: TikMatrix erkennt automatisch das verbundene Gerät.
2. **Automatische Installation**: Das System installiert automatisch die TikMatrix-Anwendung ohne Benutzereingriff.
3. **Autorisierung**: Wenn das Gerät um Erlaubnis zur Installation bittet, klicken Sie einfach auf "Zulassen".
4. **Abschluss**: Der Initialisierungsprozess wird in etwa 10 Sekunden abgeschlossen.
5. **Einsatzbereit**: Ihr Gerät ist jetzt bereit für die Ausführung automatisierter Aufgaben.

Dieser automatische Prozess eliminiert die Notwendigkeit manueller Konfiguration in den meisten Fällen.

## Manuelle Installation/Aktualisierung

Für manuelle Installation oder Aktualisierung von TikMatrix APK deaktivieren Sie den Schalter `Automatisches Agent-Aufwachen` in den Einstellungen. Führen Sie dann die folgenden Schritte aus:

1. Wählen Sie Ihr Telefon und klicken Sie auf `Allgemein` > `Agent initialisieren`.
2. TikMatrix installiert automatisch zwei APK-Dateien auf dem Telefon.
3. Klicken Sie bei Aufforderung auf "Zulassen", um die Installation zu autorisieren.
4. Wenn durch Google Play-Schutz blockiert, klicken Sie auf "Trotzdem installieren", um fortzufahren.
5. Warten Sie auf den Abschluss der Installation (etwa 10 Sekunden).
6. Aktivieren Sie den Schalter `Automatisches Agent-Aufwachen`, um das automatische Aufwachen der Anwendung beim Verbinden des Telefons zu aktivieren.
7. Klicken Sie auf `Allgemein` > `Agent öffnen`, um die Installation zu testen. Wenn die Anwendung nicht startet, wenden Sie sich an den technischen Support.
