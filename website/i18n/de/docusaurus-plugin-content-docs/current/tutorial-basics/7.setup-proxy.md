# Proxy einrichten

Sie können für jedes Telefon eine unabhängige Proxy-IP-Adresse konfigurieren.

> **Haftungsausschluss:** Alle auf dieser Seite empfohlenen Websites stammen aus dem Internet und stehen in keiner Verbindung mit dieser Website. Bitte überprüfen Sie deren Gültigkeit selbst vor der Verwendung. Wir geben keine Garantien.

## 1. Einrichten eines residentiellen VPS-Proxys (wenn Sie Proxy-IP gekauft haben, können Sie diesen Schritt überspringen)

1. Erwerben Sie eine residentielle Proxy-IP VPS aus dem Zielland (vorzugsweise native IP und Dual-ISP-VPS).
2. Verwenden Sie [https://github.com/MHSanaei/3x-ui](https://github.com/MHSanaei/3x-ui) für die schnelle Einrichtung von VPN auf VPS:

    ```shell
    bash <(curl -Ls https://raw.githubusercontent.com/MHSanaei/3x-ui/refs/tags/v2.6.0/install.sh)
    # Folgen Sie den Anweisungen zur Vervollständigung der VPN-Einrichtung.
    ```

3. Nach Abschluss der Installation greifen Sie auf die x-ui-Weboberfläche zu. URL und Anmeldedaten finden Sie in der Terminal-Ausgabe nach der Installation.
4. Erstellen Sie einen neuen Benutzer in x-ui, setzen Sie das Protokoll auf `SOCKS5` und konfigurieren Sie Port und andere Parameter nach Bedarf.
5. Nach Erstellung des Benutzers sehen Sie die Proxy-IP und den Port in der x-ui-Oberfläche.
6. Verwenden Sie die Proxy-IP und den Port in der NekoBox-App auf dem Telefon.

## 3. Proxy auf dem Telefon aktivieren

1. Laden Sie den APK-Client [NekoBoxForAndroid](https://github.com/MatsuriDayo/NekoBoxForAndroid) auf das Telefon herunter und installieren Sie ihn.
2. Öffnen Sie die NekoBox-App, klicken Sie auf die Schaltfläche `+`, um einen neuen Proxy hinzuzufügen.
3. Wählen Sie `SOCKS5` oder einen anderen Proxy-Typ.
4. Geben Sie Proxy-IP und Port ein, dann klicken Sie auf `Speichern`.
5. Klicken Sie auf die Schaltfläche `Start`, um den Proxy zu aktivieren.
6. Nach Aktivierung des Proxys können Sie den Verbindungsstatus in der NekoBox-App überprüfen.

### Hinweis für Benutzer aus China

Wenn Sie sich in China befinden, funktioniert das Protokoll `SOCKS5` möglicherweise nicht ordnungsgemäß. Sie müssen einen verschlüsselten Proxy (`VLESS`) als Frontend-Proxy hinzufügen. Sie können die folgenden Tools verwenden:

1. Fügen Sie `VLESS`-Proxy zur NekoBox-App hinzu.
2. Fügen Sie `SOCKS5`-Proxy zur NekoBox-App hinzu.
3. Fügen Sie `Chain`-Proxy zur NekoBox-App hinzu. Wählen Sie `VLESS`-Proxy als Frontend-Proxy und `SOCKS5`-Proxy als Backend-Proxy.
