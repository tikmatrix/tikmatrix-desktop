# Gruppe erstellen

**Gruppe** ist eine Sammlung von Geräten, die die Verwaltung und Durchführung von Operationen auf mehreren Geräten gleichzeitig vereinfacht.

## Schritte

1. Klicken Sie auf die Schaltfläche `Gruppe hinzufügen` in der linken Seitenleiste.
2. Geben Sie den Gruppennamen ein.
3. Drücken Sie die `Enter`-Taste, um die Gruppe zu speichern.
4. Sie können Geräte auswählen, um sie in diese Gruppe zu verschieben.
