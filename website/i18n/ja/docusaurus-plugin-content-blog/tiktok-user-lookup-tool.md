## 🧭 1. これは何か

公開TikTokプロフィールデータのための軽量、高速ビューアー — 調査、QA、競合チェックに最適です。

- URL：**[https://user.tikmatrix.com/](https://user.tikmatrix.com/)**
- 100%無料、サインイン不要
- デスクトップ&モバイルで動作

---

## 🔎 2. 何が見られるか

| カテゴリ | フィールド |
|---|---|
| プロフィール | ユーザー名、ニックネーム、アバター |
| スケール | フォロワー、フォロー中、合計いいね、動画数 |
| ID | UserID、SecUID |
| メタ | **国**、アカウント作成日 |
| 追加 | クイック分析、**動画ダウンロード**リンク（公開動画） |

> ヒント：国は公開シグナルから推測されます；居住の証明ではなく、指標として扱ってください。

---

## ⚡ 3. 使用方法（3ステップ）

1) **[https://user.tikmatrix.com/](https://user.tikmatrix.com/)** を開く  
2) TikTok **ユーザー名**を入力（例：`charlidamelio`）  
3) **検索**をクリック → 結果が即座にレンダリング

## 🔐 5. プライバシー&コンプライアンス

- **公開**プロフィール情報のみを表示します。  
- ログインなし；認証情報の収集なし。  
- コンテンツをダウンロード/表示する際はクリエイターの権利を尊重してください。

> プロフィールが非公開/制限されている場合、一部のフィールドはデザインにより利用できない場合があります。
