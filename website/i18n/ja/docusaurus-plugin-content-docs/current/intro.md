# チュートリアル紹介

5分以内に**TikMatrix**と**IgMatrix**のコア機能と使用方法を理解できます。

## ソフトウェア紹介

**TikMatrix**は、AndroidデバイスでTikTokタスクを自動実行するPCデスクトップソフトウェアです。  
**IgMatrix**は、AndroidデバイスでInstagramタスクを自動実行するPCデスクトップソフトウェアです。

**注意：** **TikMatrix**と**IgMatrix**は2つの独立したソフトウェアで、同じコンピュータ上でで同時に実行できません。

## 機能と特徴

- **マルチアカウント切り替え**：同じデバイスで複数のTikTok/Instagramアカウントにログインし、自動のに切り替えてタスクを実行できます。
- **クローンAPPサポート**：カスタムパッケージ名のTikTok/Instagramクローンアプリをサポートし、異なるアカウントを異なるクローンアプリにバインドして、ターゲットAPPを自動のに開くことができます。
- ~~**自動登録**~~：**[サービス終]**
- **自動ログイン**：Instagramは2FA自動ログインをサポートしています。TikTokは一部の認証（画像認証、メール認証など）において、依然として手動介入が必要です。
- **自動プロフィール入力**：アバター、ニックネーム、プロフィール、ユーザー名の設定をサポートしています。
- **自動認識**：ワンクリックで携帯電話にログイン済みのTikTok/Instagramアカウントを認識してインポートします。
- **自動ウォームアップ**：キーワード検索、動画視聴、いいね、フォロー、コメント、お気に入りなどのウォームアップ動作をサポートしています。
- **自動投稿**：動画または画像の投稿をサポート（ソートと一括数を設定できます能）し、カスタム音楽をサポートし、タイトル、説明、ハッシュタグ、商品リンクを含みます。
- **自動削除**：再生回数または指定されたされたルールに基づいて、パフォーマンスの悪い投稿を削除します。
- **スーパーマーケティング**：実際の人間の訪問パスをシミュレート（ホームページ訪問、投稿閲覧、ランダムなフォロー/いいね/コメント/お気に入り/シェア/ダイレクトメッセージ）し、OpenAIまたは互換APIを使用してパーソナライズされたメッセージとコメントコンテンツを生成することをサポートしています。
- **一括ブースト（ライブ配信）**：ライブ配信に一括でいいねとコメントを行います。
- **ユーザー収集**：キーワード、フォロワー、またはフォローリストを通じてターゲットユーザー名を収集します。
- **一括フォローバック**：フォロワーを一括フォローし、ウェルカムメッセージを送信します。
- **タスクスケジューリング**：分単位の精度でスケジューリングし、自動再試行、間隔実行、最大再試行回数設定をサポートしています。
- **マルチデバイス並行**：一台コンピュータできます同時に接続し管理多部安卓デバイス並行运行タスク。
- **スクリーンミラーリング**：複数のデバイスの画面をWindows/macOSにミラーリングし、マウスとキーボードで制御することをサポートしています。
- **アカウントのインポート/エクスポート**：Excel形式の一括インポートとエクスポートをサポートしています。
- **多言語サポート**：PCソフトウェアは英語、ロシア語、中国語をサポートしています。注意：一部の機能では、最適な互換性を得るために携帯電話のTikTok/Instagramアプリの言語を英語に設定する必要があります。
- **工单サポート**：提供人工工单服务以解决使用过程中の问题。

## 你需要什么

### 安卓携帯電話

你できます以使用任何安卓デバイス来运行TikMatrix，包括：

- **真实安卓デバイス**
- **[主板安卓デバイス](http://www.niaozun.shop?cid=934ec2fe)**
- **[云携帯電話](https://www.geelark.cn?invite_code=XHY6a8)**
- **シミュレート器**
- **虚拟机**

Android デバイス至少需 API 21（Android 5.0），且不需要 root 权限。请在デバイス上で启用 USB 调试（开发者选项）。在一部机型（例など小米）上で，できます能还需启用“USB 调试（安全設定）”以サポートキーボードとマウス制御。

### コンピュータ

#### Windows：Windows 10または更高版本

#### Mac：macOS 15.0または更高版本（サポートApple Silicon M系列芯片和Intel芯片）
