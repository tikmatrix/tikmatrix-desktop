# チュートリアル紹介

5分以内に**TikMatrix**と**IgMatrix**のコア機能と使用方法を理解できます。

## ソフトウェア紹介

**TikMatrix**は、AndroidデバイスでTikTokタスクを自動実行するPCデスクトップソフトウェアです。  
**IgMatrix**は、AndroidデバイスでInstagramタスクを自動実行するPCデスクトップソフトウェアです。

**注意：** **TikMatrix**と**IgMatrix**は2つの独立したソフトウェアで、同じコンピュータ上で同時に実行できません。

## 機能

- **マルチアカウント切り替え**：デバイスごとに複数のアカウントを管理し、タスク実行中に自動的に切り替えます。
- **クローンアプリサポート**：カスタムパッケージ名を持つTikTok/Instagramクローンアプリをサポートし、異なるアカウントが異なるクローンを使用して、自動的にターゲットアプリを開くことができます。
- ~~**自動登録**~~ **【終了】**
- **自動ログイン**：Instagramは2FA自動ログインをサポートしています。TikTokは一部の認証フロー（キャプチャ、メール認証）で手動介入が必要な場合があります。
- **自動プロフィール入力**：アバター、ニックネーム、プロフィール、ユーザー名を自動的に設定します。
- **自動アカウント認識**：携帯電話に既にログイン済みのアカウントをワンクリックで検出してインポートします。
- **アカウントウォーミング/成長**：キーワード検索、動画視聴、いいね、フォロー、コメント、お気に入りでアカウントをウォームアップします。
- **自動投稿**：動画または画像を公開します（並べ替えとバッチ数をサポート）。カスタマイズ可能な音楽、タイトル、説明、ハッシュタグ、商品リンクが含まれます。
- **自動削除**：視聴回数のルールに基づいて、パフォーマンスの低い投稿を削除します。
- **スーパーマーケティング**：人間のような閲覧パスをシミュレート（プロフィール訪問、投稿閲覧、ランダムなフォロー/いいね/コメント/お気に入り/シェア/DM）し、オプションでOpenAIまたは互換APIを使用してパーソナライズされたDM/コメントを生成します。
- **バッチブースト（ライブ）**：ライブストリームに一括でいいねとコメントを行います。
- **ユーザー収集**：キーワード、フォロワー、フォローリストでユーザーを収集します。
- **大量フォローバック&ウェルカムDM**：フォロワーを一括でフォローし、ウェルカムメッセージを送信します。
- **分単位のタスクスケジューリング**：再試行、間隔実行、最大再試行回数設定をサポートします。
- **マルチデバイス並行処理**：1台のPCで多数のAndroidデバイスに接続して管理し、並行してタスクを実行できます。
- **スクリーンミラーリング**：複数のデバイス画面をWindows / macOSにミラーリングし、マウスとキーボードで制御します。
- **アカウントのインポート/エクスポート**：Excelに対応した一括インポート/エクスポート。
- **多言語サポート**：ソフトウェアは英語、ロシア語、中国語をサポートしています。注意：一部の機能では、最適な互換性のために携帯電話のTikTok/Instagramアプリを英語に設定する必要があります。
- **サポート&チケット**：問題に対する人的チケットサポートを提供します。

## 必要なもの

### Android携帯電話

TikMatrixを実行するために、以下を含む任意のAndroidデバイスを使用できます：

- **実際のAndroidデバイス**
- **[マザーボードAndroidデバイス](http://www.niaozun.shop?cid=934ec2fe)**
- **[クラウド携帯電話](https://www.geelark.cn?invite_code=XHY6a8)**
- **エミュレーター**
- **仮想マシン**

Androidデバイスには少なくともAPI 21（Android 5.0）が必要で、rootは必要ありません。USBデバッグ（開発者オプション）を有効にしてください。一部の機種（例：Xiaomi）では、キーボード/マウス制御を許可するために「USBデバッグ（セキュリティ設定）」を有効にする必要がある場合があります。

### コンピュータ

#### Windows：Windows 10以降

#### Mac：macOS 15.0以降（Apple Silicon & Intelサポート）

## インストール

1. [Download TikMatrix](https://tikmatrix.com/Download)または[Download IgMatrix](https://igmatrix.com/Download-IgMatrix)ページから最新のインストーラーをダウンロードします。
2. インストーラーを実行します。
3. インストールが完了するまで待ちます。
4. コンピュータで**TikMatrix**または**IgMatrix**アプリを開きます。
