# ログイン

ログインスクリプトは、ソーシャルメディアアカウントへのログインに使用されます。

## 手順

1. アカウントがアカウントリストに追加されていることを確認し（`アカウント`をクリックしてアカウントを追加）、このアカウントに使用するデバイスインデックスを選択します。
2. ログインスクリプトを実行するデバイスを単一選択または複数選択します。
3. `スクリプト` > `ログイン`をクリックします。
4. `スクリプトを開始`をクリックして開始します。

## ログインの優先順位

システムは次のログイン優先順位を使用します：

1. **メール + パスワード**: アカウントにメールが設定されている場合、最初にログインに使用されます。
2. **ユーザー名 + パスワード**: メールが設定されていない場合、ユーザー名がログインに使用されます。

## 2FAサポート（IgMatrixのみ）

IgMatrixは自動二要素認証（2FA）ログインをサポートしています：

### 2FA用のパスワード形式

2FAを使用するには、次の形式でパスワードをフォーマットします：

```plaintext
your_password:your_2fa_secret_key
```

**例:**

```plaintext
mypassword123:JBSWY3DPEHPK3PXP
```

### 2FAの仕組み

1. **パスワード解析**: システムは、パスワードにコロン（`:`）セパレータが含まれているかどうかを自動的に検出します。
2. **トークン生成**: 2FAが必要な場合、システムはシークレットキーを使用して、2FAサービスにTOTPコードをリクエストします。
3. **自動入力**: 生成された6桁のコードは、InstagramのHFA2フィールドに自動的に入力されます。
4. **シームレスなログイン**: 2FAプロセス全体が手動介入なしに自動的に行われます。

### 2FAシークレットキー

2FAシークレットキーは通常、Instagramで2FAを設定する際に取得する32文字のBase32エンコード文字列です。これは、Google AuthenticatorやAuthyなどの認証アプリで使用されるのと同じキーです。

### 2FA対応Instagramアカウントの入手先

2FAサポート付きのInstagramアカウントは、以下から購入できます：[AccsMarket](https://accsmarket.com/en/?ref=802647)

## 注意

- TikTokのログインは100％完全自動ではありません。メール認証コードの手動入力やその他の認証手順の完了が必要になる場合があります。TikTok固有のログインチャレンジ（画像CAPTCHAや人間による検証など）をバイパスしたり、「ログイン試行回数が多すぎます」などの特定のエラーメッセージを解決したりすることはできません。
- Instagramは2FA経由で100％自動ログインを実現できます。2FA対応のInstagramアカウントの場合は、パスワードに正しいシークレットキー形式を使用してください。
