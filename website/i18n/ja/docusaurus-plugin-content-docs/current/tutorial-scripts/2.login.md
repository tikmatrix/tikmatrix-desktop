# ログイン

ログイン脚本用于ログイン社交媒体账户。

## 步骤

1. 确保账户済み添加到账户列表中（点击`账户`添加账户）, し选择你需要ログイン该账户の携帯電話序号。
2. 单选または者多选要运行ログイン脚本のデバイス。
3. 点击`脚本` > `ログイン`。
4. 点击`开始脚本`开始実行。

## ログイン优先级

系统使用以下ログイン优先级：

1. **邮箱+密码**：など果账户配置邮箱，将优先使用邮箱进行ログイン。
2. **ユーザー名+密码**：など果没有配置邮箱，则使用ユーザー名进行ログイン。

## 2FA サポート（仅限 IgMatrix）

IgMatrix サポート自動双因子认证（2FA）ログイン：

### 2FA 密码形式

要使用 2FA，请按以下形式設定密码：

```plaintext
你的密码:你的2FA密钥
```

**示例：**

```plaintext
mypassword123:JBSWY3DPEHPK3PXP
```

### 2FA 工作原理

1. **密码解析**：系统自動检测密码中是否含む冒号（`:`）分隔符。
2. **認証码生成**：当需要 2FA 認証时，系统使用您の密钥向我们の 2FA 服务请求 TOTP 認証码。
3. **自動输入**：生成の 6 位数字認証码会自動输入到 Instagram の 2FA 認証框中。
4. **无缝ログイン**：整个 2FA 过程无需手動干预，自動完成。

### 2FA 密钥说明

2FA 密钥通常是一个 32 位の Base32 编码字符串，您在 Instagram 中設定 2FA 时会得る此密钥。这と Google Authenticator または Authy など認証器アプリ使用の密钥相同。

## 注意事项

- TikTok ログイン不是完全100%自動の。你できます能需要手動输入メール認証码または完成其他認証步骤。我们也无法解决 TikTok のログイン挑战（など画像認証码、人机認証など）以及错误提示: ログイン太频繁の问题。
- Instagram できます以通じて 2FA ログイン实现100%全自動ログイン。对于启用 2FA の Instagram 账户，请确保在密码中使用正确の密钥形式。
