# 投稿を削除

投稿削除スクリプトは、視聴回数に基づいて投稿を削除するために使用されます。

## 手順

1. スクリプトを実行するデバイスを選択します。
2. `スクリプト` > `投稿を削除`をクリックします。
3. タスク設定を構成します：
    - **最大視聴回数**: 削除したい投稿の最大視聴回数を入力します。
4. `スクリプトを開始`をクリックして開始します。

## 注意

- 投稿削除スクリプトは常に成功するとは限りません。必要に応じて失敗したタスクを再試行してください。
