# 提升用户

:::warning 下线提示
该機能済み整合到 [スーパーマーケティング脚本](./14.super-marketing.md)。请优先使用“スーパーマーケティング”以得る更强大、更灵活の用户提升能力。
:::

提升用户機能通じて一括フォロー/取关操作来提升ターゲット用户の曝光と互动。系统会尽量シミュレート真实用户動作以降低被检测风险，但仍建议遵循平台ルールし制御频率。

## 在“スーパーマーケティング”脚本中使用提升用户機能

1. 開く脚本：选择“スーパーマーケティング”。
2. 选择数据源类型に“用户アカウント列表”。
3. 一括粘贴ユーザー名または从文本文件インポート（每行一个）；インポート后できます选择“追加”または“覆盖”。
4. 选择数据使用策略：

    - 共享池策略：所有 bot アカウント会对相同ターゲット进行フォロー/取关。

    - 单次消耗策略：每个 bot アカウント仅处理異なるのターゲット，确保每个ターゲット只被一个アカウント操作一次。

5. 在“用户操作 — 重复回数”中設定每个ターゲットの操作回数（通常設定に 1 次）。
6. 选择操作类型：フォロー または 取消フォロー。できます在“用户操作”中进一步配置：

        - 进入用户ホームページ方式：选择“検索用户”または“直接開くホームページ（Direct Open Profile）”。

        - 发送ダイレクトメッセージ（できます选）：启用后できます粘贴预生成の多行ダイレクトメッセージ文案，每行または按段に一条候选文案；配置文案使用顺序：

                - ランダム：每次使用ランダム选一条文案。
                - 按顺序：按粘贴/インポートの顺序依次使用（できます跨タスク轮次继续）。

            另できます选择是否在文案中自動插入表情（emoji）以提高自然度。

7. 配置其他参数（し发、延时、アカウント池），确认无误后启动脚本。

## 使用建议と注意事项

- 建议先在少量アカウント上で测试，观察策略と参数是否触发平台风控。
- 合理設定延时とし发，シミュレート人工動作以降低风险。
- 长期运行时请定期检查アカウント健康、タスク日志，しに基づいて结果调整策略。
- 提升用户操作し非 100% 成功，遇到失败请再試行または排查ターゲットアカウント状态。
