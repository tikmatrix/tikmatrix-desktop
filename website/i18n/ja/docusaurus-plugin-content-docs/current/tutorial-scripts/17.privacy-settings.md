# プライバシー設定

**プライバシー設定**スクリプトは、視聴回数フィルターに基づいてTikTok投稿のプライバシー設定を一括更新するために使用されます。

:::info
このスクリプトはTikTokアカウントでのみ利用できます。
:::

## 手順

1. スクリプトを実行するデバイスを選択します。
2. `スクリプト` > `プライバシー設定`をクリックします。
3. タスク設定を構成します：
    - **投稿フィルターパラメータ**：
      - **最大視聴回数**: 投稿をフィルタリングするための最大視聴回数を入力します。この数以下の視聴回数の投稿のみが更新されます。
      - **処理する投稿の最大数**: 処理する投稿の最大数を入力します。無制限の処理には0を設定します。
    - **プライバシー設定パラメータ**：
      - **誰がこの動画を視聴できるか**: 1つのオプションを選択します：
        - `全員`: 誰でも動画を視聴できます
        - `友達`: 友達のみが動画を視聴できます
        - `自分のみ`: 自分のみが動画を視聴できます
    - **トグルパラメータ**：
      - **コメントを許可**: 投稿のコメントを有効または無効にします。
      - **コンテンツの再利用を許可**: デュエット、スティッチ、ステッカー、ストーリーに追加を含むコンテンツの再利用を有効または無効にします。
4. `スクリプトを開始`をクリックして開始します。

## 動作原理

スクリプトは次のワークフローに従います：

1. 動画タブに移動
2. 投稿をスキャンして視聴回数を解析
3. フィルター条件に一致する投稿を見つける（視聴回数 ≤ 最大視聴回数）
4. 投稿詳細ページに入る
5. 「プライバシー設定」ボタンをクリック
6. 設定ダイアログが表示されるまで待つ
7. 選択に従って視聴権限を設定
8. 「コメントを許可」スイッチをインテリジェントに切り替え
9. 「再利用を許可」スイッチをインテリジェントに切り替え
10. リストに戻り、次の投稿の処理を続ける
11. 一致するすべての投稿が処理されるまで繰り返す

## 注意

- プライバシー設定スクリプトは常に成功するとは限りません。必要に応じて失敗したタスクを再試行してください。
- スクリプトは、視聴回数フィルターに一致する投稿のみを処理します。
- 変更はすぐに適用され、スクリプトを通じて元に戻すことはできません。
- 複数の投稿でスクリプトを実行する前に、プライバシー設定を確認してください。
