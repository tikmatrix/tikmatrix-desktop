# 隐私設定

**隐私設定**脚本用于に基づいて观看数过滤条件一括更新 TikTok 投稿の隐私設定。

:::info
此脚本仅适用于 TikTok アカウント。
:::

## 步骤

1. 选择要运行脚本のデバイス。
2. 点击 `脚本` > `隐私設定`。
3. 配置タスク設定：
    - **投稿过滤参数**：
      - **最大观看数**：输入最大观看数以过滤投稿。只有观看数 ≤ 此数字の投稿会被更新。
      - **最大处理投稿数**：输入要处理の最大投稿数。設定に 0 表示不限制处理数量。
    - **隐私設定参数**：
      - **谁できます以观看此動画**：选择一个选项：
        - `所有人`：任何人都できます以观看動画
        - `朋友`：只有朋友できます以观看動画
        - `仅自己`：只有你自己できます以观看動画
    - **开关参数**：
      - **允许コメント**：启用または禁用投稿コメント。
      - **允许コンテンツ重用**：启用または禁用コンテンツ重用，包括合拍、缝合、贴纸和添加到故事。
4. 点击 `开始脚本` 开始実行。

## 工作原理

脚本遵循以下工作流程：

1. 进入 Videos 标签页
2. 扫描投稿，解析閲覧量
3. 找到符合条件の投稿（views ≤ max_views）
4. 进入投稿详情页
5. 点击 "Privacy settings" 按钮
6. など待設定弹窗出现
7. に基づいて你の选择設定观看权限
8. 智能切换 "Allow comments" 开关
9. 智能切换 "Allow reuse" 开关
10. 返回列表，继续处理下一个投稿
11. 重复以上で步骤，直到处理完所有匹配の投稿

## 注意事项

- 隐私設定脚本できます能不总是成功；など需要请再試行失败のタスク。
- 脚本仅处理符合观看数过滤条件の投稿。
- 更改会立即アプリ，且无法通じて脚本撤销。
- 在对複数投稿运行脚本之前，请确保检查你の隐私設定。
