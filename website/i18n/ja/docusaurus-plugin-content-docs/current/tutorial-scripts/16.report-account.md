# 举报アカウント

:::warning 负责任地使用
举报機能是一个强力操作。仅当确认存在违法または违规（欺诈、诈骗、垃圾信息など）时才使用。滥用举报できます能导致アカウント被限制または其他不良后果。
:::

本指南说明など何使用 Super Marketing 脚本内置の“举报アカウント”機能。该機能会自動実行固定のアプリ内举报流程以处理一批用户アカウント，但不サポート在 UI または脚本中カスタム举报パスまたは原因。

## 快速概览

- 脚本使用の固定举报流程など下（不できます修改）：

```text
举报 → 举报账号 → Something else → Frauds and scams
```

- 脚本会尝试进入ターゲット用户ホームページ，開く更多/菜单项，按固定パス选择举报し提交（若できます用）。
- 由于携帯電話端アプリ在異なる版本とデバイス上で UI できます能存在差异，脚本使用尽力而にの选择器，因此请在小一括上で先行認証。

## 通じて Super Marketing 実行の步骤

1. 開く TikMatrix/IgMatrix し选择要运行のデバイス（サポート多选）。
2. 開く `Scripts` → `Super Marketing`。
3. 将 **数据来源（Data source）** 設定に `Usernames`，インポート文本文件または粘贴ユーザー名（一行一个）；インポート后选择“追加”または“覆盖”。
4. 在 **用户动作（User Actions）** 中启用 **举报アカウント（Report account）** 开关，し确认显示の固定举报パス。
5. 配置タスクし发、延迟など参数。
6. 点击 `Start Script` 开始处理列表，监控日志とデバイス界面以确认操作成功または失败。

## 建议と注意事项

- 先在 1–5 个アカウント上で测试，确认脚本能正确识别し実行举报流程（UI 文本または位置变化会导致失败）。
- 保持保守の节奏と延迟，一括自動举报できます能触发平台の限流または风控。
- など果アプリ UI 发生变化（按钮文本、布局变动），脚本できます能无法找到对应元素，需要更新选择器または调整など待策略。
- 仅对确凿违规コンテンツ使用举报，避免骚扰または针对无辜用户滥用该機能。
- 运行后检查日志，脚本会记录诸など“找不到菜单按钮”または“找不到举报选项”の错误，便于你调整脚本または运行参数。

## 不建议使用の场景

- 需要に每个案例选择カスタム举报原因または上で传证据の场景：脚本不サポート细粒度カスタム。
- 法律/敏感事件需要人工审核または保留证据の情况。

## 故障排查

- 报错 “Could not find Report option” / “Could not find 'Report account' option”：表示アプリ UI 文本または布局済み变化。请在小一括下再次运行し收集デバイス截图用于调试。
- など果脚本显示済み提交举报但平台没有实际处理，平台できます能对自動举报作降权または需要人工审查；针对重要个案，请使用人工流程。
