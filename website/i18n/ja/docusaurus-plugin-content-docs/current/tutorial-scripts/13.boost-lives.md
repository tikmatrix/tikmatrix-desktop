# ライブブースト

ライブブーストスクリプトを使用すると、ファームアカウントを制御してターゲットライブストリームを視聴し、オプションでいいねやコメントを行うことができます。視聴時間といいね/コメント間隔を設定して、ブースト動作をカスタマイズできます。

## 手順

1. スクリプトを実行するデバイスを選択します。
2. `スクリプト` > `ライブブースト`をクリックします。
3. タスク設定を構成します：
    - **ユーザー名**: ターゲットユーザー名を1行に1つ入力します。
    - **入力方法**: **検索**と**直接**の方法から選択します。
    - **ブーストオプション**: いいねおよび/またはコメントを有効にします。
    - **ファンクラブに参加**: ストリーマーのファンクラブが利用可能な場合に参加を有効にします。
    - **毎日チェックイン**: ライブストリーム中の毎日のチェックイン活動を有効にします。
    - **いいね間隔**: 各いいねアクション間の間隔を設定します。
    - **いいねタップ数**: いいねボタンをタップする回数。
    - **コメント間隔**: 各コメントアクション間の間隔を設定します。
    - **コメント内容**: 1行に1つのコメントを入力します。
    - **絵文字を挿入**: コメントに絵文字を挿入するかどうかを選択します。
    - **コメント順序**: コメントのランダムまたは順次順序を選択します。
    - **アカウントごとのコメント数**: アカウントごとに投稿するコメントの数を設定します。
    - **視聴時間**: ライブストリームに滞在する時間を設定します。
    - **タスク間隔**: 各タスク間の間隔を設定します。
4. `スクリプトを開始`をクリックして開始します。

## 注意

- ライブブーストスクリプトは常に成功するとは限りません。必要に応じて失敗したタスクを再試行してください。
