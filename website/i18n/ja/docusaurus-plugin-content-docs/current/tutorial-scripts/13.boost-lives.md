# 提升ライブ配信

提升ライブ配信脚本允许您制御农场アカウント观看しできます选择性地いいねまたはコメントターゲットライブ配信。您できます以設定观看时长和いいね/コメント間隔来カスタム提升動作。

## 步骤

1. 选择要运行脚本のデバイス。
2. 点击`脚本` > `提升ライブ配信`。
3. 配置タスク設定：
    - **ユーザー名**：输入ターゲットユーザー名，每行一个。
    - **进入方式**：选择**検索**または**直接**方式。
    - **提升选项**：启用いいね和/またはコメント。
    - **加入フォロワー团**：启用加入主播フォロワー团機能（など果できます用）。
    - **每日签到**：启用ライブ配信期间の每日签到活动。
    - **いいね間隔**：設定每次いいね操作之间の間隔。
    - **いいね点击回数**：設定点击いいね按钮の回数。
    - **コメント間隔**：設定每次コメント操作之间の間隔。
    - **コメントコンテンツ**：每行输入一条コメント。
    - **插入表情符号**：选择是否在コメント中插入表情符号。
    - **コメント顺序**：选择コメントのランダムまたは顺序排列。
    - **每账户コメント数**：設定每个账户发布のコメント数量。
    - **观看时长**：設定在ライブ配信中停留の时间。
    - **タスク間隔**：設定每个タスク之间の間隔。
4. 点击`开始脚本`开始実行。

## 注意事项

- 提升ライブ配信脚本できます能不总是成功；など需要请再試行失败のタスク。
