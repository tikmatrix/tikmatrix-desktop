# ユーザーを収集

ユーザー収集スクリプトは、TikTokユーザー名を収集するために使用されます。ユーザー収集機能は3つのモードをサポートします：指定されたユーザーのフォロワーリストの収集、指定されたユーザーのフォローリストの収集、またはキーワードで検索してユーザーを収集します。

## 手順

1. スクリプトを実行するデバイスを選択します。複数のデバイスを選択すると、収集タスクは選択したデバイス間で均等に分散されます。
2. `スクリプト` > `ユーザーを収集`をクリックします。
3. タスク設定を構成します：
    - **収集モード**: ユーザーを収集するモードを選択します。
      - **フォロワーで収集**: 指定されたアカウントのフォロワーリストからユーザーを収集します。
      - **フォローで収集**: 指定されたアカウントのフォローリストからユーザーを収集します。
      - **キーワードで収集**: キーワードを使用してユーザーを検索して収集します。
    - **ターゲットアカウントユーザー名**: ユーザーを収集するアカウントのユーザー名を1行に1つ入力します（フォロワーおよびフォローモード用）。
    - **検索キーワード**: ユーザーを検索するためのキーワードを入力します（キーワードモード用）。
    - **最大収集数**: 収集するユーザーの最大数を設定します。

4. `スクリプトを開始`をクリックして開始します。

## 注意

- 収集されたユーザーデータは、収集モードに基づいてファイルに保存されます：
- フォロワーモードの場合：`exported_users_following_tiktok_<username>_<count>.txt`
- フォローモードの場合：`exported_users_followers_tiktok_<username>_<count>.txt`
- キーワードモードの場合：`exported_users_search_tiktok_<keyword>_<count>.txt`
- TikTokは表示できるユーザー数のみを制限するため、1回の操作で約50件のレコードしか取得できません。
