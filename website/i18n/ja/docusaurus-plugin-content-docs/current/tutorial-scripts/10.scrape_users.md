# ユーザー収集

ユーザー収集脚本用于収集TikTokユーザー名。用户収集機能サポート三种模式：收集指定された用户のフォロワー列表、收集指定された用户のフォローリスト，または通じてキーワード検索收集用户。

## 步骤

1. 选择要运行脚本のデバイス。など果选择複数デバイス，収集タスク将在所选デバイス之间平均分配。
2. 点击`脚本` > `ユーザー収集`。
3. 配置タスク設定：
    - **収集模式**：选择ユーザー収集の模式。
      - **通じてフォロワー収集**：收集指定された账户のフォロワー列表。
      - **通じてフォロー収集**：收集指定された账户のフォローリスト。
      - **通じてキーワード収集**：使用キーワード検索收集用户。
    - **ターゲット账户ユーザー名**：输入要ユーザー収集の账户ユーザー名，每行一个（用于フォロワー和フォロー模式）。
    - **検索キーワード**：输入検索用户のキーワード（用于キーワード模式）。
    - **最大収集数量**：設定要収集の最大用户数量。

4. 点击`开始脚本`开始実行。

## 注意事项

- 収集の用户数据将に基づいて収集模式保存到文件：
  - フォロワー模式：`exported_users_followers_tiktok_<ユーザー名>_<数量>.txt`
  - フォロー模式：`exported_users_following_tiktok_<ユーザー名>_<数量>.txt`
  - キーワード模式：`exported_users_search_tiktok_<キーワード>_<数量>.txt`
- TikTok只限制できます查看の用户数量，因此每次操作只能检索大约50条记录。
