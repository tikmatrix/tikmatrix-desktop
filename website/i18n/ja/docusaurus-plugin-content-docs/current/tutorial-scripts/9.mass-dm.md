# 一括ダイレクトメッセージ

:::warning 下线提示
该機能済み整合到 [スーパーマーケティング脚本](./14.super-marketing.md)。请优先使用“スーパーマーケティング”脚本以得る更强大、更灵活の一括ダイレクトメッセージと消息管理能力。
:::

一括ダイレクトメッセージ機能用于通じて一括发送ダイレクトメッセージ（DM）とターゲット用户建立联系，适用于推广、客服回访和用户唤醒など场景。系统サポート预生成模板と实时生成两种消息来源，し会尽量シミュレート人工发送以降低被检测风险。

## 在“スーパーマーケティング”脚本中使用一括ダイレクトメッセージ機能

1. 開く脚本：选择“スーパーマーケティング”。
2. 选择数据源类型に“用户アカウント列表（ターゲット名单）”。
3. 一括粘贴ターゲットユーザー名または从文本文件インポート（每行一个）；インポート后できます选择“追加”または“覆盖”。
4. 选择数据使用策略：

    - 共享池策略：所有 bot アカウント对相同ターゲット名单进行ダイレクトメッセージ。

    - 单次消耗策略：每个 bot アカウント只处理異なるのターゲット，确保每个ターゲット仅由一个アカウント发送消息一次。

5. 設定每个ターゲットの消息发送回数（通常に 1 次，以避免重复骚扰）。
6. 在“消息設定”中配置ダイレクトメッセージコンテンツと发送策略：

    - 预生成消息模板：できます粘贴多行または多段预生成ダイレクトメッセージ文案（每行または每段视に一条候选消息）。

    - 文案使用顺序：选择“ランダム”または“按顺序”使用。

        - ランダム：每次ランダム抽取一条模板发送。

        - 按顺序：按照粘贴/インポート顺序轮换使用模板（适合需要できます复现の去重策略）。

    - 插入表情（emoji）：できます开启自動在消息中插入表情以提高自然度。

    - 实时生成消息（できます选，ChatGPT API）：

        - 启用后，脚本できます调用 ChatGPT または类似 API，に基づいてターゲットユーザー名またはできます用上で下文实时生成パーソナライズ消息，适合需要大量唯一消息の场景。

        - 注意：启用实时生成需提供有效 API Key，し注意速率限制、费用と隐私合规（避免发送敏感数据）。

    - 保证唯一消息の两种常见方式：

        1. 使用“按顺序”轮换预生成模板（每条模板只使用一次直到耗尽）。

        2. 使用 ChatGPT 实时生成每条消息（基于ターゲット信息动态生成，唯一性更高）。

7. 配置開く用户方式（など需查看用户资料）：选择“検索用户”または“直接開くホームページ（Direct Open Profile）”。
8. 配置タスク間隔（最小/最大，单位：分）とし发、アカウント池など参数，确认无误后启动脚本。

:::tip 关于タスク間隔
建议設定最小/最大間隔し启用ランダム間隔，以避免所有デバイス同時に実行导致异常峰值，从而降低被平台检测または限制の风险。
:::

## 使用建议と注意事项

- ダイレクトメッセージコンテンツ应合规且多样化，避免短时间内发送大量相同コンテンツ。
- など使用实时生成，请严格监控 API 调用量と费用，し确保不向外泄露敏感信息。
- タスクできます能失败；遇到失败请再試行または查看日志，系统会自動跳过済み完成のターゲット。
