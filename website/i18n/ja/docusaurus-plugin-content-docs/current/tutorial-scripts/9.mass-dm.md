# 一括DM

:::warning
この機能は[スーパーマーケティングスクリプト](./14.super-marketing.md)に統合されました。より強力で柔軟な大量メッセージングとメッセージ管理については、スーパーマーケティングを優先してください。
:::

一括DMは、ダイレクトメッセージ（DM）を一括で送信して接続を構築するために使用されます。プロモーション、顧客フォローアップ、再エンゲージメントに役立ちます。システムは、事前生成されたテンプレートとリアルタイム生成の両方をサポートし、検出リスクを減らすために人間のようなタイミングを模倣することを目指しています。

## スーパーマーケティング内で一括DMを使用

1. スーパーマーケティングスクリプトを開きます。
2. データソースタイプを選択：「ユーザーアカウントリスト（ターゲットリスト）」。
3. ターゲットユーザー名を一括で貼り付けるか、`.txt`ファイルをインポートします（1行に1つのユーザー名）。インポート後、現在のリストに「追加」または「置換」を選択できます。
4. データ使用戦略を選択：

   - 共有プール：すべてのボットアカウントが同じターゲットプールからメッセージを送信します。

   - 単一消費：各ボットアカウントが異なるターゲットを処理するため、各ターゲットは1つのアカウントからのみメッセージを受け取ります。

5. ターゲットごとに送信するメッセージの数を設定します（通常、繰り返し連絡を避けるために1）。
6. メッセージ設定を構成：

   - 事前生成されたテンプレート：準備されたDMコピーの複数行または段落を貼り付けます。各行/段落は候補メッセージとして扱われます。

   - メッセージ順序：「ランダム」または「順次」を選択します。

   - ランダム：各送信でランダムにテンプレートを選択します。

   - 順次：貼り付け/インポートの順序でテンプレートをローテーションします（再現可能な重複排除戦略が必要な場合に便利）。

   - 絵文字を挿入：自然さを高めるために絵文字の自動挿入を有効にします。

   - リアルタイムメッセージ生成（オプション、ChatGPT API）：

   - 有効にすると、スクリプトはChatGPTまたは同様のAPIを呼び出して、ターゲットユーザー名または利用可能なコンテキストに基づいてパーソナライズされたメッセージを生成できます。多数のユニークなメッセージが必要な場合に適しています。

   - 注意：リアルタイム生成を有効にするには、有効なAPIキーが必要であり、レート制限、コスト、プライバシーを考慮する必要があります（機密データをサードパーティAPIに送信しないでください）。

   - ユニークなメッセージを保証する2つの一般的な方法：

   1. 事前生成されたテンプレートの順次ローテーションを使用します（各テンプレートは使い果たされるまで1回使用されます）。

   2. ChatGPTリアルタイム生成を使用して、ターゲットごとにユニークなメッセージを作成します。

7. ユーザープロファイルを開く方法を構成します（タスクがプロファイルを表示する必要がある場合）：「検索」または「プロファイルを直接開く」を選択します。
8. タスク間隔（最小/最大分数）、並行性、アカウントプール設定を構成してから、スクリプトを開始します。

:::tip タスク間隔について
最小間隔と最大間隔を設定し、ランダム遅延を有効にします。時間をずらすことで、同期されたアクティビティスパイクの可能性を減らし、検出リスクを下げます。
:::

## 推奨事項と注意事項

- DMコンテンツは準拠し、多様性を保ちます。短時間に多数の同一メッセージを送信しないでください。
- リアルタイム生成を使用する場合は、API使用量とコストを注意深く監視し、機密情報を送信しないでください。
- タスクは失敗する可能性があります。ログを確認し、失敗した行を再試行してください。処理されたターゲットは、後続の実行でスキップされます。
