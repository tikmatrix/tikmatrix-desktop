# 提升投稿

:::warning 下线提示
该脚本機能済み整合到 [スーパーマーケティング脚本](./14.super-marketing.md)。请优先使用“スーパーマーケティング”脚本以得る更强大、更灵活の投稿提升能力。
:::

提升投稿機能できます通じて一括いいね、シェア、お気に入り、转发と观看など操作来增加投稿の互动量。系统会尽量シミュレート真实用户動作以降低被检测の风险，但使用时仍应遵循平台ルールし适度制御频率。

## 在“スーパーマーケティング”脚本中使用提升投稿機能

1. 開く脚本：选择“スーパーマーケティング”。
2. 选择数据源类型に“投稿リンク列表”。
3. 一括粘贴投稿リンクまたは从文本文件インポート；インポート后できます选择“追加”または“覆盖”当前列表。
4. 选择数据使用策略：

    - 共享池策略：所有 bot アカウント对相同投稿进行互动。

    - 单次消耗策略：每个 bot アカウント只对異なるの投稿进行一次互动，确保每个投稿仅被一个アカウント处理一次。

5. 在“投稿操作 — 重复回数”中設定每个投稿の互动回数。若含むコメント，建议将コメント回数設定に 1，避免出现重复コメント。
6. 在“投稿操作”中选择需要の互动类型：いいね、シェア、转发、お気に入り、コメント、观看動画など。

     など果选择“コメント”，请在コメント設定中进行以下配置：

     - 预生成コメント模板：できます粘贴多行または多段预生成コメント文案（每行または每段视に一条候选コメント）。

     - 文案使用顺序：选择“ランダム”または“按顺序”使用。
         - ランダム：每次从候选文案中ランダム抽取一条。
         - 按顺序：按粘贴/インポートの顺序依次使用（适合需要できます复现の去重策略）。

     - 插入表情（emoji）：できます开启自動在コメント中插入表情以提高自然度。

     - 实时生成コメント（できます选，ChatGPT API）：
         - 启用后，脚本できます调用 ChatGPT 类 API，に基づいて投稿タイトル/説明实时生成上で下文相关且唯一のコメント文案。此方式できます显著降低重复コメントの风险，适合需要大量唯一コメントの场景。
         - 注意事项：使用实时生成需提供できます用の API Key，し注意速率限制、费用と隐私合规（不要发送敏感数据到第三方服务）。

     - 保证唯一コメントの两种常见方式：
         1. 使用“按顺序”加载し轮换预生成コメント模版（保证每条模版只被使用一次直到耗尽）。
         2. 使用 ChatGPT API 实时生成每条コメント（基于帖文タイトル/コンテンツ动态生成，天然更易保证唯一性）。

     - 建议：先用小一括投稿测试コメント策略，检查コメント质量と平台处理情况；若使用实时生成，请监控 API 调用量と成本。
7. 配置其他脚本参数（などし发数、延时、アカウント池など），确认无误后启动脚本。

## 使用建议と注意事项

- 建议先在少量投稿上で进行测试，确认策略と参数不会触发平台风控。
- コメントコンテンツ应多样化し保持自然，避免复制粘贴相同文本。
- 合理設定延时とし发，シミュレート人工動作以降低风险。
- 长期运行时请定期检查アカウント健康とタスク日志，しに基づいて情况调整策略。
