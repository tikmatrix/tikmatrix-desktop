# 投稿ブースト

:::warning 廃止のお知らせ
この機能は[スーパーマーケティング](./14.super-marketing.md)に統合されました。より強力で柔軟な投稿ブースト機能については、スーパーマーケティングスクリプトを使用してください。
:::

投稿ブースト機能は、一括でいいね、シェア、お気に入り、リポスト、投稿の視聴を行うことでエンゲージメントを増やします。システムは実際のユーザー動作をシミュレートして検出リスクを減らそうとしますが、プラットフォームルールに従い、適度に使用する必要があります。

## スーパーマーケティングで投稿ブースト機能を使用

1. スクリプトを開く：「スーパーマーケティング」を選択します。
2. データソースタイプ「投稿URLリスト」を選択します。
3. 投稿URLを一括で貼り付けるか、テキストファイルからインポートします。インポート後、既存のリストに「追加」または「上書き」を選択します。
4. データ使用戦略を選択：

    - 共有プール：すべてのボットアカウントが同じ投稿とインタラクトします。

    - 単一消費：各ボットアカウントが異なる投稿とインタラクトするため、各投稿は1つのアカウントのみで処理されます。

5. 「投稿アクション — 繰り返し」で投稿ごとの繰り返し回数を設定します。コメントが有効になっている場合は、重複コメントを避けるためにコメントの繰り返しを1に設定します。
6. 「投稿アクション」で、いいね、シェア、リポスト、お気に入り、コメント、動画視聴など、必要なインタラクションタイプを選択します。

     「コメント」を選択した場合は、次のようにコメントオプションを構成します：

     - 事前生成されたコメントテンプレート：複数行または複数段落のコメントテンプレートを貼り付けます（各行/段落は候補コメントとして扱われます）。

     - テンプレート使用順序：**ランダム**または**順次**を選択します。
         - ランダム：毎回ランダムに1つのテンプレートを選択します。
         - 順次：貼り付け/インポートされた順序でテンプレートを使用します（決定論的な重複排除に便利）。

     - 絵文字挿入：オプションで、コメントに絵文字の自動挿入を有効にして自然さを高めます。

     - リアルタイムコメント生成（オプション — ChatGPT API）：
         - 有効にすると、スクリプトはChatGPTのようなAPIを呼び出して、投稿タイトル/説明に基づいてコンテキストに応じたユニークなコメントを生成できます。これにより、繰り返しコメントが大幅に減少し、多数のユニークなコメントが必要な場合に適しています。
         - 注意：リアルタイム生成を使用するには、有効なAPIキーが必要であり、レート制限、コスト、プライバシーに注意する必要があります（機密データをサードパーティサービスに送信しないでください）。

     - ユニークなコメントを確保する2つの一般的な方法：
         1. **順次**モードで事前生成されたテンプレートをロードしてローテーションします（各テンプレートは使い果たされるまで1回使用されます）。
         2. ChatGPT APIを使用して投稿ごとにユニークなコメントを生成します（投稿内容に基づく動的生成）。

     - 推奨：小規模バッチでコメント戦略をテストし、コメントの品質とプラットフォームの処理を確認します。リアルタイム生成を使用する場合は、API使用量とコストを監視してください。

7. 他のスクリプトパラメータ（並行性、遅延、アカウントプール）を構成し、設定を確認してから、スクリプトを開始します。

## 推奨事項と注意事項

- まず少数の投稿でテストして、設定がプラットフォームの防御をトリガーしないことを確認します。
- コメントテキストは多様で自然に保ちます。同じテキストをコピーしないでください。
- 人間の動作を模倣し、リスクを減らすために、合理的な遅延と並行性を設定します。
- 長時間実行する場合は、アカウントの健全性とタスクログを監視し、必要に応じて戦略を調整します。
