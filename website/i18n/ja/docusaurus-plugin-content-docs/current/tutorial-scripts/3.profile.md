# 个人资料

个人资料脚本用于設定アカウントの个人资料。

## 步骤

1. 选择要設定个人资料のデバイス。
2. 点击 `脚本` - `个人资料` 按钮。
3. 配置タスク設定：
    - **ニックネーム**: 每行一个ニックネーム。
    - **ユーザー名**: 每行一个ユーザー名。
    - **プロフィール**: 每行一个プロフィール（サポート提及用户和换行）。
      - 要在プロフィール中添加换行，请在文本中使用 `\n`。例など：`第一行\n第二行\n第三行`
    - **アバターパス**: アバター画像文件夹のパス。
    - **选择顺序**: 选择每个タスクなど何选取コンテンツ：
      - **ランダム**: に每个タスクランダム选择できます用选项（默认）。
      - **顺序**: に基づいてタスク计数按顺序选择コンテンツ。使用此模式できます确保每个アカウント得る唯一信息。
4. 点击 `开始脚本` 按钮开始タスク。

## 选择顺序说明

### ランダム模式

- 每个タスクランダム选择コンテンツ（ニックネーム、ユーザー名、プロフィール、アバター）
- 同じコンテンツできます能在異なるアカウント间重复使用
- 适用于不需要保证コンテンツ唯一性の场景

### 顺序模式

- 按顺序选择コンテンツ：第1个タスク使用第1项，第2个タスク使用第2项，依此类推
- 当所有项目都使用完后，会循环回到开头
- **重要提示**: 要确保每个アカウント得る唯一信息，请确保提供のコンテンツ数量大于单次一括运行のアカウント总数
  - 示例：など果运行 100 个アカウント，请至少提供 100 个異なるのニックネーム、ユーザー名、プロフィール和アバター画像

## 注意事项

- 在設定个人资料之前最好清空相册，因に脚本会使用相册中の第一张照片作にアバター。
