# プロフィール

プロフィールスクリプトは、アカウントプロフィールを設定するために使用されます。

## 手順

1. スクリプトを実行するデバイスを選択します。
2. `スクリプト` > `プロフィール入力`をクリックします。
3. タスク設定を構成します：
    - **ニックネーム**: 1行に1つのニックネーム。
    - **ユーザー名**: 1行に1つのユーザー名。
    - **プロフィール**: 1行に1つのプロフィール（ユーザーのメンションと改行をサポート）。
      - プロフィールに改行を追加するには、テキストで`\n`を使用します。例：`1行目\n2行目\n3行目`
    - **アバターパス**: アバター画像フォルダーへのファイルパス。
    - **選択順序**: 各タスクのコンテンツの選択方法を選択します：
      - **ランダム**: 各タスクで利用可能なオプションからランダムに選択します（デフォルト）。
      - **順次**: タスクカウントに基づいて順番にコンテンツを選択します。このモードを使用して、各アカウントが一意の情報を取得するようにします。
4. `スクリプトを開始`をクリックして開始します。

## 選択順序ガイド

### ランダムモード

- 各タスクでコンテンツ（ニックネーム、ユーザー名、プロフィール、アバター）がランダムに選択されます
- 同じコンテンツが異なるアカウント間で複数回使用される場合があります
- コンテンツの一意性が重要でない場合に適しています

### 順次モード

- コンテンツは順番に選択されます：1番目のタスクは1番目のアイテムを取得し、2番目のタスクは2番目のアイテムを取得します
- すべてのアイテムが使用されると、最初に戻ります
- **重要**: 各アカウントが一意の情報を取得するようにするには、単一バッチ実行のアカウント総数よりも多くのアイテムを提供してください
  - 例：100個のアカウントを実行する場合、少なくとも100個の一意のニックネーム、ユーザー名、プロフィール、アバター画像を提供してください

## 注意

- プロフィールを設定する前にギャラリーをクリアすることをお勧めします。スクリプトは最初の写真をアバターとして使用するためです。
