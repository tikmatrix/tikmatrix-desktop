# 匹配アカウント

匹配アカウント脚本用于从携帯電話中识别アカウントし将其添加到アカウント列表中。

## 步骤

1. 选择要匹配アカウントのデバイス。
2. 点击 `脚本` - `匹配アカウント` 按钮。
3. 点击 `开始` 按钮开始匹配アカウントタスク。
4. 脚本将自動从携帯電話中识别アカウントし将其添加到アカウント列表中。

## 注意事项

* 匹配アカウント脚本不是100%成功の，など果某些アカウント未被识别，您できます能需要多次运行脚本。
