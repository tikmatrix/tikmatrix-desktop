# アカウントマッチ

アカウントマッチスクリプトは、携帯電話からTikTokアカウントを認識し、アカウントリストに追加するために使用されます。

## 手順

1. スクリプトを実行するデバイスを選択します。
2. `スクリプト` > `アカウントマッチ`をクリックします。
3. `スクリプトを開始`をクリックして開始します。
4. スクリプトは携帯電話のアカウントを自動的に認識し、アカウントリストに追加します。

## 注意

- アカウントマッチスクリプトは、1回の実行ですべてのアカウントを認識できない場合があります。一部のアカウントが見つからない場合は、スクリプトを再度実行してください。
