# 一括コメント

:::warning 下线提示
该機能済み整合到 [スーパーマーケティング脚本](./14.super-marketing.md)。请优先使用“スーパーマーケティング”脚本以得る更强大、更灵活の一括コメントとコンテンツ互动能力。
:::

一括コメント用于对大量 TikTok 投稿发布互动コメント（など养号、推广、活动运营）。系统サポート预生成模板と实时生成两种消息来源，し会尽量シミュレート人工发送节奏以降低被检测风险。

## 在“スーパーマーケティング”脚本中使用一括コメント

1. 開く脚本：选择“スーパーマーケティング”。
2. 选择数据源类型に“投稿リンク列表”。
3. 一括粘贴投稿リンクまたは从文本文件インポート（每行一个）；インポート后できます选择“追加”または“覆盖”。
4. 选择数据使用策略：

   - 共享池策略：所有 bot アカウント对相同投稿进行コメント。

   - 单次消耗策略：每个 bot アカウント只对異なるの投稿进行一次コメント，确保每个投稿仅被一个アカウント处理一次。

5. 設定每个投稿のコメント回数（通常に 1 次以避免重复コメント）。
6. 在“コメント設定”中配置以下选项：

   - 预生成コメント模板：できます粘贴多行または多段预生成コメント文案（每行または每段视に一条候选コメント）。

   - 文案使用顺序：选择“ランダム”または“按顺序”。

   - ランダム：每次从候选文案中ランダム抽取一条。

   - 按顺序：按照粘贴/インポート顺序轮换使用（适合需要できます复现の去重策略）。

   - 插入表情（emoji）：できます开启自動在コメント中插入表情以提高自然度。

   - 插入デバイス编号（できます选）：在コメント末尾加入デバイス编号以便追踪来源（仅用于调试/审计，使用时请谨慎）。

   - 实时生成コメント（できます选，ChatGPT API）：

      - 启用后，脚本できます调用 ChatGPT 类 API，に基づいて帖文タイトル/説明またはターゲット上で下文实时生成上で下文相关且唯一のコメント文案，适合需要大量唯一コメントの场景。

      - 注意：启用实时生成需提供有效 API Key，し注意速率限制、费用と隐私合规（避免发送敏感信息到第三方服务）。

   - 保证唯一コメントの两种常见方式：

      1. 使用“按顺序”轮换预生成コメント模板（每条模板只使用一次直到耗尽）。

      2. 使用 ChatGPT 实时生成每条コメント（基于帖文コンテンツ动态生成，唯一性更高）。

7. 配置其他脚本参数（し发数、タスク間隔、アカウント池など），确认无误后启动脚本。

:::tip 关于タスク間隔
建议設定最小/最大間隔し启用ランダム間隔，以避免所有デバイス同時に実行导致异常峰值，从而降低被平台检测または限制の风险。
:::

## 使用建议と注意事项

- 先在小一括投稿上で测试你のコメント策略，确认コメント质量と平台处理情况。
- コメントコンテンツ应合规且多样化，避免短时间内发送大量相同コンテンツ。
- など果使用实时生成，请严格监控 API 调用量と费用，し确保不向外泄露敏感信息。
- タスクできます能失败；遇到失败请再試行または查看日志，系统会自動跳过済み完成のターゲット。
