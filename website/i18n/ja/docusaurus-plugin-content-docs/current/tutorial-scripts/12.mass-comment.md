# 一括コメント

:::warning
この機能は[スーパーマーケティングスクリプト](./14.super-marketing.md)に統合されました。より強力で柔軟な一括コメントおよびエンゲージメント機能については、スーパーマーケティングを優先してください。
:::

一括コメントは、多数のTikTok投稿にエンゲージメント返信を投稿するために使用されます。アカウント育成、プロモーション、またはアクティビティブーストに役立ちます。システムは、事前生成されたテンプレートとリアルタイム生成の両方をサポートし、検出リスクを減らすために人間のようなタイミングを模倣しようとします。

## スーパーマーケティング内で一括コメントを使用

1. スーパーマーケティングスクリプトを開きます。
2. データソースタイプを選択：「投稿URLリスト」。
3. 投稿URLを一括で貼り付けるか、`.txt`ファイルをインポートします（1行に1つのURL）。インポート後、現在のリストに「追加」または「置換」を選択できます。
4. データ使用戦略を選択：

   - 共有プール：すべてのボットアカウントが同じターゲットプールからコメントします。

   - 単一消費：各ボットアカウントが異なるターゲットを処理するため、各投稿は1つのアカウントからのみコメントを受け取ります。

5. ターゲットごとに投稿するコメントの数を設定します（通常、重複コメントを避けるために1）。
6. コメント設定を構成：

   - 事前生成されたテンプレート：準備されたコメントコピーの複数行または段落を貼り付けます。各行/段落は候補コメントとして扱われます。

   - コメント順序：「ランダム」または「順次」を選択します。

   - ランダム：各送信でランダムにテンプレートを選択します。

   - 順次：貼り付け/インポートの順序でテンプレートをローテーションします（再現可能な重複排除戦略が必要な場合に便利）。

   - 絵文字を挿入：自然さを高めるために絵文字の自動挿入を有効にします。

   - デバイス番号を挿入（オプション）：トレース用にデバイスの数値ラベルをコメントに追加します（慎重に使用してください。デバッグ/監査用）。

   - リアルタイムコメント生成（オプション、ChatGPT API）：

      - 有効にすると、スクリプトはChatGPTまたは同様のAPIを呼び出して、投稿タイトル/説明または利用可能なコンテキストに基づいてコンテキストに応じたユニークなコメントを生成できます。多数のユニークなコメントが必要な場合に適しています。

      - 注意：リアルタイム生成を有効にするには、有効なAPIキーが必要であり、レート制限、コスト、プライバシーを考慮する必要があります（機密データをサードパーティサービスに送信しないでください）。

   - ユニークなコメントを保証する2つの一般的な方法：

      1. 事前生成されたテンプレートの順次ローテーションを使用します（各テンプレートは使い果たされるまで1回使用されます）。

      2. ChatGPTリアルタイム生成を使用して、ターゲットごとにユニークなコメントを作成します。

7. 他のスクリプトパラメータ（並行性、タスク間隔、アカウントプールなど）を構成し、準備ができたらスクリプトを開始します。

:::tip タスク間隔について
最小間隔と最大間隔を設定し、ランダム遅延を有効にします。時間をずらすことで、同期されたアクティビティスパイクの可能性を減らし、検出リスクを下げます。
:::

## 推奨事項と注意事項

- まず小規模バッチでコメント戦略をテストして、品質とプラットフォームの応答を検証します。
- コメントテキストは準拠し、多様性を保ちます。短期間に多数の同一メッセージを送信しないでください。
- リアルタイム生成を使用する場合は、API使用量とコストを厳密に監視し、機密情報を送信しないでください。
- タスクは失敗する可能性があります。ログを確認し、失敗した行を再試行してください。処理されたターゲットは、後続の実行でスキップされます。
