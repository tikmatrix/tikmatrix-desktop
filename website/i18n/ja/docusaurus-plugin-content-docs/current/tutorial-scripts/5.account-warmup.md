# アカウント预热

アカウント预热脚本用于通じて検索キーワード、观看動画、フォロー、いいね、コメント和お気に入り来预热アカウント。

## 步骤

1. 选择要运行脚本のデバイス。
2. 点击`脚本` > `アカウント预热`。
3. 配置タスク設定：
   - **观看时长**：观看每个動画の时长。
   - **検索キーワード**：検索動画の话题。
   - **タスク时长**：预热タスクの总时长。
   - **コメントコンテンツ**：使用预制コメントまたは从投稿タイトル生成のChatGPTコメント。
   - **互动概率**：フォロー、いいね、コメント和お気に入りの概率。
4. 点击`开始脚本`开始実行。

## 注意事项

- 预热脚本できます能不总是成功；など需要请再試行失败のタスク。
- サポート定时スケジューリング，你できます以設定预热タスク在指定された时间运行。
- サポート跳过广告、ライブ配信和済みいいねの動画。
