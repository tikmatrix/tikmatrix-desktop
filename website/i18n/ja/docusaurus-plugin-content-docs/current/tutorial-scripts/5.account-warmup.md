# アカウントウォームアップ

アカウントウォームアップスクリプトは、キーワード検索、動画視聴、フォロー、いいね、コメント、お気に入りによってアカウントをウォームアップするために使用されます。

## 手順

1. スクリプトを実行するデバイスを選択します。
2. `スクリプト` > `アカウントウォームアップ`をクリックします。
3. タスク設定を構成します：
   - **視聴時間**: 各動画を視聴する時間。
   - **検索キーワード**: 動画を検索するトピック。
   - **タスク時間**: ウォームアップタスクの合計時間。
   - **コメント内容**: 事前に作成されたコメントまたは投稿キャプションから生成されたChatGPTコメントを使用します。
   - **インタラクション確率**: フォロー、いいね、コメント、お気に入りの確率。
4. `スクリプトを開始`をクリックして開始します。

## 注意

- ウォームアップスクリプトは常に成功するとは限りません。必要に応じて失敗したタスクを再試行してください。
- スケジューリングをサポートしているため、特定の時刻にウォームアップタスクを実行するように設定できます。
- 広告、ライブストリーム、既にいいねした動画のスキップをサポートしています。
