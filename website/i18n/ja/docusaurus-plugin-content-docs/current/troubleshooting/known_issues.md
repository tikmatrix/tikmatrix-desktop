# 既知の問題

## ポート競合エラー

ログに以下のエラーが表示される場合：

```text
tcp connect error: The connection could not be established because the target computer refused the connection request. (os error 10061)
```

これはポート競合を示しています。この問題を解決するには：

1. TikMatrix/IgMatrixアプリケーションを完全に再起動して、再度試してください。
2. TikMatrix/IgMatrixと同時に他のデバイス制御ソフトウェアを実行しないでください。ポート競合を引き起こす可能性があります。
3. 他のアプリケーションが同じ通信ポートを使用していないことを確認してください。

このエラーは通常、複数のデバイス制御アプリケーションが同時に実行され、通信ポートで競合が発生した場合に発生します。

## クラウドフォンスクリプトの失敗

PCとクラウドフォンデータセンター間に十分で安定したネットワーク帯域幅を確保してください。最良の結果を得るには、PCとクラウドフォンデータセンターを同じ国または地域に配置して、遅延とパケット損失を減らし、自動化タスクを確実に実行できるようにします。

## 不安定なスクリプト実行、ランダムエラー、一貫性のない結果

これはADB接続品質に関連していることがよくあります。USB接続を使用している場合は、別のケーブルまたはUSBポートを試してください。ワイヤレスADBを使用している場合は、PCとデバイス間の安定したネットワーク接続と良好な信号強度を確保してください。

## TikTok/Instagramアプリの更新によるスクリプトの破損

TikTokとInstagramは頻繁に更新され、自動化スクリプトが失敗する可能性があります。サポートチケットを提出してください。最新のアプリバージョンに適応するようにスクリプトをできるだけ早く更新します。
