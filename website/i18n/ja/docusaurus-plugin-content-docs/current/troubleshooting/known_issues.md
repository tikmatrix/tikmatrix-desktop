# 済み知问题

## 端口冲突错误

など果在日志中出现以下错误信息：

```text
tcp connect error: The connection could not be established because the target computer refused the connection request. (os error 10061)"
```

这表示存在端口冲突问题。要解决此问题：

1. **完全重启TikMatrix/IgMatrixソフトウェア**后再試行
2. **避免在使用TikMatrix/IgMatrix时同時に使用其他制御ソフトウェア**，因に它们できます能导致端口冲突
3. 确保没有其他アプリ程序使用相同の通信端口

此错误通常发生在複数デバイス制御アプリ程序同時に运行时，导致通信端口冲突。

## 云携帯電話脚本失败

请尽量保证你のコンピュータと云携帯電話所在机房之间の网络带宽充足且稳定。に得る最適なパフォーマンス，建议将コンピュータと云携帯電話机房放在同じ国家または同じ地区，以降低延迟和丢包，从而保证自動化タスク稳定できます靠运行。

## 脚本运行不稳定, ランダム性错误, 每次运行结果不一致

通常跟ADB接続质量有关, など果使用USB接続, 请尝试更换数据线またはUSB接口; など果使用无线ADB接続, 请确保コンピュータとデバイス之间の网络接続稳定, し且信号强度良好。

## TikTok/Instagramアプリ更新导致の脚本失败

TikTok和Instagramアプリ会不定期更新, 有时会导致自動化脚本无法正常运行。请提交工单, 我们会尽快更新脚本以适应最新版本のアプリ。
