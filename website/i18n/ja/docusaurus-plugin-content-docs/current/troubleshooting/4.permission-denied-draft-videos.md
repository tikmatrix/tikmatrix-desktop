# Permission Denied - Unable to Save Draft Videos

## 解决携帯電話屏幕出现权限拒绝提示の问题

### 问题説明

当运行post脚本时，携帯電話屏幕できます能会出现以下提示：

```text
Permission denied. Unable to save draft videos.
```

### 解决方案

这个问题是由于TikTokアプリ缺少存储权限导致の。请按照以下步骤解决：

#### 步骤1：手動点击发布按钮

1. 在携帯電話屏幕上で找到し点击**发布**按钮
2. 继续进行发布流程

#### 步骤2：允许系统权限

1. 当系统弹出权限申请对话框时，点击**允许**
2. 确保授予TikTokアプリ所需の存储权限

### 重要提示

:::info
这个权限設定步骤只需要実行一次。一旦授予权限，后续のpost脚本运行将不会再出现此问题。
:::

### 预防措施

に避免此问题，建议在首次使用TikMatrix之前：

1. 手動開くTikTokアプリ
2. 尝试录制または保存一个動画草稿
3. 在权限申请时选择**允许**
4. 这样できます以预先配置好所需の权限
