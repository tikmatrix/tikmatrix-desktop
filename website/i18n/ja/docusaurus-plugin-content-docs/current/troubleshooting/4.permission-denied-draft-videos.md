# 権限拒否 - 下書き動画を保存できません

## モバイル画面での権限拒否問題の解決

### 問題の説明

投稿スクリプトを実行する際、モバイル画面に以下のプロンプトが表示される場合があります：

```text
Permission denied. Unable to save draft videos.
```

### 解決策

この問題は、TikTokアプリにストレージ権限が不足しているために発生します。以下の手順で解決してください：

#### ステップ1：手動で公開ボタンをクリックする

1. モバイル画面で**公開**ボタンを見つけてタップします
2. 公開プロセスを続行します

#### ステップ2：システム権限を許可する

1. システム権限リクエストダイアログが表示されたら、**許可**をタップします
2. TikTokアプリに必要なストレージ権限を付与することを確認します

### 重要な注意事項

:::info
この権限設定プロセスは一度だけ実行する必要があります。権限が付与されると、その後の投稿スクリプト実行ではこの問題は発生しません。
:::

### 予防措置

この問題を回避するために、TikMatrixを初めて使用する前に以下を行うことをお勧めします：

1. TikTokアプリを手動で開く
2. 動画の録画または下書きの保存を試みる
3. 権限を求められたら**許可**を選択する
4. これにより、必要な権限が事前に設定されます
