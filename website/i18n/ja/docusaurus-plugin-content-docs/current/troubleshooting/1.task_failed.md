# タスク実行失敗のトラブルシューティング手順

## ステップ1：Agentステータスの確認（タスクが起動しない場合）

1. サイドバーをクリック：一般 → Agentを開く。
2. 端末でTikMatrixアプリが開いていることを確認してください。
3. 10秒以内に成功ステータスが表示されるか観察します。
4. Agentが失敗ステータスを示す場合は、デバイス初期化ガイドの手動初期化手順に従ってください：../tutorial-basics/2.init-device.md

## ステップ2：TikTok/Instagramアプリの設定確認（スクリプトがアプリを正しく開閉しない場合）

1. 公式のTikTok/Instagramアプリを使用している場合は、「TikTokを開く」/「Instagramを開く」をクリックして、デバイスでアプリが正しく開くか確認してください。開かない場合は、タイトルバーの設定ボタンをクリックして、正しいTikTokバージョン（インターナショナル/アジア）を選択してください。Instagramには公式版のみがあります。
2. TikTok/Instagramのクローンバージョンを使用している場合は、**アカウントリスト**でクローンアプリのパッケージ名を設定していることを確認してください。クローンアプリの使用ガイドを参照：../tutorial-basics/10.clone-apps-support.md。また、タスク実行時に**マルチアカウントモード**が有効になっていることを確認してください。

## ステップ3：TikTok/Instagramの言語設定確認

現在、TikTokとInstagramは英語のみをサポートしています。アプリが英語でない場合は、以下の手順で英語に設定してください：

1. デバイスでTikTokアプリを開きます。
2. プロフィールに移動します。
3. 右上の3つの点をタップして設定を開きます。
4. 「Language」セクションまでスクロールします。
5. アプリの言語として「English」を選択します。
6. 同じ手順を繰り返してInstagramも英語に設定します。

## ステップ4：テストスクリプトの実行

1. サイドバーをクリック：スクリプト → テストスクリプト。
2. スクリプトが完了するまで待ちます。
3. テストが失敗した場合は、ログを確認して問題を解決してみてください。一般的な問題については、下記の「既知の問題」セクションを参照してください。

## ステップ5：サポートチケットの提出

前の4つのステップで問題が特定できない場合は、サポートチケットを作成してください。ログを自動的に収集し、トラブルシューティングをサポートします。手順：

1. タイトルバーの「サポート」ボタンをクリックします。
2. 「チケット作成」をクリックします。
3. 連絡先情報と問題の説明を入力してください。迅速な解決のために、スクリプト実行中の画面録画を添付してください。
4. 影響を受けるデバイスを選択します（複数選択可）。これは重要です — 選択したデバイスのログのみを受信できます。
5. 「送信」をクリックします。
6. 失敗前の最後の画面のUI XMLファイルなど、追加情報を要求する場合があります：タスクが失敗する前の最後のページに端末を手動で移動し、デバイス画面右側の「Debug」ボタンをクリックし、「Download XML」をクリックします。ダウンロードしたXMLとチケット番号をサポートメールに送信してください：[support@tikmatrix.com](mailto:support@tikmatrix.com)
