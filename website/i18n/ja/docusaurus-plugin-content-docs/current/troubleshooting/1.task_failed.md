# タスク実行失败排查步骤

## 第1步：检查Agent状态(など果タスク未能成功启动)

1. 点击左侧边栏“通用” - “開くAgent”
2. 确认携帯電話是否済み開くTikMatrix app
3. 观察是否在10秒内显示成功状态
4. など果Agent状态に失败，请按照[初始化デバイス](../tutorial-basics/2.init-device.md)文档步骤进行手動初始化

## 第2步：检查TikTok/Instagramアプリ状态(など果脚本没有正确開くまたは关闭アプリ)

1. など果您使用の是官方のTikTok/Instagramアプリ，请点击“開くTikTok”/“開くInstagram”，观察携帯電話是否正确開くTikTokまたはInstagramアプリ,など果没有，请点击タイトル栏中の設定按钮, 选择正确のTikTok 版本(国际版/亚洲版), Instagram只有唯一の官方版本
2. など果您使用の是克隆版TikTok/Instagramアプリ，请确保您済み经在**アカウント列表**中正确配置クローンアプリの**パッケージ名**, 请参考[使用克隆版TikTok/Instagramアプリ](../tutorial-basics/10.clone-apps-support.md)文档进行配置, し且确保运行タスクの时候开启**多アカウント模式**

## 第3步：检查TikTok/Instagram言語設定

目前我们只サポート英文のTikTok和Instagram。など果您のTikTokまたはInstagramアプリ不是英文，请按照以下步骤进行設定：

1. 開く携帯電話上でのTikTokアプリ
2. 进入个人资料
3. 点击右上で角の三个点開く設定
4. 滚动到“言語”一部
5. 选择“英語”作に言語
6. 重复以上で步骤にInstagramアプリ設定言語に英語

## 第4步：运行测试脚本

1. 点击左侧边栏“脚本” - “测试脚本”
2. など待脚本运行完成
3. など果测试脚本运行失败，请查看日志し尝试解决问题，常见问题请参考下方の“済み知问题”一部

## 第5步：提交工单

など果前4步检查都没有发现问题，需要创建一个工单, 我们会自動收集日志し协助您解决问题。请按照以下步骤操作：

1. 点击タイトル栏の“サポート”按钮
2. 点击“创建工单”按钮
3. 输入您の联系方式和问题説明, に更快地解决问题, 请尽量拍摄脚本実行过程中携帯電話の屏幕录像し添加到附件中
4. 选择受影响のデバイス,できます以多选(重要, 只有选择受影响のデバイス, 我们才能收到相关日志)
5. 点击“提交”按钮
6. 我们できます能会进一步联系你以获取更多信息,比など下载指定された页面のUI xml文件: 手動操作携帯電話停留在タスク失败前の最后页面, 点击携帯電話大屏右侧の“调试”按钮, 然后点击“下载XML”按钮, 将下载のxml文件发送到我们の邮箱:`support@tikmatrix.com`し附上で工单号
