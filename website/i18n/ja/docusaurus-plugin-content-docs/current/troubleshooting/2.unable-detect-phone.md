# 无法检测到携帯電話

## 无法检测到携帯電話排查步骤

### 第一步：检查ADB接続状态

1. 在TikMatrix中点击`設定` > `開くADB终端`  
2. 実行`adb devices`命令，检查携帯電話是否出现在列表中

### Windows系统

```shell
.\adb.exe devices
```

### Mac/Linux系统

```shell
./adb devices
```

### 输出示例

```shell
List of devices attached
<设备ID>    <状态>
```

### など果デバイス没有出现在adbデバイス列表中

请按照[接続デバイス](../tutorial-basics/1.connect-to-devices.md)文档步骤进行排查

### など果デバイス状态に`unauthorized`

できます能の原因是デバイス未授权,请在携帯電話上で点击`允许USB调试`

### など果デバイス状态に`offline`

できます能の原因是USB线松动,请重新接続携帯電話,拔出USB线,重新接続

### など果デバイス状态に`device`

デバイス済み经接続成功,など果TikMatrix仍然没有检测到携帯電話, 请确认TikMatrix是否激活, 免费试用也需要在授权页面激活试用

### など果问题仍然存在

- adb devices命令输出截图
- 你済み经尝试过の步骤
