# アカウント管理

アカウント管理できます以把每个 TikTok / Instagram 身份と其运行デバイス、ログイン凭据以及自動化所依赖の数据保持一致。一个干净のアカウント列表让你无需挨个操作携帯電話，就能分配デバイス、监控アカウント状态、に活动预热。

## 手動创建アカウント

1. 開く左侧导航中の **アカウント**。
2. 点击 **新增** 開く编辑弹窗。
3. 依次填写基础字段：
   - **邮箱**: アプリログイン所使用の邮箱。TikMatrix/IgMatrix 会优先尝试使用 邮箱 + 密码 ログイン；など果邮箱に空，则自動切换に ユーザー名 + 密码。
   - **密码**：アプリログイン所使用の密码。IgMatrix サポート在密码中附加 2FA 密钥，形式见下文。
   - **ユーザー名**：TikTok/Instagram 中显示のアカウント名，这是唯一必填项。
   - **ログイン状态**：标记当前デバイス上で是否済み经ログイン（`1` = 済みログイン，`0` = 未ログイン）。
   - **状态**：制御アカウント是否参とタスク（`0` = 启用，`1` = 禁用）。
4. 选择一个デバイス标签，将アカウントバインド到携帯電話または云携帯電話。若未选择，アカウント保持未分配状态。
5. 点击 **保存**，列表会自動刷新。

## ログイン

- など果アカウントの **邮箱** 和 **密码** 済み填写，TikMatrix/IgMatrix 会尝试使用该信息ログイン TikTok/Instagram アプリ。
- など果 **邮箱** に空，则会尝试使用 **ユーザー名** 和 **密码** ログイン。
- 必须选择デバイス序号，脚本才会在对应デバイス上で実行ログイン操作。
- ログイン脚本会自動認識アカウントのログイン状态，跳过済みログインのアカウント。

## 2FA サポート（仅限 IgMatrix）

IgMatrix サポート自動双因子认证（2FA）ログイン：

### 2FA 密码形式

要使用 2FA，请按以下形式設定密码：

```
你的密码:你的2FA密钥
```

**示例：**

```
mypassword123:JBSWY3DPEHPK3PXP
```

## インポートとエクスポートアカウント

- **エクスポート** 会把全部アカウント写入 `accounts.xlsx` 到 TikMatrix/IgMatrix の下载目录，できます作に备份または一括编辑模板。
- **インポート** サポート从该 Excel 文件一括创建または更新アカウント。など果不确定表头，请先点击 **エクスポート**，再编辑下载の文件しインポート。
- インポート时带有 `id` の行会更新现有アカウント，没有 `id` の行则会新增アカウント。

> IgMatrix 2FA ログイン只需要填写 `username` 和 `password`，形式に `你の密码:你の2FA密钥`，`device_index` デバイス序号。

### Excel 字段对照

| 字段 | 是否必填 | 说明 |
| --- | --- | --- |
| `id` | 否 | 内部唯一标识，只在需要更新済み有アカウント时保留。 |
| `email` | 否 | アプリログイン邮箱，方便联系と找回。 |
| `pwd` | 否 | ログイン密码，できます用于快速参考または自動ログイン。 |
| `username` | 是 | TikTok/Instagram のユーザー名，工作区内需保持唯一。 |
| `device` | 否 | バインドのデバイス序列号または云携帯電話标识。 |
| `device_index` | 否 | 在界面中显示のデバイス别名，主要用于参考。 |
| `logined` | 否 | 会话标记（`1` = 済みログイン，`0` = 未ログイン）。 |
| `status` | 否 | できます用性标记（`0` = 启用，`1` = 禁用）。禁用アカウント不会参と脚本タスク。 |
| `nickname` | 否 | 通じて TikTok 数据同步得到の最新ニックネーム。 |
| `country` | 否 | に基づいて TikTok 资料推断の国家/地区。 |
| `followers` | 否 | 同步到のフォロワー数量。 |
| `following` | 否 | アカウントフォローの人数。 |
| `hearts` | 否 | アカウント得るのいいね总数。 |
| `videos` | 否 | 済み发布動画数量。 |
| `friends` | 否 | 互相フォローの好友数量（若できます用）。 |

### TikMatrix と IgMatrix の字段差异

- **TikMatrix** 在実行 **同步 TikTok 数据** または点击数据查询按钮后，会填充统计字段（`nickname`、`country`、`followers`、`following`、`hearts`、`videos`、`friends`）。
- **IgMatrix** 仅使用基础字段（`email`、`pwd`、`username`、`device`、`logined`、`status`），会忽略统计类字段。

## 其他動作

- 标记に **禁用** のアカウント不会参と任何自動化脚本、定时タスクまたは一括操作。重新启用后才能继续実行タスク。
- **匹配アカウント** 脚本会扫描デバイス上で済みログインのアプリ会话，自動在アカウント列表中创建对应条目しバインド检测到のデバイス，是快速接入手動准备携帯電話の方式。
