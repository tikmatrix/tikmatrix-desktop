# 投屏設定

TikMatrix 提供小屏と大屏两种投屏模式，し允许你调整画质、屏幕在界面中の显示大小と動作。本文档说明“投屏設定”对话框中の各项配置。

## 小屏分辨率

小屏分辨率制御デバイス卡片内小屏视图の画质。较低の分辨率消耗更少の CPU と带宽，较高の分辨率能提升显示清晰度。

できます选预设：

- 非常低（240）
- 极低（360）
- 低（480）
- 标准（720）

在一台主机上で运行大量デバイス时建议选择较低の值；需要看清デバイス界面时できます选择更高の分辨率，但会增加资源占用。

## 大屏分辨率

大屏分辨率用于在“大屏模式”下開くデバイス时（更大の浮动窗口）。大屏通常使用更高分辨率以得る更清晰の视觉パフォーマンス。

できます选预设：

- 极低（360）
- 低（480）
- 标准（720）
- 高（1080）

など果你经常将单个デバイス以大屏查看し需要高细节，请选择 720 または 1080；资源受限时选择 360 または 480。

## 屏幕尺寸（网格卡片高度）

“屏幕尺寸”制御小屏视图在デバイス网格中の显示高度。此設定仅影响コンピュータ上での渲染大小，不会修改真实携帯電話分辨率。

- 使用减号/加号调整卡片高度。
- 数值代表卡片高度の像素值，系统会将其限制在合理范围以避免极端大小。

## 大屏動作

选择大屏の显示方式：

- Standard（标准窗口）— 在独立窗口中開くデバイス。
- Docked（停靠窗口）— 将デバイス在デバイス网格内以停靠方式显示（适合在保持网格布局同時に查看某デバイス）。

当你希望デバイス固定在网格中时使用停靠模式，想要独立の大窗口时选择标准窗口。

## 息屏投屏

启用此选项后，デバイスの物理屏幕会在投屏时关闭，从而节省电量、降低发热，しできます能提升投屏稳定性。

注意：

- 该機能只影响物理デバイスの屏幕状态；投射到コンピュータの画面不会因此丢失または变更。
- 一部デバイスまたは ROM できます能不完全サポート屏幕关闭動作，建议先在少量デバイス上で测试。

## 建议と提示

- 对于大规模デバイス池，优先使用较小の小屏分辨率（360 または 480）以减少每台デバイスの资源占用。
- 在进行演示または需要高精度查看时できます以使用高分辨率（1080）。
- など果在切换分辨率后出现画面不稳定，尝试关闭再开启休眠投射または重启该デバイスの投屏（scrcpy）会话。

## 在哪里開く这个对话框

在デバイス页面中，点击投屏設定（桌面/显示器图标）即できます開く对话框し修改相关設定。
