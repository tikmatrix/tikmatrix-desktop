# 接続デバイス

TikMatrixサポート安卓デバイスのUSB和TCP接続。

## USB接続

1. 開く携帯電話の开发者設定し启用USB调试（参考下方截图）。
   - **小米携帯電話用户注意**：您还必须在开发者选项中启用"通じてUSB安装"和"USB调试（安全設定）"这两个选项。
2. 通じてUSB将携帯電話接続到コンピュータ。
3. 当携帯電話提示时，点击`允许`以启用USB调试。
4. など待接続建立。

## TCP接続

1. 通じてUSB将携帯電話接続到コンピュータ。
2. 在TikMatrix中，点击`ADB命令` > `启用TCP`以启用TCP/IP接続。
3. 断开USB线缆（など果使用携帯電話盒子，できます切换到OTG模式）。
4. 点击`扫描TCPデバイス`按钮し输入IP范围（你できます以在路由器管理页面找到这个信息）。
5. 点击`扫描`在指定されたIP范围内検索デバイス。
6. など待接続建立。

**云携帯電話用户**：TikMatrix现在サポート保存云携帯電話接続信息以便快速訪問。在扫描对话框中切换到`云携帯電話列表`标签页即できます保存和管理您の云携帯電話接続。详细说明请参阅[接続云携帯電話](./1.connect-to-cloudphone.md)指南。

## など何接続主板または机箱

1. 向主板または机箱销售商索要授权文件`adbkey`和`adbkey.pub`。
2. 将这些文件复制到コンピュータの`C:\Users\你のコンピュータユーザー名\.android`目录下。
3. 重启コンピュータ。

## など何接続更多USBデバイス

💡 理论上で，USB最多できますサポート127个デバイス。  
但实际上で，大多数消费级主板由于固件和芯片组の限制，通常将其限制在约40个デバイス。

🖥️ 服务器级主板（例などX79架构）则異なる — 它们通常移除这些USB限制，允许您大规模扩展。

🔧 など何成功接続更多デバイス：

1. 使用PC背面のUSB端口 — 它们と主板接続更に直接。  
2. 使用USB 2.0端口（通常是黑色の），而不是USB 3.0（蓝色），后者对于大型デバイス群できます能不稳定。  
3. 在BIOS設定中：  
   - 禁用XHCI  
   - 启用EHCI

这些步骤将帮助您の系统できます靠地识别更多デバイス。
