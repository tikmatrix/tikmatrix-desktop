# 许できます证迁移

将您のTikMatrix许できます证从一台计算机转移到另一台计算机。这在升级硬件または更换计算机时非常有用。

## 必要

- 当前计算机上で有有效许できます证（激活码またはStripe订阅）
- ターゲット计算机没有现有のTikMatrix许できます证
- 每月最多允许5次迁移

## 迁移步骤

### 步骤1：開く迁移对话框

1. 在当前计算机上で启动TikMatrix
2. 点击タイトル栏の**许できます证图标**
3. 点击**"迁移许できます证"**按钮

![许できます证迁移按钮](../img/migrate-button.webp)

### 步骤2：获取ターゲット机器ID

在ターゲット计算机上で：

1. 安装し启动TikMatrix
2. 点击タイトル栏の**许できます证图标**
3. 复制**机器ID**
4. 将此ID发送到当前计算机

![ターゲット机器ID](../img/target-machine-id.webp)

### 步骤3：認証和迁移

回到当前计算机：

1. 在迁移对话框中粘贴**ターゲット机器ID**
2. 点击**"認証"**检查互換性
3. 查看显示の许できます证详细信息

![認証成功](../img/validation-success.webp)

1. 勾选确认复选框
2. 点击**"迁移许できます证"**し确认

![迁移确认](../img/migration-confirm.webp)

### 步骤4：完成設定

1. など待迁移完成
2. 在ターゲット计算机上で重启TikMatrix
3. 您の许できます证现在在新计算机上で处于活跃状态

![迁移成功](../img/migration-success.webp)

## 重要警告

⚠️ **许できます证迁移无法撤销**

- 许できます证完全从源计算机转移到ターゲット计算机
- 您の旧计算机立即失去訪問权限
- 每月最多5次迁移
- 两台计算机都需要稳定の网络接続

## 迁移コンテンツ

### 激活码用户

- 许できます证状态和剩余天数
- 许できます证代码信息

### Stripe订阅用户

- 订阅状态和账单信息
- 续费日期和计划详情

## 故障排除

### 常见错误信息

#### "ターゲット机器済み有许できます证"

ターゲット计算机済み经有活跃の许できます证。迁移只能迁移到没有现有许できます证の计算机。

#### "超过月度迁移限制"

您每月只能迁移5次。请など到下个月または联系客服。

#### "无效の机器ID形式"

确保您正确复制完整の机器ID。它应该至少10个字符长。

#### "迁移認証失败"

检查以下コンテンツ：

- 您当前の许できます证是活跃且未过期の
- ターゲット机器ID是正确の
- 两台计算机都有网络接続

### 获取サポート

联系[Telegram客服](https://t.me/tikmatrix_agent_bot)し提供：

- 错误信息の截图
- 您当前和ターゲットの机器ID
- 问题説明

## 常见问题

**我できます以迁移回原来の计算机吗？**

できます以，但它会计入您の月度迁移限制。

**迁移后我のデバイス接続会怎样？**

デバイス接続と计算机バインド。您需要在新计算机上で重新接続デバイス。

**我できます以迁移试用许できます证吗？**

不できます以，只有付费许できます证才能迁移。

**迁移会影响剩余の许できます证天数吗？**

不会，迁移后您の剩余天数保持不变。
