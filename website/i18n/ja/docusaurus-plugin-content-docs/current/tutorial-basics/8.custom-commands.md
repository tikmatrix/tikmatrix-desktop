# カスタムADBコマンド

TikMatrixでは、カスタムADBコマンドを作成および管理して、一般的な操作を素早く実行し、ワークフローを合理化し、効率を向上させることができます。

## カスタムADBコマンドとは？

カスタムADBコマンドを使用すると、頻繁に使用するADB命令を保存し、ワンクリックで実行できるため、毎回手動でコマンドを入力する必要がなくなります。

## カスタムコマンドインターフェースへのアクセス

1. TikMatrixメインインターフェースで、左サイドバーの**カスタムコマンド**をクリックします。
2. インターフェースには、保存されたすべてのコマンドが表示され、コマンドを追加またはリセットするオプションがあります。

## プリセットコマンド

TikMatrixは、いくつかの便利なプリセットコマンドを提供します：

- **ホーム**: デバイスのホーム画面に戻る
- **戻る**: 戻る操作を実行
- **TCPを有効にする**: TCP/IP接続モードを有効にする
- **高速入力を有効にする**: 高速入力メソッドを有効にする
- **ギャラリーを開く**: デバイスギャラリーを開く
- **設定を開く**: デバイス設定を開く
- **NekoBoxを開く**: NekoBoxアプリを開く（インストールされている場合）

## カスタムコマンドの追加

1. インターフェースの上部にある**コマンドを追加**をクリックします。
2. ダイアログで入力します：
   - **コマンド名**: コマンドの認識可能な名前
   - **コマンド引数**: ADBコマンドパラメータ（「adb」プレフィックスを含める必要はありません）
3. **コマンドを追加**をクリックして保存します。

## 複数行コマンドサポート

カスタムコマンドは複数行入力をサポートします。各行は個別のADBコマンドとして扱われ、順番に実行されます。
