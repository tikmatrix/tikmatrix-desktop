# カスタムADB命令

TikMatrix允许您创建和管理カスタムADB命令，以快速実行常用操作。这些命令できます以简化您の工作流程，提高操作效率。

## 什么是カスタムADB命令

カスタムADB命令機能让您できます以保存常用のADB指令，し通じて简单の点击実行它们，而无需每次都手動输入完整命令。

## 訪問カスタム命令界面

1. 在TikMatrix主界面中，找到し点击左侧边栏中の**カスタム命令**选项。
2. カスタム命令界面将显示您済み保存の所有命令，以及添加新命令和重置命令の按钮。

## 预设命令

首次使用时，TikMatrix済み经に您准备一些实用の预设命令：

- **Home**：返回デバイス主屏幕
- **Back**：実行返回操作
- **Enable TCP**：启用TCP/IP接続模式
- **Enable Fast Input**：启用快速输入法
- **Open Gallery**：開くデバイス图库
- **Open Settings**：開くデバイス設定
- **Open NekoBox**：開くNekoBoxアプリ（など済み安装）

## 添加カスタム命令

1. 点击界面顶部の**添加命令**按钮。
2. 在弹出の对话框中，填写以下信息：
   - **命令名称**：に您の命令取一个易于识别の名称
   - **命令参数**：输入ADB命令参数（无需含む"adb"前缀）
3. 点击**添加命令**按钮保存。

## サポート多行命令

カスタム命令サポート多行输入，每行将被视に单独のADB命令，し按顺序実行。
