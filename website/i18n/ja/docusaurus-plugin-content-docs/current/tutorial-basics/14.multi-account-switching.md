# マルチアカウント切り替え

TikTokとInstagramは、同じデバイスで複数のアカウントにログインすることをサポートしています。これは公式アプリのネイティブ機能です。TikMatrixとIgMatrixは、自動化されたアカウント切り替えを提供し、手動介入なしにログインしているすべてのアカウントでタスクを実行できます。

## サポートされているスクリプト

以下のスクリプトは、自動アカウント切り替えをサポートしています：

- **Profile** — アカウントプロフィール情報を表示
- **Account Warmup** — 自然なアクティビティでアカウントをウォームアップ
- **Post** — 投稿を公開
- **Mass DM** — ダイレクトメッセージを一括送信
- **Follow Back** — フォローしてくれたユーザーをフォローバック
- **Unfollow All** — すべてのアカウントのフォローを解除
- **Delete Post** — 公開された投稿を削除
- **Boost Users** — ユーザープロフィールのエンゲージメントを増やす
- **Boost Posts** — 投稿のエンゲージメントを増やす
- **Boost Lives** — ライブストリームのエンゲージメントを増やす
- **Mass Comment** — コメントを一括投稿
- **Boost Comments** — コメントのエンゲージメントを増やす
- **Super Marketing** — 高度なマーケティング自動化

## 初期セットアップ：アカウントマッチング

マルチアカウント切り替え機能を使用する前に、まずデバイスからアカウントをマッチングする必要があります。

1. 左側のナビゲーションで**スクリプト**タブを開きます。
2. **アカウントマッチング**スクリプトを見つけて実行します。
3. このスクリプトは、デバイスでログインしているアカウントをスキャンし、自動的にアカウントリストに追加します。
4. スクリプトが完了したら、**アカウント**ボタンをクリックして、デバイスアカウントが正常に追加されたことを確認します。

## アカウント切り替えのテスト

**アカウントマッチング**を実行した後、アカウント切り替え機能をテストできます：

1. スクリプトタブから**アカウント切り替え**スクリプトを実行します。
2. このスクリプトには他のロジックはありません。アカウント切り替えフローが正しく機能することを検証するだけです。
3. スクリプトが成功すれば、アカウント切り替え機能は使用する準備ができています。

## マルチアカウントモードの使用

初期セットアップとテストを完了したら：

1. サポートされているスクリプトを実行するとき、スクリプト設定で**マルチアカウントモード**を有効にします。
2. システムは、アカウントリスト内の各アカウントに対して独立したタスクを作成します。
3. タスクはキューに入れられ、順次実行されます。システムは自動的にターゲットアカウントに切り替えて自動化を実行します。
4. 手動介入は必要ありません。

## アカウントマッチングを再実行するタイミング

**アカウントマッチング**と**アカウント切り替え**スクリプトは、通常、初期セットアップ時に1回だけ実行する必要があります。ただし、次の場合は**アカウントマッチング**を再実行する必要があります：

- デバイスに新しいアカウントをログインする
- 既存のアカウントをログアウトする
- デバイス上のアカウントが変更された

アカウントリストを更新するには：

1. **アカウント**セクションでアカウントリストを手動でクリアします。
2. **アカウントマッチング**スクリプトを再実行して、デバイス上の現在のアカウントを検出します。

---

マルチアカウント切り替えにより、手動でアカウントを切り替える必要がなくなり、単一のデバイス上のすべてのアカウントで自動化を簡単にスケールできます。
