# クローンアプリのサポート

TikMatrixとIgMatrixは、同じデバイス上で複数のクローンアプリ（GB TikTok、GB Instagram、Instagram Plusなどの改変版）を実行できます。各アプリには独自のパッケージ名があり、異なるアプリインスタンスを使用して複数のアカウントをより効率的に管理できます。

:::danger リスク警告
クローンアプリの使用は、TikTok/Instagramの利用規約に違反する可能性があり、アカウントの停止または禁止につながる可能性があります。この機能については技術サポートのみを提供しており、クローンアプリの使用による禁止、停止、制限を含むアカウント関連の問題については**責任を負いません**。自己責任でご使用ください。
:::

## パッケージ名とは？

すべてのAndroidアプリには**パッケージ名**と呼ばれる一意の識別子があります。公式アプリは標準のパッケージ名を使用しています：

- TikTok: `com.zhiliaoapp.musically`
- Instagram: `com.instagram.android`

クローンアプリは次のような異なるパッケージ名を使用します：

- GB TikTok: `com.zhiliaoapp.musically.gb`
- GB Instagram: `com.instagram.gb.android`
- Instagram Plus: `com.instagram.plus.android`

TikMatrix/IgMatrixは、各アカウントの正しいパッケージ名を自動的に検出して使用できます。

## クローンアプリのセットアップ

### ステップ1：クローンアプリのインストール

1. お好みのクローンアプリをデバイスまたはCloudPhoneにインストールします。
2. 各クローンアプリに競合を避けるための一意のパッケージ名があることを確認してください。
3. 各アプリインスタンスで異なるアカウントにログインします。

### ステップ2：パッケージ名の設定

パッケージ名を設定するには2つのオプションがあります：

#### オプションA：アカウントレベルの設定（推奨）

各アカウントに特定のパッケージ名を設定します：

1. 左側のナビゲーションで**アカウント**を開きます。
2. アカウントをクリックして編集するか、**追加**をクリックして新しいアカウントを作成します。
3. **パッケージ名**フィールドで、次のことができます：
   - クローンアプリのパッケージ名を**手動で入力**する（例：`com.instagram.gb.android`）
   - **パッケージ検出**ボタンをクリックして**自動検出**する（最初にデバイスを選択する必要があります）
4. **保存**をクリックします。

:::tip パッケージ名の自動検出
**パッケージ検出**ボタンは、選択したデバイスで現在開いているアプリを自動的に検出します。これは正しいパッケージ名を取得する最も簡単な方法です：

1. アカウントのデバイスを選択する
2. そのデバイスでクローンアプリを開く
3. **パッケージ検出**ボタンをクリックする
4. パッケージ名が自動的に入力される

この機能は、TikTok/Instagramクローン、App Clonerアプリ、NomixClonerアプリを含む、あらゆるAndroidアプリで動作します。
:::

このアカウントでスクリプトを実行すると、TikMatrix/IgMatrixは指定されたパッケージ名を自動的に使用します。

#### オプションB：システム全体のデフォルト

特定のパッケージ名を持たないすべてのアカウントのデフォルトパッケージ名を設定します：

1. 左側のナビゲーションで**設定**を開きます。
2. **TikTokパッケージ名**または**Instagramパッケージ名**フィールドを見つけます。
3. デフォルトのパッケージ名を入力します。
4. **保存**をクリックします。

:::tip 優先順位
アカウントレベルの設定は常にシステム全体のデフォルトを上書きします。アカウントに独自のパッケージ名が設定されている場合、デフォルトの代わりにそれが使用されます。
:::

## 動作の仕組み

自動化スクリプトを実行すると：

1. TikMatrix/IgMatrixは、ターゲットアカウントにカスタムパッケージ名が設定されているかどうかを確認します。
2. ある場合は、すべてのUI自動化操作にそのパッケージ名を使用します。
3. ない場合は、システム全体のデフォルトパッケージ名にフォールバックします。
4. すべてのリソースIDとXPathクエリは、正しいパッケージ名で自動的にフォーマットされます。

## ベストプラクティス

### 1. 説明的なユーザー名を使用する

異なるクローンアプリに複数のアカウントがある可能性があるため、各アカウントがどのアプリに属しているかを識別するために説明的なユーザー名を使用してください：

```text
username_gb      → GB TikTok/Instagram
username_plus    → TikTok/Instagram Plus
username         → 公式アプリ
```

### 2. グループごとにアカウントを整理する

クローンアプリのタイプに基づいてデバイスグループを作成します：

- **グループA**：すべてのGBアプリを使用するアカウント
- **グループB**：すべてのPlusアプリを使用するアカウント
- **グループC**：すべての公式アプリを使用するアカウント

### 3. 一括操作前にテストする

すべてのアカウントでスクリプトを実行する前に：

1. 最初に単一のアカウントでテストする
2. 正しいアプリが使用されていることを確認する
3. すべての操作が期待どおりに動作することを確認する

### 4. パッケージ名の一貫性を保つ

アカウントにパッケージ名を設定したら、別のクローンアプリに切り替えない限り変更しないでください。一貫性は混乱やエラーを避けるのに役立ちます。

## トラブルシューティング

### スクリプトが間違ったアプリを開く

**問題**：スクリプトがクローンアプリではなく間違ったアプリを開く。

**解決策**：

1. アカウントの**パッケージ名**フィールドが正しく設定されているか確認する
2. クローンアプリがインストールされており、パッケージ名が一致することを確認する
3. **パッケージ検出**ボタンを使用して正しいパッケージ名を自動的に取得する
4. アプリが最近インストールされた場合は、デバイスを再起動する

### UI要素が見つからない

**問題**：スクリプトが「要素が見つかりません」エラーを報告する。

**解決策**：

1. 異なるクローンアプリは、わずかに異なるUIレイアウトを持つ場合がある
2. クローンアプリのバージョンが互換性があるか確認する
3. TikMatrix/IgMatrixの最新バージョンに更新してみる
4. 問題が続く場合はサポートに連絡する

### 同じアプリ内の複数アカウント

**問題**：同じアプリインスタンス内で複数のアカウントを切り替えたい。

**解決策**：

- TikTok/Instagramは複数のログインアカウントをネイティブにサポート
- TikMatrix/IgMatrixの**アカウント切り替え**機能を使用する
- そのアプリインスタンス内のすべてのアカウントに同じパッケージ名を設定する
- TikMatrix/IgMatrixがアカウント切り替えを自動的に処理する

### パッケージ検出ボタンが機能しない

**問題**：**パッケージ検出**ボタンが機能しないか、エラーを表示する。

**解決策**：

1. 最初にデバイスを選択したことを確認する
2. アプリが現在デバイスで開いていることを確認する
3. デバイス接続が安定していることを確認する
4. アプリを再度開いて検出をクリックしてみる

## パッケージ名の検索

デバイスのパッケージ名を手動で確認したい場合：

### ADBコマンドを使用する

```bash
# すべてのTikTok関連パッケージをリスト
adb shell pm list packages | grep musically

# すべてのInstagram関連パッケージをリスト
adb shell pm list packages | grep instagram
```

### 自動検出機能を使用する

最も簡単な方法は、組み込みの**パッケージ検出**ボタンを使用することです：

1. アカウント編集ダイアログを開く
2. デバイスを選択する
3. デバイスでクローンアプリを開く
4. **パッケージ検出**をクリックする
5. パッケージ名が自動的に検出されて入力される

この方法はあらゆるAndroidアプリで動作し、推奨されるアプローチです。

## FAQ

**Q：同じデバイスで複数のクローンアプリを使用できますか？**

A：はい、各クローンアプリに一意のパッケージ名がある限り、同じデバイスで複数のアプリインスタンスを実行できます。

**Q：TikMatrix/IgMatrixを再インストールする必要がありますか？**

A：いいえ、この機能は最新バージョンに組み込まれています。最新バージョンに更新してパッケージ名を設定するだけです。

**Q：既存の自動化スクリプトはまだ動作しますか？**

A：はい、既存のスクリプトはすべて完全に互換性があります。カスタムパッケージ名を持たないアカウントは、デフォルトのパッケージ名を使用します。

**Q：TikTokとInstagramの両方で動作しますか？**

A：はい、TikMatrixとIgMatrixの両方がクローンアプリの検出と自動化を完全にサポートしています。

**Q：自動検出機能はどのように動作しますか？**

A：**パッケージ検出**をクリックすると、TikMatrix/IgMatrixは選択したデバイスで現在フォアグラウンドにあるアプリを確認し、そのパッケージ名を自動的に取得します。

**Q：App ClonerやNomixClonerのアプリを使用できますか？**

A：はい、パッケージ名機能は、App ClonerやNomixClonerを含む、あらゆるAndroidアプリクローニングツールで動作します。自動検出機能を使用して、正しいパッケージ名を簡単に取得できます。
