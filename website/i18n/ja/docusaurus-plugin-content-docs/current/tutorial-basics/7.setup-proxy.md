# プロキシを設定

各携帯電話に固有のプロキシIPアドレスを設定できます。

> **免責事項:** ここで推奨されるすべてのウェブサイトはインターネットから入手したものであり、本サイトとは関係ありません。使用前に有効性を確認してください。保証は提供しません。

## 1. レジデンシャルVPSプロキシを設定する（プロキシIPを購入した場合は、この手順をスキップできます）

1. ターゲット国からレジデンシャルプロキシIP VPSを購入します（ネイティブIPとデュアルISP VPSが望ましい）。
2. [https://github.com/MHSanaei/3x-ui](https://github.com/MHSanaei/3x-ui)を使用して、VPS上でVPNを素早く設定します：

    ```shell
    bash <(curl -Ls https://raw.githubusercontent.com/MHSanaei/3x-ui/refs/tags/v2.6.0/install.sh)
    # プロンプトに従ってVPN設定を完了します。
    ```

3. インストール後、x-ui Webインターフェースにアクセスします。URLとログイン認証情報は、インストール後のターミナル出力で確認できます。
4. x-uiで新しいユーザーを作成し、プロトコルを`SOCKS5`に設定し、必要に応じてポートとその他の設定を構成します。
5. ユーザーを作成すると、x-uiインターフェースにプロキシIPとポートが表示されます。
6. 携帯電話のNekoBoxアプリでプロキシIPとポートを使用します。

> 推奨VPSプロバイダー

- [ここをクリック](/advertising)

## 2. レジデンシャルプロキシIPを購入する（VPSを設定した場合は、この手順をスキップできます）

VPSを設定せずに、レジデンシャルプロキシIPを直接購入することもできます。

> 推奨プロキシプロバイダー

- [ここをクリック](/advertising)

> IP確認ツール

- [ipinfo](https://ipinfo.io) — IPがレジデンシャルかどうかを確認。
- [scamalytics](https://scamalytics.com/ip) — IPのリスクレベルを確認。

## 3. 携帯電話でプロキシを有効にする

1. APKをダウンロードして、携帯電話に[NekoBoxForAndroid](https://github.com/MatsuriDayo/NekoBoxForAndroid)クライアントをインストールします。
2. NekoBoxアプリを開き、`+`ボタンをクリックして新しいプロキシを追加します。
3. `SOCKS5`または他のプロキシタイプを選択します。
4. プロキシIPとポートを入力し、`保存`をクリックします。
5. `開始`ボタンをクリックしてプロキシを有効にします。
6. プロキシが有効になったら、NekoBoxアプリで接続ステータスを確認できます。

### 中国ユーザー向け

中国にいる場合、`SOCKS5`プロトコルはうまく機能しない可能性があります。フロントエンドプロキシとして暗号化プロキシ（`VLESS`）を追加する必要があります。次のツールを使用できます：

1. NekoBoxアプリに`VLESS`プロキシを追加します。
2. NekoBoxアプリに`SOCKS5`プロキシを追加します。
3. NekoBoxアプリに`Chain`プロキシを追加します。フロントエンドプロキシとして`VLESS`プロキシを選択し、バックエンドプロキシとして`SOCKS5`プロキシを選択します。
