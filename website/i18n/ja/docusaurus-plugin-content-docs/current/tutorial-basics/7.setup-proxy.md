# 設定代理

您できます以に每部携帯電話設定独立の代理IP地址。

> **免责声明：** 本页面推荐の所有网站均来源于互联网，と本站无关。使用前请自行認証其有效性。我们不提供任何保证。

## 1. 設定住宅VPS代理（など果您购买代理IP，できます以跳过此步骤）

1. 从ターゲット国家购买住宅代理IP VPS（最好是原生IP和双ISP VPS）。
2. 使用 [https://github.com/MHSanaei/3x-ui](https://github.com/MHSanaei/3x-ui) 在VPS上で快速設定VPN：

    ```shell
    bash <(curl -Ls https://raw.githubusercontent.com/MHSanaei/3x-ui/refs/tags/v2.6.0/install.sh)
    # 按照提示完成VPN设置。
    ```

3. 安装完成后，訪問x-ui Web界面。您できます以在安装后の终端输出中找到URL和ログイン凭据。
4. 在x-ui中创建新用户，将协议設定に`SOCKS5`，しに基づいて需要配置端口和其他設定。
5. 创建用户后，您将在x-ui界面中看到代理IP和端口。
6. 在携帯電話上でのNekoBoxアプリ中使用代理IP和端口。

## 3. 在携帯電話上で启用代理

1. 在携帯電話上で下载APKし安装 [NekoBoxForAndroid](https://github.com/MatsuriDayo/NekoBoxForAndroid) 客户端。
2. 開くNekoBoxアプリ，点击`+`按钮添加新代理。
3. 选择`SOCKS5`または其他代理类型。
4. 输入代理IP和端口，然后点击`保存`。
5. 点击`开始`按钮启用代理。
6. 启用代理后，您できます以在NekoBoxアプリ中检查接続状态。

### 中国用户注意事项

など果您在中国，`SOCKS5`协议できます能无法正常工作。您需要添加加密代理（`VLESS`）作に前端代理。您できます以使用以下工具：

1. 在NekoBoxアプリ中添加`VLESS`代理。
2. 在NekoBoxアプリ中添加`SOCKS5`代理。
3. 在NekoBoxアプリ中添加`链式`代理。选择`VLESS`代理作に前端代理，`SOCKS5`代理作に后端代理。
