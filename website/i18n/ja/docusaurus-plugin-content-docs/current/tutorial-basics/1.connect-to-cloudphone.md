# クラウドフォンに接続

このガイドでは、TikMatrixを使用してGeeLarkクラウドフォンに接続する方法を説明します。

:::warning ネットワーク推奨事項
コンピュータとクラウドフォンのデータセンターが十分で安定したネットワーク帯域幅を持っていることを確認してください。自動化タスクの最高の信頼性を得るために、コンピュータとクラウドフォンのデータセンターが同じ国または地域にあることを推奨します。これにより、遅延とパケットロスを減らすことができます。
:::

クラウドフォンは[GeeLark](https://www.geelark.cn?invite_code=XHY6a8)から購入できます。

## ADBを有効にする

プロファイルセクションから目的のクラウドフォンプロファイルでADBを有効にします。

![ADBを有効にする](../img/enable-adb.webp)

複数のプロファイルで一度にADBを有効にすることもできます。
![複数のプロファイルでADBを有効にする](../img/enable-adb-multiple.webp)

## プロファイルを開始

プロファイルを開いた後、🔗アイコンをクリックして、クラウドフォンのIPアドレス、ポート、接続コードを取得します。
![IPアドレスを取得](../img/get-ip-address.webp)

## 方法1: クラウドフォンリストで素早く接続（推奨）

TikMatrixは現在、クラウドフォン接続情報の保存をサポートしており、ワンクリックで素早く接続できます。

### クラウドフォン情報を保存

1. TikMatrixを開き、`TCPデバイスをスキャン`ボタンをクリックします。
2. `クラウドフォンリスト`タブに切り替えます。
3. `追加`ボタンをクリックします。
4. クラウドフォン情報を入力します：
   - **名前**: クラウドフォンを識別するためのわかりやすい名前（例：「GeeLark Phone 1」）
   - **IPアドレス**: GeeLarkから取得したIPアドレス（例：`1.2.3.4`）
   - **ポート**: ポート番号（デフォルト：`5555`）
   - **認証コード**: GeeLarkから取得した接続コード（例：`f850ef`）
5. `保存`をクリックします。

### クラウドフォンに接続

保存後、2つの方法で接続できます：

**単一接続：**

- リスト内の任意のクラウドフォンの隣にある🔌ボタンをクリックします。

**バッチ接続：**

- チェックボックスを使用して複数のクラウドフォンを選択します。
- 上部の`接続 (X)`ボタンをクリックします。

システムは自動的に以下を実行します：

1. ADB経由で接続
2. glogin認証コマンドを実行
3. 接続結果を表示

接続が成功したら、デバイスリストを更新してクラウドフォンを確認します。

## 方法2: ADBターミナル経由で手動接続

手動接続を希望する場合、またはトラブルシューティングが必要な場合：

### ADBコマンドを実行

1. TikMatrixで`設定` > `ADBターミナルを開く`をクリックします。
2. オペレーティングシステムに基づいて接続コマンドを実行し、携帯電話がステータス`device`でリストに表示されているか確認します。
3. 認証コマンドを実行して接続を認証します。

### Windowsシステムの場合

```shell
.\adb.exe connect <IP_ADDRESS>:<PORT>
.\adb.exe -s <IP_ADDRESS>:<PORT> shell glogin <CONNECTION_CODE>
```

### Mac/Linuxシステムの場合

```shell
./adb connect <IP_ADDRESS>:<PORT>
./adb -s <IP_ADDRESS>:<PORT> shell glogin <CONNECTION_CODE>
```

> 例えば、IPアドレスが`1.2.3.4`、ポートが`5555`、接続コードが`f850ef`の場合
>
> **Windows:**
>
> ```shell
> .\adb.exe connect 1.2.3.4:5555
> .\adb.exe -s 1.2.3.4:5555 shell glogin f850ef
> ```
>
> **Mac/Linux:**
>
> ```shell
> ./adb connect 1.2.3.4:5555
> ./adb -s 1.2.3.4:5555 shell glogin f850ef
> ```

## ヒント

- すべてのクラウドフォンをリストに保存して、素早くアクセスできるようにします。
- 接続ステータスは各クラウドフォンの隣に表示されます。
- 保存したクラウドフォンはいつでも編集または削除できます。
- 接続情報はコンピュータにローカルに保存されます。

## 最終ステップ

TikMatrixを開き、クラウドフォンがデバイスリストに表示されるまで待ちます。すぐに表示されない場合は、デバイスリストを更新してみてください。

おめでとうございます！これでクラウドフォンに接続され、TikMatrixで他のAndroidデバイスと同様に管理できます。
