# 接続云携帯電話

本指南将帮助你使用TikMatrix接続到GeeLark云携帯電話。

:::warning 网络提示
请尽量保证你のコンピュータと云携帯電話所在机房之间の网络带宽充足且稳定。に得る最適なパフォーマンス，建议将コンピュータと云携帯電話机房放在同じ国家または同じ地区，以降低延迟和丢包，从而保证自動化タスク稳定できます靠运行。
:::

你できます以从[GeeLark](https://www.geelark.cn?invite_code=XHY6a8)购买云携帯電話。

## 启用ADB

在配置文件一部に所需の云携帯電話配置文件启用ADB。

![启用ADB](../img/enable-adb.webp)

你也できます以同時にに複数配置文件启用ADB。
![に複数配置文件启用ADB](../img/enable-adb-multiple.webp)

## 启动配置文件

開く配置文件后，点击🔗图标获取云携帯電話のIP地址、端口和接続代码。
![获取IP地址](../img/get-ip-address.webp)

## 方法一：使用云携帯電話列表快速接続（推荐）

TikMatrix现在サポート保存云携帯電話接続信息，实现ワンクリック快速接続。

### 保存云携帯電話信息

1. 開くTikMatrix，点击`扫描TCPデバイス`按钮。
2. 切换到`云携帯電話列表`标签页。
3. 点击`添加`按钮。
4. 填写云携帯電話信息：
   - **名称**：に云携帯電話起一个便于识别の名称（例など："GeeLark携帯電話1"）
   - **IP地址**：从GeeLark获取のIP地址（例など：`1.2.3.4`）
   - **端口**：端口号（默认：`5555`）
   - **接続码**：从GeeLark获取の接続代码（例など：`f850ef`）
5. 点击`保存`。

### ワンクリック接続

保存后，你できます以通じて以下两种方式接続：

**单个接続：**

- 点击云携帯電話列表中任意云携帯電話旁边の🔌按钮。

**一括接続：**

- 使用复选框选择複数云携帯電話。
- 点击顶部の`接続 (X)`按钮。

系统会自動完成以下操作：

1. 通じてADB接続
2. 実行glogin认证命令
3. 显示接続结果

接続成功后，刷新デバイス列表即できます看到你の云携帯電話。

## 方法二：通じてADB终端手動接続

など果你更喜欢手動接続または需要排查问题：

### 运行ADB命令

1. 在TikMatrix中点击`設定` > `開くADB终端`。
2. に基づいて你の操作系统运行相应の接続命令，し检查你の携帯電話是否出现在列表中，状态に`device`。
3. 运行认证命令来認証接続。

### Windows系统

```shell
.\adb.exe connect <IP_ADDRESS>:<PORT>
.\adb.exe -s <IP_ADDRESS>:<PORT> shell glogin <CONNECTION_CODE>
```

### Mac/Linux系统

```shell
./adb connect <IP_ADDRESS>:<PORT>
./adb -s <IP_ADDRESS>:<PORT> shell glogin <CONNECTION_CODE>
```

> 例など，など果你のIP地址是`1.2.3.4`，端口是`5555`，接続代码是`f850ef`
>
> **Windows:**
>
> ```shell
> .\adb.exe connect 1.2.3.4:5555
> .\adb.exe -s 1.2.3.4:5555 shell glogin f850ef
> ```
>
> **Mac/Linux:**
>
> ```shell
> ./adb connect 1.2.3.4:5555
> ./adb -s 1.2.3.4:5555 shell glogin f850ef
> ```

## 提示

- 将所有云携帯電話保存到列表中以便快速訪問。
- 每个云携帯電話旁边会显示接続状态。
- 你できます以随时编辑または删除済み保存の云携帯電話。
- 接続信息存储在你の本地コンピュータ上で。
