# TikTokをインストール

以下の特定のTikTokバージョンをダウンロードすることをお勧めします。新しいバージョンは互換性がない場合があります。サポートされているバージョンは、更新後にグループで発表されます。

## TikTok APKをダウンロード

- グローバル版：[com.zhiliaoapp.musically](https://apkpure.com/tiktok-musically-2024/com.zhiliaoapp.musically)
- アジア版：[com.ss.android.ugc.trill](https://apkpure.com/tiktok/com.ss.android.ugc.trill)

## インストール手順

1. 上記のリンクからAPKをダウンロードします。
2. TikMatrixですべてのデバイスを選択します。
3. `一般` > `APKをインストール`をクリックし、ダウンロードしたAPKファイルを選択します。
4. インストールが完了するまで待ちます（モデルによってデバイスあたり1〜3分かかる場合があります。繰り返しクリックしないでください）。
5. インストール後、`設定`をクリックし、TikTokバージョンがインストールしたアプリと一致することを確認します。
6. `TikTokを開く`をクリックして、アプリが正しくインストールされているか確認します。
7. TikTokの言語が英語に設定されていることを確認してください。そうでない場合、スクリプトが正しく動作しない可能性があります。必要に応じてTikTok設定で言語を変更してください。
