# 高速入力キーボードを有効にする

携帯電話が接続されると、TikMatrixは自動的に高速入力キーボードに切り替わります。

## 高速入力キーボードを使用する理由

高速入力キーボードはTikMatrixに不可欠で、自動多言語サポートを提供します。これがないと、コメント、メッセージング、タイトルの公開などのテキスト入力機能が正しく動作しない場合があります。

## 高速入力キーボードを手動で有効にする方法

1. 使用するデバイスを選択します。
2. `ADBコマンド` > `高速入力を有効にする`をクリックします。

## テキストを入力する方法

TikMatrixでは、複数の方法でテキストを入力できます：

### 方法1: 直接キーボード入力 ⌨️（新機能）

⭐ **キーボード入力制御** - コンピュータのキーボードを使用して携帯電話に直接入力できるようになり、テキスト入力がより速く便利になりました。

1. 使用するデバイスを選択します。
2. 携帯電話の画面上の任意のテキストフィールドをクリックします。
3. コンピュータのキーボードで直接入力を開始すると、テキストが携帯電話にリアルタイムで表示されます。

### 方法2: 入力パネル

1. 携帯電話の画面をクリックして展開し、右側のツールバーの`入力`ボタンをクリックします。
2. 入力ボックスにテキストを入力し、送信をクリックします。

## 携帯電話とコンピュータ間でコピー＆ペースト

### 携帯電話のクリップボードをコンピュータにコピー

1. 携帯電話を選択し、`Ctrl+C`を押したままにすると、携帯電話のクリップボードがコンピュータにコピーされます。

### コンピュータのクリップボードを携帯電話にペースト

1. デバイスを選択し、`Ctrl+V`を押したままにすると、コンピュータのクリップボードが携帯電話の入力ボックスに貼り付けられます。
