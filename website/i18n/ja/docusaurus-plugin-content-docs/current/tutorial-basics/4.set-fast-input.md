# 启用快速输入キーボード

携帯電話接続后会自動切换到快速输入キーボード。

## に什么要設定快速输入キーボード

快速输入キーボード是 TikMatrix の必要工具，提供自動多言語サポート。没有它，TikMatrix の文本输入機能（などコメント、消息和发布タイトル）できます能无法正常工作。

## など何手動启用快速输入キーボード

1. 选择需要使用快速输入キーボードのデバイス。
2. 点击 `カスタムADB命令` - `启用快速输入`。

## など何输入文字

在 TikMatrix 中，您できます以通じて多种方式输入文字：

### 方式一：直接キーボード输入 ⌨️（新機能）

⭐ **キーボード输入制御** - 您现在できます以直接使用コンピュータキーボード在携帯電話上で输入文字，让文字输入更加快速便捷。

1. 选择需要使用のデバイス。
2. 点击携帯電話屏幕上での任何文字输入框。
3. 直接在コンピュータキーボード上で开始输入 - 文字将实时显示在携帯電話上で。

### 方式二：输入面板

1. 点击携帯電話屏幕，展开大屏, し点击右侧工具栏中の`输入`按钮。
2. 在输入框中输入文字，点击发送。

## など何在携帯電話和コンピュータ间复制粘贴文字

### 复制携帯電話剪切板文字到コンピュータ剪切板

1. 选择需要复制の携帯電話, 按住 `Ctrl+C` 键, 复制成功

### 粘贴コンピュータ剪切板文字到携帯電話输入框

1. 选择需要粘贴のデバイス, 按住 `Ctrl+V` 键, 粘贴成功
