# 初始化デバイス

:::warning
本アプリ通じて Android 官方 ADB 调试通道安装，用于デバイス自動化制御。某些デバイスできます能会提示是否将未知アプリ样本发送给 Google，这是 Android 系统の安全提示，とアプリ安全性无关，不会影响正常使用。点击“Don't send”即できます完成安装。
:::

TikMatrix APK是自動化タスク所需の移动代理。**当您首次将携帯電話接続到TikMatrix时，系统会自動初始化您のデバイスし安装TikMatrixアプリ** - 无需手動操作。

在自動初始化过程中，TikMatrix会安装两个APK文件：TikMatrix和TikMatrix Test（无界面）。など果您の携帯電話提示安装授权，只需点击"允许"（在小米デバイス上で尤其重要）。

## 自動安装 vs 手動安装

**自動安装（推荐）**：默认情况下，TikMatrix在首次接続デバイス时会自動初始化デバイス。这是最简单便捷の方法。

**手動安装/更新**：仅在特殊情况下または需要更新时使用。

## 自動初始化流程

当您接続新デバイス到TikMatrix时：

1. **デバイス检测**：TikMatrix自動检测済み接続のデバイス。
2. **自動安装**：系统会自動安装TikMatrixアプリ，无需用户干预。
3. **授权确认**：など果デバイス提示安装权限，只需点击"允许"。
4. **完成安装**：初始化过程大约在10秒内完成。
5. **できます以使用**：您のデバイス现在済み准备好実行自動化タスク。

这个自動化流程在大多数情况下无需手動設定。

## 手動安装/更新

要手動安装または更新TikMatrix APK，请在設定中关闭`自動唤醒Agent`开关。然后按照以下步骤操作：

1. 选择你の携帯電話し点击`通用` > `初始化Agent`。
2. TikMatrix会自動在携帯電話上で安装两个APK文件。
3. など果提示，点击"允许"授权安装。
4. など果被Google Play保护阻止，点击"仍然安装"继续。
5. など待安装完成（大约10秒钟）。
6. 開く`自動唤醒Agent`开关以在携帯電話接続时启用自動アプリ唤醒。
7. 点击`通用` > `開くAgent`测试安装。など果アプリ无法启动，请联系技术サポート。
