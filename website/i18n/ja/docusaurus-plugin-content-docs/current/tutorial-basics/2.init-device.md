# デバイスを初期化

:::warning
このアプリはAndroidの公式ADBデバッグチャネル経由でインストールされ、デバイスの自動化制御に使用されます。一部のデバイスでは、未知のアプリサンプルをGoogleに送信するかどうかを尋ねられることがありますが、これはAndroidシステムのセキュリティプロンプトであり、アプリの安全性とは関係なく、通常の使用には影響しません。「送信しない」をクリックしてインストールを完了してください。
:::

TikMatrix APKは、自動化タスクに必要なモバイルエージェントです。**初めて携帯電話をTikMatrixに接続すると、システムは自動的にデバイスを初期化し、TikMatrixアプリをインストールします** - 手動操作は不要です。

自動初期化プロセス中、TikMatrixは2つのAPKファイル（TikMatrixとTikMatrix Test（インターフェースなし））をインストールします。携帯電話でインストール許可を求められた場合は、「許可」をタップするだけです（特にXiaomiデバイスで重要）。

## 自動インストール vs 手動インストール

**自動インストール（推奨）**: デフォルトでは、TikMatrixは初回接続時にデバイスを自動的に初期化します。これが最も簡単で便利な方法です。

**手動インストール/アップデート**: 特別な場合やアップデートが必要な場合にのみ必要です。

## 自動初期化プロセス

新しいデバイスをTikMatrixに接続すると：

1. **デバイス検出**: TikMatrixは接続されたデバイスを自動的に検出します。
2. **自動インストール**: システムはユーザーの介入なしにTikMatrixアプリを自動的にインストールします。
3. **承認**: デバイスでインストール許可を求められた場合は、「許可」をタップします。
4. **完了**: 初期化プロセスは約10秒で完了します。
5. **使用準備完了**: デバイスは自動化タスクの実行準備が整いました。

この自動プロセスにより、ほとんどの場合、手動セットアップは不要です。

## 手動インストール/アップデート

TikMatrix APKを手動でインストールまたはアップデートするには、設定で`自動エージェント起動`スイッチをオフにします。その後、以下の手順に従います：

1. 携帯電話を選択し、`一般` > `エージェント初期化`をクリックします。
2. TikMatrixは自動的に両方のAPKファイルを携帯電話にインストールします。
3. プロンプトが表示されたら、「許可」をタップしてインストールを承認します。
4. Google Play プロテクトによってブロックされた場合は、「とにかくインストール」をタップして続行します。
5. インストールが完了するまで待ちます（約10秒）。
6. `自動エージェント起動`スイッチをオンにして、携帯電話接続時のアプリ自動起動を有効にします。
7. `一般` > `エージェントを開く`をクリックしてインストールをテストします。アプリが起動しない場合は、テクニカルサポートにお問い合わせください。
