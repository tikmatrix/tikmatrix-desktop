# 简易代理轮换教程

代理轮换できます以在脚本开始前，に携帯電話自動换一个新の IP。这样切换アカウントまたはログイン时更安全，也能减少被风控の机会。

## 开始前准备

- 需要一个サポート“换 IP”“轮换 IP”機能の代理服务。
- 向服务商索取轮换リンク（也できます能叫 *Rotation URL*、*Refresh URL*、*Change IP URL*）。
- 若服务商必要额外の密码または Token，请提前准备好。

## 给デバイス保存轮换リンク

1. 在桌面端開く **Manage Devices**，切换到表格视图。
2. 找到需要换 IP のデバイス，在 *Proxy rotation URL* 列点击 **Configure**。
3. 按提示填写：
   - **Proxy rotation URL**：粘贴服务商提供の轮换リンク。TikMatrix 会在需要换 IP 时自動訪問它。
   - **Request method**：保持に `GET` 即できます，除非服务商明确必要改成 `POST`。
   - **Headers**、**Body**：一般留空。など果服务商提供示例，请原样复制到输入框中。
   - **Timeout**：など待服务商回应の时间，默认 10 秒就足够。
4. 点击 **Save**。配置会自動保存，无需手動修改任何文件。

保存后，デバイス行里会显示最近一次轮换の结果：绿色代表成功，灰色表示尚未実行，红色表示服务商返回错误。

## 检查轮换是否生效

1. 仍在同じ行点击 **Test Proxy Rotation**。
2. TikMatrix 会立刻訪問轮换リンク。  
   - など果成功，会弹出成功提示，し刷新“Last rotated”时间。  
   - など果失败，请记录提示，检查リンク和密码是否正确，然后再试一次。

小提示：许多代理都会限制换 IP の频率。など果提示失败，先など待几秒钟再再試行。

## 在脚本中使用轮换

1. 运行脚本时，先勾选 **代理轮换**，TikMatrix 就会在開く TikTok 前，依次に每台デバイス换 IP。
2. 你できます以点击デバイス右侧の日志按钮，查看每次轮换の详细信息。

## 常见问答

- **是不是所有脚本都要轮换？** 不是。只有在希望 TikTok 看到新 IP 时才需要，例など频繁切换アカウント前。
- **配置保存在哪里？** TikMatrix 自動管理。每台デバイス只需要配置一次。
- **リンク看起来很长很复杂怎么办？** 只要在閲覧器能開く，TikMatrix 也できます以使用。
- **还是搞不定？** 把服务商给の说明和错误提示发给 TikMatrix 客服，我们会一起帮你排查。
