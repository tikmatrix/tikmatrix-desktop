# Pencocokan Akun

Skrip pencocokan akun digunakan untuk mengenali akun dari ponsel dan menambahkannya ke daftar akun.

## Langkah-langkah

1. Pilih perangkat yang akan mencocokkan akun.
2. Klik tombol `Skrip` - `Pencocokan Akun`.
3. Klik tombol `Mulai` untuk memulai tugas pencocokan akun.
4. Skrip akan secara otomatis mengenali akun dari ponsel dan menambahkannya ke daftar akun.

## Catatan

* Skrip pencocokan akun tidak 100% berhasil, jika beberapa akun tidak teridentifikasi, Anda mungkin perlu menjalankan skrip beberapa kali.
