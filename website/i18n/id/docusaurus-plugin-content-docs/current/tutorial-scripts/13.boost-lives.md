# Boost Live

Skrip boost live memungkinkan Anda mengontrol akun farm untuk menonton dan secara opsional memberikan like atau komentar pada live target. Anda dapat mengatur durasi menonton dan interval like/komentar untuk menyesuaikan perilaku boost.

## Langkah-langkah

1. Pilih perangkat yang akan menjalankan skrip.
2. Klik `Skrip` > `Boost Live`.
3. Konfigurasi pengaturan tugas:
    - **Username**: Masukkan username target, satu per baris.
    - **Metode Masuk**: Pilih metode **Cari** atau **Langsung**.
    - **Opsi Boost**: Aktifkan like dan/atau komentar.
    - **Gabung Fanclub**: Aktifkan fitur bergabung dengan fanclub streamer (jika tersedia).
    - **Check-in Harian**: Aktifkan aktivitas check-in harian selama live.
    - **Interval Like**: Atur interval antara setiap operasi like.
    - **Jumlah Klik Like**: Atur jumlah klik pada tombol like.
    - **Interval Komentar**: Atur interval antara setiap operasi komentar.
    - **Konten Komentar**: Masukkan satu komentar per baris.
    - **Sisipkan Emoji**: Pilih apakah akan menyisipkan emoji dalam komentar.
    - **Urutan Komentar**: Pilih pengaturan acak atau berurutan untuk komentar.
    - **Jumlah Komentar Per Akun**: Atur jumlah komentar yang dipublikasikan oleh setiap akun.
    - **Durasi Menonton**: Atur waktu tinggal di live.
    - **Interval Tugas**: Atur interval antara setiap tugas.
4. Klik `Mulai Skrip` untuk memulai eksekusi.

## Catatan

- Skrip boost live mungkin tidak selalu berhasil; silakan coba lagi tugas yang gagal jika diperlukan.
