# Login

Skrip login digunakan untuk masuk ke akun media sosial.

## Langkah-langkah

1. Pastikan akun telah ditambahkan ke daftar akun (klik `Akun` untuk menambahkan akun), dan pilih nomor urut ponsel yang perlu login ke akun tersebut.
2. Pilih satu atau beberapa perangkat yang akan menjalankan skrip login.
3. Klik `Skrip` > `Login`.
4. Klik `Mulai Skrip` untuk memulai eksekusi.

## Prioritas Login

Sistem menggunakan prioritas login berikut:

1. **Email+Password**: Jika akun dikonfigurasi dengan email, sistem akan menggunakan email terlebih dahulu untuk login.
2. **Username+Password**: Jika tidak ada email yang dikonfigurasi, sistem akan menggunakan username untuk login.

## Dukungan 2FA (Khusus IgMatrix)

IgMatrix mendukung autentikasi dua faktor (2FA) otomatis untuk login:

### Format Password 2FA

Untuk menggunakan 2FA, atur password dengan format berikut:

```plaintext
password_anda:kunci_2FA_anda
```

**Contoh:**

```plaintext
mypassword123:JBSWY3DPEHPK3PXP
```

### Cara Kerja 2FA

1. **Parsing Password**: Sistem secara otomatis mendeteksi apakah password mengandung pemisah titik dua (`:`).
2. **Generasi Kode Verifikasi**: Ketika memerlukan verifikasi 2FA, sistem menggunakan kunci Anda untuk meminta kode verifikasi TOTP dari layanan 2FA kami.
3. **Input Otomatis**: Kode verifikasi 6 digit yang dihasilkan akan secara otomatis dimasukkan ke kotak verifikasi 2FA Instagram.
4. **Login Lancar**: Seluruh proses 2FA selesai secara otomatis tanpa intervensi manual.

### Penjelasan Kunci 2FA

Kunci 2FA biasanya adalah string 32 karakter yang dikodekan Base32, yang Anda dapatkan saat mengatur 2FA di Instagram. Ini adalah kunci yang sama yang digunakan oleh aplikasi autentikator seperti Google Authenticator atau Authy.

### Dapatkan Akun Instagram dengan Dukungan 2FA

Anda dapat membeli akun Instagram dengan dukungan 2FA dari situs web berikut: [AccsMarket](https://accsmarket.com/en/?ref=802647)

## Catatan

- Login TikTok tidak 100% otomatis. Anda mungkin perlu memasukkan kode verifikasi email secara manual atau menyelesaikan langkah verifikasi lainnya. Kami juga tidak dapat menyelesaikan tantangan login TikTok (seperti captcha gambar, verifikasi manusia, dll.) serta pesan error: terlalu sering login.
- Instagram dapat mencapai login 100% otomatis melalui login 2FA. Untuk akun Instagram dengan 2FA yang diaktifkan, pastikan menggunakan format kunci yang benar dalam password.
