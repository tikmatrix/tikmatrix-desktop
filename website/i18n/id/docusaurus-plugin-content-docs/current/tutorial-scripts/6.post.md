# Posting

Skrip posting digunakan untuk memposting video atau gambar di TikTok.

## Langkah-langkah

1. **Siapkan Materi**:
   - Opsi 1: Klik tombol `Perpustakaan Materi` untuk mengunggah video/gambar ke perpustakaan materi.
   - Opsi 2: Gunakan folder lokal sebagai sumber materi, atur path folder dalam konfigurasi skrip.
2. Pilih perangkat yang akan menjalankan skrip.
3. Klik `Skrip` > `Posting`.
4. Konfigurasi pengaturan tugas:
    - **Sumber Materi**: Pilih perpustakaan materi atau folder lokal.
    - **Path Folder Lokal**: Jika menggunakan folder lokal, tentukan path yang berisi video/gambar.
    - **Metode Posting**: Pilih metode posting: berbagi sistem atau tombol tambah (`+`) atau melalui pencarian menggunakan suara.
    - **Nama Suara**: Saat menggunakan metode posting "melalui pencarian menggunakan suara", Anda dapat memilih:
      - Masukkan nama suara untuk pencarian (misalnya: "original sound", "trending music")
      - Masukkan langsung link suara TikTok (misalnya: `https://www.tiktok.com/t/ZP8hvBwCb/`)
    - **Jenis Konten**: Pilih video atau gambar.
    - **Toggle Tambah Suara**: Aktifkan/nonaktifkan penambahan suara.
    - **Waktu Tunggu Upload**: Atur waktu tunda setelah upload (5-60 detik), pastikan server TikTok sepenuhnya memproses konten sebelum melanjutkan operasi.
    - **Toggle Tambah Link Produk**: Aktifkan/nonaktifkan penambahan link produk.
    - **Format Caption**: Pilih salah satu dari dua format caption:
      - **Mode Multiline**: Setiap baris adalah caption independen, sistem menggunakan secara berurutan dan berputar.
      - **Mode Spintax**: Gunakan sintaks `{opsi1|opsi2}` untuk menghasilkan teks acak.
    - **Caption**: Masukkan caption sesuai format yang dipilih. Mendukung mention pengguna (seperti @TikMatrixPhoneFarm) dan hashtag (seperti #tag1 #tag2 #tag3). Penjelasan format detail di bawah.
    - **Tag Materi**: Filter materi berdasarkan tag. Jika tidak diatur, gunakan semua materi.
5. Klik `Mulai Skrip` untuk memulai eksekusi.

## Penjelasan Format Caption

Sistem mendukung dua format caption, memberikan Anda kontrol fleksibel atas judul postingan.

### Mode Multiline (Default)

**Cara Kerja:**
- Setiap baris mewakili satu caption independen
- Sistem menggunakan caption secara berurutan dan berputar
- Dalam satu baris gunakan `###` untuk menambahkan baris baru

**Contoh:**

```
Caption pertama###mendukung baris baru
Caption kedua###juga mendukung baris baru
Caption ketiga tanpa baris baru
```

**Perbedaan Platform:**
- **TikTok**: Gunakan `##` untuk memisahkan judul dan deskripsi (contoh: `Judul##Deskripsi panjang`)
- **Instagram**: Tidak mendukung `##`, akan menampilkan peringatan

### Mode Spintax

**Cara Kerja:**
- Hanya satu template caption
- Gunakan sintaks `{opsi1|opsi2|opsi3}` untuk mengimplementasikan variasi acak
- Gunakan **baris baru native** (tekan enter) untuk mengimplementasikan caption multi-baris
- Jangan gunakan `###` - tidak akan diparse

**Contoh:**

```
Cuaca {hari ini|besok|lusa} sangat {baik|bagus|cerah}
Silakan ikuti akun saya##Ingin tahu lebih lanjut klik link
```

**Perbedaan Platform:**
- **TikTok**: `##` dapat digunakan untuk memisahkan judul dan deskripsi
- **Instagram**: Tidak mendukung `##`

**Catatan:** Dalam mode Spintax, jika `###` terdeteksi, sistem akan meminta menggunakan baris baru native.

## Manajemen Materi

### Mode Penggunaan Musik

Ketika **Metode Posting** diatur ke "melalui pencarian menggunakan suara", skrip mendukung dua mode penambahan musik yang berbeda:

#### 1. Mode Pencarian Suara

Masukkan nama suara untuk mencari musik dalam TikTok:

- Contoh: `original sound`, `trending music`, `viral beat`
- Skrip akan mencari perpustakaan musik TikTok dan memilih hasil pertama yang cocok
- Mode ini cocok untuk menggunakan suara populer atau trending

#### 2. Mode Link Suara

Masukkan link suara TikTok langsung untuk menggunakan musik tertentu:

- Format link yang didukung:
  - Link pendek: `https://www.tiktok.com/t/ZP8hvBwCb/`
  - Halaman musik: `https://www.tiktok.com/music/original-sound-7473897776619834646`
  - Link VM: `https://vm.tiktok.com/ZMhvBwCbD/`
- Skrip secara otomatis mendeteksi link yang dimulai dengan `https://` dan langsung membuka
- Mode ini cocok untuk menggunakan suara tertentu dari video tertentu

### Perpustakaan Materi

- **Tag Materi**: Tambahkan tag untuk materi, filter saat posting. Jika tidak ada tag yang ditentukan, gunakan semua materi.
- **Judul Materi**: Anda dapat mengatur judul khusus untuk setiap materi. Jika diatur, itu lebih diprioritaskan daripada pengaturan posting.
- **Username Materi**: Anda dapat mengatur username khusus untuk setiap materi. Materi yang cocok dengan username diprioritaskan.
- **Urutan Materi**: Atur nomor urut untuk materi (berguna untuk postingan multi-gambar; semakin kecil angka semakin depan).

### Penjelasan Format File Caption.txt

Saat menggunakan **Sumber Folder Lokal**, file `caption.txt` digunakan untuk mengkonfigurasi template Spintax. File ini dapat menimpa konfigurasi caption di UI.

**Catatan Penting**: Caption.txt hanya mendukung format template Spintax, tidak mendukung mode pemilihan berurutan Multiline.

#### Format Template Spintax

Konfigurasi satu template Spintax, gunakan sintaks `{opsi1|opsi2}` untuk menghasilkan variasi teks acak. Gunakan baris baru native (tekan enter) untuk mengimplementasikan multi-baris:

```text
{Selamat pagi|Halo|Hai}! Lihat video {luar biasa|menakjubkan|hebat} ini!
{Ikuti untuk lebih banyak|Like dan share|Tag teman Anda}
```

Kemungkinan output:

- "Selamat pagi! Lihat video luar biasa ini!"
- "Halo! Lihat video menakjubkan ini!"

## FAQ

1. Mengapa tugas gagal?
   - Skrip otomasi mensimulasikan operasi berdasarkan perilaku aplikasi TikTok/Instagram, dalam beberapa kasus mungkin gagal, misalnya masalah jaringan, respons aplikasi yang tidak normal, popup tak terduga, masalah performa perangkat, masalah kompatibilitas perangkat, dll.
   - Jika masih gagal setelah beberapa kali percobaan, kirim tiket untuk mendapatkan bantuan.
2. Bagaimana cara menjalankan tanpa pengawasan?
   - Buat grup perangkat, dan gunakan fitur penjadwalan terjadwal dalam pengaturan Post grup, perangkat lunak akan secara otomatis membuat dan menjalankan tugas posting pada waktu yang ditentukan setiap hari.
3. Bagaimana cara mengimplementasikan urutan postingan slideshow?
   - Atur nomor urut untuk setiap materi gambar di perpustakaan materi, semakin kecil angka semakin depan. Skrip akan memilih gambar berdasarkan nomor urut saat posting.
   - Anda juga dapat menggunakan nama file yang sesuai dengan spesifikasi berikut untuk mengatur nomor urut saat mengimpor materi secara batch:
     - `sort_1_deskripsi_gambar.png`
     - `sort_2_deskripsi_gambar.png`
     - `sort_3_deskripsi_gambar.png`
4. Bagaimana cara menggunakan materi, caption, dan musik yang berbeda untuk setiap akun?
   - Saat menggunakan folder lokal, buat subfolder untuk setiap username, buat file `caption.txt`, `sound_names.txt` atau `sound_urls.txt` dalam subfolder untuk mendefinisikan caption dan musik khusus. Skrip akan memprioritaskan file yang cocok dengan username.
   - Saat menggunakan perpustakaan materi, atur field username dan field caption untuk setiap materi, skrip akan memprioritaskan materi dan caption yang cocok dengan username.
5. Format materi apa yang didukung?
   - Video: MP4
   - Gambar: JPG, PNG

6. Apakah materi gambar dan video dapat digunakan berulang kali?

  Tidak. Untuk melindungi akun Anda, sistem akan secara otomatis menandai materi sebagai sudah digunakan setelah dipublikasikan, menghindari dipilih berulang kali. Ini dapat mengurangi risiko posting berulang, mengurangi kemungkinan akun terkena kontrol risiko atau diblokir.
