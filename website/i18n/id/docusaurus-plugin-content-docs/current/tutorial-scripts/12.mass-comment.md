# Komentar Massal

:::warning Pemberitahuan Penghentian
Fitur ini telah diintegrasikan ke [skrip Super Marketing](./14.super-marketing.md). Silakan prioritaskan penggunaan skrip "Super Marketing" untuk kemampuan komentar massal dan interaksi konten yang lebih kuat dan fleksibel.
:::

Komentar Massal digunakan untuk mempublikasikan komentar interaktif pada banyak postingan TikTok (seperti pemanasan akun, promosi, operasi kampanye). Sistem mendukung template yang telah digenerate dan generate real-time sebagai dua sumber pesan, dan akan berusaha mensimulasikan ritme pengiriman manual untuk mengurangi risiko deteksi.

## Menggunakan Komentar Massal dalam Skrip "Super Marketing"

1. Buka skrip: Pilih "Super Marketing".
2. Pilih tipe sumber data sebagai "Daftar Link Postingan".
3. Paste link postingan secara massal atau impor dari file teks (satu per baris); setelah impor dapat memilih "Tambahkan" atau "Timpa".
4. Pilih strategi penggunaan data:

   - Strategi Shared Pool: Semua akun bot berkomentar pada postingan yang sama.

   - Strategi Single-consumption: Setiap akun bot hanya berkomentar sekali pada postingan yang berbeda, memastikan setiap postingan hanya diproses sekali oleh satu akun.

5. Atur jumlah komentar untuk setiap postingan (biasanya 1 kali untuk menghindari komentar berulang).
6. Dalam "Pengaturan Komentar" konfigurasi opsi berikut:

   - Template komentar yang telah digenerate: Dapat paste teks komentar yang telah digenerate dengan beberapa baris atau beberapa bagian (setiap baris atau setiap bagian dianggap sebagai satu kandidat komentar).

   - Urutan penggunaan teks: Pilih "Acak" atau "Berurutan".

   - Acak: Setiap kali acak mengambil satu teks dari kandidat.

   - Berurutan: Rotasi penggunaan sesuai urutan paste/impor (cocok untuk strategi deduplikasi yang dapat direproduksi).

   - Sisipkan emoji: Dapat mengaktifkan untuk secara otomatis menyisipkan emoji dalam komentar untuk meningkatkan kealamian.

   - Sisipkan nomor perangkat (opsional): Tambahkan nomor perangkat di akhir komentar untuk melacak sumber (hanya untuk debugging/audit, gunakan dengan hati-hati).

   - Generate komentar secara real-time (opsional, ChatGPT API):

      - Setelah diaktifkan, skrip dapat memanggil ChatGPT atau API serupa, generate teks komentar yang relevan dengan konteks dan unik berdasarkan judul/deskripsi postingan atau konteks target secara real-time, cocok untuk skenario yang memerlukan banyak komentar unik.

      - Catatan: Mengaktifkan generate real-time memerlukan API Key yang valid, dan perhatikan pembatasan rate, biaya, dan kepatuhan privasi (hindari mengirim informasi sensitif ke layanan pihak ketiga).

   - Dua cara umum untuk menjamin komentar unik:

      1. Gunakan "Berurutan" untuk merotasi template komentar yang telah digenerate (setiap template hanya digunakan sekali hingga habis).

      2. Gunakan ChatGPT untuk generate setiap komentar secara real-time (generate dinamis berdasarkan konten postingan, keunikan lebih tinggi).

7. Konfigurasi parameter skrip lain (jumlah konkurensi, interval tugas, pool akun, dll.), setelah konfirmasi tidak ada masalah jalankan skrip.

:::tip Tentang Interval Tugas
Disarankan untuk mengatur interval minimum/maksimum dan mengaktifkan interval acak, untuk menghindari semua perangkat mengeksekusi secara bersamaan yang menyebabkan puncak abnormal, sehingga mengurangi risiko deteksi atau pembatasan oleh platform.
:::

## Saran Penggunaan dan Catatan

- Pertama uji strategi komentar Anda pada sejumlah kecil postingan, konfirmasi kualitas komentar dan situasi pemrosesan platform.
- Konten komentar harus patuh dan beragam, hindari mengirim banyak konten yang sama dalam waktu singkat.
- Jika menggunakan generate real-time, harap monitor jumlah pemanggilan API dan biaya dengan ketat, dan pastikan tidak membocorkan informasi sensitif ke luar.
- Tugas mungkin gagal; jika gagal silakan coba lagi atau lihat log, sistem akan secara otomatis melewati target yang telah selesai.
