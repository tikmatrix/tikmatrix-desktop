# Hapus Postingan

Skrip hapus postingan digunakan untuk menghapus postingan berdasarkan jumlah views.

## Langkah-langkah

1. Pilih perangkat yang akan menjalankan skrip.
2. Klik `Skrip` > `Hapus Postingan`.
3. Konfigurasi pengaturan tugas:
    - **Views Maksimum**: Masukkan jumlah views maksimum dari postingan yang ingin Anda hapus.
4. Klik `Mulai Skrip` untuk memulai eksekusi.

## Catatan

- Skrip hapus postingan mungkin tidak selalu berhasil; silakan coba lagi tugas yang gagal jika diperlukan.
