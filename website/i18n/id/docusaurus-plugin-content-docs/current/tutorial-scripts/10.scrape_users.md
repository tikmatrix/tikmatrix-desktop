# Scrape User

Skrip scrape user digunakan untuk mengumpulkan username TikTok. Fitur scrape user mendukung tiga mode: mengumpulkan daftar pengikut user tertentu, mengumpulkan daftar following user tertentu, atau mengumpulkan user melalui pencarian kata kunci.

## Langkah-langkah

1. Pilih perangkat yang akan menjalankan skrip. Jika memilih beberapa perangkat, tugas scraping akan didistribusikan secara merata di antara perangkat yang dipilih.
2. Klik `Skrip` > `Scrape User`.
3. Konfigurasi pengaturan tugas:
    - **Mode Scraping**: Pilih mode untuk scraping user.
      - **Scraping Melalui Pengikut**: Kumpulkan daftar pengikut akun tertentu.
      - **Scraping Melalui Following**: Kumpulkan daftar following akun tertentu.
      - **Scraping Melalui Kata Kunci**: Kumpulkan user menggunakan pencarian kata kunci.
    - **Username Akun Target**: Masukkan username akun untuk scraping user, satu per baris (untuk mode pengikut dan following).
    - **Kata Kunci Pencarian**: Masukkan kata kunci untuk mencari user (untuk mode kata kunci).
    - **Jumlah Scraping Maksimum**: Atur jumlah maksimum user yang akan di-scrape.

4. Klik `Mulai Skrip` untuk memulai eksekusi.

## Catatan

- Data user yang di-scrape akan disimpan ke file berdasarkan mode scraping:
  - Mode pengikut: `exported_users_followers_tiktok_<username>_<jumlah>.txt`
  - Mode following: `exported_users_following_tiktok_<username>_<jumlah>.txt`
  - Mode kata kunci: `exported_users_search_tiktok_<keyword>_<jumlah>.txt`
- TikTok hanya membatasi jumlah user yang dapat dilihat, jadi setiap operasi hanya dapat mengambil sekitar 50 record.
