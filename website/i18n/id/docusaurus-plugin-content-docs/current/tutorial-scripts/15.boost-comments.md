# Boost Komentar

Skrip boost komentar digunakan untuk meningkatkan jumlah interaksi komentar TikTok tertentu secara massal melalui like dan reply.

## Langkah-langkah

1. Pilih perangkat yang akan menjalankan skrip.
2. Klik `Skrip` > `Boost Komentar`.
3. Konfigurasi pengaturan tugas:
    - **URL Komentar Target**: Masukkan URL komentar target, satu per baris. Tekan lama komentar untuk menyalin link komentar.
    - **Opsi Boost**: Pilih operasi boost:
        - **Like Komentar**: Memberikan like pada komentar target
        - **Reply Komentar**: Membalas komentar target
    - **Konten Reply**: Masukkan konten reply, satu per baris (hanya diperlukan saat reply komentar diaktifkan).
    - **Sisipkan Emoji**: Pilih apakah akan menyisipkan emoji acak dalam reply.
    - **Urutan Komentar**: Pilih bagaimana memilih dari beberapa konten reply:
        - **Acak**: Pilih secara acak dari konten reply
        - **Berurutan**: Gunakan konten reply secara berurutan
    - **Interval Boost Komentar**: Atur interval antara setiap tugas boost.
4. Klik `Mulai Skrip` untuk memulai eksekusi.

## Fitur Unggulan

- **Link Komentar Langsung**: Mendukung URL komentar langsung yang disalin dari TikTok
- **Operasi Multi**: Dapat memberikan like dan reply pada komentar secara bersamaan
- **Diversifikasi Konten**: Mendukung beberapa konten reply, dapat dipilih secara acak atau berurutan
- **Dukungan Emoji**: Secara otomatis menyisipkan emoji membuat reply lebih menarik
- **Pemrosesan Efisien**: Memproses beberapa URL komentar secara batch

## Catatan

- Skrip boost komentar mungkin tidak selalu berhasil; silakan coba lagi tugas yang gagal jika diperlukan.
- Menggunakan URL komentar langsung dapat memberikan hasil terbaik. Tekan lama komentar di TikTok untuk menyalin linknya.
- Konten reply harus sesuai dan mematuhi pedoman komunitas TikTok.
