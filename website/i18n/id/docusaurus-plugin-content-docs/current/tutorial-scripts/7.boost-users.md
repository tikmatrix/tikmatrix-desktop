# Boost User

:::warning Pemberitahuan Penghentian
Fitur ini telah diintegrasikan ke [skrip Super Marketing](./14.super-marketing.md). Silakan prioritaskan penggunaan "Super Marketing" untuk kemampuan boost user yang lebih kuat dan fleksibel.
:::

Fitur Boost User meningkatkan eksposur dan interaksi user target melalui operasi follow/unfollow secara massal. Sistem akan berusaha mensimulasikan perilaku user nyata untuk mengurangi risiko deteksi, tetapi tetap disarankan untuk mengikuti aturan platform dan mengontrol frekuensi.

## Menggunakan Fitur Boost User dalam Skrip "Super Marketing"

1. Buka skrip: Pilih "Super Marketing".
2. Pilih tipe sumber data sebagai "Daftar Akun User".
3. Paste username secara massal atau impor dari file teks (satu per baris); setelah impor dapat memilih "Tambahkan" atau "Timpa".
4. Pilih strategi penggunaan data:

    - Strategi Shared Pool: Semua akun bot akan melakukan follow/unfollow pada target yang sama.

    - Strategi Single-consumption: Setiap akun bot hanya memproses target yang berbeda, memastikan setiap target hanya dioperasikan sekali oleh satu akun.

5. Dalam "Operasi User â€” Jumlah Pengulangan" atur jumlah operasi untuk setiap target (biasanya diatur 1 kali).
6. Pilih jenis operasi: Follow atau Unfollow. Dapat dikonfigurasi lebih lanjut dalam "Operasi User":

        - Cara masuk ke halaman profil user: Pilih "Cari user" atau "Buka profil langsung (Direct Open Profile)".

        - Kirim pesan pribadi (opsional): Setelah diaktifkan dapat paste teks pesan pribadi yang telah digenerate dengan beberapa baris, setiap baris atau per bagian sebagai satu kandidat teks; konfigurasi urutan penggunaan teks:

                - Acak: Setiap kali menggunakan acak memilih satu teks.
                - Berurutan: Gunakan sesuai urutan paste/impor (dapat berlanjut lintas putaran tugas).

            Juga dapat memilih apakah akan secara otomatis menyisipkan emoji dalam teks untuk meningkatkan kealamian.

7. Konfigurasi parameter lain (konkurensi, delay, pool akun), setelah konfirmasi tidak ada masalah jalankan skrip.

## Saran Penggunaan dan Catatan

- Disarankan untuk menguji terlebih dahulu pada sejumlah kecil akun, amati apakah strategi dan parameter memicu kontrol risiko platform.
- Atur delay dan konkurensi dengan wajar, simulasikan perilaku manual untuk mengurangi risiko.
- Saat menjalankan jangka panjang silakan cek kesehatan akun dan log tugas secara berkala, dan sesuaikan strategi berdasarkan hasil.
- Operasi boost user tidak 100% berhasil, jika gagal silakan coba lagi atau periksa status akun target.
