# Profil

Skrip profil digunakan untuk mengatur profil akun.

## Langkah-langkah

1. Pilih perangkat yang akan mengatur profil.
2. Klik tombol `Skrip` - `Profil`.
3. Konfigurasi pengaturan tugas:
    - **Nama Panggilan**: Satu nama panggilan per baris.
    - **Username**: Satu username per baris.
    - **Bio**: Satu bio per baris (mendukung mention pengguna dan baris baru).
      - Untuk menambahkan baris baru dalam bio, gunakan `\n` dalam teks. Contoh: `Baris pertama\nBaris kedua\nBaris ketiga`
    - **Path Avatar**: Path ke folder file gambar avatar.
    - **Urutan Pemilihan**: Pilih bagaimana konten dipilih untuk setiap tugas:
      - **Acak**: Pilih opsi yang tersedia secara acak untuk setiap tugas (default).
      - **Berurutan**: Pilih konten secara berurutan berdasarkan jumlah tugas. Gunakan mode ini untuk memastikan setiap akun mendapat informasi unik.
4. Klik tombol `Mulai Skrip` untuk memulai tugas.

## Penjelasan Urutan Pemilihan

### Mode Acak

- Setiap tugas memilih konten secara acak (nama panggilan, username, bio, avatar)
- Konten yang sama mungkin digunakan ulang di berbagai akun
- Cocok untuk skenario yang tidak memerlukan keunikan konten

### Mode Berurutan

- Pilih konten secara berurutan: tugas ke-1 menggunakan item ke-1, tugas ke-2 menggunakan item ke-2, dan seterusnya
- Ketika semua item telah digunakan, akan kembali ke awal
- **Penting**: Untuk memastikan setiap akun mendapat informasi unik, pastikan jumlah konten yang disediakan lebih besar dari total jumlah akun yang dijalankan dalam satu batch
  - Contoh: Jika menjalankan 100 akun, sediakan setidaknya 100 nama panggilan, username, bio, dan gambar avatar yang berbeda

## Catatan

- Sebaiknya kosongkan album foto sebelum mengatur profil, karena skrip akan menggunakan foto pertama di album sebagai avatar.
