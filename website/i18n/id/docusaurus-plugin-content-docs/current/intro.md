# Pengenalan Tutorial

Pelajari fungsi inti dan cara menggunakan **TikMatrix** dan **IgMatrix** dalam waktu kurang dari 5 menit.

## Pengenalan Software

**TikMatrix** adalah software desktop PC yang secara otomatis menjalankan tugas TikTok pada perangkat Android.  
**IgMatrix** adalah software desktop PC yang secara otomatis menjalankan tugas Instagram pada perangkat Android.

**Catatan:** **TikMatrix** dan **IgMatrix** adalah dua software terpisah dan tidak dapat dijalankan secara bersamaan di komputer yang sama.

## Fitur Utama

- **Multi-Akun Switching**: Satu perangkat dapat login beberapa akun TikTok/Instagram dan secara otomatis beralih untuk menjalankan tugas.
- **Dukungan Aplikasi Clone**: Mendukung aplikasi clone TikTok/Instagram dengan nama paket kustom, akun berbeda dapat terikat ke aplikasi clone berbeda dan secara otomatis membuka APP target.
- ~~**Pendaftaran Otomatis**~~: **[Sudah dihentikan]**
- **Login Otomatis**: Instagram mendukung login otomatis 2FA; TikTok masih memerlukan intervensi manual untuk beberapa verifikasi (seperti verifikasi grafis, verifikasi email).
- **Isi Profil Otomatis**: Mendukung pengaturan avatar, nickname, bio, dan username.
- **Identifikasi Otomatis**: Identifikasi dan impor akun TikTok/Instagram yang sudah login di ponsel dengan satu klik.
- **Pemanasan Akun Otomatis**: Mendukung pencarian kata kunci, menonton video, like, follow, komentar, dan bookmark untuk pemanasan akun.
- **Posting Otomatis**: Mendukung posting video atau gambar (dapat mengatur urutan dan jumlah batch), mendukung musik kustom, serta termasuk judul, deskripsi, hashtag, dan link produk.
- **Penghapusan Otomatis**: Menghapus postingan dengan performa rendah berdasarkan jumlah views atau aturan yang ditentukan.
- **Super Marketing**: Mensimulasikan jalur kunjungan seperti manusia (mengunjungi homepage, browsing postingan, random follow/like/komentar/bookmark/share/DM), mendukung penggunaan OpenAI atau API kompatibel untuk menghasilkan konten DM dan komentar yang dipersonalisasi.
- **Batch Boost (Live)**: Like dan komentar batch untuk siaran langsung.
- **Scrape Users**: Mengumpulkan username target melalui kata kunci, daftar followers atau following.
- **Batch Follow Back**: Follow followers secara batch dan kirim DM selamat datang.
- **Task Scheduling**: Penjadwalan presisi level menit, mendukung retry otomatis, eksekusi interval, dan pengaturan maksimum retry.
- **Multi-Device Parallel**: Satu PC dapat terhubung dan mengelola beberapa perangkat Android secara paralel untuk menjalankan tugas.
- **Screen Mirroring**: Mendukung screen mirroring beberapa perangkat ke Windows/macOS, dan mengontrolnya melalui mouse dan keyboard.
- **Import/Export Akun**: Mendukung import dan export batch dalam format Excel.
- **Dukungan Multi-Bahasa**: Software PC mendukung Bahasa Inggris, Rusia, dan Mandarin. Catatan: Beberapa fitur memerlukan aplikasi TikTok/Instagram di ponsel dalam Bahasa Inggris untuk kompatibilitas terbaik.
- **Dukungan Tiket**: Menyediakan layanan tiket manual untuk menyelesaikan masalah selama penggunaan.

- Pembayaran tahunan dapat diskon 30%.

## Apa yang Anda Butuhkan

### Ponsel Android

Anda dapat menggunakan perangkat Android apa pun untuk menjalankan TikMatrix, termasuk:

- **Perangkat Android asli**
- **[Perangkat Android motherboard](http://www.niaozun.shop?cid=934ec2fe)**
- **[Cloud phone](https://www.geelark.cn?invite_code=XHY6a8)**
- **Emulator**
- **Virtual machine**

Perangkat Android minimal memerlukan API 21 (Android 5.0), dan tidak memerlukan akses root. Harap aktifkan USB debugging (opsi pengembang) di perangkat. Pada beberapa model (seperti Xiaomi), Anda mungkin juga perlu mengaktifkan "USB debugging (pengaturan keamanan)" untuk mendukung kontrol keyboard dan mouse.

### Komputer

#### Windows: Windows 10 atau lebih tinggi

#### Mac: macOS 15.0 atau lebih tinggi (mendukung chip Apple Silicon M-series dan chip Intel)

## Instalasi

1. Unduh installer terbaru dari [Unduh TikMatrix](https://tikmatrix.com/Download) atau [Unduh IgMatrix](https://igmatrix.com/Download-IgMatrix).
2. Jalankan installer.
3. Tunggu instalasi selesai.
4. Buka aplikasi **TikMatrix** atau **IgMatrix** di komputer Anda.
