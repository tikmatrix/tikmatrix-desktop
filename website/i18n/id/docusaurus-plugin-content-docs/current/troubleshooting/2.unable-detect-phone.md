# Tidak Dapat Mendeteksi Ponsel

## Langkah Pemecahan Masalah Tidak Dapat Mendeteksi Ponsel

### Langkah Pertama: Periksa Status Koneksi ADB

1. Di TikMatrix, klik `Pengaturan` > `Buka Terminal ADB`  
2. Jalankan perintah `adb devices`, periksa apakah ponsel muncul di daftar

### Sistem Windows

```shell
.\adb.exe devices
```

### Sistem Mac/Linux

```shell
./adb devices
```

### Contoh Output

```shell
List of devices attached
<ID Perangkat>    <Status>
```

### Jika perangkat tidak muncul di daftar perangkat adb

Silakan ikuti langkah-langkah dalam dokumen [Menghubungkan Perangkat](../tutorial-basics/1.connect-to-devices.md) untuk pemecahan masalah

### Jika status perangkat adalah `unauthorized`

Kemungkinan penyebabnya adalah perangkat tidak diotorisasi, silakan ketuk `Izinkan USB debugging` di ponsel

### Jika status perangkat adalah `offline`

Kemungkinan penyebabnya adalah kabel USB kendor, silakan sambungkan kembali ponsel, cabut kabel USB, sambungkan kembali

### Jika status perangkat adalah `device`

Perangkat telah berhasil terhubung. Jika TikMatrix masih tidak mendeteksi ponsel, pastikan TikMatrix sudah diaktivasi. Uji coba gratis juga perlu diaktifkan di halaman otorisasi

### Jika masalah masih berlanjut

Silakan hubungi tim dukungan teknis kami melalui [Telegram](https://t.me/tikmatrix_agent_bot), berikan informasi berikut:

- Screenshot output perintah adb devices
- Langkah-langkah yang sudah Anda coba
