# Permission Denied - Unable to Save Draft Videos

## Menyelesaikan Masalah Pesan Izin Ditolak di Layar Ponsel

### Deskripsi Masalah

Saat menjalankan skrip post, layar ponsel mungkin menampilkan pesan berikut:

```text
Permission denied. Unable to save draft videos.
```

### Solusi

Masalah ini disebabkan oleh aplikasi TikTok yang tidak memiliki izin penyimpanan. Ikuti langkah-langkah berikut untuk menyelesaikannya:

#### Langkah 1: Klik Tombol Post Secara Manual

1. Temukan dan klik tombol **Post** di layar ponsel
2. Lanjutkan proses posting

#### Langkah 2: Izinkan Izin Sistem

1. Ketika dialog permintaan izin sistem muncul, klik **Izinkan**
2. Pastikan untuk memberikan izin penyimpanan yang diperlukan untuk aplikasi TikTok

### Catatan Penting

:::info
Langkah pengaturan izin ini hanya perlu dilakukan sekali. Setelah izin diberikan, menjalankan skrip post selanjutnya tidak akan menampilkan masalah ini lagi.
:::

### Tindakan Pencegahan

Untuk menghindari masalah ini, disarankan sebelum menggunakan TikMatrix untuk pertama kali:

1. Buka aplikasi TikTok secara manual
2. Coba rekam atau simpan draft video
3. Pilih **Izinkan** saat permintaan izin muncul
4. Ini akan mengonfigurasi izin yang diperlukan sebelumnya
