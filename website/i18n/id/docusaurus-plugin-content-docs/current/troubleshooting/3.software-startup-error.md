# Kesalahan Startup Perangkat Lunak

## Langkah Pemecahan Masalah Kesalahan Startup Perangkat Lunak Umum

### Langkah 1: Periksa Persyaratan Sistem

1. Pastikan sistem Anda memenuhi persyaratan minimum:
   - Windows 10 atau lebih tinggi
   - Minimal 4GB memori
   - Minimal 1GB ruang disk tersedia

### Langkah 2: Periksa Microsoft Visual C++ Redistributable

Biasanya, kesalahan `vcruntime140.dll tidak ditemukan` atau `Aplikasi tidak dapat memulai dengan benar (0xc000007b)` disebabkan oleh Microsoft Visual C++ Redistributable yang tidak terinstal.

1. Tekan tombol `Windows + R`, ketik `cmd` dan tekan enter
2. Ketik `dir "C:\Windows\System32\vcruntime*.dll"` dan tekan enter
3. Jika ada output, artinya versi Visual C++ Redistributable terkait sudah terinstal. Nama file dengan sufiks vcruntime biasanya membawa nomor versi, seperti vcruntime140.dll (untuk 2015-2022).
4. Jika tidak ada output, silakan unduh dan instal Microsoft Visual C++ Redistributable: [Latest Microsoft Visual C++ Redistributable Version](https://learn.microsoft.com/en-us/cpp/windows/latest-supported-vc-redist?view=msvc-170)

### Langkah 3: Periksa Waktu Sistem

Jam komputer yang tidak akurat dapat menyebabkan kesalahan "Agent Start Timeout":

1. Klik kanan pada tampilan waktu di pojok kanan bawah layar
2. Pilih "Sesuaikan tanggal/waktu"
3. Jika belum diaktifkan, aktifkan "Atur waktu secara otomatis"
4. Jika waktu masih tidak benar, klik "Sinkronkan sekarang" atau atur tanggal dan waktu yang benar secara manual
5. Setelah memperbaiki waktu, restart TikMatrix

### Langkah 4: Periksa Log Aplikasi

1. Klik tombol `Pengaturan` - `Buka Direktori Aplikasi`
2. Navigasi ke folder `logs`
3. Periksa pesan kesalahan di file log terbaru

### Jika masalah masih berlanjut

### Langkah 5: macOS â€” Agent InvalidStateError

Di beberapa mesin macOS, Agent mungkin gagal memulai dengan kesalahan berikut:

```text
Agent start error InvalidStateError: The object is in an invalid state.
```

Kemungkinan penyebab:

- Biasanya disebabkan oleh ketidakkompatibilitas antara aplikasi dengan versi sistem macOS saat ini atau hardware/firmware lama.

Solusi / workaround:

1. Update macOS ke versi terbaru melalui "System Settings" > "General" > "Software Update". Setelah update, restart Mac dan jalankan TikMatrix lagi.
2. Jika tidak dapat upgrade atau kesalahan masih ada, coba jalankan aplikasi di Mac lain yang menjalankan macOS lebih baru.

Jika masalah masih berlanjut setelah mencoba metode di atas, silakan kumpulkan log aplikasi dan hubungi kami melalui [Telegram](https://t.me/tikmatrix_agent_bot), sertakan versi macOS dan file log Anda.

Silakan hubungi tim dukungan kami [Telegram](https://t.me/tikmatrix_agent_bot), dan berikan informasi berikut:

- Versi Windows
- File log aplikasi
- Langkah-langkah yang sudah Anda coba
