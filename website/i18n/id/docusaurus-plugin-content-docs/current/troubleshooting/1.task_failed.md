# Langkah Pemecahan Masalah Kegagalan Eksekusi Tugas

## Langkah 1: Periksa Status Agent (jika tugas gagal dimulai)

1. Klik "Umum" di sidebar kiri - "Buka Agent"
2. Pastikan aplikasi TikMatrix telah dibuka di ponsel
3. Amati apakah status berhasil ditampilkan dalam 10 detik
4. Jika status Agent gagal, ikuti langkah-langkah dalam dokumen [Inisialisasi Perangkat](../tutorial-basics/2.init-device.md) untuk melakukan inisialisasi manual

## Langkah 2: Periksa Status Aplikasi TikTok/Instagram (jika skrip tidak membuka atau menutup aplikasi dengan benar)

1. Jika Anda menggunakan aplikasi resmi TikTok/Instagram, klik "Buka TikTok"/"Buka Instagram", amati apakah ponsel membuka aplikasi TikTok atau Instagram dengan benar. Jika tidak, klik tombol pengaturan di title bar, pilih versi TikTok yang benar (Internasional/Asia), Instagram hanya memiliki satu versi resmi
2. Jika Anda menggunakan aplikasi klon TikTok/Instagram, pastikan Anda telah mengonfigurasi **nama paket** aplikasi klon dengan benar di **daftar akun**. Silakan lihat dokumen [Menggunakan Aplikasi Klon TikTok/Instagram](../tutorial-basics/10.clone-apps-support.md) untuk konfigurasi, dan pastikan **mode multi-akun** diaktifkan saat menjalankan tugas

## Langkah 3: Periksa Pengaturan Bahasa TikTok/Instagram

Saat ini kami hanya mendukung TikTok dan Instagram berbahasa Inggris. Jika aplikasi TikTok atau Instagram Anda bukan dalam bahasa Inggris, ikuti langkah-langkah berikut:

1. Buka aplikasi TikTok di ponsel Anda
2. Masuk ke profil pribadi
3. Klik tiga titik di pojok kanan atas untuk membuka pengaturan
4. Gulir ke bagian "Language"
5. Pilih "English" sebagai bahasa
6. Ulangi langkah-langkah di atas untuk mengatur bahasa aplikasi Instagram ke Inggris

## Langkah 4: Jalankan Skrip Tes

1. Klik "Skrip" di sidebar kiri - "Skrip Tes"
2. Tunggu hingga skrip selesai dijalankan
3. Jika skrip tes gagal, periksa log dan coba selesaikan masalahnya. Untuk masalah umum, lihat bagian "Masalah yang Diketahui" di bawah

## Langkah 5: Kirim Tiket

Jika keempat langkah pemeriksaan tidak menemukan masalah, Anda perlu membuat tiket. Kami akan secara otomatis mengumpulkan log dan membantu Anda menyelesaikan masalah. Ikuti langkah-langkah berikut:

1. Klik tombol "Dukungan" di title bar
2. Klik tombol "Buat Tiket"
3. Masukkan informasi kontak dan deskripsi masalah Anda. Untuk menyelesaikan masalah lebih cepat, sebaiknya rekam video layar ponsel saat menjalankan skrip dan tambahkan ke lampiran
4. Pilih perangkat yang terpengaruh, dapat memilih lebih dari satu (penting, hanya dengan memilih perangkat yang terpengaruh, kami dapat menerima log terkait)
5. Klik tombol "Kirim"
6. Kami mungkin akan menghubungi Anda lebih lanjut untuk informasi tambahan, misalnya mengunduh file UI xml dari halaman tertentu: operasikan ponsel secara manual untuk tetap berada di halaman terakhir sebelum tugas gagal, klik tombol "Debug" di sisi kanan layar besar ponsel, lalu klik tombol "Unduh XML", kirim file xml yang diunduh ke email kami: `support@tikmatrix.com` dan sertakan nomor tiket
