# Menghubungkan Cloud Phone

Panduan ini akan membantu Anda menghubungkan TikMatrix ke cloud phone GeeLark.

:::warning Catatan Jaringan
Pastikan bandwidth jaringan antara komputer Anda dan data center cloud phone mencukupi dan stabil. Untuk hasil terbaik, disarankan menempatkan komputer dan data center cloud phone di negara atau wilayah yang sama untuk mengurangi latensi dan kehilangan paket, sehingga memastikan tugas otomasi berjalan dengan andal.
:::

Anda dapat membeli cloud phone dari [GeeLark](https://www.geelark.cn?invite_code=XHY6a8).

## Mengaktifkan ADB

Aktifkan ADB untuk profil cloud phone yang diinginkan di bagian profil.

![Aktifkan ADB](../img/enable-adb.webp)

Anda juga dapat mengaktifkan ADB untuk beberapa profil sekaligus.
![Aktifkan ADB untuk beberapa profil](../img/enable-adb-multiple.webp)

## Memulai Profil

Setelah membuka profil, klik ikon ðŸ”— untuk mendapatkan alamat IP, port, dan kode koneksi cloud phone.
![Dapatkan alamat IP](../img/get-ip-address.webp)

## Metode 1: Koneksi Cepat dengan Daftar Cloud Phone (Direkomendasikan)

TikMatrix kini mendukung penyimpanan informasi koneksi cloud phone untuk akses cepat satu klik.

### Menyimpan Informasi Cloud Phone

1. Buka TikMatrix, klik tombol `Scan TCP Device`.
2. Beralih ke tab `Daftar Cloud Phone`.
3. Klik tombol `Tambah`.
4. Isi informasi cloud phone:
   - **Nama**: Berikan nama yang mudah dikenali untuk cloud phone (contoh: "GeeLark Phone 1")
   - **Alamat IP**: Alamat IP yang diperoleh dari GeeLark (contoh: `1.2.3.4`)
   - **Port**: Nomor port (default: `5555`)
   - **Kode Koneksi**: Kode koneksi dari GeeLark (contoh: `f850ef`)
5. Klik `Simpan`.

### Koneksi Satu Klik

Setelah disimpan, Anda dapat terhubung dengan dua cara:

**Koneksi Tunggal:**

- Klik tombol ðŸ”Œ di samping cloud phone mana pun dalam daftar.

**Koneksi Massal:**

- Gunakan checkbox untuk memilih beberapa cloud phone.
- Klik tombol `Hubungkan (X)` di bagian atas.

Sistem akan secara otomatis melakukan:

1. Koneksi melalui ADB
2. Menjalankan perintah autentikasi glogin
3. Menampilkan hasil koneksi

Setelah koneksi berhasil, refresh daftar perangkat untuk melihat cloud phone Anda.

## Metode 2: Koneksi Manual melalui Terminal ADB

Jika Anda lebih suka koneksi manual atau perlu troubleshooting:

### Menjalankan Perintah ADB

1. Klik `Settings` > `Open ADB Terminal` di TikMatrix.
2. Jalankan perintah koneksi sesuai sistem operasi Anda dan periksa apakah ponsel Anda muncul dalam daftar dengan status `device`.
3. Jalankan perintah autentikasi untuk memverifikasi koneksi.

### Sistem Windows

```shell
.\adb.exe connect <IP_ADDRESS>:<PORT>
.\adb.exe -s <IP_ADDRESS>:<PORT> shell glogin <CONNECTION_CODE>
```

### Sistem Mac/Linux

```shell
./adb connect <IP_ADDRESS>:<PORT>
./adb -s <IP_ADDRESS>:<PORT> shell glogin <CONNECTION_CODE>
```

> Misalnya, jika alamat IP Anda adalah `1.2.3.4`, port adalah `5555`, dan kode koneksi adalah `f850ef`
>
> **Windows:**
>
> ```shell
> .\adb.exe connect 1.2.3.4:5555
> .\adb.exe -s 1.2.3.4:5555 shell glogin f850ef
> ```
>
> **Mac/Linux:**
>
> ```shell
> ./adb connect 1.2.3.4:5555
> ./adb -s 1.2.3.4:5555 shell glogin f850ef
> ```

## Tips

- Simpan semua cloud phone ke daftar untuk akses cepat.
- Status koneksi akan ditampilkan di samping setiap cloud phone.
- Anda dapat mengedit atau menghapus cloud phone yang disimpan kapan saja.
- Informasi koneksi disimpan di komputer lokal Anda.
