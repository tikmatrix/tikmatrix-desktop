# Mengaktifkan Keyboard Input Cepat

Setelah ponsel terhubung, keyboard input cepat akan secara otomatis diaktifkan.

## Mengapa Mengatur Keyboard Input Cepat

Keyboard input cepat adalah alat yang diperlukan untuk TikMatrix, menyediakan dukungan multi-bahasa otomatis. Tanpanya, fungsi input teks TikMatrix (seperti komentar, pesan, dan judul posting) mungkin tidak berfungsi dengan baik.

## Cara Mengaktifkan Keyboard Input Cepat Secara Manual

1. Pilih perangkat yang perlu menggunakan keyboard input cepat.
2. Klik `Custom ADB Command` - `Enable Fast Input`.

## Cara Memasukkan Teks

Di TikMatrix, Anda dapat memasukkan teks dengan berbagai cara:

### Cara 1: Input Keyboard Langsung ⌨️ (Fitur Baru)

⭐ **Kontrol Input Keyboard** - Anda sekarang dapat langsung menggunakan keyboard komputer untuk memasukkan teks di ponsel, membuat input teks lebih cepat dan nyaman.

1. Pilih perangkat yang perlu digunakan.
2. Klik kotak input teks apa pun di layar ponsel.
3. Mulai mengetik langsung di keyboard komputer - teks akan ditampilkan secara real-time di ponsel.

### Cara 2: Panel Input

1. Klik layar ponsel untuk memperluas layar besar, dan klik tombol `Input` di toolbar sisi kanan.
2. Masukkan teks di kotak input, klik kirim.

## Cara Menyalin dan Menempel Teks antara Ponsel dan Komputer

### Menyalin Teks Clipboard Ponsel ke Clipboard Komputer

1. Pilih ponsel yang perlu disalin, tekan `Ctrl+C`, berhasil disalin

### Menempel Teks Clipboard Komputer ke Kotak Input Ponsel

1. Pilih perangkat yang perlu ditempel, tekan `Ctrl+V`, berhasil ditempel
