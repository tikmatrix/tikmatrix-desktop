# Perintah ADB Kustom

TikMatrix memungkinkan Anda membuat dan mengelola perintah ADB kustom untuk menjalankan operasi umum dengan cepat. Perintah-perintah ini dapat menyederhanakan alur kerja Anda dan meningkatkan efisiensi operasi.

## Apa itu Perintah ADB Kustom

Fitur perintah ADB kustom memungkinkan Anda menyimpan instruksi ADB yang sering digunakan dan menjalankannya dengan sekali klik, tanpa perlu mengetik perintah lengkap setiap kali.

## Mengakses Antarmuka Perintah Kustom

1. Di antarmuka utama TikMatrix, temukan dan klik opsi **Custom Commands** di sidebar kiri.
2. Antarmuka perintah kustom akan menampilkan semua perintah yang telah Anda simpan, beserta tombol untuk menambahkan perintah baru dan mereset perintah.

## Perintah Preset

Saat pertama kali digunakan, TikMatrix telah menyiapkan beberapa perintah preset yang berguna:

- **Home**: Kembali ke layar utama perangkat
- **Back**: Melakukan operasi kembali
- **Enable TCP**: Mengaktifkan mode koneksi TCP/IP
- **Enable Fast Input**: Mengaktifkan metode input cepat
- **Open Gallery**: Membuka galeri perangkat
- **Open Settings**: Membuka pengaturan perangkat
- **Open NekoBox**: Membuka aplikasi NekoBox (jika terinstal)

## Menambahkan Perintah Kustom

1. Klik tombol **Add Command** di bagian atas antarmuka.
2. Dalam dialog yang muncul, isi informasi berikut:
   - **Nama Perintah**: Berikan nama yang mudah dikenali untuk perintah Anda
   - **Parameter Perintah**: Masukkan parameter perintah ADB (tidak perlu menyertakan prefix "adb")
3. Klik tombol **Add Command** untuk menyimpan.

## Mendukung Perintah Multi-Baris

Perintah kustom mendukung input multi-baris, setiap baris akan diperlakukan sebagai perintah ADB terpisah dan dijalankan secara berurutan.
