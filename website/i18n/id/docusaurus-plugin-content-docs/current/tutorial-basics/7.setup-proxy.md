# Mengatur Proxy

Anda dapat mengatur alamat IP proxy independen untuk setiap ponsel.

> **Disclaimer:** Semua situs web yang direkomendasikan di halaman ini berasal dari internet dan tidak terkait dengan situs ini. Silakan verifikasi validitasnya sendiri sebelum digunakan. Kami tidak memberikan jaminan apa pun.

## 1. Mengatur Proxy VPS Residensial (jika Anda telah membeli IP proxy, Anda dapat melewati langkah ini)

1. Beli VPS proxy residensial dari negara target (sebaiknya IP native dan VPS dual ISP).
2. Gunakan [https://github.com/MHSanaei/3x-ui](https://github.com/MHSanaei/3x-ui) untuk menyiapkan VPN dengan cepat di VPS:

    ```shell
    bash <(curl -Ls https://raw.githubusercontent.com/MHSanaei/3x-ui/refs/tags/v2.6.0/install.sh)
    # Ikuti prompt untuk menyelesaikan pengaturan VPN.
    ```

3. Setelah instalasi selesai, akses antarmuka web x-ui. Anda dapat menemukan URL dan kredensial login di output terminal setelah instalasi.
4. Buat pengguna baru di x-ui, atur protokol ke `SOCKS5`, dan konfigurasikan port dan pengaturan lainnya sesuai kebutuhan.
5. Setelah membuat pengguna, Anda akan melihat IP dan port proxy di antarmuka x-ui.
6. Gunakan IP dan port proxy di aplikasi NekoBox di ponsel.

## 3. Mengaktifkan Proxy di Ponsel

1. Unduh APK dan instal klien [NekoBoxForAndroid](https://github.com/MatsuriDayo/NekoBoxForAndroid) di ponsel.
2. Buka aplikasi NekoBox, klik tombol `+` untuk menambahkan proxy baru.
3. Pilih `SOCKS5` atau jenis proxy lainnya.
4. Masukkan IP dan port proxy, lalu klik `Save`.
5. Klik tombol `Start` untuk mengaktifkan proxy.
6. Setelah mengaktifkan proxy, Anda dapat memeriksa status koneksi di aplikasi NekoBox.

### Catatan untuk Pengguna China

Jika Anda berada di China, protokol `SOCKS5` mungkin tidak berfungsi dengan baik. Anda perlu menambahkan proxy terenkripsi (`VLESS`) sebagai proxy frontend. Anda dapat menggunakan alat berikut:

1. Tambahkan proxy `VLESS` di aplikasi NekoBox.
2. Tambahkan proxy `SOCKS5` di aplikasi NekoBox.
3. Tambahkan proxy `Chain` di aplikasi NekoBox. Pilih proxy `VLESS` sebagai proxy frontend dan proxy `SOCKS5` sebagai proxy backend.
