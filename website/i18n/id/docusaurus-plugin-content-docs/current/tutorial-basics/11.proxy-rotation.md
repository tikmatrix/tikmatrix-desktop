# Tutorial Rotasi Proxy Sederhana

Rotasi proxy dapat secara otomatis mengganti IP baru untuk ponsel sebelum skrip dimulai. Ini lebih aman saat beralih akun atau login, dan dapat mengurangi kemungkinan kontrol risiko.

## Persiapan Sebelum Memulai

- Memerlukan layanan proxy yang mendukung fitur "ganti IP" "rotasi IP".
- Minta link rotasi dari penyedia layanan (juga dapat disebut *Rotation URL*, *Refresh URL*, *Change IP URL*).
- Jika penyedia layanan memerlukan password atau Token tambahan, siapkan terlebih dahulu.

## Menyimpan Link Rotasi untuk Perangkat

1. Buka **Manage Devices** di desktop, beralih ke tampilan tabel.
2. Temukan perangkat yang perlu mengganti IP, klik **Configure** di kolom *Proxy rotation URL*.
3. Isi sesuai prompt:
   - **Proxy rotation URL**: Tempelkan link rotasi yang disediakan penyedia layanan. TikMatrix akan secara otomatis mengaksesnya saat perlu mengganti IP.
   - **Request method**: Tetap `GET`, kecuali penyedia layanan secara eksplisit meminta mengubah ke `POST`.
   - **Headers**, **Body**: Umumnya kosongkan. Jika penyedia layanan memberikan contoh, salin apa adanya ke kotak input.
   - **Timeout**: Waktu tunggu respons penyedia layanan, default 10 detik sudah cukup.
4. Klik **Save**. Konfigurasi akan otomatis tersimpan, tidak perlu memodifikasi file apa pun secara manual.

Setelah disimpan, baris perangkat akan menampilkan hasil rotasi terakhir: hijau berarti berhasil, abu-abu berarti belum dijalankan, merah berarti penyedia layanan mengembalikan kesalahan.

## Memeriksa Apakah Rotasi Berhasil

1. Masih di baris yang sama klik **Test Proxy Rotation**.
2. TikMatrix akan segera mengakses link rotasi.  
   - Jika berhasil, akan muncul prompt berhasil dan refresh waktu "Last rotated".  
   - Jika gagal, catat promptnya, periksa apakah link dan password benar, lalu coba lagi.

Tips kecil: Banyak proxy membatasi frekuensi ganti IP. Jika prompt gagal, tunggu beberapa detik sebelum mencoba lagi.

## Menggunakan Rotasi dalam Skrip

1. Saat menjalankan skrip, centang **Proxy Rotation** terlebih dahulu, TikMatrix akan mengganti IP untuk setiap perangkat secara berurutan sebelum membuka TikTok.
2. Anda dapat mengklik tombol log di sisi kanan perangkat untuk melihat informasi detail setiap rotasi.

## FAQ

- **Apakah semua skrip perlu rotasi?** Tidak. Hanya perlu saat ingin TikTok melihat IP baru, misalnya sebelum sering beralih akun.
- **Di mana konfigurasi disimpan?** TikMatrix mengelola secara otomatis. Setiap perangkat hanya perlu dikonfigurasi sekali.
- **Bagaimana jika link terlihat sangat panjang dan rumit?** Selama dapat dibuka di browser, TikMatrix juga dapat menggunakan.
- **Masih tidak bisa?** Kirim instruksi dan prompt kesalahan dari penyedia layanan ke customer service TikMatrix, kami akan membantu troubleshoot bersama.
