# Menginstal Douyin/TikTok

Kami merekomendasikan mengunduh versi yang ditentukan berikut karena versi yang lebih baru mungkin tidak kompatibel. Perangkat lunak akan diperbarui secara berkala, dan versi yang didukung akan diumumkan dalam grup setelah pembaruan.

## Unduh Paket Instalasi Douyin/TikTok

* Versi Internasional: [com.zhiliaoapp.musically](https://apkpure.com/tiktok-musically-2024/com.zhiliaoapp.musically)
* Versi Asia: [com.ss.android.ugc.trill](https://apkpure.com/tiktok/com.ss.android.ugc.trill)

## Langkah Instalasi

1. Unduh paket instalasi APK dari link di atas.
2. Pilih semua perangkat.
3. Klik `General` - `Install APK` - pilih paket APK yang diunduh.
4. Tunggu instalasi selesai (mungkin memakan waktu 1-3 menit per perangkat, tergantung model ponsel; harap bersabar, jangan klik tombol "Install APK" berulang kali).
5. Setelah instalasi selesai, Anda perlu mengklik tombol `Settings`, periksa apakah pilihan versi TikTok konsisten dengan versi yang Anda instal.
6. Klik tombol `Open TikTok`, periksa apakah aplikasi TikTok terinstal dengan benar.
