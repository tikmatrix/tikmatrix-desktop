# Membuat Grup

**Grup** adalah kumpulan perangkat yang memungkinkan manajemen dan operasi terpadu yang lebih mudah untuk beberapa perangkat.

## Langkah-langkah

1. Klik tombol `Tambah Grup` di sidebar kiri.
2. Masukkan nama grup.
3. Tekan tombol `Enter` untuk menyimpan grup.
4. Anda dapat memilih perangkat untuk dipindahkan ke grup tersebut.
