# Menghubungkan Perangkat

TikMatrix mendukung koneksi USB dan TCP untuk perangkat Android.

## Koneksi USB

1. Buka pengaturan developer di ponsel dan aktifkan USB debugging (lihat tangkapan layar di bawah).
   - **Catatan untuk pengguna Xiaomi**: Anda juga harus mengaktifkan "Install via USB" dan "USB debugging (Security settings)" di opsi developer.
2. Hubungkan ponsel ke komputer melalui USB.
3. Ketika ponsel menampilkan prompt, klik `Allow` untuk mengaktifkan USB debugging.
4. Tunggu koneksi terbentuk.

## Koneksi TCP

1. Hubungkan ponsel ke komputer melalui USB.
2. Di TikMatrix, klik `ADB Command` > `Enable TCP` untuk mengaktifkan koneksi TCP/IP.
3. Lepaskan kabel USB (jika menggunakan phone box, bisa beralih ke mode OTG).
4. Klik tombol `Scan TCP Device` dan masukkan rentang IP (Anda dapat menemukan informasi ini di halaman manajemen router).
5. Klik `Scan` untuk mencari perangkat dalam rentang IP yang ditentukan.
6. Tunggu koneksi terbentuk.

**Pengguna Cloud Phone**: TikMatrix kini mendukung penyimpanan informasi koneksi cloud phone untuk akses cepat. Beralih ke tab `Daftar Cloud Phone` dalam dialog scan untuk menyimpan dan mengelola koneksi cloud phone Anda. Lihat panduan [Menghubungkan Cloud Phone](./1.connect-to-cloudphone.md) untuk petunjuk detail.

### Tips

- Langkah-langkah operasional mungkin berbeda untuk merek yang berbeda, tetapi prosesnya serupa.
- Jika tidak ada respons, coba ganti kabel data atau gunakan port USB di bagian belakang komputer.

## Cara Menghubungkan Motherboard atau Casing

1. Minta file otorisasi `adbkey` dan `adbkey.pub` dari penjual motherboard atau casing.
2. Salin file-file ini ke direktori `C:\Users\nama_pengguna_komputer_anda\.android` di komputer Anda.
3. Restart komputer.

## Tangkapan Layar File Kunci ADB

### Tips Lainnya

- Backup file otorisasi Anda. Jika Anda menginstal ulang sistem atau mengganti komputer, Anda akan membutuhkannya lagi.
- Jika file-file ini hilang, Anda perlu mengizinkan permission secara manual di layar ponsel.
