# Inisialisasi Perangkat

:::warning
Aplikasi ini diinstal melalui saluran debugging ADB resmi Android untuk kontrol otomasi perangkat. Beberapa perangkat mungkin menanyakan apakah akan mengirim sampel aplikasi tidak dikenal ke Google, ini adalah prompt keamanan sistem Android dan tidak terkait dengan keamanan aplikasi, tidak akan mempengaruhi penggunaan normal. Klik "Don't send" untuk menyelesaikan instalasi.
:::

TikMatrix APK adalah agen mobile yang diperlukan untuk tugas otomasi. **Ketika Anda pertama kali menghubungkan ponsel ke TikMatrix, sistem akan secara otomatis menginisialisasi perangkat Anda dan menginstal aplikasi TikMatrix** - tanpa operasi manual.

Selama proses inisialisasi otomatis, TikMatrix akan menginstal dua file APK: TikMatrix dan TikMatrix Test (tanpa antarmuka). Jika ponsel Anda meminta otorisasi instalasi, cukup klik "Allow" (terutama penting pada perangkat Xiaomi).

## Instalasi Otomatis vs Manual

**Instalasi Otomatis (Direkomendasikan)**: Secara default, TikMatrix akan secara otomatis menginisialisasi perangkat saat pertama kali terhubung. Ini adalah metode paling sederhana dan nyaman.

**Instalasi/Pembaruan Manual**: Hanya gunakan dalam kasus khusus atau saat perlu memperbarui.

## Alur Inisialisasi Otomatis

Ketika Anda menghubungkan perangkat baru ke TikMatrix:

1. **Deteksi Perangkat**: TikMatrix secara otomatis mendeteksi perangkat yang terhubung.
2. **Instalasi Otomatis**: Sistem akan secara otomatis menginstal aplikasi TikMatrix tanpa intervensi pengguna.
3. **Konfirmasi Otorisasi**: Jika perangkat meminta izin instalasi, cukup klik "Allow".
4. **Selesai Instalasi**: Proses inisialisasi selesai dalam sekitar 10 detik.
5. **Siap Digunakan**: Perangkat Anda sekarang siap untuk menjalankan tugas otomasi.

Proses otomatis ini tidak memerlukan pengaturan manual dalam sebagian besar kasus.

## Instalasi/Pembaruan Manual

Untuk menginstal atau memperbarui TikMatrix APK secara manual, matikan toggle `Auto Wake Agent` di pengaturan. Kemudian ikuti langkah-langkah ini:

1. Pilih ponsel Anda dan klik `General` > `Initialize Agent`.
2. TikMatrix akan secara otomatis menginstal dua file APK di ponsel.
3. Jika diminta, klik "Allow" untuk mengotorisasi instalasi.
4. Jika diblokir oleh Google Play Protect, klik "Install anyway" untuk melanjutkan.
5. Tunggu instalasi selesai (sekitar 10 detik).
6. Aktifkan toggle `Auto Wake Agent` untuk mengaktifkan wake up aplikasi otomatis saat ponsel terhubung.
7. Klik `General` > `Open Agent` untuk menguji instalasi. Jika aplikasi tidak dapat diluncurkan, hubungi dukungan teknis.
