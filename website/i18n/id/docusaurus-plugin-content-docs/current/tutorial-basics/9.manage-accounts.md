# Manajemen Akun

Manajemen akun dapat menjaga konsistensi setiap identitas TikTok / Instagram dengan perangkat yang dijalankan, kredensial login, dan data yang diandalkan untuk otomasi. Daftar akun yang rapi memungkinkan Anda mengalokasikan perangkat, memantau status akun, dan mempersiapkan aktivitas tanpa harus mengoperasikan ponsel satu per satu.

## Membuat Akun Secara Manual

1. Buka **Accounts** di navigasi kiri.
2. Klik **New** untuk membuka dialog edit.
3. Isi field dasar secara berurutan:
   - **Email**: Email yang digunakan untuk login aplikasi. TikMatrix/IgMatrix akan mencoba login menggunakan Email + Password terlebih dahulu; jika email kosong, secara otomatis akan beralih ke Username + Password.
   - **Password**: Password yang digunakan untuk login aplikasi. IgMatrix mendukung menambahkan kunci 2FA dalam password, format lihat di bawah.
   - **Username**: Nama akun yang ditampilkan di TikTok/Instagram, ini adalah satu-satunya field wajib.
   - **Login Status**: Menandai apakah sudah login di perangkat saat ini (`1` = Sudah login, `0` = Belum login).
   - **Status**: Mengontrol apakah akun berpartisipasi dalam tugas (`0` = Aktif, `1` = Nonaktif).
4. Pilih label perangkat untuk mengikat akun ke ponsel atau cloud phone. Jika tidak dipilih, akun tetap tidak dialokasikan.
5. Klik **Save**, daftar akan refresh otomatis.

## Login

- Jika **Email** dan **Password** akun telah diisi, TikMatrix/IgMatrix akan mencoba login menggunakan informasi tersebut ke aplikasi TikTok/Instagram.
- Jika **Email** kosong, akan mencoba login menggunakan **Username** dan **Password**.
- Harus memilih nomor seri perangkat, skrip baru akan menjalankan operasi login di perangkat yang sesuai.
- Skrip login akan secara otomatis mengenali status login akun dan melewati akun yang sudah login.

## Dukungan 2FA (Khusus IgMatrix)

IgMatrix mendukung login otentikasi dua faktor (2FA) otomatis:

### Format Password 2FA

Untuk menggunakan 2FA, atur password dengan format berikut:

```
password_anda:kunci_2FA_anda
```

**Contoh:**

```
mypassword123:JBSWY3DPEHPK3PXP
```

## Impor dan Ekspor Akun

- **Ekspor** akan menulis semua akun ke `accounts.xlsx` di direktori download TikMatrix/IgMatrix, dapat digunakan sebagai backup atau template edit massal.
- **Impor** mendukung pembuatan atau pembaruan akun massal dari file Excel tersebut. Jika tidak yakin tentang header, klik **Export** terlebih dahulu, lalu edit file yang diunduh dan impor.
- Saat impor, baris dengan `id` akan memperbarui akun yang ada, baris tanpa `id` akan menambahkan akun baru.

> Login IgMatrix 2FA hanya perlu mengisi `username` dan `password`, format `password_anda:kunci_2FA_anda`, `device_index` nomor seri perangkat.

### Korespondensi Field Excel

| Field | Wajib | Penjelasan |
| --- | --- | --- |
| `id` | Tidak | Identifikasi unik internal, hanya simpan saat perlu memperbarui akun yang ada. |
| `email` | Tidak | Email login aplikasi, memudahkan kontak dan pemulihan. |
| `pwd` | Tidak | Password login, dapat digunakan untuk referensi cepat atau login otomatis. |
| `username` | Ya | Username TikTok/Instagram, harus unik dalam workspace. |
| `device` | Tidak | Nomor seri perangkat atau identifikasi cloud phone yang terikat. |
| `device_index` | Tidak | Alias perangkat yang ditampilkan di antarmuka, terutama untuk referensi. |
| `logined` | Tidak | Tanda sesi (`1` = Sudah login, `0` = Belum login). |
| `status` | Tidak | Tanda ketersediaan (`0` = Aktif, `1` = Nonaktif). Akun nonaktif tidak akan berpartisipasi dalam tugas skrip. |
| `nickname` | Tidak | Nickname terbaru yang diperoleh melalui sinkronisasi data TikTok. |
| `country` | Tidak | Negara/wilayah yang disimpulkan berdasarkan profil TikTok. |
| `followers` | Tidak | Jumlah followers yang disinkronkan. |
| `following` | Tidak | Jumlah orang yang diikuti akun. |
| `hearts` | Tidak | Total likes yang diperoleh akun. |
| `videos` | Tidak | Jumlah video yang dipublikasikan. |
| `friends` | Tidak | Jumlah teman yang saling follow (jika tersedia). |

### Perbedaan Field TikMatrix dan IgMatrix

- **TikMatrix** setelah menjalankan **Sinkronisasi Data TikTok** atau klik tombol query data, akan mengisi field statistik (`nickname`, `country`, `followers`, `following`, `hearts`, `videos`, `friends`).
- **IgMatrix** hanya menggunakan field dasar (`email`, `pwd`, `username`, `device`, `logined`, `status`), akan mengabaikan field tipe statistik.

## Perilaku Lainnya

- Akun yang ditandai sebagai **Nonaktif** tidak akan berpartisipasi dalam skrip otomasi, tugas terjadwal, atau operasi massal apa pun. Harus diaktifkan kembali agar dapat melanjutkan eksekusi tugas.
- Skrip **Match Accounts** akan memindai sesi aplikasi yang sudah login di perangkat, secara otomatis membuat entri yang sesuai dalam daftar akun dan mengikat perangkat yang terdeteksi, adalah cara cepat untuk mengakses ponsel yang disiapkan secara manual.
