# Dukungan Aplikasi Kloning

TikMatrix dan IgMatrix mendukung menjalankan beberapa aplikasi kloning pada perangkat yang sama (misalnya GB TikTok, GB Instagram, Instagram Plus, atau versi modifikasi lainnya), masing-masing dengan nama paket independennya sendiri. Fitur ini memungkinkan Anda mengelola beberapa akun dengan lebih efisien dengan menggunakan instance aplikasi yang berbeda.

:::danger Peringatan Risiko
Menggunakan aplikasi kloning dapat melanggar ketentuan layanan TikTok/Instagram dan dapat menyebabkan akun diblokir atau dibatasi. Kami hanya menyediakan dukungan teknis untuk fitur ini, **tidak bertanggung jawab** atas masalah akun apa pun yang disebabkan oleh penggunaan aplikasi kloning, termasuk pemblokiran, pembatasan, atau hukuman lainnya. Gunakan dengan risiko Anda sendiri.
:::

## Apa itu Nama Paket?

Setiap aplikasi Android memiliki pengenal unik yang disebut **nama paket**. Aplikasi resmi menggunakan nama paket standar:

- TikTok: `com.zhiliaoapp.musically`
- Instagram: `com.instagram.android`

Aplikasi kloning menggunakan nama paket yang berbeda, misalnya:

- GB TikTok: `com.zhiliaoapp.musically.gb`
- GB Instagram: `com.instagram.gb.android`
- Instagram Plus: `com.instagram.plus.android`

TikMatrix/IgMatrix dapat secara otomatis mendeteksi dan menggunakan nama paket yang benar untuk setiap akun.

## Mengatur Aplikasi Kloning

### Langkah Pertama: Instal Aplikasi Kloning

1. Instal aplikasi kloning pilihan Anda di perangkat atau cloud phone.
2. Pastikan setiap aplikasi kloning memiliki nama paket unik untuk menghindari konflik.
3. Login dengan akun berbeda di setiap instance aplikasi.

### Langkah Kedua: Konfigurasi Nama Paket

Anda memiliki dua cara untuk mengkonfigurasi nama paket:

#### Cara A: Konfigurasi Tingkat Akun (Direkomendasikan)

Atur nama paket spesifik untuk setiap akun:

1. Buka **Accounts** di navigasi kiri.
2. Klik akun untuk edit, atau klik **New** untuk membuat akun baru.
3. Di field **Package Name**, Anda dapat:
   - **Input Manual** nama paket aplikasi kloning (misalnya `com.instagram.gb.android`)
   - **Deteksi Otomatis** klik tombol **Detect Package Name** (perlu memilih perangkat terlebih dahulu)
4. Klik **Save**.

:::tip Deteksi Otomatis Nama Paket
Tombol **Detect Package Name** dapat secara otomatis mendeteksi aplikasi yang sedang dibuka di perangkat yang dipilih. Ini adalah cara termudah untuk mendapatkan nama paket yang benar:

1. Pilih perangkat untuk akun
2. Buka aplikasi kloning di perangkat tersebut
3. Klik tombol **Detect Package Name**
4. Nama paket akan otomatis terisi

Fitur ini berlaku untuk semua aplikasi Android, termasuk aplikasi kloning TikTok/Instagram, aplikasi App Cloner, dan aplikasi NomixCloner.
:::

Ketika Anda menjalankan skrip untuk akun ini, TikMatrix/IgMatrix akan secara otomatis menggunakan nama paket yang ditentukan.

#### Cara B: Nilai Default Global Sistem

Atur nama paket default untuk semua akun yang tidak memiliki nama paket spesifik:

1. Buka **Settings** di navigasi kiri.
2. Temukan field **TikTok Package Name** atau **Instagram Package Name**.
3. Masukkan nama paket default.
4. Klik **Save**.

:::tip Prioritas
Pengaturan tingkat akun selalu menimpa nilai default global sistem. Jika akun mengkonfigurasi nama paketnya sendiri, nama paket tersebut akan digunakan alih-alih nilai default.
:::

## Cara Kerjanya

Ketika Anda menjalankan skrip otomasi apa pun:

1. TikMatrix/IgMatrix memeriksa apakah akun target telah mengkonfigurasi nama paket kustom.
2. Jika ada, gunakan nama paket tersebut dalam semua operasi otomasi UI.
3. Jika tidak ada, kembali ke nama paket default global sistem.
4. Semua resource ID dan query XPath akan secara otomatis diformat dengan nama paket yang benar.

## Praktik Terbaik

### 1. Gunakan Username Deskriptif

Karena Anda mungkin memiliki beberapa akun di aplikasi kloning yang berbeda, gunakan username deskriptif untuk mengidentifikasi akun mana yang termasuk aplikasi mana:

```text
username_gb      → GB TikTok/Instagram
username_plus    → TikTok/Instagram Plus
username         → Aplikasi resmi
```

### 2. Kelompokkan Akun Berdasarkan Organisasi

Buat grup perangkat berdasarkan tipe aplikasi kloning:

- **Grup A**: Semua akun yang menggunakan aplikasi GB
- **Grup B**: Semua akun yang menggunakan aplikasi Plus
- **Grup C**: Semua akun yang menggunakan aplikasi resmi

### 3. Uji Sebelum Operasi Massal

Sebelum menjalankan skrip untuk semua akun:

1. Uji dengan satu akun terlebih dahulu
2. Verifikasi aplikasi yang benar digunakan
3. Konfirmasi semua operasi berjalan sesuai harapan

### 4. Jaga Konsistensi Nama Paket

Setelah mengatur nama paket untuk akun, jangan mengubahnya kecuali beralih ke aplikasi kloning yang berbeda. Menjaga konsistensi membantu menghindari kebingungan dan kesalahan.

## Pemecahan Masalah

### Skrip Membuka Aplikasi yang Salah

**Masalah**: Skrip membuka aplikasi yang salah bukan aplikasi kloning.

**Solusi**:

1. Periksa apakah field **Package Name** akun diatur dengan benar
2. Verifikasi aplikasi kloning terinstal dan nama paket cocok
3. Gunakan tombol **Detect Package Name** untuk mendapatkan nama paket yang benar secara otomatis
4. Jika aplikasi baru diinstal, restart perangkat

### Tidak Dapat Menemukan Elemen UI

**Masalah**: Skrip melaporkan kesalahan "elemen tidak ditemukan".

**Solusi**:

1. Aplikasi kloning yang berbeda mungkin memiliki layout UI yang sedikit berbeda
2. Periksa apakah versi aplikasi kloning kompatibel
3. Coba perbarui ke versi terbaru TikMatrix/IgMatrix
4. Jika masalah berlanjut, hubungi dukungan teknis

### Beberapa Akun dalam Aplikasi yang Sama

**Masalah**: Ingin beralih beberapa akun dalam instance aplikasi yang sama.

**Solusi**:

- TikTok/Instagram secara native mendukung login multi-akun
- Gunakan fitur **Switch Account** TikMatrix/IgMatrix
- Atur nama paket yang sama untuk semua akun dalam instance aplikasi tersebut
- TikMatrix/IgMatrix akan secara otomatis menangani peralihan akun

### Tombol Detect Package Name Tidak Berfungsi

**Masalah**: Tombol **Detect Package Name** tidak berfungsi atau menampilkan kesalahan.

**Solusi**:

1. Pastikan Anda telah memilih perangkat terlebih dahulu
2. Pastikan aplikasi sedang dibuka di perangkat
3. Periksa apakah koneksi perangkat stabil
4. Coba buka ulang aplikasi dan klik detect

## Menemukan Nama Paket

Jika Anda ingin memeriksa nama paket di perangkat secara manual:

### Menggunakan Perintah ADB

```bash
# Daftar semua paket terkait TikTok
adb shell pm list packages | grep musically

# Daftar semua paket terkait Instagram
adb shell pm list packages | grep instagram
```

### Menggunakan Fitur Deteksi Otomatis

Cara termudah adalah menggunakan tombol **Detect Package Name** bawaan:

1. Buka dialog edit akun
2. Pilih perangkat
3. Buka aplikasi kloning di perangkat
4. Klik **Detect Package Name**
5. Nama paket akan otomatis terdeteksi dan terisi

Metode ini berlaku untuk semua aplikasi Android dan merupakan cara yang direkomendasikan.

## FAQ

**Q: Apakah saya dapat menggunakan beberapa aplikasi kloning di perangkat yang sama?**

A: Ya, selama setiap aplikasi kloning memiliki nama paket unik, Anda dapat menjalankan beberapa instance aplikasi di perangkat yang sama.

**Q: Apakah saya perlu menginstal ulang TikMatrix/IgMatrix?**

A: Tidak perlu, fitur ini sudah terintegrasi dalam versi terbaru. Cukup perbarui ke versi terbaru dan konfigurasikan nama paket.

**Q: Apakah skrip otomasi saya yang ada akan tetap berfungsi?**

A: Ya, semua skrip yang ada sepenuhnya kompatibel. Akun tanpa nama paket kustom akan menggunakan nama paket default.

**Q: Apakah TikTok dan Instagram keduanya didukung?**

A: Ya, TikMatrix dan IgMatrix keduanya sepenuhnya mendukung deteksi dan otomasi aplikasi kloning.

**Q: Bagaimana cara kerja fitur deteksi otomatis?**

A: Ketika Anda mengklik **Detect Package Name**, TikMatrix/IgMatrix memeriksa aplikasi foreground saat ini di perangkat yang dipilih dan secara otomatis mendapatkan nama paketnya.

**Q: Dapatkah saya menggunakan aplikasi yang dibuat dengan App Cloner atau NomixCloner?**

A: Ya, fitur nama paket berlaku untuk semua alat kloning aplikasi Android, termasuk App Cloner dan NomixCloner. Gunakan fitur deteksi otomatis untuk dengan mudah mendapatkan nama paket yang benar.
