## 🔎 2. מידע זמין

| קטגוריה | שדות |
|---|---|
| פרופיל | שם משתמש, כינוי, תמונת פרופיל |
| היקף | מספר עוקבים, עוקבים אחר, סך לייקים, מספר סרטונים |
| מזהים | UserID, SecUID |
| מטא-נתונים | **מדינה**, תאריך יצירת חשבון |
| אחר | ניתוח מהיר, **הורדת סרטונים** (סרטונים ציבוריים) |

> הערה: המדינה מוסקת מאותות ציבוריים, לעיון בלבד ואינה מייצגת מיקום משפטי.
