# התקנת Douyin/TikTok

אנו ממליצים להוריד את הגרסאות המצוינות הבאות, מכיוון שגרסאות חדשות יותר עלולות שלא להיות תואמות. התוכנה תעודכן באופן קבוע, והגרסאות הנתמכות יפורסמו בקבוצה לאחר העדכון.

## הורדת חבילת התקנה Douyin/TikTok

* גרסה בינלאומית: [com.zhiliaoapp.musically](https://apkpure.com/tiktok-musically-2024/com.zhiliaoapp.musically)
* גרסה אסיה: [com.ss.android.ugc.trill](https://apkpure.com/tiktok/com.ss.android.ugc.trill)

## שלבי התקנה

1. הורד את חבילת ה-APK מהקישורים לעיל.
2. בחר את כל המכשירים.
3. לחץ על `כללי` - `התקנת APK` - בחר את חבילת ה-APK שהורדה.
4. המתן להשלמת ההתקנה (כל מכשיר עשוי לקחת 1-3 דקות, תלוי בדגם הטלפון; אנא התאזר בסבלנות ואל תלחץ שוב על כפתור "התקנת APK").
5. לאחר השלמת ההתקנה, יש ללחוץ על כפתור `הגדרות` ולבדוק שבחירת גרסת TikTok עולה בקנה אחד עם הגרסה שהתקנת.
6. לחץ על הכפתור `פתיחת TikTok` כדי לבדוק אם אפליקציית TikTok הותקנה כהלכה.
