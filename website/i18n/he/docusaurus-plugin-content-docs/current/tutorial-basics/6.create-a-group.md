# יצירת קבוצה

**קבוצה** היא אוסף של מכשירים המאפשרת ניהול ופעולה אחידים נוחים יותר של מספר מכשירים.

## שלבים

1. לחץ על כפתור `הוספת קבוצה` בסרגל הצד השמאלי.
2. הזן שם קבוצה.
3. לחץ על מקש `Enter` כדי לשמור את הקבוצה.
4. ניתן לבחור מכשירים ולהעביר אותם לקבוצה זו.
