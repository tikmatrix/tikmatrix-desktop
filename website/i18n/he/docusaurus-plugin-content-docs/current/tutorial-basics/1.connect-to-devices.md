# חיבור למכשירים

TikMatrix תומך בחיבור USB ו-TCP למכשירי Android.

## חיבור USB

1. פתח את הגדרות המפתחים בטלפון והפעל ניפוי באגים דרך USB (ראה צילומי מסך למטה).
   - **הערה למשתמשי Xiaomi**: עליך גם להפעיל את "התקנה דרך USB" ו-"ניפוי באגים USB (הגדרות אבטחה)" באפשרויות המפתחים.
2. חבר את הטלפון למחשב דרך USB.
3. כשהטלפון מציג הודעה, לחץ על `אפשר` כדי להפעיל ניפוי באגים דרך USB.
4. המתן לביסוס החיבור.

## חיבור TCP

1. חבר את הטלפון למחשב דרך USB.
2. ב-TikMatrix, לחץ על `פקודות ADB` > `הפעלת TCP` כדי להפעיל חיבור TCP/IP.
3. נתק את כבל ה-USB (אם משתמש בקופסת טלפון, ניתן לעבור למצב OTG).
4. לחץ על הכפתור `סריקת התקני TCP` והזן טווח IP (ניתן למצוא מידע זה בדף הניהול של הנתב).
5. לחץ על `סריקה` כדי לחפש מכשירים בטווח ה-IP שצוין.
6. המתן לביסוס החיבור.

**למשתמשי טלפון ענן**: TikMatrix תומך כעת בשמירת מידע חיבור לטלפון ענן לגישה מהירה. בחלון הסריקה, עבור ללשונית `רשימת טלפוני ענן` כדי לשמור ולנהל את חיבורי טלפון הענן שלך. למידע מפורט ראה את המדריך [חיבור לטלפון ענן](./1.connect-to-cloudphone.md).

### עצות

- שלבי ההפעלה עשויים להשתנות בין מותגים שונים, אך התהליך הכללי דומה.
- אם אין תגובה, נסה להחליף כבל נתונים או השתמש בפורט USB מאחורי המחשב.

## כיצד להתחבר ללוח אם או לארגז מכשירים

1. בקש מספק לוח האם או ארגז המכשירים את קבצי ההרשאה `adbkey` ו-`adbkey.pub`.
2. העתק קבצים אלה לספרייה `C:\Users\שם_המשתמש_שלך\.android` במחשב.
3. הפעל מחדש את המחשב.

- גבה את קבצי ההרשאה שלך. אם תתקין מחדש את המערכת או תחליף מחשב, תצטרך אותם שוב.
- אם הקבצים אלה אבדו, תצטרך לאשר הרשאות ידנית על מסך הטלפון.

## כיצד לחבר יותר התקני USB

💡 תיאורטית, USB יכול לתמוך בעד 127 התקנים.  
אך בפועל, רוב לוחות האם הצרכניים מוגבלים בדרך כלל לכ-40 התקנים בגלל מגבלות קושחה וערכת שבבים.

🖥️ לוחות אם ברמת שרת (לדוגמה ארכיטקטורת X79) שונים — הם בדרך כלל מסירים את מגבלות ה-USB האלה ומאפשרים לך להרחיב בקנה מידה גדול.

🔧 כיצד להצליח לחבר יותר התקנים:

1. השתמש בפורטי USB מאחורי המחשב — הם מחוברים באופן ישיר יותר للوח האם.  
2. השתמש בפורטי USB 2.0 (בדרך כלל שחורים), לא USB 3.0 (כחולים), שעלולים להיות לא יציבים עבור קבוצות מכשירים גדולות.  
3. בהגדרות BIOS:  
   - השבת XHCI  
   - הפעל EHCI

שלבים אלה יעזרו למערכת שלך לזהות באמינות יותר מכשירים.
