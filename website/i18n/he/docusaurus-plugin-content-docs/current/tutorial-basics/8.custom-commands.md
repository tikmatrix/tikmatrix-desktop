# פקודות ADB מותאמות אישית

TikMatrix מאפשרת לך ליצור ולנהל פקודות ADB מותאמות אישית כדי לבצע במהירות פעולות נפוצות. פקודות אלה יכולות לפשט את זרימת העבודה שלך ולשפר את יעילות התפעול.

## מהן פקודות ADB מותאמות אישית

פונקציונליות פקודות ADB מותאמות אישית מאפשרת לך לשמור פקודות ADB נפוצות ולבצע אותן בלחיצה פשוטה, ללא צורך להזין ידנית את הפקודה המלאה בכל פעם.

## גישה לממשק הפקודות המותאמות אישית

1. בממשק הראשי של TikMatrix, מצא ולחץ על האפשרות **פקודות מותאמות אישית** בסרגל הצד השמאלי.
2. ממשק הפקודות המותאמות אישית יציג את כל הפקודות השמורות שלך, יחד עם כפתורים להוספת פקודות חדשות ואיפוס פקודות.

## פקודות מוגדרות מראש

בשימוש ראשון, TikMatrix כבר הכינה עבורך מספר פקודות מוגדרות מראש שימושיות:

- **Home**: חזרה למסך הבית של המכשיר
- **Back**: ביצוע פעולת חזרה
- **Enable TCP**: הפעלת מצב חיבור TCP/IP
- **Enable Fast Input**: הפעלת שיטת הקלט המהיר
- **Open Gallery**: פתיחת גלריית המכשיר
- **Open Settings**: פתיחת הגדרות המכשיר
- **Open NekoBox**: פתיחת אפליקציית NekoBox (אם מותקנת)

## הוספת פקודות מותאמות אישית

1. לחץ על כפתור **הוספת פקודה** בחלק העליון של הממשק.
2. בתיבת הדו-שיח המוקפצת, מלא את המידע הבא:
   - **שם פקודה**: תן לפקודה שלך שם שקל לזהות
   - **פרמטרי פקודה**: הזן את פרמטרי הפקודה של ADB (אין צורך לכלול את קידומת "adb")
3. לחץ על כפתור **הוספת פקודה** כדי לשמור.

## תמיכה בפקודות מרובות שורות

פקודות מותאמות אישית תומכות בהזנה מרובת שורות, כל שורה תטופל כפקודת ADB נפרדת ותבוצע לפי הסדר.
