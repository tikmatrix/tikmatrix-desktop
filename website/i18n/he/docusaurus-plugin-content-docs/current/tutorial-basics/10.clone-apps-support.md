# תמיכה באפליקציות משובטות

TikMatrix תומך באפליקציות משובטות לניהול חשבונות מרובים באותו מכשיר.

## מהן אפליקציות משובטות?

אפליקציות משובטות (Clone Apps) מאפשרות להפעיל מספר מופעים של אותה אפליקציה באותו מכשיר. זה מאפשר:

- 📱 מספר חשבונות TikTok באותו טלפון
- 🔄 החלפה מהירה בין חשבונות
- 🎭 הפרדת פרופילים

## כלי שיבוט מומלצים

### 1. Parallel Space

**יתרונות:**
- 🌟 פופולרי וי יציב
- 🚀 ביצועים טובים
- 🔒 הגנת פרטיות

**הורדה:** [Google Play](https://play.google.com/store/apps/details?id=com.lbe.parallel.intl)

### 2. Dual Space

**יתרונות:**
- 💪 קל משקל
- ⚡ מהיר
- 🎨 ממשק נקי

**הורדה:** [Google Play](https://play.google.com/store/apps/details?id=com.ludashi.dualspace)

### 3. Multiple Accounts

**יתרונות:**
- 🔢 תמיכה במספר שיבוטים
- 🛡️ אבטחה משופרת
- 🎯 התאמה אישית

**הורדה:** [Google Play](https://play.google.com/store/apps/details?id=com.excelliance.multiaccounts)

## הגדרה

### שלב 1: התקן כלי שיבוט

1. הורד מ-Google Play
2. התקן במכשיר
3. פתח האפליקציה

### שלב 2: שבט TikTok/Instagram

1. פתח את כלי השיבוט
2. לחץ על **הוסף אפליקציה**
3. בחר **TikTok** או **Instagram**
4. המתן לסיום שיבוט

### שלב 3: התחבר לחשבונות

1. פתח את האפליקציה המשובטת
2. התחבר לחשבון שונה
3. הפעל הרשאות נדרשות

### שלב 4: הגדר ב-TikMatrix

1. פתח TikMatrix Desktop
2. נווט ל**מכשירים**
3. TikMatrix יזהה אוטומטית אפליקציות משובטות
4. הקצה משימות לחשבונות שונים

## זיהוי אוטומטי

TikMatrix מזהה אוטומטית:

```
com.zhiliaoapp.musically          → TikTok רשמי
com.zhiliaoapp.musically:p0       → TikTok Clone 1
com.zhiliaoapp.musically:p1       → TikTok Clone 2

com.instagram.android             → Instagram רשמי
com.instagram.android:p0          → Instagram Clone 1
com.instagram.android:p1          → Instagram Clone 2
```

## הגדרות מותאמות

### תצורת חשבונות

```json
{
  "device_id": "ABC123",
  "accounts": [
    {
      "app_package": "com.zhiliaoapp.musically",
      "username": "@account1",
      "type": "original"
    },
    {
      "app_package": "com.zhiliaoapp.musically:p0",
      "username": "@account2",
      "type": "clone"
    }
  ]
}
```

### ניהול משימות

הקצה משימות ספציפיות לחשבונות משובטים:

```bash
curl -X POST http://localhost:50809/api/v1/task \
  -d '{
    "usernames": ["@account1", "@account2"],
    "script_name": "post",
    "script_config": {...}
  }'
```

## שיטות עבודה מומלצות

### 1. הפרד חשבונות לפי מטרה

- חשבון 1: תוכן אישי
- חשבון 2: עסקי
- חשבון 3: שיווק

### 2. השתמש בפרוקסי שונים

```
חשבון רשמי → Proxy 1
Clone 1 → Proxy 2
Clone 2 → Proxy 3
```

### 3. תזמן משימות בנפרד

הימנע מהפעלת כל החשבונות בו-זמנית:

```
@account1 → 09:00
@account2 → 11:00
@account3 → 14:00
```

### 4. נטר ביצועים

בדוק באופן קבוע:
- שימוש בזיכרון
- חיבור רשת
- מצב חשבונות

### 5. גבה הגדרות

ייצא הגדרות באופן קבוע:

```
TikMatrix → הגדרות → ייצא תצורה
```

## בעיות נפוצות

### שיבוט לא מזוהה

**פתרון:**

1. הפעל מחדש את המכשיר
2. רענן רשימת מכשירים ב-TikMatrix
3. הגדר ידנית את שם החבילה

### אפליקציה משובטת קורסת

**פתרון:**

1. נקה מטמון כלי השיבוט
2. עדכן את כלי השיבוט
3. שבט מחדש את האפליקציה

### הרשאות חסרות

**פתרון:**

```
הגדרות → כלי שיבוט → הרשאות → אפשר הכל
הגדרות → אפליקציה משובטת → הרשאות → אפשר הכל
```

### ביצועים איטיים

**פתרון:**

- סגור אפליקציות אחרות
- הגדל זיכרון RAM
- השתמש במכשיר חזק יותר

## מגבלות

### מגבלות כלי שיבוט

| כלי | מספר שיבוטים | הערות |
|-----|--------------|-------|
| Parallel Space | 2-4 | תלוי במנוי |
| Dual Space | 2 | חינמי |
| Multiple Accounts | ללא הגבלה | מנוי פרימיום |

### מגבלות מכשיר

- **RAM:** מומלץ 4GB+ לכל שיבוט
- **אחסון:** 2GB+ לכל שיבוט
- **CPU:** Octa-core מומלץ

## טיפים מתקדמים

### אופטימיזציה

הגדר לביצועים:

```
כלי שיבוט → הגדרות
→ מצב ביצועים: גבוה
→ רזולוציה: בינונית
→ חיסכון בסוללה: כבוי
```

### אוטומציה

צור סקריפט להחלפת חשבונות:

```bash
#!/bin/bash
# החלף בין חשבונות
adb shell am start -n com.lbe.parallel.intl/.SwitchActivity
```

### ניטור

עקוב אחר מצב חשבונות:

```bash
# בדוק אפליקציות פעילות
adb shell "ps | grep musically"
```
