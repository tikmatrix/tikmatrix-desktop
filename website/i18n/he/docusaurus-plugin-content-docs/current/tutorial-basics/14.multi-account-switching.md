# החלפת חשבונות מרובים

TikTok ו-Instagram תומכים בכניסה למספר חשבונות באותו מכשיר, זוהי פונקציונליות מקורית של האפליקציה הרשמית. TikMatrix ו-IgMatrix מספקים פונקציונליות החלפת חשבונות אוטומטית, המאפשרת הרצת משימות אוטומטית על כל החשבונות המחוברים ללא התערבות ידנית.

## סקריפטים נתמכים

הסקריפטים הבאים תומכים בפונקציונליית החלפת חשבונות אוטומטית:

- **Profile (הצגת פרופיל)** — הצגת מידע על פרופיל החשבון
- **Account Warmup (חימום חשבון)** — חימום החשבון באמצעות פעילות טבעית
- **Post (פרסום תוכן)** — פרסום פוסטים
- **Mass DM (הודעות מרובות)** — שליחת הודעות פרטיות בכמות גדולה
- **Follow Back (מעקב חוזר)** — מעקב חוזר אחרי משתמשים שעוקבים אחריך
- **Unfollow All (הפסקת מעקב מכולם)** — הפסקת מעקב מכל החשבונות
- **Delete Post (מחיקת פוסט)** — מחיקת תוכן שפורסם
- **Boost Users (קידום משתמשים)** — הגברת האינטראקציה בדפי משתמש
- **Boost Posts (קידום פוסטים)** — הגברת האינטראקציה בפוסטים
- **Boost Lives (קידום שידורים חיים)** — הגברת האינטראקציה בשידורים חיים
- **Mass Comment (תגובות מרובות)** — פרסום תגובות בכמות גדולה
- **Boost Comments (קידום תגובות)** — הגברת האינטראקציה בתגובות
- **Super Marketing (שיווק-על)** — אוטומציה מתקדמת לשיווק

## הגדרה ראשונית: התאמת חשבונות

לפני שימוש בפונקציונליות החלפת חשבונות מרובים, יש תחילה להתאים את החשבונות מהמכשיר.

1. פתח את הלשונית **סקריפטים** בניווט הצידי השמאלי.
2. מצא והפעל את הסקריפט **התאמת חשבונות**.
3. הסקריפט יסרוק את החשבונות המחוברים במכשיר ויוסיף אותם אוטומטית לרשימת החשבונות.
4. לאחר השלמת הסקריפט, לחץ על כפתור **חשבונות** כדי לוודא שהחשבונות מהמכשיר נוספו בהצלחה.

## בדיקת החלפת חשבונות

לאחר הרצת הסקריפט **התאמת חשבונות**, ניתן לבדוק את פונקציונליות החלפת החשבונות:

1. בלשונית הסקריפטים, הפעל את הסקריפט **החלפת חשבון**.
2. לסקריפט זה אין שום לוגיקה נוספת — מטרתו היחידה היא לוודא שתהליך החלפת החשבונות עובד כהלכה.
3. אם הסקריפט מסתיים בהצלחה, פונקציונליית החלפת החשבונות מוכנה לשימוש.

## שימוש במצב חשבונות מרובים

לאחר השלמת ההגדרה הראשונית והבדיקה:

1. בעת הרצת כל סקריפט נתמך, הפעל את **מצב חשבונות מרובים** בהגדרות הסקריפט.
2. המערכת תיצור משימה נפרדת עבור כל חשבון ברשימת החשבונות.
3. המשימות יעמדו בתור ויתבצעו ברצף — המערכת תעבור אוטומטית לחשבון היעד ותריץ את המשימה האוטומטית.
4. אין צורך בשום התערבות ידנית.

## מתי להריץ שוב התאמת חשבונות

בדרך כלל צריך להריץ את הסקריפטים **התאמת חשבונות** ו-**החלפת חשבון** רק פעם אחת בהגדרה הראשונית. עם זאת, יש להריץ שוב את **התאמת חשבונות** במקרים הבאים:

- התחברת לחשבון חדש במכשיר
- התנתקת מחשבון קיים
- החשבונות במכשיר השתנו

שלבים לרענון רשימת החשבונות:

1. בדף **חשבונות**, נקה ידנית את רשימת החשבונות.
2. הפעל מחדש את הסקריפט **התאמת חשבונות** כדי לזהות את החשבונות הנוכחיים במכשיר.

---

פונקציונליית החלפת חשבונות מרובים מבטלת את הצורך בהחלפה ידנית של חשבונות, ומאפשרת לך להרחיב בקלות את האוטומציה על פני כל החשבונות במכשיר בודד.
