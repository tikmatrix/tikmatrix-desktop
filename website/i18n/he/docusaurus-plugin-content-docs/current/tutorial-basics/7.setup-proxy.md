# הגדרת פרוקסי

ניתן להגדיר כתובת IP פרוקסי עצמאית עבור כל טלפון.

> **כתב ויתור:** כל האתרים המומלצים בדף זה מקורם באינטרנט ואינם קשורים לאתר זה. אנא אמת את תקפותם בעצמך לפני השימוש. איננו מספקים כל אחריות.

## 1. הגדרת פרוקסי VPS למגורים (אם רכשת פרוקסי IP, ניתן לדלג על שלב זה)

1. רכוש VPS עם פרוקסי IP למגורים מהמדינה היעד (עדיף IP מקורי ו-VPS עם ISP כפול).
2. השתמש ב-[https://github.com/MHSanaei/3x-ui](https://github.com/MHSanaei/3x-ui) כדי להגדיר במהירות VPN על ה-VPS:

    ```shell
    bash <(curl -Ls https://raw.githubusercontent.com/MHSanaei/3x-ui/refs/tags/v2.6.0/install.sh)
    # עקוב אחר ההנחיות כדי להשלים את הגדרת ה-VPN.
    ```

3. לאחר השלמת ההתקנה, גש לממשק האינטרנט של x-ui. ניתן למצוא את כתובת ה-URL ופרטי הכניסה בפלט המסוף לאחר ההתקנה.
4. ב-x-ui צור משתמש חדש, הגדר את הפרוטוקול ל-`SOCKS5`, והגדר פורט והגדרות אחרות לפי הצורך.
5. לאחר יצירת המשתמש, תראה את כתובת ה-IP והפורט של הפרוקסי בממשק x-ui.
6. השתמש בכתובת ה-IP והפורט של הפרוקסי באפליקציית NekoBox בטלפון.

> ספקי VPS מומלצים

- [לחץ כאן](/advertising)

## 2. רכישת פרוקסי IP למגורים (אם הגדרת VPS, ניתן לדלג על שלב זה)

ניתן גם לרכוש ישירות פרוקסי IP למגורים ללא צורך בהגדרת VPS.

> ספקי פרוקסי מומלצים

- [לחץ כאן](/advertising)

> כלי בדיקת IP

- [ipinfo](https://ipinfo.io) — בדוק אם ה-IP הוא IP למגורים.
- [scamalytics](https://scamalytics.com/ip) — בדוק את רמת הסיכון של ה-IP.

## 3. הפעלת פרוקסי בטלפון

1. הורד והתקן את לקוח [NekoBoxForAndroid](https://github.com/MatsuriDayo/NekoBoxForAndroid) בטלפון.
2. פתח את אפליקציית NekoBox, לחץ על כפתור `+` כדי להוסיף פרוקסי חדש.
3. בחר `SOCKS5` או סוג פרוקסי אחר.
4. הזן את כתובת ה-IP והפורט של הפרוקסי, ואז לחץ על `שמור`.
5. לחץ על כפתור `התחל` כדי להפעיל את הפרוקסי.
6. לאחר הפעלת הפרוקסי, ניתן לבדוק את מצב החיבור באפליקציית NekoBox.

### הערה למשתמשים בסין

אם אתה נמצא בסין, פרוטוקול `SOCKS5` עלול שלא לעבוד כהלכה. תצטרך להוסיף פרוקסי מוצפן (`VLESS`) כפרוקסי קדמי. ניתן להשתמש בכלים הבאים:

1. הוסף פרוקסי `VLESS` באפליקציית NekoBox.
2. הוסף פרוקסי `SOCKS5` באפליקציית NekoBox.
3. הוסף פרוקסי `שרשרת` באפליקציית NekoBox. בחר את פרוקסי ה-`VLESS` כפרוקסי קדמי ואת פרוקסי ה-`SOCKS5` כפרוקסי אחורי.
