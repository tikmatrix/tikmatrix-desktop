# הגדרות משימה

מדריך מקיף להגדרת ותצורת משימות ב-TikMatrix.

## סוגי משימות

### פרסום (Post)

פרסם תוכן אוטומטי:

- 📹 סרטוני וידאו
- 🖼️ תמונות
- 📝 כיתובים מותאמים

[למד עוד →](../api/post-script)

### מעקב (Follow)

עקוב אחרי משתמשים:

- 👥 מעקב אוטומטי
- 🎯 מעקב ממוקד
- 📋 ייבוא רשימות

[למד עוד →](../api/follow-script)

### ביטול מעקב (Unfollow)

בטל מעקב אחרי משתמשים:

- 🗑️ ניקוי עוקבים
- 🔄 מעקב/ביטול מעקב מחזורי
- 📊 ניהול יחס עוקבים

[למד עוד →](../api/unfollow-script)

### תגובות (Comment)

הגב על פוסטים:

- 💬 תגובות אוטומטיות
- 🎲 תגובות אקראיות
- 🤖 תגובות AI

[למד עוד →](../api/comment-script)

### חימום חשבון

בנה אמינות:

- 🔥 אינטראקציה טבעית
- 👁️ צפייה בסרטונים
- ❤️ לייקים, שמירות

[למד עוד →](../api/account-warmup-script)

## יצירת משימה

### דרך ממשק משתמש

```
TikMatrix → משימות → משימה חדשה
```

1. **בחר סוג משימה**
2. **בחר מכשירים/חשבונות**
3. **הגדר פרמטרים**
4. **תזמן (אופציונלי)**
5. **צור משימה**

### דרך API

```bash
curl -X POST http://localhost:50809/api/v1/task \
  -H "Content-Type: application/json" \
  -d '{
    "serials": ["device_1"],
    "script_name": "post",
    "script_config": {...}
  }'
```

[מדריך API →](../api/local-api)

## תזמון משימות

### תזמון בסיסי

הגדר זמן התחלה:

```
start_time: "14:30"
```

משימה תתחיל ב-14:30 באותו יום.

### תזמון חוזר

צור משימות חוזרות:

```json
{
  "schedule": {
    "type": "daily",
    "time": "09:00",
    "days": ["monday", "wednesday", "friday"]
  }
}
```

### דפוסי תזמון

#### יומי

```
כל יום בשעה 10:00
```

#### שבועי

```
כל יום שני וחמישי בשעה 15:00
```

#### התאמה אישית

```
מספר פעמים ביום עם פיזור אקראי
```

## ריבוי חשבונות

### מצב מכשיר אחד

```json
{
  "serials": ["device_1"],
  "enable_multi_account": false
}
```

רק החשבון הפעיל יבצע משימה.

### מצב ריבוי חשבונות

```json
{
  "serials": ["device_1"],
  "enable_multi_account": true
}
```

כל החשבונות במכשיר יבצעו משימה.

### בחירת חשבונות ספציפיים

```json
{
  "usernames": ["@account1", "@account2", "@account3"]
}
```

רק חשבונות אלה יבצעו משימה.

## אפשרויות תצורה

### בררות מחדל לכל המשימות

```
הגדרות → משימות → בררות מחדל
```

- ⏱️ **Timeout:** זמן מקסימלי למשימה
- 🔄 **ניסיון חוזר:** מספר נסיונות
- ⏰ **עיכוב:** השהיה בין נסיונות
- 📊 **לוגים:** רמת לוג

### הגדרות ספציפיות למשימה

עבור כל משימה:

- ⏲️ **זמן התחלה**
- 🔁 **חזרה אוטומטית**
- 🚫 **עצירה בשגיאה**
- 📢 **התראות**

### הגדרות מתקדמות

```json
{
  "retry_on_failure": true,
  "max_retries": 3,
  "retry_delay": 300,
  "stop_on_error": false,
  "priority": "normal",
  "timeout": 600
}
```

## ניהול משימות

### מצבי משימה

| מצב | תיאור | צבע |
|------|--------|-----|
| ⏳ **Pending** | ממתינה לביצוע | כחול |
| ▶️ **Running** | בביצוע | ירוק |
| ✅ **Completed** | הושלמה | ירוק |
| ❌ **Failed** | נכשלה | אדום |
| ⏸️ **Paused** | מושהית | כתום |
| 🚫 **Cancelled** | בוטלה | אפור |

### פעולות משימה

```
TikMatrix → משימות → [בחר משימה]
```

- ▶️ **הפעל** - התחל משימה
- ⏸️ **השהה** - השהה ביצוע
- ⏹️ **עצור** - עצור לחלוטין
- 🔄 **נסה שוב** - נסה שוב משימה כושלת
- 📝 **ערוך** - שנה הגדרות
- 🗑️ **מחק** - מחק משימה

### סינון ומיון

```
משימות → סינון
```

**סינון לפי:**
- מצב
- סוג
- מכשיר
- תאריך
- חשבון

**מיון לפי:**
- תאריך יצירה
- זמן התחלה
- עדיפות
- מצב

## אופטימיזציה

### שיטות עבודה מומלצות

1. **אל תבצע יותר מדי משימות בו-זמנית**
   ```
   מומלץ: 1-2 משימות פעילות לכל מכשיר
   ```

2. **השתמש בתזמון חכם**
   ```
   פזר משימות לאורך היום
   הימנע משעות עומס
   ```

3. **נטר ביצוע**
   ```
   בדוק משימות כושלות
   שפר תצורה לפי תוצאות
   ```

4. **גבה תצורות**
   ```
   ייצא משימות חוזרות
   שמור תבניות
   ```

### ניהול עדיפויות

הגדר עדיפות משימה:

| עדיפות | תיאור | שימוש |
|---------|--------|-------|
| 🔴 **High** | ביצוע מיידי | דחוף |
| 🟡 **Normal** | ביצוע רגיל | רגיל |
| 🟢 **Low** | ביצוע כשיש פנאי | לא דחוף |


**Instagram:**
```

```

```
משימות → [בחר משימה] → יומנים
```

## אוטומציה מתקדמת

### תנאים

הפעל משימות על סמך תנאים:

```json
{
  "conditions": {
    "if_last_task_success": true,
    "min_followers": 100,
    "account_age_days": 7
  }
}
```

### שרשרת משימות

צור זרימת עבודה:

```
משימה 1 (חימום) → משימה 2 (מעקב) → משימה 3 (פרסום)
```

### התראות

הגדר התראות:

```json
{
  "notifications": {
    "on_completion": true,
    "on_failure": true,
    "on_start": false,
    "channels": ["email", "webhook"]
  }
}
```

## פתרון בעיות

### משימות לא מתחילות

**בדוק:**
- ✓ מכשיר מחובר
- ✓ חשבון מחובר
- ✓ תצורה נכונה

### משימות נכשלות חוזרות

**פתרון:**

1. בדוק יומנים
2. ווידא פרמטרים
3. נסה ידנית
4. התאם הגדרות

[מדריך פתרון בעיות →](../troubleshooting/1.task_failed)

### ביצועים איטיים

**שפר:**

- הפחת משימות במקביל

- שדרג חומרה

## שאלות נפוצות

**ש: כמה משימות אפשר להפעיל בו-זמנית?**  
ת: תלוי בחומרה. מומלץ 1-2 למכשיר, 20 סה"כ.

**ש: האם משימות חוזרות אוטומטיות?**  
ת: כן, אם הגדרת תזמון חוזר.

**ש: מה קורה אם משימה נכשלת?**  
ת: ניתן להגדיר ניסיון חוזר אוטומטי או ידני.

**ש: האם אפשר לערוך משימה פעילה?**  
ת: רק לאחר השהיה או עצירה.

4. 🔄 **אוטומט חוזר** - הגדר ושכח
