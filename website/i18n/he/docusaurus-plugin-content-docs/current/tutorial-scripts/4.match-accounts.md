# התאמת חשבונות

סקריפט התאמת חשבונות משמש לזיהוי חשבונות מהטלפון והוספתם לרשימת החשבונות.

## שלבים

1. בחר את המכשירים להתאמת חשבונות.
2. לחץ על כפתור `סקריפטים` - `התאמת חשבונות`.
3. לחץ על כפתור `התחל` להתחלת משימת התאמת החשבונות.
4. הסקריפט יזהה אוטומטית חשבונות מהטלפון ויוסיף אותם לרשימת החשבונות.

## הערות

* סקריפט התאמת החשבונות אינו מצליח ב-100%, אם חלק מהחשבונות לא זוהו, ייתכן שתצטרך להריץ את הסקריפט מספר פעמים.
