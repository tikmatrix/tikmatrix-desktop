# מחיקת פוסטים

סקריפט מחיקת פוסטים משמש למחיקת פוסטים בהתאם למספר הצפיות.

## שלבים

1. בחר את המכשירים להרצת הסקריפט.
2. לחץ על `סקריפטים` > `מחיקת פוסטים`.
3. הגדר את הגדרות המשימה:
    - **מספר צפיות מקסימלי**: הזן את מספר הצפיות המקסימלי של הפוסטים שברצונך למחוק.
4. לחץ על `התחלת סקריפט` כדי להתחיל בביצוע.

## הערה

- סקריפט מחיקת הפוסטים עשוי לא להצליח תמיד; נסה שוב משימות שנכשלו במידת הצורך.
