# פרופיל

סקריפט הפרופיל משמש להגדרת פרופיל החשבון.

## שלבים

1. בחר את המכשירים להגדרת הפרופיל.
2. לחץ על כפתור `סקריפטים` - `פרופיל`.
3. הגדר את הגדרות המשימה:
    - **כינוי**: כינוי אחד בכל שורה.
    - **שם משתמש**: שם משתמש אחד בכל שורה.
    - **ביוגרפיה**: ביוגרפיה אחת בכל שורה (תומך באזכור משתמשים ושורות חדשות).
      - כדי להוסיף שורה חדשה לביוגרפיה, השתמש ב-`\n` בטקסט. לדוגמה: `שורה ראשונה\nשורה שנייה\nשורה שלישית`
    - **נתיב תמונת פרופיל**: נתיב לתיקיית קבצי תמונות הפרופיל.
    - **סדר בחירה**: בחר כיצד לבחור תוכן לכל משימה:
      - **אקראי**: בחר באקראי אפשרויות זמינות לכל משימה (ברירת מחדל).
      - **סדר**: בחר תוכן לפי סדר בהתאם למונה המשימות. השתמש במצב זה כדי להבטיח שכל חשבון יקבל מידע ייחודי.
4. לחץ על כפתור `התחל סקריפט` להתחלת המשימה.

## הסבר על סדר הבחירה

### מצב אקראי

- כל משימה בוחרת תוכן באקראי (כינוי, שם משתמש, ביוגרפיה, תמונת פרופיל)
- אותו תוכן עשוי להיות משומש חוזר בין חשבונות שונים
- מתאים לתרחישים שאינם דורשים ייחודיות תוכן

### מצב סדר

- בוחר תוכן לפי סדר: המשימה ה-1 משתמשת בפריט ה-1, המשימה ה-2 משתמשת בפריט ה-2, וכן הלאה
- כאשר כל הפריטים נוצלו, חוזר לתחילה
- **הערה חשובה**: כדי להבטיח שכל חשבון יקבל מידע ייחודי, וודא שמספר הפריטים גדול ממספר החשבונות הכולל בהרצה אצווה אחת
  - דוגמה: אם מריצים 100 חשבונות, ספק לפחות 100 כינויים, שמות משתמש, ביוגרפיות ותמונות פרופיל שונות

## הערות

- מומלץ לנקות את האלבום לפני הגדרת הפרופיל, מכיוון שהסקריפט ישתמש בתמונה הראשונה באלבום כתמונת פרופיל.
