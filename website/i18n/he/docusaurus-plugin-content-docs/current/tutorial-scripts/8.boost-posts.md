# העלאת פוסטים

:::warning הודעת הפסקה
תכונת סקריפט זה שולבה ב-[סקריפט Super Marketing](./14.super-marketing.md). אנא השתמש בעדיפות בסקריפט "Super Marketing" לקבלת יכולות העלאת פוסטים חזקות וגמישות יותר.
:::

תכונת העלאת פוסטים יכולה להגדיל את כמות האינטראקציה של פוסטים באמצעות פעולות לייק, שיתוף, שמירה, ריטוויט וצפייה באצווה. המערכת תנסה לחקות התנהגות משתמש אמיתית כדי להפחית את הסיכון לזיהוי, אך בעת השימוש עדיין יש לעקוב אחר כללי הפלטפורמה ולשלוט במידתיות בתדירות.

## שימוש בתכונת העלאת פוסטים בסקריפט "Super Marketing"

1. פתח סקריפט: בחר "Super Marketing".
2. בחר סוג מקור נתונים כ-"רשימת קישורי פוסטים".
3. הדבק באצווה קישורי פוסטים או ייבא מקובץ טקסט; לאחר הייבוא תוכל לבחור "הוסף" או "החלף" את הרשימה הנוכחית.
4. בחר אסטרטגיית שימוש בנתונים:

    - אסטרטגיית מאגר משותף: כל חשבונות ה-bot יבצעו אינטראקציה עם אותם פוסטים.

    - אסטרטגיית צריכה חד-פעמית: כל חשבון bot יבצע אינטראקציה רק עם פוסטים שונים פעם אחת, ויבטיח שכל פוסט מטופל פעם אחת בלבד על ידי חשבון אחד.

5. ב-"פעולות פוסט — מספר חזרות" הגדר את מספר האינטראקציות לכל פוסט. אם כולל תגובות, מומלץ להגדיר את מספר התגובות ל-1, כדי להימנע מהופעת תגובות כפולות.
6. ב-"פעולות פוסט" בחר את סוגי האינטראקציה הנדרשים: לייק, שיתוף, ריטוויט, שמירה, תגובה, צפייה בסרטון וכו'.

     אם בחרת "תגובה", אנא בצע את ההגדרות הבאות בהגדרות התגובה:

     - תבנית תגובה מוכנה מראש: תוכל להדביק טקסט תגובה מוכן מראש מרובה שורות או פסקאות (כל שורה או פסקה נחשבת כתגובה מועמדת אחת).

     - סדר שימוש בטקסט: בחר "אקראי" או "לפי סדר".
         - אקראי: כל פעם בחר באקראי טקסט אחד מהטקסטים המועמדים.
         - לפי סדר: השתמש לפי סדר ההדבקה/הייבוא (מתאים לאסטרטגיית דה-דופליקציה שניתן לשחזר).

     - הכנסת אמוג'י: ניתן להפעיל הכנסה אוטומטית של אמוג'י בתגובה כדי להגביר את הטבעיות.

     - יצירת תגובות בזמן אמת (אופציונלי, ChatGPT API):
         - לאחר ההפעלה, הסקריפט יכול לקרוא ל-API דמוי ChatGPT, ליצור טקסט תגובה ייחודי ורלוונטי להקשר בזמן אמת בהתאם לכותרת/תיאור הפוסט. שיטה זו יכולה להפחית משמעותית את הסיכון לתגובות כפולות, מתאימה לתרחישים הדורשים כמות גדולה של תגובות ייחודיות.
         - הערות: שימוש ביצירה בזמן אמת דורש מתן API Key זמין, ושים לב למגבלות קצב, עלויות וציות לפרטיות (אל תשלח נתונים רגישים לשירותי צד שלישי).

     - שתי דרכים נפוצות להבטחת תגובות ייחודיות:
         1. השתמש ב-"לפי סדר" לטעינה וחילוף של תבניות תגובה מוכנות מראש (להבטיח שכל תבנית משומשת רק פעם אחת עד לסיום).
         2. השתמש ב-ChatGPT API ליצירת כל תגובה בזמן אמת (יצירה דינמית המבוססת על כותרת/תוכן הפוסט, באופן טבעי קל יותר להבטיח ייחודיות).

     - המלצה: בדוק תחילה את אסטרטגיית התגובה עם מספר קטן של פוסטים, בדוק את איכות התגובה ואת המצב של טיפול הפלטפורמה; אם משתמש ביצירה בזמן אמת, אנא עקוב אחר כמות הקריאות ל-API והעלות.
7. הגדר פרמטרי סקריפט אחרים (כמו מספר מקביליות, עיכובים, מאגר חשבונות וכו'), אשר שאין שגיאות ואז הפעל את הסקריפט.

## המלצות שימוש והערות

- מומלץ לבדוק תחילה על מספר קטן של פוסטים, אשר שהאסטרטגיה והפרמטרים לא יפעילו בקרת סיכונים של הפלטפורמה.
- תוכן התגובות צריך להיות מגוון ולשמור על טבעיות, הימנע מהעתקת והדבקת אותו טקסט.
- הגדר עיכובים ומקביליות סבירים, חקה התנהגות ידנית כדי להפחית סיכון.
- בעת הרצה ארוכת טווח אנא בדוק באופן קבוע את בריאות החשבונות ולוגים של משימות, והתאם את האסטרטגיה לפי המצב.
