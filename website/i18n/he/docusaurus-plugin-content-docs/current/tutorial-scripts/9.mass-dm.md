# הודעות המוניות

:::warning הודעת הפסקה
תכונה זו שולבה ב-[סקריפט Super Marketing](./14.super-marketing.md). אנא השתמש בעדיפות בסקריפט "Super Marketing" לקבלת יכולות הודעות המוניות וניהול הודעות חזקות וגמישות יותר.
:::

תכונת הודעות המוניות משמשת ליצירת קשר עם משתמשי יעד באמצעות שליחת הודעות פרטיות (DM) באצווה, מתאימה לתרחישים כמו קידום מכירות, שירות לקוחות חוזר והשבת משתמשים. המערכת תומכת בשני מקורות הודעות: תבנית מוכנה מראש ויצירה בזמן אמת, ותנסה לחקות שליחה ידנית כדי להפחית את הסיכון לזיהוי.

## שימוש בתכונת הודעות המוניות בסקריפט "Super Marketing"

1. פתח סקריפט: בחר "Super Marketing".
2. בחר סוג מקור נתונים כ-"רשימת חשבונות משתמשים (רשימת יעדים)".
3. הדבק באצווה שמות משתמשי יעד או ייבא מקובץ טקסט (אחד לשורה); לאחר הייבוא תוכל לבחור "הוסף" או "החלף".
4. בחר אסטרטגיית שימוש בנתונים:

    - אסטרטגיית מאגר משותף: כל חשבונות ה-bot ישלחו הודעות פרטיות לאותה רשימת יעדים.

    - אסטרטגיית צריכה חד-פעמית: כל חשבון bot יעבד רק יעדים שונים, ויבטיח שכל יעד מקבל הודעה פעם אחת בלבד מחשבון אחד.

5. הגדר את מספר שליחות ההודעות לכל יעד (בדרך כלל 1 פעם, כדי להימנע מהטרדה חוזרת).
6. ב-"הגדרות הודעה" הגדר את תוכן ההודעה הפרטית ואסטרטגיית השליחה:

    - תבנית הודעה מוכנה מראש: תוכל להדביק טקסט הודעה פרטית מוכן מראש מרובה שורות או פסקאות (כל שורה או פסקה נחשבת כהודעה מועמדת אחת).

    - סדר שימוש בטקסט: בחר "אקראי" או "לפי סדר".

        - אקראי: כל פעם בחר באקראי תבנית אחת לשליחה.

        - לפי סדר: השתמש בתבניות לפי סדר ההדבקה/הייבוא בחילוף (מתאים לאסטרטגיית דה-דופליקציה שניתן לשחזר).

    - הכנסת אמוג'י: ניתן להפעיל הכנסה אוטומטית של אמוג'י בהודעה כדי להגביר את הטבעיות.

    - יצירת הודעות בזמן אמת (אופציונלי, ChatGPT API):

        - לאחר ההפעלה, הסקריפט יכול לקרוא ל-ChatGPT או API דומה, ליצור הודעות מותאמות אישית בזמן אמת בהתאם לשם משתמש היעד או הקשר זמין, מתאים לתרחישים הדורשים כמות גדולה של הודעות ייחודיות.

        - הערה: הפעלת יצירה בזמן אמת דורשת מתן API Key תקף, ושים לב למגבלות קצב, עלויות וציות לפרטיות (הימנע משליחת נתונים רגישים).

    - שתי דרכים נפוצות להבטחת הודעות ייחודיות:

        1. השתמש ב-"לפי סדר" לחילוף תבניות מוכנות מראש (כל תבנית משומשת רק פעם אחת עד לסיום).

        2. השתמש ב-ChatGPT ליצירת כל הודעה בזמן אמת (יצירה דינמית המבוססת על מידע היעד, ייחודיות גבוהה יותר).

7. הגדר את הדרך לפתיחת משתמש (במידת הצורך לראות את פרופיל המשתמש): בחר "חיפוש משתמש" או "פתיחה ישירה של פרופיל (Direct Open Profile)".
8. הגדר מרווח זמן בין משימות (מינימום/מקסימום, ביחידות: דקות) ומקביליות, מאגר חשבונות ופרמטרים אחרים, אשר שאין שגיאות ואז הפעל את הסקריפט.

:::tip לגבי מרווח זמן בין משימות
מומלץ להגדיר מרווח מינימום/מקסימום ולהפעיל מרווח אקראי, כדי להימנע מכל המכשירים מבצעים בו-זמנית שגורם לשיא חריג, ובכך להפחית את הסיכון להיות מזוהה או מוגבל על ידי הפלטפורמה.
:::

## המלצות שימוש והערות

- תוכן ההודעה הפרטית צריך להיות תואם ומגוון, הימנע משליחת כמות גדולה של אותו תוכן בזמן קצר.
- אם משתמש ביצירה בזמן אמת, אנא עקוב בקפדנות אחר כמות הקריאות ל-API והעלות, וודא שלא לדלוף מידע רגיש כלפי חוץ.
- משימות עשויות להיכשל; במקרה של כישלון נסה שוב או ראה את הלוגים, המערכת תדלג אוטומטית על יעדים שכבר הושלמו.
