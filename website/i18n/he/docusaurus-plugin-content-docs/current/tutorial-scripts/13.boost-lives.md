# קידום שידורים חיים

סקריפט קידום שידורים חיים מאפשר לך לשלוט בחשבונות חוות לצפות ובאופציה ללייק או להגיב בשידורים חיים יעד. ניתן להגדיר משך צפייה ומרווחי לייק/תגובה כדי להתאים אישית את התנהגות הקידום.

## שלבים

1. בחר את המכשירים להרצת הסקריפט.
2. לחץ על `סקריפטים` > `קידום שידורים חיים`.
3. הגדר את הגדרות המשימה:
    - **שם משתמש**: הזן שמות משתמש יעד, אחד בכל שורה.
    - **שיטת כניסה**: בחר שיטת **חיפוש** או **ישיר**.
    - **אפשרויות קידום**: הפעל לייק ו/או תגובה.
    - **הצטרפות למועדון מעריצים**: הפעל פונקציית הצטרפות למועדון המעריצים של השדר (אם זמין).
    - **צ'ק-אין יומי**: הפעל פעילות צ'ק-אין יומי במהלך השידור החי.
    - **מרווח לייק**: הגדר את המרווח בין כל פעולת לייק.
    - **מספר קליקים של לייק**: הגדר את מספר הקליקים על כפתור הלייק.
    - **מרווח תגובה**: הגדר את המרווח בין כל פעולת תגובה.
    - **תוכן תגובה**: הזן תגובה אחת בכל שורה.
    - **הוספת אמוג'י**: בחר האם להוסיף אמוג'י בתגובות.
    - **סדר תגובות**: בחר סידור אקראי או סדרתי של תגובות.
    - **מספר תגובות לחשבון**: הגדר את מספר התגובות שמפורסמות על ידי כל חשבון.
    - **משך צפייה**: הגדר את הזמן להישאר בשידור החי.
    - **מרווח משימה**: הגדר את המרווח בין כל משימה.
4. לחץ על `התחלת סקריפט` כדי להתחיל בביצוע.

## הערות

- סקריפט קידום שידורים חיים עשוי לא להצליח תמיד; נסה שוב משימות שנכשלו במידת הצורך.
