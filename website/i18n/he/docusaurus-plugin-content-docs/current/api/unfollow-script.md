עמוד זה מתאר את פרמטרי התצורה עבור סקריפט `unfollow` המשמש ליצירת משימות.

## סקירה כללית

סקריפט `unfollow` משמש לביטול מעקב אוטומטי אחרי משתמשים ב-TikTok או Instagram.

## פרמטרים

| פרמטר | סוג | נדרש | תיאור |
|------|------|------|--------|
| target_users | string[] | כן* | מערך שמות משתמש יעד |
| target_user | string | כן* | שם משתמש יחיד או מרובים |
| access_method | string | לא | direct או search |

## שיטות גישה

### גישה ישירה (`direct`)

- מהיר ואמין יותר
- מומלץ לרוב המקרים

### גישה דרך חיפוש (`search`)

- עובד כאשר גישה ישירה חסומה
- עשוי להיות פחות מדויק

## שיטות עבודה מומלצות

1. השתמש בתזמון start_time
2. העדף גישה ישירה
3. קבוצות סבירות
