עמוד זה מתאר את פרמטרי התצורה עבור סקריפט `follow` המשמש ליצירת משימות.

## סקירה כללית

סקריפט `follow` משמש למעקב אוטומטי אחרי משתמשים ב-TikTok או Instagram.

## פרמטרים

| פרמטר | סוג | נדרש | תיאור |
|------|------|------|--------|
| target_users | string[] | כן* | מערך שמות משתמש יעד |
| target_user | string | כן* | שם משתמש יחיד או מרובים |
| access_method | string | לא | direct או search |

## שיטות גישה

### גישה ישירה (`direct`)

- פתיחת פרופיל משתמש דרך URL
- מהיר ואמין יותר
- מומלץ לרוב המקרים

### גישה דרך חיפוש (`search`)

- ניווט לחיפוש והזנת שם משתמש
- עובד כאשר גישה ישירה חסומה
