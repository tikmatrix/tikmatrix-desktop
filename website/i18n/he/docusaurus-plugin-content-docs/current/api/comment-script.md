עמוד זה מתאר את פרמטרי התצורה עבור סקריפט `comment` המשמש ליצירת משימות.

## סקירה כללית

סקריפט `comment` משמש לפרסום תגובות אוטומטי בפוסטים של TikTok או Instagram. כאשר אתה מספק מספר כתובות URL של פוסטי יעד דרך ה-API, **נוצרת משימה אחת עבור כל כתובת URL של פוסט יעד**.

## פרמטרים עיקריים

| פרמטר | סוג | נדרש | תיאור |
|-----------|------|----------|-------------|
| target_post_urls | string[] | כן* | מערך כתובות URL של פוסטי יעד |
| target_post_url | string | כן* | כתובת URL יחידה או מספר כתובות URL |
| comment_content | string | כן | תוכן טקסט התגובה |
| comment_order | string | לא | אקראי או רציף |
| insert_emoji | boolean | לא | הוספת אמוג'י אקראי |
| comment_image_path | string | לא | נתיב תמונה (TikTok בלבד) |

## שיטות עבודה מומלצות

1. גוון את התגובות שלך
2. השתמש במצב רציף למגוון
3. תזמן משימות בחוכמה
