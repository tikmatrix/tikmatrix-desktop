# שגיאת הפעלת תוכנה

מדריך לפתרון בעיות כאשר TikMatrix לא מופעל כראוי.

## סיבות נפוצות

### 1. תלויות חסרות

**Windows:**
- Visual C++ Redistributable חסר
- .NET Framework מיושן

**פתרון:**
```
הורד והתקן:
- Visual C++ 2015-2022 Redistributable
- .NET Framework 4.8 ומעלה
```

**macOS:**
- חסר Xcode Command Line Tools

**פתרון:**
```bash
xcode-select --install
```

**Linux:**
```bash
sudo apt-get install libgtk-3-0 libnotify4 libnss3
```

### 2. הרשאות חסרות

**תסמינים:**
- "Access Denied"
- "Permission Error"

**פתרון:**

**Windows:**
```
לחצן ימני על TikMatrix → הפעל כמנהל
```

**macOS:**
```bash
sudo chmod +x /Applications/TikMatrix.app/Contents/MacOS/TikMatrix
```

**Linux:**
```bash
chmod +x tikmatrix-desktop.AppImage
```

### 3. קובץ תצורה פגום

**תסמינים:**
- התוכנה קופאת בהפעלה
- לולאת הפעלה מחדש

**פתרון:**

הסר קובצי תצורה:

**Windows:**
```
%APPDATA%\TikMatrix\config.json
```

**macOS:**
```
~/Library/Application Support/TikMatrix/config.json
```

**Linux:**
```
~/.config/TikMatrix/config.json
```

### 4. חסימת חומת אש/אנטי-וירוס

**תסמינים:**
- התוכנה נפתחת ונסגרת מיד
- אין התראות שגיאה

**פתרון:**

1. הוסף TikMatrix לחריגים
2. הרשה חיבורים רשת
3. השבת זמנית את האנטי-וירוס

### 5. יציאה תפוסה

**שגיאה:**
```
Error: Port 50809 already in use
```

**פתרון:**

**Windows:**
```cmd
netstat -ano | findstr :50809
taskkill /PID <PID> /F
```

**macOS/Linux:**
```bash
lsof -ti:50809 | xargs kill -9
```

## אבחון שלב אחר שלב

### שלב 1: בדוק יומנים

**מיקומי יומנים:**

**Windows:**
```
%APPDATA%\TikMatrix\logs\
```

**macOS:**
```
~/Library/Logs/TikMatrix/
```

**Linux:**
```
~/.config/TikMatrix/logs/
```

פתח את הקובץ האחרון וחפש שגיאות.

### שלב 2: הפעל במצב מפורט

**Windows:**
```cmd
TikMatrix.exe --verbose
```

**macOS:**
```bash
/Applications/TikMatrix.app/Contents/MacOS/TikMatrix --verbose
```

**Linux:**
```bash
./tikmatrix-desktop.AppImage --verbose
```

### שלב 3: אתחל הגדרות

גבה ולאחר מכן מחק:

**Windows:**
```
%APPDATA%\TikMatrix\
```

**macOS:**
```
~/Library/Application Support/TikMatrix/
```

**Linux:**
```
~/.config/TikMatrix/
```

## שגיאות ספציפיות

### "Missing DLL" (Windows)

```
Error: The code execution cannot proceed because vcruntime140.dll was not found
```

**פתרון:**
התקן [Visual C++ Redistributable](https://aka.ms/vs/17/release/vc_redist.x64.exe)

### "App is damaged" (macOS)

```
"TikMatrix.app" is damaged and can't be opened
```

**פתרון:**
```bash
xattr -cr /Applications/TikMatrix.app
sudo spctl --master-disable
```

### "Permission denied" (Linux)

```
bash: ./tikmatrix-desktop: Permission denied
```

**פתרון:**
```bash
chmod +x tikmatrix-desktop.AppImage
```

### "GLIBC version" (Linux)

```
version 'GLIBC_2.29' not found
```

**פתרון:**
```bash
# עדכן מערכת
sudo apt-get update && sudo apt-get upgrade
# או השתמש בגרסה ישנה יותר של TikMatrix
```

## פתרונות מתקדמים

### אתחל Agent

ה-Agent הוא השירות המקומי:

**Windows:**
```cmd
taskkill /F /IM tikmatrix-agent.exe
```

**macOS/Linux:**
```bash
killall tikmatrix-agent
```

### בדוק סכסוכי יציאות

```bash
# בדוק יציאה 50809
netstat -an | grep 50809

# אם תפוסה, שנה יציאה בתצורה
```

### התקנה מחדש נקייה

1. הסר התקנה מלאה
2. מחק תיקיות תצורה
3. הפעל מחדש את המחשב
4. התקן גרסה עדכנית

## טיפים למניעה

1. **עדכן באופן קבוע**
   - בדוק עדכונים בהגדרות
   - התקן תיקוני באגים

2. **גבה תצורה**
   - ייצא הגדרות באופן קבוע
   - שמור חשבונות ומשימות

3. **נטר משאבים**
   - ודא מספיק RAM זמין
   - בדוק שימוש ב-CPU

4. **הרץ כמנהל**
   - הגדר תמיד להריץ כמנהל
   - מונע בעיות הרשאות

## שאלות נפוצות

**ש: כמה זיכרון RAM נדרש?**  
ת: מינימום 4GB, מומלץ 8GB ומעלה.

**ש: האם TikMatrix תומך ב-Windows 7?**  
ת: לא, נדרש Windows 10 ומעלה.

**ש: האם אפשר להריץ מספר עותקים?**  
ת: לא מומלץ, עלול לגרום לסכסוכי יציאות.

**ש: מה עושים אם העדכון נכשל?**  
ת: הורד ידנית מהאתר והתקן מחדש.
