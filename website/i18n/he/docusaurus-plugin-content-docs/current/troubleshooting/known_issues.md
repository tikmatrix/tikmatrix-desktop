# בעיות ידועות

רשימת בעיות ידועות ופתרונות זמניים עד לתיקון קבוע.

## בעיות נוכחיות

### 1. שיקוף מסך לא יציב (Android 12+)

**תיאור:**  
שיקוף המסך מתנתק לעיתים קרובות במכשירי Android 12 ומעלה.

**סטטוס:** 🔄 בטיפול

**פתרון זמני:**
- הפחת רזולוציית שיקוף
- השתמש במצב חיסכון בחשבון
- הפעל מחדש שיקוף כל 30 דקות

**עדכון צפוי:** גרסה 2.1.0

### 2. פרסום סרטונים ארוכים (>3 דקות)

**תיאור:**  
סרטונים מעל 3 דקות נכשלים לפעמים בהעלאה.

**סטטוס:** 🔄 בטיפול

**פתרון זמני:**
- הגדל `upload_wait_time` ל-120 שניות
- דחוס סרטונים לפני העלאה
- פצל לסרטונים קצרים יותר

**עדכון צפוי:** גרסה 2.0.5

### 3. חימום חשבון - לייקים כפולים

**תיאור:**  
לפעמים נותן לייק פעמיים לאותו סרטון.

**סטטוס:** ⚠️ תעדוף נמוך

**פתרון זמני:**
- השתמש בהסתברויות נמוכות יותר
- הפעל מצב sequential
- נטר יומנים

**עדכון צפוי:** גרסה 2.2.0

### 4. אפליקציות משובטות - זיהוי לא מדויק

**תיאור:**  
אפליקציות משובטות מסוימות לא מזוהות אוטומטית.

**סטטוס:** 🔄 בטיפול

**פתרון זמני:**
- הגדר חשבונות ידנית
- השתמש בזיהוי מותאם
- עדכן רשימת אפליקציות נתמכות

**עדכון צפוי:** גרסה 2.1.0

### 5. Proxy - זמן חיבור ארוך

**תיאור:**  
פרוקסי מסוימים לוקחים זמן רב להתחבר.

**סטטוס:** 📝 נחקר

**פתרון זמני:**
- הגדל timeout בהגדרות
- השתמש בפרוקסי מהירים יותר
- בדוק חיבור פרוקסי מראש

**עדכון צפוי:** TBD

### ✅ דליפת זיכרון בשיקוף ארוך (v2.0.2)

**תוקן ב:** גרסה 2.0.2

**פתרון:** ניהול זיכרון משופר

## פתרונות כלליים

### עדכן לגרסה האחרונה

רוב הבעיות מתוקנות בגרסאות חדשות:

```
TikMatrix → הגדרות → עדכונים → בדוק עדכונים
```

### נקה מטמון

מטמון פגום עלול לגרום לבעיות:

```
TikMatrix → הגדרות → מתקדם → נקה מטמון
```

### אתחל הגדרות

כשל אחר נכשל:

```
TikMatrix → הגדרות → מתקדם → אתחל להגדרות מפעל
```

## תעדוף בעיות

### 🔴 קריטי

בעיות המונעות שימוש בסיסי:
- התרסקויות
- אובדן נתונים
- בעיות אבטחה

**זמן תגובה:** 24-48 שעות

### 🟡 בינוני

**זמן תגובה:** 1-2 שבועות

- שיפורים קוסמטיים

**זמן תגובה:** תכנון עתידי

## תאימות

### מכשירים נתמכים

✅ **תמיכה מלאה:**
- Samsung Galaxy (Android 9+)
- Google Pixel (Android 10+)
- OnePlus (Android 10+)

⚠️ **תמיכה חלקית:**
- Xiaomi MIUI (בעיות הרשאות ידועות)
- Huawei (ללא Google Services)

❌ **לא נתמך:**
- iOS/iPhone
- Windows Phone
- Tablets (בבדיקה)

### גרסאות Android

| גרסה | סטטוס | הערות |
|------|--------|-------|
| 13+ | ✅ תמיכה מלאה | מומלץ |
| 11-12 | ✅ תמיכה מלאה | בעיות שיקוף ידועות |
| 9-10 | ⚠️ תמיכה חלקית | תכונות מוגבלות |
| < 9 | ❌ לא נתמך | עדכן מערכת |

## מגבלות ידועות

### מגבלות פלטפורמה

**TikTok:**
- מגבלת קצב: 100 פעולות/שעה
- גודל סרטון: מקסימום 10 דקות
- פורמטים נתמכים: MP4, MOV

**Instagram:**
- מגבלת קצב: 60 פעולות/שעה
- גודל סרטון: מקסימום 60 שניות (Reels)
- יחס גובה-רוחב: 9:16 או 4:5

### מגבלות תוכנה

**TikMatrix:**
- מכשירים מקסימליים: 100 (תלוי בתוכנית)
- משימות במקביל: 20
- קבצי וידאו: מקסימום 5GB כל אחד
- זיכרון RAM מומלץ: 8GB+
