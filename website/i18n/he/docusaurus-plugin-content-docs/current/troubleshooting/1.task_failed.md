# משימה נכשלה

מדריך לאבחון ותיקון משימות כושלות ב-TikMatrix.

## סיבות נפוצות

### 1. בעיות חיבור רשת

- המכשיר לא מחובר לאינטרנט
- חיבור Wi-Fi חלש או לא יציב
- שרת פרוקסי לא פעיל

**פתרון:**
- בדוק את חיבור האינטרנט של המכשיר
- החלף לרשת Wi-Fi יציבה יותר
- ווודא ששרת הפרוקסי פעיל ונגיש

### 2. הרשאות לא מספיקות

- חוסר הרשאות אחסון
- הרשאות אפליקציה חסרות
- גישה לקבצים נדחתה

**פתרון:**
- הענק את כל ההרשאות הנדרשות
- הפעל מחדש את האפליקציה
- בדוק הרשאות במערכת

### 3. תצורה שגויה

- פרמטרי סקריפט לא תקינים
- נתיבי קבצים שגויים
- כיתובים חסרים

**פתרון:**
- ווידא את פרמטרי התצורה
- בדוק שנתיבי הקבצים קיימים
- השלם שדות נדרשים

### 4. גבולות פלטפורמה

- הגעת למגבלת קצב
- החשבון מוגבל זמנית
- תוכן מופרז

**פתרון:**
- המתן לפני ניסיון חוזר
- הפחת תדירות משימות
- בדוק הנחיות קהילה

## צעדי אבחון

1. **בדוק יומני משימה**
   ```
   הגדרות → יומנים → בחר משימה כושלת
   ```

2. **ווידא חיבור מכשיר**
   ```
   מכשירים → בדוק מצב מקוון
   ```

3. **נסה שוב ידנית**
   ```
   משימות → בחר משימה → נסה שוב
   ```

4. **הפעל במצב ניפוי שגיאות**
   ```
   הגדרות → מתקדם → הפעל יומני ניפוי שגיאות
   ```

## פתרונות אוטומטיים

### ניסיון חוזר אוטומטי

הגדר ניסיון חוזר אוטומטי למשימות כושלות:

```json
{
  "auto_retry": true,
  "max_retries": 3,
  "retry_delay": 300
}
```

### התראות

קבל התראות על משימות כושלות:

```
הגדרות → התראות → הפעל התראות כישלון משימה
```

## שאלות נפוצות

**ש: כמה פעמים המערכת תנסה שוב באופן אוטומטי?**  
ת: ברירת מחדל היא 3 נ יסיונות עם השהיה של 5 דקות בין ניסיונות.

**ש: האם אפשר לראות יומנים מפורטים?**  
ת: כן, הפעל מצב ניפוי שגיאות בהגדרות מתקדמות.

**ש: מה עושים אם כל המשימות נכשלות?**  
ת: בדוק את חיבור הרשת והרשאות המערכת. נסה להפעיל מחדש את האפליקציה.
