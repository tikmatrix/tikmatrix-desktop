# Erro de Inicialização do Software

## Solução de Problemas Comuns de Inicialização

### Etapa 1: Verificar Requisitos do Sistema

1. Certifique-se de que seu sistema atende aos requisitos mínimos:
   - Windows 10 ou superior
   - Pelo menos 4GB RAM
   - Pelo menos 1GB de espaço livre em disco

### Etapa 2: Verificar Microsoft Visual C++ Redistributable

Se você ver erros como `vcruntime140.dll not found` ou `The application was unable to start correctly (0xc000007b)`, você pode estar sem o Microsoft Visual C++ Redistributable.

1. Pressione `Windows + R`, digite `cmd` e pressione Enter.
2. Execute: `dir "C:\Windows\System32\vcruntime*.dll"`
3. Se você ver uma saída, o redistributable está instalado. O nome do arquivo (ex.: vcruntime140.dll) indica a versão.
4. Se não, baixe e instale a versão mais recente: [Microsoft Visual C++ Redistributable](https://learn.microsoft.com/en-us/cpp/windows/latest-supported-vc-redist?view=msvc-170)

### Etapa 3: Verificar Horário do Sistema

Horário incorreto do sistema pode causar erros de "Agent Start Timeout":

1. Clique com o botão direito na exibição de hora no canto inferior direito da sua tela
2. Selecione "Ajustar data/hora"
3. Habilite "Definir hora automaticamente" se ainda não estiver habilitado
4. Se a hora ainda estiver incorreta, clique em "Sincronizar agora" ou defina manualmente a data e hora corretas
5. Reinicie o TikMatrix após corrigir a hora

### Etapa 4: Verificar Logs da Aplicação

1. Clique em `Configurações` > `Abrir Diretório do Aplicativo` no TikMatrix.
2. Vá para a pasta `logs`.
3. Verifique o arquivo de log mais recente para mensagens de erro.

### Etapa 5: macOS — Agent InvalidStateError

Em algumas máquinas macOS, o agent pode falhar ao iniciar e mostrar um erro como:

```text
Agent start error InvalidStateError: The object is in an invalid state.
```

Causa possível:

- Isso é frequentemente causado por uma incompatibilidade entre a aplicação e a versão do sistema macOS ou hardware/firmware específico em máquinas mais antigas.

Soluções alternativas / soluções:

1. Atualize o macOS para a versão mais recente disponível usando Configurações do Sistema > Geral > Atualização de Software. Após atualizar, reinicie seu Mac e tente iniciar o TikMatrix novamente.
2. Se atualizar não for possível ou o erro persistir, tente executar a aplicação em um Mac diferente com uma versão mais recente do macOS.

Se você continuar vendo o erro após tentar o acima, colete seus logs da aplicação e entre em contato com nosso suporte no [Telegram](https://t.me/tikmatrix_agent_bot) com a versão do macOS e os logs.

### Se o Problema Persistir

Entre em contato com nossa equipe de suporte no [Telegram](https://t.me/tikmatrix_agent_bot) com:

- Sua versão do Windows
- Arquivos de log da aplicação
- Etapas que você já tentou
