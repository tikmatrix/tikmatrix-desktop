# Permissão Negada - Não é Possível Salvar Rascunhos de Vídeos

## Resolvendo Problemas de Permissão Negada na Tela do Celular

### Descrição do Problema

Ao executar scripts de postagem, você pode encontrar o seguinte aviso na tela do seu celular:

```text
Permission denied. Unable to save draft videos.
```

### Solução

Este problema ocorre porque o aplicativo TikTok não tem permissões de armazenamento. Siga estas etapas para resolver:

#### Etapa 1: Clicar Manualmente no Botão Publicar

1. Localize e toque no botão **Publicar** na tela do seu celular
2. Continue com o processo de publicação

#### Etapa 2: Conceder Permissões do Sistema

1. Quando o diálogo de solicitação de permissão do sistema aparecer, toque em **Permitir**
2. Certifique-se de conceder ao aplicativo TikTok as permissões de armazenamento necessárias

### Nota Importante

:::info
Este processo de configuração de permissões só precisa ser realizado uma vez. Uma vez que as permissões sejam concedidas, execuções subsequentes de scripts de postagem não encontrarão este problema.
:::

### Medidas de Prevenção

Para evitar este problema, é recomendado fazer o seguinte antes de usar o TikMatrix pela primeira vez:

1. Abra manualmente o aplicativo TikTok
2. Tente gravar ou salvar um rascunho de vídeo
3. Selecione **Permitir** quando solicitado para permissões
4. Isso irá pré-configurar as permissões necessárias
