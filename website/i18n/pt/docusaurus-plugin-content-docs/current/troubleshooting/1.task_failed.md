# Etapas de Solução de Problemas para Falha de Tarefa

## Etapa 1: Verificar Status do Agent (se a tarefa falhou ao iniciar)

1. Clique na barra lateral: Geral → Abrir Agent.
2. Certifique-se de que o telefone está com o aplicativo TikMatrix aberto.
3. Observe se um status de sucesso aparece dentro de 10 segundos.
4. Se o Agent mostrar um status de falha, siga as etapas de inicialização manual no guia Inicializar Dispositivo: ../tutorial-basics/2.init-device.md

## Etapa 2: Verificar Configuração do Aplicativo TikTok/Instagram (se o script falhar ao abrir/fechar o aplicativo corretamente)

1. Se você estiver usando o aplicativo oficial TikTok/Instagram, clique em "Abrir TikTok" / "Abrir Instagram" e verifique se o aplicativo abre corretamente no dispositivo. Se não abrir, clique no botão de configurações na barra de título e selecione a versão correta do TikTok (Internacional / Ásia). Instagram tem apenas a versão oficial.
2. Se você estiver usando uma versão clonada do TikTok/Instagram, certifique-se de ter configurado o nome do pacote do aplicativo clonado na **Lista de Contas**. Consulte o guia para usar aplicativos clonados: ../tutorial-basics/10.clone-apps-support.md. Também certifique-se de que o **Modo Multi-conta** está habilitado ao executar tarefas.

## Etapa 3: Verificar Configurações de Idioma do TikTok/Instagram

Atualmente, só oferecemos suporte ao idioma inglês para TikTok e Instagram. Se seus aplicativos não estiverem em inglês, configure-os para inglês da seguinte forma:

1. Abra o aplicativo TikTok no dispositivo.
2. Vá para o seu perfil.
3. Toque nos três pontos no canto superior direito para abrir Configurações.
4. Role até a seção "Language" (Idioma).
5. Selecione "English" como idioma do aplicativo.
6. Repita as etapas para configurar o Instagram para inglês.

## Etapa 4: Executar um Script de Teste

1. Clique na barra lateral: Scripts → Script de Teste.
2. Aguarde o script terminar.
3. Se o teste falhar, verifique os logs e tente resolver o problema. Consulte a seção "Problemas Conhecidos" abaixo para problemas comuns.

## Etapa 5: Enviar um Ticket de Suporte

Se as quatro etapas anteriores não identificaram o problema, por favor crie um ticket de suporte. Vamos coletar logs automaticamente e ajudá-lo a solucionar o problema. Etapas:

1. Clique no botão "Suporte" na barra de título.
2. Clique em "Criar Ticket".
3. Insira suas informações de contato e uma descrição do problema. Para acelerar a resolução, por favor anexe uma gravação de tela da execução do script.
4. Selecione os dispositivos afetados (você pode selecionar múltiplos). Isso é importante — só podemos receber logs dos dispositivos que você selecionar.
5. Clique em "Enviar".
6. Podemos solicitar informações adicionais, como o arquivo XML da interface do usuário da última tela antes da falha: navegue manualmente no telefone até a última página antes da falha da tarefa, clique no botão "Debug" no lado direito da tela do dispositivo e então clique em "Download XML". Envie o XML baixado e o número do ticket para nosso email de suporte: [support@tikmatrix.com](mailto:support@tikmatrix.com)
