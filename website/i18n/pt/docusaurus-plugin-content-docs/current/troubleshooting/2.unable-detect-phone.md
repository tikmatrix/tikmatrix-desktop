# Não Consegue Detectar o Telefone

## Etapas de Solução de Problemas

### Etapa 1: Verificar Conexão ADB

1. Clique em `Configurações` > `Abrir Terminal ADB` no TikMatrix.
2. Execute `adb devices` e verifique se seu telefone aparece na lista com status `device`.

### Para sistemas Windows

```shell
.\adb.exe devices
```

### Para sistemas Mac/Linux

```shell
./adb devices
```

### Exemplo de Saída

```shell
List of devices attached
<DEVICE_ID>    <STATUS>
```

### Se o Dispositivo Não Estiver Listado

Siga as etapas de solução de problemas em [Conectar Dispositivo](../tutorial-basics/1.connect-to-devices.md).

### Se o Status do Dispositivo For `unauthorized`

Isso significa que o dispositivo não está autorizado. Toque em `Permitir depuração USB` no seu telefone.

### Se o Status do Dispositivo For `offline`

Isso geralmente significa uma conexão USB solta. Reconecte seu telefone desconectando e reconectando o cabo USB.

### Se o Status do Dispositivo For `device`

O dispositivo está conectado com sucesso. Se o TikMatrix ainda não conseguir detectar o telefone, certifique-se de que o TikMatrix está ativado (o teste gratuito também requer ativação).
