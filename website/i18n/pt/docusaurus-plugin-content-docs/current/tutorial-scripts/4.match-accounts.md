# Combinar Contas

O script de Combinar Contas é usado para reconhecer contas do TikTok do telefone e adicioná-las à lista de contas.

## Etapas

1. Selecione o dispositivo para executar o script.
2. Clique em `Script` > `Combinar Contas`.
3. Clique em `Iniciar Script` para começar.
4. O script reconhecerá automaticamente as contas no telefone e as adicionará à lista de contas.

## Nota

- O script de combinar contas pode não reconhecer todas as contas em uma execução. Se algumas contas forem perdidas, tente executar o script novamente.
