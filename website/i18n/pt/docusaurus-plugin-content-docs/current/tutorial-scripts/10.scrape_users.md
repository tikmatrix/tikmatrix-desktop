# Coletar Usuários

O script Coletar Usuários é usado para coletar nomes de usuário do TikTok. O recurso de coleta de usuários suporta três modos: coletar a lista de seguidores de um usuário especificado, coletar a lista de seguindo de um usuário especificado ou coletar usuários pesquisando com palavras-chave.

## Etapas

1. Selecione o dispositivo para executar o script. Se você selecionar vários dispositivos, as tarefas de coleta serão distribuídas uniformemente entre os dispositivos selecionados.
2. Clique em `Script` > `Coletar Usuários`.
3. Configure as definições da tarefa:
    - **Modo de Coleta**: Selecione o modo para coletar usuários.
      - **Coletar por Seguidores**: Colete usuários da lista de seguidores de contas especificadas.
      - **Coletar por Seguindo**: Colete usuários da lista de seguindo de contas especificadas.
      - **Coletar por Palavra-chave**: Pesquise e colete usuários usando palavras-chave.
    - **Nomes de Usuário da Conta-Alvo**: Insira o nome de usuário da conta da qual deseja coletar usuários, um por linha (para modos Seguidores e Seguindo).
    - **Palavras-chave de Pesquisa**: Insira palavras-chave para pesquisar usuários (para modo Palavra-chave).
    - **Contagem máxima de coleta**: Defina o número máximo de usuários a serem coletados.

4. Clique em `Iniciar Script` para começar.

## Nota

- Os dados de usuário coletados serão salvos em um arquivo com base no modo de coleta:
- Para modo Seguidores: `exported_users_following_tiktok_<nome_de_usuario>_<contagem>.txt`
- Para modo Seguindo: `exported_users_followers_tiktok_<nome_de_usuario>_<contagem>.txt`
- Para modo Palavra-chave: `exported_users_search_tiktok_<palavra_chave>_<contagem>.txt`
- O TikTok limita apenas o número de usuários que podem ser visualizados, então apenas cerca de 50 registros podem ser recuperados por operação.
