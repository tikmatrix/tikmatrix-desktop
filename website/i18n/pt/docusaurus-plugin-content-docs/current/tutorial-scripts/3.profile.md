# Perfil

O script de Perfil é usado para configurar perfis de contas.

## Etapas

1. Selecione o dispositivo para executar o script.
2. Clique em `Script` > `Preencher Perfil`.
3. Configure as definições da tarefa:
    - **Apelidos**: Um apelido por linha.
    - **Nomes de usuário**: Um nome de usuário por linha.
    - **Biografias**: Uma biografia por linha (suporta menção a usuários e quebras de linha).
      - Para adicionar quebras de linha na biografia, use `\n` no seu texto. Por exemplo: `Primeira linha\nSegunda linha\nTerceira linha`
    - **Caminho do Avatar**: Caminho do arquivo para a pasta de imagens de avatar.
    - **Ordem de Seleção**: Escolha como o conteúdo é selecionado para cada tarefa:
      - **Aleatório**: Seleciona aleatoriamente entre as opções disponíveis para cada tarefa (padrão).
      - **Sequencial**: Seleciona o conteúdo em ordem baseada na contagem de tarefas. Use este modo para garantir que cada conta receba informações únicas.
4. Clique em `Iniciar Script` para começar.

## Guia de Ordem de Seleção

### Modo Aleatório

- O conteúdo (apelidos, nomes de usuário, biografias, avatares) é selecionado aleatoriamente para cada tarefa
- O mesmo conteúdo pode ser usado várias vezes em diferentes contas
- Adequado quando a unicidade do conteúdo não é crítica

### Modo Sequencial

- O conteúdo é selecionado em ordem: a 1ª tarefa recebe o 1º item, a 2ª tarefa recebe o 2º item, etc.
- Quando todos os itens são usados, ele volta ao início
- **Importante**: Para garantir que cada conta receba informações únicas, certifique-se de fornecer mais itens do que o número total de contas em uma única execução em lote
  - Exemplo: Se executar 100 contas, forneça pelo menos 100 apelidos, nomes de usuário, biografias e imagens de avatar únicos

## Nota

- É recomendável limpar a galeria antes de configurar o perfil, pois o script usa a primeira foto como avatar.
