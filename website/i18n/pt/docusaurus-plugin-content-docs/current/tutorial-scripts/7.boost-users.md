# Impulsionar Usuários

:::warning Aviso de Descontinuação
Esta funcionalidade foi integrada ao [Super Marketing](./14.super-marketing.md). Por favor, use o script Super Marketing para recursos de impulsionamento de usuários mais poderosos e flexíveis.
:::

O recurso Impulsionar Usuários aumenta a exposição e o engajamento seguindo ou deixando de seguir contas-alvo em massa. O sistema tenta simular o comportamento de usuários reais para reduzir o risco de detecção, mas você ainda deve seguir as regras da plataforma e usar moderadamente.

## Usando o recurso Impulsionar Usuários no Super Marketing

1. Abra o script: escolha "Super Marketing".
2. Selecione o tipo de fonte de dados "Lista de contas de usuário".
3. Cole nomes de usuário em massa ou importe de um arquivo de texto (um nome de usuário por linha); após importar escolha "Anexar" ou "Sobrescrever".
4. Escolha uma estratégia de uso de dados:

    - Pool compartilhado: todas as contas bot interagem com os mesmos alvos.

    - Consumo único: cada conta bot lida com alvos diferentes, então cada alvo é processado por apenas uma conta.

5. Defina a contagem de repetição por alvo em "Ações de Usuário — Repetições" (tipicamente definido como 1).
6. Escolha o tipo de ação: Seguir ou Deixar de Seguir. Você pode configurar ainda mais as opções em **Ações de Usuário**:

        - Método de navegação de perfil: escolha **Pesquisar Usuário** ou **Abrir Perfil Diretamente**.

        - Enviar Mensagem Direta (opcional): se ativado, cole modelos de DM multilinha pré-gerados (um modelo por linha ou parágrafo). Configure a ordem de uso do modelo:

                - **Aleatório**: escolhe aleatoriamente um modelo cada vez.
                - **Sequencial**: usa modelos na ordem colada/importada (pode continuar entre execuções de tarefas).

            Você também pode escolher inserir emoji automaticamente nos modelos para aumentar a naturalidade.

7. Configure outros parâmetros do script (concorrência, atrasos, pool de contas), verifique as configurações e inicie o script.

## Recomendações e notas

- Teste em um número pequeno de alvos primeiro para garantir que as configurações não acionem defesas da plataforma.
- Defina atrasos e concorrência razoáveis para imitar comportamento humano e reduzir riscos.
- Para execuções longas, monitore a saúde da conta e os logs de tarefas e ajuste as estratégias conforme necessário.
- Operações de Impulsionar Usuários não são garantidas para ter sucesso 100%; tente novamente tarefas falhadas ou investigue o status da conta-alvo se necessário.
