# Impulsionar Posts

:::warning Aviso de Descontinuação
Esta funcionalidade foi integrada ao [Super Marketing](./14.super-marketing.md). Por favor, use o script Super Marketing para recursos de impulsionamento de posts mais poderosos e flexíveis.
:::

O recurso Impulsionar Posts aumenta o engajamento curtindo, compartilhando, favoritando, repostando e assistindo posts em massa. O sistema tenta simular o comportamento de usuários reais para reduzir o risco de detecção, mas você ainda deve seguir as regras da plataforma e usar moderadamente.

## Usando o recurso Impulsionar Posts no Super Marketing

1. Abra o script: escolha "Super Marketing".
2. Selecione o tipo de fonte de dados "Lista de URLs de posts".
3. Cole URLs de posts em massa ou importe de um arquivo de texto; após importar escolha "Anexar" ou "Sobrescrever" a lista existente.
4. Escolha uma estratégia de uso de dados:

    - Pool compartilhado: todas as contas bot interagem com os mesmos posts.

    - Consumo único: cada conta bot interage com posts diferentes, então cada post é tratado por apenas uma conta.

5. Defina a contagem de repetição por post em "Ações de Post — Repetições". Se comentários estiverem ativados, defina repetições de comentários como 1 para evitar comentários duplicados.
6. Em "Ações de Post" selecione os tipos de interação que deseja: Curtir, Compartilhar, Repostar, Favoritar, Comentar, Assistir vídeo, etc.

     Se você selecionar "Comentar", configure as opções de comentário da seguinte forma:

     - Modelos de comentários pré-gerados: cole modelos de comentários multilinha ou multi-parágrafo (cada linha/parágrafo é tratado como um comentário candidato).

     - Ordem de uso do modelo: escolha **Aleatório** ou **Sequencial**.
         - Aleatório: escolhe aleatoriamente um modelo cada vez.
         - Sequencial: usa modelos na ordem colada/importada (útil para desduplicação determinística).

     - Inserção de emoji: opcionalmente ative a inserção automática de emoji em comentários para aumentar a naturalidade.

     - Geração de comentários em tempo real (opcional — API ChatGPT):
         - Quando ativada, o script pode chamar uma API semelhante ao ChatGPT para gerar comentários únicos com base no contexto, baseados no título/descrição do post. Isso reduz significativamente comentários repetidos e é adequado quando muitos comentários únicos são necessários.
         - Notas: usar geração em tempo real requer uma chave API válida e atenção aos limites de taxa, custos e privacidade (evite enviar dados sensíveis a serviços de terceiros).

     - Duas formas comuns de garantir comentários únicos:
         1. Carregar e rotacionar modelos pré-gerados no modo **Sequencial** (cada modelo é usado uma vez até esgotar).
         2. Usar a API ChatGPT para gerar um comentário único por post (geração dinâmica baseada no conteúdo do post).

     - Recomendação: teste sua estratégia de comentários em um lote pequeno, revise a qualidade dos comentários e o tratamento da plataforma; se usar geração em tempo real, monitore o uso e o custo da API.

7. Configure outros parâmetros do script (concorrência, atrasos, pool de contas), verifique as configurações e inicie o script.

## Recomendações e notas

- Teste em um número pequeno de posts primeiro para garantir que as configurações não acionem defesas da plataforma.
- Mantenha o texto dos comentários variado e natural; evite copiar o mesmo texto.
- Defina atrasos e concorrência razoáveis para imitar comportamento humano e reduzir riscos.
- Para execuções longas, monitore a saúde da conta e os logs de tarefas e ajuste as estratégias conforme necessário.
