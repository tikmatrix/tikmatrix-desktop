# Login

O script de Login é usado para fazer login em contas de redes sociais.

## Etapas

1. Certifique-se de que a conta foi adicionada à lista de contas (clique em `Contas` para adicionar contas) e selecione o índice do dispositivo que deseja usar para esta conta.
2. Selecione um ou vários dispositivos para executar o script de login.
3. Clique em `Script` > `Login`.
4. Clique em `Iniciar Script` para começar.

## Prioridade de Login

O sistema usa a seguinte prioridade de login:

1. **E-mail + Senha**: Se um e-mail estiver configurado para a conta, ele será usado primeiro para login.
2. **Nome de usuário + Senha**: Se nenhum e-mail estiver configurado, o nome de usuário será usado para login.

## Suporte 2FA (Apenas IgMatrix)

O IgMatrix suporta login automático com Autenticação de Dois Fatores (2FA):

### Formato de Senha para 2FA

Para usar 2FA, formate sua senha da seguinte forma:

```plaintext
sua_senha:sua_chave_secreta_2fa
```

**Exemplo:**

```plaintext
minhasenha123:JBSWY3DPEHPK3PXP
```

### Como Funciona o 2FA

1. **Análise de Senha**: O sistema detecta automaticamente se sua senha contém um separador de dois pontos (`:`).
2. **Geração de Token**: Quando o 2FA é necessário, o sistema usa sua chave secreta para solicitar um código TOTP do nosso serviço 2FA.
3. **Entrada Automática**: O código de 6 dígitos gerado é inserido automaticamente no campo 2FA do Instagram.
4. **Login Perfeito**: Todo o processo 2FA acontece automaticamente sem intervenção manual.

### Chave Secreta 2FA

A chave secreta 2FA é tipicamente uma string codificada em Base32 de 32 caracteres que você obtém ao configurar o 2FA no Instagram. É a mesma chave usada por aplicativos autenticadores como Google Authenticator ou Authy.

## Nota

- O login do TikTok não é 100% totalmente automático. Você pode precisar inserir manualmente códigos de verificação por e-mail ou concluir outras etapas de verificação. Não podemos contornar os desafios de login específicos do TikTok (como CAPTCHAs de imagem ou verificação humana) ou resolver certas mensagens de erro como "muitas tentativas de login".
- O Instagram pode alcançar login 100% automatizado via 2FA. Para contas do Instagram habilitadas com 2FA, certifique-se de usar o formato correto da chave secreta em sua senha.
