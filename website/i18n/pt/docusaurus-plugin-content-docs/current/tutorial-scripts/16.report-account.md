# Denunciar Conta

:::warning Uso Responsável
Denunciar contas é uma ação poderosa. Use o recurso de denúncia apenas para violações legítimas (fraude, golpes, spam, etc.). O uso indevido pode resultar em restrições de conta ou outros efeitos adversos.
:::

O guia de Denunciar Conta explica como usar a capacidade de denúncia integrada exposta pelo script Super Marketing. O recurso automatiza o fluxo de denúncia fixo no aplicativo para ajudar a processar uma lista de contas de usuário, mas não suporta personalização do caminho de denúncia ou razões de denúncia além da seleção fixa descrita abaixo.

## Visão geral rápida

- O fluxo de denúncia usado pelos scripts é fixo e não pode ser alterado na UI ou scripts:

```txt
Denunciar → Denunciar conta → Outra coisa → Fraudes e golpes
```

- O script tenta navegar no perfil do usuário, abrir o menu de denúncia, escolher o caminho fixo acima e enviar a denúncia quando possível.
- Como a UI móvel difere entre versões de aplicativos e dispositivos, o script usa seletores de UI de melhor esforço; sempre teste em um lote pequeno primeiro.

## Etapas para executar (via Super Marketing)

1. Abra TikMatrix/IgMatrix e selecione o(s) dispositivo(s) para executar (seleção múltipla suportada).
2. Abra `Scripts` → `Super Marketing`.
3. Defina **Fonte de dados** como `Nomes de usuário` e importe um arquivo de texto ou cole nomes de usuário (um por linha). Após a importação, escolha `Anexar` ou `Sobrescrever`.
4. Em **Ações de Usuário**, ative **Denunciar conta** (o novo alternador) e verifique o caminho de denúncia fixo exibido.
5. Configure a concorrência de tarefas, atrasos e outras configurações de tarefas conforme necessário.
6. Clique em `Iniciar Script` para começar a processar a lista. Monitore logs e telas de dispositivos para feedback de sucesso/falha.

## Dicas e recomendações

- Teste com 1–5 contas primeiro para confirmar que a UI de denúncia é detectada e a sequência funciona na versão do aplicativo-alvo e dispositivos.
- Mantenha o ritmo e os atrasos conservadores; denúncias automatizadas pesadas podem acionar defesas da plataforma ou limites de taxa.
- Este recurso usa um caminho fixo no aplicativo. Se a UI do aplicativo mudar (texto ou posições de elementos), a automação pode falhar até que os seletores sejam atualizados.
- Não use denúncia para assediar ou atingir usuários inocentes. Denuncie apenas violações claras consistentes com os termos da plataforma.
- Revise os logs após uma execução; o script registrará falhas como botões de menu ausentes ou opções de denúncia ausentes para que você possa ajustar o tempo ou os seletores.

## Quando não usar

- Se você precisar selecionar razões de denúncia personalizadas ou enviar evidências detalhadas por caso — o script não suporta esse nível de personalização.
- Para questões legais ou sensíveis onde a revisão manual é necessária.

## Solução de problemas

- "Não foi possível encontrar a opção Denunciar" / "Não foi possível encontrar a opção 'Denunciar conta'": o texto ou layout da UI provavelmente mudou. Execute novamente com um conjunto de teste pequeno e capture capturas de tela do dispositivo para depuração.
- Se denúncias estão sendo enviadas mas não têm efeito, a plataforma pode ignorar denúncias automatizadas ou aplicar regras de moderação. Considere revisão manual para casos críticos.
