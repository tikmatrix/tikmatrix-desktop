# Comentários em Massa

:::warning
Este recurso foi mesclado no [script Super Marketing](./14.super-marketing.md). Por favor, prefira Super Marketing para recursos mais poderosos e flexíveis de comentários em massa e capacidades de engajamento.
:::

Comentários em Massa é usado para postar respostas de engajamento em muitos posts do TikTok—útil para nutrição de contas, promoções ou aumento de atividade. O sistema suporta modelos pré-gerados e geração em tempo real, e tenta imitar o tempo semelhante ao humano para reduzir o risco de detecção.

## Usando Comentários em Massa dentro do Super Marketing

1. Abra o script Super Marketing.
2. Selecione o tipo de fonte de dados: "Lista de URLs de posts".
3. Cole URLs de posts em massa ou importe um arquivo `.txt` (um URL por linha). Após a importação, você pode escolher "Anexar" ou "Substituir" a lista atual.
4. Escolha uma estratégia de uso de dados:

   - Pool compartilhado: todas as contas bot comentarão do mesmo pool de alvos.

   - Consumo único: cada conta bot processa alvos diferentes, então cada post recebe um comentário de apenas uma conta.

5. Defina quantos comentários postar por alvo (geralmente 1 para evitar comentários duplicados).
6. Configure as definições de comentário:

   - Modelos pré-gerados: cole várias linhas ou parágrafos de cópia de comentário preparada. Cada linha/parágrafo é tratado como um comentário candidato.

   - Ordem do comentário: escolha "Aleatório" ou "Sequencial".

   - Aleatório: cada envio seleciona um modelo aleatório.

   - Sequencial: rotaciona modelos na ordem de colagem/importação (útil quando você precisa de uma estratégia de desduplicação reproduzível).

   - Inserir emojis: ative a inserção automática de emojis para aumentar a naturalidade.

   - Inserir número do dispositivo (opcional): anexe o rótulo numérico do dispositivo aos comentários para rastreamento (use com cautela; destinado a depuração/auditoria).

   - Geração de comentários em tempo real (opcional, API ChatGPT):

      - Quando ativada, o script pode chamar ChatGPT ou uma API semelhante para gerar comentários contextuais e únicos com base no título/descrição do post ou contexto disponível—adequado quando você precisa de muitos comentários únicos.

      - Nota: ativar a geração em tempo real requer uma chave API válida e você deve considerar limites de taxa, custos e privacidade (evite enviar dados sensíveis para serviços de terceiros).

   - Duas formas comuns de garantir comentários únicos:

      1. Use rotação sequencial de modelos pré-gerados (cada modelo é usado uma vez até esgotar).

      2. Use geração em tempo real ChatGPT para criar comentários únicos por alvo.

7. Configure outros parâmetros do script (concorrência, intervalo de tarefas, pool de contas, etc.) e inicie o script quando estiver pronto.

:::tip Sobre intervalo de tarefas
Defina um intervalo mínimo e máximo e ative atrasos aleatórios. Temporizações escalonadas reduzem a chance de picos de atividade sincronizados e diminuem o risco de detecção.
:::

## Recomendações e notas

- Teste estratégias de comentários em um lote pequeno primeiro para validar a qualidade e a resposta da plataforma.
- Mantenha o texto dos comentários compatível e variado; evite enviar muitas mensagens idênticas em um curto período.
- Se usar geração em tempo real, monitore estritamente o uso e os custos da API e evite enviar informações sensíveis.
- As tarefas podem falhar; verifique os logs e tente novamente linhas falhadas. Alvos processados serão ignorados em execuções subsequentes.
