# Excluir Posts

O script Excluir Posts é usado para remover posts com base em sua contagem de visualizações.

## Etapas

1. Selecione o dispositivo para executar o script.
2. Clique em `Script` > `Excluir Posts`.
3. Configure as definições da tarefa:
    - **Contagem Máxima de Visualizações**: Insira a contagem máxima de visualizações para posts que deseja excluir.
4. Clique em `Iniciar Script` para começar.

## Nota

- O script de excluir posts pode nem sempre ter sucesso; tente novamente tarefas falhadas se necessário.
