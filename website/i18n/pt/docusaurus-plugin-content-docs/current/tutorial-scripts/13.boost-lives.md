# Impulsionar Lives

O script Impulsionar Lives permite que você controle contas de fazenda para assistir e opcionalmente curtir ou comentar em transmissões ao vivo-alvo. Você pode definir a duração de visualização e intervalos de curtidas/comentários para personalizar o comportamento de impulsionamento.

## Etapas

1. Selecione o dispositivo para executar o script.
2. Clique em `Scripts` > `Impulsionar Lives`.
3. Configure as definições da tarefa:
    - **Nomes de usuário**: Insira nomes de usuário-alvo, um por linha.
    - **Método de Entrada**: Escolha entre os métodos **Pesquisar** e **Direto**.
    - **Opções de Impulsionamento**: Ative curtir e/ou comentar.
    - **Entrar no Clube de Fãs**: Ative para entrar no clube de fãs do streamer, se disponível.
    - **Check-in Diário**: Ative atividades de check-in diário durante a transmissão ao vivo.
    - **Intervalo de Curtidas**: Defina o intervalo entre cada ação de curtida.
    - **Contagem de Toques de Curtida**: Número de vezes para tocar no botão de curtida.
    - **Intervalo de Comentários**: Defina o intervalo entre cada ação de comentário.
    - **Conteúdo do Comentário**: Insira um comentário por linha.
    - **Inserir Emojis**: Escolha se deseja inserir emojis nos comentários.
    - **Ordem dos Comentários**: Selecione ordem aleatória ou sequencial para comentários.
    - **Comentários por Conta**: Defina o número de comentários a postar por conta.
    - **Duração de Visualização**: Defina quanto tempo ficar na transmissão ao vivo.
    - **Intervalo de Tarefas**: Defina o intervalo entre cada tarefa.
4. Clique em `Iniciar Script` para começar.

## Notas

- O script de impulsionar lives pode nem sempre ter sucesso; tente novamente tarefas falhadas se necessário.
