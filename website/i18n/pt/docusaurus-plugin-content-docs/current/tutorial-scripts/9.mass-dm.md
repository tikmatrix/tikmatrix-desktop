# Mensagens em Massa

:::warning
Este recurso foi mesclado no [script Super Marketing](./14.super-marketing.md). Por favor, prefira Super Marketing para recursos mais poderosos e flexíveis de mensagens em massa e gerenciamento de mensagens.
:::

Mensagens em Massa é usado para construir conexões enviando mensagens diretas (DMs) em massa—útil para promoção, acompanhamento de clientes e reengajamento. O sistema suporta modelos pré-gerados e geração em tempo real, e visa imitar o tempo semelhante ao humano para reduzir o risco de detecção.

## Usando Mensagens em Massa dentro do Super Marketing

1. Abra o script Super Marketing.
2. Selecione o tipo de fonte de dados: "Lista de contas de usuário (lista de alvos)".
3. Cole nomes de usuário-alvo em massa ou importe um arquivo `.txt` (um nome de usuário por linha). Após a importação, você pode escolher "Anexar" ou "Substituir" a lista atual.
4. Escolha uma estratégia de uso de dados:

   - Pool compartilhado: todas as contas bot enviarão mensagens do mesmo pool de alvos.

   - Consumo único: cada conta bot processa alvos diferentes, então cada alvo recebe uma mensagem de apenas uma conta.

5. Defina quantas mensagens enviar por alvo (geralmente 1 para evitar contato repetido).
6. Configure as definições de mensagem:

   - Modelos pré-gerados: cole várias linhas ou parágrafos de cópia de DM preparada. Cada linha/parágrafo é tratado como uma mensagem candidata.

   - Ordem da mensagem: escolha "Aleatório" ou "Sequencial".

   - Aleatório: cada envio seleciona um modelo aleatório.

   - Sequencial: rotaciona modelos na ordem de colagem/importação (útil quando você precisa de uma estratégia de desduplicação reproduzível).

   - Inserir emojis: ative a inserção automática de emojis para aumentar a naturalidade.

   - Geração de mensagens em tempo real (opcional, API ChatGPT):

   - Quando ativada, o script pode chamar ChatGPT ou uma API semelhante para gerar mensagens personalizadas com base no nome de usuário-alvo ou contexto disponível—adequado quando você precisa de muitas mensagens únicas.

   - Nota: ativar a geração em tempo real requer uma chave API válida e você deve considerar limites de taxa, custos e privacidade (evite enviar dados sensíveis para APIs de terceiros).

   - Duas formas comuns de garantir mensagens únicas:

   1. Use rotação sequencial de modelos pré-gerados (cada modelo é usado uma vez até esgotar).

   2. Use geração em tempo real ChatGPT para criar mensagens únicas por alvo.

7. Configure como abrir perfis de usuário (se a tarefa precisar visualizar perfis): escolha "Pesquisar" ou "Abrir Perfil Diretamente".
8. Configure o intervalo de tarefas (mín/máx em minutos), concorrência e configurações do pool de contas, em seguida, inicie o script.

:::tip Sobre intervalo de tarefas
Defina um intervalo mínimo e máximo e ative atrasos aleatórios. Temporizações escalonadas reduzem a chance de picos de atividade sincronizados e diminuem o risco de detecção.
:::

## Recomendações e notas

- Mantenha o conteúdo de DM compatível e diversificado; evite enviar muitas mensagens idênticas em um curto período de tempo.
- Se usar geração em tempo real, monitore o uso e os custos da API de perto e evite enviar informações sensíveis.
- As tarefas podem falhar; verifique os logs e tente novamente linhas falhadas. Alvos processados serão ignorados em execuções subsequentes.
