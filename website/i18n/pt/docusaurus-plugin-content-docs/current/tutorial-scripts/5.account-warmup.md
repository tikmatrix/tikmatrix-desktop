# Aquecimento de Contas

O script de Aquecimento de Contas é usado para aquecer contas pesquisando palavras-chave, assistindo vídeos, seguindo, curtindo, comentando e favoritando.

## Etapas

1. Selecione o dispositivo para executar o script.
2. Clique em `Script` > `Aquecimento de Contas`.
3. Configure as definições da tarefa:
   - **Duração de Visualização**: Quanto tempo assistir cada vídeo.
   - **Palavras-chave de Pesquisa**: Tópicos para buscar vídeos.
   - **Duração da Tarefa**: Duração total da tarefa de aquecimento.
   - **Conteúdo de Comentários**: Use comentários pré-fabricados ou comentários gerados por ChatGPT a partir de legendas de posts.
   - **Probabilidade de Interação**: Probabilidade de seguir, curtir, comentar e favoritar.
4. Clique em `Iniciar Script` para começar.

## Nota

- O script de aquecimento pode nem sempre ter sucesso; tente novamente tarefas falhadas se necessário.
- Suporta agendamento, para que você possa configurar a tarefa de aquecimento para executar em um horário específico.
- Suporta pular anúncios, transmissões ao vivo e vídeos já curtidos.
