# Instalar TikTok

Recomendamos baixar as versões específicas do TikTok abaixo, pois versões mais recentes podem não ser compatíveis. As versões suportadas serão anunciadas no grupo após atualizações.

## Baixar APK do TikTok

- Versão Global: [com.zhiliaoapp.musically](https://apkpure.com/tiktok-musically-2024/com.zhiliaoapp.musically)
- Versão Asiática: [com.ss.android.ugc.trill](https://apkpure.com/tiktok/com.ss.android.ugc.trill)

## Etapas de Instalação

1. Baixe o APK dos links acima.
2. Selecione todos os dispositivos no TikMatrix.
3. Clique em `General` > `Install APK` e escolha o arquivo APK baixado.
4. Aguarde a conclusão da instalação (pode levar 1-3 minutos por dispositivo dependendo do modelo; não clique repetidamente).
5. Após a instalação, clique em `Settings` e verifique se a versão do TikTok corresponde ao seu aplicativo instalado.
6. Clique em `Open TikTok` para verificar se o aplicativo está instalado corretamente.
7. Certifique-se de que o idioma do TikTok está definido como inglês; caso contrário, os scripts podem não funcionar corretamente. Altere o idioma nas configurações do TikTok se necessário.
