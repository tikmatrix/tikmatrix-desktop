# Inicializar Dispositivos

:::warning
Este aplicativo é instalado via canal oficial de depuração ADB do Android para controle de automação de dispositivos. Alguns dispositivos podem perguntar se deseja enviar uma amostra de aplicativo desconhecido para o Google; este é um aviso de segurança do sistema Android não relacionado à segurança do aplicativo e não afeta o uso normal. Clique em "Não enviar" para completar a instalação.
:::

O APK do TikMatrix é um agente móvel necessário para tarefas de automação. **Quando você conecta seu telefone ao TikMatrix pela primeira vez, o sistema inicializará automaticamente seu dispositivo e instalará o aplicativo TikMatrix** - nenhuma intervenção manual é necessária.

Durante o processo de inicialização automática, o TikMatrix instalará dois arquivos APK: TikMatrix e TikMatrix Test (sem interface). Se seu telefone solicitar autorização de instalação, simplesmente toque em "Permitir" (especialmente importante em dispositivos Xiaomi).

## Instalação Automática vs Manual

**Instalação Automática (Recomendada)**: Por padrão, o TikMatrix inicializa automaticamente os dispositivos quando são conectados pela primeira vez. Este é o método mais fácil e conveniente.

**Instalação/Atualização Manual**: Necessário apenas em casos especiais ou para atualizações.

## Processo de Inicialização Automática

Quando você conecta um novo dispositivo ao TikMatrix:

1. **Detecção de Dispositivo**: O TikMatrix detecta automaticamente o dispositivo conectado.
2. **Instalação Automática**: O sistema instalará automaticamente o aplicativo TikMatrix sem intervenção do usuário.
3. **Autorização**: Se seu dispositivo solicitar permissão de instalação, simplesmente toque em "Permitir".
4. **Conclusão**: O processo de inicialização é concluído em cerca de 10 segundos.
5. **Pronto para Usar**: Seu dispositivo agora está pronto para tarefas de automação.

Este processo automático elimina a necessidade de configuração manual na maioria dos casos.

## Instalação/Atualização Manual

Para instalar ou atualizar manualmente o APK do TikMatrix, desligue o interruptor `Auto WakeUP Agent` nas Configurações. Depois siga estes passos:

1. Selecione seu telefone e clique em `General` > `Init Agent`.
2. O TikMatrix instalará automaticamente ambos os arquivos APK no seu telefone.
3. Se solicitado, toque em "Permitir" para autorizar a instalação.
4. Se bloqueado pelo Google Play Protect, toque em "Instalar mesmo assim" para continuar.
5. Aguarde a conclusão da instalação (cerca de 10 segundos).
6. Ligue o interruptor `Auto WakeUP Agent` para habilitar a ativação automática do aplicativo quando o telefone for conectado.
7. Clique em `General` > `Open Agent` para testar a instalação. Se o aplicativo falhar ao iniciar, contate o suporte técnico.
