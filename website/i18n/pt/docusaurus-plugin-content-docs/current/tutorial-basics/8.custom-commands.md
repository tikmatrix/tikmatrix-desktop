# Comandos ADB Personalizados

O TikMatrix permite que você crie e gerencie comandos ADB personalizados para execução rápida de operações comuns, simplificando seu fluxo de trabalho e melhorando a eficiência.

## O que São Comandos ADB Personalizados?

Comandos ADB personalizados permitem que você salve instruções ADB usadas frequentemente e as execute com um único clique, eliminando a necessidade de inserir comandos manualmente cada vez.

## Acessando a Interface de Comandos Personalizados

1. Na interface principal do TikMatrix, clique em **Custom Commands** na barra lateral esquerda.
2. A interface exibe todos os seus comandos salvos, com opções para adicionar ou redefinir comandos.

## Comandos Predefinidos

O TikMatrix fornece vários comandos predefinidos úteis:

- **Home**: Retornar à tela inicial do dispositivo
- **Back**: Executar uma operação de voltar
- **Enable TCP**: Habilitar modo de conexão TCP/IP
- **Enable Fast Input**: Habilitar o método de entrada rápida
- **Open Gallery**: Abrir a galeria do dispositivo
- **Open Settings**: Abrir configurações do dispositivo
- **Open NekoBox**: Abrir o aplicativo NekoBox (se instalado)

## Adicionando Comandos Personalizados

1. Clique em **Add Command** no topo da interface.
2. No diálogo, preencha:
   - **Command Name**: Um nome reconhecível para o seu comando
   - **Command Arguments**: Os parâmetros do comando ADB (não é necessário incluir o prefixo "adb")
3. Clique em **Add Command** para salvar.

## Suporte a Comando Multi-linha

Comandos personalizados suportam entrada multi-linha; cada linha é tratada como um comando ADB separado e executada em sequência.
