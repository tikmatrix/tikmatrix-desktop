# Suporte a Aplicativos Clonados

O TikMatrix e IgMatrix suportam a execução de múltiplos aplicativos clonados (como GB TikTok, GB Instagram, Instagram Plus ou outras versões modificadas) no mesmo dispositivo, cada um com seu próprio nome de pacote. Este recurso permite que você gerencie múltiplas contas de forma mais eficiente usando diferentes instâncias de aplicativos.

:::danger Aviso de Risco
Usar aplicativos clonados pode violar os Termos de Serviço do TikTok/Instagram e pode resultar em suspensão ou banimento de conta. Fornecemos apenas suporte técnico para este recurso e **NÃO somos responsáveis** por quaisquer problemas relacionados a contas, incluindo banimentos, suspensões ou restrições causadas pelo uso de aplicativos clonados. Use por sua própria conta e risco.
:::

## O que é um Nome de Pacote?

Todo aplicativo Android tem um identificador único chamado **nome de pacote**. Os aplicativos oficiais usam nomes de pacote padrão:

- TikTok: `com.zhiliaoapp.musically`
- Instagram: `com.instagram.android`

Aplicativos clonados usam nomes de pacote diferentes como:

- GB TikTok: `com.zhiliaoapp.musically.gb`
- GB Instagram: `com.instagram.gb.android`
- Instagram Plus: `com.instagram.plus.android`

O TikMatrix/IgMatrix pode detectar e usar automaticamente o nome de pacote correto para cada conta.

## Configurando Aplicativos Clonados

### Passo 1: Instalar Aplicativos Clonados

1. Instale seu(s) aplicativo(s) clonado(s) preferido(s) no seu dispositivo ou CloudPhone.
2. Certifique-se de que cada aplicativo clonado tenha um nome de pacote único para evitar conflitos.
3. Faça login em diferentes contas em cada instância do aplicativo.

### Passo 2: Configurar Nomes de Pacote

Você tem duas opções para configurar nomes de pacote:

#### Opção A: Configuração no Nível da Conta (Recomendado)

Defina um nome de pacote específico para cada conta:

1. Abra **Accounts** na navegação esquerda.
2. Clique em uma conta para editá-la, ou clique em **Add** para criar uma nova.
3. No campo **Package Name**, você pode:
   - **Inserir manualmente** o nome do pacote do aplicativo clonado (ex.: `com.instagram.gb.android`)
   - **Auto-detectar** clicando no botão **Detect Package** (requer seleção do dispositivo primeiro)
4. Clique em **Save**.

:::tip Auto-Detectar Nome de Pacote
O botão **Detect Package** detecta automaticamente o aplicativo aberto atualmente no dispositivo selecionado. Esta é a maneira mais fácil de obter o nome de pacote correto:

1. Selecione um dispositivo para a conta
2. Abra o aplicativo clonado naquele dispositivo
3. Clique no botão **Detect Package**
4. O nome do pacote será preenchido automaticamente

Este recurso funciona com qualquer aplicativo Android, incluindo clones de TikTok/Instagram, aplicativos do App Cloner e aplicativos do NomixCloner.
:::

Quando você executar scripts para esta conta, o TikMatrix/IgMatrix usará automaticamente o nome de pacote especificado.

#### Opção B: Padrão em Todo o Sistema

Defina um nome de pacote padrão para todas as contas que não tenham um específico:

1. Abra **Settings** na navegação esquerda.
2. Encontre o campo **TikTok Package Name** ou **Instagram Package Name**.
3. Insira o nome de pacote padrão.
4. Clique em **Save**.

:::tip Prioridade
As configurações no nível da conta sempre substituem o padrão do sistema. Se uma conta tiver seu próprio nome de pacote configurado, ele será usado em vez do padrão.
:::

## Como Funciona

Quando você executa qualquer script de automação:

1. O TikMatrix/IgMatrix verifica se a conta de destino tem um nome de pacote personalizado configurado.
2. Se sim, ele usa esse nome de pacote para todas as operações de automação de UI.
3. Se não, ele recorre ao nome de pacote padrão do sistema.
4. Todos os IDs de recursos e consultas XPath são automaticamente formatados com o nome de pacote correto.

## Melhores Práticas

### 1. Use Nomes de Usuário Descritivos

Como você pode ter múltiplas contas em diferentes aplicativos clonados, use nomes de usuário descritivos para identificar a qual aplicativo cada conta pertence:

```text
username_gb      → GB TikTok/Instagram
username_plus    → TikTok/Instagram Plus
username         → Aplicativo Oficial
```

### 2. Organize Contas por Grupos

Crie grupos de dispositivos baseados no tipo de aplicativo clonado:

- **Grupo A**: Todas as contas usando aplicativos GB
- **Grupo B**: Todas as contas usando aplicativos Plus
- **Grupo C**: Todas as contas usando aplicativos oficiais

### 3. Teste Antes de Operações em Massa

Antes de executar scripts em todas as contas:

1. Teste com uma única conta primeiro
2. Verifique se o aplicativo correto está sendo usado
3. Confirme que todas as operações funcionam como esperado

### 4. Mantenha os Nomes de Pacote Consistentes

Uma vez que você definir um nome de pacote para uma conta, não o altere a menos que mude para um aplicativo clonado diferente. A consistência ajuda a evitar confusão e erros.

## Solução de Problemas

### Script Abre o Aplicativo Errado

**Problema**: O script abre o aplicativo errado em vez do aplicativo clonado.

**Solução**:

1. Verifique se o campo **Package Name** da conta está configurado corretamente
2. Verifique se o aplicativo clonado está instalado e o nome do pacote corresponde
3. Use o botão **Detect Package** para obter automaticamente o nome de pacote correto
4. Reinicie o dispositivo se o aplicativo foi instalado recentemente

### Não Consegue Encontrar Elementos de UI

**Problema**: O script relata erros de "elemento não encontrado".

**Solução**:

1. Diferentes aplicativos clonados podem ter layouts de UI ligeiramente diferentes
2. Verifique se a versão do aplicativo clonado é compatível
3. Tente atualizar para a versão mais recente do TikMatrix/IgMatrix
4. Entre em contato com o suporte se o problema persistir

### Múltiplas Contas no Mesmo Aplicativo

**Problema**: Quer alternar entre múltiplas contas na mesma instância do aplicativo.

**Solução**:

- TikTok/Instagram suporta múltiplas contas logadas nativamente
- Use o recurso **Switch Account** no TikMatrix/IgMatrix
- Defina o mesmo nome de pacote para todas as contas naquela instância do aplicativo
- O TikMatrix/IgMatrix lidará com a troca de contas automaticamente

### Botão Detect Package Não Funciona

**Problema**: O botão **Detect Package** não funciona ou mostra um erro.

**Solução**:

1. Certifique-se de ter selecionado um dispositivo primeiro
2. Verifique se o aplicativo está atualmente aberto no dispositivo
3. Confirme se a conexão do dispositivo está estável
4. Tente abrir o aplicativo novamente e clique em detectar

## Encontrando Nomes de Pacote

Se você quiser verificar manualmente os nomes de pacote no seu dispositivo:

### Usando Comando ADB

```bash
# Listar todos os pacotes relacionados ao TikTok
adb shell pm list packages | grep musically

# Listar todos os pacotes relacionados ao Instagram
adb shell pm list packages | grep instagram
```

### Usando o Recurso de Auto-Detectar

A maneira mais fácil é usar o botão **Detect Package** integrado:

1. Abra o diálogo de edição da conta
2. Selecione o dispositivo
3. Abra o aplicativo clonado no dispositivo
4. Clique em **Detect Package**
5. O nome do pacote será automaticamente detectado e preenchido

Este método funciona para qualquer aplicativo Android e é a abordagem recomendada.

## FAQ

**P: Posso usar múltiplos aplicativos clonados no mesmo dispositivo?**

R: Sim, desde que cada aplicativo clonado tenha um nome de pacote único, você pode executar múltiplas instâncias de aplicativo no mesmo dispositivo.

**P: Preciso reinstalar o TikMatrix/IgMatrix?**

R: Não, este recurso está integrado na versão mais recente. Basta atualizar para a versão mais recente e configurar os nomes de pacote.

**P: Meus scripts de automação existentes ainda funcionarão?**

R: Sim, todos os scripts existentes são totalmente compatíveis. Contas sem um nome de pacote personalizado usarão o nome de pacote padrão.

**P: Isso funciona tanto para TikTok quanto para Instagram?**

R: Sim, tanto o TikMatrix quanto o IgMatrix suportam totalmente detecção e automação de aplicativos clonados.

**P: Como funciona o recurso de auto-detectar?**

R: Quando você clica em **Detect Package**, o TikMatrix/IgMatrix verifica qual aplicativo está atualmente em primeiro plano no dispositivo selecionado e recupera seu nome de pacote automaticamente.

**P: Posso usar aplicativos do App Cloner ou NomixCloner?**

R: Sim, o recurso de nome de pacote funciona com qualquer ferramenta de clonagem de aplicativos Android, incluindo App Cloner e NomixCloner. Use o recurso de auto-detectar para obter facilmente o nome de pacote correto.
