# Gerenciamento de Contas

O gerenciamento de contas mantém cada identidade do TikTok ou Instagram sincronizada com os dispositivos em que executam, as credenciais que precisam e os dados dos quais sua automação depende. Uma lista de contas limpa facilita a delegação de dispositivos, monitoramento da saúde das contas e preparação de campanhas sem tocar em cada telefone manualmente.

## Criar uma conta manualmente

1. Abra **Accounts** na navegação esquerda.
2. Clique em **Add** para abrir o diálogo do editor.
3. Preencha os campos básicos:
   - **Email**: o endereço usado pelo aplicativo. TikMatrix/IgMatrix primeiro tenta um login com email + senha; se o email estiver em branco, automaticamente recorre a nome de usuário + senha.
   - **Password**: a credencial usada no aplicativo. IgMatrix também aceita uma senha estendida com um segredo 2FA (veja abaixo).
   - **Username** — o identificador mostrado no TikTok/Instagram. Este é o único campo obrigatório.
   - Alterne **Login Status** para marcar se a sessão do aplicativo já está autenticada no dispositivo (1 = Logado, 0 = Não logado).
   - Alterne **Status** para manter a conta ativa (0 = Habilitada) ou desabilitá-la (1 = Desabilitada).
4. Escolha um badge de dispositivo para vincular a conta a um telefone ou CloudPhone. Se nenhum dispositivo for selecionado, a conta permanece não atribuída.
5. Clique em **Save**. A lista se atualiza automaticamente.

## Login

- Quando tanto **Email** quanto **Password** são fornecidos, TikMatrix/IgMatrix fará login com esse par no dispositivo atribuído.
- Se o campo **Email** estiver vazio, tentará a combinação **Username** + **Password** em vez disso.
- Certifique-se de que um índice de dispositivo esteja selecionado para que o script de login saiba onde executar.
- A automação de login detecta sessões já autenticadas e as pula automaticamente.

## Suporte 2FA (Somente IgMatrix)

IgMatrix pode lidar com autenticação de dois fatores durante logins automatizados.

### Formato de senha 2FA

Para habilitar 2FA, armazene a senha no seguinte formato:

```text
sua_senha:seu_segredo_2fa
```

**Exemplo:**

```text
mypassword123:JBSWY3DPEHPK3PXP
```

## Importando e exportando contas

- **Export** escreve cada conta para `accounts.xlsx` na sua pasta de download do TikMatrix/IgMatrix. Use isso como um backup vivo ou como modelo para edições em massa.
- **Import** permite que você crie ou atualize contas em massa a partir dessa planilha Excel. Se você não tiver certeza sobre o formato do cabeçalho, clique em **Export** primeiro, edite o arquivo baixado e depois importe-o de volta.
- Durante a importação, qualquer linha que contenha um `id` atualiza o registro existente. Linhas sem `id` criam novas contas.

> Para logins 2FA do IgMatrix, preencha `username`, `pwd` (usando o formato `password:2fa_secret`) e a coluna `device_index`.

### Referência de colunas do Excel

| Coluna | Obrigatório | Descrição |
| --- | --- | --- |
| `id` | Não | Identificador interno. Inclua apenas quando pretender atualizar uma conta existente. |
| `email` | Não | Email de login usado pelo aplicativo. Útil para contato e recuperação. |
| `pwd` | Não | Senha armazenada para referência rápida ou logins automatizados. |
| `username` | Sim | Identificador do TikTok/Instagram. Deve ser único dentro do workspace. |
| `device` | Não | Serial do dispositivo ou identificador do CloudPhone vinculado à conta. |
| `device_index` | Não | Rótulo amigável do dispositivo mostrado na UI; principalmente para referência. |
| `logined` | Não | Flag de sessão (`1` = logado, `0` = não logado). |
| `status` | Não | Flag de disponibilidade (`0` = habilitada, `1` = desabilitada). Contas desabilitadas nunca entram em execuções de script. |
| `nickname` | Não | Último apelido recuperado da sincronização de dados do TikTok. |
| `country` | Não | País inferido do perfil do TikTok. |
| `followers` | Não | Número de seguidores sincronizados do TikTok. |
| `following` | Não | Número de contas que este perfil segue. |
| `hearts` | Não | Total de curtidas (corações) no perfil. |
| `videos` | Não | Contagem de vídeos publicados. |
| `friends` | Não | Contagem de amigos mútuos quando disponível. |

### Campos TikMatrix vs. IgMatrix

- **TikMatrix** popula os campos de estatísticas (`nickname`, `country`, `followers`, `following`, `hearts`, `videos`, `friends`) depois que você executar **Sync TikTok Data** ou usar o botão de consulta de dados.
- **IgMatrix** usa apenas as colunas básicas (`email`, `pwd`, `username`, `device`, `logined`, `status`) e ignora as colunas de análise.

## Comportamentos adicionais

- Contas marcadas como **Disabled** são excluídas de cada script de automação, tarefa agendada e ação em massa. Re-habilite-as antes de lançar novos trabalhos.
- O script **Match Accounts** escaneia dispositivos para sessões de aplicativo que já estão logadas, cria automaticamente as entradas correspondentes na lista de contas e as vincula ao dispositivo detectado. Esta é a maneira mais rápida de integrar telefones que foram preparados manualmente.
