# Guia Fácil de Rotação de Proxy

A rotação de proxy ajuda o TikMatrix a mudar automaticamente o endereço IP que um telefone usa. Use-a quando você quiser uma rede limpa antes de fazer login, trocar de contas ou executar tarefas sensíveis.

## O que você precisa

- Um serviço de proxy que suporta "rotacionar IP" ou "mudar IP" clicando em um link ou chamando uma URL simples.
- O link de rotação de proxy (às vezes chamado de *URL de rotação*, *URL de atualização* ou *URL de mudança de IP*).
- Opcional: qualquer senha ou token que seu provedor de proxy exija.

## Salvar o link de rotação para um dispositivo

1. Abra **Manage Devices** no aplicativo TikMatrix desktop e mude para a visualização de tabela.
2. Encontre o dispositivo que você deseja controlar e clique em **Configure** na coluna *Proxy rotation URL*.
3. Preencha o formulário:
   - **Proxy rotation URL**: cole o link do seu provedor de proxy. O TikMatrix abrirá este link toda vez que precisar de um novo IP.
   - **Request method**: mantenha `GET` a menos que seu provedor diga explicitamente para usar `POST`.
   - **Headers** e **Body**: deixe-os vazios a menos que seu provedor forneça um exemplo curto que os use. Se necessário, copie o exemplo deles na caixa exatamente como eles mostram.
   - **Timeout**: quanto tempo o TikMatrix espera que o serviço de proxy responda. O padrão de 10 segundos funciona para a maioria dos serviços.
4. Clique em **Save**. O TikMatrix armazena tudo automaticamente—você não precisa editar nenhum arquivo.

Após salvar, a tabela mostra o último resultado de rotação ao lado do dispositivo. Um rótulo verde significa que a última rotação foi bem-sucedida, um rótulo cinza significa que ainda não foi executada, e um rótulo vermelho significa que o serviço de proxy retornou um erro.

## Verificar se o link funciona

1. Fique na mesma linha da tabela e clique em **Test Proxy Rotation**.
2. O TikMatrix chamará o link imediatamente.  
   - Se funcionar, você verá uma mensagem de sucesso e o tempo "Last rotated" será atualizado.  
   - Se falhar, anote a mensagem, verifique novamente o link e quaisquer senhas, e tente novamente.

Dica: muitos provedores de proxy limitam com que frequência você pode rotacionar. Se o teste falhar, aguarde alguns segundos e tente novamente.

## Usar rotação de proxy durante scripts

1. Ao executar o script, primeiro marque **Proxy Rotation**. O TikMatrix então mudará o IP de cada dispositivo um por um antes de abrir o TikTok.  
2. Você pode clicar no botão de log no lado direito de cada dispositivo para ver informações detalhadas sobre cada rotação.

## Perguntas comuns

- **Preciso rotacionar para cada script?** Não. Use quando você quiser que o TikTok veja um IP novo—por exemplo, antes de alternar entre muitas contas.
- **Onde as configurações são armazenadas?** O TikMatrix gerencia o armazenamento. Você só precisa preencher o formulário uma vez por dispositivo.
- **O link do provedor parece complicado. Ainda funcionará?** Sim. Contanto que o link venha do seu provedor e funcione em um navegador, o TikMatrix também pode abri-lo.
- **Quem pode ajudar se ainda falhar?** Envie as instruções do provedor e qualquer mensagem de erro para o suporte do TikMatrix para que possamos revisá-las juntos.
