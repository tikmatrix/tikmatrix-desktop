# Criar Grupo

Um **Grupo** é uma coleção de dispositivos, facilitando o gerenciamento e operação de múltiplos dispositivos juntos.

## Como Criar um Grupo

1. Clique no botão `Add Group` na barra lateral esquerda.
2. Insira um nome de grupo.
3. Pressione `Enter` para salvar o grupo.
4. Selecione dispositivos para movê-los para o grupo.
