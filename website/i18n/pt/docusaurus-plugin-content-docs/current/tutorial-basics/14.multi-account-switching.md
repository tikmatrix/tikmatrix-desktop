# Troca de Múltiplas Contas

TikTok e Instagram suportam fazer login em múltiplas contas no mesmo dispositivo—este é um recurso nativo dos aplicativos oficiais. TikMatrix e IgMatrix fornecem troca automática de contas para que você possa executar tarefas em todas as contas logadas sem intervenção manual.

## Scripts suportados

Os seguintes scripts suportam troca automática de contas:

- **Profile** — Ver informações de perfil da conta
- **Account Warmup** — Aquecer contas com atividade natural
- **Post** — Publicar posts
- **Mass DM** — Enviar mensagens diretas em massa
- **Follow Back** — Seguir de volta usuários que te seguem
- **Unfollow All** — Deixar de seguir todas as contas
- **Delete Post** — Remover posts publicados
- **Boost Users** — Aumentar engajamento com perfis de usuários
- **Boost Posts** — Aumentar engajamento com posts
- **Boost Lives** — Aumentar engajamento com transmissões ao vivo
- **Mass Comment** — Postar comentários em massa
- **Boost Comments** — Aumentar engajamento com comentários
- **Super Marketing** — Automação de marketing avançada

## Configuração inicial: Combinar Contas

Antes de usar o recurso de troca de múltiplas contas, você deve primeiro combinar contas do seu dispositivo.

1. Abra a aba **Scripts** na navegação esquerda.
2. Encontre e execute o script **Match Accounts**.
3. Este script escaneia o dispositivo para contas logadas e as adiciona automaticamente à sua lista de contas.
4. Após o script ser concluído, clique no botão **Accounts** para verificar se as contas do seu dispositivo foram adicionadas com sucesso.

## Testando a troca de contas

Após executar **Match Accounts**, você pode testar a funcionalidade de troca de contas:

1. Execute o script **Switch Account** da aba Scripts.
2. Este script não tem outra lógica—simplesmente valida que o fluxo de troca de contas funciona corretamente.
3. Se o script tiver sucesso, o recurso de troca de contas está pronto para uso.

## Usando o modo de múltiplas contas

Uma vez que você tenha completado a configuração inicial e o teste:

1. Ao executar qualquer script suportado, habilite **Multi-Account Mode** nas configurações do script.
2. O sistema criará tarefas independentes para cada conta na sua lista de contas.
3. As tarefas são enfileiradas e executadas sequencialmente—o sistema troca automaticamente para a conta alvo e executa a automação.
4. Nenhuma intervenção manual é necessária.

## Quando executar novamente Match Accounts

Os scripts **Match Accounts** e **Switch Account** normalmente só precisam ser executados uma vez durante a configuração inicial. No entanto, você deve executar novamente **Match Accounts** se:

- Você fizer login em novas contas no dispositivo
- Você fizer logout de contas existentes
- As contas no dispositivo mudaram

Para atualizar a lista de contas:

1. Limpe manualmente a lista de contas na seção **Accounts**.
2. Execute novamente o script **Match Accounts** para detectar as contas atuais no dispositivo.

---

A troca de múltiplas contas elimina a necessidade de trocar manualmente entre contas, facilitando a escalabilidade da sua automação em todas as contas em um único dispositivo.
