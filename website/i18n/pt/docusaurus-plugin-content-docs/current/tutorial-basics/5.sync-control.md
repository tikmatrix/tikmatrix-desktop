# Controle Sincronizado

O Controle Sincronizado permite sincronizar operações em múltiplos dispositivos.

## Como Usar

1. Abra a tela de um telefone.
2. Selecione múltiplos dispositivos da lista à esquerda.
3. Todas as operações de tela serão sincronizadas para os dispositivos selecionados.
