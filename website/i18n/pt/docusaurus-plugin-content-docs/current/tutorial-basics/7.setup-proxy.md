# Configurar Proxy

Você pode definir um endereço IP proxy único para cada telefone.

> **Isenção de responsabilidade:** Todos os sites recomendados aqui são provenientes da internet e não são afiliados a este site. Por favor, verifique sua validade antes de usar. Não fornecemos nenhuma garantia.

## 1. Configurar Proxy VPS Residencial (Se você comprar um IP Proxy, pode pular esta etapa)

1. Compre um VPS de IP proxy residencial do seu país-alvo (preferencialmente com IP nativo e VPS de ISP duplo).
2. Use [https://github.com/MHSanaei/3x-ui](https://github.com/MHSanaei/3x-ui) para configurar rapidamente uma VPN no VPS:

    ```shell
    bash <(curl -Ls https://raw.githubusercontent.com/MHSanaei/3x-ui/refs/tags/v2.6.0/install.sh)
    # Siga as instruções para completar a configuração da VPN.
    ```

3. Após a instalação, acesse a interface web x-ui. Você pode encontrar a URL e credenciais de login na saída do terminal após a instalação.
4. Crie um novo usuário no x-ui, defina o protocolo como `SOCKS5` e configure a porta e outras configurações conforme necessário.
5. Após criar o usuário, você verá o IP proxy e a porta na interface x-ui.
6. Use o IP proxy e a porta no aplicativo NekoBox no seu telefone.

## 3. Habilitar Proxy no telefone

1. Baixe o APK e instale o cliente [NekoBoxForAndroid](https://github.com/MatsuriDayo/NekoBoxForAndroid) no seu telefone.
2. Abra o aplicativo NekoBox e clique no botão `+` para adicionar um novo proxy.
3. Selecione `SOCKS5` ou outros tipos de proxy.
4. Insira o IP proxy e a porta, depois clique em `Save`.
5. Clique no botão `Start` para habilitar o proxy.
6. Após o proxy ser habilitado, você pode verificar o status da conexão no aplicativo NekoBox.

### Para Usuários Chineses

Se você está na China, o protocolo `SOCKS5` pode não funcionar bem. Você precisa adicionar um proxy de criptografia (`VLESS`) como proxy frontend. Você pode usar as seguintes ferramentas:

1. Adicione um proxy `VLESS` no aplicativo NekoBox.
2. Adicione um proxy `SOCKS5` no aplicativo NekoBox.
3. Adicione um proxy `Chain` no aplicativo NekoBox. E selecione o proxy `VLESS` como proxy frontend e o proxy `SOCKS5` como proxy backend.
