# 튜토리얼 소개

**TikMatrix**와 **IgMatrix**의 핵심 기능과 사용법을 5분 이내에 알아보세요.

## 소프트웨어 소개

**TikMatrix**는 Android 기기에서 TikTok 작업을 자동화하는 PC 데스크톱 소프트웨어입니다.  
**IgMatrix**는 Android 기기에서 Instagram 작업을 자동화하는 PC 데스크톱 소프트웨어입니다.

**참고:** **TikMatrix**와 **IgMatrix**는 독립적인 두 개의 소프트웨어이며, 동일한 컴퓨터에서 동시에 실행할 수 없습니다.

## 주요 기능

- **다중 계정 전환**: 동일한 기기에서 여러 TikTok/Instagram 계정에 로그인하고 자동으로 전환하여 작업 실행.
- **클론 앱 지원**: 커스텀 패키지명을 사용하는 TikTok/Instagram 클론 앱 지원, 다른 계정을 서로 다른 클론 앱에 바인딩하고 자동으로 대상 앱 열기.
- ~~**자동 가입**~~: **[서비스 종료]**
- **자동 로그인**: Instagram은 2FA 자동 로그인 지원; TikTok은 일부 인증(예: 그래픽 인증, 이메일 인증)에서 수동 개입 필요.
- **자동 프로필 작성**: 아바타, 닉네임, 소개, 사용자 이름 설정 지원.
- **자동 인식**: 휴대폰에 이미 로그인된 TikTok/Instagram 계정을 원클릭으로 인식 및 가져오기.
- **자동 계정 워밍업**: 키워드 검색, 동영상 시청, 좋아요, 팔로우, 댓글, 즐겨찾기 등 워밍업 활동 지원.
- **자동 게시**: 동영상 또는 이미지 게시 지원(정렬 및 배치 수량 설정 가능), 커스텀 음악 지원, 제목, 설명, 해시태그, 상품 링크 포함.
- **자동 삭제**: 조회수 또는 지정된 규칙에 따라 성과가 낮은 게시물 삭제.
- **슈퍼 마케팅**: 실제 사용자 방문 경로 시뮬레이션(프로필 방문, 게시물 탐색, 랜덤 팔로우/좋아요/댓글/즐겨찾기/공유/DM), OpenAI 또는 호환 API를 사용하여 개인화된 DM 및 댓글 생성 지원.
- **일괄 부스팅(라이브)**: 라이브 방송에 대한 대량 좋아요 및 댓글.
- **사용자 수집**: 키워드, 팔로워 또는 팔로잉 목록을 통해 대상 사용자 이름 수집.
- **일괄 팔로우백**: 팔로워를 대량으로 팔로우하고 환영 DM 전송.
- **작업 스케줄링**: 분 단위 정밀 스케줄링, 자동 재시도, 간격 실행 및 최대 재시도 횟수 설정 지원.
- **다중 기기 병렬 처리**: 한 대의 컴퓨터로 여러 Android 기기를 동시에 연결하고 관리하며 병렬로 작업 실행.
- **화면 미러링**: 여러 기기 화면을 Windows/macOS에 미러링하고 마우스 및 키보드로 제어 지원.
- **계정 가져오기/내보내기**: Excel 형식으로 대량 가져오기 및 내보내기 지원.
- **다국어 지원**: PC 소프트웨어는 영어, 러시아어, 중국어 지원. 참고: 일부 기능은 최상의 호환성을 위해 휴대폰의 TikTok/Instagram 앱 언어가 영어여야 합니다.
- **티켓 지원**: 사용 중 문제 해결을 위한 전문 티켓 서비스 제공.

> 참고: **동시 실행**은 동일한 PC에서 자동화 작업을 동시에 실행할 수 있는 휴대폰 수를 의미합니다. 기본 기능(기기 관리, 연결 및 시각적 제어, 로그 및 통계 등)은 무료로 제공되며, 자동화 스크립트 기능은 플랜에 따라 해당 동시 실행 할당량이 제공됩니다.

**결제 세부사항**:

- 연간 결제 시 30% 할인.  
- 결제 방법: 신용카드, 알리페이, 암호화폐.  
- 각 라이선스는 한 대의 컴퓨터에 바인딩되며, 무료 컴퓨터 변경 지원.  
- 연결 가능한 휴대폰 수는 컴퓨터 하드웨어에 제한될 수 있음(예: 표준 메인보드는 약 40개의 USB 기기 연결 가능).

플랜 업그레이드 규칙: 시스템은 남은 기간에 따라 차액을 자동으로 계산하며, 업그레이드 시 신규 플랜 가격에서 현재 플랜 잔여 가치를 차감한 금액을 청구합니다.

컴퓨터를 변경해야 하는 경우 라이선스 마이그레이션 가이드를 참조하세요: [라이선스 마이그레이션 가이드](tutorial-basics/license-migration.md).
