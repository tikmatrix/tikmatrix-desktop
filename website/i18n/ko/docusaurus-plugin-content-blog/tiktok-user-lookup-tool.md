## 🔎 2. 확인할 수 있는 내용

| 카테고리 | 필드 |
|---|---|
| 프로필 | 사용자 이름, 닉네임, 아바타 |
| 규모 | 팔로워, 팔로잉, 총 좋아요, 비디오 수 |
| ID | UserID, SecUID |
| 메타 | **국가**, 계정 생성 날짜 |
| 추가 | 빠른 분석, **비디오 다운로드** 링크 (공개 비디오) |

> 팁: 국가는 공개 신호에서 추론됩니다. 거주지 증명이 아닌 지표로 취급하세요.

## ⚡ 3. 사용 방법 (3단계)

1) **[https://user.tikmatrix.com/](https://user.tikmatrix.com/)** 열기
2) TikTok **사용자 이름** 입력 (예: `charlidamelio`)
3) **검색** 클릭 → 결과가 즉시 렌더링됩니다
