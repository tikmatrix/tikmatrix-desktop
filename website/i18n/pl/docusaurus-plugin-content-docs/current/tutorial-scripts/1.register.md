# Rejestracja

:::warning Skrypt wycofany
Skrypt rejestracji został **wycofany** i **nie jest już utrzymywany**. Ta funkcja nie jest już dostępna w TikMatrix.
:::

~~1. Wybierz urządzenie, na którym chcesz uruchomić skrypt.~~
