# Logowanie

Skrypt logowania służy do logowania się na konta mediów społecznościowych.

## Kroki

1. Upewnij się, że konto jest dodane do listy kont (kliknij `Accounts`, aby dodać konta), i wybierz indeks urządzenia, którego chcesz użyć dla tego konta.
2. Wybierz jedno lub wiele urządzeń, na których chcesz uruchomić skrypt logowania.
3. Kliknij `Script` > `Login`.
4. Kliknij `Start Script`, aby rozpocząć.

## Priorytet logowania

System używa następującego priorytetu logowania:

1. **E-mail + Hasło**: Jeśli dla konta skonfigurowany jest e-mail, zostanie on użyty do logowania jako pierwszy.
2. **Nazwa użytkownika + Hasło**: Jeśli e-mail nie jest skonfigurowany, do logowania zostanie użyta nazwa użytkownika.

## Wsparcie dla 2FA (tylko IgMatrix)

IgMatrix obsługuje automatyczne logowanie z uwierzytelnianiem dwuskładnikowym (2FA):

### Format hasła dla 2FA

Aby użyć 2FA, sformatuj hasło w następujący sposób:

```plaintext
twoje_hasło:twój_klucz_sekretny_2fa
```

**Przykład:**

```plaintext
mojehaslo123:JBSWY3DPEHPK3PXP
```

### Jak działa 2FA

1. **Parsowanie hasła**: System automatycznie wykrywa, czy hasło zawiera separator dwukropka (`:`).
2. **Generowanie tokenu**: Gdy wymagane jest 2FA, system używa klucza sekretnego do żądania kodu TOTP z naszej usługi 2FA.
3. **Automatyczne wprowadzanie**: Wygenerowany 6-cyfrowy kod jest automatycznie wprowadzany w pole 2FA Instagram.
4. **Płynne logowanie**: Cały proces 2FA odbywa się automatycznie bez ręcznej interwencji.

### Klucz sekretny 2FA

Klucz sekretny 2FA to zazwyczaj 32-znakowy ciąg zakodowany w Base32, który otrzymujesz podczas konfigurowania 2FA w Instagram. To ten sam klucz, który jest używany przez aplikacje uwierzytelniające, takie jak Google Authenticator lub Authy.

## Uwaga

- Logowanie do TikTok nie jest w 100% w pełni automatyczne. Może być konieczne ręczne wprowadzenie kodów weryfikacyjnych e-mail lub wykonanie innych kroków weryfikacyjnych. Nie możemy ominąć wyzwań logowania specyficznych dla TikTok (takich jak obrazkowe CAPTCHA lub weryfikacja człowieka) ani rozwiązać niektórych komunikatów o błędach, takich jak "zbyt wiele prób logowania".
- Instagram może osiągnąć 100% zautomatyzowane logowanie za pomocą 2FA. W przypadku kont Instagram z włączoną funkcją 2FA upewnij się, że używasz prawidłowego formatu klucza sekretnego w haśle.
