# Dopasuj konta

Skrypt dopasowywania kont służy do rozpoznawania kont TikTok z telefonu i dodawania ich do listy kont.

## Kroki

1. Wybierz urządzenie, na którym chcesz uruchomić skrypt.
2. Kliknij `Script` > `Match Accounts`.
3. Kliknij `Start Script`, aby rozpocząć.
4. Skrypt automatycznie rozpozna konta na telefonie i doda je do listy kont.

## Uwaga

- Skrypt dopasowywania kont może nie rozpoznać wszystkich kont w jednym uruchomieniu. Jeśli niektóre konta zostaną pominięte, spróbuj uruchomić skrypt ponownie.
