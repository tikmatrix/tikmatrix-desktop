# Tworzenie grupy

**Grupa** to zbiór urządzeń, ułatwiający zarządzanie i obsługę wielu urządzeń jednocześnie.

## Jak utworzyć grupę

1. Kliknij przycisk `Add Group` na lewym pasku bocznym.
2. Wprowadź nazwę grupy.
3. Naciśnij `Enter`, aby zapisać grupę.
4. Wybierz urządzenia, aby przenieść je do grupy.
