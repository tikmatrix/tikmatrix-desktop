# Łatwy przewodnik rotacji proxy

Rotacja proxy pomaga TikMatrix automatycznie zmieniać adres IP używany przez telefon. Używaj jej, gdy chcesz czystej sieci przed zalogowaniem, przełączeniem kont lub uruchomieniem wrażliwych zadań.

## Czego potrzebujesz

- Usługa proxy obsługująca "rotację IP" lub "zmianę IP" poprzez kliknięcie linku lub wywołanie prostego URL.
- Link do rotacji proxy (czasami nazywany *URL rotacji*, *URL odświeżania* lub *URL zmiany IP*).
- Opcjonalnie: hasło lub token wymagany przez dostawcę proxy.

## Zapisz link rotacji dla urządzenia

1. Otwórz **Manage Devices** w aplikacji TikMatrix desktop i przełącz się na widok tabeli.
2. Znajdź urządzenie, którym chcesz sterować, i kliknij **Configure** w kolumnie *Proxy rotation URL*.
3. Wypełnij formularz:
   - **Proxy rotation URL**: wklej link od dostawcy proxy. TikMatrix otworzy ten link za każdym razem, gdy będzie potrzebny nowy IP.
   - **Request method**: zachowaj `GET`, chyba że Twój dostawca wyraźnie powie Ci, aby użyć `POST`.
   - **Headers** i **Body**: pozostaw puste, chyba że Twój dostawca poda Ci krótki przykład, który ich używa. Jeśli to konieczne, skopiuj ich przykład do pola dokładnie tak, jak pokazują.
   - **Timeout**: jak długo TikMatrix czeka na odpowiedź usługi proxy. Domyślne 10 sekund działa dla większości usług.
4. Kliknij **Save**. TikMatrix zapisuje wszystko automatycznie — nie musisz edytować żadnych plików.

Po zapisaniu tabela pokazuje ostatni wynik rotacji obok urządzenia. Zielona etykieta oznacza, że ostatnia rotacja się powiodła, szara etykieta oznacza, że jeszcze nie została uruchomiona, a czerwona etykieta oznacza, że usługa proxy zwróciła błąd.

## Sprawdź, czy link działa

1. Pozostań w tym samym wierszu tabeli i kliknij **Test Proxy Rotation**.
2. TikMatrix natychmiast wywoła link.  
   - Jeśli działa, zobaczysz komunikat o powodzeniu, a czas "Last rotated" zostanie zaktualizowany.  
   - Jeśli się nie powiedzie, zanotuj komunikat, sprawdź link i hasła, a następnie spróbuj ponownie.

Wskazówka: wielu dostawców proxy ogranicza częstotliwość rotacji. Jeśli test się nie powiedzie, poczekaj kilka sekund i spróbuj ponownie.

## Użyj rotacji proxy podczas skryptów

1. Podczas uruchamiania skryptu najpierw zaznacz **Proxy Rotation**. TikMatrix zmieni wtedy IP dla każdego urządzenia jeden po drugim przed otwarciem TikTok.  
2. Możesz kliknąć przycisk dziennika po prawej stronie każdego urządzenia, aby wyświetlić szczegółowe informacje o każdej rotacji.

## Często zadawane pytania

- **Czy muszę rotować dla każdego skryptu?** Nie. Używaj jej, gdy chcesz, aby TikTok widział świeży IP — na przykład przed przełączaniem między wieloma kontami.
- **Gdzie są przechowywane ustawienia?** TikMatrix obsługuje przechowywanie. Musisz wypełnić formularz tylko raz na urządzenie.
- **Link dostawcy wygląda skomplikowanie. Czy nadal będzie działać?** Tak. O ile link pochodzi od Twojego dostawcy i działa w przeglądarce, TikMatrix również może go otworzyć.
- **Kto może pomóc, jeśli nadal nie działa?** Wyślij instrukcje dostawcy i komunikat o błędzie do wsparcia TikMatrix, abyśmy mogli je razem przejrzeć.
