# Kontrola synchronizacji

Kontrola synchronizacji umożliwia synchronizację operacji na wielu urządzeniach.

## Jak używać

1. Otwórz ekran jednego telefonu.
2. Wybierz wiele urządzeń z listy po lewej stronie.
3. Wszystkie operacje na ekranie zostaną zsynchronizowane z wybranymi urządzeniami.
