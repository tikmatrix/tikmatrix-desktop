# Zarządzanie kontami

Zarządzanie kontami utrzymuje synchronizację każdej tożsamości TikTok lub Instagram z urządzeniami, na których działają, danymi uwierzytelniającymi, których potrzebują, oraz danymi, od których zależy Twoja automatyzacja. Czysta lista kont ułatwia delegowanie urządzeń, monitorowanie kondycji konta i przygotowywanie kampanii bez ręcznego dotykania każdego telefonu.

## Ręczne tworzenie konta

1. Otwórz **Accounts** w lewej nawigacji.
2. Kliknij **Add**, aby otworzyć okno dialogowe edytora.
3. Wypełnij podstawowe pola:
   - **Email**: adres używany przez aplikację. TikMatrix/IgMatrix najpierw próbuje zalogować się za pomocą email + hasło; jeśli email jest pusty, automatycznie przełącza się na username + hasło.
   - **Password**: dane uwierzytelniające używane w aplikacji. IgMatrix akceptuje również hasło rozszerzone o sekret 2FA (patrz poniżej).
   - **Username** — uchwyt pokazywany w TikTok/Instagram. To jest jedyne wymagane pole.
   - Przełącz **Login Status**, aby oznaczyć, czy sesja aplikacji jest już uwierzytelniona na urządzeniu (1 = Zalogowany, 0 = Niezalogowany).
   - Przełącz **Status**, aby utrzymać konto aktywne (0 = Włączone) lub wyłączyć je (1 = Wyłączone).
4. Wybierz oznaczenie urządzenia, aby powiązać konto z telefonem lub CloudPhone. Jeśli nie wybrano urządzenia, konto pozostaje nieprzypisane.
5. Kliknij **Save**. Lista odświeża się automatycznie.

## Logowanie

- Gdy podane są zarówno **Email**, jak i **Password**, TikMatrix/IgMatrix zaloguje się za pomocą tej pary na przypisanym urządzeniu.
- Jeśli pole **Email** jest puste, spróbuje zamiast tego kombinacji **Username** + **Password**.
- Upewnij się, że wybrano indeks urządzenia, aby skrypt logowania wiedział, gdzie uruchomić.
- Automatyzacja logowania wykrywa już uwierzytelnione sesje i automatycznie je pomija.

## Obsługa 2FA (tylko IgMatrix)

IgMatrix może obsługiwać uwierzytelnianie dwuskładnikowe podczas automatycznych logowań.

### Format hasła 2FA

Aby włączyć 2FA, zapisz hasło w następującym formacie:

```text
twoje_hasło:twój_sekret_2fa
```

**Przykład:**

```text
mojehaslo123:JBSWY3DPEHPK3PXP
```

## Import i eksport kont

- **Export** zapisuje każde konto do `accounts.xlsx` w folderze pobierania TikMatrix/IgMatrix. Użyj tego jako kopii zapasowej lub jako szablonu do masowych edycji.
- **Import** pozwala masowo tworzyć lub aktualizować konta z tego arkusza Excel. Jeśli nie jesteś pewien formatu nagłówka, najpierw kliknij **Export**, edytuj pobrany plik, a następnie zaimportuj go z powrotem.
- Podczas importu każdy wiersz zawierający `id` aktualizuje istniejący rekord. Wiersze bez `id` tworzą nowe konta.

> W przypadku logowań 2FA IgMatrix wypełnij kolumny `username`, `pwd` (używając formatu `hasło:sekret_2fa`) i `device_index`.

### Odniesienie do kolumn Excel

| Kolumna | Wymagane | Opis |
| --- | --- | --- |
| `id` | Nie | Identyfikator wewnętrzny. Dołącz go tylko wtedy, gdy zamierzasz zaktualizować istniejące konto. |
| `email` | Nie | Email logowania używany przez aplikację. Przydatny do kontaktu i odzyskiwania. |
| `pwd` | Nie | Zapisane hasło do szybkiego odniesienia lub automatycznych logowań. |
| `username` | Tak | Uchwyt TikTok/Instagram. Musi być unikalny w obszarze roboczym. |
| `device` | Nie | Numer seryjny urządzenia lub identyfikator CloudPhone powiązany z kontem. |
| `device_index` | Nie | Przyjazna etykieta urządzenia pokazywana w interfejsie użytkownika; głównie do odniesienia. |
| `logined` | Nie | Flaga sesji (`1` = zalogowany, `0` = niezalogowany). |
| `status` | Nie | Flaga dostępności (`0` = włączone, `1` = wyłączone). Wyłączone konta nigdy nie biorą udziału w uruchomieniach skryptów. |
| `nickname` | Nie | Najnowszy pseudonim pobrany z synchronizacji danych TikTok. |
| `country` | Nie | Kraj wywnioskowany z profilu TikTok. |
| `followers` | Nie | Liczba obserwujących zsynchronizowana z TikTok. |
| `following` | Nie | Liczba kont, które śledzi ten profil. |
| `hearts` | Nie | Łączna liczba polubień (serc) na profilu. |
| `videos` | Nie | Liczba opublikowanych filmów. |
| `friends` | Nie | Liczba wzajemnych przyjaciół, gdy jest dostępna. |

### Pola TikMatrix vs. IgMatrix

- **TikMatrix** wypełnia pola statystyk (`nickname`, `country`, `followers`, `following`, `hearts`, `videos`, `friends`) po uruchomieniu **Sync TikTok Data** lub użyciu przycisku zapytania o dane.
- **IgMatrix** używa tylko podstawowych kolumn (`email`, `pwd`, `username`, `device`, `logined`, `status`) i ignoruje kolumny analityczne.

## Dodatkowe zachowania

- Konta oznaczone jako **Disabled** są wykluczone z każdego skryptu automatyzacji, zaplanowanego zadania i akcji zbiorczej. Włącz je ponownie przed uruchomieniem nowych zadań.
- Skrypt **Match Accounts** skanuje urządzenia w poszukiwaniu sesji aplikacji, które są już zalogowane, automatycznie tworzy odpowiednie wpisy na liście kont i łączy je z wykrytym urządzeniem. Jest to najszybszy sposób na wdrożenie telefonów, które zostały przygotowane ręcznie.
