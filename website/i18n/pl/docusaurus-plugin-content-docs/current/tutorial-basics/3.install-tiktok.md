# Instalacja TikTok

Zalecamy pobranie konkretnych wersji TikTok poniżej, ponieważ nowsze wersje mogą nie być kompatybilne. Obsługiwane wersje będą ogłaszane w grupie po aktualizacjach.

## Pobierz APK TikTok

- Wersja globalna: [com.zhiliaoapp.musically](https://apkpure.com/tiktok-musically-2024/com.zhiliaoapp.musically)
- Wersja azjatycka: [com.ss.android.ugc.trill](https://apkpure.com/tiktok/com.ss.android.ugc.trill)

## Kroki instalacji

1. Pobierz APK z powyższych linków.
2. Wybierz wszystkie urządzenia w TikMatrix.
3. Kliknij `General` > `Install APK` i wybierz pobrany plik APK.
4. Poczekaj na zakończenie instalacji (może potrwać 1-3 minuty na urządzenie w zależności od modelu; nie klikaj wielokrotnie).
5. Po instalacji kliknij `Settings` i sprawdź, czy wersja TikTok odpowiada zainstalowanej aplikacji.
6. Kliknij `Open TikTok`, aby sprawdzić, czy aplikacja jest poprawnie zainstalowana.
7. Upewnij się, że język TikTok jest ustawiony na angielski; w przeciwnym razie skrypty mogą nie działać poprawnie. W razie potrzeby zmień język w ustawieniach TikTok.
