# Konfiguracja Proxy

Możesz ustawić unikalny adres IP proxy dla każdego telefonu.

> **Zastrzeżenie:** Wszystkie strony internetowe zalecane tutaj pochodzą z Internetu i nie są powiązane z tą stroną. Przed użyciem sprawdź ich ważność. Nie udzielamy żadnej gwarancji.

## 1. Konfiguracja VPS proxy rezydencjalnego (Jeśli kupiłeś proxy IP, możesz pominąć ten krok)

1. Kup VPS z proxy rezydencjalnym IP z kraju docelowego (najlepiej z natywnym IP i VPS z podwójnym ISP).
2. Użyj [https://github.com/MHSanaei/3x-ui](https://github.com/MHSanaei/3x-ui), aby szybko skonfigurować VPN na VPS:

    ```shell
    bash <(curl -Ls https://raw.githubusercontent.com/MHSanaei/3x-ui/refs/tags/v2.6.0/install.sh)
    # Postępuj zgodnie z instrukcjami, aby zakończyć konfigurację VPN.
    ```

3. Po instalacji uzyskaj dostęp do interfejsu internetowego x-ui. Adres URL i dane logowania znajdziesz w wyjściu terminala po instalacji.
4. Utwórz nowego użytkownika w x-ui, ustaw protokół na `SOCKS5` i skonfiguruj port oraz inne ustawienia według potrzeb.
5. Po utworzeniu użytkownika zobaczysz proxy IP i port w interfejsie x-ui.
6. Użyj proxy IP i portu w aplikacji NekoBox na swoim telefonie.

> Zalecani dostawcy VPS

- [KliknijTutaj](/advertising)

## 2. Zakup IP proxy rezydencjalnego (Jeśli skonfigurowałeś VPS, możesz pominąć ten krok)

Możesz również kupić bezpośrednio IP proxy rezydencyjne bez konfigurowania VPS.

> Zalecani dostawcy proxy

- [KliknijTutaj](/advertising)

> Narzędzia do sprawdzania IP

- [ipinfo](https://ipinfo.io) — Sprawdź, czy IP jest rezydencyjny.
- [scamalytics](https://scamalytics.com/ip) — Sprawdź poziom ryzyka IP.

## 3. Włącz proxy na telefonie

1. Pobierz APK i zainstaluj klienta [NekoBoxForAndroid](https://github.com/MatsuriDayo/NekoBoxForAndroid) na swoim telefonie.
2. Otwórz aplikację NekoBox i kliknij przycisk `+`, aby dodać nowe proxy.
3. Wybierz `SOCKS5` lub inne typy proxy.
4. Wprowadź proxy IP i port, a następnie kliknij `Save`.
5. Kliknij przycisk `Start`, aby włączyć proxy.
6. Po włączeniu proxy możesz sprawdzić status połączenia w aplikacji NekoBox.

### Dla użytkowników chińskich

Jeśli jesteś w Chinach, protokół `SOCKS5` może nie działać dobrze. Musisz dodać zaszyfrowane proxy (`VLESS`) jako proxy frontendowe. Możesz użyć następujących narzędzi:

1. Dodaj proxy `VLESS` w aplikacji NekoBox.
2. Dodaj proxy `SOCKS5` w aplikacji NekoBox.
3. Dodaj proxy `Chain` w aplikacji NekoBox. I wybierz proxy `VLESS` jako proxy frontendowe, a proxy `SOCKS5` jako proxy backendowe.
