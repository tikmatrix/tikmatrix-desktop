# Błąd uruchamiania oprogramowania

## Rozwiązywanie typowych błędów uruchamiania

### Krok 1: Sprawdź wymagania systemowe

1. Upewnij się, że Twój system spełnia minimalne wymagania:
   - Windows 10 lub nowszy
   - Co najmniej 4GB RAM
   - Co najmniej 1GB wolnego miejsca na dysku

### Krok 2: Sprawdź Microsoft Visual C++ Redistributable

Jeśli widzisz błędy takie jak `vcruntime140.dll not found` lub `The application was unable to start correctly (0xc000007b)`, możesz nie mieć zainstalowanego Microsoft Visual C++ Redistributable.

1. Naciśnij `Windows + R`, wpisz `cmd` i naciśnij Enter.
2. Uruchom: `dir "C:\Windows\System32\vcruntime*.dll"`
3. Jeśli widzisz wynik, redistributable jest zainstalowany. Nazwa pliku (np. vcruntime140.dll) wskazuje wersję.
4. Jeśli nie, pobierz i zainstaluj najnowszą wersję: [Microsoft Visual C++ Redistributable](https://learn.microsoft.com/en-us/cpp/windows/latest-supported-vc-redist?view=msvc-170)

### Krok 3: Sprawdź czas systemowy

Nieprawidłowy czas systemowy może powodować błędy „Agent Start Timeout":

1. Kliknij prawym przyciskiem myszy na wyświetlacz czasu w prawym dolnym rogu ekranu
2. Wybierz „Dostosuj datę/godzinę"
3. Włącz „Ustaw czas automatycznie", jeśli nie jest już włączony
4. Jeśli czas jest nadal nieprawidłowy, kliknij „Synchronizuj teraz" lub ręcznie ustaw prawidłową datę i godzinę
5. Uruchom ponownie TikMatrix po skorygowaniu czasu

### Krok 4: Sprawdź logi aplikacji

1. Kliknij `Ustawienia` > `Otwórz katalog aplikacji` w TikMatrix.
2. Przejdź do folderu `logs`.
3. Sprawdź najnowszy plik dziennika pod kątem komunikatów o błędach.

### Krok 5: macOS — Agent InvalidStateError

Na niektórych komputerach macOS agent może nie uruchomić się i pokazać błąd taki jak:

```text
Agent start error InvalidStateError: The object is in an invalid state.
```

Możliwa przyczyna:

- Jest to często spowodowane niekompatybilnością między aplikacją a wersją systemu macOS lub specyficznym sprzętem/firmware'em na starszych komputerach.

Obejścia / rozwiązania:

1. Zaktualizuj macOS do najnowszej dostępnej wersji, używając Ustawienia systemowe > Ogólne > Aktualizacja oprogramowania. Po aktualizacji uruchom ponownie swojego Maca i spróbuj uruchomić TikMatrix ponownie.
2. Jeśli aktualizacja nie jest możliwa lub błąd się utrzymuje, spróbuj uruchomić aplikację na innym Macu z nowszą wersją macOS.

Jeśli nadal widzisz błąd po wypróbowaniu powyższych rozwiązań, zbierz logi aplikacji i skontaktuj się z naszą pomocą na [Telegram](https://t.me/tikmatrix_agent_bot) podając wersję macOS i logi.

- Kroki, które już wypróbowałeś
