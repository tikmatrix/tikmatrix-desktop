# Nie można wykryć telefonu

## Kroki rozwiązywania problemów

### Krok 1: Sprawdź połączenie ADB

1. Kliknij `Ustawienia` > `Otwórz terminal ADB` w TikMatrix.
2. Uruchom `adb devices` i sprawdź, czy Twój telefon pojawia się na liście ze statusem `device`.

### W systemach Windows

```shell
.\adb.exe devices
```

### W systemach Mac/Linux

```shell
./adb devices
```

### Przykład wyniku

```shell
List of devices attached
<DEVICE_ID>    <STATUS>
```

### Jeśli urządzenie nie jest wymienione

Postępuj zgodnie z krokami rozwiązywania problemów w [Połącz urządzenie](../tutorial-basics/1.connect-to-devices.md).

### Jeśli status urządzenia to `unauthorized`

Oznacza to, że urządzenie nie jest autoryzowane. Dotknij `Zezwól na debugowanie USB` na swoim telefonie.

### Jeśli status urządzenia to `offline`

Zwykle oznacza to luźne połączenie USB. Ponownie podłącz telefon, odłączając i podłączając ponownie kabel USB.

### Jeśli status urządzenia to `device`

Urządzenie zostało pomyślnie połączone. Jeśli TikMatrix nadal nie może wykryć telefonu, upewnij się, że TikMatrix jest aktywowany (bezpłatny okres próbny również wymaga aktywacji).
