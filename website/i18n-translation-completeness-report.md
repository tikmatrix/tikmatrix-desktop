## ⚠️ 关于语言检测警告的说明

检测工具使用 AI 语言检测库 (franc-min) 来验证翻译文本是否使用了正确的目标语言。所有 20 种语言都显示了一些"语言检测警告"，但这些警告**并不表示翻译有问题**，而是由以下原因造成的：

### 常见的假阳性原因：

1. **技术术语和品牌名称**
   - 包含 "Windows"、"macOS"、"Ubuntu"、"TikTok"、"Instagram" 等英文品牌名和技术术语
   - 这些词在所有语言版本中通常保持英文原文

2. **混合语言文本**
   - 例如："操作系统：Windows 10/11、macOS 10.15+ 或 Ubuntu 20.04+"
   - 包含中文描述和英文技术信息的混合文本可能被误识别

3. **语言相似性**
   - 西班牙语 (es) 与葡萄牙语 (pt) 相似度高，容易误判
   - 印尼语 (id) 被检测为马来语 (zlm) - 这两种语言非常接近
   - 荷兰语 (nl) 与德语 (de) 相似

4. **短文本或技术内容**
   - API 描述、版本号、路径等技术内容
   - 短文本（< 30 字符）的语言检测准确率较低

### 语言检测警告最多的语言：

1. **泰语 (th)** - 52 个警告
2. **越南语 (vi)** - 52 个警告

这些语言显示相同数量的警告，说明它们遇到了相同的英文技术术语，这些术语在翻译中被保留。

### 语言检测警告最少的语言：

**建议**: 这些混合语言文本应该被审查和修正。
