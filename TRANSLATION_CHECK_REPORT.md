## 🛠️ 技术说明

### 检测方法

- **工具**: Python langdetect 库
- **文本预处理**:
  - 移除import语句和代码块
  - 移除HTML/JSX标签
  - 移除URL和email
  - 移除技术术语和品牌标识
  - 保留纯文本内容进行检测

### 置信度分级

| 状态 | 置信度要求 | 说明 |
|------|-----------|------|
| ✅ CORRECT | ≥ 90% | 高置信度正确翻译 |
| ⚠️ WARNING | 70-90% | 可能有问题，需人工审查 |
| ❌ INCORRECT | < 50% | 明确错误，需要修复 |

### 跳过的文件类型

- CSS/SCSS 样式文件
- 图片文件 (PNG, JPG, SVG, GIF)
- 文本内容不足50字符的文件
